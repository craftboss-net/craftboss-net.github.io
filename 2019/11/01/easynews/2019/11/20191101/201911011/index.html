飼かっている犬いぬには必かならず狂犬病きょうけんびょうのワクチンの注射ちゅうしゃをして

发表于 2019-11-01 更新于 2019-12-03 分类于 JapanEasyNews



東京都とうきょうと瑞穂町みずほまちで今年ことし８月がつ、高校生こうこうせいが犬いぬに足あしをかまれました。警察けいさつがこの犬いぬと一緒いっしょに散歩さんぽしていた人ひとを調しらべて、犬いぬが狂犬病きょうけんびょうのワクチンを受うけていないことがわかりました。検査けんさした結果けっか、この犬いぬは狂犬病きょうけんびょうになっていませんでした。
犬いぬを飼かっている人ひとは１年ねんに１回かい、犬いぬに狂犬病きょうけんびょうのワクチンを注射ちゅうしゃしなければならないと法律ほうりつで決きまっています。この犬いぬを飼かっていた人ひとは「必かならず注射ちゅうしゃをしなければならないとは思おもっていませんでした」と話はなしていました。警察けいさつは調しらべたことを検察庁けんさつちょうに伝つたえました。
狂犬病きょうけんびょうは人ひとにもうつる病気びょうきです。狂犬病きょうけんびょうの犬いぬにかまれて、ウイルスが傷きずから体からだの中なかに入はいって病気びょうきになると、ほとんどの人ひとが亡なくなります。
厚生労働省こうせいろうどうしょうは、犬いぬを飼かっている人ひとは住すんでいる所ところの役所やくしょに犬いぬの登録とうろくをして、犬いぬには必かならず狂犬病きょうけんびょうのワクチンを注射ちゅうしゃするように言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

飼う

  • 動物どうぶつにえさをあたえて育そだてる。

狂犬病

  • 犬いぬの伝染病でんせんびょう。人ひとやほかの動物どうぶつにもうつり、狂犬きょうけんにかまれると、運動神経うんどうしんけいがおかされる。

ワクチン

  • 感染症かんせんしょうのもとになるばいきんなどから作つくった薬くすり。これを接種せっしゅして、その感染症かんせんしょうにかからないようにする。

検査

  • 悪わるいところがないかを調しらべること。

結果

  • あることがもとになって起おこったことがらやようす。

検察庁

  • 犯罪はんざいを調しらべ、裁判所さいばんしょにうったえる仕事しごとをする役所やくしょ。検察官けんさつかんは、ここで仕事しごとをしている。

ウイルス

  • 日本脳炎にほんのうえんやインフルエンザなどの病気びょうきを起おこす、ふつうの顕微鏡けんびきょうでは見みえないような、非常ひじょうに小ちいさな生物せいぶつ。ビールス。

傷

  • 打うったり切きったりして、皮膚ひふや肉にくをいためたところ。
  • 品物しなもののいたんだところ。
  • 悪わるい点てん。欠点けってん。

厚生労働省

  • 国民こくみんの健康けんこうや生活せいかつを守まもる仕事しごとや、労働者ろうどうしゃが仕事しごとを見みつけるのを助たすけたり、労働者ろうどうしゃを保護ほごしたりする国くにの役所やくしょ。厚労省こうろうしょう。

役所

  • 国くにや、地方公共団体ちほうこうきょうだんたいの仕事しごとをする所ところ。官庁かんちょう。役場やくば。

登録

  • 役所やくしょなどに届とどけ出でて、おおやけに認みとめてもらうこと。
