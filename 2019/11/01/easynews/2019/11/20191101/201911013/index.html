「火事かじで焼やけた首里城しゅりじょうをもう一度いちど建たててほしい」

发表于 2019-11-01 更新于 2019-12-04 分类于 JapanEasyNews



沖縄県おきなわけん那覇市なはしにある首里城しゅりじょうが火事かじになりました。焼やけた建物たてものをもう一度いちど建たてるために、いろいろな所ところでお金かねを集あつめています。
熊本県くまもとけんは県庁けんちょうなどに募金箱ぼきんばこを置おいて、お金かねを集あつめています。熊本県くまもとけんの知事ちじは「熊本くまもとの人ひとたちの気持きもちを沖縄県おきなわけんに届とどけたいです」と話はなしていました。
首里城しゅりじょうと同おなじくユネスコの世界遺産せかいいさんになっている寺てらがある和歌山県わかやまけん高野町こうやちょうでもお金かねを集あつめています。高野町こうやちょうの職員しょくいんは「町まちの寺てらも昔むかし、火事かじになりました。１日にちも早はやく建たてることができるように願ねがっています」と話はなしていました。
東京とうきょうの銀座ぎんざにある沖縄県おきなわけんの店みせには、募金ぼきんしたいという連絡れんらくがたくさんきています。この店みせの店長てんちょうは「首里城しゅりじょうは沖縄おきなわのシンボルでした」と言いっています。
ユネスコも「世界せかいから専門家せんもんかを集あつめて、建物たてものをもう一度いちど建たてるための手伝てつだいをしたいです」と話はなしています。
