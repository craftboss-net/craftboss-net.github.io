２０２０年度ねんどの大学だいがく入学試験にゅうがくしけん　英語えいごの民間みんかん試験しけんは使つかわない

发表于 2019-11-01 更新于 2019-12-03 分类于 JapanEasyNews



文部科学省もんぶかがくしょうは２０２０年度ねんどから、大学だいがくに入学にゅうがくするための試験しけんを変かえます。英語えいごの試験しけんでは、民間みんかんの会社かいしゃが行おこなっている「英検えいけん」や「ＴＯＥＦＬ　ｉＢＴ」なども使つかう予定よていでした。今いままでの試験しけんでは読よむ問題もんだいと聞きく問題もんだいだけでしたが、これからはコミュニケーションの力ちからが大事だいじなため、話はなす問題もんだいと書かく問題もんだいがある民間みんかんの試験しけんを使つかうことにしました。
しかし、萩生田はぎうだ文部科学もんぶかがく大臣だいじんは１１月がつ１日ついたち、「みんなが平等びょうどうに安心あんしんして試験しけんを受うけることができるような準備じゅんびが十分じゅうぶんにできていません。２０２０年度ねんどは民間みんかんの試験しけんを使つかいません。これから新あたらしい試験しけんのためにしっかり準備じゅんびします」と言いいました。
高校こうこうの校長こうちょうたちは、試験しけんの方法ほうほうなどについての情報じょうほうが十分じゅうぶんに出でていないため、２０２０年度ねんどは行おこなわないでほしいと言いっていました。

民間

平等
