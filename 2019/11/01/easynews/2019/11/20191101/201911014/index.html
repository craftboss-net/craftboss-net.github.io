香港ほんこん　ハロウィーンでお面めんやマスクをした若わかい人ひとが政府せいふに抗議こうぎ

发表于 2019-11-01 更新于 2019-12-03 分类于 JapanEasyNews



香港ほんこんの政府せいふは１０月がつの初はじめ、マスクなどで顔かおが見みえないようにしてデモなどを行おこなうことを禁止きんしする規則きそくを作つくりました。このため１０月がつ３１日にち、ハロウィーンにみんなでお面めんやマスクをして、政府せいふに抗議こうぎしようという大勢おおぜいの若わかい人ひとが香港島ほんこんとうに集あつまりました。
アニメのキャラクターの服ふくを着きた人ひとのほかに、中国ちゅうごくの習しゅう近平きんぺい国家主席こっかしゅせきや香港ほんこんの政府せいふの林鄭りんてい月娥げつが行政長官ぎょうせいちょうかんの顔かおのお面めんをした人ひともいました。そして、「規則きそくをやめろ」とか「警察けいさつの暴力ぼうりょくを許ゆるしてはいけない」などとみんなで言いいました。
九龍半島きゅうりゅうはんとうでは、マスクをした若わかい人ひとが道みちでれんがを投なげたり、火ひをつけたりしました。信号機しんごうきを壊こわす人ひともいました。警察けいさつは目めが痛いたくなるガスを使つかって、夜よる遅おそくまで若わかい人ひとたちを止とめようとしました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ハロウィーン

  • 聖人せいじんを記念きねんするキリスト教きょうの行事ぎょうじ（十一月じゅういちがつ一日ついたち）の前夜祭ぜんやさい。アメリカなどではカボチャをくりぬいてかざったり、子こどもたちが仮装かそうして家々いえいえを回まわったりする。

面

  • 人ひとの顔かお。
  • 顔かおにつけるもの。おめん。
  • 名誉めいよ。ほまれ。
  • 向むき合あう。
  • 向むき。そちらのほう。
  • おもて。うわべ。
  • 平ひらたいもの。平たいらな広ひろがりをもったもの。
  • テニスコートや鏡かがみ・すずりなど、平ひらたいものを数かぞえることば。

マスク

  • お面めん。仮面かめん。
  • ほこりなどを吸すわないように、口くちや鼻はなをおおう布ぬの。
  • ガスなどを吸すわないように、消防士しょうぼうしなどが顔かおにつけるお面めん。防毒面ぼうどくめん。
  • 野球やきゅう・ソフトボールで、キャッチャーや球審きゅうしんがかぶるもの。
  • 顔かお。顔かおだち。

抗議

  • 相手あいてに対たいして反対はんたいの考かんがえを強つよく言いうこと。また、反対はんたいの意見いけん。

デモ

  • 英語えいごの「デモンストレーション」の略りゃく。自分じぶんたちの考かんがえを相手あいてに認みとめさせようとして、大勢おおぜいが集あつまって自分じぶんたちの勢いきおいを示しめすこと。また、そのための行進こうしん。

禁止

  • してはいけないと止とめること。

アニメーション

  • かいた絵えや人形にんぎょうを、動うごきに従したがって一ひとこま一ひとこま撮影さつえいし、それを映うつして実際じっさいに動うごいているように見みせる映画えいが。動画どうが。アニメ。

キャラクター

  • 性格せいかく。持もち味あじ。
  • まんが・映画えいが・劇げきなどに出でてくる人物じんぶつや動物どうぶつ。

暴力

  • 力ちからでする、乱暴らんぼうな行おこない。

許す

  • とがめないで済すませる。
  • してもよいと認みとめる。
  • 聞きき入いれる。受うけ入いれる。
  • 気きをゆるめる。打うち解とける。

れんが

  • 粘土ねんどに砂すなや石灰せっかいなどを混まぜて、水みずで練ねり固かため、長四角ながしかくの形かたちに焼やいたもの。建築けんちくなどに使つかう。

信号機

  • 道路どうろや鉄道てつどうなどで、安全あんぜん・危険きけん・注意ちゅういなどを合図あいずする機械きかい。信号しんごう。シグナル。
