ロボットが病院びょういんで漫才まんざいを見みせる

发表于 2019-01-23 更新于 2019-11-30 分类于 JapanEasyNews


神戸市こうべしにある甲南大学こうなんだいがくの研究けんきゅうグループは、漫才まんざいをするロボットの研究けんきゅうをしています。漫才まんざいは、２人ふたりがおもしろい会話かいわをして、見みている客きゃくに楽たのしんでもらう芸げいです。
ロボットの名前なまえは「あいちゃん」と「ゴン太た」です。漫才まんざいのテーマを伝つたえると、ロボットはインターネットで情報じょうほうを集あつめて、どんな会話かいわをするか考かんがえます。
２１日にち、ロボットは大阪市おおさかしにある病院びょういんで、がんという病気びょうきの人ひとたちに漫才まんざいを見みせました。ロボットはみんなに笑わらってもらうために一生懸命いっしょうけんめい漫才まんざいをしました。
７２歳さいの女性じょせいは「よくわからない部分ぶぶんもありましたが、このロボットが来きたら楽たのしいと思おもいます」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ロボット

  • 電気でんきや磁気じきの力ちからで動うごく人形にんぎょう。人造じんぞう人間にんげん。
  • 工場こうじょうなどで人間にんげんに代かわって、作業さぎょうする機械きかい。
  • （いつも）人ひとの言いいなりになって動うごく人ひと。

グループ

  • 仲間なかま。集団しゅうだん。

芸

  • 習ならって身みにつけた、わざ。
  • 芸術げいじゅつのこと。
  • 草木くさきを植うえる。

テーマ

  • 作品さくひんにかかれていることの中心ちゅうしん。また、作品さくひんの中心ちゅうしんになっている考かんがえ方かた。主題しゅだい。
  • 論文ろんぶんや、演説えんぜつの題目だいもく。
  • 一ひとつの曲きょくの中心ちゅうしんとなっているメロディー。

情報

  • できごとやものごとのようすについての知しらせ。

がん

  • 悪性あくせいのはれものができる病気びょうき。体からだの中なかにできたがん細胞さいぼうがどんどん増ふえて体からだに害がいをあたえる。

部分

  • 全体ぜんたいを、いくつかに分わけたものの一ひとつ。
