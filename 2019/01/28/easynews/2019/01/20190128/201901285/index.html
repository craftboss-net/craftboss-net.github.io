テニスの大坂おおさかなおみ選手せんしゅが世界せかいランキングの１番ばんになる

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


２６日にち、オーストラリアであったテニスの大会たいかいで、日本にっぽんの大坂おおさかなおみ選手せんしゅがチェコのペトラ?クビトバ選手せんしゅに勝かって、優勝ゆうしょうしました。この大会たいかいは世界せかいの大おおきな４つの大会たいかいの１つです。大坂おおさか選手せんしゅは去年きょねん９月がつのアメリカの大会たいかいでも優勝ゆうしょうしていて、世界せかいの大おおきな４つの大会たいかいのうち２つで続つづけて優勝ゆうしょうしました。
２７日にち、大坂おおさか選手せんしゅは集あつまった記者きしゃたちに「すごくうれしい気持きもちと、少すこしびっくりした気持きもちがあります」と話はなしました。
２８日にち、大坂おおさか選手せんしゅが女子じょしのシングルスで世界せかいランキングの１番ばんになったと発表はっぴょうがありました。シングルスで１番ばんになったアジアの選手せんしゅは、今いままで男子だんしでも女子じょしでもいませんでした。
