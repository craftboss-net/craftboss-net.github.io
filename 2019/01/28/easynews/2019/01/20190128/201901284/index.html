「嵐あらし」が来年らいねんでグループの仕事しごとを休やすむ「自由じゆうな生活せいかつをしたい」

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


アイドル グループの「嵐あらし」は、２７日にち、グループでする仕事しごとは来年らいねんの１２月がつ３１日にちまでで、そのあとは休やすむと発表はっぴょうしました。
リーダーの大野おおの智さとしさんは「２０１７年ねん、自分じぶんが『嵐あらし』というグループでする仕事しごとを終おわりにしたいとほかの４人にんに話はなしました。そのあと何なん度ども話はなし合あって、２０２０年ねんで休やすむことになりました」と説明せつめいしました。
大野おおのさんは「自由じゆうな生活せいかつをしてみたいです。この仕事しごとを一度いちど離はなれて、今いままで見みたことがない景色けしきを見みてみたいです」と理由りゆうを説明せつめいしました。そして、芸能げいのう人じんの仕事しごとも休やすみたいと話はなしました。
「嵐あらし」の二宮にのみや和也かずなりさん、櫻井さくらい翔しょうさん、相葉あいば雅紀まさきさん、松本まつもと潤じゅんさんは「リーダーの気持きもちを大切たいせつにして決きめたことです」、「２０２０年ねんまで、５人にんでファンのみなさんと楽たのしい時間じかんを過すごすことができるようにしたいです」と話はなしました。
