深ふかい海うみにいる「リュウグウノツカイ」が兵庫県ひょうごけんで見みつかる

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


兵庫県ひょうごけん豊岡市とよおかしの海うみで２３日にち、魚さかなをとる網あみの中なかに「リュウグウノツカイ」という魚さかなが入はいっていました。体からだの長ながさは４ｍ４２ｃｍで、重おもさは３２ｋｇありました。
リュウグウノツカイは、世界せかいの海うみの深ふかさが２００ｍ以上いじょうの所ところにいると言いわれていますが、どんな魚さかなかよくわかっていません。
豊岡市とよおかしにある水族館すいぞくかんの「城崎きのさきマリンワールド」では、この魚さかなが何なにを食たべているのかおなかの中なかを調しらべます。そして来月らいげつ、この魚さかなが生いきていたときと同おなじような形かたちにして、みんなに見みせる予定よていです。
水族館すいぞくかんの人ひとは「４ｍ以上いじょうのリュウグウノツカイが見みつかることは珍めずらしいのでびっくりしています。どんな魚さかなか少すこしでもわかるように調しらべたいです」と話はなしていました。
