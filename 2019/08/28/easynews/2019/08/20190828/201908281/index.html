大阪おおさか大学だいがく　ストレスを測はかることができる物質ぶっしつを見みつける

发表于 2019-08-28 更新于 2019-11-30 分类于 JapanEasyNews



大阪おおさか大学だいがくの研究けんきゅうグループは、１００人にんぐらいの男性だんせいに、ストレスがどのくらいあるか聞ききました。血液けつえきの中なかにある「αクロトー」という物質ぶっしつの濃度のうども一緒いっしょに調しらべました。
調しらべた結果けっか、αクロトーの濃度のうどがほかの人ひとより２０％から３０％ぐらい高たかい人ひとがいました。「ストレスを少すくなくすることができない」とか「寝ねても体からだは十分じゅうぶん休やすんでいない」と答こたえた人ひとです。研究けんきゅうグループは、ストレスが多おおいと濃度のうどが高たかくなると考かんがえています。
ストレスがどのくらいあるかを調しらべるとき、今いままでは、質問しつもんして答こたえてもらっていました。研究けんきゅうグループは、αクロトーの濃度のうどで、ストレスを測はかることができるようにしたいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ストレス

  • 心こころや気持きもちに悪わるい影響えいきょうをあたえる、いろいろなしげき。そのために体からだの調子ちょうしや気分きぶんが、ふだんと変かわる。

測る

  • 長ながさ・高たかさ・深ふかさ・広ひろさなどを調しらべる。

物質

  • 見みたりさわったりすることができるもの。品物しなもの。

グループ

  • 仲間なかま。集団しゅうだん。

血液

  • 血ち。

濃度

  • 液体えきたいや気体きたいのこい、うすいの度合どあい。

結果

  • あることがもとになって起おこったことがらやようす。
