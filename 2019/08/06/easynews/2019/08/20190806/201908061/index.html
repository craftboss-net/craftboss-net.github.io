北海道ほっかいどう根室市ねむろし「さんまは９月がつまであまりとれないだろう」

发表于 2019-08-06 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどうでは、８月がつの終おわりから大おおきな船ふねを使つかってさんまをとる漁りょうが始はじまります。根室市ねむろしは、漁りょうをする人ひとたちを集あつめて、今年ことしのさんまについて説明せつめいする会議かいぎを開ひらきました。
会議かいぎでは、６月がつから７月がつに国くにが行おこなった調査ちょうさの説明せつめいがありました。調査ちょうさした人ひとは「日本にっぽんの近ちかくに来くるさんまは、去年きょねんより３０％少すくなくて、特とくに９月がつまではあまりとれないと思おもいます。さんまも小ちいさいかもしれません」と話はなしました。
大おおきな船ふねを持もっている人ひとは「これからたくさんとりたいと思おもっていたので、説明せつめいを聞きいて心配しんぱいです」と話はなしていました。
