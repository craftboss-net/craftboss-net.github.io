ゴルフの大会たいかい「全英ぜんえい女子じょしオープン」で渋野しぶの選手せんしゅが優勝ゆうしょう

发表于 2019-08-05 更新于 2019-11-30 分类于 JapanEasyNews



イギリスで、ゴルフのとても有名ゆうめいな大会たいかいの「全英ぜんえい女子じょしオープン」がありました。この大会たいかいで、日本にっぽんの２０歳さいの渋野しぶの日向子ひなこ選手せんしゅが優勝ゆうしょうしました。外国がいこくのとても有名ゆうめいなゴルフの大会たいかいで日本人にっぽんじんが優勝ゆうしょうしたのは、１９７７年ねんの樋口ひぐち久子ひさこさんに続つづいて２人ふたり目めです。
渋野しぶの選手せんしゅは「今いまの気持きもちを言葉ことばにすることができません」と笑わらって話はなしました。
渋野しぶの選手せんしゅは岡山おかやま市しの出身しゅっしんです。去年きょねん、プロになるためのテストに合格ごうかくしたばかりです。渋野しぶの選手せんしゅが外国がいこくの大会たいかいに出でたのは初はじめてでした。
イギリスのテレビ局きょくのＢＢＣは「本当ほんとうに驚おどろきました。すばらしい優勝ゆうしょうでした」と伝つたえました。
