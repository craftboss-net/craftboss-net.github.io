ラグビーのワールドカップまで１か月げつ　大会たいかいをＰＲする

发表于 2019-08-21 更新于 2019-11-30 分类于 JapanEasyNews



９月がつ２０日はつかに始はじまるラグビーのワールドカップの日本にほん大会たいかいまで１か月げつになりました。試合しあいをする会場かいじょうの近ちかくでは、大会たいかいをたくさんの人ひとに知しってもらうため、幕まくなどを飾かざってＰＲしています。
東京都とうきょうと調布市ちょうふしにある東京とうきょうスタジアムでは、大会たいかいの最初さいしょの試合しあいの日本にっぽんとロシアの試合しあいを行おこないます。この会場かいじょうでは、大会たいかいのマークやキャラクターの絵えがかいてある長ながさ２．７ｍの幕まくを飾かざりました。
近ちかくにある商店街しょうてんがいには、大会たいかいのマークがかいてある旗はたをたくさん飾かざりました。まちの人ひとたちは、外国がいこくからもたくさんの人ひとが試合しあいを見みに来きてくれると楽たのしみにしています。小学生しょうがくせいの男おとこの子こは「日本にっぽんに決勝けっしょうまで進すすんでほしいです」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ラグビー

  • 一チーム１５人にんずつの選手せんしゅが、楕円形だえんけいのボールを、持もって走はしったり、けったりして相手側あいてがわのゴールライン内ないに入いれ、得点とくてんを争あらそう競技きょうぎ。

ワールドカップ

  • スポーツ競技きょうぎの世界選手権大会せかいせんしゅけんたいかいの優勝者ゆうしょうしゃにあたえられるカップ。また、そのカップを争あらそう大会たいかい。W杯ダブリューはい。

大会

  • 多おおくの人ひとが集あつまる会かい。
  • ある組織そしきの全員ぜんいんが集あつまる会かい。

PR

  • 仕事しごとの内容ないようや商品しょうひんのよさなどを、多おおくの人ひとに知しらせること。宣伝せんでん。

幕

  • 仕切しきりやかざりに使つかう、はばの広ひろい布ぬの。
  • 劇げきのひと区切くぎり。
  • 場面ばめん。場合ばあい。
  • ものごとの終おわり。
  • すもうで、幕内まくうちのこと。
  • 幕府ばくふ。

マーク

  • しるし。記号きごう。また、しるしをつけること。
  • 特とくに目めをつけて注意ちゅういすること。
  • 記録きろくを作つくること。

キャラクター

  • 性格せいかく。持もち味あじ。
  • まんが・映画えいが・劇げきなどに出でてくる人物じんぶつや動物どうぶつ。

商店街

  • 商店しょうてんがたくさん並ならんでいる通とおり。

旗

  • 布ぬのや紙かみで作つくり、さおなどの先さきにつけて、かざりや目めじるしとするもの。

決勝

  • 最後さいごの勝かち負まけを決きめること。
  • 第だい一位いちいのものを決きめること。
