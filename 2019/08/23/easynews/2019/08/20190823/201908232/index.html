外国がいこくのパラリンピックの選手せんしゅが東京とうきょうに来きて困こまったこと

发表于 2019-08-23 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねん８月がつ２５日にちに東京とうきょうでパラリンピックが始はじまります。体からだなどに障害しょうがいがある選手せんしゅがたくさん来きます。
東京とうきょうでは今月こんげつ、体からだなどに障害しょうがいがある人ひとたちの卓球たっきゅうの世界せかい大会たいかいがあって、１８０人にんぐらいの選手せんしゅが来きました。
ドイツの選手せんしゅたちは１週間しゅうかんぐらい東京とうきょうにいて、電車でんしゃでいろいろな所ところに行いきました。東京とうきょうのまちについて聞きくと、パラリンピックに７回かい出でた５４歳さいの選手せんしゅは「駅えきの入いり口ぐちに、エレベーターの場所ばしょや行いき方かたを書かいた看板かんばんが必要ひつようだと思おもいます。エレベーターの数かずも少すくないので、車くるまいすを使つかっている人ひとがたくさん来きたら大変たいへんだと思おもいます」と話はなしていました。
別べつの選手せんしゅは「英語えいごが全然ぜんぜんわからない人ひとがいて、道みちを聞きいたとき大変たいへんでした」と話はなしていました。
