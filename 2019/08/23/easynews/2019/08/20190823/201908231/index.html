７月がつに韓国かんこくから日本にっぽんへ旅行りょこうに来きた人ひとが去年きょねんより７．６％減へる

发表于 2019-08-23 更新于 2019-11-30 分类于 JapanEasyNews



日本政府観光局にほんせいふかんこうきょくによると、今年ことし７月がつに日本にっぽんへ旅行りょこうに来きた外国人がいこくじんは約やく２９９万まん人にんで、去年きょねんの７月がつより５．６％増ふえました。中国ちゅうごくから来きた人ひとが最もっとも多おおい約やく１０５万まん人にんで、去年きょねんより２０％ぐらい増ふえたためです。
しかし、２番目ばんめに多おおい韓国かんこくから来きた人ひとは約やく５６万まん人にんで、７．６％減へりました。今年ことし１月がつから７月がつに韓国かんこくから来きた人ひとも、去年きょねんより４．３％減へりました。
日本にっぽんと韓国かんこくの関係かんけいが悪わるくなっているため、旅行りょこうに来くる人ひとが減へっているようです。政府せいふの人ひとは「日本人にっぽんじんと韓国人かんこくじんが旅行りょこうなどで交流こうりゅうして、相手あいてを理解りかいしていくことが大切たいせつだと思おもいます」と話はなしていました。
