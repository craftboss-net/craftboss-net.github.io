海うみで流ながされた男性だんせいを６９歳さいと６６歳さいの姉あねと妹いもうとが助たすける

发表于 2019-08-22 更新于 2019-11-30 分类于 JapanEasyNews



今月こんげつ１８日にち、新潟市にいがたしの海うみで、男性だんせいが海岸かいがんから１００ｍぐらい遠とおくに流ながされました。男性だんせいを見みつけた６９歳さいの小旗おばたはるみさんと、妹いもうとの６６歳さいの河合かわい京子きょうこさんが泳およいで行いって、男性だんせいを助たすけました。
２人ふたりは２０日はつか、警察けいさつから賞状しょうじょうをもらいました。小旗おばたさんは１週間しゅうかんに３回かいぐらいプールで泳およいでいて、河合かわいさんの趣味しゅみはスキューバダイビングです。２人ふたりは泳およぐことに自信じしんがあると話はなしていました。
小旗おばたさんは「波なみが高たかかったので、助たすけに行いこうかどうしようか考かんがえました。男性だんせいが遠とおくへ流ながされていったので、妹いもうとに『行いくぞ』と言いって助たすけに行いきました。いま考かんがえると、無理むりなことをしたかもしれないと思おもいます」と話はなしていました。
