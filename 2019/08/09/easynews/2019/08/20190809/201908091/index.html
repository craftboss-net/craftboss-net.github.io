がんが見みつかってから治療ちりょうで５年ねん以上いじょう生いきている人ひとは６６％

发表于 2019-08-09 更新于 2019-11-30 分类于 JapanEasyNews



国立こくりつがん研究けんきゅうセンターは、２０１０年ねんまでの２年ねんの間あいだに２７７の病院びょういんで「がん」の治療ちりょうをした５７万まん人にんについて調しらべました。
治療ちりょうをした人ひとの中なかで、がんが見みつかってから５年ねん以上いじょう生いきている人ひとは６６％いました。この割合わりあいで、どのくらいの人ひとを治療ちりょうで助たすけることができたかがわかります。
割合わりあいがいちばん高たかかったのは、男性だんせいだけにある前立腺ぜんりつせんのがんで９９％でした。次つぎに高たかかった乳にゅうがんは９３％でした。女性じょせいだけのがんの子宮体しきゅうたいがんは８２％、子宮頸しきゅうけいがんは７５％でした。日本にっぽんで多おおい大腸だいちょうがんは７３％、胃いがんは７２％、ぼうこうがんは７０％でした。
５年ねん以上いじょう生いきている人ひとの割合わりあいがいちばん低ひくかったのは、すい臓ぞうがんで１０％でした。次つぎに低ひくかった肝臓かんぞうがんは４０％、肺はいがんは４１％、食道しょくどうがんは４４％でした。

がん

  • 悪性あくせいのはれものができる病気びょうき。体からだの中なかにできたがん細胞さいぼうがどんどん増ふえて体からだに害がいをあたえる。

治療

  • 病気びょうきやけがの手当てあてをして治なおすこと。

割合

  • 物ものと物ものとの関係かんけいを、数かずで表あらわしたもの。割わり。率りつ。歩合ぶあい。
  • 思おもいのほか。比較的ひかくてき。

助ける

  • 危あぶないことや、苦くるしいことから、救すくう。
  • 力ちからを貸かす。手伝てつだう。

乳房

  • 女おんなの人ひとや哺乳動物ほにゅうどうぶつの雌めすの、乳ちちを出だすところ。にゅうぼう。

大腸

  • 消化器しょうかきの一ひとつ。小腸しょうちょうに続つづき、肛門こうもんに至いたる部分ぶぶん。おもに水分すいぶんを吸すい取とるはたらきをする。

胃

  • 口くちから食道しょくどうでつながっている消化器官しょうかきかん。食たべたものをこなすところ。胃いぶくろ。

ぼうこう

  • 腹はらの下したのほうにある器官きかん。じん臓ぞうから送おくられてくるにょうを、一時いちじためておくところ。

膵臓

  • 胃いの後うしろにある内臓ないぞう。食たべ物ものをこなすはたらきをする、すい液えきを作つくり出だす。

肝臓

  • 胃いの右上みぎうえにある器官きかん。胆汁たんじゅうを出だして消化しょうかを助たすけたり、血液けつえきの中なかの毒物どくぶつをこわしたり、余分よぶんな栄養えいようをたくわえたりする。

肺

  • 胸むねの左ひだりと右みぎにあって、息いきを吸すったり、はいたりするはたらきをするところ。肺臓はいぞう。

食道

  • のどと胃いをつなぎ、食たべた物ものを胃いに送おくる管くだ。
