台風たいふう１０号ごうが来くる　太平洋たいへいよう側がわの海岸かいがんでは気きをつけて

发表于 2019-08-09 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうによると、台風たいふう１０号ごうが東京とうきょうから離はなれた南みなみの太平洋たいへいようから北西ほくせいに向むかって進すすんでいます。そして１０日とおかから１１日にちの間あいだに、小笠原諸島おがさわらしょとうの近ちかくに来きそうです。とても強つよい風かぜが吹ふいて、波なみが高たかくなりそうなので、気きをつける必要ひつようがあります。
そのあとも日本にっぽんに向むかって進すすんで、１４日じゅうよっかごろには本州ほんしゅうから九州きゅうしゅうの間あいだの海岸かいがんに来きそうです。台風たいふうが近ちかくに来きていなくても、波なみが高たかくなる危険きけんがあるので、海うみに行いくときは十分じゅうぶんに気きをつけてください。
気象庁きしょうちょうや県けんなどが出だす台風たいふうについての新あたらしい情報じょうほうをチェックしてください。「ＮＥＷＳ　ＷＥＢ　ＥＡＳＹ」の「台風たいふう」のページも読よんでください。
