空そらを飛とぶことができる車くるまのテストをする

发表于 2019-08-07 更新于 2019-11-30 分类于 JapanEasyNews



ＮＥＣは空そらを飛とぶことができる車くるまの研究けんきゅうをしています。５日いつか、まだ人ひとが乗のっていない車くるまを飛とばすテストをしました。
車くるまの長ながさは４ｍぐらいで、重おもさは１５０ｋｇです。電気でんきで動うごくモーターで４つのプロペラを回まわして飛とびます。ＮＥＣの人ひとが無線むせんでモーターを動うごかすと、車くるまがまっすぐ上うえに浮うきました。そして、３ｍぐらいの高たかさまでゆっくり上あがりました。
空そらを飛とんで人ひとや荷物にもつを運はこぶ車くるまは、道みちが混こまないようにするために役やくに立たちます。人口じんこうが少すくない町まちの交通こうつうも便利べんりになると言いわれています。ＮＥＣは、２０２３年ねんに荷物にもつを運はこぶテストをして、そのあと、人ひとが乗のることができるようにしたいと考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

飛ばす

  • 飛とぶようにする。
  • 散ちらす。
  • 走はしらせる。
  • 間あいだをぬいて、先さきに進すすむ。
  • 言いいふらす。
  • 勢いきおいよく…する。

モーター

  • 電気でんきや蒸気じょうき・ガソリンなどで、物ものを動うごかす力ちからを起おこす機械きかい。電動機でんどうき。発動機はつどうき。
  • 自動車じどうしゃ。

プロペラ

  • 軸じくの周まわりを回まわる、回転羽根かいてんばね。飛行機ひこうきや船ふねなどを動うごかすもの。

回す

  • 円えんをかくように動うごかす。
  • 順じゅんに送おくる。
  • 必要ひつようとするところへふり向むける。さし向むける。
  • 周まわりを取とり巻まくようにする。
  • 行いき届とどかせる。

無線

  • 通信つうしんや放送ほうそうで、電波でんぱを使つかい、電線でんせんを使つかわないこと。
  • 「無線電信むせんでんしん」や「無線電話むせんでんわ」の略りゃく。

動かす

  • 動うごくようにする。
  • 位置いちを変かえる。
  • 感動かんどうさせる。

浮く

  • 水みずの上うえにある。うかぶ。
  • 地面じめんからはなれて、空中くうちゅうにある。
  • 表面ひょうめんに現あらわれる。
  • 心こころがはずむ。
  • お金かねや時間じかんの余あまりが出でる。
  • しっかりと、くっついていない。
  • 相手あいてにされなくなる。
