暑あつい日ひが続つづく　熱中症ねっちゅうしょうに気きをつけて

发表于 2019-08-07 更新于 2019-11-30 分类于 JapanEasyNews



７日なのかも日本にっぽんのいろいろな所ところで３５℃以上いじょうになりました。暑あつい日ひが続つづいているので、熱中症ねっちゅうしょうにならないように、よく水みずを飲のんだり、エアコンで部屋へやを涼すずしくしたりしてください。
国くにによると、８月がつ４日よっかまでの１週間しゅうかんに、熱中症ねっちゅうしょうで病院びょういんに運はこばれた人ひとは１万まん８０００人にん以上いじょういました。その前まえの１週間しゅうかんの３倍ばい以上いじょうです。この中なかの５７人にんが亡なくなって、７３０人にんぐらいが長ながい入院にゅういんが必要ひつようになりました。病院びょういんに運はこばれた人ひとは６５歳さい以上いじょうの人ひとが多おおくて、１万まん人にんぐらいいました。熱中症ねっちゅうしょうになった場所ばしょでいちばん多おおかったのは、家いえの中なかでした。
これからも暑あつい日ひが続つづきそうです。熱中症ねっちゅうしょうにならないために気きをつけることは「ＮＥＷＳ　ＷＥＢ　ＥＡＳＹ」の「熱中症ねっちゅうしょう」のページにも書かいてあります。
