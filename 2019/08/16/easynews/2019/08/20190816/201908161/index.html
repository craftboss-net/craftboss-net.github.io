動物園どうぶつえんの猿さるに果物くだものが入はいった氷こおりをプレゼント

发表于 2019-08-16 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんではとても暑あつい日ひが続つづいていて、４０℃以上いじょうになった所ところもあります。
長野県ながのけんにある飯田市立動物園いいだしりつどうぶつえんでは毎年まいとし、夏なつの暑あついときに氷こおりを猿さるにプレゼントしています。
１４日じゅうよっか、猿さるが好すきなバナナやすいかなどを入いれた氷こおりを２０ｋｇ用意よういしました。猿さるに食たべ物ものをあげる午後ごご２時じには３２℃以上いじょうになりました。動物園どうぶつえんの人ひとが氷こおりをあげると、猿さるはおいしそうに食たべていました。
横浜市よこはましから来きた女おんなの子こは「暑あついので、私わたしも冷つめたいアイスやかき氷ごおりなどを食たべたいです」と話はなしていました。

猿

  • 森林しんりんにすみ、木登きのぼりがうまく、群むれを作つくって暮くらしている動物どうぶつ。チンパンジー・ゴリラ・オランウータン・テナガザルなど種類しゅるいが多おおい。日本にっぽんにはニホンザルがいる。

氷

  • 温度おんどが０度れいどより低ひくくなって、水みずが固かたまったもの。

西瓜

  • 畑はたけに作つくる作物さくもつ。夏なつ、大おおきな実みが、地面じめんをはうつるにでき、中身なかみは赤あかや黄色きいろで水分すいぶんが多おおくあまい。

アイスクリーム

  • 牛乳ぎゅうにゅう・砂糖さとう・卵たまごの黄身きみなどをかき混まぜて、こおらせた食たべ物もの。アイス。
