「百日ひゃくにちせき」になる人ひとが増ふえる　赤あかちゃんは特とくに気きをつけて

发表于 2019-08-14 更新于 2019-11-30 分类于 JapanEasyNews



「百日ひゃくにちせき」は激はげしい せきが続つづく病気びょうきで、細菌さいきんが原因げんいんでうつります。生うまれてから６か月げつまでの赤あかちゃんがこの病気びょうきになると、肺炎はいえんになって亡なくなることがあります。
国くにの研究所けんきゅうじょによると、今年ことし１月がつから今月こんげつ４日よっかまでに百日ひゃくにちせきになった人ひとが１万まん人にん以上いじょうになりました。去年きょねん１年ねんでこの病気びょうきになった人ひとと同おなじぐらいです。
百日ひゃくにちせきは子こどもに多おおい病気びょうきですが、去年きょねんこの病気びょうきになった人ひとのうち２０％以上いじょうは２０歳さい以上いじょうの人ひとでした。大人おとなはこの病気びょうきになってもひどくならないため、かぜだと思おもって病院びょういんに行いかないで、周まわりの人ひとにうつしてしまうことがあります。
研究所けんきゅうじょの人ひとは「赤あかちゃんにはできるだけ早はやくワクチンの注射ちゅうしゃをしてください。赤あかちゃんが近ちかくにいる人ひとは、せきが出でたら病院びょういんに行いってください」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

百日ぜき

  • 子こどもに多おおい感染症かんせんしょうの一ひとつ。せきが、百日ひゃくにちも続つづくといわれるほど、治なおりがおそい。

激しい

  • 勢いきおいが強つよい。
  • ものごとの程度ていどがきつい。

咳

  • のどや気管きかんがしげきされて、急きゅうに激はげしく出でる強つよい息いき。

細菌

  • 一ひとつの細胞さいぼうからできている、ごく小ちいさい生物せいぶつ。ウイルスよりも大おおきい。病気びょうきのもとになる病原菌びょうげんきんや、物ものをはっこうさせる乳酸菌にゅうさんきん、くさらせるふはい菌きんなどがある。バクテリア。

肺炎

  • 細菌さいきんによって起おこる肺はいの病気びょうき。高たかい熱ねつが出でる。

研究所

  • ものごとを研究けんきゅうする所ところ。研究けんきゅうをする施設しせつ。

移す

  • 場所ばしょを変かえる。
  • ものごとを進すすめる。
  • 感染かんせんさせる。

ワクチン

  • 感染症かんせんしょうのもとになるばいきんなどから作つくった薬くすり。これを接種せっしゅして、その感染症かんせんしょうにかからないようにする。
