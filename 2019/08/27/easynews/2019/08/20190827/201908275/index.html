スマートフォンにこのメッセージが来きたら詐欺さぎかもしれない

发表于 2019-08-27 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどうに住すむ女性じょせいのスマートフォンに７月がつ、「利用りよう料金りょうきんが高額こうがくになっていて確認かくにんが必要ひつようです」というメッセージが来きました。携帯電話けいたいでんわの会社かいしゃから来きたメッセージに見みえますが、うそでした。
女性じょせいはメッセージに書かいてあったウェブサイトを開ひらいて、ＩＤやパスワードを入いれました。そのあと、買かい物ものをしていないのに、インターネットで９万まん円えん以上いじょうの買かい物ものをしたという連絡れんらくが来きました。
携帯電話けいたいでんわの会社かいしゃには、買かい物ものなどの料金りょうきんと電話でんわの料金りょうきんを一緒いっしょに払はらうことができるサービスがあります。このサービスを利用りようした詐欺さぎでした。
ＩＤやパスワードなどを盗ぬすむためのウェブサイトが急きゅうに増ふえています。このようなウェブサイトは７月がつに２０００件けん以上いじょう見みつかっていて、これからもっと増ふえそうだと言いわれています。専門家せんもんかは同おなじようなメッセージが来きたら、簡単かんたんに信しんじないでくださいと言いっています。
