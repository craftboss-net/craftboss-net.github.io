東京とうきょうパラリンピックのメダル　触さわってわかるデザイン

发表于 2019-08-26 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうパラリンピックが始はじまるまであと１年ねんになった２５日にち、メダルの発表はっぴょうがありました。
パラリンピックのメダルは、表おもても裏うらも扇子せんすのデザインになっています。「みんなの心こころを一ひとつにして、世界せかいに新あたらしい風かぜを送おくる」という意味いみです。
表おもてには「ＴＯＫＹＯ２０２０」と書かいてあって、目めの見みえない人ひとがわかるように点字てんじもあります。横よこには丸まるいくぼみがあって、くぼみの数かずでどのメダルかわかります。裏うらには岩いわ、花はな、木き、葉は、水みずの５つの自然しぜんもデザインされていて、触さわってわかるようになっています。
メダルをデザインした人ひとは「日本にっぽんらしいデザインにしたいと思おもって、扇子せんすにしました」と話はなしています。
使つかわなくなった携帯電話けいたいでんわなどから集あつめた金属きんぞくを使つかって、来年らいねん５月がつまでに全部ぜんぶで約やく２５００個このメダルを作つくります。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

メダル

  • 金属きんぞくの板いたに、絵えや文字もじなどをうきぼりにしたもの。記念品きねんひんや賞品しょうひんなどにする。

デザイン

  • 物ものを作つくるときに、形かたちや色いろなどを工夫くふうすること。
  • 図案ずあん。意匠いしょう。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

パラリンピック

  • 身体障害者しんたいしょうがいしゃの国際こくさいスポーツ大会たいかい。四年よねんに一度いちど、オリンピック開催地かいさいちで行おこなわれる。

扇子

  • あおいで風かぜを起おこすもので、折おりたたむようにできている。おうぎ。すえひろ（がり）。

点字

  • 目めの不自由ふじゆうな人ひとが、指ゆびの先さきでさわって読よむ、文字もじの代かわりになるしるし。紙かみなどの表面ひょうめんにうき出ださせた小ちいさな点てんを組くみ合あわせて使つかう。

窪み

  • 周まわりより低ひくく、へこんだ所ところ。

数

  • 一ひとつ、二ふたつ、三みっつなどと数かぞえた物ものの数量すうりょう。すう。
  • いくつもあること。たくさん。
  • 数かぞえあげる値打ねうちのあるもの。なかま。

岩

  • 石いしの大おおきなもの。岩石がんせき。

自然

  • 山やま・川かわ・草くさ・木き・星ほし・雲くも・雨あめ・雪ゆきなど、人ひとが作つくったものでない物もの。
  • もともとの性質せいしつを備そなえていること。
  • わざとらしくないようす。
  • ひとりでに。いつのまにか。

金属

  • ふつうは固体こたいで、熱ねつや電気でんきをよく伝つたえるもの。鉄てつ・銅どう・金きん・銀ぎん・アルミニウムなど。

約

  • ちかう。取とり決きめる。
  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。
