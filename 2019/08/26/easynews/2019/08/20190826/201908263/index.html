群馬県ぐんまけん大泉町おおいずみまちで外国人がいこくじんも参加さんかして地震じしんのときの訓練くんれんを行おこなう

发表于 2019-08-26 更新于 2019-11-30 分类于 JapanEasyNews



群馬県ぐんまけん大泉町おおいずみまちで２５日にち、大おおきな地震じしんがあったときのことを考かんがえて、訓練くんれんを行おこないました。大泉町おおいずみまちに住すんでいる人ひとの１８．６％は外国人がいこくじんです。訓練くんれんにはいろいろな国くにの人ひとが参加さんかして、全部ぜんぶで１０００人にんぐらい集あつまりました。
ネパール人じんは車くるまの中なかからマイクを使つかって、気きをつけることを国くにの言葉ことばで伝つたえました。ブラジル人じんの学校がっこうの先生せんせいは消火器しょうかきで火ひを消けす練習れんしゅうをしました。ベトナムやフィリピンから来きた技能実習生ぎのうじっしゅうせいは、みんなのためにほかの国くにの人ひとたちと一緒いっしょにご飯はんを作つくりました。
参加さんかしたブラジル人じんは「日本にっぽんに住すんでいる外国人がいこくじんもこのような訓練くんれんを経験けいけんしてしっかり準備じゅんびすることが大事だいじだと思おもいます」とか「どの国くにの人ひととも協力きょうりょくできるようにしたいです」などと話はなしていました。
