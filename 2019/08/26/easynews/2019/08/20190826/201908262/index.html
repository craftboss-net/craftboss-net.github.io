「はやぶさ２」のカプセルは来年らいねん１１月がつか１２月がつに地球ちきゅうに戻もどる予定よてい

发表于 2019-08-26 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのＪＡＸＡが打うち上あげた「はやぶさ２」は今いま、地球ちきゅうと火星かせいの間あいだにあるリュウグウという小ちいさい星ほしの近ちかくにいます。今いままでに２回かいこの星ほしに下おりました。そして、取とった石いしをカプセルに入いれて、地球ちきゅうに持もって帰かえる予定よていです。
ＪＡＸＡは、来年らいねん１１月がつか１２月がつに、このカプセルを「はやぶさ２」から離はなして、オーストラリアの砂漠さばくに落おとす予定よていだと発表はっぴょうしました。この砂漠さばくは、２０１０年ねんに「はやぶさ」が別べつの小ちいさい星ほしから石いしなどを持もって帰かえった所ところです。
ＪＡＸＡは「はやぶさ２が問題もんだいなく地球ちきゅうに戻もどって、砂漠さばくにカプセルを落おとすことができるように、しっかり準備じゅんびを続つづけます」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

カプセル

  • ゼラチンで作つくった、筒形つつがたの小ちいさな入いれ物もの。薬くすりなどを入いれてそのまま飲のむ。
  • 密閉みっぺいした入いれ物もの。

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから三番さんばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

打ち上げる

  • （勢いきおいよく）物ものを空中くうちゅうに上あげる。
  • 波なみが物ものを陸りくに運はこび上あげる。
  • （仕事しごとなどを）すっかり終おえる。

火星

  • 惑星わくせいの一ひとつ。太陽たいようから四番よばんめの星ほし。星ほしの色いろは赤あかく見みえ、衛星えいせいが二ふたつある。

離す

  • くっついていたものを分わける。
  • 距離きょりをあける。遠とおくへやる。

砂漠・

  • 砂地すなじがむき出だしになっている広ひろい地域ちいき。かわいた気候きこうのため植物しょくぶつが育そだちにくい。ゴビ砂漠さばく・サハラ砂漠さばくなど。
