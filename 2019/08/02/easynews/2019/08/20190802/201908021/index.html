京都きょうとアニメーションの新あたらしい映画えいが　９月がつに映画館えいがかんで上映じょうえい

发表于 2019-08-02 更新于 2019-11-30 分类于 JapanEasyNews



京都きょうとアニメーションの建物たてものに男おとこが火ひをつけて大おおきな火事かじになった事件じけんでは、絵えをかく仕事しごとをしていた人ひとなど３５人にんが亡なくなって、３３人にんがけがをしました。
この事件じけんの前まえから京都きょうとアニメーションで作つくっていた映画えいがが、９月がつ６日むいかから２６日にちまで７３の映画館えいがかんで上映じょうえいされることになりました。映画えいがは「ヴァイオレット?エヴァーガーデン外伝がいでん」です。この映画えいがは、前まえにテレビで放送ほうそうしていたアニメの新あたらしい映画えいがです。このアニメは、戦争せんそうの中なかで育そだった女おんなの子こが、誰だれかの代かわりに手紙てがみを書かく仕事しごとをして、愛あいや人ひとの気持きもちを知しる話はなしです。
事件じけんのあと、京都きょうとアニメーションで作つくった新あたらしい映画えいがが上映じょうえいされるのは初はじめてです。ファンなどはツイッターに「中止ちゅうしになるかもしれないと思おもっていたのでうれしいです」や、「必かならず見みに行いきます」などと書かいています。
