手紙てがみの宛先あてさきの書かき方かたがわからない子こどもが増ふえている

发表于 2019-08-02 更新于 2019-11-30 分类于 JapanEasyNews



文部科学省もんぶかがくしょうの今年ことしの「全国学力ぜんこくがくりょくテスト」では、中学校ちゅうがっこう３年生ねんせいの国語こくごで、手紙てがみの宛先あてさきの書かき方かたの問題もんだいが出でました。
正ただしい書かき方かたは、封筒ふうとうの右みぎに住所じゅうしょを書かいて、真まん中なかに「様さま」などをつけた相手あいての名前なまえを書かきます。しかし、正ただしく答こたえた中学生ちゅうがくせいは５７％だけでした。間違まちがった答こたえでは、相手あいての名前なまえを右みぎに書かいたり、住所じゅうしょと一緒いっしょにメールのアドレスを書かいたりしていました。７年ねん前まえのテストで正ただしく答こたえた中学生ちゅうがくせいは７４％で、今年ことしは１７ポイント低ひくくなりました。
インターネットのＳＮＳを使つかうことが増ふえて、子こどもたちが手紙てがみを書かく機会きかいが少すくなくなっているようです。
日本語にほんごを研究けんきゅうしている大学だいがくの先生せんせいは「手紙てがみだから伝つたえることができる気持きもちもあると思おもいます。子こどもたちには手紙てがみのいいところを大切たいせつにしてほしいです」と話はなしています。
