台風たいふう１０号ごう　多おおくの雨あめが降ふる危険きけんがある

发表于 2019-08-15 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうによると、台風たいふう１０号ごうは１５日にちに九州きゅうしゅうと四国しこくの間あいだを通とおって、広島県ひろしまけんに入はいりました。このあと、中国地方ちゅうごくちほうを通とおって日本海にほんかいに出でて、北東ほくとうに進すすみそうです。
四国しこくや紀伊半島きいはんとうなどでは雨あめがたくさん降ふっています。九州きゅうしゅうから北海道ほっかいどうまでの広ひろい場所ばしょでも、雨あめがたくさん降ふるかもしれません。
雨あめがたくさん降ふった場所ばしょでは、山やまが崩くずれたり川かわの水みずがあふれたりするかもしれません。強つよい風かぜや高たかい波なみにも気きをつけてください。
台風たいふうについては、気象庁きしょうちょうや市しなどが出だす新あたらしい情報じょうほうをチェックしてください。台風たいふうが来きたときに気きをつけることは「NEWS　WEB　EASY」の「台風たいふう」のページに書かいてあります。

九州地方

  • 日本にっぽんの南西部なんせいぶにある地方ちほう。福岡ふくおか・佐賀さが・長崎ながさき・熊本くまもと・大分おおいた・宮崎みやざき・鹿児島かごしま・沖縄おきなわの八県けんがある。

四国地方

  • 瀬戸内海せとないかいをはさんで本州ほんしゅうと向むかい合あっている大おおきな島しま。香川かがわ・高知こうち・愛媛えひめ・徳島とくしまの四県けんがある。

中国地方

  • 本州ほんしゅうの西部せいぶにあり、北きたは日本海にほんかいに、南みなみは瀬戸内海せとないかいに面めんする地方ちほう。岡山おかやま・広島ひろしま・山口やまぐち・島根しまね・鳥取とっとりの五県ごけんがある。中国ちゅうごく。

日本海

  • アジア大陸たいりくの東ひがしと日本列島にほんれっとうにはさまれた海うみ。

北東

  • 北きたと東ひがしの真まん中なかの方角ほうがく。東北とうほく。

紀伊半島

  • 近畿地方きんきちほうの南部なんぶにある日本にっぽんでいちばん大おおきな半島はんとう。和歌山わかやま・奈良なら・三重みえの各県かくけんをふくむ。
