グーグル　スマートフォンで見みた文章ぶんしょうを翻訳ほんやくするＡＩの技術ぎじゅつ

发表于 2019-05-09 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのカリフォルニア州しゅうで７日なのか、ＩＴの会社かいしゃのグーグルが新あたらしい技術ぎじゅつを紹介しょうかいするイベントが始はじまりました。
グーグルは、ＡＩを使つかった新あたらしい技術ぎじゅつを紹介しょうかいしました。ＡＩが入はいっているスマートフォンのカメラで外国語がいこくごの文章ぶんしょうを見みると、日本語にほんごなど１０以上いじょうの言葉ことばに翻訳ほんやくして読よみます。
耳みみが聞きこえない人ひとや英語えいごが上手じょうずではない人ひとのために、ビデオに出でてくる英語えいごの会話かいわに文字もじを出だす技術ぎじゅつも紹介しょうかいしました。
アメリカのＩＴの会社かいしゃはセキュリティーが十分じゅうぶんではないという意見いけんがあります。グーグルは「インターネットで何なにを見みたかなどのデータは、しばらくしたら消きえるようにします」と言いっています。
