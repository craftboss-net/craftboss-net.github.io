東京都とうきょうと調布市ちょうふし「たばこを吸すってはいけない店みせ」を紹介しょうかいする

发表于 2019-05-09 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうと調布市ちょうふしは今年ことし９月がつに始はじまるラグビーのワールドカップの会場かいじょうになっています。調布市ちょうふしは、住すんでいる人ひとや市しに来きた人ひとたちの健康けんこうのため、公園こうえんや駅えきの前まえ、学校がっこうの近ちかくでたばこを吸すってはいけないことにしました。たばこを吸すわない人ひとが、ほかの人ひとのたばこの煙けむりを吸すうと、病気びょうきになることがあるからです。
調布市ちょうふしは、たばこを吸すってはいけないと決きめているレストランや喫茶店きっさてんなどを紹介しょうかいする本ほんも作つくりました。市しにある７０ぐらいの店みせを日本語にほんごと英語えいごで紹介しょうかいしています。たばこを吸すってもいい場所ばしょと吸すってはいけない場所ばしょを分わけている店みせや、火ひを使つかわない「加熱式かねつしきたばこ」を吸すってもいい店みせは入はいっていません。
