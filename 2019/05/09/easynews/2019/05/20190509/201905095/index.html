犬いぬの「わさお」　一緒いっしょにいた雌めすの犬いぬが死しんで元気げんきがない

发表于 2019-05-09 更新于 2019-11-30 分类于 JapanEasyNews



青森県あおもりけん鰺ヶ沢町あじがさわまちの店みせにいる雄おすの犬いぬの「わさお」は、１０年ねん前まえにインターネットで紹介しょうかいされて、ライオンのように長ながい毛けと小ちいさな目めがかわいいと人気にんきになりました。
「わさお」は５年ねん前まえから、雌めすの犬いぬの「つばき」と一緒いっしょにいましたが、５日いつか、「つばき」が急きゅうに死しにました。「つばき」は６歳さいの若わかい犬いぬでした。
そのあと、「わさお」は元気げんきがなくなってしまいました。「わさお」は長ながい時間じかん何なにもしないで外そとを見みていたり、時々ときどき何なにかを思おもい出だしたようにほえたりしています。散歩さんぽに行いくと、何なにかをさがすように歩あるくことがあります。「わさお」の世話せわをしている人ひとは、「つばき」をさがしているように見みえると話はなしています。
