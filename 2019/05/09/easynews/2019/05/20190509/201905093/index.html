白血病はっけつびょうの池江いけえ選手せんしゅがウェブサイトを作つくる

发表于 2019-05-09 更新于 2019-11-30 分类于 JapanEasyNews



池江いけえ璃花子りかこ選手せんしゅは水泳すいえいの１００ｍバタフライなどで日本記録にほんきろくを持もっています。白血病はっけつびょうという病気びょうきになったことがわかって、いま治療ちりょうを続つづけています。
池江いけえ選手せんしゅは８日ようか、最近さいきんの自分じぶんのことなどを知しらせるウェブサイトを作つくって、メッセージを出だしました。メッセージは池江いけえ選手せんしゅが書かいた字じがそのまま出でています。
池江いけえ選手せんしゅは「治療ちりょうは問題もんだいなく進すすんでいます。体からだの具合ぐあいがいいときは、ぬり絵えや映画えいがなど、水泳すいえいをしていたときにできなかったことを楽たのしんでいます」と書かいています。いまの気持きもちについては、「心こころが折おれそうなときもあります。でも、みなさんのたくさんの言葉ことばから元気げんきをもらって、最後さいごまでがんばりたい、負まけたくないと思おもっています」と書かいています。
ウェブサイトでは、池江いけえ選手せんしゅの写真しゃしんを見みたり、池江いけえ選手せんしゅへのメッセージを書かいたりできます。
