奈良市ならし　殺ころした犬いぬや猫ねこが１０年ねんで６６３匹びきから１匹ぴきに減へる

发表于 2019-05-13 更新于 2019-11-30 分类于 JapanEasyNews



奈良市ならしでは、誰だれが飼かっているかわからない犬いぬや猫ねこを殺ころすことがあります。２００８年度ねんどは６６３匹びきいました。
奈良市ならしは１匹ぴきも殺ころさないようにしたいと考かんがえて、４年ねん前まえからウェブサイトで犬いぬや猫ねこを紹介しょうかいして、飼かう人ひとを探さがすことを始はじめました。
その結果けっか、殺ころした犬いぬや猫ねこはとても減へって、２０１７年度ねんどは３匹びき、２０１８年度ねんどは１匹ぴきになりました。
２０１８年度ねんどから奈良市ならしは、保健所ほけんじょが助たすけた犬いぬや猫ねこの赤あかちゃんの世話せわを、しばらくボランティアの人ひとにお願ねがいしています。このやり方かたで新あたらしく飼かう人ひとが見みつかった犬いぬや猫ねこは４２匹ひきいました。
奈良市ならしの市長しちょうは「犬いぬや猫ねこと最後さいごまで一緒いっしょに生活せいかつするという考かんがえ方かたをみんなに持もってほしいです。１匹ぴきも殺ころさないようにするために、これからも頑張がんばります」と話はなしています。
