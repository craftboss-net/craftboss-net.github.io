汚よごれたプラスチックのごみ　世界せかいの新あたらしいルールができる

发表于 2019-05-13 更新于 2019-11-30 分类于 JapanEasyNews



プラスチックのごみは、きれいに洗あらわなかったりほかの物ものが付ついていたりすると、リサイクルがしにくくなります。
スイスのジュネーブであった国際こくさい会議かいぎで１０日とおか、汚よごれたプラスチックのごみについて、新あたらしいルールが決きまりました。ごみを輸出ゆしゅつしにくくして、国くにの中なかでもっとリサイクルをしてもらうため、日本にっぽんとノルウェーがルールの案あんを出だしました。
新あたらしいルールでは、汚よごれたプラスチックのごみを輸出ゆしゅつするためには、輸入ゆにゅうする国くにが「自分じぶんの国くにに入いれてもいい」と言いうことが必要ひつようです。
新あたらしいルールは２０２１年ねんに始はじまります。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

プラスチック

  • 熱ねつや圧力あつりょくを加くわえて、自由じゆうに形かたちを作つくることができる物質ぶっしつ。特とくに合成樹脂ごうせいじゅしを指さす。使つかい道みちが広ひろい。

ルール

  • 規則きそく。決きまり。

付く

  • ものがくっつく。
  • つきそう。
  • 加くわわる。
  • 決きまる。
  • 草木くさきが、かれずに根ねをおろす。
  • 感かんじる。
  • ある値段ねだんになる。

リサイクル

  • いらなくなった物ものや使つかえなくなった物ものを、もう一度利用いちどりようできるようにすること。

案

  • 思おもうこと。考かんがえ。計画けいかく。
  • 相談そうだんのことがら。
