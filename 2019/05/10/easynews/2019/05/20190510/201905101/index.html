セブン＆アイ「２０３０年ねんまでにプラスチックの袋ふくろをやめる」

发表于 2019-05-10 更新于 2019-11-30 分类于 JapanEasyNews



「セブン＆アイ?ホールディングス」は、コンビニのセブン－イレブンやスーパーのイトーヨーカドーなどの会社かいしゃです。
会社かいしゃは、プラスチックのごみを少すくなくするための計画けいかくを発表はっぴょうしました。
２０３０年ねんまでに、買かい物ものをした客きゃくにレジで渡わたすプラスチックの袋ふくろをやめて、植物しょくぶつからつくった紙かみなどの袋ふくろにかえます。弁当べんとうなどの入いれ物ものには、紙かみやリサイクルした材料ざいりょうを使つかいます。そして、会社かいしゃが使つかうプラスチックの量りょうを２０３０年ねんまでに半分はんぶん以下いか、２０５０年ねんまでにゼロにします。
ほかのコンビニやスーパーの会社かいしゃも、プラスチックのごみを少すくなくします。ローソンは今月こんげつから、アイスコーヒーを入いれるコップを紙かみのコップにかえます。ファミリーマートも弁当べんとうの入いれ物ものなどに使つかうプラスチックの量りょうを少すくなくすると決きめました。
