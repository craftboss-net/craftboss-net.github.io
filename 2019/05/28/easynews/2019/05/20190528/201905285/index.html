男おとこが小学生しょうがくせいなど１９人にんを刺さす　２人ふたりが亡なくなる

发表于 2019-05-28 更新于 2019-11-30 分类于 JapanEasyNews



２８日にちの朝あさ、神奈川県かながわけん川崎市かわさきしで両方りょうほうの手てに包丁ほうちょうを持もった男おとこが小学生しょうがくせいなどを刺さしました。警察けいさつによると、１１歳さいの女おんなの子こ１人ひとりと男性だんせい１人ひとりが亡なくなって、小学生しょうがくせいなど１７人にんがけがをしました。
小学生しょうがくせいたちは学校がっこうのバスを待まっていました。警察けいさつによると、亡なくなった男性だんせいは小学生しょうがくせいの父親ちちおやのようです。
小学生しょうがくせいなどを刺さした男おとこは、自分じぶんの首くびを刺さして死しにました。近ちかくでは、事件じけんに使つかわれたと考かんがえられる包丁ほうちょうが２本ほん見みつかっています。男おとこのリュックサックの中なかには、使つかっていない包丁ほうちょうも２本ほん入はいっていました。警察けいさつは、男おとこが小学生しょうがくせいたちを刺さした理由りゆうなどを調しらべています。
