「キャッシュレス」の利用りようが増ふえて世界せかいのＡＴＭが減へる

发表于 2019-05-28 更新于 2019-11-30 分类于 JapanEasyNews



イギリスの調査ちょうさ会社がいしゃは、銀行ぎんこうなどのＡＴＭが世界せかいに何なん台だいあるか調しらべました。去年きょねんの終おわりには３２４万まん台だいあって、おととしより１％減へりました。ＡＴＭの数かずが減へったのは初はじめてです。お金かねを払はらうとき「キャッシュレス」を利用りようする人ひとが増ふえて、現金げんきんをあまり使つかわなくなっているためです。
ＡＴＭが世界せかいで最もっとも多おおい中国ちゅうごくでは、スマートフォンを使つかった「キャッシュレス」を利用りようする人ひとが増ふえています。ＡＴＭが２番目ばんめに多おおいアメリカでは、銀行ぎんこうの店みせが減へっています。４番目ばんめに多おおい日本にっぽんでも、去年きょねんはおととしより５００台だい減へりました。１０年ねんの間あいだで初はじめて減へりました。
調査ちょうさ会社がいしゃは、世界せかいのＡＴＭはこれからも減へっていくと考かんがえています。
