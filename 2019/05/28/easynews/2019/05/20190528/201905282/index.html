アメリカで日本にっぽんの畳たたみのよさを紹介しょうかい

发表于 2019-05-28 更新于 2019-11-30 分类于 JapanEasyNews



山形県やまがたけん寒河江市さがえしで畳たたみを作つくっている鏡かがみ芳昭よしあきさんたち５人にんが、アメリカで日本にっぽんの畳たたみを紹介しょうかいしました。鏡かがみさんたちは、いぐさという草くさで作つくった昔むかしからの畳たたみのよさをたくさんの人ひとに知しってほしいと考かんがえています。
鏡かがみさんたちはシカゴにある「アンダーソン日本庭園にほんていえん」で、古ふるくなった畳たたみの直なおし方かたをみんなに見みせました。部屋へやにある畳たたみを１枚まい１枚まい丁寧ていねいに直なおしてきれいにしました。そして、畳たたみが体からだや地球ちきゅうの環境かんきょうにいいことなども説明せつめいしました。
鏡かがみさんは「外国がいこくの人ひとも畳たたみの香かおりがいいと言いってくれて、うれしいです」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

いぐさ

  • 湿地しっちに生はえ、水田すいでんにも植うえる草くさ。茎くきは緑色みどりいろで細長ほそながく、畳表たたみおもてやござを作つくる。イ。トウシンソウ。

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから三番さんばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

環境

  • 人ひとや生いき物ものを取とり巻まき、影響えいきょうをあたえる周まわりの世界せかい。

香り

  • いいにおい。
  • そのものから感かんじられる品ひんの高たかさ。
