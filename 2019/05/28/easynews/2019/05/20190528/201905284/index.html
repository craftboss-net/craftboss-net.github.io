安倍あべ総理大臣そうりだいじんとトランプ大統領だいとうりょう　貿易ぼうえきの問題もんだいなどを話はなし合あう

发表于 2019-05-28 更新于 2019-11-30 分类于 JapanEasyNews



安倍あべ総理大臣そうりだいじんとアメリカのトランプ大統領だいとうりょうは２７日にち、北朝鮮きたちょうせんや貿易ぼうえきの問題もんだいなどについて話はなし合あいました。
２人ふたりは、北朝鮮きたちょうせんの核兵器かくへいきをなくすためにしっかり協力きょうりょくすることを確認かくにんしました。日本人にっぽんじんが北朝鮮きたちょうせんに連つれて行いかれた拉致らちの問題もんだいを解決かいけつするために、トランプ大統領だいとうりょうも協力きょうりょくすると約束やくそくしました。
トランプ大統領だいとうりょうは貿易ぼうえきの問題もんだいについて、「８月がつにいい発表はっぴょうができると思おもいます」と話はなしました。日本にっぽんの政府せいふでは、問題もんだいを早はやく解決かいけつするためにアメリカが強つよく言いってくるかもしれないという意見いけんも出でています。
トランプ大統領だいとうりょうは２８日にち、安倍あべ総理大臣そうりだいじんと一緒いっしょに神奈川県かながわけんにある横須賀基地よこすかきちに行いったあと、アメリカに帰かえりました。
