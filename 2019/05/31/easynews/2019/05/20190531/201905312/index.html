日本高血圧学会にほんこうけつあつがっかい「上うえの血圧けつあつが１３０以上いじょうにならないように」

发表于 2019-05-31 更新于 2019-11-30 分类于 JapanEasyNews



血圧けつあつが高たかくなると、脳のうや心臓しんぞうなどの病気びょうきの危険きけんが高たかくなると言いわれています。
日本高血圧学会にほんこうけつあつがっかいは今いままで、７４歳さいまでの大人おとなは、上うえの血圧けつあつが１４０以上いじょう、下したの血圧けつあつが９０以上いじょうにならないようにしたほうがいいと言いっていました。しかし、日本高血圧学会にほんこうけつあつがっかいはこの数字すうじを１０ずつ下さげて、新あたらしい目標もくひょうを発表はっぴょうしました。上うえの血圧けつあつが１３０以上いじょう、下したの血圧けつあつが８０以上いじょうにならないように言いっています。
７５歳さい以上いじょうの人ひとは今いままで、上うえの血圧けつあつが１５０以上いじょう、下したの血圧けつあつが９０以上いじょうにならないように言いっていました。しかし、新あたらしい目標もくひょうでは、上うえの血圧けつあつが１４０以上いじょうにならないように言いっています。下したの血圧けつあつは今いままでと同おなじです。
日本高血圧学会にほんこうけつあつがっかいは、今いままでの目標もくひょうと新あたらしい目標もくひょうを比くらべると、病気びょうきになる危険きけんが１０％から２０％ぐらい低ひくくなると言いっています。
