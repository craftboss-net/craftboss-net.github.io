日本にっぽんの海岸かいがんを掃除そうじして「プラスチックのごみをなくそう」

发表于 2019-05-31 更新于 2019-11-30 分类于 JapanEasyNews



５月がつ３０日にちは「５３０」が「ごみゼロ」と読よむこともできるため、日本にっぽんでは、ごみについて考かんがえる日ひになっています。環境省かんきょうしょうと日本にっぽん財団ざいだんは５月がつ３０日にちから、日本にっぽんの海岸かいがんを掃除そうじするキャンペーンを始はじめました。海うみが汚よごれる原因げんいんになるプラスチックのごみをなくしたいと考かんがえています。神奈川県かながわけん藤沢市ふじさわしの海岸かいがんには４００人にん以上いじょうが集あつまって、ペットボトルやお菓子かしの袋ふくろなどプラスチックのごみを１つ１つ拾ひろいました。環境省かんきょうしょうによると、プラスチックのごみは海うみに流ながれると集あつめることが難むずかしいため、海うみに流ながれないようにすることが大切たいせつです。キャンペーンは、海うみについて考かんがえる「世界せかい海洋かいようデー」の６月がつ８日ようかまで行おこないます。環境省かんきょうしょうなどは、日本にっぽんの２０００の場所ばしょで８０万まん人にんぐらいが海岸かいがんの掃除そうじを行おこなうことを考かんがえています。
