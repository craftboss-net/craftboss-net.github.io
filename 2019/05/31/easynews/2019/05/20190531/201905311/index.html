５つのレベルで雨あめや避難ひなんの情報じょうほうを伝つたえる

发表于 2019-05-31 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねん、西にし日本にほんで雨あめがたくさん降ふって多おおくの人ひとが亡なくなりました。このとき、気象庁きしょうちょうや役所やくしょなどは避難ひなんや災害さいがいの情報じょうほうをたくさん出だしました。しかし避難ひなんした人ひとはあまりいませんでした。
このため国くにや気象庁きしょうちょうは、もっとわかりやすく災害さいがいの情報じょうほうを伝つたえるために、２９日にちから５つのレベルで知しらせることにしました。
最もっとも低ひくいレベル１が出でたら、雨あめや川かわなどの情報じょうほうをテレビやインターネットなどで見みるように気きをつけてください。レベル２が出でたら、どこにどうやって避難ひなんするか調しらべてください。
レベル３が出でたら、お年寄としよりや体からだに障害しょうがいがある人ひとなどは、避難ひなんを始はじめてください。レベル４が出でたら、みんな避難ひなんを始はじめてください。
最もっとも高たかいレベル５が出でたら、災害さいがいが起おこっています。レベル５が出でる前まえに安全あんぜんな場所ばしょに避難ひなんすることが大切たいせつです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ
