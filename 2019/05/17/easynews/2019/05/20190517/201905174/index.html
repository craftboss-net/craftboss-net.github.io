東京とうきょうオリンピック　工事こうじの人ひとが働はたらく時間じかんを短みじかくしてほしい

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



国際的こくさいてきな労働組合ろうどうくみあいの団体だんたいは、東京とうきょうオリンピックとパラリンピックの会場かいじょうなどの工事こうじをしている人ひとがどのように働はたらいているか調しらべました。
東京とうきょう大会たいかいで選手せんしゅが泊とまる建物たてものや、新あたらしい国立競技場こくりつきょうぎじょうをつくっている人ひとの中なかには、１か月げつに２８日にち続つづけて働はたらいた人ひとがいました。暗くらい所ところや、頭あたまの上うえにコンクリートがつるしてある所ところで働はたらく人ひともいました。団体だんたいは、安全あんぜんではない働はたらき方かたをしている人ひとがいると言いっています。
団体だんたいは、東京とうきょう大会たいかいの組織委員会そしきいいんかいや東京都とうきょうとに、工事こうじの人ひとが働はたらく時間じかんを短みじかくしたり、自分じぶんたちと一緒いっしょに工事こうじをしている所ところを見みに行いったりしてほしいと言いっています。
組織委員会そしきいいんかいなどは、団体だんたいの言いっていることを確たしかめてから、どうするか考かんがえると話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

工事

  • 建物たてもの・道路どうろ・橋はしなどを造つくったり、直なおしたりすること。また、その仕事しごと。

国際的

  • 世界せかいの国々くにぐにに関係かんけいのあるようす。

労働組合

  • 労働者ろうどうしゃが、自分じぶんたちの権利けんりを守まもり、生活せいかつをよくするためにつくる団体だんたい。

団体

  • 大勢おおぜいの人ひとの集あつまり。
  • 同おなじ目的もくてきを持もった人々ひとびとの集あつまり。

オリンピック

  • ４年よねんごとに開ひらかれ、世界せかいじゅうの国々くにぐにから選手せんしゅが参加さんかする競技大会きょうぎたいかい。古代こだいギリシャのオリンピアで開ひらかれた古代こだいオリンピックにならって、フランスのクーベルタンの力ちからで、１８９６年ねんにギリシャのアテネで開ひらかれたのが、近代きんだいオリンピックの始はじまり。五輪ごりん。

パラリンピック

  • 身体障害者しんたいしょうがいしゃの国際こくさいスポーツ大会たいかい。４年よねんに一度いちど、オリンピック開催地かいさいちで行おこなわれる。

大会

  • 多おおくの人ひとが集あつまる会かい。
  • ある組織そしきの全員ぜんいんが集あつまる会かい。

選手

  • 競技きょうぎに出でるために選えらばれた人ひと。

コンクリート

  • セメント・砂すな・じゃりを水みずで混まぜて、石いしのように固かたまらせたもの。コンクリ。

つるす

  • 上うえからつって下さげる。ぶら下さげる。

確かめる

  • まちがいがないようにする。あやふやな点てんを、はっきりさせる。
