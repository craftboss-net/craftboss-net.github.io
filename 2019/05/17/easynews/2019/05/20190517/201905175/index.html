飛行機ひこうき代だいをなくした高校生こうこうせいと６万まん円えん渡わたした男性だんせいがまた会あう

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



先月せんげつ２４日にじゅうよっか、沖縄県おきなわけん那覇市なはしの高校生こうこうせいの崎元さきもと颯馬そうまさんは、おじさんの葬式そうしきに出でるために空港くうこうに向むかいました。しかし空港くうこうの駅えきに着ついたとき、財布さいふをなくしたことがわかりました。財布さいふには飛行機ひこうきのチケットを買かうための約やく６万まん円えんが入はいっていました。崎元さきもとさんはどうしたらいいかわからなくて、とても困こまっていました。
そのとき、知しらない男性だんせいが「早はやく飛行機ひこうきに乗のって」と言いって、崎元さきもとさんに６万まん円えんを渡わたしました。崎元さきもとさんは、この男性だんせいのおかげで葬式そうしきに間まに合あいました。しかし、男性だんせいの名前なまえや電話でんわ番号ばんごうを聞きいていませんでした。
このため沖縄県おきなわけんの新聞しんぶんに記事きじを出だしてもらって、男性だんせいをさがしていました。そして、埼玉県さいたまけんの医者いしゃで６８歳さいの猪野屋いのや博ひろしさんが、インターネットにこの記事きじが出でていることを知しって連絡れんらくしました。
２人ふたりは来週らいしゅう沖縄県おきなわけんで会あう予定よていです。崎元さきもとさんは「こんなに優やさしい人ひとがいることに感謝かんしゃの気持きもちでいっぱいです。早はやく会あってお金かねを返かえして、お礼れいを言いいたいです」と話はなしています。
