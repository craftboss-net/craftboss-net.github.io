イギリスの大英博物館だいえいはくぶつかんで日本にっぽんの漫画まんがの展覧会てんらんかいが始はじまる

发表于 2019-05-24 更新于 2019-11-30 分类于 JapanEasyNews



イギリスのロンドンにある大英博物館だいえいはくぶつかんで、日本にっぽんの漫画まんがを紹介しょうかいする展覧会てんらんかいが始はじまりました。「鉄腕てつわんアトム」や「キャプテン翼つばさ」、「ＯＮＥ　ＰＩＥＣＥ」など、５０人にんの漫画まんが家かがかいた７０ぐらいの漫画まんがを見みることができます。
漫画まんがの文化ぶんかが日本にっぽんに何百なんびゃく年ねんも前まえからある浮世絵うきよえや絵巻物えまきものから始はじまったことや、漫画まんがからアニメやゲームが生うまれたことなども紹介しょうかいしています。漫画まんがのキャラクターの人形にんぎょうなどもあります。
大英博物館だいえいはくぶつかんは２００６年ねんに初はじめて漫画まんがを展示てんじしてから、漫画まんがの絵えや本ほんを集あつめてきました。
大英博物館だいえいはくぶつかんの人ひとは「漫画まんがは見みて楽たのしむものなので、インターネットのインスタグラムなどがよく使つかわれる今いまの時代じだいに合あっています。たくさんの人ひとに漫画まんがをもっと知しってほしいです」と話はなしています。
展覧会てんらんかいは８月がつ２６日にちまでです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

浮世絵

  • 江戸時代えどじだいに始はじまったもので、人物じんぶつや生活せいかつ、景色けしきなどをかいた日本にっぽんの絵え。筆ふででかいたものと、版画はんがとがある。歌麿うたまろ・北斎ほくさい・広重ひろしげなどが名高なだかい。

絵巻物

  • 物語ものがたりや伝説でんせつなどを、絵え、または絵えと文ぶんで表あらわして、巻まき物ものにしたもの。

アニメーション

  • かいた絵えや人形にんぎょうを、動うごきに従したがって一ひとこま一ひとこま撮影さつえいし、それを映うつして実際じっさいに動うごいているように見みせる映画えいが。動画どうが。アニメ。

ゲーム

  • 勝かち負まけを争あらそう遊あそび。
  • 試合しあい。

キャラクター

  • 性格せいかく。持もち味あじ。
  • まんが・映画えいが・劇げきなどに出でてくる人物じんぶつや動物どうぶつ。

展示

  • 品物しなものを並ならべて、多おおくの人ひとに見みせること。
