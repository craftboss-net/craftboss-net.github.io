日本にっぽんの大おおきな３つの銀行ぎんこう　店みせや働はたらく人ひとを減へらす

发表于 2019-05-24 更新于 2019-11-30 分类于 JapanEasyNews



低ひくい利子りしが長ながい間あいだ続つづいて、銀行ぎんこうは経営けいえいが厳きびしくなっています。日本にっぽんの大おおきな３つの銀行ぎんこうのグループは、働はたらく人ひとや店みせなどを減へらそうと考かんがえています。
三井みつい住友すみとも銀行ぎんこうのグループは、５０００人にんぐらいで行おこなっている量りょうの仕事しごとを、来年らいねん３月がつまでに機械きかいで行おこなうことにします。働はたらく人ひとも４０００人にんぐらい減へりそうです。
三菱みつびしＵＦＪ銀行ぎんこうは、２０２４年ねん３月がつまでに店みせを１８０ぐらい減へらします。今いまある店みせのうち３５％がなくなります。仕事しごとの量りょうも減へらして、１万まん人にん以上いじょうの人ひとで行おこなっている仕事しごとをなくします。
みずほフィナンシャルグループも、２０２５年ねん３月がつまでに店みせを１３０減へらします。働はたらく人ひとは２０２７年ねん３月がつまでに、１万まん９０００人にん減へらします。
ＩＴの会社かいしゃなども銀行ぎんこうの仕事しごとを始はじめているため、競争きょうそうが厳きびしくなっています。銀行ぎんこうは、客きゃくが不便ふべんにならないようにしながら店みせや働はたらく人ひとを減へらすことや、利益りえきが出でそうな仕事しごとに社員しゃいんを使つかうことが必要ひつようになっています。
