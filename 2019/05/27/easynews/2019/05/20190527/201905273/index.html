ＷＨＯ「ゲーム障害しょうがい」を病気びょうきなどのリストに入いれる

发表于 2019-05-27 更新于 2019-11-30 分类于 JapanEasyNews



ＷＨＯの会議かいぎで２５日にち、病気びょうきやけがなどのリストに「ゲーム障害しょうがい」を入いれることが決きまりました。
「ゲーム障害しょうがい」は、パソコンやスマートフォンのゲームをやめることができなくて、生活せいかつや仕事しごとに問題もんだいが出でていることです。
これから、「ゲーム障害しょうがい」の人ひとが世界せかいにどのくらいいるか、どうやって治なおすかなど、研究けんきゅうが進すすみそうです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ
