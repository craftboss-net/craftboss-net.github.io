日本にっぽんが作つくる新あたらしいスーパーコンピューター　名前なまえは「富岳ふがく」

发表于 2019-05-27 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんが作つくっている新あたらしいスーパーコンピューターの名前なまえが、「富岳ふがく」に決きまりました。「富岳ふがく」は富士山ふじさんという意味いみです。
世界せかいでは、速はやいスーパーコンピューターを作つくる競争きょうそうが激はげしくなっています。日本にっぽんの「京けい」はスーパーコンピューターの計算けいさんの速はやさで、２０１１年ねんに世界せかいで１番ばんになりました。しかしそのあと、もっと速はやいコンピューターが出でてきました。
新あたらしい「富岳ふがく」は、コンピューターの脳のうの働はたらきをするＣＰＵの速はやさが「京けい」の２０倍ばいです。災害さいがいや医療いりょうなど９つの分野ぶんやでよく使つかうソフトウエアを速はやく動うごかすこともできるようになります。このため、「京けい」の１００倍ばいぐらい速はやく作業さぎょうすることができます。世界せかいで最もっとも速はやくなりそうです。
理化学研究所りかがくけんきゅうしょと富士通ふじつうは今年ことし３月がつから「富岳ふがく」のハードウエアなどを作つくり始はじめています。国くには２０２１年ねんごろから「富岳ふがく」を使つかいたいと考かんがえています。
