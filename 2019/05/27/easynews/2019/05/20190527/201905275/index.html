トランプ大統領だいとうりょうが天皇てんのう陛下へいかと皇后こうごうさまに会あって話はなしをする

发表于 2019-05-27 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのトランプ大統領だいとうりょうが２５日にちに日本にっぽんへ来きました。２７日にちの朝あさ、皇居こうきょでトランプ大統領だいとうりょうと夫人ふじんを歓迎かんげいする式しきがありました。朝あさ９時じ半はんごろ、天皇てんのう陛下へいかと皇后こうごうさまが２人ふたりを迎むかえて、笑顔えがおで握手あくしゅをしてあいさつしました。天皇てんのう陛下へいかと皇后こうごうさまがトランプ大統領だいとうりょうに会あったのは初はじめてです。
歓迎かんげいの式しきには、天皇てんのう陛下へいかの弟おとうとの秋篠宮あきしののみやさまや、安倍あべ総理大臣そうりだいじんも出席しゅっせきしました。日本にっぽんの小学生しょうがくせいやアメリカ大使館たいしかんの人ひとたちも、国くにの旗はたを振ふってトランプ大統領だいとうりょうと夫人ふじんを歓迎かんげいしました。式しきのあと、天皇てんのう陛下へいかと皇后こうごうさまは２人ふたりと話はなしをしました。
夜よるは、天皇てんのう陛下へいかと皇后こうごうさまがトランプ大統領だいとうりょうと夫人ふじんを招待しょうたいして、皇居こうきょで晩ばんさん会かいを行おこないます。
