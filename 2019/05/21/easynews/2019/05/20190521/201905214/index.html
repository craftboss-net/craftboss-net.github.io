プロ野球やきゅうの上原うえはら浩治こうじさんが選手せんしゅをやめる

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



２０日はつか、プロ野球やきゅうのピッチャーの上原うえはら浩治こうじさんが「選手せんしゅをやめます」と言いいました。上原うえはらさんは４４歳さいで、日本にっぽんのプロ野球やきゅうのピッチャーでは年齢ねんれいがいちばん上うえでした。
上原うえはらさんは巨人きょじんというチームに入はいって１９９９年ねんから活躍かつやくしました。日本にっぽんの代表だいひょうでオリンピックにも２回かい出でました。２００９年ねんにアメリカの大だいリーグのチームに入はいりましたが、去年きょねんまた巨人きょじんに戻もどりました。
上原うえはらさんは「けがをしたひざの手術しゅじゅつをして、体からだの具合ぐあいはよかったですが、試合しあいでいい結果けっかを出だすことができませんでした。気持きもちと体からだが合あわなくなりました」と話はなしました。
