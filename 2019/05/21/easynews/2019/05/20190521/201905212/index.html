「日本にほん遺産いさん」に１６件けんが決きまる　沖縄県おきなわけんの琉球りゅうきゅう料理りょうりや泡盛あわもり

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



文化庁ぶんかちょうは、日本にっぽんのいろいろな地域ちいきの文化ぶんかや歴史れきしのすばらしさを紹介しょうかいして、観光かんこうに来くる人ひとを多おおくしたいと考かんがえて「日本にほん遺産いさん」を決きめています。
２０１９年度ねんどは、沖縄県おきなわけんや群馬県ぐんまけんなどの１６件けんが日本にほん遺産いさんに決きまりました。
沖縄県おきなわけんでは、琉球りゅうきゅう料理りょうりや泡盛あわもりというお酒さけ、琉球りゅうきゅう舞踊ぶようという踊おどりが日本にほん遺産いさんになりました。１４２９年ねんから４００年ねん以上いじょう続つづいた琉球りゅうきゅう王国おうこくの時代じだいに、このような料理りょうりやお酒さけで客きゃくを大切たいせつに迎むかえていました。沖縄県おきなわけんの知事ちじは「日本にほん遺産いさんに決きまったものは、沖縄県おきなわけんの人ひとの宝たからです。沖縄県おきなわけんの観光かんこうをもっとよくしていきたいです」と話はなしていました。
群馬県ぐんまけん館林市たてばやししでは、沼ぬまなどの自然しぜんと人ひとが一緒いっしょに生いきていく文化ぶんかが、日本にほん遺産いさんになりました。
日本にほん遺産いさんは全部ぜんぶで８３件けんになりました。
