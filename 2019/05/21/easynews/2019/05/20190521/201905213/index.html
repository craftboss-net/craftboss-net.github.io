東京とうきょうで雨あめが降ふったときに地面じめんにたまる水みずの深ふかさを予想よそう

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうなどの大おおきなまちで雨あめがたくさん降ふると、地面じめんの上うえに水みずがたまって危険きけんになることがあります。早稲田大学わせだだいがくと東京大学とうきょうだいがくなどのグループは、雨あめが降ふったときに東京とうきょうの２３区くで水みずが地面じめんにどのくらいたまるかすぐに予想よそうするシステムをつくりました。
このシステムでは、雲くもの情報じょうほうを調しらべてどのくらい雨あめが降ふるか予想よそうします。そして、道みちや建物たてもの、川かわ、下水道げすいどうの情報じょうほうも利用りようして、たまる水みずの深ふかさをすぐに計算けいさんします。そして地図ちずに、水みずの深ふかさを色いろで知しらせます。今いまの水みずの深ふかさだけではなくて、３０分ぷんあとにどのくらいの深ふかさになるかもわかります。
グループは、６月がつごろからインターネットでこの予想よそうを見みせる予定よていです。
