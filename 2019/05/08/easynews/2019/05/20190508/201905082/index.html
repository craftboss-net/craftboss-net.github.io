「動物どうぶつや植物しょくぶつ１００万まん種類しゅるいがいなくなる心配しんぱいがある」

发表于 2019-05-08 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんなど１３０以上いじょうの国くにの科学かがくの専門家せんもんかが集あつまるＩＰＢＥＳによると、今いままでの５００年ねんの間あいだ、人ひとが原因げんいんで６８０種類しゅるいの脊椎動物せきついどうぶつがいなくなりました。ガラパゴス諸島しょとうにいた亀かめ「ピンタゾウガメ」や、インド洋ようのモーリシャスにいた大おおきな鳥とり「ドードー」などです。
ＩＰＢＥＳは、人ひとが農業のうぎょうや漁業ぎょぎょうのために陸りくや海うみの環境かんきょうを変かえたこと、動物どうぶつをたくさんとったこと、地球ちきゅうの気温きおんが上あがったことなどが原因げんいんだと言いっています。
ＩＰＢＥＳによると、今いまも動物どうぶつや植物しょくぶつ１００万まん種類しゅるいがいなくなる心配しんぱいがあります。この動物どうぶつや植物しょくぶつの多おおくが、これから何なん十じゅう年ねんかの間あいだにいなくなるかもしれないため、ＩＰＢＥＳは世界せかいで対策たいさくが必要ひつようだと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

植物

  • 生物せいぶつを大おおきく二ふたつに分わけたときの一ひとつで、動物どうぶつに対たいするもう一ひとつの生物せいぶつ。木き・草くさ・きのこ・海藻かいそうなど。

種類

  • 共通きょうつうする形かたちや性質せいしつによって分わけたもの。

脊椎動物

  • 背骨せぼねを持もつ動物どうぶつ。脳のうが発達はったつしていて、動物どうぶつの中なかでも高等こうとうなもの。哺乳類ほにゅうるい・鳥類ちょうるい・爬虫類はちゅうるい・魚類ぎょるいなど。

ガラパゴス諸島

  • 東太平洋ひがしたいへいようの赤道直下せきどうちょっかにある火山かざんの島々しまじま。南みなみアメリカ大陸たいりくの西にし、約やく１０００キロメートルの所ところにあり、エクアドル領りょう。どの大陸たいりくからも離はなれているので、巨大きょだいなゾウガメやウミトカゲなど、他たの地域ちいきには見みられない独自どくじの進化しんかをとげた生物せいぶつがたくさん生息せいそくしている。ここを観察かんさつしたことが、ダーウィンが「進化論しんかろん」を唱となえるもとになった。世界遺産せかいいさん。

亀

  • 水中すいちゅうや陸上りくじょうで生活せいかつし、体からだがかたいこうらでおおわれた動物どうぶつ。池いけや川かわにすむイシガメ・スッポン、陸りくにすむゾウガメ、海うみにすむアカウミガメなど種類しゅるいが多おおい。ほとんどの種類しゅるいが首くびや尾お、足あしをこうらの中なかに引ひっこめることができる。

インド洋

  • アジア・アフリカ・オーストラリア・南極なんきょくの大陸たいりくに囲かこまれた大おおきな海うみ。

農業

  • 田たや畑はたけで、穀物こくもつ・野菜やさい・果物くだものなどを作つくる仕事しごと。牛うしなどの家畜かちくを飼かう仕事しごともふくめていうことがある。

漁業

  • 魚さかなや貝かい・海藻かいそうなどの水産物すいさんぶつを、とったり育そだてたりする仕事しごと。

陸

  • 広ひろい土つちのある所ところ。

環境

  • 人ひとや生いき物ものを取とり巻まき、影響えいきょうをあたえる周まわりの世界せかい。

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから三番さんばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

気温

  • 大気たいき（空気くうき）の温度おんど。

対策

  • ある問題もんだいや、できごとに対たいして、どうしたらよいかという、そのやり方かた。
