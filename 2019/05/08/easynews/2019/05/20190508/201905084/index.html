奈良公園ならこうえん　令和れいわになって初はじめての鹿しかの赤あかちゃんが生うまれる

发表于 2019-05-08 更新于 2019-11-30 分类于 JapanEasyNews



奈良市ならしにある奈良公園ならこうえんにはたくさんの鹿しかがいて、国くにの天然記念物てんねんきねんぶつになっています。
７日なのかの朝あさ、鹿しかの赤あかちゃんが生うまれているのが見みつかりました。元号げんごうが令和れいわになってから初はじめて生うまれた赤あかちゃんです。体からだの長ながさが５４ｃｍで３０００ｇの雌めすです。
奈良公園ならこうえんでは、鹿しかの赤あかちゃんは５月がつから６月がつごろたくさん生うまれます。今年ことしは２００匹ぴきぐらい生うまれそうです。
鹿しかの赤あかちゃんは来月らいげつ１日ついたちから見みることができます。

鹿

  • 森林しんりんや山野さんやにすむ、草食そうしょくのおとなしいけもの。足あしは細長ほそながく、雄おすは木きの枝えだのような角つのを持もつ。世界各地せかいかくちにいる。

天然記念物

  • 法律ほうりつで、だいじにするように決きめられている、めずらしい動物どうぶつや植物しょくぶつ・鉱物こうぶつのこと。例たとえば、北海道ほっかいどうの阿寒湖あかんこのマリモなど。

年号

  • 日本にっぽんで、年としにつける呼よび名な。元号げんごう。平成へいせいや昭和しょうわなど。

雌

  • 動物どうぶつで、子こや卵たまごを生うむ能力のうりょくがあるほう。
