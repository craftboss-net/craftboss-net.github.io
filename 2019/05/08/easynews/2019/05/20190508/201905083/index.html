ミャンマー　ロヒンギャの問題もんだいを調しらべた記者きしゃが刑務所けいむしょから出でる

发表于 2019-05-08 更新于 2019-11-30 分类于 JapanEasyNews



ミャンマーの政府せいふは７日なのか、約やく６５００人にんを刑務所けいむしょから出だして自由じゆうにすると言いいました。ロイター通信つうしんの記者きしゃのワ?ローンさんと、チョー?ソー?ウーさんも刑務所けいむしょから出でて自由じゆうになりました。
２人ふたりの記者きしゃは、ミャンマーでイスラム教きょうを信しんじているロヒンギャの人ひとたちがひどいことをされたという問題もんだいについて調しらべていました。しかし、おととし、国くにの書類しょるいを正ただしくないやり方かたでもらったという理由りゆうで警察けいさつに捕つかまりました。そして、先月せんげつの裁判さいばんで刑務所けいむしょに７年ねん入はいることが決きまりました。このため、「ミャンマーでは本当ほんとうのことを伝つたえることができなくなっている」などと世界せかいで言いわれていました。
ワ?ローンさんは「本当ほんとうにうれしいです。早はやくニュースの仕事しごとに戻もどりたいです」と話はなしました。
ロイター通信つうしんは「２人ふたりはニュースを自由じゆうに伝つたえる大切たいせつさをみんなに知しらせました」と言いいました。
