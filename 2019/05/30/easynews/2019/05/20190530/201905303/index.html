パワーハラスメントをなくすための法律ほうりつができる

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



パワーハラスメントは、働はたらく人ひとに暴力ぼうりょくをしたり、つらい気持きもちにさせたりすることです。短みじかくして、「パワハラ」と言いうこともあります。
会社かいしゃや工場こうじょうなどで、パワハラをなくすための法律ほうりつができました。この法律ほうりつで、会社かいしゃなどは働はたらく人ひとからパワハラの相談そうだんを聞きかなければならないと決きまりました。相談そうだんをした人ひとをやめさせたり、嫌いやな仕事しごとに変かえたりすることも禁止きんしします。そして、妊娠にんしんした女性じょせいや子こどもを産うんだ女性じょせいを公平こうへいに働はたらかせることも決きめました。
厚生労働省こうせいろうどうしょうは、パワハラについての細こまかいルールなどをこれから決きめると言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

暴力

  • 力ちからでする、乱暴らんぼうな行おこない。

つらい

  • （体からだや気持きもちが）がまんできないほど苦くるしい。

辞める

  • 勤つとめや役目やくめから退しりぞく。

禁止

  • してはいけないと止とめること。

妊娠

  • おなかの中なかに子こどもを宿やどすこと。みごもること。

産む

  • 赤あかんぼうや卵たまごを、母親ははおやが体からだから出だす。

公平

  • えこひいきをしないこと。かたよらないこと。

厚生労働省

  • 国民こくみんの健康けんこうや生活せいかつを守まもる仕事しごとや、労働者ろうどうしゃが仕事しごとを見みつけるのを助たすけたり、労働者ろうどうしゃを保護ほごしたりする国くにの役所やくしょ。厚労省こうろうしょう。

ルール

  • 規則きそく。決きまり。
