虐待ぎゃくたいから子こどもを守まもる仕事しごとをＡＩが手伝てつだう実験じっけん

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



虐待ぎゃくたいされる子こどもが増ふえていて、子こどもを守まもる児童相談所じどうそうだんじょの職員しょくいんが足たりません。
産業技術総合研究所さんぎょうぎじゅつそうごうけんきゅうしょなどは、児童相談所じどうそうだんじょの仕事しごとをコンピューターのＡＩが手伝てつだう実験じっけんを三重県みえけんですることにしました。
実験じっけんで使つかうコンピューターのＡＩは、三重県みえけんで集あつめた子こどもの傷きずの場所ばしょなど６０００件けんのデータを覚おぼえています。
職員しょくいんは、虐待ぎゃくたいされているかもしれない子こどものデータをコンピューターに入いれます。ＡＩはその子こどもにこれからどのくらい危険きけんがありそうかなどを計算けいさんします。職員しょくいんは、安全あんぜんな場所ばしょに子こどもを連つれていったほうがいいかどうか早はやく決きめることができます。
実験じっけんは、２つの児童相談所じどうそうだんじょにコンピューターを２０台だい入いれて、来月らいげつから始はじまります。
