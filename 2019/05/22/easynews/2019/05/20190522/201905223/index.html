法律ほうりつの専門家せんもんか以外いがいの人ひとも参加さんかする「裁判員さいばんいん制度せいど」が１０年ねん

发表于 2019-05-22 更新于 2019-11-30 分类于 JapanEasyNews



法律ほうりつの専門家せんもんかではない人ひとたちも裁判さいばんに参加さんかする「裁判員さいばんいん制度せいど」が始はじまって、２１日にちで１０年ねんになりました。今いままでに９万まん人にんの裁判員さいばんいんが参加さんかして、約やく１万まん２０００人にんの被告ひこくに判決はんけつが出でました。
人ひとを殺ころした事件じけんと性せい的てきな暴行ぼうこうの事件じけんでは、裁判員さいばんいんが参加さんかしたときは、裁判官さいばんかんだけのときよりも刑けいが重おもくなっていました。そして、裁判員さいばんいんが参加さんかしたときは、被告ひこくにもう一度いちど頑張がんばってほしいという気持きもちで判決はんけつを考かんがえることが多おおくなっています。
東京とうきょうで２１日にち、裁判員さいばんいんに選えらばれてもできないと言いう人ひとが増ふえている問題もんだいについて会議かいぎがありました。裁判員さいばんいんの経験けいけんがある男性だんせいは、裁判員さいばんいんに選えらばれたとき会社かいしゃの人ひとが困こまっていたため、裁判所さいばんしょから会社かいしゃにも連絡れんらくしてほしいと話はなしました。検察庁けんさつちょうの人ひとは、裁判さいばんに必要ひつような期間きかんを短みじかくして参加さんかしやすいようにしたいと話はなしました。裁判所さいばんしょの人ひとは「裁判員さいばんいんになった人ひとのうち９６％がいい経験けいけんだったと答こたえています。このことをもっと知しってほしいです」と話はなしました。
