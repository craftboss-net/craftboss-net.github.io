去年きょねんは災害さいがいが多おおかった　保険ほけん会社がいしゃが払はらったお金かねも多おおくなる

发表于 2019-05-22 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねんは、台風たいふう２１号ごうが来きたり、西にし日本にほんでとてもたくさん雨あめが降ふったりして、夏なつから災害さいがいが続つづきました。損害そんがい保険ほけんの会社かいしゃが、災害さいがいで被害ひがいを受うけた人ひとに払はらったお金かねがとても多おおくなりました。
三井住友海上みついすみともかいじょうとあいおいニッセイ同和どうわのグループは６５５０億おく円えん払はらいました。東京とうきょう海上かいじょうホールディングスは５０６７億おく円えん、ＳＯＭＰＯホールディングスは４９８８億おく円えん払はらいました。
この３つのグループが払はらったお金かねを足たすと１兆ちょう６０００億おく円えんになります。会社かいしゃは、今いままでで最もっとも多おおいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

災害

  • 台風たいふう・地震じしん・大水おおみずなどによる災難さいなん。

保険

  • 大勢おおぜいで少すこしずつお金かねを積つみ立たてておき、病気びょうきや事故じこなどの災難さいなんにあった人ひとが、決きまったお金かねを受うけ取とる仕組しくみ。

損害

  • 物ものがなくなったり、こわれたりして受うけた損そん。

被害

  • 損害そんがいを受うけること。また、受うけた害がい。

グループ

  • 仲間なかま。集団しゅうだん。

兆

  • 一億いちおくの一万倍いちまんばい。
  • しるし。前まえぶれ。
