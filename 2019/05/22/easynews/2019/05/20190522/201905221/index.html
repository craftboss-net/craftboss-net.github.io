ＡＩが司法試験しほうしけんの予備試験よびしけんの問題もんだいを６０％予想よそうできた

发表于 2019-05-22 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうにある会社かいしゃが作つくったコンピューターのＡＩが、国くにの司法試験しほうしけんの予備試験よびしけんにどんな問題もんだいが出でるか予想よそうしました。予備試験よびしけんに合格ごうかくした人ひとは、弁護士べんごしや裁判官さいばんかんになるための司法試験しほうしけんを受うけることができます。
ＡＩは、去年きょねんまでの予備試験よびしけんに出でた問題もんだいや法律ほうりつの言葉ことばなどを勉強べんきょうしました。そして、予備試験よびしけんの中なかで、法律ほうりつについて正ただしい答こたえを選えらぶ問題もんだいを予想よそうして、答こたえも考かんがえました。
１９日にち、今年ことしの予備試験よびしけんがありました。会社かいしゃによると、９５ある問題もんだいの中なかで、ＡＩの予想よそうと同おなじ問題もんだいは６０％の５７ありました。会社かいしゃは、答こたえも合あっていたため、合格ごうかくした可能性かのうせいが高たかいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

予想

  • 前まえもって、こうだろうと考かんがえること。または、その考かんがえ。

合格

  • 学校がっこうや会社かいしゃの試験しけんに受うかること。パス。
  • 品物しなものなどが決きめられたことがらに合あっていること。

弁護士

  • 裁判さいばんなどで、事件じけんに関係かんけいのある人ひとの、権利けんりや利益りえきを守まもるために、本人ほんにんの代理だいりや弁護べんごをする職業しょくぎょうの人ひと。

裁判官

  • 裁判所さいばんしょで、裁判さいばんをする国くにの公務員こうむいん。国民こくみんの自由じゆうや権利けんりを守まもるために、よい悪わるいを決きめたり、悪わるい人ひとを裁さばいたりする。

可能性

  • その状態じょうたいになりそうなようす。できそうなようす。
