福岡市ふくおかし　傘かさを安やすく借かりることができるサービスが始はじまる

发表于 2019-05-22 更新于 2019-11-30 分类于 JapanEasyNews



もうすぐ梅雨つゆの季節きせつになります。福岡市ふくおかしで２１日にち、スマートフォンのアプリを使つかって、傘かさを借かりることができるサービスが始はじまりました。
傘かさは福岡市ふくおかしの市役所しやくしょや地下鉄ちかてつの駅えきなど４１の場所ばしょに全部ぜんぶで１０００本ぼんぐらい置おいてあります。傘かさには鍵かぎがかかっていますが、傘かさにあるＱＲコードをスマートフォンのアプリで写うつすと、鍵かぎを開あけるための番号ばんごうがわかります。
傘かさは１日いちにち７０円えんで借かりることができます。６日むいか目めから１か月げつまでは、何なん日にち使つかっても４２０円えんです。使つかい終おわったら、４１の場所ばしょのどこにでも返かえすことができます。
サービスを始はじめた会社かいしゃは「傘かさを買かうより安やすく借かりることができます。傘かさのごみも少すくなくなると思おもいます」と話はなしています。
