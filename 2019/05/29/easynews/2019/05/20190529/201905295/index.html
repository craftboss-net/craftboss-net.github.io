車くるまの「自動じどう運転うんてん」が来年らいねん高速道路こうそくどうろで始はじまる予定よてい

发表于 2019-05-29 更新于 2019-11-30 分类于 JapanEasyNews



２８日にちの国会こっかいで、交通こうつうの法律ほうりつを変かえることが決きまりました。人ひとが運転うんてんしなくても車くるまが走はしる「自動じどう運転うんてん」が、高速道路こうそくどうろが混こんでいるときなどにできるようになります。
新あたらしい法律ほうりつでは、自動じどう運転うんてんをしている間あいだ、運転うんてん席せきにいる人ひとはすぐに運転うんてんできるようにしていなければなりません。すぐに運転うんてんできるなら、運転うんてん席せきの人ひとは携帯電話けいたいでんわを持もって話はなすことも、メールやゲームもできます。本ほんを読よんだり食事しょくじしたりすることもできそうです。
新あたらしい法律ほうりつは、来年らいねん５月がつまでに始はじまる予定よていです。

自動

  • 機械きかいなどが自分じぶんの力ちからで動うごくこと。

高速道路

  • 自動車じどうしゃが速はやく走はしれるように、立体交差りったいこうさにしたり、上のぼり下くだりを分わけたりした道路どうろ。ハイウエー。

国会

  • 国会議員こっかいぎいんが集あつまって、法律ほうりつを作つくり、政治せいじのやり方かたを相談そうだんして決きめる議会ぎかい。衆議院しゅうぎいんと参議院さんぎいんとがある。
