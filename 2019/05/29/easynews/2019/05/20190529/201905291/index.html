中国ちゅうごく　体からだが全部ぜんぶ白しろいパンダが見みつかる

发表于 2019-05-29 更新于 2019-11-30 分类于 JapanEasyNews



中国ちゅうごくの四川省しせんしょうで自然しぜんを守まもる 活動かつどうをしている役所やくしょが動物どうぶつを調しらべるために山やまの中なかに置おいていたカメラに、体からだが全部ぜんぶ白しろいパンダが写うつっていました。
このパンダは１歳さいか２歳さいぐらいで、毛けや爪つめの色いろなどが全部ぜんぶ白しろくて、目めは少すこし赤あかくなっています。役所やくしょの人ひとは、体からだが全部ぜんぶ白しろいパンダの写真しゃしんを撮とったのは初はじめてだと言いっています。
このパンダは、遺伝子いでんしが変かわって色いろがない体からだで生うまれたと役所やくしょは考かんがえています。役所やくしょの人ひとは「これからカメラを増ふやして、白しろいパンダの生活せいかつなどを調しらべたいです」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

パンダ

  • アジア大陸たいりくの中央部ちゅうおうぶにすむけもの。ジャイアントパンダとレッサーパンダがいる。
  • 特とくにジャイアントパンダ。中国西部ちゅうごくせいぶの山地さんちにすむ。体からだは白しろと黒くろに色分いろわけされて、顔かおつきや動作どうさがかわいい。

自然

  • 山やま・川かわ・草くさ・木き・星ほし・雲くも・雨あめ・雪ゆきなど、人ひとが作つくったものでない物もの。
  • もともとの性質せいしつを備そなえていること。
  • わざとらしくないようす。
  • ひとりでに。いつのまにか。

守る

  • 害がいを受うけないように、防ふせぐ。
  • 決きめたとおりにする。

活動

  • 元気げんきよく動うごいたり、働はたらいたりすること。

役所

  • 国くにや、地方公共団体ちほうこうきょうだんたいの仕事しごとをする所ところ。官庁かんちょう。役場やくば。

写る

  • （下したの字じや絵えが）すけて見みえる。
  • 写真しゃしんにとられる。

爪

  • 指ゆびの先さきにあり、皮かわがかたく変かわったもの。
  • 琴ことなどをひくときに、指先ゆびさきにはめるもの。
  • 引ひっかけてとめるための、小ちいさなもの。

遺伝子

  • 染色体せんしょくたいの中なかにあって、遺伝いでんのはたらきをする物質ぶっしつ。

増やす

  • 多おおくする。
