暑あつい日ひが続つづいて病院びょういんに運はこばれた人ひとが１週間しゅうかんに２０００人にん以上いじょう

发表于 2019-05-29 更新于 2019-11-30 分类于 JapanEasyNews



総務省消防庁そうむしょうしょうぼうちょうによると、５月がつ２０日はつかから２６日にちまでの１週間しゅうかんに熱中症ねっちゅうしょうのようだと言いって病院びょういんに運はこばれた人ひとは２０００人にん以上いじょういました。その前まえの週しゅうの４倍ばいぐらいです。
総務省消防庁そうむしょうしょうぼうちょうは、この１週間しゅうかんはいつもの年としよりずっと暑あつい日ひが続つづいたため、病院びょういんに運はこばれた人ひとが増ふえたと考かんがえています。
６月がつ１日ついたちから１０日とおかもいつもの年としよりずっと暑あつくなりそうです。総務省消防庁そうむしょうしょうぼうちょうは「体からだが暑あつさにまだ慣なれていません。冷房れいぼうを使つかったり、よく水みずを飲のんだりして、熱中症ねっちゅうしょうにならないようにしてください」と言いっています。
