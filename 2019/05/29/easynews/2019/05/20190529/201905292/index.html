エベレストで男性だんせいが亡なくなる　大勢おおぜいの人ひとで混こんでいた

发表于 2019-05-29 更新于 2019-11-30 分类于 JapanEasyNews



２５日にち、世界せかいでいちばん高たかいエベレストの頂上ちょうじょうの近ちかくで、イギリス人じんの４４歳さいの男性だんせいが亡なくなりました。
頂上ちょうじょうの近ちかくは大勢おおぜいの人ひとで混こんでいて、長ながい列れつができていました。このため、男性だんせいは酸素さんそが少すくない所ところで待またなければならなくて、体からだの具合ぐあいが悪わるくなりました。
世界せかいでは山やまに登のぼる人ひとが増ふえています。フランスの通信社つうしんしゃは、今年ことしエベレストに登のぼる人ひとは今いままででいちばん多おおいかもしれないと言いっています。専門家せんもんかは「ネパールの政府せいふは、エベレストに登のぼってもいい人ひとをもっと少すくなくしたほうがいいと思おもいます」と言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

エベレスト

  • 中国ちゅうごくとネパールとの国境こっきょう、ヒマラヤ山脈さんみゃくにある山やま。高たかさ８８４８メートルで、世界一せかいいち高たかい。

頂

  • 山やまなどのいちばん高たかい所ところ。頂上ちょうじょう。

列

  • 長ながく並ならぶ。並ならんだもの。
  • 順序じゅんじょ。
  • 加くわわる。

酸素

  • 色いろも、においもない気体きたい。空気くうきや水みずの中なかにふくまれていて、生物せいぶつはこれを吸すって生いきている。また、物ものが燃もえるには、酸素さんそが必要ひつようである。

通信社

  • ニュースを集あつめて、新聞社しんぶんしゃ・放送局ほうそうきょく・雑誌社ざっししゃなどに送おくる仕事しごとをする会社かいしゃ。

政府

  • 国くにの政治せいじを行おこなうところ。
  • 内閣ないかく。
