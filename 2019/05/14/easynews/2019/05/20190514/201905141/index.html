たくさん引用いんようされた科学かがくの論文ろんぶん　１番ばんはアメリカで２番ばんは中国ちゅうごく

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



いい研究けんきゅうの論文ろんぶんは、別べつの論文ろんぶんで一部いちぶが使つかわれることがあります。これを「引用いんよう」と言いいます。
日本にっぽんの科学技術振興機構かがくぎじゅつしんこうきこうは、どの国くにの論文ろんぶんがいちばん引用いんようされたか、１５１の科学かがくの分野ぶんやについて調しらべました。
２０１５年ねんから２０１７年ねんまで平均へいきんすると、１番ばん多おおかったのはアメリカの論文ろんぶんで、８０の分野ぶんやでいちばん引用いんようされていました。２番目ばんめは中国ちゅうごくの論文ろんぶんで、７１の分野ぶんやでいちばん引用いんようされていました。どの分野ぶんやでもいちばん引用いんようされたのはこの２つの国くにの論文ろんぶんだけでした。
次つぎに、引用いんようされた回数かいすうが上うえから５番目ばんめまでだった論文ろんぶんの国くにを調しらべると、日本にっぽんは１８の分野ぶんやで５番ばん以内いないでした。２０年ねん前まえは８３の分野ぶんやで、そのときよりずっと少すくなくなっています。
科学技術振興機構かがくぎじゅつしんこうきこうでは「日本にっぽんはいい研究けんきゅうをする力ちからが低ひくくなっていることがわかりました。大おおきな問題もんだいだと思おもいます」と話はなしています。
