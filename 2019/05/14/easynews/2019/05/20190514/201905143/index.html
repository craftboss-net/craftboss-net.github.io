横浜市よこはましに住すむ外国人がいこくじん　初はじめて１０万まん人にん以上いじょうになる

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



横浜市よこはましに住すむ外国人がいこくじんが先月せんげつ、１０万まん２２７人にんになりました。横浜市よこはましの外国人がいこくじんの人口じんこうが１０万まん人にん以上いじょうになったのは初はじめてで、５年ねん前まえより３０％以上いじょう増ふえています。
いちばん多おおいのは中国人ちゅうごくじんで、約やく４万まん人にん住すんでいます。最近さいきん増ふえているベトナム人じんは約やく７５００人にんで、ネパール人じんは約やく４０００人にんです。どちらも、５年ねん前まえの３倍ばい以上いじょうに増ふえました。
横浜市よこはましはこれからも外国人がいこくじんが増ふえると考かんがえています。このため、外国語がいこくごで相談そうだんできる場所ばしょを増ふやしたり、日本語にほんごを勉強べんきょうする手伝てつだいをしたりして、外国人がいこくじんが生活せいかつしやすい市しにしようとしています。
