アメリカ「中国ちゅうごくから輸入ゆにゅうする品物しなものほとんどの関税かんぜいを上あげる」

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



アメリカと中国ちゅうごくは、貿易ぼうえきの問題もんだいについて話はなし合あいをしていますが、うまく進すすんでいません。アメリカは１０日とおか、中国ちゅうごくから輸入ゆにゅうしている一部いちぶの品物しなものの関税かんぜいを２５％上あげました。関税かんぜいを上あげた品物しなものを全部ぜんぶ足たすと、２０００億おくドルになります。
中国ちゅうごくも１３日にち、アメリカから輸入ゆにゅうしている一部いちぶの品物しなものの関税かんぜいを、来月らいげつ１日ついたちから上あげると言いいました。関税かんぜいが上あがる品物しなものを足たすと６００億おくドルで、この中なかには２５％高たかくなる品物しなものもあります。
中国ちゅうごくが関税かんぜいを上あげると言いったあと、アメリカはまだ関税かんぜいを上あげていなかった品物しなものも、上あげる準備じゅんびを始はじめたと言いいました。この品物しなものを全部ぜんぶ足たすと、３０００億おくドルになります。
アメリカの政府せいふは来月らいげつに会議かいぎを開ひらいて、いろいろな人ひとの意見いけんを聞きいてから決きめると言いっています。残のこっている品物しなものの関税かんぜいが上あがると、アメリカが中国ちゅうごくから輸入ゆにゅうしているほとんど全部ぜんぶの品物しなものの関税かんぜいが上あがることになります。
