小ちいさな子こどもを助たすけよう　ＡＥＤの使つかい方かたを教おしえる会かいを開ひらく

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



日本赤十字社にっぽんせきじゅうじしゃは、小ちいさな子こどもの命いのちを助たすけるために、「ＡＥＤ」をどのように使つかったらいいか教おしえる会かいを開ひらきました。ＡＥＤは、心臓しんぞうが急きゅうに止とまった人ひとの体からだに電気でんきを流ながすパッドを貼はって、心臓しんぞうを動うごかす機械きかいです。
大阪市おおさかしで開ひらいた会かいには、保育園ほいくえんの先生せんせいなど３３人にんが参加さんかしました。大人おとなとは違ちがって子こどもには、パッドは小ちいさなものを使つかって、胸むねと背中せなかに貼はったほうがいいと説明せつめいがありました。そのあと、赤あかちゃんの人形にんぎょうを使つかって、２本ほんの指ゆびで心臓しんぞうのマッサージをするやり方かたやＡＥＤの使つかい方かたを練習れんしゅうしました。
ＡＥＤは事故じこで心臓しんぞうが急きゅうに止とまった場合ばあいに役やくに立たつため、専門家せんもんかは練習れんしゅうして使つかってほしいと言いっています。
滋賀県しがけん大津市おおつしで今月こんげつの初はじめに交通事故こうつうじこがあって、保育園ほいくえんの子こどもが２人ふたり亡なくなりました。事故じこのときなどに、小ちいさな子こどもをどのように助たすけたらいいか考かんがえる人ひとが増ふえています。
