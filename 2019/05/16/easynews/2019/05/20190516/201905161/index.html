今年ことし「はしか」になった人ひとが去年きょねん１年ねんの１．７倍ばいになる

发表于 2019-05-16 更新于 2019-11-30 分类于 JapanEasyNews



「はしか」は高たかい熱ねつやせきが出でたり、体からだに赤あかい発疹はっしんが出でたりする病気びょうきです。ウイルスでうつりやすくて、とてもひどくなることがあります。妊娠にんしんしている女性じょせいがはしかになると、おなかの中なかの赤あかちゃんが亡なくなることもあります。
国立感染症研究所こくりつかんせんしょうけんきゅうしょによると、５月がつ５日いつかまでの１週間しゅうかんに、はしかになったことがわかった人ひとは１１人にんいました。今年ことし、はしかになった人ひとは４６７人にんで、去年きょねん１年ねんの１．７倍ばいに増ふえています。大阪府おおさかふでは１３６人にんがはしかになりました。東京都とうきょうとや三重県みえけん、愛知県あいちけんなどでも多おおくなっています。
厚生労働省こうせいろうどうしょうは、４月がつの終おわりから５月がつの初はじめの休やすみの間あいだに、はしかのウイルスがうつった人ひとは、これから熱ねつやせきが出でるかもしれないと言いっています。そして、高たかい熱ねつなどが出でた場合ばあいは、まず電話でんわで病院びょういんに相談そうだんしてほしいと言いっています。
