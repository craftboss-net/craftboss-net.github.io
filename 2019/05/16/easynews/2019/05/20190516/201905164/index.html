東京とうきょうオリンピック　「ガンダム」が宇宙うちゅうから選手せんしゅを応援おうえんする

发表于 2019-05-16 更新于 2019-11-30 分类于 JapanEasyNews



「機動戦士きどうせんしガンダム」はロボットが宇宙うちゅうで戦たたかう日本にっぽんのアニメで、とても人気にんきがあります。東京とうきょうオリンピックとパラリンピックの組織委員会そしきいいんかいとＪＡＸＡは、このアニメに出でてくる「ガンダム」と「ザク」のロボットの模型もけいを人工衛星じんこうえいせいに入いれます。そして、国際こくさい宇宙うちゅうステーションに行いく宇宙船うちゅうせんに乗のせて、来年らいねん３月がつに打うち上あげます。
人工衛星じんこうえいせいは宇宙うちゅうで、２つの模型もけいを外そとへ出だします。模型もけいは高たかさ１０ｃｍぐらいで、人工衛星じんこうえいせいの上うえに立たつような形かたちになります。来年らいねん７月がつにオリンピックが始はじまると、選手せんしゅを応援おうえんするメッセージを模型もけいの下したのディスプレーに出だします。人工衛星じんこうえいせいのカメラが撮とったメッセージや模型もけいは、インターネットのＳＮＳなどで見みせる予定よていです。
組織委員会そしきいいんかいの室伏むろふし広治こうじさんは「宇宙うちゅうからガンダムに応援おうえんしてもらったら、選手せんしゅも頑張がんばることができると思おもいます。楽たのしみにしています」と話はなしています。

宇宙

オリンピック

人工衛星
