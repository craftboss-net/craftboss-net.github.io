箱根山はこねやま　火山かざんの活動かつどうが原因げんいんの地震じしんが増ふえている

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



神奈川県かながわけんの箱根山はこねやまでは、１８日にちから火山かざんの活動かつどうが原因げんいんの地震じしんが増ふえています。気象庁きしょうちょうは１９日にち、「火口周辺警報かこうしゅうへんけいほう」を出だして、大涌谷おおわくだにでは噴火ふんかで飛とんでくる石いしに気きをつけるように言いいました。大涌谷おおわくだには、いつもは大勢おおぜいの人ひとが旅行りょこうに来くる場所ばしょです。
箱根町はこねまちは、大涌谷おおわくだにの火口かこうの近ちかくには入はいらないように言いいました。大涌谷おおわくだにに向むかう道みちは一部いちぶ通とおることができません。大涌谷おおわくだにの上うえを通とおるロープウエーも運転うんてんを休やすんでいます。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ

火山

  • 地下ちかのマグマが、地表ちひょうにふき出だして山やまとなっている所ところ。

活動

  • 元気げんきよく動うごいたり、働はたらいたりすること。

噴火

  • 火山かざんが爆発ばくはつして、とけた溶岩ようがんや、火山灰かざんばい・水蒸気すいじょうき・ガスをふき出だすこと。

噴火口

  • 噴火ふんかのときに、とけた岩いわやガスなどのふき出でる所ところ。火口かこう。

一部

  • 全体ぜんたいの中なかの、ある部分ぶぶん。
  • 新聞しんぶんや雑誌ざっしなどを数かぞえるときの、一ひとつ。

ロープウエー

  • 空中くうちゅうにロープを張はって箱はこをつるし、人ひとや物ものを運はこぶ乗のり物もの。
