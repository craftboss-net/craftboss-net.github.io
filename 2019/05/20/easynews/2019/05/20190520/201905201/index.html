アメリカ　窓まどの掃除そうじのゴンドラが高たかさ２６０ｍでひどく揺ゆれる

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのオクラホマ州しゅうで１５日にち、窓まどの掃除そうじをする２人ふたりが乗のったゴンドラが、ビルの高たかさ２６０ｍの所ところでコントロールできなくなりました。ゴンドラは、前まえや後うしろ、左ひだりや右みぎに揺ゆれて、だんだんひどく揺ゆれるようになって、ビルにぶつかって窓まどが割われました。
この揺ゆれるゴンドラを撮とったビデオには、落おちないようにしっかりつかまっている２人ふたりがうつっています。２人ふたりは、消防しょうぼうの人ひとが用意よういしたロープを使つかって揺ゆれるゴンドラを止とめたあと、無事ぶじに助たすけられました。ビルの周まわりには割われたガラスが落おちましたが、けがをした人ひとはいませんでした。
このゴンドラのビデオはインターネットに出でて、６時間じかんで１００万まん回かいぐらい見みられました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ゴンドラ

  • イタリアのベネチアで使つかっている、細長ほそながい小船こぶね。
  • 気球ききゅうやロープウエーなどの、つりかご。

コントロール

  • うまく調節ちょうせつすること。また、自分じぶんの思おもうように動うごかすこと。
  • 野球やきゅう・ソフトボールで、投手とうしゅが自分じぶんの思おもうところに、球たまを投なげる力ちから。

ぶつかる

  • 打うち当あたる。つき当あたる。
  • 困難こんなんなことに出であう。
  • 争あらそう。立たち向むかう。
  • 重かさなる。

ビデオ

  • 映像えいぞう。
  • 「ビデオテープ」の略りゃく。
  • 映像えいぞうや音おとを、磁気じきテープに記録きろくしたり再生さいせいしたりする装置そうち。

つかまる

  • 手てでしっかりとにぎる。

写る

  • （下したの字じや絵えが）すけて見みえる。
  • 写真しゃしんにとられる。

消防

  • 火事かじを消けしたり、火災かさいを防ふせいだりすること。

ロープ

  • あさ糸いとや、針金はりがねなどを太ふとくよった丈夫じょうぶなつな。

無事

  • 変かわったことがないこと。
  • 健康けんこうなこと。元気げんき。

助ける

  • 危あぶないことや、苦くるしいことから、救すくう。
  • 力ちからを貸かす。手伝てつだう。
