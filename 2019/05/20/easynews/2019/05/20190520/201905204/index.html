岐阜県ぎふけん白川村しらかわむら　世界せかいの職人しょくにんが合掌造がっしょうづくりの家いえの屋根やねを新あたらしくする

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



岐阜県ぎふけん白川村しらかわむらには、山やまのような形かたちをした合掌造がっしょうづくりの家いえがたくさんあって、世界遺産せかいいさんになっています。
合掌造がっしょうづくりの家いえのように、かやを使つかった屋根やねの文化ぶんかがある国くにの人ひとたちが集あつまって、日本にっぽんで会議かいぎを行おこなっています。白川村しらかわむらでは１９日にち、オランダなど世界せかいの職人しょくにんたちと村むらの人ひとたち１８０人にんが、屋根やねを新あたらしくする作業さぎょうを行おこないました。
作業さぎょうでは、長ながさ２ｍ５０ｃｍぐらいのかやを屋根やねの上うえにたくさん置おいて、道具どうぐを使つかってかやを強つよくたたいて、屋根やねを新あたらしくしました。
スウェーデンから来きた職人しょくにんは「新あたらしいことを習ならってすばらしい経験けいけんになりました。同おなじ仕事しごとをしている人ひとたちなので、ことばがわからなくても交流こうりゅうできました」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

職人

  • 手先てさきの技術ぎじゅつで物ものを作つくる仕事しごとをしている人ひと。大工だいくさん・石屋いしやさん・植木屋うえきやさんなど。

合掌造り

  • 木造もくぞうの大おおきな家いえの屋根やねの造つくり方かたの一ひとつで、木材もくざいを山やまの形かたちに組くみ合あわせたもの。

屋根

  • 雨あめなどを防ふせぐために、家いえをおおうもの。
  • 上うえをおおうもの。
  • いちばん高たかい所ところ。

世界遺産

  • ユネスコの「世界せかいの文化遺産ぶんかいさん及および自然遺産しぜんいさんの保護ほごに関かんする条約じょうやく」（「世界遺産保護条約せかいいさんほごじょうやく」）にもとづいて決きめられた、世界的せかいてきに残のこす価値かちがあると認みとめられた文化ぶんかや自然しぜん。日本にっぽんでは、文化遺産ぶんかいさんとして姫路城ひめじじょうや法隆寺ほうりゅうじ・沖縄おきなわの首里城しゅりじょうなど、自然遺産しぜんいさんとして屋久島やくしまや白神山地しらかみさんちなどが指定していされている。

かや

  • ススキ・スゲ・チガヤなど、葉はの細長ほそながい草くさ。

作業

  • 仕事しごとをすること。

たたく

  • ぶつ。なぐる。
  • 打うって音おとを出だす。
  • 悪口わるくちを言いって、やっつける。
  • やたらにしゃべる。
  • 値段ねだんをまけさせる。

交流

  • たがいに行いき来きすること。交まじり合あうこと。
  • 決きまった時間じかんごとに、流ながれの方向ほうこうが逆ぎゃくになる電流でんりゅう。
