ロシア　飛行機ひこうきの事故じこで４１人にんが亡なくなる

发表于 2019-05-07 更新于 2019-11-30 分类于 JapanEasyNews



５日いつか、ロシアの航空こうくう会社がいしゃ「アエロフロート」の飛行機ひこうきが、モスクワの空港くうこうを出発しゅっぱつして飛とんでいる途中とちゅうに問題もんだいが出でました。このため飛行機ひこうきは４０分ぷんあとに空港くうこうに戻もどりましたが、下おりるときに失敗しっぱいして火ひが出でました。この事故じこで、飛行機ひこうきに乗のっていた７８人にんのうち４１人にんが亡なくなりました。
この飛行機ひこうきのパイロットは６日むいか、「飛行機ひこうきに雷かみなりが落おちて、空港くうこうなどと連絡れんらくする機械きかいが壊こわれました」と話はなしました。
ロシア政府せいふの委員会いいんかいは、事故じこの原因げんいんを調しらべています。
