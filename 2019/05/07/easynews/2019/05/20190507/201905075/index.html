原子力げんしりょく発電所はつでんしょで事故じこがあった福島県ふくしまけん大熊町おおくままちに役場やくばが戻もどる

发表于 2019-05-07 更新于 2019-11-30 分类于 JapanEasyNews



福島県ふくしまけん大熊町おおくままちは、２０１１年ねんに原子力げんしりょく発電所はつでんしょで事故じこがあったあと、人ひとが住すむことができなくなりました。国くにが放射線ほうしゃせんの量りょうを少すくなくする作業さぎょうをして、今年ことし４月がつ１０日とおかから大熊町おおくままちの一部いちぶに人ひとが住すむことができるようになりました。
町まちの役場やくばも７日なのか、新あたらしい建物たてもので仕事しごとを始はじめました。事故じこのあと、役場やくばは福島県ふくしまけんの別べつの市しなどで仕事しごとをしていました。
渡辺わたなべ利綱としつな町長ちょうちょうは仕事しごとを始はじめる前まえに、約やく８０人にんの職員しょくいんに「町まちの人ひとたちが安心あんしんすることができる安全あんぜんな町まちにしましょう」と話はなしました。

原子力

  • 原子核げんしかくがこわれたり（核分裂かくぶんれつ）、とけあったり（核融合かくゆうごう）するときに出でる、ものすごく大おおきな力ちからと高たかい熱ねつ。原子げんしエネルギー。

  • 電気でんきを起おこす所ところ。

  • 町まちや村むらの仕事しごとをする役所やくしょ。

放射線

  • ウランやラジウムなどの放射性ほうしゃせい元素げんそがこわれて、ほかの元素げんそに変かわるときに出だすもの。大量たいりょうに浴あびると有害ゆうがいである。
