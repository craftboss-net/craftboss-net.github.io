去年きょねんとれた水産物すいさんぶつ　スルメイカが少すくなくなってサバは増ふえる

发表于 2019-05-07 更新于 2019-11-30 分类于 JapanEasyNews



農林水産省のうりんすいさんしょうによると、去年きょねん漁りょうでとれた魚さかななどの水産物すいさんぶつは全部ぜんぶで４３８万まん９０００ｔでした。おととしと比くらべて２％増ふえましたが、いちばん多おおかった３０年ねんぐらい前まえと比くらべると３０％ぐらいになりました。
スルメイカは約やく４万まん６０００ｔで、おととしより２８％少すくなくなりました。調しらべ始はじめた１９５６年ねんからでいちばん少すくなくなりました。
サンマは５４％、サケの仲間なかまも２３％増ふえましたが、どちらもいちばん多おおかった１０年ねん前まえと比くらべると半分はんぶん以下いかになっています。
反対はんたいに、サバの仲間なかまは２年ねん続つづけて増ふえて、約やく５３万まん７０００ｔでした。マイワシなども増ふえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

捕れる

  • 魚さかな・鳥とりなどが得えられる。
  • とることができる。

水産物

  • 川かわ・海うみ・湖みずうみなどでとれる、魚さかな・貝かい・海藻かいそうなど。

烏賊

  • 海うみにすむ動物どうぶつ。スルメイカ・ヤリイカ・ホタルイカなど。胴どうは細長ほそながいふくろの形かたちで、１０本ぽんの足あしが頭あたまの部分ぶぶんから出でている。敵てきにあうと、すみをはいてにげる。

鯖

  • 暖あたたかい海うみに群むれを作つくってすむ魚さかな。食用しょくようにする。

農林水産省

  • 農業のうぎょう・林業りんぎょう・水産業すいさんぎょう・畜産業ちくさんぎょうなどについての仕事しごとをする、国くにの役所やくしょ。農水省のうすいしょう。

漁

  • 魚さかななどをとること。

トン

  • メートル法ほうで、重おもさの単位たんいの一ひとつ。一トンは、１０００キログラム。記号きごうは「t」。
  • 船ふねの容積ようせきの単位たんい。
  • 貨車かしゃやトラックの容積ようせきの単位たんい。

約

  • ちかう。取とり決きめる。
  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。

秋刀魚

  • 冷つめたい海うみにすむ細長ほそながい魚さかな。秋あきに多おおくとれる。

鮭

  • 川かわで生うまれ、海うみに下くだって育そだち、寒流かんりゅうにすむ魚さかな。四し、五ご年ねんたつと、秋あき、生うまれた川かわをさかのぼって卵たまごを産うむ。身みだけでなく、卵たまごも「すじこ」「イクラ」と呼よばれて食用しょくようにする。シャケ。アキアジ。

仲間

  • いっしょにものごとをする人ひと。友達ともだち。グループ。
  • 種類しゅるいの同おなじもの。

鰯

  • 近海きんかいの暖あたたかな海うみにいる魚さかな。マイワシ・ウルメイワシ・カタクチイワシがいる。背せは青黒あおぐろくて、腹はらは白しろい。食用しょくようにする。
