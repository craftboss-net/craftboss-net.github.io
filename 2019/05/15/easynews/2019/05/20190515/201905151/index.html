３８００年ねん前まえに日本にっぽんにいた縄文人じょうもんじんは酒さけに強つよかった

发表于 2019-05-15 更新于 2019-11-30 分类于 JapanEasyNews



国立科学博物館こくりつかがくはくぶつかんなどのグループは、３８００年ねんぐらい前まえの縄文時代じょうもんじだいに日本にっぽんに住すんでいた縄文人じょうもんじんの女性じょせいのＤＮＡを調しらべました。この女性じょせいは、北海道ほっかいどうの礼文島れぶんとうで見みつかりました。
グループは、女性じょせいの歯はのＤＮＡがしっかり残のこっていたので、遺伝子いでんしの情報じょうほうを全部ぜんぶ調しらべることができたと言いっています。調しらべた結果けっか、この女性じょせいは目めと肌はだの色いろが茶色ちゃいろで、酒さけに強つよいことなどがわかりました。
脂肪しぼうが多おおい食たべ物ものを消化しょうかしやすい遺伝子いでんしを持もっていることもわかりました。グループは、動物どうぶつや魚さかなをとって生活せいかつしていたようだと言いっています。そして、縄文人じょうもんじんの遺伝子いでんしの１０％ぐらいが今いまの日本人にっぽんじんに伝つたわっていると考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

グループ

  • 仲間なかま。集団しゅうだん。

縄文時代

  • 縄文式土器じょうもんしきどきを作つくって使つかっていた時代じだい。１２０００年ねんほど前まえから２３００年前ねんまえごろの間あいだ。

DNA

  • 生物せいぶつの細胞さいぼうの中なかにあって、遺伝子いでんしを構成こうせいする物質ぶっしつ。

遺伝子

  • 染色体せんしょくたいの中なかにあって、遺伝いでんのはたらきをする物質ぶっしつ。

情報

  • できごとやものごとのようすについての知しらせ。

結果

  • あることがもとになって起おこったことがらやようす。

肌

  • 体からだの皮膚ひふ。
  • 土地とちなどの表面ひょうめん。
  • 性質せいしつ。

脂肪

  • 動物どうぶつや植物しょくぶつの中なかにふくまれているあぶら。栄養素えいようその一ひとつ。

消化

  • 食たべた物ものを、体からだのためになるものにこなすこと。
  • 十分じゅうぶん理解りかいして、自分じぶんのものにすること。
  • ものごとを残のこさずかたづけてしまうこと。

伝わる

  • 物ものに沿そって、動うごいていく。伝つたう。
  • 世よの中なかに知しれわたる。
  • 昔むかしから受うけつがれる。
  • よそから来くる。
  • 熱ねつや電気でんきなどが通とおる。
