東京都とうきょうとの小池こいけ知事ちじ「２０５０年ねんに二酸化炭素にさんかたんそをゼロにする」

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



地球ちきゅうの気温きおんが上あがって、天気てんきなどが変かわっているため、自然しぜんや人ひとの生活せいかつに問題もんだいが出でるようになりました。このため世界せかいの国くには、二酸化炭素にさんかたんそなど気温きおんが上あがる原因げんいんになるガスを減へらすことを決きめました。日本にっぽんの政府せいふは２０５０年ねんまでに、このようなガスを８０％減へらすことを考かんがえています。
東京都とうきょうとの小池こいけ知事ちじは２１日にち、２０５０年ねんに東京都とうきょうとから出でる二酸化炭素にさんかたんそをゼロにしたいと発表はっぴょうしました。政府せいふの目標もくひょうより早はやくゼロにしたいと考かんがえています。
そのために何なにをしたらいいか、１２月がつごろまでに計画けいかくを決きめると小池こいけ知事ちじは言いっています。家庭かていや大おおきなビルのプラスチックのごみを焼やく量りょうを、２０３０年ねんまでに４０％減へらすことも計画けいかくの中なかに入いれる予定よていです。

二酸化炭素

  • 炭素たんそと酸素さんその化合物かごうぶつで、色いろもにおいもない気体きたい。炭火すみびの燃もえるときなどに発生はっせいし、人ひとのはく息いきの中なかにもふくまれている。ドライアイス・ソーダ水すいなどに使つかわれる。炭酸たんさんガス。記号きごうは「CO2シーオーツー」。

地球
