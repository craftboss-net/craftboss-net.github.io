ティッシュペーパーが店みせに少すくなくても心配しんぱいありません

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



４月がつごろから、店みせに置おいてあるティッシュペーパーが少すくなくなっている所ところがあります。１人ひとりが買かうことができるティッシュペーパーの数かずを決きめている店みせもあります。
ティッシュペーパーを作つくる会社かいしゃの団体だんたいなどは、先月せんげつ、紙かみを作つくる大おおきな工場こうじょうで火事かじがあって、工場こうじょうが止とまったことなどが少すくなくなった理由りゆうだと言いっています。春はるは花粉かふんが飛とぶ季節きせつなので、花粉かふんのアレルギーがある人ひとがたくさんティッシュペーパーを買かったことも理由りゆうの１つだと言いっています。
団体だんたいは、花粉かふんの季節きせつが終おわって、工場こうじょうも紙かみを作つくり始はじめたので、店みせにティッシュペーパーが戻もどると考かんがえています。そして、店みせにティッシュペーパーが少すくなくても、心配しんぱいしないでくださいと言いっています。
