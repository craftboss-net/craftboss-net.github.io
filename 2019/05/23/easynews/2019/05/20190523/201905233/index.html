皇后こうごうさま　日本赤十字社にっぽんせきじゅうじしゃの大会たいかいに出席しゅっせきする

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



皇室こうしつは赤十字せきじゅうじの活動かつどうを応援おうえんしています。日本赤十字社にっぽんせきじゅうじしゃの名誉めいよ総裁そうさいは、昔むかしからずっと皇后こうごうさまがしています。
２２日にち、東京とうきょうで日本赤十字社にっぽんせきじゅうじしゃの今年ことしの大会たいかいがありました。大会たいかいには１９００人にんが集あつまって、今月こんげつから名誉めいよ総裁そうさいになった皇后こうごうさまが皇室こうしつの女性じょせいたちと一緒いっしょに出席しゅっせきしました。皇后こうごうさまは、赤十字せきじゅうじの活動かつどうに協力きょうりょくした人ひとや団体だんたいに賞状しょうじょうなどを渡わたしました。
そのあと、岡山赤十字病院おかやませきじゅうじびょういんで働はたらく男性だんせいの医者いしゃが、去年きょねん７月がつに西にし日本にほんで大雨おおあめが降ふったときに多おおくの人ひとを助たすけた活動かつどうについて発表はっぴょうしました。男性だんせいは、大雨おおあめが降ふる３か月げつ前まえに訓練くんれんをしていたため、すぐに活動かつどうできたと話はなしました。皇后こうごうさまは、男性だんせいの話はなしを聞きいてうなずいていました。皇后こうごうさまは会場かいじょうを出でるときに男性だんせいに「いつも訓練くんれんをしておくことがとても大事だいじですね」と話はなしました。
５月がつ１日ついたちに天皇てんのう陛下へいかが即位そくいしてから、皇后こうごうさまがお一人ひとりで仕事しごとをするのは初はじめてです。
