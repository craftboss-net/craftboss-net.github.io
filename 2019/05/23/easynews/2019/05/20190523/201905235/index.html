パナソニックがファーウェイに部品ぶひんを売うることをやめる

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのパナソニックは、中国ちゅうごくのファーウェイがスマートフォンに使つかう部品ぶひんなどを作つくっています。アメリカの政府せいふは、アメリカの会社かいしゃがファーウェイに部品ぶひんなどを売うることを禁止きんししようとしています。
パナソニックの部品ぶひんなどの中なかには、アメリカの会社かいしゃの部品ぶひんや技術ぎじゅつを使つかって作つくった物ものがあって、ファーウェイに売うることができなくなるかもしれません。このため、パナソニックはファーウェイに部品ぶひんなどを売うることをやめると決きめました。
ファーウェイによると、今いままでにファーウェイに部品ぶひんなどを売うった日本にっぽんの会社かいしゃは１００以上いじょうあります。パナソニックが売うることをやめるため、ほかの会社かいしゃもやめるかもしれません。
日本にっぽんの携帯電話けいたいでんわの会社かいしゃのＫＤＤＩ、ソフトバンク、楽天らくてんモバイルなどは、ファーウェイの新あたらしいスマートフォンを売うり始はじめる日ひを予定よていより遅おそくすることにしました。ＮＴＴドコモは、ファーウェイの新あたらしいスマートフォンの予約よやくの受うけ付つけをやめると言いいました。
