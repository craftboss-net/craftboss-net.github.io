怪あやしい船ふねから１０００ｋｇの覚醒かくせい剤ざい　中国人ちゅうごくじん７人にんを逮捕たいほ

发表于 2019-06-06 更新于 2019-11-30 分类于 JapanEasyNews



３日みっかの夜よる、静岡県しずおかけん南伊豆町みなみいずちょうの海うみで、船ふねの名前なまえが書かいていない怪あやしい船ふねが見みつかりました。警察けいさつなどが船ふねの中なかを調しらべると、たくさんの覚醒かくせい剤ざいが見みつかりました。覚醒かくせい剤ざいは体からだに悪わるくて危険きけんな薬くすりで、法律ほうりつで禁止きんしされています。
警察けいさつによると、覚醒かくせい剤ざいは全部ぜんぶで１０００ｋｇぐらいあって、値段ねだんは６００億おく円えんぐらいです。日本にっぽんで１回かいに見みつかった覚醒かくせい剤ざいの中なかでいちばん多おおいと警察けいさつは言いっています。
警察けいさつは船ふねなどにいた７人にんの中国人ちゅうごくじんを、覚醒かくせい剤ざいを持もっていた疑うたがいで逮捕たいほしました。覚醒かくせい剤ざいは香港ほんこんから輸入ゆにゅうされて、日本にっぽんや外国がいこくの犯罪はんざい グループが関係かんけいしていると警察けいさつは考かんがえています。
