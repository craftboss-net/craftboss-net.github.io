ＪＲ東日本ひがしにほん　水素すいそからつくった電気でんきで走はしる電車でんしゃをつくる

发表于 2019-06-06 更新于 2019-11-30 分类于 JapanEasyNews



ＪＲ東日本ひがしにほんは、水素すいそからつくった電気でんきで走はしる電車でんしゃをつくることにしました。タンクに入いれた水素すいそと空気くうきの中なかにある酸素さんそを使つかって、電気でんきをつくります。
電車でんしゃの上うえにあるタンクに水素すいそを全部ぜんぶ入いれると、１４０ｋｍぐらい走はしることができます。電線でんせんや変電所へんでんしょは要いりません。
水素すいそからつくった電気でんきで走はしる電車でんしゃはドイツでは運転うんてんしていますが、日本にっぽんの法律ほうりつでは運転うんてんできません。このため、ＪＲ東日本ひがしにほんは、法律ほうりつを変かえてほしいと国くにに言いうことにしています。

水素

  • いちばん軽かるくて、色いろも、においもない気体きたい。燃もえやすい。

タンク

  • 水みず・ガス・石油せきゆなどをためておく、大おおきな入いれ物もの。
  • 戦車せんしゃ。

酸素

  • 色いろも、においもない気体きたい。空気くうきや水みずの中なかにふくまれていて、生物せいぶつはこれを吸すって生いきている。また、物ものが燃もえるには、酸素さんそが必要ひつようである。

電線

  • 電気でんきを通とおす金属きんぞくの線せん。

変電所

  • 発電所はつでんしょから送おくられる電流でんりゅうを弱よわくして工場こうじょうや家庭かていに配くばったり、電圧でんあつを高たかくして遠とおくに送おくったりする設備せつびのある所ところ。
