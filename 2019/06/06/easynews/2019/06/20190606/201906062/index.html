サーロー節子せつこさんがカナダのトロント大学だいがくの名誉めいよ博士はくしになる

发表于 2019-06-06 更新于 2019-11-30 分类于 JapanEasyNews



サーロー節子せつこさんは１９４５年ねんに広島ひろしまに原爆げんばくが落おとされたとき、１３歳さいで被爆ひばくしました。サーローさんは１９５５年ねんからカナダのトロント大学だいがくの大学院だいがくいんで社会福祉しゃかいふくしを勉強べんきょうしました。そのあと、仕事しごとをしながら、世界中せかいじゅうで被爆ひばくの経験けいけんについて話はなして、核兵器かくへいきをなくすための活動かつどうを続つづけてきました。
トロント大学だいがくは４日よっか、サーローさんの活動かつどうがすばらしいと考かんがえて、サーローさんを大学だいがくの名誉めいよ博士はくしにしました。
サーローさんは「もう誰だれも、被爆ひばくした私わたしたちと同おなじ経験けいけんをしないでほしいです」と言いいました。そして、「たくさんの人ひとや国くにが努力どりょくして、核兵器かくへいきを禁止きんしする条約じょうやくが国連こくれんで決きまりました。世界せかいから核兵器かくへいきをなくすための努力どりょくをやめてはいけません」と話はなしました。
