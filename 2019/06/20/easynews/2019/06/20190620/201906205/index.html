東京とうきょうオリンピックのチケット　うそのメールに気きをつけて

发表于 2019-06-20 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックの組織委員会そしきいいんかいは２０日はつか、チケットの抽ちゅうせんの結果けっかをウェブサイトで発表はっぴょうしました。チケットを申もうし込こんだ人ひとには、結果けっかを知しらせるメールも送おくっています。そして、お金かねの払はらい方かたなどはウェブサイトを見みてほしいと言いっています。
警察けいさつなどは、抽ちゅうせんの結果けっかだとうそを書かいて、だましてお金かねやメールのアドレスなど個人情報こじんじょうほうを盗ぬすもうとするメールがくるかもしれないので気きをつけてほしいと言いっています。
組織委員会そしきいいんかいが送おくるメールには、ウェブサイトのＵＲＬやお金かねを払はらう銀行ぎんこうの口座こうざ番号ばんごうなどは書かいてありません。組織委員会そしきいいんかいは「チケットが当あたったかどうかはウェブサイトで調しらべてください。だまされないように気きをつけてください」と言いっています。
