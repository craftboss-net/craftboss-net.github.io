香港ほんこん　新あたらしい規則きそくに反対はんたいする若わかい人ひとの抗議こうぎが続つづく

发表于 2019-06-13 更新于 2019-11-30 分类于 JapanEasyNews



香港ほんこんの政府せいふは、香港ほんこんに逃にげてきた容疑者ようぎしゃを中国ちゅうごくに渡わたすことができる新あたらしい規則きそくを作つくろうとしています。
香港ほんこんでは、この規則きそくに反対はんたいする若わかい人ひとたちの抗議こうぎが続つづいています。１２日にちは何なん万まん人にんもの人ひとがまちの中心ちゅうしんにある道みちに座すわったり、議会ぎかいの建物たてものの周まわりに集あつまったりしました。一部いちぶの人ひとは傘かさや柵さくを投なげたり、議会ぎかいの土地とちに入はいったりしました。警察けいさつはこの人ひとたちを止とめるため、ゴムでできた弾たまを撃うったり、目めが痛いたくなるガスを使つかったりしました。
香港ほんこんの政府せいふは７２人にんがけがをしたと言いっています。
警察けいさつは「まちは混乱こんらんしている。若わかい人ひとたちは安全あんぜんを壊こわす危険きけんなことをしている」と強つよく言いって、香港ほんこんの人ひとたちにまちの中心ちゅうしんに行いかないように言いいました。
１２日にちは新あたらしい規則きそくについて議会ぎかいで話はなし合あう予定よていでしたが、中止ちゅうしになりました。香港ほんこんの政府せいふは、新あたらしい規則きそくを作つくるという考かんがえ方かたを変かえていません。香港ほんこんの混乱こんらんは続つづきそうです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

抗議

  • 相手あいてに対たいして反対はんたいの考かんがえを強つよく言いうこと。また、反対はんたいの意見いけん。

容疑者

  • 罪つみをおかした疑うたがいのある人ひと。

中心

  • 真まん中なか。
  • ものごとのいちばんだいじなところ。

議会

  • 国民こくみんに選えらばれた議員ぎいんが集あつまって、国くにや地方ちほうの政治せいじについて話はなし合あい、取とり決きめをする所ところ。国くにの議会ぎかいの国会こっかいと、都道府県とどうふけんなどの地方議会ちほうぎかいとがある。

一部

  • 全体ぜんたいの中なかの、ある部分ぶぶん。
  • 新聞しんぶんや雑誌ざっしなどを数かぞえるときの、一ひとつ。

柵

  • 木きや竹たけなどを立たて、横木よこぎを打うちつけて作つくった囲かこい。

土地

  • つち。大地だいち。
  • 地面じめん。地所じしょ。
  • その地方ちほう。

ゴム

  • 熱帯地方ねったいちほうに生はえる、ゴムの木きのしるで作つくったもの。のび縮ちぢみする。タイヤ、ボール、くつの底そこなどに使つかう。

弾丸

  • ピストルや鉄砲てっぽうのたま。
  • 非常ひじょうに速はやいもののたとえ。

撃つ

  • （鉄砲てっぽうなどを）発射はっしゃする。

混乱

  • 入いり乱みだれて、まとまりがなくなること。

話し合う

  • たがいに話はなす。
  • 意見いけんを出だし合あう。
