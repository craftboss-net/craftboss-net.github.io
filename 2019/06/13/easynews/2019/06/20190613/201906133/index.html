ペルーで「ギラン?バレー症候群しょうこうぐん」という病気びょうきの人ひとが増ふえる

发表于 2019-06-13 更新于 2019-11-30 分类于 JapanEasyNews



「ギラン?バレー症候群しょうこうぐん」は、免疫めんえきが自分じぶんの神経しんけいを攻撃こうげきするため、手てや足あしに力ちからが入はいらなくなる病気びょうきです。呼吸こきゅうができなくなって亡なくなる人ひともいます。
ペルーの政府せいふによると、首都しゅとのリマや北きたにある観光地かんこうちでは、今年ことしになってからギラン?バレー症候群しょうこうぐんだとわかった人ひとが２０６人にんいました。このうち４人にんが亡なくなりました。
ペルーの政府せいふは今月こんげつ８日ようか、この病気びょうきになる人ひとが急きゅうに増ふえているため、健康けんこうに危険きけんがある状況じょうきょうだと言いいました。そして、体からだに力ちからが入はいらなくなったら、すぐに病院びょういんに行いくように言いっています。
ギラン?バレー症候群しょうこうぐんは、蚊かやウイルスなどでうつる病気びょうきと関係かんけいがあるかもしれないと言いわれています。政府せいふは専門家せんもんかと原因げんいんを調しらべています。
ペルーには世界遺産せかいいさんのマチュピチュがあって、日本人にっぽんじんもたくさん観光かんこうに行いきます。ペルーの日本大使館にっぽんたいしかんは、手てや食たべ物ものをきれいに洗あらって、衛生えいせいに気きをつけるように言いっています。
