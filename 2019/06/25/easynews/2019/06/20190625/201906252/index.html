アメリカで鹿児島県かごしまけんの焼酎しょうちゅうを紹介しょうかい「カクテルでもおいしい」

发表于 2019-06-25 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのワシントンにある日本大使館にっぽんたいしかんで、鹿児島県かごしまけんの焼酎しょうちゅうをアメリカでもっと飲のんでもらうためのイベントがありました。
アメリカのレストランの人ひとなど１２０人にんぐらいが来きて、芋いもや麦むぎでつくった焼酎しょうちゅうを楽たのしんでいました。
アメリカでは、日本酒にほんしゅや日本にっぽんのウイスキーを買かう人ひとが増ふえています。しかし、焼酎しょうちゅうを知しっている人ひとは少すくないため、焼酎しょうちゅうの売うり上あげは日本酒にほんしゅの５％ぐらいです。
イベントでは、焼酎しょうちゅうに合あう魚さかなの料理りょうりなどを出だしました。焼酎しょうちゅうを使つかったいろいろなカクテルも紹介しょうかいしました。参加さんかした女性じょせいは「焼酎しょうちゅうはカクテルでもすばらしい味あじで、私わたしは好すきです」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ

焼酎

  • 穀物こくもつやサツマイモなどからつくった、強つよい酒さけ。

イベント

  • 行事ぎょうじ。できごと。
  • 勝負しょうぶ。試合しあい。

芋

  • 植物しょくぶつの根ねや地下茎ちかけいが、でんぷんなどの養分ようぶんをたくわえて大おおきくなったもの。サツマイモ・サトイモ・ジャガイモなど。ダリアなどの球根きゅうこんもいう。

麦

  • 畑はたけに作つくる作物さくもつ。大麦おおむぎ・小麦こむぎ・ハダカ麦むぎ・ライ麦むぎなど、種類しゅるいが多おおい。世界的せかいてきに重要じゅうような穀物こくもつ。

日本酒

  • 米こめから造つくる、日本特有にっぽんとくゆうの酒さけ。清酒せいしゅ。

ウイスキー

  • 大麦おおむぎなどを原料げんりょうとして作つくった、強つよい酒さけ。

売り上げ

  • 品物しなものを売うって得えたお金かね。
