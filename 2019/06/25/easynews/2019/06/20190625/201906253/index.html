荷物にもつを届とどけるいろいろな国くにのロボットが集あつまった

发表于 2019-06-25 更新于 2019-11-30 分类于 JapanEasyNews



荷物にもつを届とどける会社かいしゃでは働はたらく人ひとが足たりなくなっています。荷物にもつが増ふえたため、特とくに、家いえや会社かいしゃなどに１軒けんずつ荷物にもつを届とどける人ひとが足たりません。このため、荷物にもつを届とどける会社かいしゃや国くには、人ひとの代かわりに自動じどうで走はしるロボットを使つかおうと考かんがえています。
２４日にじゅうよっか、経済産業省けいざいさんぎょうしょうは日本にっぽんやアメリカ、中国ちゅうごくなどの会社かいしゃが作つくったロボットを集あつめて、どのように走はしるかなどをチェックしました。
アメリカのロボットは、狭せまい道みちでも人ひとなどにぶつからないように走はしることができます。日本にっぽんのロボットはカメラやセンサーで周まわりを見みて、人ひとの近ちかくを通とおるとき「こんにちは」と言いいます。
アメリカやヨーロッパでは、このようなロボットが道みちを走はしることができる新あたらしい法律ほうりつを作つくり始はじめています。しかし、日本にっぽんの今いまの法律ほうりつではみんなが利用りようする道みちを走はしることはできません。
国くには来年らいねん３月がつまでに、ロボットが安全あんぜんに走はしるためのルールの案あんをいろいろな会社かいしゃと一緒いっしょに考かんがえます。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ロボット

  • 電気でんきや磁気じきの力ちからで動うごく人形にんぎょう。人造じんぞう人間にんげん。
  • 工場こうじょうなどで人間にんげんに代かわって、作業さぎょうする機械きかい。
  • （いつも）人ひとの言いいなりになって動うごく人ひと。

自動

  • 機械きかいなどが自分じぶんの力ちからで動うごくこと。

経済産業省

  • 外国がいこくとの取とり引ひきや、商業しょうぎょう・工業こうぎょうなどの産業さんぎょうをさかんにするための仕事しごとをする国くにの役所やくしょ。経産省けいさんしょう。

ぶつかる

  • 打うち当あたる。つき当あたる。
  • 困難こんなんなことに出であう。
  • 争あらそう。立たち向むかう。
  • 重かさなる。

センサー

  • 光ひかりや音おと、温度おんどなどに反応はんのうして、電気的でんきてきな信号しんごうを送おくる装置そうち。

ヨーロッパ

  • 世界せかいの六大州ろくだいしゅうの一ひとつ。アジアの北西ほくせい、アフリカの北きたにある。産業さんぎょうや文化ぶんかが発達はったつした国くにが多おおい。

ルール

  • 規則きそく。決きまり。

案

  • 思おもうこと。考かんがえ。計画けいかく。
  • 相談そうだんのことがら。
