東京とうきょうオリンピックではペットボトルを持もって会場かいじょうに入はいれる

发表于 2019-06-28 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックとパラリンピックは、とても暑あつい季節きせつに開ひらくので、熱中症ねっちゅうしょうが心配しんぱいです。大会たいかいの組織委員会そしきいいんかいは、見みに来きた人ひとが熱中症ねっちゅうしょうにならないように、ペットボトルを持もって会場かいじょうに入はいることができるようにしたいと考かんがえています。
オリンピックやパラリンピックではしばらく前まえから、ペットボトルを持もって会場かいじょうに入はいることはできません。飲のみ物ものではない液体えきたいをペットボトルに入いれてテロを行おこなうことを心配しんぱいしているためです。
組織委員会そしきいいんかいは、安全あんぜんのため、飲のみ物ものをどのくらい持もってきてもいいか決きめたり、入はいる前まえに飲のんでもらって調しらべたりするつもりです。
飲のみ物ものは会場かいじょうでも買かうことができますが、買かうために並ならんで時間じかんがかかることもありそうです。組織委員会そしきいいんかいは、会場かいじょうでみんなが飲のみ物ものを十分じゅうぶん飲のむことができるようにしたいと考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

オリンピック

  • ４年ごとに開ひらかれ、世界せかいじゅうの国々くにぐにから選手せんしゅが参加さんかする競技大会きょうぎたいかい。古代こだいギリシャのオリンピアで開ひらかれた古代こだいオリンピックにならって、フランスのクーベルタンの力ちからで、１８９６年ねんにギリシャのアテネで開ひらかれたのが、近代きんだいオリンピックの始はじまり。五輪ごりん。

ペットボトル

  • 樹脂製じゅしせいの容器ようき。軽かるくて丈夫じょうぶなので飲のみ物ものなどの容器ようきに使つかわれる。ポリエチレンテレフタレートの頭文字かしらもじを取とってPETぺットという。

パラリンピック

  • 身体障害者しんたいしょうがいしゃの国際こくさいスポーツ大会たいかい。４年ねんに一度いちど、オリンピック開催地かいさいちで行おこなわれる。

熱中症

  • 夏なつの暑あつさなどで、熱ねつが体内たいないにたまって起おこる病気びょうき。ひどい頭痛ずつうがしたり、気きを失うしなったりする。

大会

  • 多おおくの人ひとが集あつまる会かい。
  • ある組織そしきの全員ぜんいんが集あつまる会かい。

液体

  • 水みずや油あぶらのように、決きまった形かたちがなく、流ながれる性質せいしつのあるもの。

テロ

  • 英語えいごの「テロリズム」の略りゃく。政治的せいじてきな目的もくてきを成なしとげるためには、人ひとの命いのちをうばうような暴力ぼうりょくを使つかってもよいとする考かんがえ。また、そのような考かんがえで起おこす事件じけん。
