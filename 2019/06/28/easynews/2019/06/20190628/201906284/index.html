中国ちゅうごくの習しゅう主席しゅせき「来年らいねんの春はるに国くにのお客きゃくさまで日本にっぽんに来くる」

发表于 2019-06-28 更新于 2019-11-30 分类于 JapanEasyNews



２８日にちと２９日にち、大阪おおさかでＧ２０サミットがあります。サミットには国くにや国連こくれんなど３７の代表だいひょうが出席しゅっせきします。２７日にち、サミットに出席しゅっせきするため大阪おおさかに来きた中国ちゅうごくの習しゅう近平きんぺい国家主席こっかしゅせきと、安倍あべ総理大臣そうりだいじんが会あって話はなし合あいました。
安倍あべ総理大臣そうりだいじんは「日本にっぽんと中国ちゅうごくはいい関係かんけいに戻もどりました。今年ことし、日本にっぽんは令和れいわの時代じだいになって、中国ちゅうごくは国くにができてから７０年ねんになります。協力きょうりょくして新あたらしい時代じだいを作つくりましょう」と話はなしました。
習しゅう主席しゅせきは「中国ちゅうごくと日本にっぽんは新あたらしい歴史れきしのスタートラインに立たっています。一緒いっしょに新あたらしい時代じだいに合あった関係かんけいを作つくりましょう」と話はなしました。
２人ふたりは、政府せいふの高たかい地位ちいの人ひとたちが会あって、もっと話はなしをすることが大切たいせつだと考かんがえました。そして、習しゅう主席しゅせきが、来年らいねんの春はるに国くにのお客きゃくさまで日本にっぽんに来くることができるように、協力きょうりょくしていくことになりました。
