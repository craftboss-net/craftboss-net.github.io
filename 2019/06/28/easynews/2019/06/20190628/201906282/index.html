アメリカ国境こっきょうの川かわで亡なくなった父ちちと娘むすめの写真しゃしんがニュースになる

发表于 2019-06-28 更新于 2019-11-30 分类于 JapanEasyNews



アメリカとメキシコの国境こっきょうにある川かわで亡なくなった父ちちと１歳さい１１か月げつの娘むすめの写真しゃしんが、大おおきなニュースになっています。２人ふたりはメキシコの南みなみにあるエルサルバドルの人ひとです。川かわを渡わたってアメリカに向むかいましたが、溺おぼれて亡なくなりました。
写真しゃしんでは、２人ふたりが背中せなかを上うえにして、川かわの中なかで倒たおれています。娘むすめは手てを父ちちの首くびに伸のばしていて、最後さいごまで父ちちに強つよくつかまっていたように見みえます。
アメリカのニュースは、この写真しゃしんは、法律ほうりつを守まもらないでアメリカに入はいろうとする移民いみんの問題もんだいの大おおきさを表あらわしていると言いっています。
アメリカのトランプ大統領だいとうりょうは「写真しゃしんを見みたくなかった。きっとすばらしいお父とうさんだったと思おもう」と言いいました。
去年きょねん、２８３人にんがこの川かわを渡わたろうとして亡なくなったと言いわれています。インターネットではたくさんの人ひとが、移民いみんの問題もんだいと関係かんけいがある国くには、早はやく問題もんだいを解決かいけつしなければならないと言いっています。
