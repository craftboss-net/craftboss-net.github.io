サンフランシスコ市しが電子でんしたばこを売うることを禁止きんしする

发表于 2019-06-28 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのサンフランシスコ市しは、若わかい人ひとたちに人気にんきがある「電子でんしたばこ」を町まちの中なかで売うることを禁止きんしする規則きそくを作つくりました。電子でんしたばこは、たばこなどのにおいがする液体えきたいを電気でんきで温あたためて煙けむりを吸すいます。
市しは、インターネットで買かった電子でんしたばこを町まちの中なかに届とどけることも禁止きんしします。この規則きそくに違反いはんすると罰金ばっきんを払はらわなければなりません。アメリカの大おおきな町まちが電子でんしたばこを禁止きんしするのは初はじめてです。この規則きそくは７か月げつ後ごから始はじまる予定よていです。
サンフランシスコ市しの職員しょくいんは、子こどもたちをニコチンから守まもるためだと話はなしています。
サンフランシスコにある電子でんしたばこを作つくっている会社かいしゃの人ひとは「この規則きそくができると、大人おとなたちが昔むかしからあるたばこに戻もどってしまいます。昔むかしからあるたばこは、大勢おおぜいの人ひとが死しぬ原因げんいんになっているのに、禁止きんしされていません」と言いって、この規則きそくに反対はんたいしています。
