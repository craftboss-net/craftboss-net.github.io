１人ひとり６０億おく円えんで宇宙うちゅうステーションに行いくことができる

发表于 2019-06-11 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのＮＡＳＡは、早はやい場合ばあい来年らいねん、宇宙飛行士うちゅうひこうしではない人ひとでも国際こくさい宇宙うちゅうステーションへ旅行りょこうに行いくことができると発表はっぴょうしました。
旅行りょこうに行いく人ひとは、健康けんこうなことと、行いく前まえに訓練くんれんを受うけることが必要ひつようです。１年ねんに２人ふたりだけ、長ながい場合ばあい３０日にち宇宙うちゅうステーションにいることができます。
宇宙うちゅうステーションに泊とまるお金かねは、食事しょくじ代だいも入いれて１泊ぱく３８０万まん円えんで、３０日にちいる場合ばあいは１億おく円えん以上いじょう必要ひつようです。宇宙船うちゅうせんで行いくため６３億おく円えんぐらいのお金かねもかかります。
ＮＡＳＡは、５年ねんあとに宇宙飛行士うちゅうひこうしが月つきに下おりる計画けいかくを進すすめています。このため、旅行りょこうに行いく人ひとが払はらったお金かねを、この計画けいかくに使つかいたいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

宇宙ステーション

  • スペースシャトルなどで行いく宇宙うちゅう旅行りょこうや、宇宙うちゅうでの研究けんきゅうの基地きちとして、その中なかでたくさんの人ひとが生活せいかつを続つづけられる大おおきな人工衛星じんこうえいせい。

NASA

  • 宇宙開発うちゅうかいはつのために作つくられた、アメリカ政府せいふの機関きかん。スペースシャトルの開発かいはつなどをする。
