世界せかいで１億おく人にん以上いじょうの男性だんせいが１８歳さいになる前まえに結婚けっこんしている

发表于 2019-06-11 更新于 2019-11-30 分类于 JapanEasyNews



ユニセフは、１８歳さいになる前まえに結婚けっこんすることについて調しらべています。今いままでは女性じょせいについて調しらべていましたが、初はじめて男性だんせいについて調しらべて、どのくらいいるか計算けいさんしました。
計算けいさんによると、世界せかいの２０歳さいから２４歳さいの男性だんせいのうち、１億おく１５００万まん人にんが１８歳さいになる前まえに結婚けっこんしています。中央ちゅうおうアフリカやニカラグア、マダガスカルなどの国くにで多おおくなっています。２３００万まん人にんは１５歳さいになる前まえに結婚けっこんしています。
ユニセフは「結婚けっこんすると、家族かぞくのために学校がっこうをやめて働はたらかなければなりません。教育きょういくを受うけていないと、給料きゅうりょうが高たかい仕事しごとができなくて、貧まずしい生活せいかつになりやすいです」と言いっています。
