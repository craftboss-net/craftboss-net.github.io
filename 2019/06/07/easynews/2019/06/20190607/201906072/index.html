安全あんぜんに運転うんてんするために車くるまに付つける機械きかいがよく売うれている

发表于 2019-06-07 更新于 2019-11-30 分类于 JapanEasyNews



お年寄としよりが運転うんてんする車くるまの事故じこが続つづいています。車くるまと関係かんけいがある機械きかいなどを売うっている店みせでは、安全あんぜんに運転うんてんするために車くるまに付つける機械きかいがよく売うれています。
オートバックスセブンでは、ブレーキと間違まちがえてアクセルを強つよく踏ふんでも、車くるまが急きゅうに動うごかないようにする機械きかいが売うれています。先月せんげつは去年きょねんの５月がつの２６倍ばい売うれました。値段ねだんは３万まん円えんからです。
店みせは「安全あんぜんのための機械きかいを車くるまに付つけようと考かんがえる人ひとが増ふえています。店みせでもいろいろな機械きかいを用意よういしたいと思おもいます」と話はなしています。
