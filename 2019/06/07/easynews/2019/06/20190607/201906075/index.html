東海道新幹線とうかいどうしんかんせんの「Ｎ７００Ｓ」　時速じそく３６０ｋｍで走はしるテスト

发表于 2019-06-07 更新于 2019-11-30 分类于 JapanEasyNews



ＪＲ東海とうかいは来年らいねん７月がつから、東海道新幹線とうかいどうしんかんせんで「Ｎ７００Ｓ」という新あたらしい新幹線しんかんせんを使つかう予定よていです。
６日むいかの夜よる、新あたらしい新幹線しんかんせんが米原駅まいばらえきから京都駅きょうとえきまで走はしるテストが行おこなわれました。新あたらしい新幹線しんかんせんは、今いまと同おなじ時速じそく２８５ｋｍで走はしることになっています。しかし、テストでは時速じそく３６０ｋｍまでスピードを上あげて、気持きもちよく乗のることができるか、音おとがうるさくないかなどをチェックしました。
ＪＲ東海とうかいによると、新あたらしい新幹線しんかんせんは、走はしっているときに前まえから受うける風かぜの力ちからが今いままでより小ちいさくなっています。そして、ブレーキがよくなっていて、地震じしんなどでも安全あんぜんにスピードを下さげることができると言いっています。
ＪＲ東海とうかいは、新幹線しんかんせんをアメリカや台湾たいわんに輸出ゆしゅつするために、新あたらしい新幹線しんかんせんのよさをもっと知しってほしいと考かんがえています。
