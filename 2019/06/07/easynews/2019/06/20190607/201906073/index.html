エアビーアンドビー「空あき家やで民泊みんぱくの仕事しごとを大おおきくしたい」

发表于 2019-06-07 更新于 2019-11-30 分类于 JapanEasyNews



旅行りょこうに来きた人ひとなどにホテルの代かわりに家いえやマンションの部屋へやを貸かす「民泊みんぱく」の法律ほうりつができてから、今月こんげつ１５日にちで１年ねんになります。
民泊みんぱくを紹介しょうかいする世界せかいで最もっとも大おおきなウェブサイト「エアビーアンドビー」を作つくったネイサン?ブレチャージクさんが、日本にっぽんで民泊みんぱくの仕事しごとを大おおきくする計画けいかくについて話はなしました。
ブレチャージクさんは「来年らいねん、東京とうきょうでオリンピックとパラリンピックがありますが、ホテルだけでは泊とまる場所ばしょが足たりません。日本にっぽんには誰だれも住すんでいない空あき家やが８５０万まん軒けんもあります。これを使つかいたいと考かんがえています」と話はなしました。
法律ほうりつができたばかりのころ、エアビーアンドビーのウェブサイトで紹介しょうかいしている家いえの中なかには、役所やくしょに書類しょるいを出だしていないところがありました。ブレチャージクさんは「役所やくしょに出だした書類しょるいを会社かいしゃにも出ださせて、法律ほうりつを守まもるように気きをつけています」と話はなしました。
