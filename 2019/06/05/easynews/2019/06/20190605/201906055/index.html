８１歳さいが運転うんてんする車くるまがすごいスピードで交差点こうさてんに入はいって事故じこ

发表于 2019-06-05 更新于 2019-11-30 分类于 JapanEasyNews



福岡市ふくおかしで４日よっかの夜よる、８１歳さいの男性だんせいが運転うんてんする車くるまがすごいスピードで走はしりながら、交差点こうさてんに入はいりました。そして、２台だいの車くるまとぶつかって、人ひとが歩あるいている道みちに突つっ込こみました。この事故じこで男性だんせいと車くるまに乗のっていた妻つまの２人ふたりが亡なくなって、歩あるいていた人ひとなど７人にんがけがをしました。
警察けいさつによると、男性だんせいの車くるまは交差点こうさてんに入はいる６００ｍぐらい前まえで、前まえにいた車くるまにぶつかりました。そのあと、反対はんたい側がわの道みちに入はいって、ほかの車くるまにぶつかりながら、すごいスピードのまま交差点こうさてんに入はいっていきました。
警察けいさつは、事故じこの原因げんいんを調しらべています。
