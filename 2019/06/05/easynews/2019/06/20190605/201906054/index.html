将棋しょうぎの羽生はぶさんが新あたらしい記録きろくを作つくる　勝かった試合しあいが最もっとも多おおい

发表于 2019-06-05 更新于 2019-11-30 分类于 JapanEasyNews



将棋しょうぎのプロで４８歳さいの羽生はぶ善治よしはるさんは先月せんげつ、プロになってから勝かった試合しあいが１４３３回かいになりました。これは、大山おおやま康晴やすはるさんが１９９２年ねんに６９歳さいで作つくった記録きろくと同おなじで、プロの中なかで最もっとも多おおい数かずです。
羽生はぶさんは４日よっか、大おおきな大会たいかいの１つ「王位戦おういせん」の試合しあいをしました。試合しあいは午前ごぜん１０時じに始はじまりました。羽生はぶさんがうまく攻せめて、午後ごご８時じ前まえに勝かちました。
羽生はぶさんが勝かった試合しあいは１４３４回かいになりました。羽生はぶさんは勝かった試合しあいがプロの中なかで最もっとも多おおい新あたらしい記録きろくを作つくりました。
