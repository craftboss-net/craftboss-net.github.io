３万まん年ねん前まえのライオンがロシアの凍こおった土つちから見みつかる

发表于 2019-06-05 更新于 2019-11-30 分类于 JapanEasyNews



ロシアと日本にっぽんの大学だいがくなどのチームは去年きょねん７月がつ、シベリアの凍こおった土つちの中なかから「ホラアナライオン」の赤あかちゃんを見みつけました。ホラアナライオンは、１万まん年ねんぐらい前まえにいなくなったと言いわれる動物どうぶつです。
見みつかったホラアナライオンは、体からだの長ながさが４０ｃｍ、重おもさが８００ｇぐらいです。生うまれたばかりの雌めすの赤あかちゃんで、３万まん年ねんぐらい前まえに生いきていたと考かんがえられています。このホラアナライオンは、前まえに見みつけた３つよりきれいで、毛けや皮かわ、脳のう、心臓しんぞうなども残のこっていました。
近ちかくの場所ばしょでは、３万まん年ねんぐらい前まえのオオカミの頭あたまも見みつかりました。このオオカミも、骨ほねや筋肉きんにくだけではなくて目めや舌したなどもきれいに残のこっていました。
チームの人ひとは、昔むかしの動物どうぶつについて詳くわしいことがわかるかもしれないと考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

凍る

  • 寒さむさのために、水みずなどの液体えきたいが固かたまる。氷こおりになる。

土

  • 岩いわや石いしがくだけて、粉こなになったもの。どろ。
  • 陸地りくちの表面ひょうめん。地面じめん。

チーム

  • 競技きょうぎや仕事しごとをするときの、組くみや団体だんたい。

シベリア

  • アジアの北部ほくぶにあるロシアの領土りょうど。北きたは北極海ほっきょくかいに面めんし、冬ふゆの寒さむさが厳きびしい。

雌

  • 動物どうぶつで、子こや卵たまごを生うむ能力のうりょくがあるほう。

皮

  • 動物どうぶつや植物しょくぶつなどの外側そとがわを包つつんでいるもの。
  • 中身なかみをおおって包つつんでいるもの。

脳

  • 頭あたまの中なかにあって、考かんがえたり体からだを動うごかしたりするはたらきを受うけ持もつところ。
  • 頭あたまのはたらき。
  • 中心ちゅうしんとなる人ひと。

心臓

  • 胸むねの左側ひだりがわにあって、血液けつえきを体からだじゅうに送おくり出だすポンプの役目やくめをする器官きかん。こぶしぐらいの大おおきさで、左右さゆうの心房しんぼうと左右さゆうの心室しんしつの四よっつの部分ぶぶんからできている。
  • ものごとの大切たいせつな部分ぶぶん。
  • 度胸どきょうがよくて、物ものおじしないこと。

狼

  • ユーラシアや北きたアメリカ大陸たいりくの原野げんやにすむ動物どうぶつ。形かたちは犬いぬのシェパードに似にているが、性質せいしつがあらく、家畜かちくや人ひとをおそうこともある。かつては日本にっぽんにもすんでいた。

骨

  • 動物どうぶつの体からだの中なかにあって、体からだを支ささえているかたいもの。
  • 物もののしんにする細長ほそながい竹たけや金属きんぞく。
  • ものごとの中心ちゅうしん。
  • へこたれない強つよい性質せいしつ。
  • 苦労くろうなこと。

筋肉

  • 動物どうぶつの体からだを動うごかすはたらきをする、細ほそい筋すじが集あつまってできているもの。骨ほねにつながっているものと、内臓ないぞうのかべを作つくっているものとがある。

舌

  • 口くちの中なかにあって、物ものを飲のみこんだり、味あじを感かんじたり、発音はつおんを助たすけたりするはたらきをするもの。

詳しい

  • 細こまかい。
  • 細こまかいことまでよく知しっている。
