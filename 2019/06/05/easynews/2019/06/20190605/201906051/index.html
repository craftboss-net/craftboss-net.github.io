プラスチックの袋ふくろを無料むりょうで客きゃくに渡わたしてはいけない法律ほうりつを作つくる

发表于 2019-06-05 更新于 2019-11-30 分类于 JapanEasyNews



政府せいふは先月せんげつ、海うみが汚よごれないようにプラスチックのごみを少すくなくするための計画けいかくを決きめました。計画けいかくでは、スーパーやコンビニなどがレジで客きゃくに渡わたしているプラスチックの袋ふくろを無料むりょうにしないことにしました。
環境省かんきょうしょうの原田はらだ大臣だいじんは３日みっか、プラスチックの袋ふくろを用意よういしている店みせは全部ぜんぶ、無料むりょうで客きゃくに袋ふくろを渡わたしてはいけないという法律ほうりつを作つくると言いいました。原田はらだ大臣だいじんは「店みせが袋ふくろを売うったときは、そのお金かねを環境かんきょうを大切たいせつにするために使つかうのがいいと思おもいます」と言いっています。
環境省かんきょうしょうは法律ほうりつを作つくるため、これから話はなし合あいを始はじめます。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

プラスチック

  • 熱ねつや圧力あつりょくを加くわえて、自由じゆうに形かたちを作つくることができる物質ぶっしつ。特とくに合成樹脂ごうせいじゅしを指さす。使つかい道みちが広ひろい。

袋

  • 紙かみや布ぬのなどで作つくり、中なかに物ものを入いれて、口くちをしめるようにした入いれ物もの。
  • 「1)」に似にた形かたちのもの。

無料

  • 料金りょうきんのいらないこと。ただ。

政府

  • 国くにの政治せいじを行おこなうところ。
  • 内閣ないかく。

コンビニ

  • 食料品しょくりょうひんや日用品にちようひんが、手軽てがるにいつでも買かえるように開ひらいている、小型こがたのスーパー。

環境省

  • 公害こうがいをなくし、自然しぜんを守まもる仕事しごとをする国くにの役所やくしょ。

大臣

  • 内閣ないかくをつくり、国くにの政治せいじで、もっとも責任せきにんのある人ひと。総理大臣そうりだいじんと国務大臣こくむだいじんとがある。

環境

  • 人ひとや生いき物ものを取とり巻まき、影響えいきょうをあたえる周まわりの世界せかい。

話し合い

  • 話はなし合あうこと。相談そうだん。
