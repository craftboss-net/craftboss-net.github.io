自動じどうで走はしる電車でんしゃが反対はんたいに向むかって走はしって１４人にんがけがをする

发表于 2019-06-03 更新于 2019-11-30 分类于 JapanEasyNews



神奈川県かながわけん横浜市よこはましで６月がつ１日ついたちの夜よる、駅えきを出発しゅっぱつした「シーサイドライン」という電車でんしゃが、反対はんたいに向むかって走はしり出だしました。そして、２５ｍぐらい走はしって、電車でんしゃを止とめるための壁かべにぶつかりました。この事故じこで、電車でんしゃに乗のっていた３０人にんのうち１４人にんがけがをしました。
シーサイドラインは、人ひとが運転うんてんしないでコンピューターを使つかって自動じどうで走はしっています。
国土交通省こくどこうつうしょうは、自動じどうで運転うんてんするための機械きかいなどをチェックして、事故じこの原因げんいんを詳くわしく調しらべることにしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

自動

  • 機械きかいなどが自分じぶんの力ちからで動うごくこと。

ぶつかる

  • 打うち当あたる。つき当あたる。
  • 困難こんなんなことに出であう。
  • 争あらそう。立たち向むかう。
  • 重かさなる。

国土交通省

  • 川かわ・道路どうろ・建物たてものなどに関かんする仕事しごとや、交通こうつう・荷物にもつの運送うんそうなどに関かんする仕事しごとをする国くにの役所やくしょ。国交省こっこうしょう。

詳しい

  • 細こまかい。
  • 細こまかいことまでよく知しっている。
