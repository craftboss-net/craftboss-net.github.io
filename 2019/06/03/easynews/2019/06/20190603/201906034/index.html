東京とうきょうオリンピックの聖火せいかリレー　８５７の市しや町まちを走はしる

发表于 2019-06-03 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックの聖火せいかリレーで、８５７の市しや町まちを走はしることが決きまりました。日本にっぽんにある市しや町まちの約やく半分はんぶんです。
岩手県いわてけん陸前高田市りくぜんたかたしにある「奇跡きせきの一本松いっぽんまつ」や熊本県くまもとけんにある「熊本城くまもとじょう」も選えらばれました。「奇跡きせきの一本松いっぽんまつ」は、２０１１年ねんの大おおきな地震じしんで津波つなみが来きたとき、流ながされないで１本ぽんだけ残のこった木きです。「熊本城くまもとじょう」は、２０１６年ねんの大おおきな地震じしんで壊こわれましたが、今いま直なおしています。このように災害さいがいのあと、町まちを元気げんきにするために頑張がんばっている所ところを回まわります。
「富士山ふじさん」や広島県ひろしまけんの「厳島神社いつくしまじんじゃ」、兵庫県ひょうごけんの「姫路城ひめじじょう」や山口県やまぐちけんの「錦帯橋きんたいきょう」など有名ゆうめいな所ところも回まわります。１９６４年ねんの東京とうきょうオリンピックや、１９９８年ねんの長野ながのオリンピックで日本にっぽんの選手せんしゅが活躍かつやくした会場かいじょうも回まわります。
聖火せいかリレーは、来年らいねん３月がつ２６日にちに福島県ふくしまけんを出発しゅっぱつして、１２１日にちで８５７の市しや町まちを回まわります。

オリンピック

  • ４年よねんごとに開ひらかれ、世界せかいじゅうの国々くにぐにから選手せんしゅが参加さんかする競技大会きょうぎたいかい。古代こだいギリシャのオリンピアで開ひらかれた古代こだいオリンピックにならって、フランスのクーベルタンの力ちからで、１８９６年ねんにギリシャのアテネで開ひらかれたのが、近代きんだいオリンピックの始はじまり。五輪ごりん。

聖火

  • 神かみにささげる火ひ。特とくに、オリンピック大会たいかいの期間中きかんちゅう、燃もやし続つづける火ひ。ギリシャのオリンピアで太陽たいようから点火てんかされ、会場かいじょうまで聖火せいかリレーで運はこばれる。

リレー

  • 順々じゅんじゅんに伝つたえわたしていくこと。
  • 陸上競技りくじょうきょうぎ・水泳すいえい競技きょうぎで、ひと組くみの選手せんしゅが、つぎつぎに決きめられた距離きょりを受うけついで、速はやさを争あらそうもの。

約

  • ちかう。取とり決きめる。
  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。

津波

  • 地震じしんなどのために、大おおきな波なみが急きゅうに海岸かいがんにおし寄よせてくること。

流す

  • 液体えきたいなどが低ひくいほうへ動うごくようにする。
  • 水みずなどにうかべて動うごかす。
  • よごれを洗あらい落おとす。
  • うわさなどを広ひろめる。
  • 予定よていしていたことを、取とりやめる。
  • そのことを気きに留とめない。

災害

  • 台風たいふう・地震じしん・大水おおみずなどによる災難さいなん。

富士山

  • 静岡県しずおかけんと山梨県やまなしけんの境さかいにある、日本一にっぽんいち高たかい山やま。高たかさは３７７６メートル。江戸時代えどじだいに大おおきな噴火ふんかがあった。

選手

  • 競技きょうぎに出でるために選えらばれた人ひと。

活躍

  • めざましく働はたらくこと。
