駐車場ちゅうしゃじょうの建物たてものの６階かいから車くるまが落おちそうになる

发表于 2019-06-03 更新于 2019-11-30 分类于 JapanEasyNews



５月がつ３１日にち、福岡市ふくおかしの町まちの中なかで、車くるまが駐車場ちゅうしゃじょうの建物たてものの６階かいから落おちそうになりました。この車くるまを運転うんてんしていた女性じょせいは「アクセルとブレーキを間違まちがえて踏ふんでしまった」と話はなしています。車くるまは後うしろの半分はんぶんが建物たてものの外そとに出でました。けがをした人ひとはいませんでした。
近ちかくにいた男性だんせいは「大おおきな音おとが聞きこえたので見みたら、車くるまが落おちそうになっていました」と話はなしていました。

会社かいしゃやグループなどの名前なまえ

  • 英語えいごの「アクセレレーター」の略りゃく。自動車じどうしゃで、足あしでふんで速はやさをかげんする仕しかけ。

ブレーキ

  • 車くるまを止とめる仕しかけ。
  • ものごとの進すすむのをおさえたり、止とめたりすること。
