珍めずらしい猿さるをスーツケースに隠かくして運はこんで逮捕たいほされる

发表于 2019-06-12 更新于 2019-11-30 分类于 JapanEasyNews



世界せかいでも数かずが少すくない動物どうぶつは輸入ゆにゅうや輸出ゆしゅつをしてはいけないことになっています。しかし、最近さいきんかばんなどに動物どうぶつを隠かくして、飛行機ひこうきでタイから日本にっぽんに運はこんでくる人ひとたちがいます。
珍めずらしい猿さるや亀かめなど１７匹ひきを去年きょねん９月がつにタイから成田空港なりたくうこうに運はこんできたため、東京とうきょうに住すんでいる男おとこが逮捕たいほされました。動物どうぶつは男おとこのスーツケースの中なかに入はいっていました。
男おとこは「１０万まん円えんもらって、日本にっぽんに運はこぶように言いわれた」と話はなしています。
専門家せんもんかは「見みつかった動物どうぶつのうち２匹ひきの猿さるは、アフリカなどにいる猿さるの仲間なかまで、日本にっぽんには来きたことがない珍めずらしい猿さるのようです」と言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

猿

  • 森林しんりんにすみ、木登きのぼりがうまく、群むれを作つくって暮くらしている動物どうぶつ。チンパンジー・ゴリラ・オランウータン・テナガザルなど種類しゅるいが多おおい。日本にっぽんにはニホンザルがいる。

隠す

  • 人ひとに見みえないようにする。
  • 人ひとに知しられないようにする。秘密ひみつにする。

逮捕

  • 罪つみを犯おかした疑うたがいのある人ひとを、警察けいさつがつかまえること。

数

  • 一ひとつ、二ふたつ、三みっつなどと数かぞえた物ものの数量すうりょう。すう。
  • いくつもあること。たくさん。
  • 数かぞえあげる値打ねうちのあるもの。なかま。

亀

  • 水中すいちゅうや陸上りくじょうで生活せいかつし、体からだがかたいこうらでおおわれた動物どうぶつ。池いけや川かわにすむイシガメ・スッポン、陸りくにすむゾウガメ、海うみにすむアカウミガメなど種類しゅるいが多おおい。ほとんどの種類しゅるいが首くびや尾お、足あしをこうらの中なかに引ひっこめることができる。

仲間

  • いっしょにものごとをする人ひと。友達ともだち。グループ。
  • 種類しゅるいの同おなじもの。
