「子こども食堂しょくどう」が増ふえている

发表于 2019-06-27 更新于 2019-11-30 分类于 JapanEasyNews



「子こども食堂しょくどう」は、親おやが働はたらいていて１人ひとりで食事しょくじをしなければならない子こどもなどが、無料むりょうか少すくないお金かねで食事しょくじができる所ところです。
子こども食堂しょくどうを応援おうえんしているＮＰＯによると、子こども食堂しょくどうは今年ことし５月がつに３７１８ありました。去年きょねん４月がつより１４００以上いじょう増ふえて、１．６倍ばいになりました。子こども食堂しょくどうがいちばん多おおい所ところは東京都とうきょうとで、４８８ありました。人口じんこうが多おおい所ところで多おおくなっています。少すくない所ところは１１ある秋田県あきたけんなどでした。
このＮＰＯは、小学校しょうがっこうがある所ところに子こども食堂しょくどうも１つあるのがいちばんいいと考かんがえています。沖縄県おきなわけんにある子こども食堂しょくどうの数かずは小学校しょうがっこうの数かずの６０．５％で、日本にっぽんでいちばん高たかい割合わりあいになっています。いちばん低ひくい所ところは秋田県あきたけんで５．５％でした。
ＮＰＯは「子こども食堂しょくどうは、お金かねに困こまっている家庭かていの子こどものためだけではなくて、子こどもからお年寄としよりまでいろいろな人ひとが交流こうりゅうできる所ところです。このことをみなさんに知しってほしいです」と話はなしています。

国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

  • いくつもあること。たくさん。
