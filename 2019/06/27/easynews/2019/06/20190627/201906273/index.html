障害しょうがいがある人ひと　去年きょねんは８２万まん人にん以上いじょうが会社かいしゃで働はたらいた

发表于 2019-06-27 更新于 2019-11-30 分类于 JapanEasyNews



厚生労働省こうせいろうどうしょうは５年ねんに１回かい、日本にっぽんの会社かいしゃで体からだや心こころに障害しょうがいがある人ひとが何なん人にん働はたらいているか計算けいさんしています。去年きょねん働はたらいた人ひとは８２万まん１０００人にんになりました。５年ねん前まえより１９万まん人にん増ふえて、今いままででいちばん多おおくなりました。
働はたらいている人ひとのうち、体からだに障害しょうがいがある人ひとの５０％以上いじょうが会社かいしゃの社員しゃいんになっています。しかし、心こころや知能ちのうに障害しょうがいがある人ひとで社員しゃいんになっている人ひとは２０％ぐらいです。
厚生労働省こうせいろうどうしょうは「会社かいしゃの中なかの障害しょうがいがある人ひとの割合わりあいは法律ほうりつで決きまっていて、去年きょねん上あがりました。障害しょうがいがある人ひとに働はたらいてほしいと考かんがえる会社かいしゃも増ふえています」と言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

障害

  • 何なにかをするときに、じゃまになるものごと。さまたげ。
  • 体からだのはたらきに故障こしょうがあること。

厚生労働省

  • 国民こくみんの健康けんこうや生活せいかつを守まもる仕事しごとや、労働者ろうどうしゃが仕事しごとを見みつけるのを助たすけたり、労働者ろうどうしゃを保護ほごしたりする国くにの役所やくしょ。厚労省こうろうしょう。

計算

  • 数かずや量りょうを数かぞえること。
  • 式しきを解といて、答こたえを出だすこと。
  • 考かんがえに入いれておくこと。

社員

  • その会社かいしゃに勤つとめている人ひと。会社員かいしゃいん。

知能

  • ものごとを考かんがえたり、理解りかいしたりするはたらき。

割合

  • 物ものと物ものとの関係かんけいを、数かずで表あらわしたもの。割わり。率りつ。歩合ぶあい。
  • 思おもいのほか。比較的ひかくてき。
