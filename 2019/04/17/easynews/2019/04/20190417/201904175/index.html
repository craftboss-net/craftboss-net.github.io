フランスの大統領だいとうりょう「ノートルダム大聖堂だいせいどうを５年ねん以内いないに直なおす」

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



フランスのノートルダム大聖堂だいせいどうは、歴史れきしがある建物たてもので、世界遺産せかいいさんにもなっています。しかし、日本にっぽんの時間じかんの１６日にちに大おおきな火事かじがあって、塔とうや屋根やねが焼やけて落おちてしまいました。
マクロン大統領だいとうりょうは１７日にち、テレビでフランスの人ひとたちに「大聖堂だいせいどうを直なおして、５年ねん以内いないに火事かじの前まえより美うつくしくするつもりです」と話はなしました。そして、みんなに協力きょうりょくしてほしいと言いいました。
フランスでは会社かいしゃの社長しゃちょうや市しなどがお金かねを出だすと言いっています。大聖堂だいせいどうを直なおすためのお金かねは、１７日にちの昼ひるまでに９００億おく円えんぐらい集あつまっています。

世界遺産

  • ユネスコの「世界せかいの文化遺産ぶんかいさん及および自然遺産しぜんいさんの保護ほごに関かんする条約じょうやく」（「世界遺産保護条約せかいいさんほごじょうやく」）にもとづいて決きめられた、世界的せかいてきに残のこす価値かちがあると認みとめられた文化ぶんかや自然しぜん。日本にっぽんでは、文化遺産ぶんかいさんとして姫路城ひめじじょうや法隆寺ほうりゅうじ・沖縄おきなわの首里城しゅりじょうなど、自然遺産しぜんいさんとして屋久島やくしまや白神山地しらかみさんちなどが指定していされている。
