オリンピックの仕事しごとをしている桜田さくらだ大臣だいじんがやめる

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



オリンピックとパラリンピックの仕事しごとをしている桜田さくらだ大臣だいじんは１０日とおか、岩手県いわてけんの高橋たかはし議員ぎいんのパーティーに出席しゅっせきしました。岩手県いわてけんは２０１１年ねんに地震じしんと津波つなみで大おおきな被害ひがいを受うけました。このパーティーで桜田さくらだ大臣だいじんは、被害ひがいを受うけた町まちを元気げんきにすることより大事だいじなのが高橋たかはし議員ぎいんだと言いいました。
パーティーのあと桜田さくらだ大臣だいじんは「被害ひがいを受うけた人ひとたちの気持きもちを傷きずつけるようなことを言いって、とても申もうし訳わけないと思おもっています」と言いって、大臣だいじんをやめました。
安倍あべ総理大臣そうりだいじんは「これからも私わたしたちは被害ひがいを受うけた人ひとたちの気持きもちを大切たいせつにして、町まちを元気げんきにするために頑張がんばります」と言いって、謝あやまりました。
野党やとうは、大臣だいじんを選えらんだ安倍あべ総理大臣そうりだいじんにも責任せきにんがあると言いっています。今月こんげつ５日いつかにも、自分じぶんが話はなしたことが原因げんいんで副ふく大臣だいじんがやめたばかりでした。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

オリンピック

  • ４年よねんごとに開ひらかれ、世界せかいじゅうの国々くにぐにから選手せんしゅが参加さんかする競技大会きょうぎたいかい。古代こだいギリシャのオリンピアで開ひらかれた古代こだいオリンピックにならって、フランスのクーベルタンの力ちからで、１８９６年ねんにギリシャのアテネで開ひらかれたのが、近代きんだいオリンピックの始はじまり。五輪ごりん。

大臣

  • 内閣ないかくをつくり、国くにの政治せいじで、もっとも責任せきにんのある人ひと。総理大臣そうりだいじんと国務大臣こくむだいじんとがある。

辞める

  • 勤つとめや役目やくめから退しりぞく。

パラリンピック

  • 身体障害者しんたいしょうがいしゃの国際こくさいスポーツ大会たいかい。４年よねんに一度いちど、オリンピック開催地かいさいちで行おこなわれる。

議員

  • 選挙せんきょで選えらばれ、国会こっかいや地方ちほうの議会ぎかいで、政治せいじに関かんすることをいろいろ相談そうだんする人ひと。国会議員こっかいぎいん・県議会議員けんぎかいぎいんなど。

津波

  • 地震じしんなどのために、大おおきな波なみが急きゅうに海岸かいがんにおし寄よせてくること。

被害

  • 損害そんがいを受うけること。また、受うけた害がい。

傷つける

  • （相手あいてに）けがをさせる。
  • （物ものに）傷きずをつける。
  • 人ひとのめいよや、気持きもちに傷きずをつける。

申し訳ない

  • 言いい訳わけができない。大変たいへんすまない。

辞める

  • 勤つとめや役目やくめから退しりぞく。

総理大臣

  • 内閣総理大臣ないかくそうりだいじんのこと。内閣ないかくの最高責任者さいこうせきにんしゃで、国会議員こっかいぎいんの中なかから議員ぎいんが選えらび、天皇てんのうが認みとめて決きまる。首相しゅしょう。総理そうり。

野党

  • 内閣ないかくを作つくっていない政党せいとう。

責任

  • 自分じぶんが引ひき受うけてしなければならない務つとめ。

副

  • おもなものの次つぎの。ひかえの。
  • そえるもの。つきもの。
