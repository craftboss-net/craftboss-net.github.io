働はたらく外国人がいこくじんの新あたらしいビザ　国くにが考かんがえたより試験しけんを受うける人ひとが多おおい

发表于 2019-04-08 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんで働はたらく外国人がいこくじんを増ふやすために、４月がつから１４の仕事しごとについて「特定技能とくていぎのう」というビザができました。このビザは、日本語にほんごの試験しけんとそれぞれの仕事しごとの試験しけんに合格ごうかくした外国人がいこくじんに出だします。
１４の仕事しごとのうち、レストランなどで働はたらく人ひとの試験しけんは、２５日にちにあります。試験しけんの申もうし込こみが始はじまった日ひ、会場かいじょうに入はいる人数にんずうの３倍ばいの約やく１０００人にんが申もうし込こみました。このため、国くには２６日にちと６月がつにも試験しけんをすることを急いそいで決きめました。
ホテルなどで働はたらく人ひとの試験しけんは、１４日じゅうよっかにあります。約やく７００人にんが受うける予定よていで、７つの会場かいじょうはほとんどいっぱいになります。国くには試験しけんの回数かいすうを増ふやすことも考かんがえています。
