パリのノートルダム大聖堂だいせいどうで火事かじ　高たかい塔とうが燃もえる

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



フランスのパリで１５日にち午後ごご７時じごろ、ノートルダム大聖堂だいせいどうが火事かじになりました。建物たてものの真まん中なかの部分ぶぶんがひどく燃もえて、高たかさ９０ｍの細ほそい塔とうや、建物たてものの屋根やねの半分はんぶん以上いじょうが落おちました。１０時間じかんぐらいが過すぎたころ、火ひは大体だいたい消きえました。
火事かじになったとき大聖堂だいせいどうは閉しまっていたため、観光かんこうに来きた人ひとは中なかにいなかったようです。
大聖堂だいせいどうでは、去年きょねん４月がつから建物たてものを直なおす工事こうじをしていました。消防しょうぼうは、大聖堂だいせいどうの屋根やねの裏うらから火ひが出でたと考かんがえていますが、原因げんいんはまだわかっていません。
近ちかくの店みせにいた女性じょせいは「歴史れきしがある建物たてものなので、私わたしたちはショックを受うけています」と話はなしていました。
ノートルダム大聖堂だいせいどうには、１年ねんに１２００万まん人にんが世界中せかいじゅうから観光かんこうに来きます。１９９１年ねんにはユネスコの世界遺産せかいいさんになりました。

UNESCO

  • 国連こくれんの機関きかんの一ひとつで、教育きょういくや科学かがく、また文化ぶんかを通つうじて、たがいに理解りかいし合あい、世界せかいの平和へいわと安全あんぜんを守まもることを目的もくてきとしている。

世界遺産

  • ユネスコの「世界せかいの文化遺産ぶんかいさん及および自然遺産しぜんいさんの保護ほごに関かんする条約じょうやく」（「世界遺産保護条約せかいいさんほごじょうやく」）にもとづいて決きめられた、世界的せかいてきに残のこす価値かちがあると認みとめられた文化ぶんかや自然しぜん。日本にっぽんでは、文化遺産ぶんかいさんとして姫路城ひめじじょうや法隆寺ほうりゅうじ・沖縄おきなわの首里城しゅりじょうなど、自然遺産しぜんいさんとして屋久島やくしまや白神山地しらかみさんちなどが指定していされている。
