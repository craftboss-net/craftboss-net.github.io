宮城県みやぎけん柴田町しばたまち　花見はなみに来きた外国人がいこくじんに子こどもたちが英語えいごで案内あんない

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



宮城県みやぎけんの柴田町しばたまちには、たくさんの桜さくらの木きが並ならんでいる「一目ひとめ千本桜せんぼんざくら」という有名ゆうめいな場所ばしょがあります。毎年まいとし２５万まん人にんが花見はなみに来きて、最近さいきんはタイや香港ほんこんなどから来くる人ひとも増ふえています。
１４日じゅうよっか、柴田町しばたまちの小学生しょうがくせいや中学生ちゅうがくせい３８人にんが、駅えきの前まえなど６つの場所ばしょで観光かんこうに来きた外国人がいこくじんを迎むかえました。
この子こどもたちは、学校がっこうで授業じゅぎょうが終おわってから英語えいごの会話かいわを勉強べんきょうしています。子こどもたちは笑顔えがおで、英語えいごや中国語ちゅうごくごで書かいてある地図ちずを渡わたしたり、英語えいごで「写真しゃしんを撮とりましょうか」と話はなしたりして案内あんないしました。
案内あんないした女おんなの子こは「英語えいごを習ならっていますが、外国がいこくの人ひとが何なにを話はなしているかよくわかりませんでした。これからも外国がいこくの人ひとに親切しんせつにして、柴田町しばたまちのきれいな桜さくらを知しってもらいたいです」と話はなしていました。
