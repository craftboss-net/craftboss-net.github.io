京都市きょうとしのお寺てらなどで観光かんこうに来きた人ひとが猿さるにかまれる

发表于 2019-04-05 更新于 2019-11-30 分类于 JapanEasyNews



京都市きょうとしでは、観光かんこうに来きた人ひとが、山やまから下おりて来きた猿さるにかまれてけがをする被害ひがいが続つづいています。
知恩院ちおんいんというお寺てらでは３日みっか、女性じょせいが足あしをかまれて病院びょういんに運はこばれました。女性じょせいが持もっていた傘かさの近ちかくに子こどもの猿さるが来きたあと、突然とつぜん、親おやだと考かんがえられる大おおきな猿さるが来きて足あしをかみました。
このお寺てらでは、先週せんしゅうの終おわりから２日ふつかまでに、外国人がいこくじんなど３人にんが猿さるに足あしをかまれました。このお寺てらの近ちかくでも先月せんげつ、女性じょせいが足あしをかまれました。
京都市きょうとしは「猿さるの近ちかくに行いって食たべ物ものをあげたり、写真しゃしんを撮とったりすることは危険きけんです。猿さるがいたら、目めを見みないで猿さるから遠とおくに離はなれてください」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

観光

  • 景色けしきのよい所ところや名所めいしょなどを見物けんぶつして回まわること。

猿

  • 森林しんりんにすみ、木登きのぼりがうまく、群むれを作つくって暮くらしている動物どうぶつ。チンパンジー・ゴリラ・オランウータン・テナガザルなど種類しゅるいが多おおい。日本にっぽんにはニホンザルがいる。

被害

  • 損害そんがいを受うけること。また、受うけた害がい。

突然

  • 急きゅうに。思いがけなく。

親

  • 父ちちや母はは。両親りょうしん。
  • 子こや卵たまごをうんだもの。
  • トランプやかるたなどの遊あそびで、カードを配くばる人ひと。
  • 中心ちゅうしんとなるもの。大おおきいもの。

離れる

  • くっついていたものが分わかれる。
  • 距離きょりがある。
  • 間あいだがあく。
  • やめる。
