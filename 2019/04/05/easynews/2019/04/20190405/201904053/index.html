アメリカの通信つうしん会社がいしゃ「世界せかいで初はじめて５Ｇを始はじめた」と発表はっぴょう

发表于 2019-04-05 更新于 2019-11-30 分类于 JapanEasyNews



アメリカの通信つうしん会社がいしゃのベライゾン?コミュニケーションズは３日みっか、スマートフォンのための５Ｇのサービスを始はじめたと発表はっぴょうしました。会社かいしゃは、世界せかいで初はじめてだと言いっています。５Ｇは、今いままでより速はやく大おおきなデータを送おくることができるサービスです。
会社かいしゃは、シカゴとミネアポリスの２つの市しでこのサービスを始はじめました。今いま使つかうことができるのは、モトローラが作つくった１種類しゅるいのスマートフォンだけです。
この会社かいしゃは最初さいしょ、今月こんげつ１１日にちにサービスを始はじめる予定よていでした。しかし、韓国かんこくの会社かいしゃが今週こんしゅう、５Ｇのサービスを５日いつかに始はじめると発表はっぴょうしたため、予定よていより早はやくしたようです。
ベライゾンのトップは、今年中ことしじゅうにアメリカの３０の市しでサービスを始はじめる計画けいかくだと言いっています。
日本にっぽんでも早はやい場合ばあいは今年ことしの夏なつごろ、一部いちぶの場所ばしょで５Ｇの試験しけんが始はじまる予定よていです。
