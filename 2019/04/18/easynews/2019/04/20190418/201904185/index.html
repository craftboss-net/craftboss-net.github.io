天皇てんのう陛下へいかと皇后こうごうさまが伊勢神宮いせじんぐうに参拝さんぱいする

发表于 2019-04-18 更新于 2019-11-30 分类于 JapanEasyNews



天皇てんのう陛下へいかは今月こんげつ３０日にちに退位たいいして、５月がつ１日ついたちに皇太子こうたいしさまが新あたらしい天皇てんのうになります。天皇てんのう陛下へいかと皇后こうごうさまは１８日にち、三重県みえけんにある伊勢神宮いせじんぐうに参拝さんぱいしました。伊勢神宮いせじんぐうは皇室こうしつの祖先そせんをまつる神社じんじゃで、皇室こうしつと深ふかい関係かんけいがあります。
天皇てんのう陛下へいかが退位たいいする前まえに東京とうきょう以外いがいの地方ちほうに行いくのは、これが最後さいごです。神社じんじゃへ向むかう道みちでは、朝あさ早はやくから大勢おおぜいの人ひとが待まっていました。天皇てんのう陛下へいかと皇后こうごうさまの車くるまがゆっくりと進すすむと、大勢おおぜいの人ひとが手てや旗はたを振ふってお２人ふたりを迎むかえました。
大阪おおさかから来きていた女性じょせいは「大おおきな地震じしんがあったときにも来きていただきました。国民こくみんを大切たいせつにしていると感かんじます。本当ほんとうにお疲つかれさまでしたと伝つたえたいです」と話はなしていました。
