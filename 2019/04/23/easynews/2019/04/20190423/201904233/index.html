札幌さっぽろ　家いえが集あつまる場所ばしょで「熊くまを見みた」

发表于 2019-04-23 更新于 2019-11-30 分类于 JapanEasyNews



札幌市さっぽろし清田きよた区くで２１日にち午後ごご１１時じごろ、家いえが集あつまる場所ばしょの道みちやコンビニの駐車場ちゅうしゃじょうで熊くまを見みたという連絡れんらくが警察けいさつにありました。
１時間じかんぐらいあとには、１ｋｍぐらい離はなれた北広島市きたひろしましにある店みせの駐車場ちゅうしゃじょうで、警官けいかんが熊くまを見みつけました。
警察けいさつは、同おなじ熊くまの可能性かのうせいが高たかいと考かんがえて、気きをつけるように言いっています。
近ちかくの小学校しょうがっこうに通かよっている１年生ねんせいの母親ははおやは２２日にちの朝あさ、「今日きょうは心配しんぱいなので、一緒いっしょに来きました」と話はなしていました。
