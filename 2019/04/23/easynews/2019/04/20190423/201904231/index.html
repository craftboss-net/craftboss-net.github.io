岩手県いわてけん　珍めずらしい恐竜きょうりゅうの歯はの化石かせきを高校生こうこうせいが見みつける

发表于 2019-04-23 更新于 2019-11-30 分类于 JapanEasyNews



岩手県いわてけん久慈市くじしにある久慈琥珀博物館くじこはくはくぶつかんのイベントで去年きょねん、高校生こうこうせいが９ｍｍぐらいの化石かせきを見みつけました。この化石かせきを専門家せんもんかが調しらべて、ティラノサウルスという恐竜きょうりゅうの仲間なかまの歯はの化石かせきだとわかりました。
この歯はの化石かせきは９０００万まん年ねんぐらい前まえの土つちの中なかから見みつかりました。恐竜きょうりゅうの体からだの大おおきさは３ｍぐらいだろうと言いわれています。
専門家せんもんかは「日本にっぽんでこのような形かたちの歯はが見みつかることは珍めずらしいです。ティラノサウルスの仲間なかまで、まだ見みたことがない恐竜きょうりゅうの可能性かのうせいもあります」と話はなしています。
見みつけた高校生こうこうせいは「本当ほんとうに驚おどろきました。とてもラッキーだと思おもいました」と話はなしていました。
この化石かせきは、久慈琥珀博物館くじこはくはくぶつかんで８月がつ１９日にちまで見みることができます。
