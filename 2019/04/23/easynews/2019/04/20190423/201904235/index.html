ルノー「日産にっさんとルノーを１つの会社かいしゃで経営けいえいしたい」

发表于 2019-04-23 更新于 2019-11-30 分类于 JapanEasyNews



フランスの自動車じどうしゃ会社がいしゃのルノーが、日産自動車にっさんじどうしゃとルノーの経営けいえいを１つの会社かいしゃで行おこないたいと言いっていることがわかりました。そして、日産にっさんの経営けいえいにもっと強つよく関係かんけいすることができるように、ルノーで働はたらいていた人ひとを日産にっさんの取締役とりしまりやくなどにするように言いっています。
ルノーの株かぶをいちばん多おおく持もっているフランスの政府せいふは前まえにも、ルノーと日産にっさんの経営けいえいを１つの会社かいしゃで行おこないたいと言いいました。しかし、日産にっさんは自分じぶんたちで経営けいえいしたいと言いって反対はんたいしてきました。
ゴーン前会長ぜんかいちょうの事件じけんのあと、ルノーと日産にっさんは今いまのままの関係かんけいを続つづけることにしていました。
ルノーが考かんがえを変かえたことについて、日産にっさんの中なかでは反対はんたいする意見いけんが出でているようです。
