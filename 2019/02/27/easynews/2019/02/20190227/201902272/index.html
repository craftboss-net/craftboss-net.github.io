うその電話でんわやメール　お年寄としより以外いがいの人ひともお金かねをとられる

发表于 2019-02-27 更新于 2019-11-30 分类于 JapanEasyNews



うその電話でんわやメールなどでだまして、お金かねをとる詐欺さぎが続つづいています。警察けいさつによると、去年きょねん、このような詐欺さぎでとられたお金かねは全部ぜんぶで３５７億おく円えんでした。
その中なかでも、使つかっていないウェブサイトの料金りょうきんなどを払はらうように言いう詐欺さぎが増ふえています。このような詐欺さぎでとられたお金かねは１３７億おく円えんでした。うそのメールなどには、有名ゆうめいなウェブサイトの名前なまえが書かいてあったり、コンビニで電子でんしマネーを買かうように書かいてあったりします。いろいろな方法ほうほうでだましていて、うそだとわかりにくくなっています。
このような詐欺さぎでお金かねをとられた人ひとの４２％は、２０歳さいから５９歳さいの人ひとでした。お年寄としより以外いがいの人ひともたくさんだまされていました。
警察けいさつは、お金かねを払はらうように言いわれても、変へんだと思おもったら、すぐに払はらわないで、周まわりの人ひとに相談そうだんするか警察けいさつに連絡れんらくしてほしいと言いっています。
