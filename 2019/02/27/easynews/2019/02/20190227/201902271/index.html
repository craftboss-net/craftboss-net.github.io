高たかさ５４ｍまではしごが伸のびる消防しょうぼう車しゃ

发表于 2019-02-27 更新于 2019-11-30 分类于 JapanEasyNews



高たかい建物たてもので火事かじがあったとき、はしごを長ながく伸のばして、火ひを消けすことができる消防しょうぼう車しゃがあります。
日本にっぽんの消防しょうぼう車しゃでいちばん長ながいはしごは、高たかさ５４ｍまで伸のびます。この消防しょうぼう車しゃを愛知県あいちけん豊橋市とよはししでも使つかうことになりました。
２５日にち、市役所しやくしょの前まえで子こどもたちに新あたらしい消防しょうぼう車しゃを見みせました。はしごは市役所しやくしょの１３階がい、高たかさ４７ｍまで伸のびました。
この消防しょうぼう車しゃは１８階かい建だてのマンションの火事かじにも使つかうことができます。水みずを出だすホースは車くるまの中なかから動うごかすことができます。
