東北地方とうほくちほうで大おおきな地震じしんの可能性かのうせい「９０％以上いじょうの所ところもある」

发表于 2019-02-27 更新于 2019-11-30 分类于 JapanEasyNews



東北地方とうほくちほうから関東地方かんとうちほうの沖おきの海うみの底そこにある「日本海溝にほんかいこう」の近ちかくでは、何なん度ども地震じしんが起おこっています。８年ねん前まえには東日本大震災ひがしにほんだいしんさいがありました。
地震じしんについて調しらべている政府せいふの委員会いいんかいは、この場所ばしょでこれから３０年ねん以内いないに地震じしんが起おこる 可能性かのうせいを新あたらしく計算けいさんしました。
計算けいさんによると、８年ねん前まえと同おなじマグニチュード９ぐらいの地震じしんが起おこる 可能性かのうせいはほとんど０％でした。しかし、マグニチュード７から７．５ぐらいの地震じしんが起おこる 可能性かのうせいは高たかくなっています。
青森県あおもりけんの東ひがしと岩手県いわてけんの北きたの沖おきでは９０％以上いじょう、宮城県みやぎけんの沖おきでは９０％ぐらいでした。茨城県いばらきけんの沖おきでは８０％ぐらい、福島県ふくしまけんの沖おきでは５０％ぐらい、岩手県いわてけんの南みなみの沖おきでは３０％ぐらいでした。
調しらべた委員会いいんかいの人ひとは「東北地方とうほくちほうではもう大おおきな地震じしんは起おこらないと考かんがえないで、強つよい揺ゆれや高たかい津波つなみが来きたときのことを考かんがえて、大おおきな被害ひがいが出でないように準備じゅんびしてほしいです」と話はなしています。
