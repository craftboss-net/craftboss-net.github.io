神奈川県かながわけん箱根町はこねまちでチョコレートの風呂ふろを楽たのしむ

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



神奈川県かながわけん箱根町はこねまちにある「箱根はこね小涌園こわきえんユネッサン」には、いろいろな温泉おんせんがあります。今いまの季節きせつはバレンタインデーがあるため、毎年まいとしチョコレートの風呂ふろを楽たのしむことができます。風呂ふろには、チョコレートの原料げんりょうのカカオを使つかった入浴にゅうよく剤ざいが入はいっています。
この風呂ふろでは、入はいっている人ひとに１Ｌぐらいのチョコレートのソースをかけるイベントもあります。風呂ふろの中なかの人ひとは、チョコレートを顔かおや体からだに塗ぬったり、香かおりを楽たのしんだりしていました。風呂ふろを楽たのしんだ人ひとは「すごくいいにおいです」とか、「バレンタインデーがとても楽たのしみです」などと話はなしていました。
箱根はこね小涌園こわきえんユネッサンは「チョコレートの香かおりでリラックスしながら楽たのしむことができる風呂ふろなので、たくさんの人ひとに入はいってほしいです」と話はなしています。チョコレートの風呂ふろは３月がつ１７日にちまでです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

温泉

  • 地下水ちかすいが火山かざんなどの熱ねつで温あたためられて、地下ちかからわき出だす湯ゆ。いろいろな成分せいぶんがとけていて病気びょうきに効きく。また、その湯ゆの出でる場所ばしょ。

バレンタインデー

  • キリスト教きょうの聖人せいじんバレンタインを記念きねんする日ひ。二月にがつ十四日じゅうよっか。日本にっぽんでは女性じょせいから男性だんせいへ、チョコレートなどのプレゼントをおくる習慣しゅうかんがある。

原料

  • 品物しなものを作つくるための、もとになる物もの。

カカオ

  • 熱帯ねったい地方ちほうで栽培さいばいする高木こうぼく。枝えだや幹みきにできる実みの中なかの種たねを、ココアやチョコレートの原料げんりょうにする。

入浴

  • ふろに入はいること。

剤

  • 薬くすりなどを混まぜ合あわせる。また、混まぜ合あわせた薬くすり。

リットル

  • メートル法ほうで、量りょうの基本きほんの単位たんい。１リットルは、１０００立方りっぽうセンチメートル。一辺ぺんが１０センチメートルの立方体りっぽうたいの体積たいせきを１リットルとする。記号きごうは「L」。

ソース

  • 西洋料理せいようりょうりの味あじつけに使つかう、液体えきたいの調味料ちょうみりょう。種類しゅるいが多おおい。

イベント

  • 行事ぎょうじ。できごと。
  • 勝負しょうぶ。試合しあい。

香り

  • いいにおい。
  • そのものから感かんじられる品ひんの高たかさ。

リラックス

  • 緊張きんちょうしないで、のんびりと楽らくにすること。
