ロボット掃除機そうじき　電気でんきストーブの火事かじに気きをつけて

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



ロボット掃除機そうじきは、自動じどうで部屋へやの中なかを動うごいて掃除そうじをします。東京とうきょうのマンションで去年きょねん１２月がつ、電気でんきストーブがソファーにぶつかって焼やける火事かじがありました。
火事かじになったとき部屋へやの中なかには誰だれもいませんでしたが、ロボット掃除機そうじきが動うごいていました。東京消防庁とうきょうしょうぼうちょうが実験じっけんを行おこなうと、ロボット掃除機そうじきが電気でんきストーブを押おしたり、ストーブのコードを巻まき込こんだりすることがわかりました。
東京消防庁とうきょうしょうぼうちょうは、ロボット掃除機そうじきが電気でんきストーブを５０ｃｍぐらい押おして、ストーブがソファーにぶつかって火ひが出でたと考かんがえています。ロボット掃除機そうじきが原因げんいんだと考かんがえられている火事かじは、今月こんげつも１件けんありました。
東京消防庁とうきょうしょうぼうちょうは、ロボット掃除機そうじきを使つかうときには、電気でんきストーブのコードをコンセントから抜ぬいてほしいと言いっています。
