切きっても元もとに戻もどる新あたらしいゴムができる

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



理化学研究所りかがくけんきゅうしょなどのグループが、切きっても元もとに戻もどる新あたらしいゴムを作つくりました。グループによると、ゴムの切きった所ところを軽かるく合あわせると、何なん分ぷんかあとに元もとのようにつながります。傷きずもほとんど見みえなくなります。このゴムで作つくった袋ふくろは、穴あながあいてもすぐに閉とじます。
グループは、分子ぶんしと分子ぶんしが引ひき合あう 物理学ぶつりがくのルールを利用りようしてこのゴムを作つくりました。
切きっても元もとに戻もどるゴムは前まえからありましたが、つけるためには光ひかりや熱ねつなどのエネルギーが必要ひつようでした。新あたらしくできたゴムは、光ひかりや熱ねつなどが必要ひつようありません。そして、水みずの中なかや宇宙うちゅうなどいろいろな場所ばしょで使つかうことができます。
ゴムのほかにも、切きっても元もとに戻もどるガラスやコンクリート、金属きんぞくなどの研究けんきゅうが進すすんでいます。
