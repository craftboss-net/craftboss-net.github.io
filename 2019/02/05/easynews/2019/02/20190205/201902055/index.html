東京とうきょうタワーを赤あかくライトアップ　春節しゅんせつのお祝いわいをする

发表于 2019-02-05 更新于 2019-11-30 分类于 JapanEasyNews


中国ちゅうごくでは昔むかしの暦こよみの正月しょうがつ「春節しゅんせつ」の休やすみが始はじまりました。
日本にっぽんに住すんでいる中国人ちゅうごくじんの団体だんたいなどは春節しゅんせつのお祝いわいをするため、４日よっかの夜よる、東京とうきょうタワーを赤あかい光ひかりでライトアップしました。中国ちゅうごくでは赤あかは幸しあわせの色いろだと言いわれています。東京とうきょうタワーの下したにある広場ひろばでは、中国ちゅうごくの子こどもたちが歌うたを歌うたいました。
記念きねんの式しきで、中国ちゅうごくの程てい永華えいか大使たいしは「今年ことしは中国ちゅうごくと日本にっぽんが新あたらしい未来みらいをつくることができるといいと思おもいます」と話はなしました。日本にっぽんの安倍あべ総理大臣そうりだいじんもビデオで「今年ことしは日本人にっぽんじんと中国人ちゅうごくじんが今いままでよりたくさん交流こうりゅうできるようにしたいです」と言いいました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

暦

  • 一年いちねんじゅうの月つき・日ひ・週しゅう、行事ぎょうじ、祝日しゅくじつなどを、日ひの順じゅんに書かきこんだもの。カレンダー。

団体

  • 大勢おおぜいの人ひとの集あつまり。
  • 同おなじ目的もくてきを持もった人々ひとびとの集あつまり。

幸せ

  • 十分じゅうぶんに満足まんぞくしている状態じょうたい。幸福こうふく。

広場

  • 大勢おおぜいの人ひとが集あつまれる、広ひろい場所ばしょ。

記念

  • 思おもい出でに残のこしておくこと。また、その物もの。

大使

  • 国くにを代表だいひょうして外交がいこうの仕事しごとをする、外交官がいこうかんのいちばん上うえの役やくの人ひと。

未来

  • これから先さきのこと。将来しょうらい。

総理大臣

  • 内閣総理大臣ないかくそうりだいじんのこと。内閣ないかくの最高責任者さいこうせきにんしゃで、国会議員こっかいぎいんの中なかから議員ぎいんが選えらび、天皇てんのうが認みとめて決きまる。首相しゅしょう。総理そうり。

ビデオ

  • 映像えいぞう。
  • 「ビデオテープ」の略りゃく。
  • 映像えいぞうや音おとを、磁気じきテープに記録きろくしたり再生さいせいしたりする装置そうち。

交流

  • たがいに行いき来きすること。交まじり合あうこと。
  • 決きまった時間じかんごとに、流ながれの方向ほうこうが逆ぎゃくになる電流でんりゅう。
