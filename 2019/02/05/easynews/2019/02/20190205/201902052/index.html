山梨県やまなしけん甲府市こうふしでゆっくり動うごく金属きんぞくの作品さくひんを展示てんじ

发表于 2019-02-05 更新于 2019-11-30 分类于 JapanEasyNews


山梨大学やまなしだいがくの先生せんせいの村松むらまつ俊夫としおさんは、金属きんぞくのパイプで「おきあがりこぼし」のような作品さくひんを作つくりました。山梨県やまなしけん甲府市こうふしで７日なのかまで作品さくひんを展示てんじしています。
展示てんじしている８つの作品さくひんはバランスが計算けいさんしてあって、押おすと揺ゆれたり転ころがったりしてゆっくり動うごきます。１本ぽんのパイプを曲まげた作品さくひんは、端はしにある鉄てつのボールの重おもさを使つかって、大おおきく倒たおれたり起おきたりします。巻まいたような形かたちの作品さくひんは、ゆっくり転ころがります。
村松むらまつさんは「科学かがくと美術びじゅつと技術ぎじゅつはつながっています。特とくに子こどもに楽たのしんでもらいたいです」と話はなしています。
