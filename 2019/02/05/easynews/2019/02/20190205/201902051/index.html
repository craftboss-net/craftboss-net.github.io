香港ほんこんの工場こうじょうでじゃがいもと一緒いっしょに小ちいさい爆弾ばくだんが見みつかる

发表于 2019-02-05 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんの会社かいしゃの「カルビー」の香港ほんこんにある工場こうじょうでは、じゃがいもを使つかってポテトチップスを作つくっています。
この工場こうじょうでじゃがいもを機械きかいで分わけていたとき、土つちがついた丸まるい金属きんぞくが見みつかりました。２日ふつかの朝あさ、警察けいさつがこの金属きんぞくを調しらべると、手てで投なげる小ちいさい爆弾ばくだんだとわかりました。警察けいさつが工場こうじょうの中なかから安全あんぜんに外そとに出だしたため、働はたらいている人ひとや工場こうじょうに被害ひがいはありませんでした。
警察けいさつによると、この小ちいさい爆弾ばくだんは、フランスから輸入ゆにゅうしたたくさんのじゃがいもと一緒いっしょに入はいっていました。重おもさは１ｋｇで、大おおきさは直径ちょっけい８ｃｍありました。１００年ねんぐらい前まえの第だい１次じ世界大戦せかいたいせんのときに、ドイツが作つくったと考かんがえられています。

じゃがいも

  • 地下ちかにできるイモを食用しょくようにする作物さくもつ。夏なつのはじめ、白しろまたはうすむらさき色いろの花はなが咲さく。ばれいしょ。

爆弾

  • 中なかに火薬かやくをつめて、投なげたり落おとしたりして爆発ばくはつさせる兵器へいき。

  • 配くばる。分配ぶんぱいする。

  • 引ひき分わけにする。

  • ふつうは固体こたいで、熱ねつや電気でんきをよく伝つたえるもの。鉄てつ・銅どう・金きん・銀ぎん・アルミニウムなど。

直径

第一次世界大戦

  • １９１４年ねんから１９１８年ねんまで、ドイツ・オーストリアなどの同盟国どうめいこくとイギリス・フランス・ロシア・アメリカ・日本にっぽんなどの連合国れんごうこくとの間あいだで行おこなわれた戦争せんそう。ドイツ側がわが敗やぶれて終おわった。
