客きゃくの荷物にもつを飛行機ひこうきに入いれる仕事しごとにパワーアシストスーツを使つかう

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews



日本航空にほんこうくうのグループの会社かいしゃは２月がつから、空港くうこうで客きゃくの荷物にもつを飛行機ひこうきに入いれる仕事しごとなどに、パワーアシストスーツという機械きかいを使つかい始はじめました。このスーツを体からだにつけると、少すくない力ちからで重おもい荷物にもつを持もつことができます。
羽田空港はねだくうこうでは女性じょせいもこのスーツをつけて、スーツケースを飛行機ひこうきのコンテナに入いれていました。
スーツを作つくった会社かいしゃによると、スーツの重おもさは４．５ｋｇぐらいです。モーターを使つかって、荷物にもつを持もつときに腰こしにかかる力ちからを少すくなくします。スーツをつけていないときと比くらべると、同おなじ時間じかんで荷物にもつを２０％以上いじょう多おおく運はこぶことができます。
日本にっぽんの航空こうくう会社がいしゃでは、全日空ぜんにっくうも同おなじようなスーツを使つかっています。

コンテナー

  • 貨物かもつの輸送ゆそうに使つかう大おおきな箱はこ。品物しなものを入いれて、そのまま貨車かしゃやトラックで運はこぶ。コンテナ。

モーター

  • 電気でんきや蒸気じょうき・ガソリンなどで、物ものを動うごかす力ちからを起おこす機械きかい。電動機でんどうき。発動機はつどうき。
  • 自動車じどうしゃ。

腰

  • 体からだの、胴どうと足あしとの間あいだの部分ぶぶん。
  • 物ものの真まん中なか辺あたりより少すこし下したの部分ぶぶん。
  • 粉こなや、もちなどの、ねばり。

  • 飛行機ひこうきなどで空そらを飛とぶこと。
