さいたま市し　子こどもを保育園ほいくえんに入いれるかＡＩが決きめる

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews



多おおくの市しや町まちは今いま、４月がつから保育園ほいくえんに入はいりたい子こどもを保育園ほいくえんに入いれるか決きめて家族かぞくに連絡れんらくする仕事しごとで忙いそがしくなっています。
さいたま市しでは、今年ことし４月がつから保育園ほいくえんに入はいりたい子こどもが８８００人にんぐらいいます。市しが保育園ほいくえんを決きめるときは、子こどもの家いえがある場所ばしょや兄あにや姉あねが通かよっている保育園ほいくえんなどを考かんがえなければなりません。このため、長ながい時間じかんがかかります。
さいたま市しは、２０１９年度ねんどからコンピューターのＡＩを使つかって、子こどもを保育園ほいくえんに入いれるか決きめることにしました。
コンピューターに、家族かぞくの考かんがえなどを入いれると、保育園ほいくえんに通かようことができるかすぐにわかります。さいたま市しは、今いまより１週間しゅうかんぐらい早はやく家族かぞくに連絡れんらくすることができるようになると言いっています。
