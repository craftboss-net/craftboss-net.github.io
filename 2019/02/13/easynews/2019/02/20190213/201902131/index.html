チョコレートを食たべながら飲のむとおいしい梅酒うめしゅを輸出ゆしゅつする

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews



和歌山県わかやまけん田辺市たなべしにある会社かいしゃが、チョコレートを食たべながら飲のむとおいしい梅酒うめしゅをつくりました。梅酒うめしゅは、梅うめの実みでつくるお酒さけです。イタリアでチョコレートの会社かいしゃをつくった人ひとと一緒いっしょに考かんがえました。
この梅酒うめしゅには、田辺市たなべしなどで育そだてている「古城梅ごじろうめ」という梅うめを使つかっていて、アーモンドに似にた香かおりがします。あまり甘あまくなくて、チョコレートによく合あいます。
会社かいしゃは、この梅酒うめしゅを２０００本ぼんぐらいつくって、半分はんぶんはヨーロッパで売うる計画けいかくです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

梅

  • 春はるの初はじめ、葉はよりも先さきに、白しろや紅くれないのかおりのよい花はなが咲さく木き。六月ろくがつごろ実みが大おおきくなり、梅うめぼしや梅酒うめしゅなどにする。

実

  • 花はなが終おわったあとにできるもの。中なかに種たねがある。
  • 植物しょくぶつの種たね。
  • みそしるなどの中なかに入いれる、野菜やさい・肉にくなど。
  • 中身なかみ。

香り

  • いいにおい。
  • そのものから感かんじられる品ひんの高たかさ。

ヨーロッパ

  • 世界せかいの六大州ろくだいしゅうの一ひとつ。アジアの北西ほくせい、アフリカの北きたにある。産業さんぎょうや文化ぶんかが発達はったつした国くにが多おおい。
