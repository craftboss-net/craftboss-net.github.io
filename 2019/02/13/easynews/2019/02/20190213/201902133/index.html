水泳すいえいの池江いけえ璃花子りかこ選手せんしゅが白血病はっけつびょうになったと発表はっぴょう

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews



水泳すいえいの池江いけえ璃花子りかこ選手せんしゅは、女子じょし１００mバタフライなどたくさんの日本記録にほんきろくを持もっています。
池江いけえ選手せんしゅは１２日にち、白血病はっけつびょうという病気びょうきになったと発表はっぴょうしました。池江いけえ選手せんしゅはインターネットのツイッターに「体からだの具合ぐあいが悪わるかったので検査けんさをして、白血病はっけつびょうだとわかりました。しっかり治療ちりょうをしたら治なおる病気びょうきなので、今いまは少すこし休やすみます。１日にちでも早はやく、もっと強つよくなって戻もどることができるように頑張がんばります」と書かいています。
池江いけえ選手せんしゅは１８歳さいで、来年らいねん東京とうきょうで行おこなうオリンピックでメダルを取とるために頑張がんばっています。
