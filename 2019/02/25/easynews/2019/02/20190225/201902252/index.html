埼玉県さいたまけん秩父市ちちぶし　怖こわさを楽たのしむつり橋ばしができる

发表于 2019-02-25 更新于 2019-11-30 分类于 JapanEasyNews



埼玉県さいたまけん秩父市ちちぶしの山やまを流ながれる川かわに、長ながさ１００ｍのつり橋ばしができました。川かわからの高たかさは５０ｍ以上いじょうです。たくさんの人ひとに楽たのしんでもらうために、秩父市ちちぶしの会社かいしゃがつくりました。２１日にち、この橋はしを記者きしゃに見みせて、秩父市ちちぶしの人ひとなどに渡わたってもらいました。
橋はしはワイヤーと木きの板いたでできています。板いたと板いたの間あいだから、下したを流ながれる川かわがよく見みえます。渡わたる人ひとは安全あんぜんベルトをつけて、橋はしの両側りょうがわにあるワイヤーをしっかり持もって歩あるきます。渡わたった女性じょせいは「風かぜが強つよくて怖こわかったですが、思おもったより楽たのしかったです」と話はなしていました。
この橋はしは３月がつの終おわりまでにオープンする予定よていです。そのあと、体からだにロープなどをつけて、高たかい所ところから飛とんで降おりる「バンジージャンプ」もできるようにする予定よていです。
