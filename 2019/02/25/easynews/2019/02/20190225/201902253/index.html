日本にほん文学ぶんがくを長ながい間あいだ研究けんきゅうしたドナルド?キーンさんが亡なくなる

发表于 2019-02-25 更新于 2019-11-30 分类于 JapanEasyNews



日本にほん文学ぶんがくを５０年ねん以上いじょう研究けんきゅうしていたドナルド?キーンさんが、２４日にじゅうよっか、東京とうきょうの病院びょういんで亡なくなりました。９６歳さいでした。
キーンさんは、１９２２年ねんにアメリカで生うまれて、海軍かいぐんの学校がっこうで日本語にほんごを勉強べんきょうしました。そして京都大学きょうとだいがくに留学りゅうがくしたあと、アメリカのコロンビア大学だいがくの教授きょうじゅになって、日本にほん文学ぶんがくの研究けんきゅうを続つづけました。
キーンさんは、日本にほん文学ぶんがくについてたくさんの本ほんを書かいたり、日本にっぽんで昔むかしから続つづいている能のうや歌舞伎かぶき、日本にっぽんの文化ぶんかについて意見いけんを発表はっぴょうしたりしました。古ふるい時代じだいの本ほんなどたくさんの本ほんを英語えいごに翻訳ほんやくして、日本にほん文学ぶんがくを世界せかいに紹介しょうかいしました。
キーンさんは日本にっぽんでいろいろな賞しょうをもらっていて、２００８年ねんには文化ぶんか勲章くんしょうをもらいました。
２０１１年ねんの東日本大震災ひがしにほんだいしんさいのあとには日本にっぽんにずっと住すむことを決きめて、日本にっぽんの国籍こくせきを取とりました。キーンさんは「日本にっぽんにいることは幸しあわせです。日本人にっぽんじんが大好だいすきです」と話はなしていました。
