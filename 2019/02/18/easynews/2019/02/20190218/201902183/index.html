北海道ほっかいどう　地震じしんで壊こわれた建物たてものにあった木きで家具かぐを作つくって売うる

发表于 2019-02-18 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどうでは去年きょねん９月がつに大おおきな地震じしんがあって、厚真町あつまちょうでは家いえなどがたくさん壊こわれました。
札幌市さっぽろしでは、北海道ほっかいどうの大工だいくやデザイナーが厚真町あつまちょうの壊こわれた建物たてものにあった木きで家具かぐを作つくって、売うっています。作つくった家具かぐはいすやテーブルなどです。壊こわれた建物たてものから木きを運はこんだときの写真しゃしんをインターネットで見みることもできます。
家具かぐは２４日にじゅうよっかまで売うっていて、売うったお金かねの一部いちぶは地震じしんで家いえが壊こわれた人ひとに渡わたします。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

家具

  • 家いえの中なかに、備そなえつけて使つかう道具どうぐ。たんす・テーブル・本箱ほんばこなど。

大工

  • 家いえを建たてたり、直なおしたりするのを仕事しごとにしている人ひと。また、その仕事しごと。

デザイナー

  • 洋服ようふくや建物たてものなどの形かたちや模様もようなどを考かんがえることを仕事しごとにしている人ひと。

一部
