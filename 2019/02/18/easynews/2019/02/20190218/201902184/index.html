タイの認知症にんちしょうの女性じょせいが家いえから６４０ｋｍの中国ちゅうごくで見みつかる

发表于 2019-02-18 更新于 2019-11-30 分类于 JapanEasyNews



タイのチェンライ県けんに住すんでいる５９歳さいの女性じょせいは、去年きょねん６月がつ、近ちかくに住すんでいる息子むすこに会あうために家いえを出でました。そのあと、どこへ行いったかわからなくなりました。この女性じょせいは、考かんがえたり覚おぼえたりすることが難むずかしくなる認知症にんちしょうという病気びょうきになっています。
約やく７か月げつが過すぎた１月がつの終おわりに、この女性じょせいは、家いえから北きたへ６４０ｋｍぐらいの所ところにある中国ちゅうごくの昆明こんめいで見みつかりました。女性じょせいがＩＤカードを持もっていたためわかりました。
女性じょせいは、２０ｋｇぐらい体重たいじゅうが少すくなくなっていましたが元気げんきそうで、２月がつ１５日にちに飛行機ひこうきでタイへ帰かえりました。
女性じょせいは「私わたしは息子むすこに会あうことができたので、そのあと歩あるき続つづけました。おなかがすいたときは、店みせで食たべ物ものをもらいました」と話はなしています。
女性じょせいの家いえから中国ちゅうごくへ行いくまでに、ミャンマーかラオスを通とおらなければなりませんが、女性じょせいがどうやって中国ちゅうごくへ行いったかはわかっていません。
