バレンタインデー「食たべないチョコを子こどもに届とどけよう」

发表于 2019-02-14 更新于 2019-11-30 分类于 JapanEasyNews



２月がつ１４日じゅうよっかはバレンタインデーです。
山梨県やまなしけんのＮＰＯの「フードバンク山梨やまなし」は、食たべないで余あまっているチョコレートを、生活せいかつに困こまっている家庭かていの子こどもに届とどけようと考かんがえました。バレンタインデーにチョコレートをもらっても、全部ぜんぶは食たべないという男性だんせいが３０％ぐらいいるという調査ちょうさもあるためです。
フードバンク山梨やまなしは、山梨県やまなしけんに住すんでいる人ひとや会社かいしゃに「食たべないチョコレートを持もってきてください」と言いっています。２０日はつかまで甲府市こうふしにある事務所じむしょなどでチョコレートを集あつめて、そのあと子こどもに届とどける予定よていです。
