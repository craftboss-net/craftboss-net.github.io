東京都とうきょうとでいつもの年としより早はやく杉すぎの花粉かふんが飛とび始はじめる

发表于 2019-02-14 更新于 2019-11-30 分类于 JapanEasyNews



花粉症かふんしょうの人ひとは体からだの中なかに花粉かふんが入はいると、くしゃみが出でたり目めがかゆくなったりします。
東京都とうきょうとで１１日にちから杉すぎの花粉かふんが飛とび始はじめました。東京都とうきょうとの発表はっぴょうによると、今年ことしは去年きょねんより３日みっか早はやく飛とび始はじめていて、最近さいきんの１０年ねん間かんの平均へいきんより５日いつか早はやくなっています。
東京都とうきょうとは「今月こんげつ初はじめに暖あたたかい日ひが続つづいて、杉すぎの花はなが咲さくのが早はやくなったからでしょう」と話はなしています。飛とぶ花粉かふんの量りょうはいつもの年としと同おなじぐらいになりそうです。
東京都とうきょうとはどこでどのくらい花粉かふんが飛とんでいるかインターネットに出だしていて、花粉かふんが多おおい日ひにはマスクをしたり眼鏡めがねをかけたりするように言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

杉

  • すぎ。幹みきがまっすぐで、針はりのような葉はが一年いちねんじゅう緑色みどりいろをしている高木こうぼく。建築けんちくや家具かぐなどに使つかわれる。

花粉

  • 花はなの雄おしべの先さきから出だす細こまかな粉こな。雌めしべにつくと、実みを結むすぶ。

花粉症

  • アレルギーの一ひとつ。目めや鼻はなが、スギやヒノキなどの花粉かふんにしげきされて、なみだ・くしゃみ・鼻水はなみずなどが出でる。

くしゃみ

  • 鼻はなのねんまくが、しげきを受うけて、急きゅうにふき出だす息いき。はくしょん。

かゆい

  • 皮膚ひふがむずむずして、かきたくなる。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

間

  • あいだ。へだたり。
  • ひろがり。
  • ある関係かんけいをもったまとまり。
  • 相手あいてのようすをうかがう。
  • へや。へやを数かぞえることば。
  • 昔むかしの尺貫法しゃっかんほうで、長ながさの単位たんいの一ひとつ。一間いっけんは約やく一・八メートル。

平均

  • 多おおい少すくないや高たかい低ひくいなどがないように、ならすこと。
  • つり合あいがとれること。バランス。

量

マスク

  • お面めん。仮面かめん。
  • ほこりなどを吸すわないように、口くちや鼻はなをおおう布ぬの。
  • ガスなどを吸すわないように、消防士しょうぼうしなどが顔かおにつけるお面めん。防毒面ぼうどくめん。
