今年ことしも風疹ふうしんになる人ひとが増ふえている「ワクチンの注射ちゅうしゃをして」

发表于 2019-02-14 更新于 2019-11-30 分类于 JapanEasyNews



風疹ふうしんは熱ねつが出でたり体からだに赤あかい発疹はっしんが出でたりする病気びょうきで、ウイルスでうつります。
国くにの研究所けんきゅうじょによると、去年きょねん、風疹ふうしんになった人ひとは２９１７人にんで、今いままでの１０年ねんで２番目ばんめに多おおくなりました。今年ことしも２月がつ３日みっかまでに３６７人にんが風疹ふうしんになっています。この中なかの６０％以上いじょうは東京都とうきょうととその周まわりの県けんの人ひとで、大阪府おおさかふや福岡県ふくおかけんなどでも増ふえています。
研究所けんきゅうじょによると、風疹ふうしんになる人ひとは毎年まいとし春はるから多おおくなるため、これからもっと増ふえる心配しんぱいがあります。
風疹ふうしんになる人ひとが特とくに多おおい３９歳さいから５６歳さいの男性だんせいは、ワクチンの注射ちゅうしゃをしていない人ひとがたくさんいます。このため、国くにはこれから、この年齢ねんれいの男性だんせいに無料むりょうでワクチンの注射ちゅうしゃをすることができるようにします。
妊娠にんしんしている女性じょせいが風疹ふうしんになると、赤あかちゃんの目めや耳みみなどに障害しょうがいが出でることがあります。研究所けんきゅうじょは「女性じょせいは妊娠にんしんする前まえにワクチンの注射ちゅうしゃを２回かいしてください」と言いっています。
