辺野古へのこの工事こうじに賛成さんせいか反対はんたいか　沖縄県おきなわけんの人ひとたちが２４日にじゅうよっかに投票とうひょう

发表于 2019-02-14 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの政府せいふは、沖縄県おきなわけん宜野湾市ぎのわんしにあるアメリカ軍ぐんの普天間基地ふてんまきちを名護市なごし辺野古へのこに移うつすため、辺野古へのこの海うみを埋うめる 工事こうじをしています。しかし、沖縄県おきなわけんは工事こうじに反対はんたいしています。
沖縄県おきなわけんは１４日じゅうよっか、工事こうじに賛成さんせいか反対はんたいか、沖縄県おきなわけんの人ひとたちの意見いけんを聞きく投票とうひょうを２４日にじゅうよっかに行おこなうと発表はっぴょうしました。投票とうひょうする人ひとは「賛成さんせい」「反対はんたい」「どちらでもない」の３つの意見いけんの中なかから１つ選えらびます。
投票とうひょうでいちばん多おおかった意見いけんが有権者ゆうけんしゃの２５％以上いじょうだった場合ばあい、知事ちじがその意見いけんを日本にっぽんの総理そうり大臣だいじんとアメリカの大統領だいとうりょうに伝つたえることになっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

工事

  • 建物たてもの・道路どうろ・橋はしなどを造つくったり、直なおしたりすること。また、その仕事しごと。

賛成

  • 人ひとの考かんがえをよいと認みとめること。同意どういすること。

投票

  • 選挙せんきょや採決さいけつで、選えらびたい人ひとの名前なまえや、賛成さんせいか反対はんたいかを、紙かみに書かいて出だすこと。

政府

  • 国くにの政治せいじを行おこなうところ。
  • 内閣ないかく。

軍

  • 兵隊へいたい。兵隊へいたいの集あつまり。
  • 戦たたかい。戦争せんそう。

移す

  • 場所ばしょを変かえる。
  • ものごとを進すすめる。
  • 感染かんせんさせる。

埋める

  • 穴あなをほって物ものを入いれ、土つちや砂すなをかぶせる。
  • ふさいで平たいらにする。
  • すき間まなく、いっぱいにする。
  • お湯ゆに水みずを入いれて、ぬるくする。
  • 足たりない分ぶんを補おぎなう。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

有権者

  • 権利けんりを持もっている人ひと。特とくに、選挙権せんきょけんを持もっている人ひと。

知事

  • 都と・道どう・府ふ・県けんなどの政治せいじをとる、いちばん上うえの役目やくめ。また、その人ひと。

総理大臣

  • 内閣総理大臣ないかくそうりだいじんのこと。内閣ないかくの最高責任者さいこうせきにんしゃで、国会議員こっかいぎいんの中なかから議員ぎいんが選えらび、天皇てんのうが認みとめて決きまる。首相しゅしょう。総理そうり。

大統領

  • 共和国きょうわこくで、その国くにを代表だいひょうする人ひと。
