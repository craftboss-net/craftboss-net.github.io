コンビニを経営けいえいする人ひとの団体だんたい　「営業えいぎょう時間じかんを短みじかくしたい」

发表于 2019-02-28 更新于 2019-11-30 分类于 JapanEasyNews



今月こんげつ、東大阪市ひがしおおさかしでセブン－イレブンの店みせを経営けいえいする人ひとが、働はたらく人ひとが足たりないため店みせの営業えいぎょう時間じかんを短みじかくしました。セブン－イレブンの会社かいしゃは、２４時間じかん営業えいぎょうをしないのは契約けいやくに違反いはんしていると言いって、この店みせを経営けいえいする人ひとに１７００万まん円えん払はらうように言いいました。
コンビニの店みせを経営けいえいしている人ひとなどの団体だんたいは２７日にち、セブン－イレブンの会社かいしゃに行いって、営業えいぎょう時間じかんなどについて会社かいしゃと話はなし合あいたいと言いいました。
団体だんたいの人ひとは「私わたしたちは会社かいしゃの許可きょかがないと、営業えいぎょう時間じかんを変かえることができません。会社かいしゃと話はなし合あって、どのような場合ばあいに時間じかんを短みじかくすることができるかなどを決きめたいです」と言いっています。
セブン－イレブンの会社かいしゃは「団体だんたいと話はなし合あうことは考かんがえていません。２４時間じかん営業えいぎょうすることが難むずかしい場合ばあいは、会社かいしゃの社員しゃいんが店みせを手伝てつだいます」と話はなしています。
