女性じょせいの消防しょうぼう隊員たいいんの訓練くんれん「災害さいがいがあった場所ばしょで役やくに立たちたい」

发表于 2019-02-28 更新于 2019-11-30 分类于 JapanEasyNews



消防しょうぼうの仕事しごとをする消防しょうぼう隊員たいいんは、地震じしんや台風たいふうなどが起おこった場所ばしょで、人ひとを助たすける仕事しごともします。女性じょせいの消防しょうぼう隊員たいいんにもこのような仕事しごとをしてもらうために２５日にちと２６日にち、さいたま市しで初はじめての訓練くんれんがありました。
１５０人にんぐらいが集あつまって、重おもさが１００ｋｇ以上いじょうあるテントを建たてたり、たくさんのお湯ゆを用意よういしたりする訓練くんれんをしました。
さいたま市しによると、女性じょせいの消防しょうぼう隊員たいいんが災害さいがいがあった場所ばしょに行いく機会きかいは、今いままではほとんどありませんでした。しかし、災害さいがいのとき役やくに立たちたいと考かんがえる女性じょせいの消防しょうぼう隊員たいいんが増ふえています。訓練くんれんに参加さんかした女性じょせいは「被害ひがいがあった場所ばしょや人ひとのために、女性じょせいができることがあると強つよく思おもいました」と話はなしていました。
さいたま市しは「災害さいがいがあった場所ばしょでも女性じょせいが働はたらきやすいようにして、女性じょせいが活躍かつやくできるようにしたいです」と話はなしています。
