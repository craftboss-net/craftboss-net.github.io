ブラジル　死しんだ鯨くじらがアマゾン川がわで見みつかる

发表于 2019-02-28 更新于 2019-11-30 分类于 JapanEasyNews



２月がつ２２日にち、ブラジルのアマゾン川がわの河口かこうにある林はやしの中なかで、死しんだ鯨くじらが見みつかりました。海うみに住すむ鯨くじらが林はやしの中なかで見みつかって、ブラジルの人ひとたちは驚おどろいています。
ロイター通信つうしんなどによると、鯨くじらは若わかい雄おすで、体からだの長ながさは８ｍぐらいです。死しんでから５日いつかぐらい過すぎていたようです。
自然しぜんを大切たいせつにする活動かつどうをしている団体だんたいなどによると、ブラジルの海うみに来くる鯨くじらは、今いまの季節きせつは南極なんきょくにいます。
専門家せんもんかは死しんだ鯨くじらのＤＮＡなどを調しらべて、どこから来きた鯨くじらか調しらべたいと言いっています。
