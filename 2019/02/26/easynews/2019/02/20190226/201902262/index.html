大阪おおさかで銭湯せんとうのよさを知しってもらうイベント

发表于 2019-02-26 更新于 2019-11-30 分类于 JapanEasyNews



大阪おおさかでは、年としをとって経営けいえいをやめる人ひとなどがいて、銭湯せんとうが少すくなくなっています。銭湯せんとうを経営けいえいする若わかい人ひとたちは、銭湯せんとうのよさを知しってもらいたいと考かんがえて、大阪市おおさかし住吉区すみよしくでイベントを行おこないました。
会場かいじょうには、車輪しゃりんをつけたおけを使つかって、カーリングのように遊あそぶ場所ばしょを作つくりました。子こどもたちは、点数てんすうが書かいてあるところに止とまるように、おけを走はしらせていました。
銭湯せんとうで使つかっていたいろいろな色いろののれんを、迷路めいろのように並ならべた場所ばしょもありました。家族かぞくで来きた人ひとたちが、写真しゃしんを撮とりながら歩あるいて楽たのしんでいました。
イベントを行おこなった人ひとは「大阪おおさかでは、台風たいふうや地震じしんの被害ひがいもあって、銭湯せんとうを休やすんだりやめたりしています。どうやって銭湯せんとうを元気げんきにするか、これからも考かんがえていきたいです」と話はなしていました。
