いちばん売うれた音楽おんがく　外国がいこくのアーティストは「クイーン」

发表于 2019-02-26 更新于 2019-11-30 分类于 JapanEasyNews



日本にほんレコード協会きょうかいは毎年まいとし、日本にっぽんで活躍かつやくした歌手かしゅやバンドなどのアーティストに賞しょうを贈おくっています。「アーティスト?オブ?ザ?イヤー」は、ＣＤと音楽おんがくビデオ、インターネットで曲きょくを聞きくサービスなどの売うり上あげが１年ねんでいちばん多おおかったアーティストに贈おくる賞しょうです。
外国がいこくのアーティストでは、イギリスのロックバンド「クイーン」がこの賞しょうをもらいました。去年きょねん１１月がつ、クイーンについての映画えいがが日本にっぽんでも始はじまって、とても人気にんきになりました。この映画えいがの音楽おんがくが入はいったＣＤやクイーンのアルバムがたくさん売うれています。
ギターのブライアン?メイさんは「日本にっぽんの皆みなさんを愛あいしています。また近ちかいうちに会あいましょう」と話はなしています。
日本にっぽんのアーティストでは、去年きょねん９月がつに歌手かしゅをやめた安室あむろ奈美恵なみえさんがもらいました。安室あむろさんは２年ねん続つづけてこの賞しょうをもらいました。
