アカデミー賞しょうのパーティー　２年ねん続つづけて宮崎県みやざきけんの牛肉ぎゅうにくを使つかう

发表于 2019-02-22 更新于 2019-11-30 分类于 JapanEasyNews



アカデミー賞しょうは、アメリカでいちばん有名ゆうめいな映画えいがの賞しょうだと言いわれています。アカデミー賞しょうを発表はっぴょうする式しきのあとには、毎年まいとしパーティーがあります。
去年きょねんのパーティーでは、宮崎県みやざきけんの牛肉ぎゅうにくを使つかった料理りょうりや焼酎しょうちゅうが出でました。宮崎県みやざきけんの出身しゅっしんでロサンゼルスに住すんでいる映画えいが監督かんとくの曽原そはら三友紀みゆきさんが、よく知しっている料理りょうり人にんに頼たのんで宮崎県みやざきけんの牛肉ぎゅうにくや焼酎しょうちゅうを使つかってもらいました。
宮崎県みやざきけんによると、日本にっぽんの時間じかんで今月こんげつ２５日にちに行おこなわれる今年ことしのパーティーでも宮崎県みやざきけんの牛肉ぎゅうにくを使つかったステーキとハンバーガーや焼酎しょうちゅうが出でます。
宮崎県みやざきけんの河野こうの知事ちじは「２年ねん続つづけてチャンスをもらいました。宮崎県みやざきけんの牛肉ぎゅうにくをたくさんの人ひとに知しってもらいたいです」と話はなしています。
