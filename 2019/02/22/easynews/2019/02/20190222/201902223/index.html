北海道ほっかいどうで震度しんど６弱じゃくの地震じしん　「去年きょねん９月がつの地震じしんと関係かんけいがある」

发表于 2019-02-22 更新于 2019-11-30 分类于 JapanEasyNews



２１日にち午後ごご９時じ２２分ふんごろ、北海道ほっかいどうの胆振いぶり地方ちほうでマグニチュード５．８の地震じしんがありました。
北海道ほっかいどう厚真町あつまちょうでは震度しんど６弱じゃくの大おおきな揺ゆれがありました。震度しんど５強きょうは安平町あびらちょうとむかわ町ちょう、震度しんど５弱じゃくは札幌市さっぽろしの北きた区くと手稲ていね区く、千歳市ちとせし、長沼町ながぬまちょう、平取町びらとりちょうでした。津波つなみはありませんでした。この地震じしんで大おおきなけがをした人ひとはいませんでした。
胆振いぶり地方ちほうでは去年きょねん９月がつ６日むいかにもマグニチュード６．７の地震じしんがあって、厚真町あつまちょうで震度しんど７の揺ゆれがありました。
気象庁きしょうちょうは「２１日にちの地震じしんは去年きょねん９月がつの地震じしんと関係かんけいがあると考かんがえています。大おおきく揺ゆれた場所ばしょでは家いえが壊こわれたり山やまなどが崩くずれたりする心配しんぱいがあります。これからも同おなじぐらい揺ゆれる地震じしんがあるかもしれないため、気きをつけてください」と言いっています。
