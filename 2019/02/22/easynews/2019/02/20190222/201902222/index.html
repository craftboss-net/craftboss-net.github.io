折おって小ちいさくすることができるスマホ　競争きょうそうが激はげしくなりそう

发表于 2019-02-22 更新于 2019-11-30 分类于 JapanEasyNews



韓国かんこくのサムスン電子でんしは２０日はつか、画面がめんを半分はんぶんに折おって小ちいさくすることができる新あたらしいスマートフォンを発表はっぴょうしました。
画面がめんを開ひらいたときはタブレット端末たんまつのように使つかうことができます。本ほんのように半分はんぶんに折おると、今いままでのスマートフォンのように使つかうことができます。半分はんぶんに折おったとき見みていた地図ちずなどは、画面がめんを開ひらくと自動じどうで大おおきくなります。
サムスン電子でんしによると、この新あたらしいスマートフォンは４月がつ２６日にちにアメリカで売うり始はじめます。いちばん値段ねだんが安やすいタイプは日本にっぽんのお金かねで２２万まん円えんぐらいです。
折おって小ちいさくすることができるスマートフォンは今いま、中国ちゅうごくのロヨルという会社かいしゃが売うっています。中国ちゅうごくのファーウェイや、モトローラも売うり始はじめる予定よていで、これから競争きょうそうが激はげしくなりそうです。
