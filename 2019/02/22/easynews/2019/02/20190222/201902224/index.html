日本にっぽんの「はやぶさ２」がリュウグウという星ほしに下おりた

发表于 2019-02-22 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのＪＡＸＡは、地球ちきゅうから３億おくｋｍの所ところにあるリュウグウという小ちいさい星ほしを調しらべるため、４年ねんぐらい前まえに「はやぶさ２」を打うち上あげました。
「はやぶさ２」は２１日にちの午後ごご、高たかさ２万まんｍの所ところからリュウグウに下おり始はじめました。高たかさ５００ｍぐらいの所ところからは、センサーを使つかって高たかさや下おりる場所ばしょを自分じぶんで計算けいさんしながら下おりました。ＪＡＸＡは、「はやぶさ２」は２２日にちの午前ごぜん７時じ半はんごろリュウグウに着ついたと発表はっぴょうしました。
「はやぶさ２」は、リュウグウの表面ひょうめんに弾丸だんがんを発射はっしゃして、細こまかくなった石いしや砂すなを取とります。ＪＡＸＡによると、弾丸だんがんを発射はっしゃできたというデータが届とどきました。
「はやぶさ２」はこれから高たかさ２万まんｍの所ところまで戻もどります。そのあと１回かいか２回かいリュウグウに下おりて、来年らいねん１２月がつに地球ちきゅうに帰かえる予定よていです。

地球
