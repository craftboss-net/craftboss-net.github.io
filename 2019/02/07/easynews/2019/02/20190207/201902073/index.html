「くら寿司ずし」　アルバイトが撮とったビデオについて謝あやまる

发表于 2019-02-07 更新于 2019-11-30 分类于 JapanEasyNews


「くら寿司ずし」の制服せいふくを着きた男性だんせいが魚さかなをまな板いたで切きったあとごみ箱ばこに入いれて、そのあとごみ箱ばこから拾ひろってまたまな板いたの上うえに置おくビデオがインターネットのツイッターに出でました。ビデオを撮とっている人ひとが笑わらっている声こえも入はいっていました。
「くら寿司ずし」の会社かいしゃは６日むいか、ビデオに映うつっている男性だんせいとビデオを撮とった人ひとは、大阪府おおさかふ守口市もりぐちしの店みせで働はたらいているアルバイトだと言いいました。そして、ビデオに映うつっていた魚さかなは、客きゃくに出だしていないと言いいました。
会社かいしゃは「お客様きゃくさまを嫌いやな気持きもちにしてしまって、申もうし訳わけありません。同おなじようなことが起おこらないように、店みせで働はたらく人ひとをしっかり教育きょういくします」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ビデオ

  • 映像えいぞう。
  • 「ビデオテープ」の略りゃく。
  • 映像えいぞうや音おとを、磁気じきテープに記録きろくしたり再生さいせいしたりする装置そうち。

制服

  • 学校がっこうや会社かいしゃなどで決きめられている、形かたちや色いろが同おなじ服装ふくそう。ユニフォーム。

まな板

  • 料理りょうりをするとき、材料ざいりょうをのせて切きったり、刻きざんだりするための厚あつい板いた。

映る

  • 物ものの形かたちやかげなどが、ほかの物ものの表面ひょうめんに現あらわれる。
  • 色いろの取とり合あわせがよい。

申し訳ない

  • 言いい訳わけができない。大変たいへんすまない。

起こる

  • 始はじまる。
  • 生しょうじる。起おきる。
