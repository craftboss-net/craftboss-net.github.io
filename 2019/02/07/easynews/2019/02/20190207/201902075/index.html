アメリカとロシアが核兵器かくへいきを減へらす条約じょうやくをやめると言いう

发表于 2019-02-07 更新于 2019-11-30 分类于 JapanEasyNews


アメリカとロシアは３２年ねん前まえ、核兵器かくへいきを減へらすために条約じょうやくを決きめました。この条約じょうやくで、地上ちじょうから発射はっしゃして５００ｋｍから５５００ｋｍ飛とぶミサイルを持もつことを禁止きんししました。
アメリカは２日ふつか、ロシアが条約じょうやくを守まもらないので、この条約じょうやくをやめると言いいました。ロシアも６日むいか、アメリカと同おなじようにすると言いいました。
アメリカとロシアが考かんがえを変かえないと、この条約じょうやくは今年ことし８月がつに終おわります。多おおくの国くには、アメリカとロシアがまた核兵器かくへいきをたくさん作つくるかもしれないと心配しんぱいしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

核兵器

  • 核分裂かくぶんれつや核融合かくゆうごうによって出でるエネルギーを利用りようした兵器へいき。原子爆弾げんしばくだんや、水素爆弾すいそばくだんなど。

減らす

  • 少すくなくする。

条約

  • 国くにと国くにとの間あいだで決きめ、文章ぶんしょうに書かいた約束やくそく。

地上

  • 地面じめんの上うえ。

発射

  • 鉄砲てっぽうのたまやロケットなどを、打うち出だすこと。

ミサイル

  • ロケットなどの力ちからで飛とび、誘導ゆうどう装置そうちによって、目標もくひょうをとらえる爆弾ばくだん。誘導弾ゆうどうだん。

禁止

  • してはいけないと止とめること。

守る

  • 害がいを受うけないように、防ふせぐ。
  • 決きめたとおりにする。

考え

  • 考かんがえること。また、考かんがえた内容ないよう。
