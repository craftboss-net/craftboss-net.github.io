奈良公園ならこうえん　鹿しかが原因げんいんでけがをした人ひとが増ふえている

发表于 2019-02-07 更新于 2019-11-30 分类于 JapanEasyNews


奈良公園ならこうえんは、野生やせいの鹿しかが自由じゆうに歩あるいている世界せかいでも珍めずらしい公園こうえんで、たくさんの人ひとが観光かんこうに来きます。鹿しかはたいてい静しずかにしていますが、餌えさを見みせながらなかなかあげなかったりすると、怒おこって人ひとを押おしたりします。
奈良県ならけんによると、鹿しかに押おされて転ころんだりしてけがをした人ひとは最近さいきんとても増ふえていて、去年きょねん４月がつから今年ことし１月がつまでに２０９人にんいました。骨ほねが折おれるようなひどいけがをした人ひとは８人にんで、この中なかの５人にんは外国人がいこくじんでした。ひどいけがになった事故じこは、鹿しかが怒おこりやすくなる９月がつから１１月がつに多おおくなっています。
奈良公園ならこうえんの職員しょくいんは「公園こうえんの鹿しかは野生やせいなので、遠とおくから見みていたほうがいいと思おもいます。餌えさをあげるときはすぐにあげて、優やさしくしてください」と話はなしています。
