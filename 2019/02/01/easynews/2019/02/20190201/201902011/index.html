お菓子かしを作つくる技術ぎじゅつで大おおきな絵えのような作品さくひんを作つくる

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


東京都とうきょうと立川市たちかわしにある国際製菓専門学校こくさいせいかせんもんがっこうはお菓子かしやパンの作つくり方かたなどを勉強べんきょうする学校がっこうです。この学校がっこうの学生がくせいたちは毎年まいとし、習ならった技術ぎじゅつを使つかって、大おおきな絵えのような作品さくひんなどを作つくっています。
今年ことしは江戸時代えどじだいの伊藤若冲いとうじゃくちゅうという画家がかの絵えをテーマにしました。学生がくせいたちは、縦たてと横よこが１ｍ２０ｃｍのふすまに、砂糖さとうや卵たまごの白身しろみを使つかって、にわとりやひよこの絵えをかきました。絵えは細こまかい所ところまでとても丁寧ていねいにかいてあります。絵えの背景はいけいは、パンを使つかって金色きんいろにしました。
作つくった学生がくせいは「色いろをつけるのが難むずかしかったですが、うまくできました。たくさんの人ひとに見みてほしいです」と話はなしていました。
この作品さくひんは２月がつ３日みっかまで立川市たちかわしにあるデパートに飾かざってあります。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

作品

  • 作つくったもの。文学ぶんがく・音楽おんがく・美術びじゅつなどで、作者さくしゃが創造そうぞうしたもの。

江戸時代

  • 徳川家康とくがわいえやすが、１６０３年ねんに江戸えどに幕府ばくふを開ひらいてから、１８６７年ねんにほろびるまでの約やく２６０年間ねんかん。鎖国さこくのために日本にっぽん独特どくとくの文化ぶんかが栄さかえた。「徳川時代とくがわじだい」ともいう。

画家

  • 絵えをかくことを仕事しごとにしている人ひと。絵えかき。

テーマ

  • 作品さくひんにかかれていることの中心ちゅうしん。また、作品さくひんの中心ちゅうしんになっている考かんがえ方かた。主題しゅだい。
  • 論文ろんぶんや、演説えんぜつの題目だいもく。
  • 一ひとつの曲きょくの中心ちゅうしんとなっているメロディー。

ふすま

  • 木きの骨組ほねぐみの上うえに両側りょうがわから紙かみや布ぬのを張はった建具たてぐ。ふすま障子しょうじ。からかみ。

白身

  • 卵たまごの身みの、白しろいところ。
  • 魚さかなや肉にくの身みが白しろいところ。

鶏

  • 鳥とりの中なかで、おもに卵たまごや肉にくをとるために、飼かわれるもの。雄おすは、とさかが大おおきい。

ひよこ

  • 鳥とりの子こ。特とくに、ニワトリの子こ。
  • まだ十分じゅうぶんに成長せいちょうせず、幼稚ようちなもの。

背景

  • 絵え・写真しゃしんなどの、後うしろの景色けしき。バック。
  • 演劇えんげきで、舞台ぶたいの後うしろの景色けしき。
  • 裏うらにかくれている事情じじょう。

金色

  • 金きんのように黄色きいろく光ひかっている色いろ。
