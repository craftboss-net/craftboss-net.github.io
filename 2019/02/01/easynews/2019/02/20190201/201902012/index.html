国連こくれん　北朝鮮きたちょうせんは石油せきゆ製品せいひんの輸入ゆにゅうや石炭せきたんの輸出ゆしゅつを続つづけている

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


国連こくれんは、北朝鮮きたちょうせんが核兵器かくへいきやミサイルの実験じっけんをしたあと、北朝鮮きたちょうせんの輸入ゆにゅうや輸出ゆしゅつについての制裁せいさいを決きめました。そして、北朝鮮きたちょうせんについての制裁せいさいが守まもられているかどうか調しらべています。
国連こくれんによると、北朝鮮きたちょうせんの船ふねは去年きょねん１月がつから８月がつまでに１５０回かいぐらい、外国がいこくの船ふねから海うみの上うえでガソリンなど石油せきゆの製品せいひんをもらっていました。そして、国連こくれんが決きめたよりたくさんの石油せきゆの製品せいひんを輸入ゆにゅうしていました。
国連こくれんは、北朝鮮きたちょうせんが石炭せきたんや鉄てつを輸出ゆしゅつすることを禁止きんししています。しかし北朝鮮きたちょうせんは、ロシアの石炭せきたんだとうそを言いって、ロシアの港みなとを通とおって石炭せきたんを輸出ゆしゅつしていました。うそを言いって輸出ゆしゅつした石炭せきたんを売うると、５５８万まんドルぐらいになります。
国連こくれんによると、北朝鮮きたちょうせんはほかにも、近ちかくの海うみで魚さかなをとる権利けんりを中国ちゅうごくの船ふねに売うったりして、制裁せいさいに違反いはんすることをいろいろ行おこなっていました。
