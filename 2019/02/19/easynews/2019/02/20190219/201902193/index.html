滋賀県しがけん甲賀市こうかし　市役所しやくしょの職員しょくいんが忍者にんじゃの服ふくを着きて仕事しごとをする

发表于 2019-02-19 更新于 2019-11-30 分类于 JapanEasyNews



滋賀県しがけん甲賀市こうかしや三重県みえけん伊賀市いがしなどは、２月がつ２２日にちを「忍者にんじゃの日ひ」にしています。そして、昔むかし忍者にんじゃが住すんでいたことをみんなに知しってもらうため、いろいろなことをしています。
甲賀市こうかしの市役所しやくしょでは１８日にちから２２日にちまで、一部いちぶの職員しょくいんが黒くろや青あおの忍者にんじゃの服ふくを着きて仕事しごとをしています。赤あかの忍者にんじゃの服ふくで仕事しごとをしている女性じょせいは「忍者にんじゃのように早はやく仕事しごとができそうです」と話はなしていました。
