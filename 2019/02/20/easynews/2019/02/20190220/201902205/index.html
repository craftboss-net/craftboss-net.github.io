羽田空港はねだくうこうへ行いく京急電鉄けいきゅうでんてつの電車でんしゃの料金りょうきんが１０月がつから安やすくなる

发表于 2019-02-20 更新于 2019-11-30 分类于 JapanEasyNews



京急電鉄けいきゅうでんてつは、羽田空港はねだくうこうと東京とうきょうの中心ちゅうしんや横浜よこはまなどとの間あいだを走はしる電車でんしゃの料金りょうきんを今年ことし１０月がつ１日ついたちから安やすくします。
羽田空港はねだくうこう国内線こくないせんターミナル駅えきと東京とうきょうの品川駅しながわえきとの間あいだは今いま４１０円えんですが、２９０円えんになります。横浜駅よこはまえきとの間あいだは今いま４８０円えんですが、３６０円えんになります。
最近さいきん、羽田空港はねだくうこうを利用りようする外国人がいこくじんなど京急電鉄けいきゅうでんてつの電車でんしゃに乗のる人ひとが増ふえているため、会社かいしゃは料金りょうきんを安やすくすることにしました。来年らいねんは東京とうきょうでオリンピックとパラリンピックがあります。京急電鉄けいきゅうでんてつは「日本にっぽんへ来くる外国人がいこくじんなどにもっと乗のってほしいです」と話はなしています。
