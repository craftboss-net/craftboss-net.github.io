日本にっぽんに住すんでいる外国人がいこくじんの生活せいかつの様子ようすをアンケートで調しらべる

发表于 2019-02-20 更新于 2019-11-30 分类于 JapanEasyNews



４月がつから法律ほうりつが変かわって、外国人がいこくじんが日本にっぽんで働はたらきやすくなるため、日本にっぽんに住すむ外国人がいこくじんが増ふえそうです。国くには、外国人がいこくじんも住すみやすい社会しゃかいにするため、日本にっぽんに住すんでいる外国人がいこくじんの生活せいかつの様子ようすを調しらべることにしました。
日本にっぽんのいろいろな所ところに住すんでいる外国人がいこくじんにアンケートをして、悩なやみや不便ふべんなこと、差別さべつされたかどうかなどについて聞ききます。
いつごろ、何なん人にんぐらいに聞きくかなど詳くわしいことはこれから決きめることにしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

様子

  • ありさま。状態じょうたい。
  • 姿すがた。身みなり。
  • そぶり。気配けはい。
  • わけ。事情じじょう。

アンケート

  • たくさんの人ひとに問とい合あわせ、答こたえを書かいてもらって、人ひとの考かんがえを調しらべる方法ほうほう。

悩み

  • 心こころの苦くるしみ。心配しんぱいごと。

差別

  • あるものとほかのものとの間あいだに差さをつけて、あつかい方かたをちがわせること。

詳しい

  • 細こまかい。
  • 細こまかいことまでよく知しっている。
