奈良県ならけん　顔かおにつけると仏ほとけ様さまのように見みえる美容びようのパック

发表于 2019-02-08 更新于 2019-11-30 分类于 JapanEasyNews


奈良県ならけんには有名ゆうめいな仏像ぶつぞうがたくさんあります。奈良県ならけんにある化粧けしょう品ひんの会社かいしゃは奈良県ならけんと近畿大学きんきだいがくと一緒いっしょに、顔かおにつけると仏ほとけ様さまのように見みえる美容びようのパックをつくりました。
パックは青銅せいどうの色いろになっていて、額ひたいに丸まるい模様もようがかいてあります。仏ほとけ様さまの髪かみに見みえる模様もようもかいてあります。
このパックには、奈良県ならけんの日本酒にほんしゅや柿かきの葉はからつくった化粧けしょう品ひんが入はいっています。日本酒にほんしゅには肌はだを乾かわきにくくする成分せいぶん、柿かきの葉はには肌はだを若わかくする成分せいぶんが入はいっていると言いわれています。
動物どうぶつから取とった成分せいぶんは入はいっていません。会社かいしゃの人ひとは、アレルギーのある人ひとや肉にくや魚さかななどを食たべない人ひとにも使つかってほしいと言いっています。
