ミス?ベネズエラ　雪ゆきまつりで政府せいふに反対はんたいする意見いけんを言いう

发表于 2019-02-08 更新于 2019-11-30 分类于 JapanEasyNews



札幌市さっぽろしでは１１日にちまで、「さっぽろ雪ゆきまつり」を行おこなっています。ミス?インターナショナルの世界せかいの代表だいひょうたちが７日なのか、去年きょねん大おおきな地震じしんがあった北海道ほっかいどうを元気げんきにするために札幌市さっぽろしに来きて、雪ゆきまつりを見みました。
去年きょねん、世界せかい大会たいかいで優勝ゆうしょうしたベネズエラ代表だいひょうのマリエム?ベラスコさんも来きました。ベネズエラでは多おおくの人ひとが、マドゥーロ大統領だいとうりょうは民主的みんしゅてきな政治せいじをしていないと言いって、政府せいふに反対はんたいしています。
ベラスコさんは「私わたしたちは自由じゆうと人権じんけんを守まもるために闘たたかっています。私わたしの仕事しごとは、ベネズエラで起おこっていることを世界中せかいじゅうの人ひとに知しらせることです。私わたしたちは闘たたかい続つづけなければなりません」と話はなしていました。
