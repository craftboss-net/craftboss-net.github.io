子こどもの虐待ぎゃくたい　５年ねん前まえの約やく４倍ばいに増ふえる

发表于 2019-02-08 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねん、親おやなどから虐待ぎゃくたいされた可能性かのうせいがあると警察けいさつが児童相談所じどうそうだんじょに連絡れんらくした子こどもの数かずが、初はじめて８万まん人にん以上いじょうになりました。５年ねん前まえの約やく４倍ばいに増ふえました。
このうち、親おやがひどいことを言いったり子こどもの前まえでけんかをしたりして、子こどもの心こころに傷きずをつける 虐待ぎゃくたいがいちばん多おおい５万まん７３２６人にんで、７０％以上いじょうでした。次つぎは、子こどもが殴なぐられたりする体からだの虐待ぎゃくたいで１万まん４８２１人にんでした。
千葉県ちばけん野田市のだしでは先月せんげつ小学校しょうがっこう４年生ねんせいの女おんなの子こが亡なくなっているのが見みつかって、両親りょうしんが逮捕たいほされました。日本にっぽんでは、子こどもへの虐待ぎゃくたいが大おおきな問題もんだいになっています。
