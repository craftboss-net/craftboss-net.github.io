太平洋戦争たいへいようせんそうで海うみに沈しずんだ日本にっぽんの軍ぐんの船ふね「比叡ひえい」が見みつかる

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


太平洋戦争たいへいようせんそうが続つづいていた１９４２年ねん、日本にっぽんの軍ぐんの船ふね「比叡ひえい」は太平洋たいへいようの南みなみにあるソロモン諸島しょとうの近ちかくでアメリカなどに攻撃こうげきされました。そのあと船ふねは海うみに沈しずみましたが、どうやって沈しずんだかはわかっていません。
太平洋戦争たいへいようせんそうで沈しずんだ船ふねを探さがしているアメリカのチームによると、１月がつ３１日にちにソロモン諸島しょとうの近ちかくの深ふかさ９８５ｍの海うみの底そこで「比叡ひえい」が見みつかりました。このチームは、アメリカのＩＴの会社かいしゃのマイクロソフトを作つくったポール?アレンさんのチームです。
「比叡ひえい」は上うえと下したが反対はんたいになって沈しずんでいました。船ふねの長ながさは２２２ｍですが、見みつかった船ふねは前まえの方ほうの７０ｍぐらいがなくなっていました。このため、大おおきな爆発ばくはつが原因げんいんで沈しずんだのかもしれないと初はじめてわかりました。
チームはこれから船ふねの写真しゃしんなどをみんなに見みせて、研究けんきゅうの役やくに立たてたいと言いっています。
