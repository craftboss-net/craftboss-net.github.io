去年きょねんの西日本にしにほんの雨あめ　１２の川かわで水みずが堤防ていぼうの中なかを通とおって外そとに出でる

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


去年きょねん７月がつ、西日本にしにほんでとてもたくさんの雨あめが降ふって、多おおくの川かわで水みずが増ふえました。
国土交通省こくどこうつうしょうが調しらべると、このとき１２の川かわの２８の場所ばしょで、川かわの水みずが堤防ていぼうの中なかや下したを通とおって外そとに出でていたことがわかりました。これは、堤防ていぼうが壊こわれる前まえに起おこると言いわれています。川かわの水みずが増ふえて水みずの圧力あつりょくがとても強つよくなったことが原因げんいんです。堤防ていぼうに穴あながあいていた川かわもありました。
国土交通省こくどこうつうしょうは、２８の場所ばしょで川かわと堤防ていぼうの間あいだに金属きんぞくの板いたを入いれて、水みずが通とおらないようにする工事こうじなどをします。

堤防

  • 川かわや海うみがあふれないように、川岸かわぎしや海岸かいがんに、石いしやコンクリートなどで築きずいたもの。つつみ。土手どて。

国土交通省

  • 川かわ・道路どうろ・建物たてものなどに関かんする仕事しごとや、交通こうつう・荷物にもつの運送うんそうなどに関かんする仕事しごとをする国くにの役所やくしょ。国交省こっこうしょう。

圧力

  • 物ものをおしつける力ちから。空気くうきの圧力あつりょくを気圧きあつ、水みずの圧力あつりょくを水圧すいあつという。
  • 相手あいてに強つよくはたらきかけること。
