成田空港なりたくうこう　飛行機ひこうきが出発しゅっぱつや到着とうちゃくできる時間じかんを長ながくする

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


成田空港なりたくうこうで飛行機ひこうきが出発しゅっぱつしたり到着とうちゃくしたりできるのは夜よる１１時じまでです。国くにや空港くうこうの会社かいしゃは、２０２０年ねんに東京とうきょうでオリンピックとパラリンピックがあるため、成田空港なりたくうこうを使つかうことができる時間じかんを１時間じかん長ながくしたいと考かんがえました。
空港くうこうの周まわりにあるいくつかの市しや町まちは最初さいしょ、時間じかんを長ながくすることに反対はんたいしました。このため空港くうこうの会社かいしゃは、飛行機ひこうきの音おとが近ちかくの家いえに聞きこえないようにする工事こうじのお金かねの一部いちぶを出だすと言いいました。
４日よっか、国くにと空港くうこうの会社かいしゃ、空港くうこうの周まわりの９つの市しと町まちなどが集あつまって、話はなし合あいました。その結果けっか、今年ことし１０月がつからは１時間じかん長ながくして夜よるの０時じまで使つかうことができることに決きまりました。
