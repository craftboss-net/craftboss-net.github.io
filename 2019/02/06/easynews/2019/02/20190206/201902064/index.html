パスワードを知しるＣＥＯが亡なくなって仮想通貨かそうつうかが下おろせない

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


カナダの「クアドリガＣＸ」は、インターネットの中なかでお金かねと同おなじように使つかう「仮想通貨かそうつうか」を売うったり買かったりする会社かいしゃです。この会社かいしゃをつくったジェラルド?コットンＣＥＯが去年きょねん１２月がつに３０歳さいで突然とつぜん亡なくなりました。
カナダのニュースなどによると、仮想通貨かそうつうかを下おろすことができる状態じょうたいにするためのパスワードは、亡なくなったコットンＣＥＯだけが知しっていました。専門家せんもんかもいろいろ調しらべましたが、うまくいきませんでした。このため、１１万まん５０００人にんの客きゃくの２００億おく円えんぐらいの仮想通貨かそうつうかが、下おろすことができなくなっています。
ニュースは「カナダは仮想通貨かそうつうかの規則きそくが十分じゅうぶんではないことが問題もんだいだ」などと伝つたえています。
