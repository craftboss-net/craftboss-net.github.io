ＡＩロボットが京都きょうとの観光地かんこうちを案内あんないするタクシー

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうで１９日にち、日本にっぽんの魅力みりょく「クールジャパン」を世界せかいに伝つたえている会社かいしゃに賞しょうを贈おくる式しきがありました。
最もっともすばらしい賞しょうは、小ちいさいロボットを使つかってタクシーの客きゃくに観光かんこうの案内あんないをしている京都市きょうとしのタクシー会社がいしゃ「エムケイ」がもらいました。
ロボットにはＡＩが入はいっていて、京都きょうとの観光地かんこうちを声こえで案内あんないしたり、客きゃくからの質問しつもんに答こたえたりすることができます。賞しょうを贈おくった団体だんたいは、日本にっぽんのロボットやＡＩの技術ぎじゅつをうまく使つかっていてすばらしいと言いっています。
タクシー会社がいしゃの人ひとは「ロボットは外国がいこくからのお客きゃくさんにもとても人気にんきがあります。このタクシーに乗のって、日本にっぽんをもっと好すきになってもらいたいです」と話はなしていました。
