滋賀県しがけん彦根市ひこねし　江戸時代えどじだいのひな人形にんぎょうの展覧会てんらんかい

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



滋賀県しがけん彦根市ひこねしにある彦根城ひこねじょう博物館はくぶつかんで、昔むかしのひな人形にんぎょうの展覧会てんらんかいを開ひらいています。展覧会てんらんかいでは、江戸時代えどじだいに彦根城ひこねじょうに住すんでいた大名だいみょうの娘むすめが結婚けっこんするときに持もって行いったひな人形にんぎょうや道具どうぐなどを２００ぐらい見みることができます。
江戸時代えどじだいの始はじめごろの高たかさ３０ｃｍぐらいのひな人形にんぎょうは、顔かおが丸まるくて、着物きものが和紙わしでできています。ひな人形にんぎょうと一緒いっしょに飾かざる小ちいさなおわんや重箱じゅうばこなどの道具どうぐには、金きんの模様もようがあります。
博物館はくぶつかんの人ひとは「細こまかいところまできれいに作つくってある道具どうぐをぜひ見みてください」と話はなしていました。この展覧会てんらんかいは３月がつ４日よっかまでです。

江戸時代

  • 徳川家康とくがわいえやすが、１６０３年ねんに江戸えどに幕府ばくふを開ひらいてから、１８６７年ねんにほろびるまでの約やく２６０年間ねんかん。鎖国さこくのために日本にっぽん独特どくとくの文化ぶんかが栄さかえた。「徳川時代とくがわじだい」ともいう。
