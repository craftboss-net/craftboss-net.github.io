「学校がっこうに通かようことができない子こどもは２億おく６０００万まん人にん」

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



イギリスの首相しゅしょうだったブラウンさんは、今いまは国連こくれんの教育きょういく特使とくしで、世界せかいの教育きょういくをよくするための仕事しごとをしています。
ブラウンさんは１９日にち、「難民なんみんになった子こどもの中なかで、小学校しょうがっこうの教育きょういくを受うけることができる子こどもは２０％だけです。ギリシャでシリアからの難民なんみんなどがたくさん住すんでいる所ところでは、１０歳さいぐらいの２人ふたりの男おとこの子こが自殺じさつしようとしました。難民なんみんたちが住すんでいる所ところでも教育きょういくを行おこなうことが必要ひつようです」と言いいました。
そして「学校がっこうに通かようことができない子こどもは今いま、世界せかいに２億おく６０００万まん人にんもいます。世界せかいの国くには、２０３０年ねんまでにみんなが十分じゅうぶんな教育きょういくを受うけることができるようにすると決きめました。しかし、このままでは２０３０年ねんには２億おく人にんの子こどもが学校がっこうに通かようことができません」と言いいました。
ブラウンさんは、子こどもの教育きょういくのために、世界せかいの国くにはもっと協力きょうりょくしてほしいと言いっています。
