徳島県とくしまけんで８５年ねん前まえの「阿波あわおどり」のビデオが見みつかる

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



「阿波あわおどり」は徳島とくしまの有名ゆうめいな夏なつの祭まつりです。徳島とくしま県立けんりつ文書館もんじょかんが、阿南市あなんしに住すんでいる女性じょせいからもらった古ふるいビデオを調しらべると、８５年ねんぐらい前まえの「阿波あわおどり」などがうつっていました。
ビデオは全部ぜんぶで１４分ぷんです。三味線しゃみせんを弾ひいている女性じょせいや踊おどっている男性だんせいがうつっています。今いまの「阿波あわおどり」とは違ちがう形かたちのかさをかぶって、派手はでな着物きものを着きている人ひとたちもうつっています。祭まつりのほかには、８５年ねん前まえの大おおきな台風たいふうのあとの川かわや強つよい風かぜで倒たおれた列車れっしゃなどもありました。
徳島とくしま県立けんりつ文書館もんじょかんはこのビデオをみんなに見みせる予定よていです。館長かんちょうは「昔むかしの徳島とくしまのことがわかるビデオはとても珍めずらしいです。たくさんの人ひとに見みてほしいです」と話はなしています。
