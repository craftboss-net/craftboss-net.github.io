「未来みらいのミライ」がアメリカでアニメ映画えいがの賞しょうをもらう

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


アニメの「アカデミー賞しょう」と言いわれている「アニー賞しょう」は、アニメのすばらしい映画えいがや監督かんとくなどに贈おくる賞しょうで、１９７２年ねんに始はじまりました。
アメリカで２日ふつか、日本にっぽんの細田守ほそだまもる監督かんとくの「未来みらいのミライ」が、アニー賞しょうの中なかの「インディペンデント作品さくひん賞しょう」をもらいました。この賞しょうは、アメリカで上映じょうえいした映画館えいがかんが少すくない映画えいがの中なかから選えらばれます。細田ほそだ監督かんとくは５１歳さいで、今いままでも外国がいこくの映画えいがのイベントなどで賞しょうをもらっています。
「未来みらいのミライ」は４歳さいの男おとこの子こが、未来みらいから来きた妹いもうとに会あっていろいろな経験けいけんをしながら、家族かぞくの愛あいを知しる話はなしです。
細田ほそだ監督かんとくは「僕ぼくの２人ふたりの子こどもと一緒いっしょにいて、小ちいさな子こどもと過すごすことは本当ほんとうにすばらしいと思おもって、この映画えいがを作つくりました」と話はなしました。
