音楽おんがくが聞きこえる歯はブラシができた「子こどもに使つかってほしい」

发表于 2019-07-05 更新于 2019-11-30 分类于 JapanEasyNews



歯はを磨みがいているときに音楽おんがくが聞きこえる歯はブラシができました。この歯はブラシはスマートフォンとつないで使つかいます。歯はブラシが歯はに当あたると、スマートフォンで選えらんだ音楽おんがくが、口くちの中なかで鳴なります。
京きょうセラとソニーとライオンの３つの会社かいしゃが一緒いっしょにこの歯はブラシを作つくりました。いろいろな会社かいしゃの技術ぎじゅつを集あつめて、新あたらしい物ものや技術ぎじゅつを作つくることは、これからの時代じだいに必要ひつようだと言いわれています。
３つの会社かいしゃは、３日みっかからウェブサイトでこの歯はブラシを紹介しょうかいして、どのくらい売うれるかを考かんがえます。そして、売うるかどうかを決きめます。
歯はブラシを作つくった人ひとは「違ちがう技術ぎじゅつを持もった会社かいしゃが協力きょうりょくして、今いままでになかった物ものを作つくることができました。歯はを磨みがくのが嫌きらいな子こどもにぜひ使つかってほしいです」と話はなしています。
