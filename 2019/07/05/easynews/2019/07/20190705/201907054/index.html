東京とうきょうオリンピックの組織委員会そしきいいんかい「２回かい目めの抽ちゅうせんをします」

发表于 2019-07-05 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックの組織委員会そしきいいんかいは先月せんげつ、チケットの抽ちゅうせんを行おこないました。大勢おおぜいの人ひとが申もうし込こんだので、当あたらなかった人ひとがたくさんいました。
このため組織委員会そしきいいんかいは、１枚まいも当あたらなかった人ひとのために、２回かい目めの抽ちゅうせんをすると発表はっぴょうしました。
この抽ちゅうせんで売うるのは、最初さいしょの抽ちゅうせんで申もうし込こみが少すくなかったチケットや、当あたった人ひとがお金かねを払はらわなかったチケットです。２回かい目めの抽ちゅうせんは、８月がつに行おこなう予定よていです。申もうし込こむことができるのは、１人ひとり１種類しゅるいだけです。
組織委員会そしきいいんかいは今年ことしの秋あきから、早はやく申もうし込こんだ人ひとから順番じゅんばんにチケットを売うる予定よていでしたが、抽ちゅうせんにすることを考かんがえています。
