フランスで日本にっぽんの漫画まんがやアニメを紹介しょうかいするイベントが始はじまる

发表于 2019-07-05 更新于 2019-11-30 分类于 JapanEasyNews



漫画まんがやアニメなどの新あたらしい日本にっぽんの文化ぶんかを紹介しょうかいする「ジャパンエキスポ」が４日よっか、フランスで始はじまりました。
ステージでは、東京とうきょうのバレエのチームが人気にんき ゲームの「ドラゴンクエスト」をテーマにしたバレエを踊おどりました。見みに来きた２６歳さいの男性だんせいは、「ドラゴンクエストが大好だいすきです。すばらしい踊おどりでゲームの世界せかいを表あらわしていました」と話はなしていました。
会場かいじょうでは東京都とうきょうとや鎌倉市かまくらしなどが観光地かんこうちを紹介しょうかいするパンフレットを配くばっていました。
このイベントは７日なのかまでで、２４万まん人にんぐらいが来くると考かんがえられています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

アニメーション

  • かいた絵えや人形にんぎょうを、動うごきに従したがって一ひとこま一ひとこま撮影さつえいし、それを映うつして実際じっさいに動うごいているように見みせる映画えいが。動画どうが。アニメ。

イベント

  • 行事ぎょうじ。できごと。
  • 勝負しょうぶ。試合しあい。

ステージ

  • 舞台ぶたい。

バレエ

  • 西洋せいようのおどり。歌うたやことばを使つかわずに、音楽おんがくに合あわせて、身みぶりや、おどりだけで物語ものがたりを表あらわす。

チーム

  • 競技きょうぎや仕事しごとをするときの、組くみや団体だんたい。

人気

  • 世よの中なかの人ひとたちのよい評判ひょうばん。

ゲーム

  • 勝かち負まけを争あらそう遊あそび。
  • 試合しあい。

テーマ

  • 作品さくひんにかかれていることの中心ちゅうしん。また、作品さくひんの中心ちゅうしんになっている考かんがえ方かた。主題しゅだい。
  • 論文ろんぶんや、演説えんぜつの題目だいもく。
  • 一ひとつの曲きょくの中心ちゅうしんとなっているメロディー。

表す

  • 気持きもちや考かんがえなどを、ことばや表情ひょうじょうに出だしたり、絵えや音楽おんがくなどにしたりする。

観光地

  • 景色けしきが美うつくしかったり、名所めいしょがあったりして、多おおくの人々ひとびとが見物けんぶつに集あつまる所ところ。

パンフレット

  • 簡単かんたんにとじた、うすい本ほん。パンフ。

配る

  • 分わけて、わたす。
  • 気持きもちを行いきわたらせる。
