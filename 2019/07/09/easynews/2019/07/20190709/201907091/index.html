日本語にほんごの教育きょういくが必要ひつような子こどものために「支援員しえんいん」を増ふやす

发表于 2019-07-09 更新于 2019-11-30 分类于 JapanEasyNews



外国がいこくで育そだったりしたため、日本語にほんごの教育きょういくが必要ひつような公立こうりつの高校こうこうの生徒せいとは、２０１７年度ねんどに約やく４０００人にんいました。しかし、この中なかの９．６％が途中とちゅうで高校こうこうをやめました。
９．６％は、公立こうりつの高校こうこうの全部ぜんぶの生徒せいとの中なかで、途中とちゅうで高校こうこうをやめた人ひとの割合わりあいの７倍ばいです。文部科学省もんぶかがくしょうは、授業じゅぎょうがよくわからなくてやめた人ひとも多おおいと考かんがえています。
文部科学省もんぶかがくしょうは、日本語にほんごの教育きょういくが必要ひつような高校生こうこうせいのために、「支援員しえんいん」を増ふやすことにしました。支援員しえんいんは、授業じゅぎょうの内容ないようをわかりやすいことばで翻訳ほんやくして説明せつめいします。
日本にっぽんでは、働はたらく外国人がいこくじんを多おおくしているため、同おなじように日本語にほんごの教育きょういくが必要ひつような小学生しょうがくせいや中学生ちゅうがくせいが増ふえそうです。このため文部科学省もんぶかがくしょうは、支援員しえんいんを増ふやしたり、たくさんのことばで翻訳ほんやくできるシステムを使つかったりしたいと考かんがえています。

国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

  • 多おおくする。

育つ

  • 生いき物ものが、大おおきくなる。
  • 一人前いちにんまえになる。

公立

  • 都道府県とどうふけんや市町村しちょうそんなどが運営うんえいすること。

年度

  • 仕事しごとのつごうで決きめた１年いちねんの期間きかん。ふつう４月がつ１日ついたちに始はじまり、翌年よくねんの３月がつ３１日にちに終おわる。

  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。

割合

  • 物ものと物ものとの関係かんけいを、数かずで表あらわしたもの。割わり。率りつ。歩合ぶあい。
  • 思おもいのほか。比較的ひかくてき。

文部科学省

  • 教育きょういくや文化ぶんかについての仕事しごとや科学技術かがくぎじゅつを発展はってんさせる仕事しごとをする国くにの役所やくしょ。文科省もんかしょう。

内容

  • 文章ぶんしょうや話はなしによって、表あらわされたもの。

小学生
