富士山ふじさん　山梨県やまなしけんから頂上ちょうじょうまで登のぼることができるようになる

发表于 2019-07-09 更新于 2019-11-30 分类于 JapanEasyNews



富士山ふじさんは山梨県やまなしけんと静岡県しずおかけんの間あいだにあります。山梨県やまなしけんから登のぼる道みちは１日ついたちにオープンしましたが、崩くずれた石いしで３４５０ｍより上うえに登のぼることができませんでした。
９日ここのかの午後ごご、簡単かんたんな工事こうじが終おわって、頂上ちょうじょうまで登のぼることができるようになりました。工事こうじがまだ全部ぜんぶ終おわっていないため、道みちはいつもより狭せまくなっています。山梨県やまなしけんは「道みちには案内あんないする人ひとがいるので、その人ひとの言いうとおりにして、安全あんぜんに登のぼってください」と言いっています。
１０日とおかに静岡県しずおかけんから登のぼる道みちもオープンします。
近ちかくの旅館りょかんの人ひとは「これから山やまに登のぼるのにいい季節きせつなので、たくさんの人ひとに来きてほしいです」と話はなしています。
