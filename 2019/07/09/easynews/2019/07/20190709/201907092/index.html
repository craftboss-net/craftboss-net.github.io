働はたらき方かたが変かわってノートパソコンの生産せいさんが増ふえている

发表于 2019-07-09 更新于 2019-11-30 分类于 JapanEasyNews



今年ことし３月がつまでの１年ねんに工場こうじょうで生産せいさんして店みせに出だしたパソコンは７３９万まん８０００台だいでした。パソコンの生産せいさんは、４年ねん続つづけて前まえの年としより少すくなくなっていましたが、久ひさしぶりに前まえの年としより９．３％増ふえました。
デスクトップは前まえの年としより３．９％、ノートパソコンは１１．２％増ふえました。日本にっぽんの政府せいふは、会社かいしゃで長ながい時間じかん働はたらいて自分じぶんの生活せいかつの時間じかんがなくなるような働はたらき方かたをやめようと言いっています。会社かいしゃ以外いがいの場所ばしょで仕事しごとができるようにノートパソコンを買かう人ひとが増ふえているようです。
パソコンを作つくっている会社かいしゃは、運はこびやすくするためパソコンを小ちいさく軽かるくしています。そして、インターネットを使つかった会議かいぎで使つかいやすいように、マイクを増ふやしたり、スピーカーをよくしたりしています。
