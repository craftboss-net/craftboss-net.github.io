「友達ともだちの『投票とうひょうしたよ』というスタンプを見みて投票とうひょうに行いって」

发表于 2019-07-19 更新于 2019-11-30 分类于 JapanEasyNews



参議院さんぎいんの議員ぎいんを選えらぶ投票とうひょうが２１日にちにあります。インターネットのインスタグラムの会社かいしゃは、若わかい人ひとに投票とうひょうに行いってほしいと考かんがえて、投票とうひょうしたことを友達ともだちに知しらせるスタンプを作つくりました。
インスタグラムに友達ともだちに見みせる写真しゃしんを出だすとき、「投票とうひょう完了かんりょう」という文字もじや「投票とうひょう箱ばこ」の絵えなどのスタンプを選えらびます。スタンプは２４時間じかん出でていて、自分じぶんが投票とうひょうしたことを家族かぞくや友達ともだちなどに知しらせることができます。
「友達ともだちがやったことを自分じぶんもやってみたい」と思おもう人ひとは多おおいと言いわれています。会社かいしゃの人ひとは「たくさんの若わかい人ひとが『友達ともだちも投票とうひょうしたから私わたしもしよう』と思おもって、投票とうひょうに行いってほしいです」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

投票

  • 選挙せんきょや採決さいけつで、選えらびたい人ひとの名前なまえや、賛成さんせいか反対はんたいかを、紙かみに書かいて出だすこと。

参議院

  • 国会こっかいの議院ぎいんの一ひとつ。衆議院しゅうぎいんで決きまった予算よさんや法律ほうりつなどを、もう一度いちど検討けんとうするところ。解散かいさんはない。

議員

  • 選挙せんきょで選えらばれ、国会こっかいや地方ちほうの議会ぎかいで、政治せいじに関かんすることをいろいろ相談そうだんする人ひと。国会議員こっかいぎいん・県議会議員けんぎかいぎいんなど。

完了

  • すっかり終おわること。

文字

  • ことばを書かき表あらわすための記号きごう。もんじ。字じ。日本にっぽんでは、かたかな・ひらがな・漢字かんじ・アルファベットや数字すうじを使つかう。
