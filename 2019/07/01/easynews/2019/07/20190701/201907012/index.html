ＮＡＳＡ　土星どせいを回まわる星ほしにドローンのような探査たんさ機きを送おくる

发表于 2019-07-01 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのＮＡＳＡは２７日にち、「２０３４年ねんに、土星どせいの周まわりを回まわっている星ほしの中なかでいちばん大おおきいタイタンに、探査たんさ機きを送おくる」と発表はっぴょうしました。
探査たんさ機きは「ドラゴンフライ」という名前なまえです。ドローンのように空そらを飛とんだり、タイタンに下おりたりすることができます。
タイタンの周まわりにある空気くうきの層そうはとても厚あつくて、液体えきたいになったメタンガスが川かわや湖みずうみなど地球ちきゅうのような景色けしきをつくっていると考かんがえられています。そして有機物ゆうきぶつがたくさんあって、生いき物ものに必要ひつようなものが全部ぜんぶあると言いわれています。
ＮＡＳＡは２０２６年ねんに探査たんさ機きを打うち上あげて、タイタンには２０３４年ねんに着つく計画けいかくです。
そのあと２年ねん半はん以上いじょう、探査たんさ機きはタイタンで写真しゃしんを撮とったり、空気くうきや有機物ゆうきぶつを細こまかく調しらべたりします。ＮＡＳＡはタイタンを調しらべて、地球ちきゅうの生いき物ものがどうやって生うまれたか知しりたいと考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

NASA

  • 宇宙開発うちゅうかいはつのために作つくられた、アメリカ政府せいふの機関きかん。スペースシャトルの開発かいはつなどをする。

土星

  • 惑星わくせいの一ひとつ。太陽たいようから六番ろくばんめの星ほし。小ちいさな星ほしの集あつまりが周まわりを回まわっているので、輪わに囲かこまれたように見みえる。

探査

  • 探さがして調しらべること。

機

  • 仕しかけ。仕組しくみ。
  • とき。きっかけ。
  • 心こころの動うごき。
  • はたらき。
  • だいじな点てん。
  • 「飛行機ひこうき」の略りゃく。また、飛行機ひこうきを数かぞえることば。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

層

  • 重かさなり。
  • 建物たてものの階かい。
  • 人々ひとびとを年齢ねんれいや階級かいきゅうなどによって分わけた集あつまり。

液体

  • 水みずや油あぶらのように、決きまった形かたちがなく、流ながれる性質せいしつのあるもの。

メタンガス

  • 無色無臭むしょくむしゅうの気体きたい。天然てんねんガスに多おおくふくまれ、うす青色あおいろのほのおを出だして燃もえる。都市としガスの原料げんりょうなどにする。

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから３番ばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

有機物

  • 生いき物ものの体からだを作つくっている物質ぶっしつ。また、生いき物ものが体内たいないで作つくり出だす物質ぶっしつ。

生き物

  • 生いきているもの。生物せいぶつ。特とくに、動物どうぶつのこと。

打ち上げる

  • （勢いきおいよく）物ものを空中くうちゅうに上あげる。
  • 波なみが物ものを陸りくに運はこび上あげる。
  • （仕事しごとなどを）すっかり終おえる。
