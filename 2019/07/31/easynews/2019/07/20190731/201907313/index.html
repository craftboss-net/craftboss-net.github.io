ジンベエザメのロープをダイバーが切きって助たすける

发表于 2019-07-31 更新于 2019-11-30 分类于 JapanEasyNews



７月がつ２９日にち、沖縄県おきなわけんの西表島いりおもてじまの海うみで、体からだにロープを巻まいたまま泳およぐ大おおきな魚さかなをダイバーの大工だいく泰幸やすゆきさんが見みつけました。この魚さかなは７ｍぐらいのジンベエザメでした。太ふといロープが体からだをきつく 巻まいていて、外はずれなくなっていました。
大工だいくさんは、船ふねからロープを切きる道具どうぐを取とってきました。そして、ほかのダイバーと一緒いっしょにジンベエザメのそばを泳およぎながら、ロープを少すこしずつ切きりました。ロープが体からだから外はずれると、ジンベエザメは泳およいでいきました。
大工だいくさんは「ジンベエザメが『助たすけてください』と言いっているように見みえました。助たすけることができてよかったです」と話はなしていました。
