長野県ながのけん佐久市さくしが「北斗ほくとの拳けん」のマンホールのふたを作つくる

发表于 2019-07-31 更新于 2019-11-30 分类于 JapanEasyNews



人気にんきのアクション漫画まんが「北斗ほくとの拳けん」の作者さくしゃの武論尊ぶろんそんさんは、長野県ながのけん佐久市さくしの出身しゅっしんです。佐久市さくしは「北斗ほくとの拳けん」の主人公しゅじんこうや恋人こいびとの絵えをかいた７種類しゅるいのマンホールのふたを作つくりました。
佐久市さくしはこのふたを９月がつに佐久平さくだいら駅えきの近ちかくに置おく予定よていです。佐久市さくしはこのふたの絵えのカードなどを作つくって、観光かんこうに来くる人ひとを集あつめたいと考かんがえています。
武論尊ぶろんそんさんは「少すこしでもふるさとの役やくに立たつことができたらうれしいです」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

マンホール

  • 地下ちかの下水道げすいどうなどに出入でいりするための穴あな。入いり口ぐちに鉄てつやコンクリートのふたがある。

蓋

  • 入いれ物ものの口くちをおおうもの。

人気

  • 世よの中なかの人ひとたちのよい評判ひょうばん。

アクション

  • 体からだの動うごき。行動こうどう。活動かつどう。
  • 俳優はいゆうの演技えんぎ。特とくに、はげしい動うごき。

作者

  • 物語ものがたりや詩し、また絵え・陶器とうきなどの作品さくひんを作つくった人ひと。

出身

  • その土地とちで生うまれたこと。
  • その学校がっこうを卒業そつぎょうしたこと。

主人公

  • 小説しょうせつや劇げきなどの、中心ちゅうしんになる人物じんぶつ。

恋人

  • とても好すきで、こいしく思おもっている相手あいて。

種類

  • 共通きょうつうする形かたちや性質せいしつによって分わけたもの。

カード

  • 紙かみを、四角しかくに小ちいさく切きったもの。記入きにゅうしたり、整理せいりするときなどに使つかう。
  • 「クレジットカード」「キャッシュカード」などの略りゃく。
  • トランプ（の札ふだ）。
  • 野球やきゅうなどの、試合しあいの組くみ合あわせ。

観光

  • 景色けしきのよい所ところや名所めいしょなどを見物けんぶつして回まわること。

ふるさと

  • 自分じぶんが生うまれ育そだった所ところ。故郷こきょう。
