日本にっぽんの人口じんこうが１０年ねん続つづけて減へった

发表于 2019-07-11 更新于 2019-11-30 分类于 JapanEasyNews



総務省そうむしょうによると、今年ことし１月がつ１日ついたちに日本にっぽんに住すんでいる日本人にっぽんじんは１億おく２４７０万まん人にんでした。１０年ねん続つづけて人口じんこうが減へりました。
東京とうきょうは７万まん人にん増ふえて、２３年ねん続つづけて増ふえています。しかし、名古屋なごやと関西かんさいの都市としでは減へっています。東京とうきょうに今いままでよりもっと多おおくの人ひとが集あつまっています。
専門家せんもんかは「２５歳さいから３９歳さいの女性じょせいが２００８年ねんからの１０年ねんで２１％も減へりました。子こどもを産うむ年齢ねんれいの女性じょせいが減へっていることが、人口じんこうが減へったことのいちばん大おおきな原因げんいんです。これからも人口じんこうが減へると考かんがえて、社会しゃかいや経済けいざいをどうするのか考かんがえることが必要ひつようです」と話はなしています。
日本にっぽんに住すむ外国人がいこくじんは２６６万まん人にんでした。外国人がいこくじんの人口じんこうは増ふえていて、日本にっぽんに住すんでいる人ひとの中なかで外国人がいこくじんの割合わりあいが初はじめて２％以上いじょうになりました。
