「はやぶさ２」がまたリュウグウという星ほしに下おりた

发表于 2019-07-11 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのＪＡＸＡが打うち上あげた「はやぶさ２」は、去年きょねん６月がつに地球ちきゅうと火星かせいの間あいだにあるリュウグウという小ちいさい星ほしに着ついて、この星ほしを調しらべています。
ＪＡＸＡは、「はやぶさ２」が１１日にち午前ごぜん１０時じ過すぎにリュウグウに下おりたと発表はっぴょうしました。「はやぶさ２」は高たかさ２万まんｍの所ところからリュウグウに下おり始はじめました。下おりる場所ばしょにはボールのような目印めじるしを落おとしておきました。そして、この目印めじるしをカメラで見みながら下おりました。
「はやぶさ２」は今年ことし２月がつにもリュウグウに下おりて、地面じめんの石いしを取とりました。今年ことし４月がつには地面じめんに金属きんぞくの塊かたまりを落おとして、穴あなを作つくりました。１１日にちはこの穴あなの近ちかくに下おりました。「はやぶさ２」は、穴あなを作つくったときに地面じめんの中なかから出でた石いしを取とって、地球ちきゅうへ持もって帰かえる予定よていです。
地面じめんの中なかの石いしは、宇宙うちゅうからの影響えいきょうがないため、地球ちきゅうなどの星ほしがどうやってできたか知しることができるかもしれません。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

打ち上げる

  • （勢いきおいよく）物ものを空中くうちゅうに上あげる。
  • 波なみが物ものを陸りくに運はこび上あげる。
  • （仕事しごとなどを）すっかり終おえる。

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから三番さんばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

火星

  • 惑星わくせいの一ひとつ。太陽たいようから四番よばんめの星ほし。星ほしの色いろは赤あかく見みえ、衛星えいせいが二ふたつある。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

目印

  • 見みつけやすいようにつけるしるし。目当めあて。目標もくひょう。

地面

  • 土地とちの表面ひょうめん。土つちの上うえ。地上ちじょう。地じべた。
  • 土地とち。地所じしょ。

金属

  • ふつうは固体こたいで、熱ねつや電気でんきをよく伝つたえるもの。鉄てつ・銅どう・金きん・銀ぎん・アルミニウムなど。

固まり・

  • かたまったもの。
  • 集あつまり。群むれ。
  • 心こころをそのことだけに向むける。

穴

  • くぼんだ所ところ。
  • 向むこうまでつきぬけている部分ぶぶん。
  • 弱よわいところ。
  • 損そん。不足ふそく。

宇宙

  • 地球ちきゅう・太陽たいよう・星ほしなどのある、果はてしなく広ひろい空間くうかんのこと。地球ちきゅうは太陽たいようを中心ちゅうしんにして銀河系宇宙ぎんがけいうちゅうにあり、この銀河系宇宙ぎんがけいうちゅうのようなものがたくさん集あつまって宇宙うちゅうを作つくっている。

影響

  • あるものごとが、ほかのものごとに力ちからをおよぼして、そのようすを変かえること。
