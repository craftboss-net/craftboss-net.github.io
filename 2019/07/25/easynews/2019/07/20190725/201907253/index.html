安倍あべ総理大臣そうりだいじんがハンセン病びょうになった人ひとの家族かぞくに会あって謝あやまる

发表于 2019-07-25 更新于 2019-11-30 分类于 JapanEasyNews



ハンセン病びょうになった人ひとの家族かぞくは、国くにが間違まちがった考かんがえ方かたでハンセン病びょうの人ひとを療養りょうよう所じょに入いれ続つづけたため家族かぞくも差別さべつされたと言いって、裁判さいばんに訴うったえました。裁判所さいばんしょは先月せんげつ２８日にち、家族かぞくが差別さべつされたのは国くにに責任せきにんがあると言いいました。
安倍あべ総理大臣そうりだいじんは２４日にじゅうよっか、家族かぞくの人ひとたちに会あって「長ながい間あいだ、皆みな様さまを苦くるしませてしまいました。政府せいふを代表だいひょうして、心こころから深ふかくおわびします」と言いって謝あやまりました。そして「裁判さいばんに参加さんかしなかった人ひとにも補償ほしょうのお金かねを払はらうための新あたらしい法律ほうりつを作つくります」と話はなしました。
国くにを訴うったえた林はやし力ちからさんは「長ながい間あいだハンセン病びょうの人ひとを差別さべつしてきた社会しゃかいが変かわるのは簡単かんたんではありません。国くには間違まちがった考かんがえ方かたがなくなるように努力どりょくしてほしいです」と安倍あべ総理大臣そうりだいじんに言いいました。
