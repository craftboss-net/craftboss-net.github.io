「世界遺産せかいいさんになった昔むかしの人ひとの墓はかを見みせてほしい」

发表于 2019-07-25 更新于 2019-11-30 分类于 JapanEasyNews



大阪府おおさかふにある昔むかしの偉えらい人ひとたちの墓はか「百舌鳥もず?古市ふるいち古墳群こふんぐん」は、今月こんげつユネスコの世界文化遺産せかいぶんかいさんになりました。
「百舌鳥もず?古市ふるいち古墳群こふんぐん」の中なかには、昔むかしの天皇てんのうの墓はかだと言いわれている墓はかもあります。この墓はかは宮内庁くないちょうが管理かんりしていて、中なかに入はいったり見みたりすることができません。古ふるい歴史れきしなどを研究けんきゅうしている人ひとたちの団体だんたいが２３日にち、墓はかを誰だれでも見みることができるようにしてほしいと言いいました。
団体だんたいの人ひとは「今いまのままでは昔むかしの人ひとの墓はかについて勉強べんきょうすることができません。少すこしでも見みることができるようにしてほしいです」と話はなしています。

世界遺産

  • ユネスコの「世界せかいの文化遺産ぶんかいさん及および自然遺産しぜんいさんの保護ほごに関かんする条約じょうやく」（「世界遺産保護条約せかいいさんほごじょうやく」）にもとづいて決きめられた、世界的せかいてきに残のこす価値かちがあると認みとめられた文化ぶんかや自然しぜん。日本にっぽんでは、文化遺産ぶんかいさんとして姫路城ひめじじょうや法隆寺ほうりゅうじ・沖縄おきなわの首里城しゅりじょうなど、自然遺産しぜんいさんとして屋久島やくしまや白神山地しらかみさんちなどが指定していされている。
