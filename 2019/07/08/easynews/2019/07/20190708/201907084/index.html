地震じしんがあった新潟県にいがたけん村上市むらかみし　お祭まつりに大勢おおぜいの人ひとが来きた

发表于 2019-07-08 更新于 2019-11-30 分类于 JapanEasyNews



新潟県にいがたけん村上市むらかみしは、先月せんげつ１８日にちに大おおきな地震じしんがあって、家いえが壊こわれたりけがをしたりした人ひとがいました。それから３週間しゅうかんぐらい過すぎて、「村上大祭むらかみたいさい」というお祭まつりがありました。
このお祭まつりは、３８０年ねん以上いじょう前まえ、神社じんじゃの場所ばしょが移うつることを祝いわって木きの車くるまに太鼓たいこをのせて町まちを歩あるいたことから始はじまりました。
７日なのかは、馬うまに乗のった武士ぶしや昔むかしの人ひとの格好かっこうで歩あるく子こどもたちに続つづいて、１９台だいの山車だしが引ひかれて行いきました。彫刻ちょうこくで飾かざった立派りっぱな山車だしには子こどもが乗のって太鼓たいこをたたいたりしました。
大勢おおぜいの人ひとがお祭まつりを見みに来きました。新潟市にいがたしから来きた人ひとは「朝あさ早はやくからたくさんの山車だしが出でてすごいお祭まつりだと思おもいます」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

祝う

  • めでたいことを喜よろこび、ことばや行おこないに表あらわす。
  • 幸しあわせであるようにいのる。

太鼓

  • 打楽器だがっきの一ひとつ。木きや金属きんぞくで作つくった胴どうに皮かわを張はり、ばちで打うち鳴ならすもの。

載せる

  • 上うえに置おく。
  • 新聞しんぶんや雑誌ざっしなどの記事きじにする。

馬

  • 家畜かちくとして人ひとに飼かわれる動物どうぶつ。力ちからが強つよく、走はしるのがはやい。荷物にもつの運搬うんぱんや農業のうぎょう、競馬けいばなどに使つかわれる。

武士

  • 昔むかし、武芸ぶげいを身みにつけて、いくさで戦たたかった人ひと。さむらい。

格好・

  • 姿すがた。形かたち。
  • ていさい。
  • ちょうど合あっているようす。

山車

  • 祭まつりのとき、いろいろなかざり物ものをつけて引ひいて歩あるく大おおきな車くるま。

彫刻

  • 木き・石いし・金属きんぞくなどを、ほったりけずったりして、いろいろな形かたちをつくること。また、その作品さくひん。

たたく

  • ぶつ。なぐる。
  • 打うって音おとを出だす。
  • 悪口わるくちを言いって、やっつける。
  • やたらにしゃべる。
  • 値段ねだんをまけさせる。
