夏なつに外国がいこくへ旅行りょこうに行いく人ひと　今いままででいちばん多おおい２９９万まん人にん

发表于 2019-07-08 更新于 2019-11-30 分类于 JapanEasyNews



旅行会社りょこうがいしゃのＪＴＢは、ツアーに予約よやくしている人ひとの数かずなどから今年ことしの夏なつに外国がいこくへ旅行りょこうに行いく人ひとの数かずを計算けいさんしました。その結果けっか、今月こんげつ１５日にちから来月らいげつ３１日にちまでに外国がいこくへ旅行りょこうに行いく人ひとは２９９万まん人にんぐらいになりました。去年きょねんより３．５％多おおくなって、今いままででいちばん多おおくなりそうです。
来月らいげつは休日きゅうじつが３日みっか続つづいているため、長ながい休やすみを取とる人ひとが多おおいことや、アメリカやヨーロッパに行いく飛行機ひこうきの数かずが増ふえていることなどが理由りゆうだと言いっています。
旅行りょこうに使つかうお金かねは、平均へいきんで１人ひとり２２万まん７７００円えんで、去年きょねんより６．２％増ふえそうです。
ＪＴＢは「８泊ぱく以上いじょうの長ながい旅行りょこうに行いく人ひとが増ふえそうです。特とくにヨーロッパなど遠とおい国くにに行いく人ひとが多おおくなると考かんがえています」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

ツアー

  • 観光かんこうやスポーツなどのために、団体だんたいで行いく旅行りょこう。

数

  • 一ひとつ、二ふたつ、三みっつなどと数かぞえた物ものの数量すうりょう。すう。
  • いくつもあること。たくさん。
  • 数かぞえあげる値打ねうちのあるもの。なかま。

計算

  • 数かずや量りょうを数かぞえること。
  • 式しきを解といて、答こたえを出だすこと。
  • 考かんがえに入いれておくこと。

結果

  • あることがもとになって起おこったことがらやようす。

休日

  • 休やすみの日ひ。日曜にちようや祝日しゅくじつなど。

ヨーロッパ

  • 世界せかいの六大州ろくだいしゅうの一ひとつ。アジアの北西ほくせい、アフリカの北きたにある。産業さんぎょうや文化ぶんかが発達はったつした国くにが多おおい。

平均

  • 多おおい少すくないや高たかい低ひくいなどがないように、ならすこと。
  • つり合あいがとれること。バランス。

泊

  • 船ふねが港みなとにとまる。
  • 自分じぶんの家いえでない所ところにとまる。
  • あっさりしている。
  • とまった数かずを数かぞえることば。
