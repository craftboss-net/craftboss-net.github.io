アメリカのエミー賞しょう　「こんまり」の番組ばんぐみが候補こうほになる

发表于 2019-07-22 更新于 2019-11-30 分类于 JapanEasyNews



アメリカでは、すばらしいテレビの番組ばんぐみなどにエミー賞しょうという賞しょうを贈おくります。賞しょうの候補こうほの発表はっぴょうが１６日にちにありました。この中なかの２つの部門ぶもんで、日本にっぽんの近藤こんどう麻理恵まりえさんの番組ばんぐみが候補こうほになりました。近藤こんどうさんはみんなに「こんまり」と呼よばれていて、家いえの中なかなどを片かたづける方法ほうほうを教おしえています。アメリカでもとても人気にんきがあります。
近藤こんどうさんの番組ばんぐみでは、片かたづけ方かたについて困こまっているアメリカの家庭かていに近藤こんどうさんが行いって片かたづけるのを手伝てつだいます。
近藤こんどうさんのやり方かたでは、片かたづけたい物ものを手てに持もって、うれしくなったり楽たのしみになったりする「ときめき」を感かんじるかどうかで、捨すてるかどうかを決きめます。アメリカでは、この片かたづけ方かたがとても人気にんきになりました。
エミー賞しょうの発表はっぴょうは９月がつ２２日にちです。
