東京都とうきょうと　電車でんしゃや道みちが混こまないように「スムーズビズ」を行おこなう

发表于 2019-07-22 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックとパラリンピックの２つの大会たいかいのとき、東京都とうきょうとやその周まわりには、１０００万まん人にん以上いじょうの人ひとが来くると考かんがえられています。電車でんしゃや道みちがいつもより混こむ心配しんぱいがあるため、東京都とうきょうとは会社かいしゃなどに「スムーズビズ」を行おこなってほしいと言いっています。スムーズビズは、会社かいしゃに行いく時間じかんを変かえたり、会社かいしゃに行いかないで家いえなどでパソコンを使つかって仕事しごとをしたりすることです。
東京都とうきょうとや電車でんしゃの会社かいしゃは、２２日にちからスムーズビズのテストを始はじめました。来年らいねんの２つの大会たいかいと同おなじような期間きかんの９月がつ６日むいかまで行おこなう予定よていです。
東京都とうきょうとは２２日にちから５日いつか間かん、東京都庁とうきょうとちょうで働はたらく約やく１万まん人にんに、午前ごぜん８時じから午前ごぜん１０時じまで電車でんしゃやバスに乗のらないように言いっています。
電車でんしゃの会社かいしゃは２２日にちから、朝あさ早はやい時間じかんや夕方ゆうがたに電車でんしゃを増ふやしました。混こまない時間じかんに電車でんしゃに乗のってもらうためにキャンペーンも始はじめました。
東京都とうきょうとは、テストの結果けっかを来年らいねんの大会たいかいで役やくに立たつようにしたいと考かんがえています。
