香港ほんこん　デモをしていた大勢おおぜいの若わかい人ひとが議会ぎかいの中なかに入はいる

发表于 2019-07-02 更新于 2019-11-30 分类于 JapanEasyNews



７月がつ１日ついたち、イギリスが香港ほんこんを中国ちゅうごくに返かえしてから２２年ねんになりました。香港ほんこんでは毎年まいとしこの日ひに、中国ちゅうごくの政治せいじに反対はんたいする団体だんたいなどがデモをしています。今年ことしは、香港ほんこんに逃にげてきた容疑者ようぎしゃを中国ちゅうごくに渡わたすことができる規則きそくを作つくることに反対はんたいするデモになりました。
この規則きそくに反対はんたいする大勢おおぜいの若わかい人ひとたちが朝あさ早はやくから集あつまって、道みちを通とおることができないようにしました。夜よるになると、議会ぎかいの建物たてものに入はいってカメラを壊こわしたり、「長官ちょうかんはやめろ」と壁かべに書かいたりしました。
警察けいさつは涙なみだが出でるガスなどを使つかって、集あつまった人ひとたちを帰かえらせました。香港ほんこんの政府せいふによると、市民しみんや警官けいかんなど５９人にんがけがをしました。
香港ほんこんの政府せいふの長官ちょうかんは、議会ぎかいに入はいったりした若わかい人ひとたちの暴力ぼうりょくに強つよく反対はんたいすると言いいました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

デモ

  • 英語えいごの「デモンストレーション」の略りゃく。自分じぶんたちの考かんがえを相手あいてに認みとめさせようとして、大勢おおぜいが集あつまって自分じぶんたちの勢いきおいを示しめすこと。また、そのための行進こうしん。

議会

  • 国民こくみんに選えらばれた議員ぎいんが集あつまって、国くにや地方ちほうの政治せいじについて話はなし合あい、取とり決きめをする所ところ。国くにの議会ぎかいの国会こっかいと、都道府県とどうふけんなどの地方議会ちほうぎかいとがある。

団体

  • 大勢おおぜいの人ひとの集あつまり。
  • 同おなじ目的もくてきを持もった人々ひとびとの集あつまり。

容疑者

  • 罪つみをおかした疑うたがいのある人ひと。

長官

  • 官庁かんちょうのいちばん上うえの役目やくめ。また、その役目やくめの人ひと。次官じかんの上うえ。

辞める

  • 勤つとめや役目やくめから退しりぞく。

涙

  • 泣ないたときや目めにごみが入はいったときなどに、目めから出でる水みずのようなもの。

暴力

  • 力ちからでする、乱暴らんぼうな行おこない。
