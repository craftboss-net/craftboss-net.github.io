富士山ふじさんに登のぼる道みちがオープン　頂上ちょうじょうまで登のぼることはできない

发表于 2019-07-02 更新于 2019-11-30 分类于 JapanEasyNews



山梨県やまなしけんと静岡県しずおかけんの間あいだにある富士山ふじさんで１日ついたち、山梨県やまなしけんから登のぼる道みちがオープンしました。
富士山ふじさんでは去年きょねん、頂上ちょうじょうの神社じんじゃにある石いしの塀へいが崩くずれて、近ちかくの道みちを通とおることができなくなりました。このため、今いまは高たかさ３４５０ｍの８．５合ごう目めより上うえには登のぼることができません。１日ついたちは時々ときどき雨あめが降ふりましたが、お土産みやげの店みせなどがある５合ごう目めには大勢おおぜいの人ひとが来きました。８．５合ごう目めまで登のぼる人ひともいました。
山梨県やまなしけんは「頂上ちょうじょうまで登のぼることができるように、できるだけ早はやく崩くずれた石いしを片付かたづけます」と言いっています。
静岡県しずおかけんから富士山ふじさんに登のぼる道みちは１０日とおかにオープンします。
