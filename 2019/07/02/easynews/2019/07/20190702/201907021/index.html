京都きょうとの寺てら「カードやスマートフォンで払はらわないでほしい」

发表于 2019-07-02 更新于 2019-11-30 分类于 JapanEasyNews



京都きょうとにある約やく１０００の寺てらが入はいっている京都仏教会きょうとぶっきょうかいは、寺てらに来きた人ひとがお金かねを払はらうとき、クレジットカードやスマートフォンを利用りようしてもいいかどうか話はなし合あってきました。そして、寺てらを見みるときや祈いのるとき、それに葬式そうしきのときにはカードやスマートフォンを利用りようしないでお金かねで払はらってもらうことにすると発表はっぴょうしました。
京都仏教会きょうとぶっきょうかいは理由りゆうも説明せつめいしました。説明せつめいによると、亡なくなった人ひとのために祈いのることは商売しょうばいとは違ちがいます。カードやスマートフォンからその人ひとの宗教しゅうきょうなどの情報じょうほうが外そとに出でてしまうことも心配しんぱいです。
京都仏教会きょうとぶっきょうかいは、仏教ぶっきょう以外いがいの宗教しゅうきょうの団体だんたいも同おなじようにしてほしいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

カード

  • 紙かみを、四角しかくに小ちいさく切きったもの。記入きにゅうしたり、整理せいりするときなどに使つかう。
  • 「クレジットカード」「キャッシュカード」などの略りゃく。
  • トランプ（の札ふだ）。
  • 野球やきゅうなどの、試合しあいの組くみ合あわせ。

約

  • ちかう。取とり決きめる。
  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。

クレジットカード

  • 現金げんきんを払はらわなくても、後払あとばらいで買かい物ものをすることができるカード。

話し合う

  • たがいに話はなす。
  • 意見いけんを出だし合あう。

葬式

  • 死しんだ人ひとを、とむらう儀式ぎしき。とむらい。葬儀そうぎ。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

商売

  • 品物しなものを、仕入しいれて売うること。あきない。
  • 生活せいかつのためにしている仕事しごと。職業しょくぎょう。

宗教

  • 神かみや仏ほとけを信しんじること。また、神かみや仏ほとけの教おしえ。

情報

  • できごとやものごとのようすについての知しらせ。

仏教

  • 世界せかい三大宗教さんだいしゅうきょうの一ひとつ。紀元前きげんぜん５００年ねんごろ、インドで釈迦しゃかが説といた教おしえ。日本にっぽんには、六ろく世紀せいきの中なかごろ中国ちゅうごくを通とおして伝つたわった。

団体

  • 大勢おおぜいの人ひとの集あつまり。
  • 同おなじ目的もくてきを持もった人々ひとびとの集あつまり。
