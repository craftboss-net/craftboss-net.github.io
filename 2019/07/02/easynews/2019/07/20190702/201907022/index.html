売うる目的もくてきで鯨くじらをとることを日本にっぽんがまた始はじめる

发表于 2019-07-02 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんは売うる目的もくてきで鯨くじらをとっていましたが、３１年ねん前まえからやめていました。世界せかいの国くにが鯨くじらについて話はなし合あうＩＷＣが「鯨くじらが少すくなくなってしまうため、調査ちょうさ以外いがいの目的もくてきで鯨くじらをとってはいけない」と決きめたからです。しかし日本にっぽんはＩＷＣから出でて、売うる目的もくてきで鯨くじらをとることを７月がつ１日ついたちからまた始はじめました。日本にっぽんは、海うみにいる鯨くじらが少すくなくならないように量りょうを決きめてとると言いっています。
山口県やまぐちけん下関市しものせきしでは、鯨くじらをとる船ふねの出発しゅっぱつをお祝いわいする式しきがありました。下関市しものせきしでは昔むかし、鯨くじらをたくさんとっていたため、「鯨くじらのまち」で有名ゆうめいです。
下関市しものせきしの市長しちょうは式しきで「売うる目的もくてきで鯨くじらをとりたいと長ながい間あいだずっと思おもっていました。出発しゅっぱつの式しきを行おこなうことができて、とてもうれしいです」と話はなしました。港みなとには、船ふねを送おくりに来きた家族かぞくが集あつまりました。船ふねはこれから３か月げつの間あいだ、鯨くじらをとります。
１日ついたちは、北海道ほっかいどう釧路市くしろしの港みなとからも鯨くじらをとる小ちいさな船ふね５隻せきが出発しゅっぱつしました。
