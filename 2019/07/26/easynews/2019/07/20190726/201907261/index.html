東京とうきょうオリンピックの試合しあいを見みるツアーの受うけ付つけが始はじまる

发表于 2019-07-26 更新于 2019-11-30 分类于 JapanEasyNews



ＪＴＢなど３つの旅行りょこうの会社かいしゃが、来年らいねんの東京とうきょうオリンピックのチケットとホテルをセットにしたツアーの抽ちゅうせんの受うけ付つけを始はじめました。
ＪＴＢのツアーは全部ぜんぶで３７あります。いちばん高たかいツアーは４５０万まん円えんで、開会かいかい式しきと閉会へいかい式しき、そして柔道じゅうどうや陸上競技りくじょうきょうぎなどの試合しあいをオリンピックの間あいだ、毎日まいにち見みることができます。
いちばん安やすいツアーは２８万まん８０００円えんで、バレーボールなど１つのスポーツの試合しあいを５日いつか間かん見みることができます。このほかに、メダルが決きまる試合しあいだけを見みることができるツアーもあります。
８月がつ１５日にちまでにウェブサイトで申もうし込こんだ人ひとの中なかから、抽ちゅうせんで選えらばれた７８０人にんぐらいがツアーを買かうことができます。
近畿日本きんきにっぽんツーリストと東武とうぶトップツアーズも、ウェブサイトでツアーの受うけ付つけを始はじめました。３つの会社かいしゃは、これからツアーをもっと多おおくする予定よていです。
