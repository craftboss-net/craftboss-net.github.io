ＳＭＡＰだった３人にんのテレビ出演しゅつえんでジャニーズ事務所じむしょを注意ちゅうい

发表于 2019-07-18 更新于 2019-11-30 分类于 JapanEasyNews



ジャニーズ事務所じむしょの「ＳＭＡＰ」にいた稲垣いながき吾郎ごろうさんと草くさなぎ剛つよしさんと香取かとり慎吾しんごさんの３人にんは事務所じむしょをやめました。３人にんは今いま、ＮＨＫ以外いがいのテレビに出演しゅつえんしていません。
公正取引委員会こうせいとりひきいいんかいは３人にんがテレビに出演しゅつえんしていないことについて、ジャニーズ事務所じむしょやテレビ局きょくの人ひとなどに話はなしを聞ききました。
公正取引委員会こうせいとりひきいいんかいは、テレビ局きょくの人ひとが「３人にんをテレビに出だすとジャニーズ事務所じむしょのタレントが出でなくなる」と感かんじるようなことを事務所じむしょが言いっていたと考かんがえています。これは、仕事しごとを独占どくせんすることを禁止きんしする法律ほうりつに違反いはんする可能性かのうせいがあると考かんがえて、ジャニーズ事務所じむしょを注意ちゅういしました。
ジャニーズ事務所じむしょは「テレビ局きょくに３人にんを出だすなと言いったことはありません。しかし、これからはそのように思おもわれないように気きをつけます」と言いっています。
