東京とうきょうオリンピックで使つかう新国立競技場しんこくりつきょうぎじょうに芝しばを植うえ始はじめる

发表于 2019-07-18 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックとパラリンピックで開会かいかい式しきや陸上競技りくじょうきょうぎ、サッカーの試合しあいなどを行おこなう新国立競技場しんこくりつきょうぎじょうは、９０％ぐらい工事こうじが終おわりました。競技場きょうぎじょうには屋根やねがついて、客きゃくが座すわる席せきも７０％ぐらいできました。
１６日にちからは、サッカーの試合しあいなどをする場所ばしょに芝しばを植うえ始はじめました。芝しばを植うえる場所ばしょの広ひろさは縦たて７１ｍ、横よこ１０７ｍです。１７日にちまでに半分はんぶん以上いじょう植うえました。冬ふゆでも芝しばに光ひかりが当あたるように、屋根やねの南みなみ側がわは透明とうめいになっています。土つちの中なかには冷つめたい水みずや温あたたかい水みずが通とおる管くだがあって、いつも芝しばが育そだちやすい温度おんどにすることができます。
芝しばは今週こんしゅう中ちゅうに植うえ終おわって、８月がつからは陸上競技りくじょうきょうぎで選手せんしゅが走はしるトラックをつくり始はじめます。新国立競技場しんこくりつきょうぎじょうの工事こうじは、今年ことし１１月がつに全部ぜんぶ終おわる予定よていです。

オリンピック

  • ４年ごとに開ひらかれ、世界せかいじゅうの国々くにぐにから選手せんしゅが参加さんかする競技大会きょうぎたいかい。古代こだいギリシャのオリンピアで開ひらかれた古代こだいオリンピックにならって、フランスのクーベルタンの力ちからで、１８９６年ねんにギリシャのアテネで開ひらかれたのが、近代きんだいオリンピックの始はじまり。五輪ごりん。
