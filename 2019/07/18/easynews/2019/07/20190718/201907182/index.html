「７ｐａｙ」　１５００人にん以上いじょうが不正ふせいにお金かねを使つかわれる

发表于 2019-07-18 更新于 2019-11-30 分类于 JapanEasyNews



コンビニのセブンーイレブンは、今月こんげつ１日ついたちからスマートフォンを使つかってお金かねを払はらう「７ｐａｙ」というサービスを始はじめました。しかし、知しらない人ひとにパスワードなどを不正ふせいに使つかわれて、買かい物ものをされる被害ひがいがありました。
会社かいしゃが調しらべると、今月こんげつ１１日にちまでに１５７４人にんが被害ひがいにあったことがわかりました。不正ふせいに使つかわれたお金かねは３２４０万まん円えん以上いじょうでした。会社かいしゃは被害ひがいを受うけた人ひとみんなにお金かねを返かえすと言いっています。
被害ひがいは、サービスが始はじまった次つぎの日ひの２日ふつかの夜よるから３日みっかに多おおくなっていました。会社かいしゃはこのサービスに新あたらしく登録とうろくしたり、お金かねを入いれたりすることができないようにしています。そして、不正ふせいに使つかわれた原因げんいんや被害ひがいがまた出でないようにする方法ほうほうを考かんがえると言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

不正

  • 正ただしくないこと。

コンビニ

  • 食料品しょくりょうひんや日用品にちようひんが、手軽てがるにいつでも買かえるように開ひらいている、小型こがたのスーパー。

サービス

  • 人ひとのためにつくすこと。
  • 客きゃくをもてなすこと。奉仕ほうし。

パスワード

  • コンピューターを使つかうときなどに、その人ひとであることを確認かくにんする、特別とくべつなことばや符号ふごう。

被害

  • 損害そんがいを受うけること。また、受うけた害がい。

登録

  • 役所やくしょなどに届とどけ出でて、おおやけに認みとめてもらうこと。

方法

  • ものごとを行おこなうやり方かた。しかた。
