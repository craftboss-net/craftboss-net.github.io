「投票とうひょうするときに男性だんせいか女性じょせいかを聞きかないでほしい」

发表于 2019-07-17 更新于 2019-11-30 分类于 JapanEasyNews



宮城県みやぎけん女川町おながわちょうに住すんでいる人ひとが、心こころと体からだの性せいが違ちがうトランスジェンダーの人ひとたちが安心あんしんして選挙せんきょの投票とうひょうができるようにしてほしいと町まちにお願ねがいしました。
女川町おながわちょうに住すんでいる２９歳さいの堀ほりみのりさんは、戸籍こせきでは女性じょせいになっていますが、自分じぶんは男性だんせいだと思おもって生活せいかつしています。
堀ほりさんは投票とうひょうに行いったときに「女性じょせいに見みえない」と言いわれて、嫌いやな気持きもちになったことがあります。このため、投票とうひょうするときに必要ひつような入場券にゅうじょうけんに男性だんせいか女性じょせいかを印刷いんさつしたり、会場かいじょうで男性だんせいか女性じょせいかを聞きいたりすることをやめてほしいと言いいました。
堀ほりさんは「自分じぶんと同おなじような人ひとたちが不安ふあんな気持きもちにならないで投票とうひょうできるようにしてほしいです」と話はなしていました。
女川町おながわちょうで選挙せんきょの準備じゅんびをしている人ひとは「いろいろな人ひとの意見いけんを聞きいて、どうしたらいいか考かんがえます」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

投票

  • 選挙せんきょや採決さいけつで、選えらびたい人ひとの名前なまえや、賛成さんせいか反対はんたいかを、紙かみに書かいて出だすこと。

性

  • 生うまれつき。
  • 男女だんじょの区別くべつ。
  • …の性質せいしつを持もつ、という意味いみを表あらわす。

選挙

  • ある役目やくめにつく人ひとを、大勢おおぜいの中なかから選えらぶこと。

戸籍

  • 本籍地ほんせきちや氏名しめい、生年月日せいねんがっぴ、家族かぞくの関係かんけいなどを書かいて、役所やくしょに置おかれている、おおやけの書かき物もの。

印刷

  • 文字もじや絵えなどを組くみ合あわせて版はんを作つくり、印刷機いんさつきでインクをつけて同おなじものをたくさん刷すること。

不安

  • 気きがかりなこと。心配しんぱいなこと。
