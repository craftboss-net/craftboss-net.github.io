手足口病てあしくちびょう　これから２週間しゅうかんぐらいがいちばん多おおくなりそう

发表于 2019-07-17 更新于 2019-11-30 分类于 JapanEasyNews



手足口病てあしくちびょうは子こどもがなることが多おおい病気びょうきで、手てや足あし、口くちの中なかなどに小ちいさな発疹はっしんができます。この病気びょうきはウイルスでうつって、脳のうが重おもい病気びょうきになることもあります。
国立感染症研究所こくりつかんせんしょうけんきゅうしょによると、今月こんげつ７日なのかまでの１週間しゅうかんにこの病気びょうきになった子こどもは３万まん１０００人にんぐらいでした。７月がつの初はじめごろでは、今いままでの１０年ねんで最もっとも多おおくなりました。特とくに福井県ふくいけんや石川県いしかわけんで多おおくなっています。
国立感染症研究所こくりつかんせんしょうけんきゅうしょは、手足口病てあしくちびょうになる子こどもはこれから２週間しゅうかんぐらいの間あいだにいちばん多おおくなると考かんがえています。
国立感染症研究所こくりつかんせんしょうけんきゅうしょは「小ちいさな子こどもがいる家庭かていや保育園ほいくえんなどでは、赤あかちゃんのおむつを取とり替かえたあとはきれいに片付かたづけてください。よく手てを洗あらって、ほかの人ひとと同おなじタオルを使つかわないようにしてください」と言いっています。
