恐竜きょうりゅうの展覧会てんらんかい　世界せかいで初はじめて見みせる化石かせきもある

发表于 2019-07-17 更新于 2019-11-30 分类于 JapanEasyNews



恐竜きょうりゅうの化石かせきなどを１００以上いじょう紹介しょうかいする展覧会てんらんかいが、東京都とうきょうとにある国立科学博物館こくりつかがくはくぶつかんで始はじまりました。
この展覧会てんらんかいでは、「デイノケイルス」という恐竜きょうりゅうの頭あたまと足あしの化石かせきを世界せかいで初はじめて見みせています。「デイノケイルス」は前まえの足あしが長ながくて、くちばしの幅はばが広ひろい、珍めずらしい恐竜きょうりゅうです。
北海道ほっかいどうむかわ町ちょうで見みつかった「むかわ竜りゅう」の体からだの形かたちや大おおきさがわかる骨ほねも見みせています。「むかわ竜りゅう」は、日本にっぽんで体からだのほとんど全部ぜんぶの骨ほねが見みつかった中なかでは、いちばん大おおきい恐竜きょうりゅうです。
新あたらしい研究けんきゅうでわかった恐竜きょうりゅうの様子ようすをＣＧで見みることもできます。
この展覧会てんらんかいは１０月がつ１４日じゅうよっかまでです。
