若わかい社員しゃいんが奨学金しょうがくきんを返かえすのを手伝てつだう会社かいしゃが増ふえる

发表于 2019-07-17 更新于 2019-11-30 分类于 JapanEasyNews



若わかい社員しゃいんの中なかには、学生がくせいのとき借かりた奨学金しょうがくきんを返かえすのが大変たいへんな人ひともいます。このため、奨学金しょうがくきんを返かえすのを手伝てつだう大おおきな会社かいしゃが増ふえています。
「あおぞら銀行ぎんこう」は今年ことし４月がつから、働はたらき始はじめて３年ねんまでの社員しゃいんに、まだ返かえしていないお金かねの１５％までを払はらうことにしました。
「大和証券だいわしょうけんグループ本社ほんしゃ」も去年きょねん夏なつから、社員しゃいんにお金かねを貸かして、奨学金しょうがくきんを一度いちどに全部ぜんぶ返かえすことができるようにしました。会社かいしゃが貸かすお金かねに利子りしはかかりません。
「トヨタ自動車じどうしゃ」のグループは、大学だいがくや大学院だいがくいんで機械きかいなどの技術ぎじゅつを勉強べんきょうしている女子じょし学生がくせいに、１年ねんに６０万まん円えん貸かしています。その学生がくせいがグループの会社かいしゃに入はいったら、お金かねを返かえす必要ひつようはありません。
会社かいしゃは、若わかい社員しゃいんがお金かねで困こまらないように手伝てつだって、能力のうりょくが高たかい人ひとに会社かいしゃに入はいってほしいと考かんがえています。
