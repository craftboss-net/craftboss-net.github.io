参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまる

发表于 2019-07-04 更新于 2019-11-30 分类于 JapanEasyNews



４日よっか、参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまりました。投票とうひょうは７月がつ２１日にちです。
参議院さんぎいんは３年ねんに１度ど、半分はんぶんの数かずの議員ぎいんを選挙せんきょで選えらびます。去年きょねん、法律ほうりつが変かわって議員ぎいんを増ふやしたため、前まえの選挙せんきょより３人にん多おおい１２４人にんを選えらびます。
参議院さんぎいんの議員ぎいんは２つの方法ほうほうで選えらびます。都道府県とどうふけんの中なかで議員ぎいんを選えらぶ「選挙区せんきょく」の選挙せんきょでは、７４人にんを選えらびます。人口じんこうが少すくない県けんでは、２つの県けんで１人ひとりを選えらぶところもあります。政党せいとうと立候補りっこうほした人ひとが取とった票ひょうの割合わりあいで議員ぎいんの数かずが決きまる「比例代表ひれいだいひょう」の選挙せんきょでは、５０人にんを選えらびます。
３７０人にんが立候補りっこうほしました。立候補りっこうほした人ひとたちは、安倍あべ総理大臣そうりだいじんの政治せいじや、１０月がつに消費税しょうひぜいを１０％に上あげること、年金ねんきんの問題もんだいなどについて意見いけんを言いって、選挙せんきょを戦たたかいます。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

参議院

  • 国会こっかいの議院ぎいんの一ひとつ。衆議院しゅうぎいんで決きまった予算よさんや法律ほうりつなどを、もう一度いちど検討けんとうするところ。解散かいさんはない。

議員

  • 選挙せんきょで選えらばれ、国会こっかいや地方ちほうの議会ぎかいで、政治せいじに関かんすることをいろいろ相談そうだんする人ひと。国会議員こっかいぎいん・県議会議員けんぎかいぎいんなど。

選挙

  • ある役目やくめにつく人ひとを、大勢おおぜいの中なかから選えらぶこと。

投票

  • 選挙せんきょや採決さいけつで、選えらびたい人ひとの名前なまえや、賛成さんせいか反対はんたいかを、紙かみに書かいて出だすこと。

数

  • 一ひとつ、二ふたつ、三みっつなどと数かぞえた物ものの数量すうりょう。すう。
  • いくつもあること。たくさん。
  • 数かぞえあげる値打ねうちのあるもの。なかま。

増やす

  • 多おおくする。

方法

  • ものごとを行おこなうやり方かた。しかた。

都道府県

  • 全国ぜんこくを４７に分わけた区画くかく。東京都とうきょうと・北海道ほっかいどう・大阪府おおさかふ・京都府きょうとふと、４３の県けん。

政党

  • 政治せいじについて、同おなじ考かんがえを持もつ人ひとたちが集あつまって作つくった団体だんたい。

立候補

  • 選挙せんきょのとき、候補者こうほしゃとして名なのり出でること。

票

  • 紙かみのふだ。
  • 選挙せんきょで投票とうひょうした紙かみ。投票用紙とうひょうようし。また、その紙かみを数かぞえることば。

割合

  • 物ものと物ものとの関係かんけいを、数かずで表あらわしたもの。割わり。率りつ。歩合ぶあい。
  • 思おもいのほか。比較的ひかくてき。

総理大臣

  • 内閣総理大臣ないかくそうりだいじんのこと。内閣ないかくの最高責任者さいこうせきにんしゃで、国会議員こっかいぎいんの中なかから議員ぎいんが選えらび、天皇てんのうが認みとめて決きまる。首相しゅしょう。総理そうり。

消費税

  • 物ものを買かったり、サービスを受うけたりしたときに、かけられる税金ぜいきん。

年金

  • （国くになどから）毎年まいとしいくらと決きまって受うけ取とるお金かね。

戦う

  • 戦争せんそうをする。
  • わざを比くらべて、勝かち負まけを決きめる。
