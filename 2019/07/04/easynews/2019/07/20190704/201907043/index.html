雨あめがやんだあとでも山やまが崩くずれることがあるので気きをつけて

发表于 2019-07-04 更新于 2019-11-30 分类于 JapanEasyNews



雨あめがとてもたくさん降ふった鹿児島県かごしまけんなどでは、地面じめんの中なかに水みずがたくさん入はいっているので、山やまや崖がけなどが崩くずれる心配しんぱいがあります。
鹿児島県かごしまけんと宮崎県みやざきけんでは先月せんげつ２８日にちから降ふった雨あめが、いつもの年としの７月がつの２倍ばい以上いじょうになっている場所ばしょがあります。
鹿児島県かごしまけんでは１９９７年ねん、雨あめがやんでから４時間じかんあとに、山やまから土つちや石いしが崩くずれてきて、２１人にんが亡なくなりました。
気象庁きしょうちょうは「鹿児島県かごしまけんなど九州きゅうしゅうの南みなみ側がわは、少すこしの雨あめや雨あめがやんだあとでも、山やまなどが崩くずれるかもしれません。山やまや崖がけの近ちかくに行いかないでください」と言いっています。
