富士山ふじさんに登のぼるときお金かねを払はらってくれる人ひとを多おおくしたい

发表于 2019-07-23 更新于 2019-11-30 分类于 JapanEasyNews



山梨県やまなしけんは富士山ふじさんに登のぼる人ひとに１人ひとり１０００円えん払はらってほしいと言いっています。このお金かねは、富士山ふじさんをきれいで安全あんぜんにするために使つかっています。しかし、去年きょねんお金かねを払はらった人ひとは、山梨県やまなしけんから登のぼった人ひとの５８．６％だけでした。
このため、山梨県やまなしけんは今年ことしから、お金かねを払はらってくれた人ひとに、腕うでにつけるリストバンドをあげることにしました。
リストバンドにあるＱＲコードをスマートフォンなどで写うつすと、ホテルの情報じょうほうを見みたり、レストランでお金かねが安やすくなるチケットをもらったりできます。このリストバンドは光ひかる材料ざいりょうでできているため、朝あさ早はやくや夜よるに登のぼるとき役やくに立たちます。
リストバンドは２万まん個こあります。山梨県やまなしけんから登のぼる道みちの５合ごう目めと６合ごう目めにお金かねを払はらう場所ばしょがあって、そこで８月がつ１１日にちからリストバンドをもらうことができます。

  • 静岡県しずおかけんと山梨県やまなしけんの境さかいにある、日本一にっぽんいち高たかい山やま。高たかさは３７７６メートル。江戸時代えどじだいに大おおきな噴火ふんかがあった。
