コヤマコウモリが北海道ほっかいどうで子こどもを育そだてているかもしれない

发表于 2019-07-23 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどう上ノ国町かみのくにちょうにある風力ふうりょく発電所はつでんしょで、おととし８月がつ、５匹ひきのこうもりが死しんでいるのが見みつかりました。電気でんきをつくるために回まわっていた風車ふうしゃにぶつかった可能性かのうせいがあります。
５匹ひきは「コヤマコウモリ」の子こどもだとわかりました。コヤマコウモリは体からだの長ながさが８ｃｍぐらいで、日本にっぽんだけにいると言いわれています。今いままでに見みつかったのは東北地方とうほくちほうや関東地方かんとうちほうの２７匹ひきだけで、将来しょうらいいなくなることが心配しんぱいされています。日本にっぽんのどこにいるのか、子こどもをどこで育そだてているのかなどまだわかっていません。
こうもりを研究けんきゅうしたり守まもったりしている団体だんたいは、コヤマコウモリが子こどもを育そだてている場所ばしょが近ちかくにあると考かんがえています。このため、発電所はつでんしょに調査ちょうさをするように言いっています。団体だんたいの人ひとは「大切たいせつなコヤマコウモリを守まもるためにできることをしてほしいです」と話はなしています。

発電所

  • 電気でんきを起おこす所ところ。

こうもり

  • 日中にっちゅうは暗くらいほら穴あなの岩いわなどにぶら下さがり、夜よるになると鳥とりのように空そらを飛とび回まわるけもの。顔かおと体からだつきはネズミに似にている。種類しゅるいが多おおい。
  • 「こうもりがさ」の略りゃく。

風車

  • 風かぜの力ちからで回まわす大おおきな羽根車はねぐるま。回まわる力ちからを使つかって、粉こなひきや米こめつきなどをする。かざぐるま。

ぶつかる

  • 打うち当あたる。つき当あたる。
  • 困難こんなんなことに出であう。
  • 争あらそう。立たち向むかう。
  • 重かさなる。

可能性

  • その状態じょうたいになりそうなようす。できそうなようす。

東北地方

  • 本州ほんしゅうの北東部ほくとうぶにあたる地方ちほう。青森あおもり・秋田あきた・岩手いわて・宮城みやぎ・山形やまがた・福島ふくしまの六県ろっけんがある。

関東地方

  • 本州ほんしゅうの中なかほどにある、太平洋側たいへいようがわの地方ちほう。東京とうきょう・神奈川かながわ・埼玉さいたま・千葉ちば・群馬ぐんま・栃木とちぎ・茨城いばらきの一都六県いっとろっけんがある。
