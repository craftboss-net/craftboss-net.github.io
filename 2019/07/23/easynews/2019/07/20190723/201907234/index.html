世界せかい選手権せんしゅけんでメダルを取とった選手せんしゅが池江いけえ選手せんしゅにメッセージ

发表于 2019-07-23 更新于 2019-11-30 分类于 JapanEasyNews



水泳すいえいの世界せかい選手権せんしゅけんは２２日にち、女子じょしの１００ｍバタフライの決勝けっしょうがありました。
この種目しゅもくでメダルを取とった３人にんの選手せんしゅは、手てを前まえに出だして記念きねんの写真しゃしんを撮とりました。３人にんの手てには英語えいごで「Ｒｉｋａｋｏ　ＮＥＶＥＲ　ＧＩＶＥ　ＵＰ」と池江いけえ選手せんしゅへのメッセージが書かいてありました。池江いけえ選手せんしゅはこの種目しゅもくがいちばん得意とくいですが、白血病はっけつびょうのため出でることができませんでした。
３人にんのうち、銀ぎんメダルを取とったスウェーデンのショーストロム選手せんしゅは、池江いけえ選手せんしゅと一緒いっしょに練習れんしゅうをしたこともあります。銅どうメダルを取とった選手せんしゅは「ショーストロム選手せんしゅがメッセージを出だそうと言いって、すぐにやることに決きまりました。メダルをもらう式しきに出でる少すこし前まえに書かきました」と話はなしていました。
