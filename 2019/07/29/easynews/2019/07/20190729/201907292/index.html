かばんは閉しめて　東京とうきょうを旅行りょこう中ちゅうの高校生こうこうせいが財布さいふを盗ぬすまれる

发表于 2019-07-29 更新于 2019-11-30 分类于 JapanEasyNews



２３日にちの朝あさ、東京とうきょうのＪＲ新宿駅しんじゅくえきで電車でんしゃに乗のろうとした女子じょし高校生こうこうせいが、肩かたにかけるかばんの中なかから男おとこに財布さいふを盗ぬすまれました。ホームで警備けいびしていた警察けいさつの人ひとが男おとこを見みていたため、男おとこをすぐ捕つかまえました。
高校生こうこうせいは夏休なつやすみの旅行りょこうで愛知県あいちけんから来きていました。かばんは開あいていて、財布さいふがいちばん上うえにありました。
かばんが開あいていると、泥棒どろぼうは中なかの物ものを盗ぬすみやすいと考かんがえます。警察けいさつは、かばんは必かならず閉しめて気きをつけてほしいと言いっています。
