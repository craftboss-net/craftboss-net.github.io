ＪＡＸＡ　宇宙うちゅうで水みずをきれいにする機械きかいを作つくる

发表于 2019-07-29 更新于 2019-11-30 分类于 JapanEasyNews



宇宙うちゅうでは、水みずがとても大切たいせつです。国際こくさい宇宙うちゅうステーションでは、ＮＡＳＡが作つくった機械きかいで宇宙飛行士うちゅうひこうしの尿にょうなどをきれいにした水みずを使つかっています。
ＪＡＸＡは、汚よごれた水みずを入いれると８５％以上いじょうがきれいになって出でてくる新あたらしい機械きかいを作つくりました。この機械きかいは縦たてが５０ｃｍ、横よこが８０ｃｍ、幅はばが７０ｃｍぐらいで、フィルターや電気でんきを使つかって水みずをきれいにします。飲のむことができる水みずを１日にちに８００ｍＬまでつくることができます。
ＪＡＸＡはこの機械きかいを国際こくさい宇宙うちゅうステーションに打うち上あげて正ただしく動うごくかどうかテストをします。そして、２０２５年ねんごろまでにもっと大おおきい機械きかいを作つくって、新あたらしく作つくることを考かんがえている宇宙うちゅうステーションなどで使つかってほしいと言いっています。
ＪＡＸＡは「人ひとが月つきや火星かせいに行いくためには水みずをリサイクルする技術ぎじゅつが必要ひつようなので、日本にっぽんも役やくに立たちたいです」と話はなしていました。
