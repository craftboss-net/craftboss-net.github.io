テニスの若わかい選手せんしゅの世界せかい大会たいかい　日本にっぽんの望月もちづき選手せんしゅが優勝ゆうしょうする

发表于 2019-07-16 更新于 2019-11-30 分类于 JapanEasyNews



テニスの世界せかい大会たいかい、ウィンブルドン選手権せんしゅけんの若わかい選手せんしゅが出でる試合しあいで、日本にっぽんの１６歳さいの望月もちづき慎太郎しんたろう選手せんしゅが優勝ゆうしょうしました。望月もちづき選手せんしゅは、決勝けっしょうでスペインの選手せんしゅに２?０で勝かちました。
テニスの４つの大おおきな世界せかい大会たいかいの若わかい選手せんしゅの試合しあいで、日本にっぽんの男子だんしの選手せんしゅが優勝ゆうしょうしたのは初はじめてです。世界せかいの若わかい選手せんしゅのランキングでも、望月もちづき選手せんしゅが１番ばんになりました。
望月もちづき選手せんしゅは３歳さいでテニスを始はじめて、１２歳さいでアメリカにある有名ゆうめいなスポーツの学校がっこうに入はいりました。望月もちづき選手せんしゅは「今いままで努力どりょくを続つづけてよかったと思おもいます。満足まんぞくしないで、これからも頑張がんばります」と話はなしていました。
