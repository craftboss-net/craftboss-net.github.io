サッカー　女子じょしのプロリーグを作つくるために準備じゅんびを始はじめる

发表于 2019-07-16 更新于 2019-11-30 分类于 JapanEasyNews



サッカー 女子じょしの２０１１年ねんのワールドカップで、日本にっぽんのチーム「なでしこジャパン」が優勝ゆうしょうしました。しかし、今年ことしのワールドカップでは、決勝けっしょうトーナメントの最初さいしょの試合しあいで負まけました。しっかりしたプロリーグがあるヨーロッパのチームが今年ことしは特とくに強つよくなっていました。
このため日本にほんサッカー協会きょうかいは１１日にち、女子じょしのプロリーグを作つくるために、専門家せんもんかなどと一緒いっしょに準備じゅんびをすることを決きめました。来年らいねんの東京とうきょうオリンピックのあとに、プロリーグを作つくりたいと考かんがえています。
しかし、お客きゃくさんやお金かねを十分じゅうぶんに集あつめることができるかどうか心配しんぱいがあります。日本にほんサッカー協会きょうかいは、女子じょしのプロリーグを本当ほんとうに作つくることができるか基本きほんから話はなし合あうことにしています。
日本にほんサッカー協会きょうかいの人ひとは「なでしこジャパンをもう一度いちど世界せかいで１番ばんにするために、選手せんしゅの力ちからをもっと上あげていきたいです」と話はなしていました。
