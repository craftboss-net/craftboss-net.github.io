雨あめや曇くもりの日ひが多おおい　野菜やさいの値段ねだんが高たかくなる

发表于 2019-07-16 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの太平洋たいへいよう側がわなどでは６月がつの終おわりごろから、曇くもりや雨あめの日ひが続つづいています。太陽たいようが出でている時間じかんが短みじかくて、いつもの年としより気温きおんが低ひくい日ひが多おおくなっています。
東京とうきょうでは気温きおんが２５℃まで上あがらなかった日ひが、今月こんげつ１２日にちまで８日ようか続つづきました。太陽たいようが出でていた時間じかんが３時間じかんより短みじかかった日ひも、１４日じゅうよっかまで１８日にち続つづきました。こんなに長ながく続つづいた年としは、１９８８年ねんからあとはありませんでした。
特とくに関東地方かんとうちほうでは太陽たいようが出でる時間じかんが短みじかくなっていて、野菜やさいが育そだつのが遅おくれています。東京都とうきょうとの市場いちばでは、きゅうりの値段ねだんがいつもの年としより５０％以上いじょう高たかくなっています。なすも２０％、ピーマンも１０％ぐらい高たかくなっています。
気象庁きしょうちょうは、これから１週間しゅうかんぐらいは関東地方かんとうちほうから九州きゅうしゅうまでで、曇くもりや雨あめの日ひが多おおくなりそうだと言いっています。

太平洋

  • アジア・オーストラリア・南北なんぼくアメリカ・南極なんきょくの五いつつの大陸たいりくに囲かこまれた、世界せかいでいちばん広ひろい海うみ。

太陽

  • 太陽系たいようけいの中心ちゅうしんで高たかい熱ねつと光ひかりを出だしている星ほし。地球ちきゅうに熱ねつや光ひかりをあたえ、生物せいぶつを育そだてる。お日ひさま。

気温

  • 大気たいき（空気くうき）の温度おんど。

関東地方

  • 本州ほんしゅうの中なかほどにある、太平洋側たいへいようがわの地方ちほう。東京とうきょう・神奈川かながわ・埼玉さいたま・千葉ちば・群馬ぐんま・栃木とちぎ・茨城いばらきの一都六県いっとろっけんがある。

育つ

  • 生いき物ものが、大おおきくなる。
  • 一人前いちにんまえになる。

市場

  • 商人しょうにんが集あつまって、魚さかなや野菜やさいなどを売うり買かいする所ところ。市いち。
  • 食たべ物ものや日用品にちようひんなどの小売店こうりてんが、一いっか所しょに集あつまって品物しなものを売うっている所ところ。マーケット。

胡瓜

  • 畑はたけに作つくる野菜やさい。夏なつに黄色きいろの花はなが咲さき、緑色みどりいろの細長ほそながい実みがなる。つけ物ものやサラダにして食たべる。

茄子

  • 野菜やさいの一ひとつ。夏なつにむらさき色いろの花はなが咲さく。実みは、こいむらさき色いろで、つけ物ものなどにして食たべる。なすび。

ピーマン

  • トウガラシのなかま。実みは長ながい円形えんけいで、中なかは空からになっている。外側そとがわのやわらかい緑色みどりいろの部分ぶぶんを食たべる。
