「地球温暖化ちきゅうおんだんかを止とめたい」スウェーデンの１６歳さいの女性じょせいに賞しょう

发表于 2019-07-24 更新于 2019-11-30 分类于 JapanEasyNews



スウェーデンの１６歳さいのグレタ?トゥーンベリさんは、地球温暖化ちきゅうおんだんかを止とめるための活動かつどうをしています。去年きょねん８月がつから毎週まいしゅう金曜日きんようびに学校がっこうを休やすんで、議会ぎかいの前まえで、地球ちきゅうの気温きおんが上あがらないようにしてほしいと言いっています。この活動かつどうはインターネットで世界中せかいじゅうの若わかい人ひとたちに広ひろがっています。
トゥーンベリさんは、フランスのノルマンディー地方ちほうが平和へいわのために活動かつどうしている人ひとたちに贈おくる「自由賞じゆうしょう」という賞しょうをもらうことになりました。
トゥーンベリさんは、賞しょうをもらう式しきで「これは私わたしだけではなくて、活動かつどうを続つづけている世界せかいの若わかい人ひとみんながもらった賞しょうです」と言いいました。そして、フランスの若わかい人ひとに「私わたしたちの活動かつどうは広ひろがりましたが、二酸化炭素にさんかたんそはまだ減へっていません。活動かつどうを続つづけましょう」と話はなしました。

  • 大気中たいきちゅうの二酸化炭素にさんかたんそ（CO2シーオーツー）が増ふえて、地球ちきゅうの気温きおんが上あがること。南極なんきょくや北極ほっきょくの氷こおりが解とけて海面かいめんが高たかくなったり、農作物のうさくぶつにひがいが出でたりする。
