イギリス　ジョンソンさんが新あたらしい首相しゅしょうになることが決きまる

发表于 2019-07-24 更新于 2019-11-30 分类于 JapanEasyNews



イギリスでは、ＥＵから出でるときのＥＵとの約束やくそくについて、メイ首相しゅしょうの意見いけんに議会ぎかいが反対はんたいしています。このため、メイ首相しゅしょうは首相しゅしょうを辞やめると言いって、保守党ほしゅとうのトップを辞やめました。
２３日にち、保守党ほしゅとうの新あたらしいトップを選えらぶ選挙せんきょがありました。ボリス?ジョンソンさんがトップに選えらばれて、２４日にじゅうよっかに首相しゅしょうになることが決きまりました。ジョンソンさんは５５歳さいで、今いままでにロンドンの市長しちょうや外務がいむ大臣だいじんの仕事しごとをしました。
イギリスは今年ことし１０月がつ３１日にちまでにＥＵから出でることになっています。ジョンソンさんは、その日ひまでにＥＵとの約束やくそくを決きめることができなくても、ＥＵから出でると言いっています。しかし、ジョンソンさんに反対はんたいして大臣だいじんを辞やめると言いっている人ひともいます。議会ぎかいには、約束やくそくを決きめてからＥＵを出でたほうがいいと考かんがえている人ひとが大勢おおぜいいます。
