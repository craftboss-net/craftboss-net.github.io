埼玉県さいたまけん和光市わこうし　子こどもが学校がっこうに通かよう外国人がいこくじんのため市民しみんが通訳つうやくする

发表于 2019-07-03 更新于 2019-11-30 分类于 JapanEasyNews



埼玉県さいたまけん和光市わこうしは、日本語にほんごがわからない外国人がいこくじんのお父とうさんやお母かあさんのために、市民しみんが学校がっこうで通訳つうやくするサービスを始はじめます。
２日ふつか、通訳つうやくする市民しみんの勉強会べんきょうかいが始はじまりました。英語えいごや韓国語かんこくごや中国語ちゅうごくごが上手じょうずな市民しみん７０人にんぐらいが集あつまりました。市民しみんは、子こどもの入学にゅうがく式しきやＰＴＡなどでお父とうさんやお母かあさんにわかる言葉ことばで通訳つうやくします。勉強会べんきょうかいでは、日本にっぽんの学校がっこうの習慣しゅうかんを説明せつめいするときに大事だいじなことや、困こまっている人ひとの話はなしはできるだけ丁寧ていねいに聞きくことなどを習ならいました。
勉強会べんきょうかいに来きた女性じょせいは「外国がいこくで子こどもを育そだてることは大変たいへんなので、手伝てつだいたいと思おもっています」と話はなしていました。
このサービスは来月らいげつから始はじまります。１回かい２時間じかんまで２０００円えんです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

通訳

  • ちがうことばを話はなす人ひとの間あいだに立たって、両方りょうほうのことばを翻訳ほんやくして伝つたえること。また、その人ひと。

サービス

  • 人ひとのためにつくすこと。
  • 客きゃくをもてなすこと。奉仕ほうし。

PTA

  • 親おやと先生せんせいが力ちからを合あわせて、子こどもたちをりっぱに育そだてるための相談そうだんなどをする会かい。親おやと教師きょうしの会かい。
