青森県あおもりけんのりんごの木き　許可きょかがないのに中国ちゅうごくで売うっている可能性かのうせい

发表于 2019-07-03 更新于 2019-11-30 分类于 JapanEasyNews



外国がいこくで、日本にっぽんのいちごやぶどうなどを育そだてる許可きょかをもらわないで育そだてている人ひとたちがいて、問題もんだいになっています。
青森県あおもりけんの「千雪ちゆき」というりんごは、外国がいこくでも人気にんきがあります。中国ちゅうごくでは、青森県あおもりけんの産業技術さんぎょうぎじゅつセンターの許可きょかをもらわないで、このりんごを売うったり育そだてたりすることが禁止きんしされています。しかし、許可きょかがないのに、中国ちゅうごくの会社かいしゃがウェブサイトで「千雪ちゆき」の木きの苗なえを売うっている可能性かのうせいがあることがわかりました。
産業技術さんぎょうぎじゅつセンターは、この苗なえが規則きそくを守まもらないで中国ちゅうごくに運はこばれたと考かんがえて、本当ほんとうに「千雪ちゆき」かどうか国くにと一緒いっしょに調しらべることにしました。「千雪ちゆき」だとわかったら、会社かいしゃに売うることをやめるように言いいます。
産業技術さんぎょうぎじゅつセンターは「日本にっぽんで初はじめて作つくったりんごや、りんごを育そだてている日本にっぽんの人ひとたちを守まもらなければなりません」と話はなしています。
