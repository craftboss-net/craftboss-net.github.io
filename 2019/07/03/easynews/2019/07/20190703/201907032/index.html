プラスチックの代かわりに竹たけでストローをつくる

发表于 2019-07-03 更新于 2019-11-30 分类于 JapanEasyNews



プラスチックのごみで海うみが汚よごれることが大おおきな問題もんだいになっています。静岡県しずおかけん浜松市はままつしは、プラスチックの代かわりに竹たけでストローを作つくって、レストランなどで使つかってもらうテストを始はじめます。
浜松市はままつしには使つかっていない竹たけの林はやしが９００ｈａ以上いじょうあります。浜松市はままつしはこの林はやしにある太ふとさ３ｍｍから５ｍｍの細ほそい竹たけでストローをつくります。竹たけは２０ｃｍの長ながさに切きって、熱あつい湯ゆで消毒しょうどくしたあと乾かわかします。
浜松市はままつしは竹たけのストローを１０００本ぼん作つくります。そして、レストランなどで使つかって問題もんだいがないかを調しらべて、これからたくさんの人ひとに使つかってほしいと考かんがえています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

プラスチック

  • 熱ねつや圧力あつりょくを加くわえて、自由じゆうに形かたちを作つくることができる物質ぶっしつ。特とくに合成樹脂ごうせいじゅしを指さす。使つかい道みちが広ひろい。

竹

  • 中なかが空からの茎くきに、節ふしのある植物しょくぶつ。日用品にちようひんや、細工物さいくものなどに使つかう。若わかい芽めを「たけのこ」という。

ストロー

  • 麦むぎわらやビニルなどで作つくった細ほそい管くだ。ジュースなどを飲のむときに使つかう。

ヘクタール

  • メートル法ほうで、土地とちの面積めんせきの単位たんいの一ひとつ。記号きごうは「ha」。１ヘクタールは、１００アールで、１万平方いちまんへいほうメートル。

消毒

  • 薬くすりや、熱ねつなどで、ばいきんを殺ころすこと。

乾かす

  • 日光にっこうや風かぜに当あてて、しめり気けを取とる。干ほす。
