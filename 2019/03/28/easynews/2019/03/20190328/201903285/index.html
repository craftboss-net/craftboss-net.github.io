外国人がいこくじんが来きた病院びょういんの２０%「お金かねを払はらっていない人ひとがいる」

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



厚生労働省こうせいろうどうしょうは去年きょねん、けがや病気びょうきで外国人がいこくじんが来きたことがあるかどうか、全部ぜんぶの病院びょういんに聞ききました。１０月がつの１か月げつに、けがや病気びょうきで来きた外国人がいこくじんを治療ちりょうした病院びょういんは１９６５でした。調査ちょうさに答こたえた病院びょういんの半分はんぶんぐらいでした。
この中なかの２０%ぐらいの病院びょういんは、お金かねを払はらっていない外国人がいこくじんがいると言いっています。払はらっていないお金かねの平均へいきんは、１つの病院びょういんで４２万まん円えんぐらいでした。１か月げつで１００万まん円えん以上いじょうになる病院びょういんもありました。
日本にっぽんで病院びょういんに行いく外国人がいこくじんは、これから増ふえると考かんがえられています。厚生労働省こうせいろうどうしょうは、外国人がいこくじんがお金かねを払はらいやすくする方法ほうほうなどをこれから考かんがえると言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

厚生労働省

  • 国民こくみんの健康けんこうや生活せいかつを守まもる仕事しごとや、労働者ろうどうしゃが仕事しごとを見みつけるのを助たすけたり、労働者ろうどうしゃを保護ほごしたりする国くにの役所やくしょ。厚労省こうろうしょう。

治療

  • 病気びょうきやけがの手当てあてをして治なおすこと。

調査

  • 事実じじつを明あきらかにするために、調しらべること。

平均

  • 多おおい少すくないや高たかい低ひくいなどがないように、ならすこと。
  • つり合あいがとれること。バランス。

方法

  • ものごとを行おこなうやり方かた。しかた。
