静岡県しずおかけんの海うみでサクラエビの春はるの漁りょうが始はじまる

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



サクラエビは体からだの長ながさが４ｃｍぐらいの小ちいさいえびで、色いろは桜さくらのようなピンクです。
静岡県しずおかけんの駿河湾するがわんでは、春はると秋あきにサクラエビの漁りょうを行おこないます。しかし、去年きょねんは子こどものえびが多おおかったため、秋あきの漁りょうはしませんでした。
とることができる大おおきさのえびが増ふえたため、今年ことしは３月がつ２６日にちから春はるの漁りょうが始はじまりました。最初さいしょの日ひにとったサクラエビは去年きょねんの１５％ぐらいの４ｔでした。
２７日にちの朝あさの競せりでは、１５ｋｇの値段ねだんが平均へいきんで１０万まん円えん以上いじょうになりました。去年きょねんの２倍ばい以上いじょうの高たかい値段ねだんです。
今年ことしの春はるの漁りょうは６月がつ５日いつかまで続つづきます。

国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

漁

  • 魚さかななどをとること。

海老・

  • 海うみや川かわなどにすみ、体からだが、節ふしのあるかたいからに包つつまれた動物どうぶつ。１０本ぽんの足あしと、２本にほんの長ながいひげがある。

桜

  • 山地さんちに生はえ、公園こうえんや庭にわにも植うえる木き。ソメイヨシノ・シダレザクラ・ヤマザクラなど種類しゅるいが多おおい。春はる、うすもも色いろの美うつくしい花はなが咲さく。日本にっぽんの「国花こっか」とされる。

ピンク

  • もも色いろ。うすい赤色あかいろ。

トン

  • メートル法ほうで、重おもさの単位たんいの一ひとつ。１トンは、１０００キログラム。記号きごうは「t」。
  • 船ふねの容積ようせきの単位たんい。
  • 貨車かしゃやトラックの容積ようせきの単位たんい。

競り売り

  • 買かい手てに競争きょうそうで値段ねだんをつけさせ、いちばん高たかい値段ねだんをつけた人ひとに、その品物しなものを売うる方法ほうほう。競売きょうばい。オークション。せり。

平均

  • 多おおい少すくないや高たかい低ひくいなどがないように、ならすこと。
  • つり合あいがとれること。バランス。
