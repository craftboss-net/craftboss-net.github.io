小学校しょうがっこうで使つかう教科書きょうかしょに初はじめてＬＧＢＴについて書かいた

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



２６日にち、来年らいねん４月がつから小学校しょうがっこうで使つかうことができる教科書きょうかしょが決きまりました。この中なかには、小学校しょうがっこうの教科書きょうかしょで初はじめて、ＬＧＢＴについて書かいた教科書きょうかしょがあります。ＬＧＢＴは、心こころと体からだの性せいが同おなじではないと感かんじる人ひとや、自分じぶんと同おなじ性せいの人ひとを好すきになる人ひとなどのことです。
光文書院こうぶんしょいんという会社かいしゃが作つくった３年生ねんせいと４年生ねんせいの教科書きょうかしょでは、心こころと体からだの性せいが同おなじではない人ひとの悩なやみなどについて書かいてあります。悩なやんでいる子こどもが電話でんわで相談そうだんできる所ところも紹介しょうかいしています。
文教社ぶんきょうしゃが作つくった５年生ねんせいと６年生ねんせいの教科書きょうかしょには、性せいのことで悩なやんでいる友達ともだちがいたら、みんなの個性こせいを大切たいせつにするという考かんがえ方かたを持もつことが必要ひつようだと書かいてあります。
早稲田大学わせだだいがくの棚村たなむら政行まさゆき先生せんせいは「まだ最初さいしょの一歩いっぽです。ＬＧＢＴについて子こどもたちにどうやって伝つたえたらいいか、もっと考かんがえなければならないと思おもいます」と話はなしています。
