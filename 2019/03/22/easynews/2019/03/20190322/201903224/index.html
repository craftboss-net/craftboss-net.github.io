セブン－イレブン　店みせの営業えいぎょう時間じかんを短みじかくする実験じっけんを始はじめる

发表于 2019-03-22 更新于 2019-11-30 分类于 JapanEasyNews



コンビニのセブン－イレブンは、１日にちに２４時間じかん店みせを開あけて営業えいぎょうしています。しかし、働はたらく人ひとが足たりなくて、２４時間じかん営業えいぎょうすることが難むずかしいと言いう店みせが出でています。
セブン－イレブンの会社かいしゃは、営業えいぎょう時間じかんを短みじかくする実験じっけんを１０の店みせで始はじめました。東京とうきょうの足立区あだちくの店みせは、営業えいぎょう時間じかんを午前ごぜん５時じから次つぎの日ひの午前ごぜん１時じまでにしました。午前ごぜん１時じになると、店員てんいんが看板かんばんの電気でんきを消けして、ドアに鍵かぎをかけました。そのあと掃除そうじなどをして、朝あさ５時じに店みせを開あける準備じゅんびをしていました。
会社かいしゃの人ひとは「これからも２４時間じかんの営業えいぎょうを続つづけますが、社会しゃかいは変かわっています。実験じっけんをして、どんな問題もんだいが出でるのかしっかり調しらべます」と話はなしていました。
