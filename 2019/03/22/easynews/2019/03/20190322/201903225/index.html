ニュージーランド　事件じけんで亡なくなった人ひとのために祈いのる会かい

发表于 2019-03-22 更新于 2019-11-30 分类于 JapanEasyNews



ニュージーランドのクライストチャーチで１５日にち、イスラム教きょうの人ひとが祈いのる場所ばしょのモスクで、男おとこが銃じゅうを撃うちました。この事件じけんで５０人にんが亡なくなりました。この事件じけんのあと、アーダーン首相しゅしょうは、大勢おおぜいの人ひとを殺ころす力ちからがある銃じゅうを売うったり持もったりすることを禁止きんししたいと話はなしました。
事件じけんから１週間しゅうかんが過すぎた２２日にち、モスクの近ちかくの公園こうえんで、亡なくなった人ひとのために祈いのる会かいがありました。アーダーン首相しゅしょうが「私わたしたちは一ひとつです」と話はなしたあと、集あつまったみんなで目めを閉とじて静しずかに祈いのりました。
事件じけんがあったモスクの人ひとは「ニュージーランドは、こんな事件じけんでは壊こわれないことがわかりました」と話はなしました。そして、事件じけんで被害ひがいを受うけた人ひとや家族かぞくを、たくさんの人ひとが助たすけてくれたことに感謝かんしゃしていると話はなしました。
