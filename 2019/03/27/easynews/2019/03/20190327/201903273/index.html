１０歳さいの仲邑なかむら菫すみれさん「世界せかいで戦たたかう囲碁いごのプロになりたい」

发表于 2019-03-27 更新于 2019-11-30 分类于 JapanEasyNews



囲碁いごの団体だんたいの日本棋院にほんきいんが２６日にち、囲碁いごのプロだと証明しょうめいする免状めんじょうを渡わたす式しきを行おこないました。４月がつ１日ついたちにプロになる１３人にんが免状めんじょうをもらいました。この中なかには、今いままででいちばん若わかい１０歳さいでプロになる仲邑なかむら菫すみれさんもいました。仲邑なかむらさんは「世界せかいで戦たたかうことができるプロになりたいです」と言いいました。
仲邑なかむらさんは、日本棋院にほんきいんの新あたらしい制度せいどでプロになりました。この制度せいどは、世界せかいの試合しあいで勝かつことができる人ひとを育そだてるために、強つよい小学生しょうがくせいを選えらんでプロにします。日本棋院にほんきいんは「囲碁いごが強つよい中国ちゅうごくや韓国かんこくとも戦たたかうことができるように若わかい人ひとたちを手伝てつだっていきたいと思おもいます」と話はなしています。
