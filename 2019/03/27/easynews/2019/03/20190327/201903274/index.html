２０年ねん前まえに盗ぬすまれた４億おく円えんのピカソの絵えが見みつかる

发表于 2019-03-27 更新于 2019-11-30 分类于 JapanEasyNews



パブロ?ピカソはスペインの有名ゆうめいな画家がかです。２０年ねん前まえにフランスで盗ぬすまれたピカソの絵えが、今月こんげつオランダで見みつかりました。
この絵えは、ピカソが１９３８年ねんにかいた女性じょせいの絵えです。外国がいこくの新聞しんぶんによると、サウジアラビア人じんのお金持かねもちが４億おく円えんで絵えを買かいましたが、１９９９年ねんにフランスの港みなとで船ふねの中なかから盗ぬすまれました。それからずっと、絵えがどこにあるかわかりませんでした。
オランダ人じんの美術品びじゅつひんの専門家せんもんかは、首都しゅとのアムステルダムにこの絵えを持もっている人ひとがいるようだと知しって、その人ひとと何なん度ども話はなしをしました。そして、絵えを渡わたしてもらいました。アメリカの専門家せんもんかが調しらべると、本当ほんとうのピカソの絵えだとわかりました。
