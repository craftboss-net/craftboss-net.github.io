大分県おおいたけん　タトゥーがある人ひとも入はいれる温泉おんせんを英語えいごで紹介しょうかいする

发表于 2019-03-07 更新于 2019-11-30 分类于 JapanEasyNews



タトゥーは肌はだに傷きずをつけてかいた字じや絵えで、入いれ墨ずみとも言いいます。世界せかいではタトゥーが昔むかしからの文化ぶんかになっている所ところもあります。しかし、日本にっぽんではタトゥーがある人ひとは怖こわいと思おもう人ひとがいるため、タトゥーがあると入はいることができない温泉おんせんもあります。
温泉おんせんが有名ゆうめいな大分県おおいたけんでは、今年ことしの秋あき、ラグビーのワールドカップで５つの試合しあいを行おこないます。外国がいこくから来くる人ひとたちに温泉おんせんを楽たのしんでもらうため、大分県おおいたけんはタトゥーがある人ひとも入はいることができる温泉おんせんを英語えいごのウェブサイトで紹介しょうかいすることにしました。ウェブサイトは３月がつ中旬ちゅうじゅんから始はじめる予定よていです。
大分県おおいたけんは「文化ぶんかが違ちがう外国がいこくの人ひとも日本にっぽんの温泉おんせんを楽たのしむことができるようにしたいです」と話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

温泉

  • 地下水ちかすいが火山かざんなどの熱ねつで温あたためられて、地下ちかからわき出だす湯ゆ。いろいろな成分せいぶんがとけていて病気びょうきに効きく。また、その湯ゆの出でる場所ばしょ。

肌

  • 体からだの皮膚ひふ。
  • 土地とちなどの表面ひょうめん。
  • 性質せいしつ。

傷

  • 打うったり切きったりして、皮膚ひふや肉にくをいためたところ。
  • 品物しなもののいたんだところ。
  • 悪わるい点てん。欠点けってん。

付ける

  • 物ものが、はなれないようにする。
  • しるしを残のこす。
  • 書かきこむ。しるす。
  • 加くわえる。
  • そっとあとについて行いく。
  • ものごとを収おさめる。
  • 注意ちゅういを向むける。
  • 値段ねだんを決きめる。
  • そばに付つきそわせる。
  • …し慣なれている。

ラグビー

  • １チーム１５人にんずつの選手せんしゅが、楕円形だえんけいのボールを、持もって走はしったり、けったりして相手側あいてがわのゴールライン内ないに入いれ、得点とくてんを争あらそう競技きょうぎ。

ワールドカップ

  • スポーツ競技きょうぎの世界選手権大会せかいせんしゅけんたいかいの優勝者ゆうしょうしゃにあたえられるカップ。また、そのカップを争あらそう大会たいかい。W杯ダブリューはい。

ウェブサイト

  • インターネットで、ひとまとまりの情報じょうほうが置おかれている場所ばしょ。サイト。

中旬

  • 月つきの１１じゅういち日にちから２０日はつかまでの間あいだ。
