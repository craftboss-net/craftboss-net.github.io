仕事しごとの途中とちゅうに少すこし寝ねて休やすむことができる「睡眠すいみんカフェ」

发表于 2019-03-07 更新于 2019-11-30 分类于 JapanEasyNews



食たべ物ものの会社かいしゃのネスレが、仕事しごとの途中とちゅうなどに少すこし寝ねて休やすむことができる「睡眠すいみんカフェ」という店みせを東京都とうきょうと品川区しながわくに作つくりました。
店みせにはベッドが６台だいと、背中せなかの部分ぶぶんが後うしろに倒たおれるいすが４つあります。隣となりの人ひとから見みえないように、カーテンのような物ものもあります。
利用りようする時間じかんの長ながさは５つのコースから選えらびます。１時間じかんのコースは寝ねる前まえにカフェインが入はいっていないコーヒー、起おきたあとにカフェインが入はいったコーヒーが１杯ぱいずつ出でて、１５００円えんです。
ネスレは、仕事しごとをしている人ひとなど寝ねる時間じかんが足たりない人ひとに店みせを利用りようしてほしいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ
