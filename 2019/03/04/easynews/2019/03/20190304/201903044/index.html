ザギトワ選手せんしゅが日本にっぽんからもらった「マサル」が１歳さいになる

发表于 2019-03-04 更新于 2019-11-30 分类于 JapanEasyNews



ロシアのモスクワで３日みっか、日本にっぽんの文化ぶんかを紹介しょうかいする「サクラフェスティバル」というイベントがありました。
会場かいじょうには、オリンピックのフィギュアスケートで金きんメダルを取とったアリーナ?ザギトワ選手せんしゅが来きました。そして、日本にっぽんからもらった秋田犬あきたいぬの「マサル」をみんなに紹介しょうかいしました。
ザギトワ選手せんしゅがもらったとき、生うまれて３か月げつだったマサルは、先月せんげつ１歳さいになって、体からだの重おもさは２２ｋｇ以上いじょうになりました。ザギトワ選手せんしゅが座すわってと指示しじしてマサルが座すわると、みんな喜よろこんでいました。
ザギトワ選手せんしゅは、両親りょうしんに「秋田犬あきたいぬを飼かいたい」と頼たのんだときの話はなしをしました。お母かあさんは「オリンピックでいい演技えんぎができたらいいよ」と答こたえたので、ザギトワ選手せんしゅは頑張がんばろうという気持きもちになったと言いっていました。
ザギトワ選手せんしゅは「マサルはとてもおとなしくて、みんながかわいいと思おもうような犬いぬで、そして頭あたまがいいです」と話はなしていました。
