藤井聡太ふじいそうたさんなど多おおくの将棋しょうぎのプロが食事しょくじをした店みせが閉しまる

发表于 2019-03-29 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうの将棋しょうぎ会館かいかんの近ちかくにあるそば 店てんの「みろく庵あん」には、３０年ねん以上いじょう前まえから、加藤かとう一二三ひふみさんなどたくさんの将棋しょうぎのプロが来きていました。プロたちが将棋しょうぎ会館かいかんで試合しあいをしている途中とちゅうで、この店みせに食事しょくじを届とどけてもらうこともありました。
プロで高校生こうこうせいの藤井聡太ふじいそうたさんが、おととし２９回かい続つづけて勝かった記録きろくをつくったとき、この店みせの「豚ぶた キムチ うどん」を食たべたことも有名ゆうめいになりました。
みろく庵あんは、店みせが入はいっているビルの工事こうじがあるため、３月がつ３１日にちで閉しまることになりました。店みせには、将棋しょうぎのファンなどがたくさん来きています。将棋しょうぎ会館かいかんに将棋しょうぎをしに来きた高校生こうこうせいは「プロと同おなじごはんを食たべることができて、いい思おもい出でになりました」と話はなしていました。
この店みせの人ひとは「最後さいごまで一生懸命いっしょうけんめい食事しょくじを作つくりたいです」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

将棋

  • 縦たて・横よこに１０本ぽんの線せんを引ひいた板いたの上うえで、２０枚まいずつのこまを動うごかし、相手あいての王おうを先さきに取とるゲーム。

プロ

  • 英語えいごの「プロフェッショナル」の略りゃく。職業しょくぎょうにすること。本職ほんしょく。専門せんもん。
  • 「プロダクション」の略りゃく。

そば

  • 畑はたけに作つくる作物さくもつ。赤あかみのある茎くきに、秋あきの初はじめごろ、白しろい花はなをつける。実みからそば粉こをとる。
  • そば粉こを水みずでこねて細ほそく切きった、めん類るい

店

  • みせ。

記録

  • あとのために書かきつけておくこと。また、書かきつけたもの。
  • 運動競技うんどうきょうぎなどの最高さいこうの成績せいせき。レコード。

豚

  • 人ひとに飼かわれているけもの。ずんぐり太ふとっている。肉にくは食たべられ、皮かわも使つかわれる。

キムチ

  • 白菜はくさい・大根だいこんなどをトウガラシ・ニンニクなどで漬つけこんだ漬つけ物もの。

うどん

  • 小麦粉こむぎこを、水みずでこねてうすくのばし、細長ほそながく切きってゆでた食たべ物もの。

工事

  • 建物たてもの・道路どうろ・橋はしなどを造つくったり、直なおしたりすること。また、その仕事しごと。

ファン

  • スポーツや映画えいが・演劇えんげきなどが、特別とくべつに好すきな人ひと。または、ある俳優はいゆうや、選手せんしゅなどを熱心ねっしんに応援おうえんする人ひと。フアン。
  • 扇風機せんぷうき。送風機そうふうき。換気扇かんきせん。

思い出

  • 過すぎ去さったことを、思おもい返かえすこと。また、そのことがら。
