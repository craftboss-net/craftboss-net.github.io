「ゲノム編集へんしゅう」の食品しょくひんを売うるときのルールが決きまる

发表于 2019-03-19 更新于 2019-11-30 分类于 JapanEasyNews



「ゲノム編集へんしゅう」は遺伝子いでんしの情報じょうほうを変かえる技術ぎじゅつです。この技術ぎじゅつを使つかって、米こめがたくさんとれるようにしたり、魚さかなの体からだを大おおきくしたりする研究けんきゅうが進すすんでいます。今いまのほとんどの研究けんきゅうは、別べつの遺伝子いでんしを外そとから入いれないで、中なかにある遺伝子いでんしを変かえるやり方かたです。
１８日にち、厚生労働省こうせいろうどうしょうは会議かいぎを開ひらいて「ゲノム編集へんしゅう」の食品しょくひんを売うるときのルールを決きめました。
遺伝子いでんしを変かえただけの食品しょくひんの場合ばあい、体からだに悪わるくないかどうかなど、安全あんぜんについては調しらべる必要ひつようがなくなりました。そして、国くにに書類しょるいを出だしたら売うることができるようにします。
厚生労働省こうせいろうどうしょうは、これから詳くわしい ルールを決きめて、夏なつごろから始はじめる予定よていです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

食品

  • 食たべ物ものとなるもの。食たべ物もの。

ルール

  • 規則きそく。決きまり。

遺伝子

  • 染色体せんしょくたいの中なかにあって、遺伝いでんのはたらきをする物質ぶっしつ。

情報

  • できごとやものごとのようすについての知しらせ。

取れる

  • はなれて落おちる。
  • つかれなどが、なくなる。
  • その状態じょうたいになる。
  • 取とることができる。
  • そう考かんがえられる。
  • 作物さくもつなどが得えられる。

厚生労働省

  • 国民こくみんの健康けんこうや生活せいかつを守まもる仕事しごとや、労働者ろうどうしゃが仕事しごとを見みつけるのを助たすけたり、労働者ろうどうしゃを保護ほごしたりする国くにの役所やくしょ。厚労省こうろうしょう。

書類

  • 書かき物もの。書かきつけ。文書ぶんしょ。

詳しい

  • 細こまかい。
  • 細こまかいことまでよく知しっている。
