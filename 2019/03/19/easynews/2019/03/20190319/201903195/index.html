オランダ　男おとこが電車でんしゃの中なかで銃じゅうを撃うって３人にんが亡なくなる

发表于 2019-03-19 更新于 2019-11-30 分类于 JapanEasyNews



オランダのユトレヒトで１８日にちの午前ごぜん、電車でんしゃの中なかで男おとこが銃じゅうを撃うちました。電車でんしゃに乗のっていた３人にんが亡なくなって、５人にんがけがをしました。
警察けいさつが電車でんしゃにあるカメラの映像えいぞうなどを調しらべると、トルコ出身しゅっしんの３７歳さいの男おとこが銃じゅうを撃うって、車くるまで逃にげたことがわかりました。警察けいさつは１８日にちの夜よる、３ｋｍぐらい離はなれたビルで男おとこを見みつけて逮捕たいほしました。
トルコのテレビなどは、男おとこは自分じぶんの親戚しんせきを狙ねらっていたと伝つたえていますが、男おとこがどうして銃じゅうを撃うったかはまだわかっていません。オランダの政府せいふは、テロの可能性かのうせいもあると考かんがえて調しらべています。
