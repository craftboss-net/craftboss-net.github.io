ＪＡＸＡ「地球ちきゅうから３億おくｋｍの小ちいさい星ほしに水みずの成分せいぶんがある」

发表于 2019-03-20 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのＪＡＸＡは、地球ちきゅうから３億おくｋｍの所ところにあるリュウグウという小ちいさい星ほしを調しらべるため、４年ねんぐらい前まえに「はやぶさ２」を打うち上あげました。
「はやぶさ２」がリュウグウに着ついたあと、ＪＡＸＡと大学だいがくなどのチームは赤外線せきがいせんの光ひかりを使つかって、リュウグウの表面ひょうめんの岩いわや石いしを２か月げつ調しらべました。その結果けっか、岩いわや石いしの中なかに水みずの成分せいぶんがあることがわかりました。
ＪＡＸＡは、「はやぶさ２」は今年ことし２月がつにリュウグウの石いしを取とることができたようだと言いっています。この石いしを地球ちきゅうに持もって帰かえって調しらべたら、地球ちきゅうで生いき物ものが生うまれるために必要ひつような水みずがどこから来きたかわかるかもしれません。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

地球

  • 太陽系たいようけいの惑星わくせいの一ひとつ。太陽たいようから三番さんばんめの星ほしで、わたしたちが住すんでいる天体てんたい。自分じぶんで回まわりながら（自転じてん）、さらに太陽たいようの周まわりを３６５日にちで回まわっている（公転こうてん）。

成分

  • 物ものを作つくり上あげている一ひとつ一ひとつのもとになる物質ぶっしつ。

打ち上げる

  • （勢いきおいよく）物ものを空中くうちゅうに上あげる。
  • 波なみが物ものを陸りくに運はこび上あげる。
  • （仕事しごとなどを）すっかり終おえる。

チーム

  • 競技きょうぎや仕事しごとをするときの、組くみや団体だんたい。

赤外線

  • プリズムで日光にっこうを分わけたとき、赤色あかいろの外側そとがわに現あらわれる目めに見みえない光線こうせん。物ものを温あたためるので熱線ねっせんともいう。医療いりょう・通信つうしん・写真しゃしんなどに使つかわれる。

表面

  • 物ものの外側そとがわ。表おもて。
  • うわべ。見みせかけ。

岩

  • 石いしの大おおきなもの。岩石がんせき。

結果

  • あることがもとになって起おこったことがらやようす。

生き物

  • 生いきているもの。生物せいぶつ。特とくに、動物どうぶつのこと。
