女性じょせい議員ぎいんの割合わりあい　日本にっぽんは世界せかいで１６５番目ばんめ

发表于 2019-03-08 更新于 2019-11-30 分类于 JapanEasyNews



スイスにある団体だんたいは、世界中せかいじゅうの国くにの議会ぎかいの中なかで女性じょせい議員ぎいんがどのくらいいるか調しらべて、今年ことし１月がつ１日ついたちの結果けっかを発表はっぴょうしました。
１９３の国くにのうち、下院かいんの議会ぎかいなどで女性じょせい議員ぎいんの割合わりあいがいちばん多おおいのはアフリカのルワンダで６１．３％でした。
日本にっぽんは１０．２％で、１６５番目ばんめでした。前まえの年としより７つ順番じゅんばんが下さがりました。
７つの先進国せんしんこく「Ｇ７」の中なかで最もっとも割合わりあいが多おおいフランスは３９．７％で、世界せかいで１６番目ばんめでした。次つぎのイタリアは３５．７％で３０番目ばんめ、その次つぎのアメリカは２３．５％で７８番目ばんめでした。Ｇ７の中なかで１００番ばん以下いかなのは、日本にっぽんだけでした。
世界せかい全体ぜんたいの女性じょせい議員ぎいんの割合わりあいは、１９９５年ねんに１１．３％でした。今年ことしは２４．３％で、２倍ばいになりました。
