福岡市ふくおかし　猫ねこを飼かう人ひとだけが住すむマンション

发表于 2019-03-06 更新于 2019-11-30 分类于 JapanEasyNews



福岡市ふくおかしに、猫ねこを飼かっている人ひとや飼かう予定よていの人ひとだけが住すむマンションができました。部屋へやは全部ぜんぶで２０あります。
部屋へやの壁かべは、猫ねこの爪つめでも傷きずつきにくくなっていて、猫ねこが運動うんどうできる階段かいだんも付ついています。猫ねこが外そとを見みるための小ちいさい窓まどや、猫ねこが通とおることができる小ちいさいドアもあります。住すんでいる人ひとは、出でかけていても、猫ねこのためにスマートフォンで部屋へやの電気でんきやエアコンをつけることができます。
マンションの会社かいしゃは「最近さいきん、猫ねこを飼かう人ひとがとても多おおくなっています。猫ねこも飼かっている人ひともこのマンションで幸しあわせに生活せいかつしてほしいです」と話はなしています。
