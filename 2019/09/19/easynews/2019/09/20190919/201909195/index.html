日本にっぽんの政府せいふ　１０月がつに北方四島ほっぽうよんとうへ観光かんこうに行いくツアーを行おこなう

发表于 2019-09-19 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんとロシアの政府せいふは、北海道ほっかいどうの東ひがしにある北方四島ほっぽうよんとうの経済けいざいをよくするために、一緒いっしょにいろいろな仕事しごとをすることにしています。今年ことし６月がつの話はなし合あいで、安倍あべ総理大臣そうりだいじんとロシアのプーチン大統領だいとうりょうは、日本にっぽんから北方四島ほっぽうよんとうへ観光かんこうに行いくツアーを行おこなってみることにしました。
ツアーは１０月がつ１１日にちから１６日にちに行おこないます。５０人にんぐらいが北方四島ほっぽうよんとうの国後島くなしりとうと択捉島えとろふとうへ行いきます。北海道ほっかいどうの根室ねむろから船ふねで国後島くなしりとうへ行いって、ロシア正教せいきょうの教会きょうかいや博物館はくぶつかん、海岸かいがんの景色けしきを見みます。択捉島えとろふとうでは白しろい崖がけがある海岸かいがんや温泉おんせんへ行いきます。
日本にっぽんの政府せいふはこのツアーを行おこなって、これから北方四島ほっぽうよんとうでロシアと一緒いっしょに仕事しごとをするときにどんな問題もんだいがあるか知しりたいと考かんがえています。そして、ロシアと平和へいわ条約じょうやくを結むすぶための話はなし合あいを進すすめたいと考かんがえています。

  • 観光かんこうやスポーツなどのために、団体だんたいで行いく旅行りょこう。

話し合い

  • 話はなし合あうこと。相談そうだん。

総理大臣

  • 内閣総理大臣ないかくそうりだいじんのこと。内閣ないかくの最高責任者さいこうせきにんしゃで、国会議員こっかいぎいんの中なかから議員ぎいんが選えらび、天皇てんのうが認みとめて決きまる。首相しゅしょう。総理そうり。

大統領

  • 共和国きょうわこくで、その国くにを代表だいひょうする人ひと。

博物館

崖

  • 山やまや海岸かいがんなどの、けずり取とられて険けわしい所ところ。

温泉

  • 地下水ちかすいが火山かざんなどの熱ねつで温あたためられて、地下ちかからわき出だす湯ゆ。いろいろな成分せいぶんがとけていて病気びょうきに効きく。また、その湯ゆの出でる場所ばしょ。
