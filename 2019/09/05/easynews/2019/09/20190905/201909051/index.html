障害しょうがいがある人ひともない人ひとも使つかいたくなる機械きかいを紹介しょうかいする

发表于 2019-09-05 更新于 2019-11-30 分类于 JapanEasyNews



体からだなどに障害しょうがいがある人ひとだけではなくて、みんなが使つかいたくなるような便利べんりでかっこいい機械きかいを紹介しょうかいするイベントが、東京都とうきょうと渋谷区しぶやくで開ひらかれています。
会場かいじょうに集あつまった２０以上いじょうの機械きかいの中なかには、電気でんきで動うごく車くるまいすがあります。この車くるまいすは、カメラやセンサーを使つかって周まわりの物ものにぶつからないようにしながら、行いきたい所ところまで自動じどうで進すすみます。広ひろいショッピングセンターなどで、たくさん歩あるくことが大変たいへんな人ひとも利用りようすることができます。
会場かいじょうではほかに、眼鏡めがねに付つけて使つかう機械きかいも紹介しょうかいしています。この機械きかいは、中なかにカメラが入はいっていて、眼鏡めがねをかけた人ひとが見みた字じを読よんでくれます。腕うで時計とけいを見みるときのように腕うでを動うごかすと、今いまの時間じかんも教おしえてくれます。目めに障害しょうがいがある人ひとだけではなくて、漢字かんじがわからない子こどもなどにも役やくに立たちそうです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

障害

  • 何なにかをするときに、じゃまになるものごと。さまたげ。
  • 体からだのはたらきに故障こしょうがあること。

イベント

  • 行事ぎょうじ。できごと。
  • 勝負しょうぶ。試合しあい。

車椅子

  • 足あしの不自由ふじゆうな人ひとや病気びょうきの人ひとなどが、腰こしかけたまま移動いどうできるように車くるまをつけた、いす。

センサー

  • 光ひかりや音おと、温度おんどなどに反応はんのうして、電気的でんきてきな信号しんごうを送おくる装置そうち。

ぶつかる

  • 打うち当あたる。つき当あたる。
  • 困難こんなんなことに出であう。
  • 争あらそう。立たち向むかう。
  • 重かさなる。

自動

  • 機械きかいなどが自分じぶんの力ちからで動うごくこと。

付ける

  • 物ものが、はなれないようにする。
  • しるしを残のこす。
  • 書かきこむ。しるす。
  • 加くわえる。
  • そっとあとについて行いく。
  • ものごとを収おさめる。
  • 注意ちゅういを向むける。
  • 値段ねだんを決きめる。
  • そばに付つきそわせる。
  • …し慣なれている。

動かす

  • 動うごくようにする。
  • 位置いちを変かえる。
  • 感動かんどうさせる。
