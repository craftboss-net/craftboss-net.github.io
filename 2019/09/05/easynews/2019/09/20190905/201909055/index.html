香港ほんこんの政府せいふ「中国ちゅうごくに容疑者ようぎしゃを渡わたす規則きそくを作つくることをやめる」

发表于 2019-09-05 更新于 2019-11-30 分类于 JapanEasyNews



香港ほんこんの政府せいふは４日よっか、香港ほんこんに逃にげてきた容疑者ようぎしゃを中国ちゅうごくに渡わたすことができる規則きそくを作つくることをやめると発表はっぴょうしました。
香港ほんこんでは、この規則きそくを作つくることに反対はんたいする人ひとたちがデモなどをして３か月げつぐらい抗議こうぎを続つづけています。国際こくさい空港くうこうや地下鉄ちかてつを使つかうことができなくなったこともありました。香港ほんこんの政府せいふは、デモなどが起おこらないようにしようと考かんがえているようです。
抗議こうぎしている人ひとたちは、民主的みんしゅてきな選挙せんきょをしてほしいなど、ほかにも政府せいふに４つの要求ようきゅうをしています。このため、多おおくの人ひとがこれからも抗議こうぎを続つづけると言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

政府

  • 国くにの政治せいじを行おこなうところ。
  • 内閣ないかく。

容疑者

  • 罪つみをおかした疑うたがいのある人ひと。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

デモ

  • 英語えいごの「デモンストレーション」の略りゃく。自分じぶんたちの考かんがえを相手あいてに認みとめさせようとして、大勢おおぜいが集あつまって自分じぶんたちの勢いきおいを示しめすこと。また、そのための行進こうしん。

抗議

  • 相手あいてに対たいして反対はんたいの考かんがえを強つよく言いうこと。また、反対はんたいの意見いけん。

起こる

  • 始はじまる。
  • 生しょうじる。起おきる。

民主的

  • 大勢おおぜいの人ひとの考かんがえを、だいじにするようす。

選挙

  • ある役目やくめにつく人ひとを、大勢おおぜいの中なかから選えらぶこと。

要求

  • こうしてほしいと強つよく求もとめること。
