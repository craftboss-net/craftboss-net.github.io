真珠しんじゅを作つくる貝かいがたくさん死しんでいる

发表于 2019-09-05 更新于 2019-11-30 分类于 JapanEasyNews



愛媛県えひめけんと三重県みえけんは、真珠しんじゅの生産せいさんがとても盛さかんです。真珠しんじゅはアコヤガイという貝かいの中なかにできます。今年ことしの夏なつ、２つの県けんで育そだてているアコヤガイがたくさん死しんでいることがわかりました。県けんなどによると、原因げんいんはわかっていません。
愛媛県えひめけんで４０年ねんぐらいアコヤガイを育そだてている人ひとは「８０％ぐらいの子こどもの貝かいが死しんでいます。来年らいねんは、真珠しんじゅを作つくる大人おとなの貝かいがとても少すくなくなるかもしれません」と話はなしています。
三重県みえけんは、世界せかいで初はじめて、アコヤガイを育そだてて真珠しんじゅの生産せいさんをした所ところです。真珠しんじゅの生産せいさんをしている人ひとは「店みせで売うる真珠しんじゅはこれから少すくなくなると思おもいます」と話はなしています。
