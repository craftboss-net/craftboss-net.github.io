最後さいごの「エアレース」の世界せかい選手権せんしゅけんで室屋むろや選手せんしゅが優勝ゆうしょう

发表于 2019-09-09 更新于 2019-11-30 分类于 JapanEasyNews



「エアレース」は、海うみの上うえに作つくったコースを、時速じそく３７０ｋｍが出でる小ちいさい飛行機ひこうきで飛とんで、タイムの速はやさを競争きょうそうするスポーツです。千葉市ちばしで世界せかい選手権せんしゅけんがあって、世界せかいの国くにから１４人にんが出でました。
日本にっぽんの室屋むろや義秀よしひで選手せんしゅは、今いままで２回かい優勝ゆうしょうしたことがあります。室屋むろや選手せんしゅは最初さいしょのレースで負まけましたが、負まけた人ひとの中なかでタイムがいちばん速はやかったため、次つぎのレースに進すすむことができました。そして最後さいごに４人にんで行おこなったレースで勝かって、優勝ゆうしょうしました。
エアレースの世界せかい選手権せんしゅけんは今年ことしが最後さいごです。室屋むろや選手せんしゅは「世界せかい選手権せんしゅけんはエアレースのことを全然ぜんぜん知しらなかった人ひとにも知しってもらう機会きかいになりました。すごく感謝かんしゃしています」と話はなしていました。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

選手権

  • 試合しあいや大会たいかいで、いちばんすぐれた選手せんしゅやチームにあたえられる資格しかくや地位ちい。また、その試合しあいや大会たいかい。

選手

  • 競技きょうぎに出でるために選えらばれた人ひと。

優勝

  • 競技きょうぎなどで、一位いちいで勝かつこと。

コース

  • 進すすんで行いく道みち。進路しんろ。
  • 競走きょうそうで、そこを通とおるように決きめられた道みち。
  • 学習がくしゅうの内容ないようや順序じゅんじょ。課程かてい。

時速

  • 一時間いちじかんに進すすむ速はやさ。

タイム

  • 時間じかん。時刻じこく。
  • 試合しあいなどを、少すこしの間あいだ、中止ちゅうしすること。
  • 競走きょうそうなどでかかった時間じかん。

レース

  • 競走きょうそう・競泳きょうえいなどの競技きょうぎ。
  • 競争きょうそう。

感謝

  • ありがたいと思おもう気持きもち。また、その気持きもちを表あらわすこと。
