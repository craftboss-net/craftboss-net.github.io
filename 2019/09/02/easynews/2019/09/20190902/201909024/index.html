来月らいげつから消費税しょうひぜいが上あがる　レジを作つくる工場こうじょうが忙いそがしい

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



来月らいげつ１日ついたちから消費税しょうひぜいが上あがります。食たべ物ものや飲のみ物ものを買かう場合ばあい、消費税しょうひぜいは今いまの８％のままですが、店みせの中なかで食たべたり飲のんだりする場合ばあいは１０％になります。この複雑ふくざつな計算けいさんができる新あたらしいレジを買かう店みせが増ふえています。
岩手県いわてけんにあるレジを作つくっている会社かいしゃの工場こうじょうには、日本中にほんじゅうのスーパーや店みせから去年きょねんの３倍ばいの注文ちゅうもんが来きています。工場こうじょうは、働はたらく人ひとや働はたらく時間じかんをいつもより増ふやして、急いそいでレジを作つくっています。社長しゃちょうは「レジを新あたらしくする店みせには国くにがお金かねを出だすので、注文ちゅうもんが多おおくなっているようです。お客様きゃくさまが困こまらないように頑張がんばって届とどけます」と話はなしていました。
レジを作つくっているほかの会社かいしゃも売うり上あげが去年きょねんの２倍ばいぐらいになっています。
注文ちゅうもんしたレジが今月中こんげつちゅうに届とどかない店みせでは、電卓でんたくで計算けいさんして、領収書りょうしゅうしょを書かくことになりそうです。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

消費税

  • 物ものを買かったり、サービスを受うけたりしたときに、かけられる税金ぜいきん。

計算

  • 数かずや量りょうを数かぞえること。
  • 式しきを解といて、答こたえを出だすこと。
  • 考かんがえに入いれておくこと。

注文

  • 品物しなものを作つくらせたり、届とどけさせたりすること。
  • 希望きぼうや条件じょうけんを出だすこと。

増やす

  • 多おおくする。

売り上げ

  • 品物しなものを売うって得えたお金かね。

届く

  • 送おくった物ものが着つく。
  • のばしたものが、つく。
  • 行いきわたる。
  • 願ねがいなどがかなう。

電卓

  • 電子でんしのはたらきで計算けいさんをする、小型こがたの機械きかい。

領収書

  • お金かねを受うけ取とったしるしに出だす証明書しょうめいしょ。受取うけとり。
