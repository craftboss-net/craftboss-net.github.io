スーパーコンピューター「京けい」のスイッチを切きる式しきを行おこなう

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



神戸市こうべしにある理化学りかがく研究所けんきゅうしょでは、新あたらしいスーパーコンピューター「富岳ふがく」を使つかうことになりました。８月がつ３０日にち、今いままで使つかっていた「京けい」のスイッチを切きる式しきがありました。
理化学りかがく研究所けんきゅうしょの人ひとは「少すこし寂さびしいですが、『富岳ふがく』も『京けい』と同おなじように役やくに立たつようにしたいと思おもいます」とあいさつしました。そして理化学りかがく研究所けんきゅうしょの人ひとなどが、「京けい」が置おいてある部屋へやで４つのスイッチを１つずつ切きっていきました。
国くにの計画けいかくでつくった「京けい」は２０１２年ねんにできました。計算けいさんの速はやさで世界せかいで１番ばんになったことがあります。ビッグデータを分析ぶんせきする能力のうりょくでは、何なん回かいも世界せかいで１番ばんになりました。地震じしんの被害ひがいの予測よそくや薬くすりの研究けんきゅうなどにとても役やくに立たちました。
「京けい」のあとに使つかう「富岳ふがく」は、「京けい」の１００倍ばい速はやく計算けいさんすることができます。理化学りかがく研究所けんきゅうしょは、２年ねんあとに「富岳ふがく」を使つかい始はじめたいと言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

スイッチ

  • 電流でんりゅうを通とおしたり、止とめたり、切きりかえたりする装置そうち。
  • ほかのものに切きりかえること。

計算

  • 数かずや量りょうを数かぞえること。
  • 式しきを解といて、答こたえを出だすこと。
  • 考かんがえに入いれておくこと。

分析

  • ものごとを、細こまかく分わけて、成なり立たちなどを調しらべること。

能力

  • ものごとを成なしとげることのできる力ちから。

被害

  • 損害そんがいを受うけること。また、受うけた害がい。

予測

  • 前まえもって、こうなるだろうと、見当けんとうをつけること。
