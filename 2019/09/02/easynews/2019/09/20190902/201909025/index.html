韓国かんこくで交流こうりゅうのイベントを今年ことしも開ひらく　日本にっぽんと韓国かんこくの文化ぶんかを紹介しょうかい

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



９月がつ１日ついたち、韓国かんこくで日本にっぽんと韓国かんこくの文化ぶんかを紹介しょうかいして交流こうりゅうするイベントがありました。
ステージではまず、日本にっぽんと韓国かんこくの子こどもたちがそれぞれの国くにの歌うたを一緒いっしょに歌うたいました。そして、韓国かんこくの政府せいふの人ひとと日本にっぽんの国会議員こっかいぎいんが、隣となりの国くにをよく知しって大切たいせつに思おもうことが必要ひつようだなどとあいさつをしました。そのあと、日本にっぽんの踊おどりや韓国かんこくの音楽おんがくを紹介しょうかいしました。
会場かいじょうに来きた人ひとたちは、日本にっぽんの着物きものや韓国かんこくのチマチョゴリを着きたりして楽たのしんでいました。
このイベントは毎年まいとし開ひらいていますが、今年ことしは初はじめて、会場かいじょうで荷物にもつのチェックがありました。
韓国かんこくの女性じょせいは「今いまは２つの国くにの関係かんけいがよくなくても、このようなイベントで関係かんけいがよくなっていくといいと思おもいます」と話はなしていました。
