タトゥーシールで顔かおや腕うでが痛いたくなることがある　気きをつけて

发表于 2019-09-25 更新于 2019-11-30 分类于 JapanEasyNews



「タトゥーシール」は絵えや字じがかいてあるシールで、顔かおや腕うでなどに貼はって楽たのしみます。スポーツのイベントなどのときに使つかう人ひとが増ふえています。
消費者庁しょうひしゃちょうによると、タトゥーシールを貼はった所ところがかゆくなったり、痛いたくなったりした人ひとがいます。
このため、国民生活こくみんせいかつセンターは、いろいろなタトゥーシールを調しらべました。顔かおや体からだに絵えをかくときに使つかう絵えの具ぐも調しらべました。その結果けっか、アレルギーの危険きけんがある金属きんぞくを使つかっているのに、袋ふくろに書かいていない物ものがありました。化粧けしょう品ひんに使つかってはいけない成分せいぶんを使つかっている物ものもありました。
消費者庁しょうひしゃちょうは、タトゥーシールなどを使つかうときは、最初さいしょに腕うでの内側うちがわなどでテストをして、大丈夫だいじょうぶかどうか調しらべるように言いっています。もし痛いたくなったら、病院びょういんへ行いくように言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

シール

  • ふうとうなどの、閉とじたところにはる紙かみ。
  • かざりや目印めじるしに使つかう、絵えや模様もようのついた小ちいさな紙かみ。

イベント

  • 行事ぎょうじ。できごと。
  • 勝負しょうぶ。試合しあい。

かゆい

  • 皮膚ひふがむずむずして、かきたくなる。

絵の具

  • 絵えに色いろをつけるために使つかうもの。水彩すいさい絵えの具ぐや油あぶら絵えの具ぐなどがある。

結果

  • あることがもとになって起おこったことがらやようす。

アレルギー

  • 生物せいぶつがある特定とくていの物質ぶっしつに対たいして示しめす異常いじょうな反応はんのう。人ひとによって、薬くすりや食たべ物もの、花粉かふんなどに対たいして、皮膚ひふがかゆくなったり、ぶつぶつができたりすること。

金属

  • ふつうは固体こたいで、熱ねつや電気でんきをよく伝つたえるもの。鉄てつ・銅どう・金きん・銀ぎん・アルミニウムなど。

袋

  • 紙かみや布ぬのなどで作つくり、中なかに物ものを入いれて、口くちをしめるようにした入いれ物もの。
  • 「1)」に似にた形かたちのもの。

化粧

  • おしろいや、紅べになどをつけて、顔かおをきれいに見みせること。
  • 外側そとがわをきれいに見みせること。

品

  • 物もの。品物しなもの。
  • 品物しなものの性質せいしつ。品質ひんしつ。
  • 種類しゅるい。

成分

  • 物ものを作つくり上あげている一ひとつ一ひとつのもとになる物質ぶっしつ。

内側

  • 中なかのほう。
