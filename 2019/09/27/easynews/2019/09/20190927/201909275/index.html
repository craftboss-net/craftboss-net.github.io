関西電力かんさいでんりょくの社長しゃちょうなど２０人にんが３億おく円えん以上いじょうもらう

发表于 2019-09-27 更新于 2019-11-30 分类于 JapanEasyNews



関西電力かんさいでんりょくの岩根いわね茂樹しげき社長しゃちょうが２７日にち、社長しゃちょうや役員やくいんなど２０人にんが、去年きょねんまでの７年ねんに３億おく２０００万まん円えんぐらいのお金かねや品物しなものをもらっていたと話はなしました。
お金かねなどを渡わたしたのは、関西電力かんさいでんりょくの原子力げんしりょく発電所はつでんしょがある福井県ふくいけん高浜町たかはまちょうの役場やくばで助役じょやくをしていた人ひとです。助役じょやくをしていた人ひとは今年ことし３月がつに亡なくなりました。この人ひとが、原発げんぱつに関係かんけいする工事こうじをしている会社かいしゃからお金かねをもらって、その一部いちぶを関西電力かんさいでんりょくの役員やくいんなどに渡わたしたことが金沢国税局かなざわこくぜいきょくの調査ちょうさでわかったようです。
岩根いわね社長しゃちょうたちは、金沢国税局かなざわこくぜいきょくに言いわれて、お金かねや品物しなものの一部いちぶや全部ぜんぶを返かえしたと言いっています。岩根いわね社長しゃちょうは「皆みな様さまに大おおきな心配しんぱいをかけて、申もうし訳わけありません。会社かいしゃから処分しょぶんを受うけて、給料きゅうりょうなどの一部いちぶを会社かいしゃに返かえしました」と話はなしました。
経済産業けいざいさんぎょう大臣だいじんは「本当ほんとうのことだったらとても大おおきな問題もんだいです」と言いっています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

役員

  • 会社かいしゃや団体だんたいなどで、特別とくべつの役やくについている人ひと。幹部かんぶ。重役じゅうやく。
  • 特とくに決きめられた仕事しごとを受うけ持もつ人ひと。

原子力

  • 原子核げんしかくがこわれたり（核分裂かくぶんれつ）、とけあったり（核融合かくゆうごう）するときに出でる、ものすごく大おおきな力ちからと高たかい熱ねつ。原子げんしエネルギー。

発電所

  • 電気でんきを起おこす所ところ。

役場

  • 町まちや村むらの仕事しごとをする役所やくしょ。

助役

  • 市長しちょう・町長ちょうちょうや、駅長えきちょうなどの仕事しごとを助たすける役やく。また、その人ひと。

原発

  • 「原子力発電所げんしりょくはつでんしょ」の略りゃく。

工事

  • 建物たてもの・道路どうろ・橋はしなどを造つくったり、直なおしたりすること。また、その仕事しごと。

一部

  • 全体ぜんたいの中なかの、ある部分ぶぶん。
  • 新聞しんぶんや雑誌ざっしなどを数かぞえるときの、一ひとつ。

調査

  • 事実じじつを明あきらかにするために、調しらべること。

申し訳ない

  • 言いい訳わけができない。大変たいへんすまない。

処分

  • いらなくなった物ものなどを始末しまつすること。
  • 規則きそくを破やぶった人ひとに、あるばつをあたえること。

給料

  • やとい主ぬしが、働はたらいた人ひとにはらうお金かね。給与きゅうよ。サラリー。

大臣

  • 内閣ないかくをつくり、国くにの政治せいじで、もっとも責任せきにんのある人ひと。総理大臣そうりだいじんと国務大臣こくむだいじんとがある。
