ｉＰＳ細胞さいぼうから２つ以上いじょうの臓器ぞうきを一度いちどに作つくる

发表于 2019-09-27 更新于 2019-11-30 分类于 JapanEasyNews



ｉＰＳ細胞さいぼうは、体からだのいろいろな部分ぶぶんに変かわることができる細胞さいぼうです。
東京医科歯科大学とうきょういかしかだいがくの武部たけべ貴則たかのり教授きょうじゅのグループは、ｉＰＳ細胞さいぼうから肝臓かんぞうやすい臓ぞうなど２つ以上いじょうの臓器ぞうきを一度いちどに作つくったと発表はっぴょうしました。
グループは最初さいしょに、人ひとのｉＰＳ細胞さいぼうを消化器しょうかきに変かわる前まえまで育そだてました。そして、違ちがう臓器ぞうきになる細胞さいぼうを一ひとつに合あわせて育そだてると、肝臓かんぞうとすい臓ぞうと、両方りょうほうをつなぐ胆管たんかんが一度いちどにできました。大おおきさは約やく１ｃｍで、お母かあさんのお腹なかの中なかにいる１か月げつの赤あかちゃんの臓器ぞうきと同おなじぐらいです。肝臓かんぞうから液体えきたいが流ながれて、臓器ぞうきが働はたらくこともわかりました。
ｉＰＳ細胞さいぼうから２つ以上いじょうの臓器ぞうきを一度いちどに作つくったのは世界せかいで初はじめてです。グループはこれから、血管けっかんなども作つくって、人ひとに移植いしょくして病気びょうきを治なおすために使つかいたいと話はなしています。

人ひとの名前なまえ
国くにや県けん、町まち、場所ばしょなどの名前なまえ
会社かいしゃやグループなどの名前なまえ

臓器

  • 体からだの中なかにある、心臓しんぞう・胃い・腸ちょうなどの器官きかん。

部分

  • 全体ぜんたいを、いくつかに分わけたものの一ひとつ。

細胞

  • 生物せいぶつの体からだを組くみ立たてている、いちばん小ちいさい単位たんい。

教授

  • 大学だいがくの先生せんせい。
  • 人ひとにものを教おしえること。また、その人ひと。

グループ

  • 仲間なかま。集団しゅうだん。

肝臓

  • 胃いの右上みぎうえにある器官きかん。胆汁たんじゅうを出だして消化しょうかを助たすけたり、血液けつえきの中なかの毒物どくぶつをこわしたり、余分よぶんな栄養えいようをたくわえたりする。

膵臓

  • 胃いの後うしろにある内臓ないぞう。食たべ物ものをこなすはたらきをする、すい液えきを作つくり出だす。

発表

  • 多おおくの人ひとに広ひろく知しらせること。

消化器

  • 食たべた物ものをこなして、養分ようぶんを体からだに取とり入いれる器官きかん。口くち・食道しょくどう・胃い・腸ちょうなど。

合わせる

  • 足たして一ひとつにする。いっしょにする。
  • 同おなじにする。そろえる。
  • 同おなじかどうか比くらべる。

つなぐ

  • はなれないように、ほかのものに結むすびつける。
  • はなれているものを、結むすび合あわせてひと続つづきのものにする。
  • 長ながく続つづくようにする。

約

  • ちかう。取とり決きめる。
  • 縮ちぢめる。省はぶく。簡単かんたんにする。
  • おおよそ。ほぼ。だいたい。

液体

  • 水みずや油あぶらのように、決きまった形かたちがなく、流ながれる性質せいしつのあるもの。

流れる

  • 液体えきたいなどが、低ひくいほうへ動うごく。
  • 時ときがたつ。
  • うかんで行いく。
  • 広ひろがって行いく。
  • とりやめになる。

血管

  • 体からだの中なかを血液けつえきが流ながれる管くだ。動脈どうみゃく・静脈じょうみゃく・毛細血管もうさいけっかんがある。

移植

  • 草くさや木きを、ほかの場所ばしょに植うえかえること。
  • 体からだの一部分いちぶぶんを、ほかの部分ぶぶんやほかの人ひとに移うつしかえること。

治す

  • 病気びょうきやけがをよくする。
