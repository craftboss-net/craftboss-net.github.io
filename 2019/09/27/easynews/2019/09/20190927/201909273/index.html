台湾たいわんで人気にんきがある本屋ほんやが日本にっぽんで初はじめて店みせを出だす

发表于 2019-09-27 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうの日本橋にほんばしに２７日にち、台湾たいわんで人気にんきがある「誠品生活せいひんせいかつ」という本屋ほんやがオープンしました。
「コレド室町むろまちテラス」というビルの中なかに広ひろさ２９００m²の売うり場ばがあって、本ほんや文房具ぶんぼうぐ、台湾たいわんで人気にんきがある雑貨ざっかも売うっています。
この本屋ほんやは、店みせを出だす場所ばしょの文化ぶんかや、生活せいかつする人ひとが好すきな物ものに合あわせて売うる物ものを決きめています。店みせでは日本にっぽんや外国がいこくの小説しょうせつなど、１０万まん冊さつ以上いじょうの本ほんや雑誌ざっしを売うっています。
店みせの中なかでは、いろいろな所ところで座すわって本ほんを読よむことができて、好すきな本ほんを探さがしながらゆっくり楽たのしむことができます。
「誠品生活せいひんせいかつ」が日本にっぽんに店みせを出だすのは初はじめてです。店みせの人ひとは「本ほんを読よむ人ひとが日本にっぽんでも世界せかいでも少すくなくなっています。この店みせに来きた人ひとが、人生じんせいに影響えいきょうを受うけるような本ほんを見みつけることができる場所ばしょになってほしいです」と話はなしています。
