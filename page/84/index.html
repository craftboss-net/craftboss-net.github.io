ロシアの島しま　ホッキョクグマが町まちに来きてとても危険きけん

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews


北極海ほっきょくかいにあるロシアの島しまでは去年きょねん１２月がつから、たくさんのホッキョクグマが町まちに入はいって来きています。
ロシアのテレビは１２日にち、クマがアパートの廊下ろうかを歩あるいているところや、人ひとが襲おそわれそうになっているビデオを何なん度ども放送ほうそうしました。町まちには今いままでに５０頭とう以上いじょうのクマが来きて、いつも１０頭とうぐらいが町まちにいるようです。
この町まちには３０００人にんぐらいが住すんでいます。役所やくしょはとても危険きけんなので気きをつけるように言いって、パトロールをしたりフェンスを作つくったりしています。
専門家せんもんかは「地球ちきゅうの気温きおんが上あがって、クマが生活せいかつしている海うみの氷こおりがとけていることが原因げんいんだと思おもいます」と話はなしています。

客きゃくの荷物にもつを飛行機ひこうきに入いれる仕事しごとにパワーアシストスーツを使つかう

发表于 2019-02-13 更新于 2019-11-30 分类于 JapanEasyNews



日本航空にほんこうくうのグループの会社かいしゃは２月がつから、空港くうこうで客きゃくの荷物にもつを飛行機ひこうきに入いれる仕事しごとなどに、パワーアシストスーツという機械きかいを使つかい始はじめました。このスーツを体からだにつけると、少すくない力ちからで重おもい荷物にもつを持もつことができます。
羽田空港はねだくうこうでは女性じょせいもこのスーツをつけて、スーツケースを飛行機ひこうきのコンテナに入いれていました。
スーツを作つくった会社かいしゃによると、スーツの重おもさは４．５ｋｇぐらいです。モーターを使つかって、荷物にもつを持もつときに腰こしにかかる力ちからを少すくなくします。スーツをつけていないときと比くらべると、同おなじ時間じかんで荷物にもつを２０％以上いじょう多おおく運はこぶことができます。
日本にっぽんの航空こうくう会社がいしゃでは、全日空ぜんにっくうも同おなじようなスーツを使つかっています。

ロボット掃除機そうじき　電気でんきストーブの火事かじに気きをつけて

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



ロボット掃除機そうじきは、自動じどうで部屋へやの中なかを動うごいて掃除そうじをします。東京とうきょうのマンションで去年きょねん１２月がつ、電気でんきストーブがソファーにぶつかって焼やける火事かじがありました。
火事かじになったとき部屋へやの中なかには誰だれもいませんでしたが、ロボット掃除機そうじきが動うごいていました。東京消防庁とうきょうしょうぼうちょうが実験じっけんを行おこなうと、ロボット掃除機そうじきが電気でんきストーブを押おしたり、ストーブのコードを巻まき込こんだりすることがわかりました。
東京消防庁とうきょうしょうぼうちょうは、ロボット掃除機そうじきが電気でんきストーブを５０ｃｍぐらい押おして、ストーブがソファーにぶつかって火ひが出でたと考かんがえています。ロボット掃除機そうじきが原因げんいんだと考かんがえられている火事かじは、今月こんげつも１件けんありました。
東京消防庁とうきょうしょうぼうちょうは、ロボット掃除機そうじきを使つかうときには、電気でんきストーブのコードをコンセントから抜ぬいてほしいと言いっています。

切きっても元もとに戻もどる新あたらしいゴムができる

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



理化学研究所りかがくけんきゅうしょなどのグループが、切きっても元もとに戻もどる新あたらしいゴムを作つくりました。グループによると、ゴムの切きった所ところを軽かるく合あわせると、何なん分ぷんかあとに元もとのようにつながります。傷きずもほとんど見みえなくなります。このゴムで作つくった袋ふくろは、穴あながあいてもすぐに閉とじます。
グループは、分子ぶんしと分子ぶんしが引ひき合あう 物理学ぶつりがくのルールを利用りようしてこのゴムを作つくりました。

ゴムのほかにも、切きっても元もとに戻もどるガラスやコンクリート、金属きんぞくなどの研究けんきゅうが進すすんでいます。
