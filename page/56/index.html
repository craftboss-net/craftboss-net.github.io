今年ことし「はしか」になった人ひとが去年きょねん１年ねんの１．７倍ばいになる

发表于 2019-05-16 更新于 2019-11-30 分类于 JapanEasyNews



「はしか」は高たかい熱ねつやせきが出でたり、体からだに赤あかい発疹はっしんが出でたりする病気びょうきです。ウイルスでうつりやすくて、とてもひどくなることがあります。妊娠にんしんしている女性じょせいがはしかになると、おなかの中なかの赤あかちゃんが亡なくなることもあります。
国立感染症研究所こくりつかんせんしょうけんきゅうしょによると、５月がつ５日いつかまでの１週間しゅうかんに、はしかになったことがわかった人ひとは１１人にんいました。今年ことし、はしかになった人ひとは４６７人にんで、去年きょねん１年ねんの１．７倍ばいに増ふえています。大阪府おおさかふでは１３６人にんがはしかになりました。東京都とうきょうとや三重県みえけん、愛知県あいちけんなどでも多おおくなっています。
厚生労働省こうせいろうどうしょうは、４月がつの終おわりから５月がつの初はじめの休やすみの間あいだに、はしかのウイルスがうつった人ひとは、これから熱ねつやせきが出でるかもしれないと言いっています。そして、高たかい熱ねつなどが出でた場合ばあいは、まず電話でんわで病院びょういんに相談そうだんしてほしいと言いっています。

広島県ひろしまけん　雨あめで崩くずれた山やまにドローンを使つかって草くさの種たねをまく

发表于 2019-05-16 更新于 2019-11-30 分类于 JapanEasyNews



広島県ひろしまけんでは、去年きょねん７月がつにとてもたくさんの雨あめが降ふって、８５００の場所ばしょで山やまなどが崩くずれました。今いまも崩くずれたままの場所ばしょがたくさんあります。
崩くずれた場所ばしょを前まえのように木きや草くさがある緑みどりの山やまにするため、１４日じゅうよっか、呉市くれしの山やまでドローンを使つかった実験じっけんがありました。実験じっけんでは、ドローンを飛とばして、広ひろさ１００㎡ぐらいの土つちが見みえている場所ばしょに、空そらからよもぎなど３種類しゅるいの草くさの種たねをまきました。
実験じっけんをした会社かいしゃは、ドローンから種たねをまいて山やまを緑みどりにしようとするのは珍めずらしいと言いっています。会社かいしゃは、これから毎月まいつき草くさの様子ようすをチェックして、大おおきくなっていることがわかったら、種たねをまく場所ばしょをもっと広ひろくすることにしています。

沖縄県おきなわけんの石垣島いしがきじま　もう稲いねを刈かる季節きせつになる

发表于 2019-05-16 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんでは秋あきが、米こめができて稲いねを刈かる季節きせつです。沖縄県おきなわけんの八重山地方やえやまちほうは暖あたたかいため、１年ねんに２回かい、米こめを作つくる農家のうかがたくさんあります。毎年まいとし５月がつに１回かい目めの米こめができて、稲いねを刈かります。
八重山地方やえやまちほうの石垣島いしがきじまの田たんぼでは、今年ことし１月がつに植うえた稲いねに米こめができて、農家のうかの人ひとが１５日にちから稲いねを刈かり始はじめました。
今年ことし２月がつから３月がつ、石垣島いしがきじまでは暑あつくなったり寒さむくなったりしました。このため、稲いねが十分じゅうぶんに育そだつかどうか心配しんぱいでしたが、高たかさ１ｍぐらいまで大おおきくなりました。農家のうかの人ひとは機械きかいを使つかって、たくさんの稲いねを刈かっていました。
できた米こめは、問題もんだいがないかどうか調しらべたあと、５月がつ３１日にちから石垣島いしがきじまにあるＪＡの店みせで売うることになっています。

人工衛星じんこうえいせいは宇宙うちゅうで、２つの模型もけいを外そとへ出だします。模型もけいは高たかさ１０ｃｍぐらいで、人工衛星じんこうえいせいの上うえに立たつような形かたちになります。来年らいねん７月がつにオリンピックが始はじまると、選手せんしゅを応援おうえんするメッセージを模型もけいの下したのディスプレーに出だします。人工衛星じんこうえいせいのカメラが撮とったメッセージや模型もけいは、インターネットのＳＮＳなどで見みせる予定よていです。
組織委員会そしきいいんかいの室伏むろふし広治こうじさんは「宇宙うちゅうからガンダムに応援おうえんしてもらったら、選手せんしゅも頑張がんばることができると思おもいます。楽たのしみにしています」と話はなしています。
