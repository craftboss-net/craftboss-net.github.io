切きっても元もとに戻もどる新あたらしいゴムができる

发表于 2019-02-12 更新于 2019-11-30 分类于 JapanEasyNews



理化学研究所りかがくけんきゅうしょなどのグループが、切きっても元もとに戻もどる新あたらしいゴムを作つくりました。グループによると、ゴムの切きった所ところを軽かるく合あわせると、何なん分ぷんかあとに元もとのようにつながります。傷きずもほとんど見みえなくなります。このゴムで作つくった袋ふくろは、穴あながあいてもすぐに閉とじます。
グループは、分子ぶんしと分子ぶんしが引ひき合あう 物理学ぶつりがくのルールを利用りようしてこのゴムを作つくりました。
切きっても元もとに戻もどるゴムは前まえからありましたが、つけるためには光ひかりや熱ねつなどのエネルギーが必要ひつようでした。新あたらしくできたゴムは、光ひかりや熱ねつなどが必要ひつようありません。そして、水みずの中なかや宇宙うちゅうなどいろいろな場所ばしょで使つかうことができます。
ゴムのほかにも、切きっても元もとに戻もどるガラスやコンクリート、金属きんぞくなどの研究けんきゅうが進すすんでいます。

魚さかなは鏡かがみに映うつった自分じぶんがわかる　大阪市立大学おおさかしりつだいがくなどが調しらべる

发表于 2019-02-08 更新于 2019-11-30 分类于 JapanEasyNews



サルやイルカは、鏡かがみに映うつった自分じぶんがわかります。魚さかなも同おなじようにわかるかどうか、大阪市立大学おおさかしりつだいがく大学院だいがくいんの幸田こうだ正典まさのり先生せんせいたちのグループが調しらべました。
グループは、鏡かがみがついている水槽すいそうで育そだてているホンソメワケベラという魚さかなを使つかいました。この魚さかなのあごの近ちかくに茶色ちゃいろの汚よごれをつけた場合ばあいとつけなかった場合ばあいについて調しらべました。
その結果けっか、汚よごれをつけた４匹ひきのうち３匹びきは、鏡かがみを見みたあと水槽すいそうにある砂すなにあごの近ちかくを強つよく当あてて、また鏡かがみの前まえに戻もどりました。汚よごれをつけなかった魚さかなは、このようなことはしませんでした。
グループによると、汚よごれをつけた魚さかなは鏡かがみを見みて虫むしか何なにかがついていると思おもって、取とろうとしたと考かんがえられます。
グループは、魚さかなが鏡かがみに映うつった自分じぶんがわかると確認かくにんできたのは初はじめてだと言いっています。幸田こうだ先生せんせいは「今いままでみんなが考かんがえていたより、魚さかなは頭あたまがいい可能性かのうせいが高たかいと思おもいます」と話はなしています。

イタリアの大学生だいがくせいが日本にっぽんの学校がっこうの給食きゅうしょくを見みに来くる

イタリアの大学だいがくで世界せかいの食事しょくじの文化ぶんかを勉強べんきょうしている学生がくせい１４人にんが、日本にっぽんの給食きゅうしょくを見みるために山形県やまがたけん鶴岡市つるおかしへ来きました。
鶴岡市つるおかしでは１３０年ねん前まえに日本にっぽんで初はじめて給食きゅうしょくが始はじまったと言いわれています。学校がっこうでみんなに同おなじ食事しょくじを出だす日本にっぽんの給食きゅうしょくは、世界せかいでも珍めずらしいと言いわれています。

そのあと、学生がくせいたちは近ちかくの小学校しょうがっこうへ行いって、子こどもたちと一緒いっしょに給食きゅうしょくを食たべました。学生がくせいは子こどもに「好すきな食たべ物ものは？」などと聞きいていました。この日ひの給食きゅうしょくには２月がつ３日みっかの節分せつぶんに食たべる豆まめもあって、学生がくせいたちは昔むかしから続つづく行事ぎょうじの食たべ物ものが給食きゅうしょくに出でることに驚おどろいていました。
