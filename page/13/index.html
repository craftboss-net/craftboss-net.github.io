日本にっぽんとアメリカ　貿易ぼうえきの新あたらしいルールの話はなし合あいがまとまる

发表于 2019-09-26 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんは太平洋たいへいようの周まわりにある１０の国くにと「ＴＰＰ」という貿易ぼうえきのルールを決きめていますが、アメリカはこの中なかに入はいりませんでした。このため、日本にっぽんとアメリカは貿易ぼうえきの新あたらしいルールを作つくるために話はなし合あいをしてきました。
２６日にち、日本にっぽんの安倍あべ総理大臣そうりだいじんとアメリカのトランプ大統領だいとうりょうがニューヨークで会あって、話はなし合あいがまとまりました。
新あたらしいルールでは、ＴＰＰで決きまっている範囲はんいで、日本にっぽんがアメリカからの農産物のうさんぶつの輸入ゆにゅうを認みとめることになりました。安倍あべ総理大臣そうりだいじんとトランプ大統領だいとうりょうは、新あたらしいルールのとおりに貿易ぼうえきが続つづいている間あいだは、アメリカが日本にっぽんから輸入ゆにゅうする車くるまの関税かんぜいを今いまより高たかくしないことを確認かくにんしました。
安倍あべ総理大臣そうりだいじんは「日本にっぽんの人ひとにもアメリカの人ひとにも利益りえきがあるルールになりました。２つの国くにの経済けいざいは必かならず発展はってんすると思おもいます」と話はなしました。

タトゥーシールで顔かおや腕うでが痛いたくなることがある　気きをつけて

发表于 2019-09-25 更新于 2019-11-30 分类于 JapanEasyNews



「タトゥーシール」は絵えや字じがかいてあるシールで、顔かおや腕うでなどに貼はって楽たのしみます。スポーツのイベントなどのときに使つかう人ひとが増ふえています。
消費者庁しょうひしゃちょうによると、タトゥーシールを貼はった所ところがかゆくなったり、痛いたくなったりした人ひとがいます。
このため、国民生活こくみんせいかつセンターは、いろいろなタトゥーシールを調しらべました。顔かおや体からだに絵えをかくときに使つかう絵えの具ぐも調しらべました。その結果けっか、アレルギーの危険きけんがある金属きんぞくを使つかっているのに、袋ふくろに書かいていない物ものがありました。化粧けしょう品ひんに使つかってはいけない成分せいぶんを使つかっている物ものもありました。
消費者庁しょうひしゃちょうは、タトゥーシールなどを使つかうときは、最初さいしょに腕うでの内側うちがわなどでテストをして、大丈夫だいじょうぶかどうか調しらべるように言いっています。もし痛いたくなったら、病院びょういんへ行いくように言いっています。

外国人がいこくじんに災害さいがいの情報じょうほうを伝つたえるアプリ　外国語がいこくごを増ふやす

发表于 2019-09-25 更新于 2019-11-30 分类于 JapanEasyNews



観光庁かんこうちょうが作つくった「Ｓａｆｅｔｙ　ｔｉｐｓ」は、台風たいふうや地震じしんなど災害さいがいの情報じょうほうを外国人がいこくじんに伝つたえるアプリです。
このアプリをスマートフォンなどに入いれると、強つよい地震じしんが来くるときや雨あめがとてもたくさん降ふるときなどに連絡れんらくがきます。避難ひなんするときに気きをつけることや病院びょういんの場所ばしょなども知しることができます。
今いままでは英語えいご、中国語ちゅうごくご、韓国語かんこくご、日本語にほんごだけでしたが、今月こんげつからスペイン語ご、ポルトガル語ご、ベトナム語ご、タイ語ご、インドネシア語ご、タガログ語ご、ネパール語ごを増ふやしました。そして来年らいねん３月がつまでにクメール語ごとビルマ語ご、モンゴル語ごも入いれる予定よていです。市しや町まちが避難ひなんするように言いった場合ばあいも、外国語がいこくごで伝つたえるようにします。
観光庁かんこうちょうは「外国人がいこくじんが安心あんしんして旅行りょこうを楽たのしむことができるように、アプリやＳＮＳなどいろいろな方法ほうほうを使つかって、災害さいがいの情報じょうほうを伝つたえます」と話はなしています。

ベッドが軽かるいため、選手せんしゅはベッドを置おく場所ばしょを自分じぶんで変かえることができます。よく寝ねることができるように、自分じぶんにちょうどいい硬かたさを選えらぶことができるマットレスを使つかいます。重おもさ２００ｋｇの人ひとも寝ねることができます。

組織委員会そしきいいんかいは「オリンピックなどと関係かんけいがある人ひとたちに段だんボールのベッドを見みてもらいましたが、心配しんぱいする人ひとはいませんでした。選手せんしゅたちが気持きもちよく寝ねることができるようにしたいです」と話はなしています。

２４時間じかん開あいているコンビニや店みせなどは、１０月がつ１日ついたちの０時じになるとすぐ１０％に変かわります。レストランのガスト、ジョナサン、ロイヤルホストは、１０月がつ１日ついたちの０時じを過すぎてから店みせを閉しめる場合ばあい、店みせを閉しめる時間じかんまで８％のままです。

電気でんき代だいやガス代だいは、９月がつに使つかった料金りょうきんが入はいっている場合ばあいは８％です。
