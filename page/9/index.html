ラグビーのワールドカップ　日本にっぽんは今いままでの試合しあいで全部ぜんぶ勝かつ

发表于 2019-10-07 更新于 2019-11-30 分类于 JapanEasyNews



先月せんげつ２０日はつかから日本にっぽんで始はじまったラグビーのワールドカップで、日本にっぽんは今いままで試合しあいを３つして、全部ぜんぶ勝かちました。
最初さいしょは、ロシアとの試合しあいで３０－１０で勝かちました。次つぎは、アイルランドとの試合しあいで１９－１２で勝かちました。とても強つよいアイルランドに日本にっぽんが勝かったため、外国がいこくのテレビや新聞しんぶんは「今いままでのワールドカップでいちばん驚おどろいた」などと伝つたえました。
今月こんげつ５日いつかのサモアとの試合しあいでは、日本にっぽんはトライを４つして、３８－１９で勝かちました。優勝ゆうしょうを決きめるための「決勝けっしょうトーナメント」に日本にっぽんが初はじめて出でる可能性かのうせいが高たかくなりました。
日本にっぽんの次つぎの試合しあいは１３日にちで、とても強つよいスコットランドと戦たたかいます。日本にっぽんの選手せんしゅは「試合しあいまでの間あいだ、一生懸命いっしょうけんめい練習れんしゅうして準備じゅんびしたいと思おもいます」と話はなしています。

地震じしんで壊こわれた熊本城くまもとじょうの一部いちぶを見みることができるようになる

发表于 2019-10-07 更新于 2019-11-30 分类于 JapanEasyNews



熊本市くまもとしにある熊本城くまもとじょうは３年ねん半はん前まえの地震じしんで壊こわれましたが、城しろでいちばん高たかい「大天守だいてんしゅ」という建物たてものの外側そとがわはほとんど直なおりました。このため熊本市くまもとしは５日いつかから、城しろの一部いちぶに入はいることができるようにしました。午後ごご１時じ、楽たのしみに待まっていた人ひとたちがどんどん城しろの中なかに入はいっていきました。
入はいることができるのは、二にの丸まる広場ひろばから大天守だいてんしゅの前まえにある広場ひろばまでの４５０ｍの道みちです。立派りっぱな形かたちに戻もどった大天守だいてんしゅや、壊こわれたままの石垣いしがきを近ちかくで見みることができます。
７０歳さいの男性だんせいは「小ちいさいときから城しろと一緒いっしょだったので、本当ほんとうにうれしいです。城しろが前まえのような形かたちに戻もどると、私わたしたちも元気げんきになります」と話はなしていました。
熊本城くまもとじょうの大天守だいてんしゅは、来年らいねん３月がつの終おわりまで日曜日にちようびと祝日しゅくじつ、今月こんげつ１４日じゅうよっかまでは、平日へいじつの午後ごごと土曜日どようびにも見みることができます。

日本航空にほんこうくうの席せきを予約よやくするときの「幼児ようじマーク」に賛成さんせいと反対はんたい

发表于 2019-10-07 更新于 2019-11-30 分类于 JapanEasyNews



日本航空にほんこうくうは２０１３年ねんから、飛行機ひこうきの席せきを予約よやくするためのウェブサイトで、３歳さいにならない子こどもがいる席せきに「幼児ようじマーク」を付つけています。
日本航空にほんこうくうに乗のったアメリカの男性だんせいがこのマークについて、「大おおきな声こえで泣なく赤あかちゃんがどこにいるか教おしえてくれてありがとう」とインターネットに書かきました。そして、男性だんせいが「ほかの会社かいしゃも同おなじようにしなければならない」と書かいたため、インターネットで議論ぎろんになりました。
小ちいさな子こどもがいる親おやや静しずかに休やすみたいという人ひとたちはマークに賛成さんせいしています。しかし「子こどもの近ちかくにはいたくないという考かんがえ方かたは差別さべつになる」という意見いけんもあります。イギリスのＢＢＣなどもこの議論ぎろんを伝つたえました。
日本航空にほんこうくうは「小ちいさな子こどもを連つれたお客きゃくさまからの意見いけんで、マークを付つけ始はじめました。どのお客きゃくさまも気持きもちよく利用りようできるようにしていきます」と話はなしています。

電子でんしたばこと病気びょうきの研究けんきゅう「肺はいは毒どくがある物質ぶっしつを吸すったときと同おなじ」

发表于 2019-10-07 更新于 2019-11-30 分类于 JapanEasyNews



液体えきたいを電気でんきで温あたためて煙けむりのようなものを吸すう「電子でんしたばこ」について調しらべた結果けっかが、アメリカの有名ゆうめいな医学いがくの雑誌ざっしに出でました。
アメリカの病院びょういんなどが、電子でんしたばこと関係かんけいがある肺はいなどの病気びょうきになった１７人にんを調しらべました。その結果けっか、全部ぜんぶの人ひとの肺はいなどが、毒どくがある物質ぶっしつを吸すったときと同おなじように傷きずついていました。どんな物質ぶっしつが原因げんいんで傷きずついたかわからないと言いっています。
健康けんこうについて研究けんきゅうしているアメリカ政府せいふのＣＤＣによると、電子でんしたばこが原因げんいんだと考かんがえられる肺はいなどの病気びょうきになった人ひとは１０８０人にんいて、１８人にんが亡なくなっています。
アメリカのいくつかの州しゅうは、においが出でる電子でんしたばこを売うることを禁止きんししています。インドも電子でんしたばこを売うることや輸入ゆにゅうを禁止きんししていて、電子でんしたばこについての規則きそくが厳きびしくなっています。

荷物にもつに入はいったナイフ　羽田空港はねだくうこうでも検査けんさ場じょうを通とおった

发表于 2019-10-04 更新于 2019-11-30 分类于 JapanEasyNews



先月せんげつ２６日にち、飛行機ひこうきに乗のる人ひとの荷物にもつを調しらべる大阪空港おおさかくうこうの検査けんさ場じょうで、全日空ぜんにっくうのスタッフが客きゃくの荷物にもつにナイフのような物ものが入はいっているのを見みつけました。しかし、スタッフはこの客きゃくをそのまま通とおしてしまいました。検査けんさ場じょうを閉しめて、飛行機ひこうきに乗のる人ひとたちの荷物にもつなどをもう一度いちど調しらべましたが、ナイフは見みつかりませんでした。検査けんさ場じょうを閉しめたので、たくさんの人ひとが飛行機ひこうきに乗のることができなくなりました。
この客きゃくは、大阪空港おおさかくうこうから飛行機ひこうきに乗のって羽田空港はねだくうこうに着ついたあと、外国がいこくに行いく飛行機ひこうきに乗のり換かえました。羽田空港はねだくうこうでも荷物にもつの検査けんさを受うけましたが、羽田空港はねだくうこうでは荷物にもつに入はいったナイフを見みつけることができませんでした。
客きゃくはナイフを荷物にもつに入いれたまま、２回かい飛行機ひこうきに乗のっていたことになります。
国土交通省こくどこうつうしょうは、飛行機ひこうきの会社かいしゃや空港くうこうに「飛行機ひこうきに持もって入はいってはいけない物ものは必かならず見みつけて、見みつけたらどうするかスタッフの教育きょういくもしっかりしてください」と言いいました。

埼玉県さいたまけん日高市ひだかしにある「巾着田きんちゃくだ曼珠沙華まんじゅしゃげ公園こうえん」では、秋あきになると、広ひろさ３ｈａぐらいの所ところに５００万まん本ぼんの赤あかい彼岸花ひがんばなが咲さきます。

ロシアのプーチン大統領だいとうりょうは２日ふつか、「グレタさんは世界せかいが複雑ふくざつだということを知しらないのです。地球温暖化ちきゅうおんだんかを止とめるために太陽たいようの光ひかりで電気でんきを作つくっていたら、発展途上国はってんとじょうこくはスウェーデンのように発展はってんしません。グレタさんは優やさしいですが、情報じょうほうが少すくない若わかい人ひとです」と言いいました。
