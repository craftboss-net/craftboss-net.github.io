イギリスの会社かいしゃ「女性じょせいの客室きゃくしつ乗務員じょうむいんは化粧けしょうをしなくてもいい」

发表于 2019-03-07 更新于 2019-11-30 分类于 JapanEasyNews



イギリスの航空こうくう会社がいしゃのヴァージン?アトランティック航空こうくうは、女性じょせいの客室きゃくしつ乗務員じょうむいんだけに決きめていた化粧けしょうや制服せいふくについての規則きそくを今月こんげつやめました。
飛行機ひこうきで客きゃくにサービスなどをする女性じょせいの客室きゃくしつ乗務員じょうむいんは、仕事しごとのとき化粧けしょうをしなければならないと会社かいしゃは決きめていました。しかし、これからは化粧けしょうをしなくてもいいと決きめました。そして、制服せいふくはスカートだけでしたが、ズボンも選えらぶことができるようにしました。
会社かいしゃの人ひとは「このように規則きそくを変かえたことは、とても大おおきな変化へんかです。女性じょせいがもっと気持きもちよく働はたらくことができます。そして、自分じぶんに合あう制服せいふくや化粧けしょうを自分じぶんで選えらぶことができます」と話はなしています。
世界せかいの航空こうくう会社がいしゃでは、男性だんせいと女性じょせいで違ちがう規則きそくを変かえる会社かいしゃが増ふえています。

大分県おおいたけん　タトゥーがある人ひとも入はいれる温泉おんせんを英語えいごで紹介しょうかいする

发表于 2019-03-07 更新于 2019-11-30 分类于 JapanEasyNews



タトゥーは肌はだに傷きずをつけてかいた字じや絵えで、入いれ墨ずみとも言いいます。世界せかいではタトゥーが昔むかしからの文化ぶんかになっている所ところもあります。しかし、日本にっぽんではタトゥーがある人ひとは怖こわいと思おもう人ひとがいるため、タトゥーがあると入はいることができない温泉おんせんもあります。
温泉おんせんが有名ゆうめいな大分県おおいたけんでは、今年ことしの秋あき、ラグビーのワールドカップで５つの試合しあいを行おこないます。外国がいこくから来くる人ひとたちに温泉おんせんを楽たのしんでもらうため、大分県おおいたけんはタトゥーがある人ひとも入はいることができる温泉おんせんを英語えいごのウェブサイトで紹介しょうかいすることにしました。ウェブサイトは３月がつ中旬ちゅうじゅんから始はじめる予定よていです。
大分県おおいたけんは「文化ぶんかが違ちがう外国がいこくの人ひとも日本にっぽんの温泉おんせんを楽たのしむことができるようにしたいです」と話はなしています。

大戸屋おおとや　アルバイトなどの研修けんしゅうのためにほとんどの店みせを休やすむ

发表于 2019-03-06 更新于 2019-11-30 分类于 JapanEasyNews



「大戸屋おおとや」は日本中にほんじゅうに３５０の店みせがある食堂しょくどうです。先月せんげつ、大戸屋おおとやのアルバイトの人ひとが店みせでズボンを脱ぬいで遊あそんでいるビデオをインターネットに出だして、問題もんだいになりました。
このため大戸屋おおとやの会社かいしゃは、店みせで働はたらく人ひとの研修けんしゅうをするために、今月こんげつ１２日にちにほとんどの店みせを休やすむことにしました。
研修けんしゅうでは、スマートフォンなどを仕事しごと中ちゅうに持もつことは禁止きんしだと伝つたえます。そして、仕事しごと中ちゅうにビデオを撮とったりインターネットに出だしたりしないことを会社かいしゃに約束やくそくしてもらいます。
会社かいしゃは「お客様きゃくさまに安心あんしんして安全あんぜんな物ものを食たべてもらうために、働はたらく人ひとの教育きょういくと研修けんしゅうが必要ひつようだと考かんがえました」と言いっています。

子こどもがインターネットを使つかう時間じかんは１日にちに３時間じかんぐらい

发表于 2019-03-05 更新于 2019-11-30 分类于 JapanEasyNews



内閣府ないかくふは毎年まいとし、１０歳さいから１７歳さいの子こどもにインターネットについてのアンケートを行おこなっています。去年きょねんは３０７９人にんが答こたえました。
インターネットを使つかっている子こどもに、月曜日げつようびから金曜日きんようびに１日にち何なん時間じかんぐらい使つかっているか聞きくと、平均へいきんで２時間じかん４９分ふんでした。おととしより９分ふん長ながくなりました。
小学生しょうがくせいはおととしより２１分ぷん長ながい１時間じかん５８分ふん、中学生ちゅうがくせいは１５分ふん長ながい２時間じかん４４分ぷん、高校生こうこうせいは３分ぷん長ながい３時間じかん３７分ふんでした。
インターネットを使つかって何なにをしているか聞きくと、７９％が「ビデオを見みる」と答こたえました。「ゲーム」は７６％、「メールやＳＮＳ」は６６％でした。
内閣府ないかくふは「子こどもがインターネットを使つかう時間じかんはだんだん長ながくなっています。どんな影響えいきょうがあるか調しらべて、安全あんぜんに使つかうことができるようにしたいです」と話はなしています。
