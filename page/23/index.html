東京とうきょうパラリンピックのメダル　触さわってわかるデザイン

发表于 2019-08-26 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうパラリンピックが始はじまるまであと１年ねんになった２５日にち、メダルの発表はっぴょうがありました。
パラリンピックのメダルは、表おもても裏うらも扇子せんすのデザインになっています。「みんなの心こころを一ひとつにして、世界せかいに新あたらしい風かぜを送おくる」という意味いみです。
表おもてには「ＴＯＫＹＯ２０２０」と書かいてあって、目めの見みえない人ひとがわかるように点字てんじもあります。横よこには丸まるいくぼみがあって、くぼみの数かずでどのメダルかわかります。裏うらには岩いわ、花はな、木き、葉は、水みずの５つの自然しぜんもデザインされていて、触さわってわかるようになっています。
メダルをデザインした人ひとは「日本にっぽんらしいデザインにしたいと思おもって、扇子せんすにしました」と話はなしています。
使つかわなくなった携帯電話けいたいでんわなどから集あつめた金属きんぞくを使つかって、来年らいねん５月がつまでに全部ぜんぶで約やく２５００個このメダルを作つくります。

发表于 2019-08-26 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックとパラリンピックの組織委員会そしきいいんかいは、開会かいかい式しきの会場かいじょうに選手せんしゅを運はこんでみるテストを行おこないました。泊とまっている選手村せんしゅむらから開会かいかい式しきをする新国立競技場しんこくりつきょうぎじょうまで、バスを使つかって選手せんしゅを運はこぶ予定よていです。

韓国かんこくが日本にっぽんとの国くにと国くにの安全あんぜんについての約束やくそくをやめる

发表于 2019-08-23 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんと韓国かんこくは、国くにの安全あんぜんのための情報じょうほうを交換こうかんする「ＧＳＯＭＩＡ」という約束やくそくをしています。韓国かんこくの政府せいふは２２日にち、日本にっぽんとのＧＳＯＭＩＡをやめると発表はっぴょうしました。
やめることを決きめた理由りゆうは、日本にっぽんの政府せいふが、日本にっぽんから輸出ゆしゅつが簡単かんたんにできるようにしているグループから韓国かんこくを外はずしたことだと言いっています。韓国かんこくの政府せいふは「日本にっぽんは外はずした理由りゆうをはっきり説明せつめいしていません。日本にっぽんとＧＳＯＭＩＡを続つづけることは韓国かんこくのためになりません」と言いっています。
安倍あべ総理大臣そうりだいじんは「韓国かんこくは、歴史れきしの問題もんだいについて国くにと国くにの約束やくそくを守まもっていません。信頼しんらいできる関係かんけいをつくるためには、約束やくそくを守まもってほしいです」と話はなしています。
韓国かんこくがＧＳＯＭＩＡをやめることを決きめて、日本にっぽんと韓国かんこくの問題もんだいは、歴史れきしや経済けいざいだけではなくて国くにの安全あんぜんの問題もんだいにも広ひろがっています。

東京とうきょうの虎ノ門とらのもん?麻布台地区あざぶだいちくに高たかさ３３０ｍ、６４階かい建だてのビルができることになりました。「森もりビル」という会社かいしゃが２０２３年ねん３月がつまでに建たてる予定よていです。大阪市おおさかしにある高たかさ３００ｍの「あべのハルカス」より高たかくなって、日本にっぽんでいちばん高たかいビルになります。
建たてるのにかかるお金かねは５８００億おく円えんぐらいです。ビルの中なかにはいろいろな会社かいしゃの事務所じむしょが入はいって、５４階かいより上うえは人ひとが住すむ場所ばしょになります。
東京とうきょうでは高たかいビルを建たてる計画けいかくが続つづいています。２０２７年ねんごろにはＪＲの東京駅とうきょうえきの北きた側がわに高たかさ３９０ｍのビルができる予定よていです。
世界せかいの大おおきな都市としでは高たかいビルが増ふえています。いろいろな国くにの会社かいしゃが集あつまって、ビジネスの中心ちゅうしんになっています。森もりビルの社長しゃちょうは「このビルが東京とうきょうのシンボルになってほしいです」と話はなしています。

コンビニのローソンは、横浜市よこはましの店みせで午前ごぜん０時じから午前ごぜん５時じまで売うり場ばやレジに店員てんいんがいない実験じっけんをしています。
店みせの入いり口ぐちで機械きかいがＱＲコードか顔かおの写真しゃしんで客きゃくをチェックできると、ドアが開あきます。客きゃくは自分じぶんでレジやスマートフォンを使つかってお金かねを払はらいます。

「１人ひとりで悩なやまないで」子こども食堂しょくどうで相談そうだんの電話でんわ番号ばんごうを知しらせる

发表于 2019-08-22 更新于 2019-11-30 分类于 JapanEasyNews



ＮＰＯの「チャイルドライン支援しえんセンター」は、子こどもが電話でんわやインターネットで悩なやみを相談そうだんできる所ところです。
毎年まいとし、夏休なつやすみが終おわるころは子こどもの自殺じさつが増ふえると言いわれています。センターは、悩なやみを相談そうだんできる所ところがあることを子こどもたちに知しらせたいと考かんがえました。そして、センターの電話でんわ番号ばんごうなどを書かいたカードを初はじめて「子こども食堂しょくどう」に送おくることにしました。
「子こども食堂しょくどう」は、生活せいかつのお金かねが足たりない家庭かていの子こどもや、親おやが働はたらいていて１人ひとりで食事しょくじをしなければならない子こどもが、無料むりょうか安やすいお金かねで食事しょくじができる所ところです。
センターは７０ぐらいの「子こども食堂しょくどう」にカードを５万まん枚まいぐらい送おくって、子こどもたちに渡わたしてもらいます。センターの人ひとは「心配しんぱいなことがあるときや苦くるしいときは、１人ひとりで悩なやまないでください。誰だれかに話はなすことが大切たいせつです」と言いっています。
