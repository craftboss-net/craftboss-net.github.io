２０２６年ねんの冬ふゆのオリンピック　イタリアの２つの町まちで開ひらく

发表于 2019-06-25 更新于 2019-11-30 分类于 JapanEasyNews



２０２６年ねんに冬ふゆのオリンピックを開ひらく町まちを決きめるＩＯＣの会議かいぎがスイスでありました。イタリアのミラノとコルティナダンペッツォ、スウェーデンのストックホルムとオーレが開ひらきたいと言いっていました。
会議かいぎでは、どちらで開ひらくかを決きめる投票とうひょうを行おこないました。
その結果けっか、イタリアの２つの町まちで開ひらくことが決きまりました。コルティナダンペッツォは１９５６年ねんにも冬ふゆのオリンピックを開ひらいています。イタリアでオリンピックを開ひらくのは、夏なつと冬ふゆを合あわせて４回かい目めです。
日本にっぽんの札幌市さっぽろしも２０２６年ねんにオリンピックを開ひらきたいと言いっていましたが、途中とちゅうで２０３０年ねんに変かえました。

子こどもへの体罰たいばつを禁止きんしする法律ほうりつができる

发表于 2019-06-24 更新于 2019-11-30 分类于 JapanEasyNews



子こどもへの虐待ぎゃくたいをなくすための法律ほうりつができて、親おやが子こどもを教育きょういくするために、たたいたり蹴けったりする体罰たいばつを禁止きんしすることになりました。
最近さいきん、親おやに虐待ぎゃくたいされた子こどもを児童相談所じどうそうだんじょが守まもることができなくて、子こどもが亡なくなる事件じけんが続つづいています。このため、法律ほうりつでは児童相談所じどうそうだんじょの力ちからを強つよくしました。医者いしゃや保健師ほけんしが児童相談所じどうそうだんじょで意見いけんを言いったり相談そうだんを聞きいたりするようにしました。児童相談所じどうそうだんじょは、相談そうだんする弁護士べんごしを決きめておかなければなりません。
法律ほうりつでは虐待ぎゃくたいをした親おやに専門家せんもんかが子こどもの育そだて方かたを教おしえたり、子こどもを守まもる仕事しごとをする専門せんもんの職員しょくいんを増ふやしたりすることも決きまりました。
日本にっぽんに前まえからある法律ほうりつでは、親おやには子こどもを叱しかる権利けんりがあると決きめています。このため、虐待ぎゃくたいしたときに叱しかっていただけだと言いう親おやがいます。新あたらしい法律ほうりつでは、親おやの叱しかる権利けんりをなくしたり変かえたりすることも話はなし合あうと決きめています。

働はたらく場所ばしょでの暴力ぼうりょくやハラスメントを禁止きんしする国際こくさい条約じょうやくが決きまる

发表于 2019-06-24 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんのＩＬＯは２１日にち、スイスで開ひらいた会議かいぎで、働はたらく場所ばしょでの暴力ぼうりょくやハラスメントを禁止きんしする国際こくさい条約じょうやくを決きめました。
条約じょうやくによると、暴力ぼうりょくやハラスメントは体からだや心こころなどを傷きずつけるため、法律ほうりつで禁止きんしします。仕事しごとの実習じっしゅうに来きている学生がくせいやボランティア、仕事しごとを探さがしている人ひとにもしてはいけません。
条約じょうやくを決きめるとき、日本にっぽんは、政府せいふと働はたらく人ひとたちの団体だんたいは賛成さんせいに投票とうひょうしました。しかし、会社かいしゃを経営けいえいしている人ひとたちの団体だんたいは投票とうひょうしませんでした。
ＩＬＯの１８７の国くには、この条約じょうやくに参加さんかするかどうかこれから考かんがえます。参加さんかした国くには、暴力ぼうりょくやハラスメントを禁止きんしする法律ほうりつをつくります。

１８日にち、新潟県にいがたけんや山形県やまがたけんで大おおきな地震じしんがありました。
山形県やまがたけんでは、さくらんぼが日本にっぽんでいちばんたくさん取とれます。今いまはさくらんぼがいちばんおいしい季節きせつで、農園のうえんでは観光かんこう客きゃくがさくらんぼを取とって楽たのしむことができます。
鶴岡市つるおかしにある農園のうえんでは、「佐藤錦さとうにしき」という種類しゅるいのさくらんぼがおいしそうに赤あかくなっていて、いつもの年としはたくさんのお客きゃくさんが来きます。
１８日にちの地震じしんでこの農園のうえんも大おおきく揺ゆれましたが、被害ひがいはありませんでした。しかし、農園のうえんに来くる予定よていだったお客きゃくさんからのキャンセルが続つづいています。農園のうえんに１年ねんに入はいるお金かねの半分はんぶんは、この季節きせつにお客きゃくさんが払はらう入園にゅうえん料りょうです。農園のうえんではお客きゃくさんが減へってとても困こまっています。
農園のうえんの人ひとは「地震じしんの被害ひがいはありません。ぜひ遊あそびに来きて、さくらんぼをたくさん食たべてください」と話はなしていました。

消費生活しょうひせいかつセンターによると、買かい物ものなどのトラブルの相談そうだんの中なかで、インターネットのＳＮＳを使つかった人ひとからの相談そうだんが増ふえています。去年きょねんは今いままでで最もっとも多おおい１万まん６８０５件けんの相談そうだんがありました。
この中なかでは２０歳さいから２９歳さいの人ひとの相談そうだんが３０％で最もっとも多おおくなっています。年齢ねんれいが高たかい人ひとでも、５０歳さいから５９歳さいの人ひとは１５％ぐらい、６０歳さいから６９歳さいの人ひとは７％ぐらいいました。５０歳さい以上いじょうの人ひとの相談そうだんは４年ねん前まえの１．８倍ばいになりました。

政府せいふは、トラブルの相談そうだんが増ふえた原因げんいんの１つは、年齢ねんれいが高たかい人ひとのＳＮＳの利用りようが多おおくなったことだと考かんがえています。
