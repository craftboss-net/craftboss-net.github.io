どこへ行いったかわからなくなった認知症にんちしょうの人ひと　去年きょねん最もっとも多おおい

发表于 2019-06-24 更新于 2019-11-30 分类于 JapanEasyNews



認知症にんちしょうという病気びょうきになると、自分じぶんの名前なまえを忘わすれたり、どこにいるかわからなくなったりします。認知症にんちしょうの人ひとが、出でかけたあとどこへ行いったかわからなくなることが増ふえています。
警察庁けいさつちょうによると、「認知症にんちしょうの人ひとがどこへ行いったかわからない」と家族かぞくなどが警察けいさつに知しらせた数かずは、去年きょねんは１万まん６９２７でした。６年ねん続つづけて１万まん以上いじょうになって、７年ねん前まえに調しらべ始はじめてから最もっとも多おおくなりました。
ほとんどの人ひとは去年きょねん見みつかりましたが、１９７人にんはまだ見みつかっていません。
警察けいさつでは、早はやく見みつけることができるように、認知症にんちしょうの人ひとがどのようなことをするかなど専門せんもんの勉強べんきょうをしています。そして、さがしている人ひとの写真しゃしんや情報じょうほうをまとめて、すぐ調しらべることができるようにしています。

働はたらく場所ばしょでの暴力ぼうりょくやハラスメントを禁止きんしする国際こくさい条約じょうやくが決きまる

发表于 2019-06-24 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんのＩＬＯは２１日にち、スイスで開ひらいた会議かいぎで、働はたらく場所ばしょでの暴力ぼうりょくやハラスメントを禁止きんしする国際こくさい条約じょうやくを決きめました。
条約じょうやくによると、暴力ぼうりょくやハラスメントは体からだや心こころなどを傷きずつけるため、法律ほうりつで禁止きんしします。仕事しごとの実習じっしゅうに来きている学生がくせいやボランティア、仕事しごとを探さがしている人ひとにもしてはいけません。
条約じょうやくを決きめるとき、日本にっぽんは、政府せいふと働はたらく人ひとたちの団体だんたいは賛成さんせいに投票とうひょうしました。しかし、会社かいしゃを経営けいえいしている人ひとたちの団体だんたいは投票とうひょうしませんでした。
ＩＬＯの１８７の国くには、この条約じょうやくに参加さんかするかどうかこれから考かんがえます。参加さんかした国くには、暴力ぼうりょくやハラスメントを禁止きんしする法律ほうりつをつくります。

最後さいごに大おおきな戦たたかいがあった糸満市いとまんしの平和へいわ祈念きねん公園こうえんでは２３日にち、亡なくなった人ひとたちのために祈いのる式しきがありました。式しきには亡なくなった人ひとの家族かぞくなど５１００人にんが出席しゅっせきして、１２時じになるとみんなで目めを閉とじて静しずかに祈いのりました。

日本にっぽんの学校がっこうの先生せんせいは世界せかいでいちばん忙いそがしい

发表于 2019-06-21 更新于 2019-11-30 分类于 JapanEasyNews



ＯＥＣＤは５年ねんに１度ど、世界せかいの学校がっこうの先生せんせいの仕事しごとについて調しらべています。去年きょねんは日本にっぽんの小学校しょうがっこうと中学校ちゅうがっこうの先生せんせい７０００人にんなど、４８の国くにと地域ちいきを調しらべました。
中学校ちゅうがっこうの先生せんせいが１週間しゅうかんに働はたらく時間じかんは、世界せかいの平均へいきんで３８．３時間じかんでした。働はたらく時間じかんがいちばん長ながいのは日本にっぽんの先生せんせいで５６時間じかんでした。次つぎはカザフスタンが４８．８時間じかん、カナダが４７時間じかん、イギリスが４６．９時間じかん、アメリカが４６．２時間じかんでした。どんな仕事しごとをしているか調しらべると、日本にっぽんの先生せんせいは授業じゅぎょうの時間じかんが１８時間じかんで、ほかの国くになどと変かわりませんでした。しかし、授業じゅぎょうではない部活動ぶかつどうなどの仕事しごとは７．５時間じかんで、平均へいきんの４倍ばいでした。書類しょるいを作つくる仕事しごとなども平均へいきんの２倍ばいでした。日本にっぽんの先生せんせいが忙いそがしい理由りゆうは、授業じゅぎょう以外いがいの仕事しごとの時間じかんが長ながいためです。教育きょういくの専門家せんもんかは「日本にっぽんは、ほかの国くにのやり方かたも聞ききながら、先生せんせいの働はたらき方かたを変かえていく必要ひつようがあると思おもいます」と話はなしています。

发表于 2019-06-21 更新于 2019-11-30 分类于 JapanEasyNews



１８日にち、新潟県にいがたけんや山形県やまがたけんで大おおきな地震じしんがありました。
山形県やまがたけんでは、さくらんぼが日本にっぽんでいちばんたくさん取とれます。今いまはさくらんぼがいちばんおいしい季節きせつで、農園のうえんでは観光かんこう客きゃくがさくらんぼを取とって楽たのしむことができます。
鶴岡市つるおかしにある農園のうえんでは、「佐藤錦さとうにしき」という種類しゅるいのさくらんぼがおいしそうに赤あかくなっていて、いつもの年としはたくさんのお客きゃくさんが来きます。
１８日にちの地震じしんでこの農園のうえんも大おおきく揺ゆれましたが、被害ひがいはありませんでした。しかし、農園のうえんに来くる予定よていだったお客きゃくさんからのキャンセルが続つづいています。農園のうえんに１年ねんに入はいるお金かねの半分はんぶんは、この季節きせつにお客きゃくさんが払はらう入園にゅうえん料りょうです。農園のうえんではお客きゃくさんが減へってとても困こまっています。
農園のうえんの人ひとは「地震じしんの被害ひがいはありません。ぜひ遊あそびに来きて、さくらんぼをたくさん食たべてください」と話はなしていました。

消費生活しょうひせいかつセンターによると、買かい物ものなどのトラブルの相談そうだんの中なかで、インターネットのＳＮＳを使つかった人ひとからの相談そうだんが増ふえています。去年きょねんは今いままでで最もっとも多おおい１万まん６８０５件けんの相談そうだんがありました。

高知県こうちけんは毎月まいつき、生うまれた人ひとや亡なくなった人ひと、引ひっ越こして来きた人ひとや出でて行いった人ひとを計算けいさんして、県けんの人口じんこうがどのくらいか発表はっぴょうしています。
高知県こうちけんによると、６月がつ１日ついたちの人口じんこうは６９万まん９５２２人にんになりました。高知県こうちけんの人口じんこうが７０万まん人にん以下いかになったのは、１９４５年ねんに戦争せんそうが終おわってから初はじめてです。
