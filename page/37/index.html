日本にっぽんの人口じんこうが１０年ねん続つづけて減へった

发表于 2019-07-11 更新于 2019-11-30 分类于 JapanEasyNews



総務省そうむしょうによると、今年ことし１月がつ１日ついたちに日本にっぽんに住すんでいる日本人にっぽんじんは１億おく２４７０万まん人にんでした。１０年ねん続つづけて人口じんこうが減へりました。
東京とうきょうは７万まん人にん増ふえて、２３年ねん続つづけて増ふえています。しかし、名古屋なごやと関西かんさいの都市としでは減へっています。東京とうきょうに今いままでよりもっと多おおくの人ひとが集あつまっています。
専門家せんもんかは「２５歳さいから３９歳さいの女性じょせいが２００８年ねんからの１０年ねんで２１％も減へりました。子こどもを産うむ年齢ねんれいの女性じょせいが減へっていることが、人口じんこうが減へったことのいちばん大おおきな原因げんいんです。これからも人口じんこうが減へると考かんがえて、社会しゃかいや経済けいざいをどうするのか考かんがえることが必要ひつようです」と話はなしています。
日本にっぽんに住すむ外国人がいこくじんは２６６万まん人にんでした。外国人がいこくじんの人口じんこうは増ふえていて、日本にっぽんに住すんでいる人ひとの中なかで外国人がいこくじんの割合わりあいが初はじめて２％以上いじょうになりました。

人間にんげんがアフリカから出でたのは考かんがえていたより前まえだった

发表于 2019-07-11 更新于 2019-11-30 分类于 JapanEasyNews



ドイツの大学だいがくなどのグループは、ギリシャの南みなみにある洞窟どうくつから見みつかった２つの頭あたまの骨ほねを調しらべました。放射線ほうしゃせんを使つかって調しらべた結果けっか、１つは２１万まん年ねん前まえの人間にんげんの骨ほねだとわかりました。
今いままでに見みつかった人間にんげんの骨ほねは、アフリカで見みつかったものが３０万まん年ねん前まえの骨ほねでした。中東ちゅうとうは１８万まん年ねん前まえ、東ひがしアジアは９万まん年ねん前まえ、ヨーロッパは５万まん年ねん前まえの骨ほねでした。ギリシャで見みつかった骨ほねは、人間にんげんが最初さいしょに住すんでいたアフリカ以外いがいで見みつかった骨ほねの中なかでいちばん古ふるいことになります。
グループは「人間にんげんがアフリカから外そとへ出でたのは、今いままでに考かんがえていたより何なん万まん年ねんも前まえだったことになります。人間にんげんがどのように生いきてきたかを調しらべるのに役やくに立たつと思おもいます」と話はなしています。

「はやぶさ２」がまたリュウグウという星ほしに下おりた

发表于 2019-07-11 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのＪＡＸＡが打うち上あげた「はやぶさ２」は、去年きょねん６月がつに地球ちきゅうと火星かせいの間あいだにあるリュウグウという小ちいさい星ほしに着ついて、この星ほしを調しらべています。
ＪＡＸＡは、「はやぶさ２」が１１日にち午前ごぜん１０時じ過すぎにリュウグウに下おりたと発表はっぴょうしました。「はやぶさ２」は高たかさ２万まんｍの所ところからリュウグウに下おり始はじめました。下おりる場所ばしょにはボールのような目印めじるしを落おとしておきました。そして、この目印めじるしをカメラで見みながら下おりました。
「はやぶさ２」は今年ことし２月がつにもリュウグウに下おりて、地面じめんの石いしを取とりました。今年ことし４月がつには地面じめんに金属きんぞくの塊かたまりを落おとして、穴あなを作つくりました。１１日にちはこの穴あなの近ちかくに下おりました。「はやぶさ２」は、穴あなを作つくったときに地面じめんの中なかから出でた石いしを取とって、地球ちきゅうへ持もって帰かえる予定よていです。
地面じめんの中なかの石いしは、宇宙うちゅうからの影響えいきょうがないため、地球ちきゅうなどの星ほしがどうやってできたか知しることができるかもしれません。

发表于 2019-07-10 更新于 2019-11-30 分类于 JapanEasyNews



ダイキン工業こうぎょうなど９つの会社かいしゃが、東京とうきょうのビルの中なかに新あたらしい技術ぎじゅつを使つかって働はたらきやすい場所ばしょをつくりました。１０００㎡ぐらいの広ひろさで、会議かいぎ室しつや１人ひとりで仕事しごとをするための机つくえなどがあります。
会議かいぎ室しつでは、みんながどのような話はなし方かたをしているか機械きかいが調しらべます。話はなし方かたが激はげしくなると、リラックスできるような香かおりを出だします。香かおりは３種類しゅるいあって、会議かいぎの様子ようすで出でてくる香かおりが変かわります。
１人ひとりで仕事しごとをする机つくえには小ちいさいカメラがあって、仕事しごとをしている人ひとの目めの動うごき方かたを調しらべます。その人ひとが眠ねむくなっていることがわかると、眠ねむらないようにエアコンが温度おんどを下さげます。
ダイキン工業こうぎょうの人ひとは「９つの会社かいしゃの技術ぎじゅつを集あつめてつくりました。たくさんの人ひとに使つかってほしいです」と話はなしています。
この場所ばしょは１６日にちから使つかうことができます。

家族かぞくは、国くにがハンセン病びょうになった人ひとを療養りょうよう所じょに入いれ続つづけたため、家族かぞくも差別さべつされたと裁判さいばんで訴うったえました。裁判所さいばんしょは先月せんげつ２８日にち、家族かぞくが差別さべつされたのは国くにに責任せきにんがあると言いって、国くにが家族かぞくに賠償ばいしょうのためのお金かねを払はらうように命令めいれいしました。
安倍あべ総理大臣そうりだいじんは「とてもつらい経験けいけんをした家族かぞくの人ひとたちをこれ以上いじょう苦くるしめることはできません」と言いって、裁判さいばんで争あらそわないと言いいました。

日本語にほんごの教育きょういくが必要ひつような子こどものために「支援員しえんいん」を増ふやす

发表于 2019-07-09 更新于 2019-11-30 分类于 JapanEasyNews



外国がいこくで育そだったりしたため、日本語にほんごの教育きょういくが必要ひつような公立こうりつの高校こうこうの生徒せいとは、２０１７年度ねんどに約やく４０００人にんいました。しかし、この中なかの９．６％が途中とちゅうで高校こうこうをやめました。
９．６％は、公立こうりつの高校こうこうの全部ぜんぶの生徒せいとの中なかで、途中とちゅうで高校こうこうをやめた人ひとの割合わりあいの７倍ばいです。文部科学省もんぶかがくしょうは、授業じゅぎょうがよくわからなくてやめた人ひとも多おおいと考かんがえています。
文部科学省もんぶかがくしょうは、日本語にほんごの教育きょういくが必要ひつような高校生こうこうせいのために、「支援員しえんいん」を増ふやすことにしました。支援員しえんいんは、授業じゅぎょうの内容ないようをわかりやすいことばで翻訳ほんやくして説明せつめいします。
日本にっぽんでは、働はたらく外国人がいこくじんを多おおくしているため、同おなじように日本語にほんごの教育きょういくが必要ひつような小学生しょうがくせいや中学生ちゅうがくせいが増ふえそうです。このため文部科学省もんぶかがくしょうは、支援員しえんいんを増ふやしたり、たくさんのことばで翻訳ほんやくできるシステムを使つかったりしたいと考かんがえています。
