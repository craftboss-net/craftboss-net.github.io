スキーのジャンプのワールドカップ　小林こばやし陵侑りょうゆう選手せんしゅが総合そうごう優勝ゆうしょう

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



スキーのジャンプの小林こばやし陵侑りょうゆう選手せんしゅは、去年きょねんから今年ことしのワールドカップの試合しあいで１１回かい優勝ゆうしょうしています。全部ぜんぶの成績せいせきを足たした「個人こじん総合そうごう」では１番ばんになっています。
１０日とおかにノルウェーで試合しあいがあって、小林こばやし選手せんしゅは５番ばんでした。まだ試合しあいは５つありますが、個人こじん総合そうごうの成績せいせきでほかの選手せんしゅが小林こばやし選手せんしゅより上うえになる可能性かのうせいはなくなりました。このため、小林こばやし選手せんしゅが総合そうごう優勝ゆうしょうすることが決きまりました。
１９７９年ねんから始はじまったスキーのジャンプ男子だんしのワールドカップで、日本にっぽんの選手せんしゅが総合そうごう優勝ゆうしょうするのは初はじめてです。
小林こばやし選手せんしゅは岩手県いわてけんの出身しゅっしんで２２歳さいです。「すごくうれしいです。こんなに早はやく決きまるとは思おもっていませんでした。周まわりの人ひとにすごく感謝かんしゃしています」と話はなしていました。

「がんばろう！石巻いしのまき」　震災しんさいで亡なくなった人ひとのために祈いのる

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



宮城県みやぎけん石巻いしのまき市しで、８年ねん前まえの東日本大震災ひがしにほんだいしんさいで亡なくなった人ひとのために祈いのる会かいがありました。
１１日にち、「がんばろう！石巻いしのまき」と書かいた大おおきな看板かんばんの前まえに市民しみんやボランティアなどが３００人にんぐらい集あつまりました。地震じしんが起おこった午後ごご２時じ４６分ぷんになると、亡なくなった人ひとのために祈いのって、青あおや白しろの風船ふうせんを空そらに飛とばしました。
そのあと看板かんばんの周まわりに、石巻いしのまき市しで亡なくなった人ひとと同おなじ数かずの３６００個この灯籠とうろうを並ならべました。午後ごご５時じ半はんごろ、灯籠とうろうのろうそくに火ひをつけて、もう一度いちど祈いのりました。
看板かんばんを作つくった石巻いしのまき市しの男性だんせいは「震災しんさいから８年ねんになって、前まえのように戻もどったと思おもう人ひとや、まだ時間じかんが止とまっている人ひとなどいろいろな人ひとがいます。これからも活動かつどうを続つづけようと思おもっています」と話はなしていました。

赤あかちゃん用ようの液体えきたいミルク　日本にっぽんでも売うり始はじめる

赤あかちゃん用ようの「液体えきたいミルク」は、外国がいこくではたくさん売うっていますが、日本にっぽんでは売うっていませんでした。液体えきたいミルクはそのまま飲のませることができて便利べんりなため、日本にっぽんでも売うってほしいと多おおくの人ひとが言いっていました。このため国くには液体えきたいミルクの基準きじゅんを作つくって、日本にっぽんでも売うることができるようにしました。
「江崎えざきグリコ」は１１日にち、紙かみパックに入はいった液体えきたいミルクを店みせで売うり始はじめました。このミルクは冷蔵庫れいぞうこに入いれなくても６か月げつ置おいておくことができます。「明治めいじ」ももうすぐ新あたらしい液体えきたいミルクを売うり始はじめる予定よていです。
消費者庁しょうひしゃちょうは、紙かみパックや缶かんなどに入はいった液体えきたいミルクは開あけたらすぐに、きれいな瓶びんなどに入いれて飲のませるように言いっています。そして、残のこったミルクは菌きんが増ふえやすいため、使つかわないように言いっています。

发表于 2019-03-11 更新于 2019-11-30 分类于 JapanEasyNews



９日ここのか、新潟市にいがたしの港みなとから佐渡市さどしの港みなとに向むかっていた船ふねが、海うみの中なかにいた何なにかにぶつかりました。船ふねには１２５人にんが乗のっていました。事故じこで８０人にんがけがをして、この中なかの１３人にんがひどいけがです。船ふねを調しらべると、後うしろの部分ぶぶんに１５ｃｍぐらいの割われ目めがありました。船ふねの後うしろにある水中翼すいちゅうよくは曲まがっていました。
船長せんちょうは「船ふねが時速じそく７５ｋｍぐらいで進すすんでいるとき、すぐ前まえに白しろい大おおきな物ものが見みえたので、スピードを遅おそくしたがぶつかった」と話はなしています。

船ふねの会社かいしゃは１１日にち、「深ふかくおわびします」と謝あやまりました。そして安全あんぜんのために、船ふねのスピードを遅おそくして進すすむ所ところを今いままでより多おおくすることを考かんがえたいと話はなしました。

取とった土つちなどは、まず家いえの庭にわや駐車場ちゅうしゃじょうなど人ひとが生活せいかつしている場所ばしょの近ちかくに置おいています。そのあと畑はたけなどに集あつめてから、福島県ふくしまけん双葉ふたば町まちと大熊町おおくままちに作つくっている特別とくべつな建物たてものに運はこびます。
