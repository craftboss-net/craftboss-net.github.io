東京とうきょうオリンピックまで５００日にち　競技きょうぎのピクトグラムを発表はっぴょう

发表于 2019-03-13 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックの開会かいかい式しきまであと５００日にちになりました。東京とうきょうで１２日にち、競技きょうぎを絵えで表あらわした「ピクトグラム」を発表はっぴょうするイベントがありました。ピクトグラムは３３の競技きょうぎで５０種類しゅるいです。
会場かいじょうでは、陸上りくじょうの飯塚いいづか翔太しょうた選手せんしゅや空手からての清水しみず希容きよう選手せんしゅ、小学生しょうがくせいたちが新あたらしいピクトグラムを紹介しょうかいしました。
競技きょうぎのピクトグラムは１９６４年ねんの東京とうきょうオリンピックで初はじめて作つくられてから、全部ぜんぶのオリンピックで新あたらしいピクトグラムが作つくられています。
新あたらしいピクトグラムは、競技きょうぎの会場かいじょうやチケット、ウェブサイトなどで使つかわれます。
東京とうきょうパラリンピックのピクトグラムは来月らいげつ１３日にちに発表はっぴょうする予定よていです。

埼玉県さいたまけんで始はじまる「夜間中学やかんちゅうがく」　外国人がいこくじんの生徒せいとが説明せつめいを聞きく

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



外国人がいこくじんが大勢おおぜい住すんでいる埼玉県さいたまけん川口市かわぐちしは、今年ことし４月がつから「夜間中学やかんちゅうがく」を始はじめます。夜間中学やかんちゅうがくは、中学校ちゅうがっこうで十分じゅうぶんに勉強べんきょうできなかった人ひとや外国人がいこくじんのために、中学校ちゅうがっこうで夜よる授業じゅぎょうをします。
９日ここのか、川口市かわぐちしは入学にゅうがくする生徒せいとを集あつめて、学校がっこうの生活せいかつやルールなどを説明せつめいしました。入学にゅうがくする８０人にんのうち４４人にんが外国人がいこくじんのため、説明せつめいがわかるように資料しりょうには漢字かんじの読よみ方かたや英語えいごなども書かきました。授業じゅぎょうでは、外国人がいこくじんのために日本語にほんごや日本にっぽんの習慣しゅうかんや文化ぶんかも教おしえる予定よていです。
出席しゅっせきした外国人がいこくじんは「子こどもの学校がっこうの先生せんせいと話はなすときなど、日本語にほんごは生活せいかつの中なかで必要ひつようです」とか「日本にっぽんのことを勉強べんきょうして花屋はなやを開ひらきたいです」などと話はなしていました。

地震じしんのとき役やくに立たつ公衆電話こうしゅうでんわ　子こどもに使つかい方かたを教おしえる

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



ＮＴＴ東日本ひがしにほんは、町まちの中なかや駅えきにある「公衆電話こうしゅうでんわ」の使つかい方かたを子こどもに教おしえるウェブサイトを作つくりました。ウェブサイトでは、ビデオやクイズで公衆電話こうしゅうでんわの使つかい方かたを紹介しょうかいしています。公衆電話こうしゅうでんわがどこにあるか調しらべることができます。
公衆電話こうしゅうでんわは、大おおきな地震じしんがあったときや電気でんきが止とまったときでも、ほかの電話でんわよりつながりやすくなっています。８年ねん前まえの東日本大震災ひがしにほんだいしんさいのとき、携帯電話けいたいでんわはつながりにくくなりましたが、公衆電話こうしゅうでんわで多おおくの人ひとが連絡れんらくすることができました。
しかしＮＴＴ東日本ひがしにほんが調しらべると、「公衆電話こうしゅうでんわを知しらない」とか「使つかったことがない」という子こどもがたくさんいることがわかりました。
ＮＴＴ東日本ひがしにほんの人ひとは「子こどもたちに公衆電話こうしゅうでんわを使つかってみてほしいです。そして、地震じしんなどのとき使つかうことができるようにしてほしいです」と話はなしていました。

スキーのジャンプのワールドカップ　小林こばやし陵侑りょうゆう選手せんしゅが総合そうごう優勝ゆうしょう

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



スキーのジャンプの小林こばやし陵侑りょうゆう選手せんしゅは、去年きょねんから今年ことしのワールドカップの試合しあいで１１回かい優勝ゆうしょうしています。全部ぜんぶの成績せいせきを足たした「個人こじん総合そうごう」では１番ばんになっています。
１０日とおかにノルウェーで試合しあいがあって、小林こばやし選手せんしゅは５番ばんでした。まだ試合しあいは５つありますが、個人こじん総合そうごうの成績せいせきでほかの選手せんしゅが小林こばやし選手せんしゅより上うえになる可能性かのうせいはなくなりました。このため、小林こばやし選手せんしゅが総合そうごう優勝ゆうしょうすることが決きまりました。
１９７９年ねんから始はじまったスキーのジャンプ男子だんしのワールドカップで、日本にっぽんの選手せんしゅが総合そうごう優勝ゆうしょうするのは初はじめてです。
小林こばやし選手せんしゅは岩手県いわてけんの出身しゅっしんで２２歳さいです。「すごくうれしいです。こんなに早はやく決きまるとは思おもっていませんでした。周まわりの人ひとにすごく感謝かんしゃしています」と話はなしていました。

宮城県みやぎけん石巻いしのまき市しで、８年ねん前まえの東日本大震災ひがしにほんだいしんさいで亡なくなった人ひとのために祈いのる会かいがありました。
１１日にち、「がんばろう！石巻いしのまき」と書かいた大おおきな看板かんばんの前まえに市民しみんやボランティアなどが３００人にんぐらい集あつまりました。地震じしんが起おこった午後ごご２時じ４６分ぷんになると、亡なくなった人ひとのために祈いのって、青あおや白しろの風船ふうせんを空そらに飛とばしました。
そのあと看板かんばんの周まわりに、石巻いしのまき市しで亡なくなった人ひとと同おなじ数かずの３６００個この灯籠とうろうを並ならべました。午後ごご５時じ半はんごろ、灯籠とうろうのろうそくに火ひをつけて、もう一度いちど祈いのりました。

東日本大震災ひがしにほんだいしんさいから８年ねん　今いまも５万まん人にん以上いじょうが家いえに戻もどれない

发表于 2019-03-11 更新于 2019-11-30 分类于 JapanEasyNews



８年ねん前まえの３月がつ１１日にち、東日本大震災ひがしにほんだいしんさいがありました。地震じしんで大おおきく揺ゆれて、東北地方とうほくちほうや関東地方かんとうちほうには高たかさ１０ｍ以上いじょうの津波つなみが来きました。原子力げんしりょく発電所はつでんしょの事故じこもありました。この震災しんさいで１万まん５８９７人にんが亡なくなって、２５３３人にんが行方不明ゆくえふめいになっています。
家いえに戻もどることができなくて別べつの場所ばしょで生活せいかつしている人ひとは、まだ５万まん人にん以上いじょういます。震災しんさいのあと、慣なれない生活せいかつで体からだの具合ぐあいが悪わるくなったりして３７０１人にんが亡なくなりました。１人ひとりで生活せいかつしていて、亡なくなってしばらくしてから見みつかるお年寄としよりも増ふえています。
岩手県いわてけんと宮城県みやぎけんと福島県ふくしまけんでは、半分はんぶん以上いじょうの市しや町まちで震災しんさいの前まえより１０％以上いじょう人口じんこうが減へっています。

新潟県にいがたけんの佐渡市さどしの海うみで船ふねの事故じこ　鯨くじらにぶつかった可能性かのうせい

发表于 2019-03-11 更新于 2019-11-30 分类于 JapanEasyNews



９日ここのか、新潟市にいがたしの港みなとから佐渡市さどしの港みなとに向むかっていた船ふねが、海うみの中なかにいた何なにかにぶつかりました。船ふねには１２５人にんが乗のっていました。事故じこで８０人にんがけがをして、この中なかの１３人にんがひどいけがです。船ふねを調しらべると、後うしろの部分ぶぶんに１５ｃｍぐらいの割われ目めがありました。船ふねの後うしろにある水中翼すいちゅうよくは曲まがっていました。
船長せんちょうは「船ふねが時速じそく７５ｋｍぐらいで進すすんでいるとき、すぐ前まえに白しろい大おおきな物ものが見みえたので、スピードを遅おそくしたがぶつかった」と話はなしています。
専門家せんもんかは「船ふねが通とおる日本海にほんかいでは、１年ねん中じゅう鯨くじらが泳およいでいて、２０ｍぐらいの大おおきな鯨くじらもいます。息いきをするために上あがってきた鯨くじらに船ふねがぶつかった可能性かのうせいがあります」と話はなしています。
船ふねの会社かいしゃは１１日にち、「深ふかくおわびします」と謝あやまりました。そして安全あんぜんのために、船ふねのスピードを遅おそくして進すすむ所ところを今いままでより多おおくすることを考かんがえたいと話はなしました。

福島県ふくしまけん　原発げんぱつの事故じこで汚よごれた土つちが今いまも家いえの近ちかくに置おいたまま

发表于 2019-03-08 更新于 2019-11-30 分类于 JapanEasyNews



８年ねん前まえ、東京電力とうきょうでんりょく福島第一原発ふくしまだいいちげんぱつで事故じこがありました。事故じこのあと、福島県ふくしまけんでは、放射線ほうしゃせんを出だす物質ぶっしつで汚よごれた土つちや草くさ、木きなどを取とる作業さぎょうをしています。
取とった土つちなどは、まず家いえの庭にわや駐車場ちゅうしゃじょうなど人ひとが生活せいかつしている場所ばしょの近ちかくに置おいています。そのあと畑はたけなどに集あつめてから、福島県ふくしまけん双葉ふたば町まちと大熊町おおくままちに作つくっている特別とくべつな建物たてものに運はこびます。
４年ねん前まえから建物たてものに運はこび始はじめましたが、今いままでに運はこんだ土つちなどは予定よていの１７％だけです。
環境省かんきょうしょうなどによると、今いま、福島県ふくしまけんで汚よごれた土つちなどが置おいたままの場所ばしょは、家いえの庭にわなどでは１０万まん以上いじょう、畑はたけなどでは９００以上いじょうあります。環境省かんきょうしょうは、２０２２年ねん３月がつまでにほとんど全部ぜんぶの土つちなどを運はこびたいと言いっています。
住すんでいる人ひとたちは「遅おそすぎる」とか「できるだけ早はやく、家いえの近ちかくから汚よごれた土つちなどをなくしてほしい」などと言いっています。
