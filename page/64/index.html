成田空港なりたくうこう　飛行機ひこうきをとめる場所ばしょで働はたらく車くるまの事故じこが増ふえる

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



成田空港なりたくうこうでは、飛行機ひこうきをとめる場所ばしょで働はたらく人ひとが運転うんてんする車くるまの事故じこが増ふえています。成田空港なりたくうこうの会社かいしゃによると、１年ねんに１１９件けんの事故じこがあって、前まえの年としの１．５倍ばいぐらいに増ふえました。
多おおいのは車くるまと車くるまがぶつかる事故じこでした。会社かいしゃが調しらべると、事故じこを起おこした人ひとの半分はんぶん以上いじょうは、空港くうこうで車くるまを運転うんてんする仕事しごとを始はじめてから３年ねんになっていませんでした。
空港くうこうの会社かいしゃは、働はたらく人ひとが足たりないため、新あたらしく働はたらく人ひとが十分じゅうぶんな教育きょういくを受うけていないことがあると考かんがえています。
成田空港なりたくうこうの会社かいしゃは「日本にっぽんへ観光かんこうに来くる外国人がいこくじんが増ふえて、空港くうこうを利用りようする飛行機ひこうきはもっと増ふえると思おもいます。事故じこが起おこらないようにしっかり教育きょういくします」と話はなしています。

赤あかちゃん３０人にん以上いじょうがベッドで亡なくなる事故じこ

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのフィッシャープライスという会社かいしゃは、赤あかちゃんのベッドなどを売うっています。この会社かいしゃの「ロックン?プレイ?スリーパー」というベッドで寝ねていた赤あかちゃんが３０人にん以上いじょう亡なくなっていたことがわかりました。
会社かいしゃは、このベッドを使つかうときは赤あかちゃんにベルトをするように言いっていました。しかし、亡なくなった赤あかちゃんはベルトをしていなかったため、体からだの向むきが変かわって呼吸こきゅうができなくなりました。
このベッドは日本にっぽんでもインターネットなどで売うっていたため、日本にっぽんの消費者庁しょうひしゃちょうは「使つかうのをやめてください」と言いっています。そして、「ほかの会社かいしゃのベッドでも、安全あんぜんのためのベルトがあったら、必かならずベルトをしてください」と言いっています。

東京とうきょうオリンピックの開会かいかい式しきの時間じかんなどが決きまる

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックの開会かいかい式しきの時間じかんや試合しあいをする日ひなどが決きまりました。
開会かいかい式しきは来年らいねん７月がつ２４日にじゅうよっか、閉会へいかい式しきは８月がつ９日ここのかに新国立競技場しんこくりつきょうぎじょうで行おこないます。どちらも午後ごご８時じから始はじまります。
最初さいしょの試合しあいはソフトボールで、開会かいかい式しきの前まえの７月がつ２２日にちに福島県ふくしまけんで行おこないます。この試合しあいには日本にっぽんのチームが出でる予定よていです。２５日にちからは、柔道じゅうどうなどいろいろな競技きょうぎの試合しあいが始はじまります。
８月がつ２日ふつかの日曜日にちようびの夜よるには、１００ｍの男子だんしの決勝けっしょうがあります。
世界せかいのランキングで日本にっぽんの選手せんしゅが１番ばんになっているバドミントンの男子だんしは、８月がつ３日みっかにシングルスの決勝けっしょうがあります。日本にっぽんの選手せんしゅが強つよい卓球たっきゅうの女子じょしは、８月がつ６日むいかに団体だんたいの決勝けっしょうがあります。
マラソンは８月がつ２日ふつかに女子じょし、８月がつ９日ここのかに男子だんしの試合しあいがあります。暑あつさが心配しんぱいなため、最初さいしょの計画けいかくより早はやい午前ごぜん６時じにスタートします。

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



１０月がつから消費税しょうひぜいが８％から１０％に上あがります。食たべ物ものや酒さけ以外いがいの飲のみ物ものの消費税しょうひぜいは、食たべたり飲のんだりする場所ばしょで変かわります。食たべる場所ばしょがあるコンビニの場合ばあい、店みせで食たべたら消費税しょうひぜいは１０％で、店みせから持もって帰かえったら８％です。
コンビニの会社かいしゃの団体だんたいは、客きゃくが食たべ物ものを買かったときの消費税しょうひぜいの決きめ方かたを考かんがえました。コンビニでは、店みせの中なかで食たべる場合ばあい、客きゃくが買かうときに店みせの人ひとに言いうことにして、店みせの人ひとからは質問しつもんしないことにします。店みせのレジの近ちかくに「店みせで食たべる場合ばあい、お金かねを払はらうときに言いってください」と書かいた紙かみを貼はって、客きゃくに知しらせることにしました。
コンビニと同おなじように食たべる場所ばしょがあるスーパーでも、やり方かたを考かんがえることになりそうです。

お年寄としよりの家いえへ行いくヘルパーの５０％がハラスメントをされた

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



三菱総合研究所みつびしそうごうけんきゅうしょは２月がつに、お年寄としよりの家いえへ行いくヘルパーなど介護かいごの仕事しごとをしている１万まん人にん以上いじょうに調査ちょうさをしました。
調査ちょうさによると、お年寄としよりの家いえへ行いくヘルパーの５０％がハラスメントをされたことがあると答こたえました。ハラスメントは、相手あいてが嫌いやだと思おもうことをしたり言いったりすることです。
いちばん多おおかったハラスメントは、ひどいことを言いわれたりする「心こころへの暴力ぼうりょく」で８１％でした。物ものを投なげられたりする「体からだへの暴力ぼうりょく」は４２％、体からだを触さわられたりする「セクハラ」は３７％でした。
ハラスメントをされて仕事しごとをやめたいと思おもったことがある人ひとは３０％ぐらいいました。
調査ちょうさのあと、介護かいごの仕事しごとをする人ひとたちがハラスメントをされないように、マニュアルを作つくりました。マニュアルには「２人ふたり以上いじょうで家いえへ行いく」とか「何なにがハラスメントになるのか初はじめに伝つたえておく」などと書かいてあります。

宮城県みやぎけんの柴田町しばたまちには、たくさんの桜さくらの木きが並ならんでいる「一目ひとめ千本桜せんぼんざくら」という有名ゆうめいな場所ばしょがあります。毎年まいとし２５万まん人にんが花見はなみに来きて、最近さいきんはタイや香港ほんこんなどから来くる人ひとも増ふえています。
１４日じゅうよっか、柴田町しばたまちの小学生しょうがくせいや中学生ちゅうがくせい３８人にんが、駅えきの前まえなど６つの場所ばしょで観光かんこうに来きた外国人がいこくじんを迎むかえました。

パリのノートルダム大聖堂だいせいどうで火事かじ　高たかい塔とうが燃もえる

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



フランスのパリで１５日にち午後ごご７時じごろ、ノートルダム大聖堂だいせいどうが火事かじになりました。建物たてものの真まん中なかの部分ぶぶんがひどく燃もえて、高たかさ９０ｍの細ほそい塔とうや、建物たてものの屋根やねの半分はんぶん以上いじょうが落おちました。１０時間じかんぐらいが過すぎたころ、火ひは大体だいたい消きえました。
火事かじになったとき大聖堂だいせいどうは閉しまっていたため、観光かんこうに来きた人ひとは中なかにいなかったようです。
大聖堂だいせいどうでは、去年きょねん４月がつから建物たてものを直なおす工事こうじをしていました。消防しょうぼうは、大聖堂だいせいどうの屋根やねの裏うらから火ひが出でたと考かんがえていますが、原因げんいんはまだわかっていません。
近ちかくの店みせにいた女性じょせいは「歴史れきしがある建物たてものなので、私わたしたちはショックを受うけています」と話はなしていました。
ノートルダム大聖堂だいせいどうには、１年ねんに１２００万まん人にんが世界中せかいじゅうから観光かんこうに来きます。１９９１年ねんにはユネスコの世界遺産せかいいさんになりました。
