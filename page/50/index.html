警察けいさつによると、父親ちちおやは「息子むすこは仕事しごとをしないで、ほとんど家いえにいました。家族かぞくは息子むすこが中学生ちゅうがくせいのときから暴力ぼうりょくを受うけていました」などと話はなしています。

警察けいさつは、父親ちちおやが息子むすこのことを誰だれにも相談そうだんできなくて悩なやんでいたと考かんがえています。

「動物どうぶつ看護師かんごし」　国くにが試験しけんを行おこなう国家こっか資格しかくになる

发表于 2019-06-04 更新于 2019-11-30 分类于 JapanEasyNews



動物どうぶつの病院びょういんなどで、獣医師じゅういしの手伝てつだいをする「動物どうぶつ看護師かんごし」は２万まん人にんぐらいいます。動物どうぶつ看護師かんごしの資格しかくを取とるための試験しけんは、民間みんかんの団体だんたいが行おこなっていて、国くには行おこなっていません。しかし、ペットの病気びょうきやけがを高たかい技術ぎじゅつで治なおしてほしいと考かんがえる人ひとが増ふえていて、獣医師じゅういしや看護師かんごしの技術ぎじゅつを上あげることが必要ひつようになっています。
このため国会議員こっかいぎいんのグループは、「動物どうぶつ看護師かんごし」を国家こっか資格しかくにするための法律ほうりつの案あんを作つくりました。国家こっか資格しかくは、国くにの試験しけんに合格ごうかくした人ひとが取とることができる資格しかくです。
案あんでは、獣医師じゅういしの指示しじで「動物どうぶつ看護師かんごし」がペットの血ちを採とったり、ペットに薬くすりをあげたりできるようになります。
いま開ひらいている国会こっかいで、新あたらしい法律ほうりつができることになりそうです。

中国ちゅうごく　多おおくの学生がくせいなどが亡なくなった天安門てんあんもん事件じけんから３０年ねん

发表于 2019-06-04 更新于 2019-11-30 分类于 JapanEasyNews



１９８９年ねん６月がつ４日よっか、中国ちゅうごくの北京ぺきんで「天安門てんあんもん事件じけん」がありました。民主的みんしゅてきな国くににしてほしいと言いって天安門てんあんもん広場ひろばに集あつまっていた市民しみんや学生がくせいを、軍ぐんが銃じゅうで撃うって、たくさんの人ひとが亡なくなりました。
この事件じけんから４日よっかで３０年ねんになりました。天安門てんあんもん広場ひろばには、いつもと同おなじように大勢おおぜいの人ひとが観光かんこうに来きています。しかし、広場ひろばや周まわりには警官けいかんや警察けいさつの車くるまがいつもよりたくさん出でています。

事件じけんのあと中国ちゅうごくでは、民主的みんしゅてきな国くににしてほしいと言いったり、政府せいふを悪わるく言いったりすることができないように厳きびしくチェックされています。４日よっかの朝あさ、外国がいこくで見みることができるＮＨＫのニュースがこの事件じけんについて伝つたえると、中国ちゅうごくでは画面がめんが８分ふん黒くろくなって音おとも聞きこえなくなりました。

世界せかいのワインが集あつまる国際こくさい大会たいかい「デキャンター?ワールド?ワイン?アワード」が、４月がつから５月がつにイギリスのロンドンでありました。大会たいかいでは、１万まん６５００種類しゅるい以上いじょうのワインの味あじをチェックして、賞しょうを決きめました。

東京とうきょうオリンピックの聖火せいかリレー　８５７の市しや町まちを走はしる

发表于 2019-06-03 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねんの東京とうきょうオリンピックの聖火せいかリレーで、８５７の市しや町まちを走はしることが決きまりました。日本にっぽんにある市しや町まちの約やく半分はんぶんです。
岩手県いわてけん陸前高田市りくぜんたかたしにある「奇跡きせきの一本松いっぽんまつ」や熊本県くまもとけんにある「熊本城くまもとじょう」も選えらばれました。「奇跡きせきの一本松いっぽんまつ」は、２０１１年ねんの大おおきな地震じしんで津波つなみが来きたとき、流ながされないで１本ぽんだけ残のこった木きです。「熊本城くまもとじょう」は、２０１６年ねんの大おおきな地震じしんで壊こわれましたが、今いま直なおしています。このように災害さいがいのあと、町まちを元気げんきにするために頑張がんばっている所ところを回まわります。
「富士山ふじさん」や広島県ひろしまけんの「厳島神社いつくしまじんじゃ」、兵庫県ひょうごけんの「姫路城ひめじじょう」や山口県やまぐちけんの「錦帯橋きんたいきょう」など有名ゆうめいな所ところも回まわります。１９６４年ねんの東京とうきょうオリンピックや、１９９８年ねんの長野ながのオリンピックで日本にっぽんの選手せんしゅが活躍かつやくした会場かいじょうも回まわります。
聖火せいかリレーは、来年らいねん３月がつ２６日にちに福島県ふくしまけんを出発しゅっぱつして、１２１日にちで８５７の市しや町まちを回まわります。
