「平成へいせい３１年ねん」の硬貨こうか　６種類しゅるいのセットが人気にんき

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



今年ことしの元号げんごうは４月がつまで平成へいせいですが、５月がつから新あたらしくなります。
大阪おおさかにある造幣局ぞうへいきょくでは、１円えんから５００円えんまで６種類しゅるいの硬貨こうかをつくっています。造幣局ぞうへいきょくの中なかにある店みせでは「平成へいせい３１年ねん」の硬貨こうかのセットを売うっていて、人気にんきになっています。
店みせには毎日まいにちたくさんの人ひとが買かいに来きていて、午前ごぜん中ちゅうに全部ぜんぶなくなってしまう日ひもあります。インターネットなどでも３５万まんセットの注文ちゅうもんがありました。
孫まごと一緒いっしょに店みせに買かいに来きた男性だんせいは「昭和しょうわ最後さいごの年としの硬貨こうかを持もっているので、平成へいせい最後さいごの硬貨こうかも欲ほしくなりました」と話はなしていました。
新あたらしい元号げんごうの硬貨こうかは、今年ことしの夏なつごろにできる予定よていです。

静岡県しずおかけんの海うみでサクラエビの春はるの漁りょうが始はじまる

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



サクラエビは体からだの長ながさが４ｃｍぐらいの小ちいさいえびで、色いろは桜さくらのようなピンクです。
静岡県しずおかけんの駿河湾するがわんでは、春はると秋あきにサクラエビの漁りょうを行おこないます。しかし、去年きょねんは子こどものえびが多おおかったため、秋あきの漁りょうはしませんでした。
とることができる大おおきさのえびが増ふえたため、今年ことしは３月がつ２６日にちから春はるの漁りょうが始はじまりました。最初さいしょの日ひにとったサクラエビは去年きょねんの１５％ぐらいの４ｔでした。
２７日にちの朝あさの競せりでは、１５ｋｇの値段ねだんが平均へいきんで１０万まん円えん以上いじょうになりました。去年きょねんの２倍ばい以上いじょうの高たかい値段ねだんです。
今年ことしの春はるの漁りょうは６月がつ５日いつかまで続つづきます。

发表于 2019-03-28 更新于 2019-11-30 分类于 JapanEasyNews



厚生労働省こうせいろうどうしょうは去年きょねん、けがや病気びょうきで外国人がいこくじんが来きたことがあるかどうか、全部ぜんぶの病院びょういんに聞ききました。１０月がつの１か月げつに、けがや病気びょうきで来きた外国人がいこくじんを治療ちりょうした病院びょういんは１９６５でした。調査ちょうさに答こたえた病院びょういんの半分はんぶんぐらいでした。
この中なかの２０%ぐらいの病院びょういんは、お金かねを払はらっていない外国人がいこくじんがいると言いっています。払はらっていないお金かねの平均へいきんは、１つの病院びょういんで４２万まん円えんぐらいでした。１か月げつで１００万まん円えん以上いじょうになる病院びょういんもありました。
日本にっぽんで病院びょういんに行いく外国人がいこくじんは、これから増ふえると考かんがえられています。厚生労働省こうせいろうどうしょうは、外国人がいこくじんがお金かねを払はらいやすくする方法ほうほうなどをこれから考かんがえると言いっています。

发表于 2019-03-27 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうは大おおきな災害さいがいがあったときなどに、急いそいで伝つたえたい情報じょうほうをテレビや新聞しんぶんの記者きしゃたちに説明せつめいしています。例たとえば、震度しんど５弱じゃく以上いじょうの地震じしんがあったときや、大雨おおあめなどで「特別とくべつ警報けいほう」を出だすとき、台風たいふうが近ちかくに来きているとき、火山かざんの噴火ふんかの警報けいほうを出だすときなどです。
気象庁きしょうちょうは、耳みみに障害しょうがいがある人ひとにも情報じょうほうを伝つたえることができるように、説明せつめいするときに手話しゅわも使つかうことにしました。今いまは昼間ひるまだけですが、気象庁きしょうちょうはできるだけ早はやく、２４時間じかんいつでも手話しゅわを使つかうようにしたいと考かんがえています。

日本相撲協会にほんすもうきょうかいは２７日にち、貴景勝たかけいしょうを大相撲おおずもうで上うえから２番目ばんめの「大関おおぜき」にすることを決きめました。
大関おおぜきにすることを伝つたえる式しきで、貴景勝たかけいしょうは「大関おおぜきになって恥はずかしくないように、武士ぶしが守まもっていた考かんがえ方かたを大切たいせつにして頑張がんばります。感謝かんしゃする気持きもちや優やさしい気持きもちを忘わすれないようにします」と話はなしました。

パブロ?ピカソはスペインの有名ゆうめいな画家がかです。２０年ねん前まえにフランスで盗ぬすまれたピカソの絵えが、今月こんげつオランダで見みつかりました。
この絵えは、ピカソが１９３８年ねんにかいた女性じょせいの絵えです。外国がいこくの新聞しんぶんによると、サウジアラビア人じんのお金持かねもちが４億おく円えんで絵えを買かいましたが、１９９９年ねんにフランスの港みなとで船ふねの中なかから盗ぬすまれました。それからずっと、絵えがどこにあるかわかりませんでした。

ＮＨＫは今年ことし１月がつから３月がつ、都道府県とどうふけんや市町村しちょうそんなど地方ちほうの議会ぎかいに女性じょせいの議員ぎいんが何なん人にんいるか調しらべました。全部ぜんぶで３万まん２４５０人にんいる議員ぎいんの中なかで、女性じょせいは１３％の４２７０人にんでした。
