１０連休れんきゅう　旅行りょこうに行いく人ひとが今いままででいちばん多おおくなりそう

发表于 2019-04-08 更新于 2019-11-30 分类于 JapanEasyNews



今年ことしは４月がつ２７日にちから５月がつ６日むいかまで１０連休れんきゅうです。旅行会社りょこうがいしゃのＪＴＢによると、４月がつ２５日にちから５月がつ５日いつかまでの間あいだに２４６７万まん人にんが旅行りょこうに行いく予定よていです。去年きょねんより３０万まん人にん多おおくなっていて、１９６９年ねんに調しらべ始はじめてからいちばん多おおくなりそうです。ＪＴＢは、１０連休れんきゅうがあるため、旅行りょこうに行いく人ひとが多おおくなったと考かんがえています。
このうち、日本にっぽんの中なかで旅行りょこうをする人ひとは２４０１万まん人にんです。北海道ほっかいどうや沖縄おきなわ、皇室こうしつと関係かんけいが深ふかい伊勢神宮いせじんぐうに行いくツアーが人気にんきになっています。外国がいこくへ旅行りょこうに行いく人ひとは６６万まん２０００人にんで、ヨーロッパやオーストラリアなどに長ながい間あいだ旅行りょこうをする人ひとが増ふえています。
アンケートに答こたえた２０００人にんのうち、半分はんぶん以上いじょうの人ひとは、この間あいだに２回かい以上いじょう旅行りょこうに行いくと答こたえていました。
ＪＴＢの人ひとは「新あたらしい元号げんごうになる初はじめての連休れんきゅうなので、旅行りょこうに行いく人ひとが増ふえたと思おもいます。旅行りょこうに使つかうお金かねも今いままででいちばん多おおくなりそうです」と話はなしています。

发表于 2019-04-08 更新于 2019-11-30 分类于 JapanEasyNews



宮城県みやぎけん気仙沼市けせんぬましと大島おおしまという島しまの間あいだに、長ながさ３５６ｍの「気仙沼けせんぬま大島おおしま大橋おおはし」という橋はしができました。橋はしと周まわりの道路どうろができるまでに７年ねん、お金かねは全部ぜんぶで２７０億おく円えんかかりました。
大島おおしまでは７日なのか、お祝いわいの式しきに２００人にんが集あつまりました。そして式しきのあと、島しまの人ひとたちが初はじめて橋はしを渡わたりました。そのあと、車くるまも通とおることができるようになりました。
大島おおしまには２５００人にんが住すんでいます。今いままで、気仙沼市けせんぬましの中心ちゅうしんには船ふねなどで行いっていましたが、風かぜや雨あめがひどいときは行いくことができませんでした。島しまに住すんでいるお年寄としよりは「自分じぶんが生いきている間あいだに橋はしができるとは思おもっていなかったので、とてもうれしいです」と話はなしていました。
式しきに出でた宮城県みやぎけんの村井むらい知事ちじは「本当ほんとうにうれしいです。この橋はしを利用りようして、８年ねん前まえの東日本大震災ひがしにほんだいしんさいで被害ひがいがあった大島おおしまや気仙沼市けせんぬましをにぎやかにしたいです」と話はなしていました。

鎌倉市かまくらし　「食たべながら歩あるかないでください」

发表于 2019-04-05 更新于 2019-11-30 分类于 JapanEasyNews



神奈川県かながわけん鎌倉市かまくらしは、ほかの人ひとの迷惑めいわくになることをしないように観光かんこう客きゃくなどにお願ねがいする規則きそくをつくりました。例たとえば、人ひとが大勢おおぜいいる場所ばしょで食たべながら歩あるくことや、線路せんろの周まわりなど危険きけんな場所ばしょで写真しゃしんを撮とること、山やまの中なかの道みちで走はしって前まえの人ひとを追おい越こすことなどです。
鎌倉市かまくらしには観光かんこう客きゃくが毎年まいとし２０００万まん人にんぐらい来きます。観光かんこう客きゃくのマナーが悪わるくて困こまるという相談そうだんが多おおいため、この規則きそくをつくりました。
この規則きそくは４月がつ１日ついたちから始はじまりました。しかし、お土産みやげや食たべ物ものを売うる店みせなどが並ならんでいる通とおりでは、観光かんこう客きゃくがソフトクリームなどを食たべながら歩あるいていました。
多おおくの人ひとがハイキングに来くる山やまでは、市しの職員しょくいんが看板かんばんを立たてて、前まえの人ひとを追おい越こすときは走はしらないで歩あるいてくださいとお願ねがいしていました。
鎌倉市かまくらしはこれから外国語がいこくごでも規則きそくを知しらせる予定よていです。

ハイブリッド車しゃは、ガソリンと電気でんきで走はしる車くるまです。ハイブリッド車しゃを２０年ねん以上いじょう作つくっているトヨタ自動車じどうしゃは、ハイブリッド車しゃを作つくる技術ぎじゅつの特許とっきょを２万まん３０００件けん以上いじょう持もっています。
トヨタは、このたくさんの技術ぎじゅつをほかの会社かいしゃが無料むりょうで使つかってもいいことにしました。
トヨタは、ハイブリッド車しゃを作つくる会社かいしゃを多おおくして、世界せかいでもっとたくさんのハイブリッド車しゃが売うれるようにしたいと考かんがえています。
ハイブリッド車しゃの部品ぶひんの中なかには、電気でんき自動車じどうしゃでも使つかう部品ぶひんがたくさんあります。これからは電気でんき自動車じどうしゃがもっと作つくられるようになるため、トヨタは電気でんき自動車じどうしゃを作つくる会社かいしゃにもトヨタのグループの会社かいしゃから部品ぶひんを買かってもらうことができると考かんがえています。
トヨタの副ふく社長しゃちょうは「ハイブリッド車しゃや電気でんき自動車じどうしゃが増ふえて、地球ちきゅうの気温きおんを上あげる二酸化炭素にさんかたんそを少すくなくできるといいと思おもいます」と話はなしています。
