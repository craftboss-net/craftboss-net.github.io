アフガニスタンの子こどもに贈おくるランドセルを集あつめる

发表于 2019-03-18 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうとの文京区ぶんきょうくはＮＧＯと一緒いっしょに、アフガニスタンの子こどもたちに贈おくるために、使つかい終おわったランドセルを集あつめました。
アフガニスタンには、生活せいかつのために働はたらかなければならないため、学校がっこうに通かようことができない子こどもがたくさんいます。ＮＧＯは子こどもたちにランドセルを贈おくって、学校がっこうに行いくことの大切たいせつさを親おやたちに知しってもらおうと考かんがえています。
３年ねん前まえに小学校しょうがっこうを卒業そつぎょうした息子むすこのランドセルを贈おくった女性じょせいは「アフガニスタンの子こどもがこのランドセルを持もって学校がっこうに通かよってくれるとうれしいです」と話はなしていました。
前まえにランドセルをもらったアフガニスタンの子こどもたちは「勉強べんきょうを頑張がんばって、学校がっこうの先生せんせいになりたいです」などと話はなしているそうです。

長ながい時間じかん働はたらく人ひとは突然とつぜん「心筋しんきん梗塞こうそく」になる可能性かのうせいが高たかい

发表于 2019-03-18 更新于 2019-11-30 分类于 JapanEasyNews



国立こくりつがん研究けんきゅうセンターや大阪おおさか大学だいがくのグループは、働はたらく時間じかんと「心筋しんきん梗塞こうそく」の関係かんけいを研究けんきゅうしました。心筋しんきん梗塞こうそくは心臓しんぞうに血ちが流ながれにくくなって亡なくなることもある病気びょうきです。
グループは、４０歳さいから５９歳さいの男性だんせい、約やく１万まん５０００人にんを２０年ねん調しらべました。１日いちにちに働はたらく時間じかんが７時間じかん以上いじょうで９時間じかんより短みじかい人ひとたちと、ほかの人ひとたちを比くらべました。１日いちにちに１１時間じかん以上いじょう働はたらく人ひとは、突然とつぜん、心筋しんきん梗塞こうそくになる可能性かのうせいが１．６３倍ばい高たかくなりました。５０歳さいから５９歳さいで長ながい時間じかん働はたらく人ひとは、２．６倍ばい高たかくなりました。
長ながい時間じかん働はたらく人ひとは十分じゅうぶん寝ねていないために、疲つかれたりストレスが増ふえたりすることが原因げんいんだとグループは考かんがえています。大阪おおさか大学だいがくの磯いそ博康ひろやす先生せんせいは「長ながい時間じかん働はたらく生活せいかつをずっと続つづけていると、定年ていねんで仕事しごとをやめたあとに心筋しんきん梗塞こうそくになる可能性かのうせいが上あがると考かんがえています」と話はなしています。

ニュージーランド　モスクで男おとこが銃じゅうを撃うって大勢おおぜいが亡なくなる

发表于 2019-03-18 更新于 2019-11-30 分类于 JapanEasyNews



１５日にち、ニュージーランドのクライストチャーチにある２つのモスクで男おとこが銃じゅうを周まわりに撃うちました。モスクはイスラム教きょうの人ひとが祈いのる場所ばしょです。警察けいさつはオーストラリアで生うまれた男おとこを捕つかまえました。
この事件じけんで５０人にんが亡なくなりました。半分はんぶん以上いじょうは、パキスタンやエジプト、ヨルダンなどの外国がいこくから来きた人ひとたちでした。
ニュージーランドは、いろいろな文化ぶんかや宗教しゅうきょうを大事だいじにする社会しゃかいをつくろうと考かんがえていて、去年きょねんは外国がいこくから約やく５万まん人にんが来きて生活せいかつしています。このような国くにで起おこった事件じけんにみんな驚おどろいています。
１６日にち、ニュージーランドのアーダーン首相しゅしょうが亡なくなった人ひとの家族かぞくなどに会あって、安全あんぜんと自由じゆうを約束やくそくしました。モスクの近ちかくでは大勢おおぜいの人ひとが、亡なくなった人ひとのために祈いのって花はなを置おいています。

发表于 2019-03-15 更新于 2019-11-30 分类于 JapanEasyNews



ロボットがコンビニの品物しなものを客きゃくに届とどける実験じっけんが、神奈川県かながわけんにある大学だいがくの中なかでありました。コンビニの「ローソン」や自動じどう運転うんてんの技術ぎじゅつを研究けんきゅうしている会社かいしゃなどが行おこないました。
ロボットは１ｍぐらいの大おおきさで、箱はこのような形かたちをしています。客きゃくがスマートフォンのアプリで品物しなものを注文ちゅうもんすると、ロボットは店みせを出でて客きゃくの所ところまで品物しなものを届とどけます。ロボットはセンサーやカメラを使つかって、近ちかくに人ひとがいたら止とまります。人ひとに会あうと声こえを出だしてあいさつもします。
ローソンなどは、このロボットを使つかうと、お年寄としよりや店みせに行いくことができない人ひとの買かい物ものが便利べんりになると考かんがえています。運はこぶ人ひとがいなくて品物しなものを届とどけることができないという問題もんだいもなくなります。このため、ロボットが道みちを通とおることができるように法律ほうりつを変かえてもらったりして、早はやく使つかい始はじめたいと考かんがえています。

ガザ地区ちくでは、日本にっぽんにお礼れいの気持きもちを伝つたえるために、東日本ひがしにほん大震災だいしんさいがあった３月がつ１１日にちごろに、毎年まいとしイベントを行おこなっています。子こどもたちが空そらにたこをあげて、被害ひがいがあった町まちが元気げんきになるように祈いのります。

新あたらしいビザで外国人がいこくじんを雇やとうときの規則きそくができる

发表于 2019-03-15 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの政府せいふは来月らいげつから、国くにが決きめた仕事しごとの技術ぎじゅつがある外国人がいこくじんに「特定技能とくていぎのう１号ごう」と「特定技能とくていぎのう２号ごう」という新あたらしいビザを出だします。法務省ほうむしょうは１５日にち、新あたらしいビザで外国人がいこくじんを雇やとう会社かいしゃなどが守まもらなければならない規則きそくを発表はっぴょうしました。
規則きそくでは、外国人がいこくじんの給料きゅうりょうは日本人にっぽんじんと同おなじか、それ以上いじょうにしなければならないと決きまっています。給料きゅうりょうは、銀行ぎんこうや郵便局ゆうびんきょくなどの口座こうざに入いれたりして、どのくらい払はらったかわかるようにしなければなりません。外国人がいこくじんが生活せいかつしやすいように、携帯電話けいたいでんわを契約けいやくするときなどは手伝てつだわなければなりません。
外国人がいこくじんの中なかには、日本にっぽんで働はたらくために悪わるい人ひとにたくさんお金かねを払はらう人ひとがいて、問題もんだいになっています。このような問題もんだいをなくすために、外国人がいこくじんを雇やとう会社かいしゃなどは、悪わるい人ひとにお金かねを払はらっていないかどうか確認かくにんしなければなりません。仕事しごとをやめたあと、国くにに帰かえるお金かねがない場合ばあいは、会社かいしゃがお金かねを出ださなければなりません。
この規則きそくは、法務省ほうむしょうのウェブサイトで見みることができます。
