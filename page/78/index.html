スマートフォンの充電じゅうでんのケーブルの事故じこが５年ねんで８０件けん以上いじょう

发表于 2019-03-04 更新于 2019-11-30 分类于 JapanEasyNews



製品せいひんの安全あんぜんなどを調しらべているＮＩＴＥが、スマートフォンなどを充電じゅうでんするケーブルの事故じこについて発表はっぴょうしました。ケーブルがとても熱あつくなったり火ひが出でたりした事故じこは、去年きょねん３月がつまでの５年ねんに８６件けんあって、２０人にんぐらいがやけどをしています。
いちばん多おおかったのは、スマートフォンに入いれる部分ぶぶんの「コネクター」に細こまかいごみや水分すいぶんなどが入はいったまま使つかったときの事故じこでした。次つぎに多おおかったのは、曲まがったりして形かたちが変かわったコネクターを使つかったときの事故じこでした。電気でんきが通とおっているコネクターに長ながい時間じかん体からだが触ふれていて、ひどいやけどをした人ひともいました。
ＮＩＴＥは「安全あんぜんのため、ごみが入はいったり曲まがったりしたコネクターは使つかわないでください。そして、電気でんきが通とおっているコネクターに長ながい時間じかん体からだが触ふれないようにしてください」と言いっています。

寝ねる時間じかんが足たりない人ひと　休やすみの日ひだけ長ながく寝ねても十分じゅうぶんではない

发表于 2019-03-04 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのコロラド大学だいがくの研究けんきゅうチームは、寝ねる時間じかんが足たりない人ひとが休やすみの日ひに長ながく寝ねるとどうなるか調しらべる実験じっけんをしました。
実験じっけんでは１８歳さいから３９歳さいまでの人ひとに平日へいじつは５時間じかん、休やすみの２日ふつか間かんは好すきなだけ寝ねてもらいました。
９日ここのか間かんの実験じっけんの結果けっか、休やすみの日ひに寝ねた時間じかんはいつもより１時間じかんぐらい長ながいだけでした。その代かわり、平日へいじつに寝ねるのが遅おそくなって、十分じゅうぶん寝ねることができなくなることがわかりました。
休やすみの日ひだけ長ながく寝ねた人ひとの中なかには、起おきている時間じかんが長ながくなって、食事しょくじ以外いがいにお菓子かしなどを食たべて、体重たいじゅうが１．３ｋｇ増ふえた人ひともいました。糖尿病とうにょうびょうになる可能性かのうせいも高たかくなると言いっています。
研究けんきゅうチームは、毎日まいにち７時間じかん以上いじょう寝ねたほうがいいと言いっています。
