「アフリカ豚とんコレラ」が中国ちゅうごくから入はいらないように空港くうこうで検査けんさ

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


中国ちゅうごくでは、豚ぶたの病気びょうきの「アフリカ豚とんコレラ」がとても多おおくなっています。人ひとにはうつりませんが、この病気びょうきになった豚ぶたはほとんどが死しにます。日本にっぽんで見みつかっている「豚とんコレラ」とは別べつの病気びょうきです。
２月がつ４日よっかから中国ちゅうごくの春節しゅんせつの休やすみが始はじまって、日本にっぽんへ旅行りょこうに来くる中国人ちゅうごくじんが増ふえています。農林水産省のうりんすいさんしょうは「アフリカ豚とんコレラ」がうつった豚肉ぶたにくが日本にっぽんに入はいらないように、空港くうこうで検査けんさする人ひとを２月がつの終おわりまで多おおくしています。
北海道ほっかいどうの新千歳空港しんちとせくうこうには５日いつか、中国ちゅうごくの上海しゃんはいや北京ぺきんからの飛行機ひこうきで大勢おおぜいの中国人ちゅうごくじんが来きました。自分じぶんの荷物にもつを取とる場所ばしょでは、肉にくで作つくった食たべ物ものがかばんに入はいっていないかどうか、犬いぬがにおいをチェックしていました。
農林水産省のうりんすいさんしょうの人ひとは「病気びょうきのウイルスが日本にっぽんに入はいったら、豚ぶたを育そだてている農家のうかに大おおきな被害ひがいが出でるかもしれません。豚肉ぶたにくなどの輸出ゆしゅつもできなくなります」と話はなしています。

成田空港なりたくうこう　飛行機ひこうきが出発しゅっぱつや到着とうちゃくできる時間じかんを長ながくする

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


成田空港なりたくうこうで飛行機ひこうきが出発しゅっぱつしたり到着とうちゃくしたりできるのは夜よる１１時じまでです。国くにや空港くうこうの会社かいしゃは、２０２０年ねんに東京とうきょうでオリンピックとパラリンピックがあるため、成田空港なりたくうこうを使つかうことができる時間じかんを１時間じかん長ながくしたいと考かんがえました。
空港くうこうの周まわりにあるいくつかの市しや町まちは最初さいしょ、時間じかんを長ながくすることに反対はんたいしました。このため空港くうこうの会社かいしゃは、飛行機ひこうきの音おとが近ちかくの家いえに聞きこえないようにする工事こうじのお金かねの一部いちぶを出だすと言いいました。
４日よっか、国くにと空港くうこうの会社かいしゃ、空港くうこうの周まわりの９つの市しと町まちなどが集あつまって、話はなし合あいました。その結果けっか、今年ことし１０月がつからは１時間じかん長ながくして夜よるの０時じまで使つかうことができることに決きまりました。

去年きょねんの西日本にしにほんの雨あめ　１２の川かわで水みずが堤防ていぼうの中なかを通とおって外そとに出でる

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


去年きょねん７月がつ、西日本にしにほんでとてもたくさんの雨あめが降ふって、多おおくの川かわで水みずが増ふえました。
国土交通省こくどこうつうしょうが調しらべると、このとき１２の川かわの２８の場所ばしょで、川かわの水みずが堤防ていぼうの中なかや下したを通とおって外そとに出でていたことがわかりました。これは、堤防ていぼうが壊こわれる前まえに起おこると言いわれています。川かわの水みずが増ふえて水みずの圧力あつりょくがとても強つよくなったことが原因げんいんです。堤防ていぼうに穴あながあいていた川かわもありました。
国土交通省こくどこうつうしょうは、２８の場所ばしょで川かわと堤防ていぼうの間あいだに金属きんぞくの板いたを入いれて、水みずが通とおらないようにする工事こうじなどをします。

太平洋戦争たいへいようせんそうで海うみに沈しずんだ日本にっぽんの軍ぐんの船ふね「比叡ひえい」が見みつかる

发表于 2019-02-06 更新于 2019-11-30 分类于 JapanEasyNews


太平洋戦争たいへいようせんそうが続つづいていた１９４２年ねん、日本にっぽんの軍ぐんの船ふね「比叡ひえい」は太平洋たいへいようの南みなみにあるソロモン諸島しょとうの近ちかくでアメリカなどに攻撃こうげきされました。そのあと船ふねは海うみに沈しずみましたが、どうやって沈しずんだかはわかっていません。
太平洋戦争たいへいようせんそうで沈しずんだ船ふねを探さがしているアメリカのチームによると、１月がつ３１日にちにソロモン諸島しょとうの近ちかくの深ふかさ９８５ｍの海うみの底そこで「比叡ひえい」が見みつかりました。このチームは、アメリカのＩＴの会社かいしゃのマイクロソフトを作つくったポール?アレンさんのチームです。
「比叡ひえい」は上うえと下したが反対はんたいになって沈しずんでいました。船ふねの長ながさは２２２ｍですが、見みつかった船ふねは前まえの方ほうの７０ｍぐらいがなくなっていました。このため、大おおきな爆発ばくはつが原因げんいんで沈しずんだのかもしれないと初はじめてわかりました。
チームはこれから船ふねの写真しゃしんなどをみんなに見みせて、研究けんきゅうの役やくに立たてたいと言いっています。

「さっぽろ雪ゆきまつり」が始はじまる　今年ことしで７０回かい目め

北海道ほっかいどう札幌市さっぽろしで行おこなう「さっぽろ雪ゆきまつり」は今年ことしで７０回かい目めです。大通おおどおり公園こうえんなどでは４日よっかから始はじまりました。会場かいじょうには、市民しみんなどが雪ゆきや氷こおりで作つくった像ぞうが１９０ぐらいあります。高たかさ１０ｍ以上いじょうの大おおきな像ぞうは５つあります。
そのうちの１つは、札幌さっぽろの有名ゆうめいな観光地かんこうちの前まえを走はしる馬うまの像ぞうです。去年きょねん９月がつの地震じしんで大おおきな被害ひがいがあった町まちでは、馬うまをたくさん育そだてています。この像ぞうは、町まちが元気げんきになるように願ねがって作つくられました。
３０歳さいの男性だんせいは「すばらしい像ぞうです。馬うまのように元気げんきに走はしって前まえのような町まちに戻もどってほしいです」と話はなしていました。
「さっぽろ雪ゆきまつり」は今月こんげつ１１日にちまでで、２５０万まん人にんぐらいが見みに来くると言いわれています。

ドイツのメルケル首相しゅしょうが日本にっぽんに来きて、安倍あべ総理大臣そうりだいじんと会あって話はなしをしました。
