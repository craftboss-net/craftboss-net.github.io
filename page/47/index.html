パンダのシャンシャン　２歳さいの誕生日たんじょうび

发表于 2019-06-12 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうとの上野動物園うえのどうぶつえんで生うまれた雌めすのパンダのシャンシャンは、１２日にちに２歳さいになりました。
シャンシャンは体からだの重おもさが６０ｋｇ以上いじょうになって、今いまはお母かあさんと別べつの部屋へやで生活せいかつしています。シャンシャンは、木きに登のぼったりタケノコを食たべたりして、元気げんきに動うごいていました。
東京都とうきょうとはシャンシャンのお父とうさんとお母かあさんを中国ちゅうごくから借かりています。シャンシャンは、２歳さいになったら中国ちゅうごくに返かえす予定よていでした。東京都とうきょうとは中国ちゅうごくと話はなし合あって、来年らいねん１２月がつにシャンシャンを返かえすことになりました。
上野動物園うえのどうぶつえんは「これからもシャンシャンを大切たいせつに育そだてます」と話はなしています。

運転うんてんするお年寄としよりに事故じこで家族かぞくが亡なくなった人ひとの手紙てがみを送おくる

发表于 2019-06-12 更新于 2019-11-30 分类于 JapanEasyNews



お年寄としよりが運転うんてんする車くるまの事故じこが続つづいています。
大分県おおいたけんの警察けいさつはお年寄としよりに運転うんてんをやめることを考かんがえてもらうため、運転うんてん免許めんきょがある８０歳さい以上いじょうの人ひとに手紙てがみを送おくっています。１０日とおかからは、今年ことし４月がつに事故じこで妻つまと３歳さいの娘むすめが亡なくなった男性だんせいからの手紙てがみも一緒いっしょに送おくっています。
この男性だんせいの妻つまと娘むすめは、東京とうきょうの池袋いけぶくろで横断歩道おうだんほどうを渡わたっていたとき、８７歳さいの人ひとが運転うんてんする車くるまがぶつかってきて、亡なくなりました。
男性だんせいの手紙てがみには「大切たいせつな２人ふたりがいなくなって、本当ほんとうに悲かなしいです。少すこしでも車くるまの運転うんてんに心配しんぱいがある人ひとは運転うんてんをしないでほしいです」と書かいてあります。
大分県おおいたけんの警察けいさつは来年らいねん３月がつまでに１万まん人にんぐらいに男性だんせいの手紙てがみを送おくる予定よていです。

会社かいしゃはこれからバルーンを作つくって、２０２１年ねんにサービスを始はじめたいと考かんがえています。

東京福祉大学とうきょうふくしだいがく　留学生りゅうがくせいが３年ねんで１６００人にん行方不明ゆくえふめいになる

发表于 2019-06-12 更新于 2019-11-30 分类于 JapanEasyNews



東京福祉大学とうきょうふくしだいがくでは、ベトナム人じんやネパール人じんなどの留学生りゅうがくせいが大学だいがくに来こなくなってどこにいるかわからなくなりました。
国くにが調しらべると、３年ねんで１６１０人にんが行方不明ゆくえふめいになっていました。この中なかの１１１３人にんは「学部研究生がくぶけんきゅうせい」で、大学だいがくに入はいる前まえの準備じゅんびの勉強べんきょうをしているはずでした。
国くには「しっかり勉強べんきょうできる環境かんきょうではないのに、大勢おおぜいの留学生りゅうがくせいを入いれた大学だいがくに問題もんだいがある」と言いいました。そして、東京福祉大学とうきょうふくしだいがくの学部研究生がくぶけんきゅうせいには留学生りゅうがくせいのビザを出ださないことにしました。
このようなことがないように、国くには日本中にほんじゅうの大学だいがくを調しらべることにしました。大学だいがくをやめた人ひとや、やめたあと国くにに帰かえらないで日本にっぽんに残のこっている人ひとの数かずを調しらべます。そして、大学だいがくに留学生りゅうがくせいの勉強べんきょうや生活せいかつをしっかりチェックするように言いいます。よくならない場合ばあい、その大学だいがくには留学生りゅうがくせいが入学にゅうがくできないようにして、大学だいがくの名前なまえも発表はっぴょうします。

旅行りょこうに行いく人ひとは、健康けんこうなことと、行いく前まえに訓練くんれんを受うけることが必要ひつようです。１年ねんに２人ふたりだけ、長ながい場合ばあい３０日にち宇宙うちゅうステーションにいることができます。
宇宙うちゅうステーションに泊とまるお金かねは、食事しょくじ代だいも入いれて１泊ぱく３８０万まん円えんで、３０日にちいる場合ばあいは１億おく円えん以上いじょう必要ひつようです。宇宙船うちゅうせんで行いくため６３億おく円えんぐらいのお金かねもかかります。
ＮＡＳＡは、５年ねんあとに宇宙飛行士うちゅうひこうしが月つきに下おりる計画けいかくを進すすめています。このため、旅行りょこうに行いく人ひとが払はらったお金かねを、この計画けいかくに使つかいたいと言いっています。

このため国立国語研究所こくりつこくごけんきゅうしょは、日本にっぽんで生活せいかつしていて簡単かんたんな日本語にほんごを読よんだり書かいたりできない人ひとがどのくらいいるか調しらべることにしました。
