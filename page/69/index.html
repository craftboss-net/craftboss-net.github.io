新あたらしい元号げんごうが決きまって「万葉集まんようしゅう」という本ほんが人気にんき

发表于 2019-04-03 更新于 2019-11-30 分类于 JapanEasyNews



「万葉集まんようしゅう」は日本にっぽんの１２００年ねん以上いじょう前まえの本ほんです。昔むかしの人ひとが作つくった詩しがたくさん書かいてあります。５月がつからの新あたらしい元号げんごうの「令和れいわ」は、万葉集まんようしゅうにある言葉ことばの中なかから選えらばれました。
本屋ほんやでは、万葉集まんようしゅうや、書かいてある詩しを今いまの日本語にほんごで説明せつめいした本ほんなどが人気にんきになっています。東京とうきょうの新宿しんじゅくにある本屋ほんやの人ひとは「いつもはあまり売うれないので驚おどろいています」と話はなしていました。
本ほんを作つくっている会社かいしゃには、本屋ほんやからの注文ちゅうもんがたくさん来きています。本ほんが足たりないため、新あたらしく１万まん冊さつ作つくることにした会社かいしゃもあります。

水爆すいばくの実験じっけんで被ひばくした船ふねを展示てんじしている建物たてものが新あたらしくなる

发表于 2019-04-03 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうと江東区こうとうくに「第五福竜丸だいごふくりゅうまる」という船ふねを展示てんじしている建物たてものがあります。この船ふねは１９５４年ねん３月がつ、太平洋たいへいようへ魚さかなをとりに行いったとき、アメリカが行おこなった水爆すいばくの実験じっけんで被ひばくしました。水爆すいばくは爆発ばくはつしたときの力ちからが原爆げんばくよりとても大おおきい核兵器かくへいきです。原爆げんばくや水爆すいばくの被害ひがいを世界せかいからなくしたいと考かんがえて、この建物たてものができました。
去年きょねん７月がつから古ふるくなった建物たてものを直なおしていましたが、４月がつ２日ふつか、新あたらしくなってまたオープンしました。新あたらしい建物たてものでは、船ふねの中なかの３Ｄの映像えいぞうも見みることができるようになりました。船ふねに乗のっていた人ひとが被ひばくしたときのことなどを話はなしているビデオも見みることができます。
船ふねを展示てんじしている団体だんたいの人ひとは「ここが平和へいわを祈いのる場所ばしょになって、『第五福竜丸だいごふくりゅうまる』のことを将来しょうらいに伝つたえ続つづけてほしいです」と話はなしていました。

福島県ふくしまけん大熊町おおくままち　町まちを守まもる「じじい部隊ぶたい」の活動かつどうが終おわる

发表于 2019-04-02 更新于 2019-11-30 分类于 JapanEasyNews



福島県ふくしまけん大熊町おおくままちは、８年ねん前まえの原子力げんしりょく発電所はつでんしょの事故じこのあと、町まちに入はいることができなくなりました。
人ひとがいなくなった町まちを守まもるため、６年ねん前まえから「じじい（おじいさん）部隊ぶたい」というグループが、町まちの掃除そうじや安全あんぜんのチェックを続つづけてきました。グループは、６人にんの男性だんせいで平均へいきん６６歳さいのため、この名前なまえになりました。
大熊町おおくままちは４月がつ１０日とおかから町まちの一部いちぶに入はいることができるようになって、役場やくばも仕事しごとを始はじめます。このため、グループの活動かつどうは３月がつ３１日にちに終おわることになりました。
最後さいごの日ひは、町まちの人ひとたちが６人にんにお礼れいを言いって花はなを渡わたしました。グループの人ひとは「私わたしたちができることはもう全部ぜんぶしました。あとは若わかい人ひとたちにお願ねがいします」と話はなしていました。

新あたらしい元号げんごうを使つかうことができるようにシステムを変かえる

发表于 2019-04-02 更新于 2019-11-30 分类于 JapanEasyNews



５月がつからの新あたらしい元号げんごうが「令和れいわ」になることが決きまりました。
役所やくしょや会社かいしゃは、新あたらしい元号げんごうを使つかうことができるように、コンピューターのシステムを変かえています。政府せいふは５月がつ１日ついたちまでに、国くになどが管理かんりしている５７６のシステムを変かえて、「令和れいわ」を使つかうことができるようにします。
東京都とうきょうと新宿区しんじゅくくには、元号げんごうに慣なれていない外国人がいこくじんがたくさん住すんでいます。このため、役所やくしょに出だす書類しょるいの一部いちぶでは、元号げんごうのほかに、「２０１９年ねん」のように西暦せいれきで書かくことができるようにしました。
日本にっぽんで運転うんてん免許めんきょ証しょうを持もつ外国人がいこくじんが増ふえているため、運転うんてん免許めんきょ証しょうには運転うんてんできる期間きかんを元号げんごうと西暦せいれきの両方りょうほうで書かくことになりました。
東京都とうきょうと荒川区あらかわくにあるカレンダーの会社かいしゃは、新あたらしい元号げんごうが決きまってすぐ、書道しょどうの芸術家げいじゅつかに「令和れいわ」の字じを書かいてもらいました。そして、その字じが書かいてあるカレンダーを作つくり始はじめました。４月がつ１７日にちから売うり始はじめます。

風疹ふうしんはウイルスでうつる病気びょうきで、熱ねつが出でたり体からだに赤あかい発疹はっしんが出でたりします。妊娠にんしんしている女性じょせいが風疹ふうしんになると、赤あかちゃんの目めや耳みみ、心臓しんぞうに障害しょうがいが出でることがあります。
国くにの研究所けんきゅうじょによると、今年ことし１月がつから３月がつ２４日にじゅうよっかまでに１０３３人にんが風疹ふうしんになりました。風疹ふうしんになった人ひとがとても多おおかった２０１３年ねんから、３月がつに１０００人にん以上いじょうになった年としはありませんでした。今年ことし風疹ふうしんになった人ひとの６０％ぐらいは東京とうきょうとその周まわりの県けんに住すんでいて、大阪おおさかなど近畿地方きんきちほうでも増ふえています。
