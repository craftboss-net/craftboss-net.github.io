「鹿しかの赤あかちゃんに触さわらないで」　１０万まん回かい以上いじょうのリツイート

发表于 2019-05-31 更新于 2019-11-30 分类于 JapanEasyNews



奈良市ならしにある奈良公園ならこうえんでは今いま、鹿しかの赤あかちゃんがたくさん生うまれています。
奈良公園ならこうえんで鹿しかを守まもる 活動かつどうをしている「奈良ならの鹿しか愛護会あいごかい」は２４日にじゅうよっか、インターネットのツイッターに「鹿しかの赤あかちゃんに触さわらないでください。お母かあさんが赤あかちゃんを育そだてなくなります」とメッセージを書かきました。このメッセージを友達ともだちなどに知しらせるリツイートの数かずは３０日にちまでに、１０万まん６０００回かい以上いじょうになりました。
団体だんたいによると、生うまれたばかりの赤あかちゃんに人ひとが触さわって、人ひとのにおいがつくと、お母かあさんが乳ちちをあげなくなります。このため赤あかちゃんが死しんでしまうことがあります。
団体だんたいの人ひとは「かわいい赤あかちゃんは、遠とおくから見みたり写真しゃしんを撮とったりしてください」と話はなしています。

虐待ぎゃくたいから子こどもを守まもる仕事しごとをＡＩが手伝てつだう実験じっけん

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



虐待ぎゃくたいされる子こどもが増ふえていて、子こどもを守まもる児童相談所じどうそうだんじょの職員しょくいんが足たりません。
産業技術総合研究所さんぎょうぎじゅつそうごうけんきゅうしょなどは、児童相談所じどうそうだんじょの仕事しごとをコンピューターのＡＩが手伝てつだう実験じっけんを三重県みえけんですることにしました。
実験じっけんで使つかうコンピューターのＡＩは、三重県みえけんで集あつめた子こどもの傷きずの場所ばしょなど６０００件けんのデータを覚おぼえています。

実験じっけんは、２つの児童相談所じどうそうだんじょにコンピューターを２０台だい入いれて、来月らいげつから始はじまります。

東京とうきょうオリンピックで首都高速道路しゅとこうそくどうろが混こまないようにする

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



政府せいふや東京都とうきょうとなどは、来年らいねんの東京とうきょうオリンピックとパラリンピックで選手せんしゅなどが会場かいじょうへ行いくとき、東京都とうきょうととその周まわりにある首都高速道路しゅとこうそくどうろがたくさんの車くるまで混こまないようにする案あんを考かんがえました。
案あんでは、オリンピックとパラリンピックの間あいだの毎日まいにち午前ごぜん６時じから午後ごご１０時じまで、東京都とうきょうとの中なかでは首都高速道路しゅとこうそくどうろの料金りょうきんをいつもより１０００円えん高たかくします。タクシーやバス、店みせの品物しなものを運はこぶトラックなどはいつもと同おなじ料金りょうきんです。
午前ごぜん０時じから午前ごぜん４時じまでは首都高速道路しゅとこうそくどうろの全部ぜんぶの道みちで、どの車くるまも料金りょうきんを半分はんぶんぐらいにします。
開会かいかい式しきと閉会へいかい式しきの日ひは、大勢おおぜいの選手せんしゅなどが会場かいじょうへ行いきます。このため、式しきと関係かんけいがない車くるまのほとんどは長ながい時間じかん、首都高速道路しゅとこうそくどうろを通とおることができなくなります。
政府せいふなどは、道みちが混こまないようにする計画けいかくを今年ことし８月がつごろ決きめる予定よていです。

アフリカや南みなみアメリカではデータを集あつめていないため、中国ちゅうごく以外いがいの国くにから出でている可能性かのうせいもあると言いっています。

新あたらしい法律ほうりつは、来年らいねん５月がつまでに始はじまる予定よていです。
