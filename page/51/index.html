フランスの美術館びじゅつかんが「裸はだかのモナリザ」を展示てんじ

发表于 2019-05-31 更新于 2019-11-30 分类于 JapanEasyNews



今年ことしは、レオナルド?ダビンチが亡なくなってから５００年ねんになります。フランスのパリの近ちかくにあるコンデ美術館びじゅつかんは、「裸はだかのモナリザ」と呼よばれる絵えを来月らいげつから展示てんじします。この絵えは、ダビンチがかいた「モナリザ」とよく似にています。
美術館びじゅつかんによると、この絵えはダビンチが仕事しごとをしていた場所ばしょで「モナリザ」のあとにかかれました。この絵えには、左ひだりの手てでかいた部分ぶぶんがあります。このため、左ひだりの手てで絵えをかいていたダビンチが絵えの一部いちぶをかいた可能性かのうせいが高たかいと言いわれています。絵えの女性じょせいは胸むねが膨ふくらんでいますが、腕うでには男性だんせいのような筋肉きんにくがついています。
美術館びじゅつかんの人ひとは「男性だんせいの美うつくしさと女性じょせいの美うつくしさをかいたのだと思おもいます。ダビンチが考かんがえる理想りそうの美うつくしさだと思おもいます」と話はなしていました。

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



三菱みつびしＵＦＪ銀行ぎんこうは６月がつ１０日とおかから、新あたらしく作つくる口座こうざではできるだけ紙かみの通帳つうちょうを使つかわないことに決きめました。口座こうざを作つくった人ひとは、紙かみの通帳つうちょうの代かわりにパソコンやスマートフォンで口座こうざのお金かねの記録きろくを見みます。パソコンなどでは、１０年ねん前まえまでのお金かねの記録きろくを見みることができるようにしています。

最近さいきん、インターネットの銀行ぎんこうを始はじめるＩＴの会社かいしゃなどが増ふえています。このため、いろいろな銀行ぎんこうが店みせやＡＴＭを少すくなくして、インターネットを使つかったサービスを多おおくしています。

東京とうきょうオリンピックで首都高速道路しゅとこうそくどうろが混こまないようにする

发表于 2019-05-30 更新于 2019-11-30 分类于 JapanEasyNews



政府せいふや東京都とうきょうとなどは、来年らいねんの東京とうきょうオリンピックとパラリンピックで選手せんしゅなどが会場かいじょうへ行いくとき、東京都とうきょうととその周まわりにある首都高速道路しゅとこうそくどうろがたくさんの車くるまで混こまないようにする案あんを考かんがえました。
案あんでは、オリンピックとパラリンピックの間あいだの毎日まいにち午前ごぜん６時じから午後ごご１０時じまで、東京都とうきょうとの中なかでは首都高速道路しゅとこうそくどうろの料金りょうきんをいつもより１０００円えん高たかくします。タクシーやバス、店みせの品物しなものを運はこぶトラックなどはいつもと同おなじ料金りょうきんです。
午前ごぜん０時じから午前ごぜん４時じまでは首都高速道路しゅとこうそくどうろの全部ぜんぶの道みちで、どの車くるまも料金りょうきんを半分はんぶんぐらいにします。
開会かいかい式しきと閉会へいかい式しきの日ひは、大勢おおぜいの選手せんしゅなどが会場かいじょうへ行いきます。このため、式しきと関係かんけいがない車くるまのほとんどは長ながい時間じかん、首都高速道路しゅとこうそくどうろを通とおることができなくなります。
政府せいふなどは、道みちが混こまないようにする計画けいかくを今年ことし８月がつごろ決きめる予定よていです。
