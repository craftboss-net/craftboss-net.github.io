ごみを集あつめる車くるまで火事かじ　ガスのボンベなどは正ただしく捨すてて

发表于 2019-10-28 更新于 2019-11-30 分类于 JapanEasyNews



製品せいひんの安全あんぜんなどを調しらべているＮＩＴＥによると、名古屋市なごやしでは今年ことし８月がつまでの５年ねんの間あいだに、ごみを集あつめる車くるまの火事かじが１５０件けんありました。
火事かじの原因げんいんでいちばん多おおかったのは、ガスが入はいったボンベやスプレーで、５８件けんでした。次つぎはライターで２７件けんでした。リチウムイオン電池でんちが原因げんいんの火事かじも１６件けんありました。
ＮＩＴＥによると、リチウムイオン電池でんちは強つよくぶつけると火ひが出でる危険きけんがあります。ごみを集あつめる車くるまがリチウムイオン電池でんちをつぶす 実験じっけんをすると、火ひが出でたり破裂はれつしたりしました。
ＮＩＴＥは、ガスのボンベなどを捨すてるときは、ほかのごみと一緒いっしょに袋ふくろに入いれないで、分わけるように言いっています。充電じゅうでんできる電化製品でんかせいひんは、中なかにリチウムイオン電池でんちが入はいっているか調しらべてから、住すんでいる所ところの決きまりどおりに捨すてるように言いっています。

東京とうきょうの渋谷しぶやで最もっとも高たかい２３０ｍのビルが１１月がつにオープン

发表于 2019-10-28 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうの渋谷駅しぶやえきのそばに「渋谷しぶやスクランブルスクエア」という新あたらしいビルができました。４７階かい建だてで、高たかさは２３０ｍ、地下ちかは７階かいまであります。渋谷駅しぶやえきの周まわりでいちばん高たかいビルです。
地下ちか２階かいから１４階かいは店みせやレストランで、ＮＨＫのテレビ放送ほうそうの新あたらしい技術ぎじゅつを楽たのしむことができる場所ばしょもあります。１７階かいから４５階かいはいろいろな会社かいしゃの事務所じむしょになっていて、ビルのいちばん上うえは「ＳＨＩＢＵＹＡ　ＳＫＹ」という展望台てんぼうだいです。外国人がいこくじんに人気にんきがあるスクランブル交差点こうさてんや東京とうきょうのまちを見みることができます。展望台てんぼうだいは午前ごぜん９時じから午後ごご１１時じまでで、１８歳さい以上いじょうは２０００円えん、インターネットで予約よやくすると１８００円えんです。
大おおきな地震じしんが起おこったときのために、多おおくの階かいに倉庫そうこがあります。家いえに帰かえることができなくなった２０００人にんが３日みっか間かん生活せいかつできるように、食たべ物ものや水みずなどが用意よういしてあります。
このビルは、１１月がつ１日ついたちにオープンします。

アフリカでたくさんの象ぞうが死しんでいる

发表于 2019-10-25 更新于 2019-11-30 分类于 JapanEasyNews



アフリカの南みなみの地方ちほうのジンバブエとボツワナでは、先月せんげつから、死しんだ象ぞうが１５０頭とう以上いじょう見みつかっています。
ジンバブエの国立公園こくりつこうえんの人ひとは、雨あめがほとんど降ふっていないため、食たべ物ものがなくなったり水みずが足たりなくなったりしたことが原因げんいんだと考かんがえています。国立公園こくりつこうえんの人ひとは「今いま生いきている象ぞうも食たべ物ものがなくて苦くるしんでいます。雨あめが降ふらない季節きせつになったら、今いまよりもっとひどくなると思おもいます」と話はなしています。
ボツワナの政府せいふは、長ながい間あいだ食たべ物ものがなくて死しんだり、草くさがないため土つちを食たべて、土つちの中なかの細菌さいきんが体からだに入はいって死しんだりした可能性かのうせいがあると考かんがえています。
環境かんきょうを守まもる活動かつどうをしているＷＷＦによると、アフリカの南みなみの地方ちほうには象ぞうが約やく２９万まん頭とういますが、象牙ぞうげを取とるために殺ころされて数かずが減へっています。ＷＷＦは、地球ちきゅうの気温きおんが上あがって雨あめが少すくなくなって環境かんきょうが変かわると、象ぞうがもっと少すくなくなるかもしれないと心配しんぱいしています。
