４年ねん前まえに噴火ふんかした鹿児島県かごしまけんの口永良部島くちのえらぶじま　また噴火ふんかの可能性かのうせい

发表于 2019-10-28 更新于 2019-11-30 分类于 JapanEasyNews



鹿児島県かごしまけんの口永良部島くちのえらぶじまで、今月こんげつ１８日にちと２７日にちに大おおきな地震じしんがありました。気象庁きしょうちょうは噴火ふんかがあるかもしれないと考かんがえて、２８日にち、１から５までの数字すうじで噴火ふんかの可能性かのうせいと危険きけんを知しらせる「噴火ふんか警戒けいかい」のレベルを２から３に上あげました。
気象庁きしょうちょうは、火口かこうの周まわりや火口かこうから海岸かいがんまでの場所ばしょでは、噴火ふんかで飛とんでくる石いしや流ながれてくる熱あつい灰はいなどに気きをつけるように言いっています。
口永良部島くちのえらぶじまでは４年ねん前まえに大おおきな噴火ふんかがありました。灰はいなどが海岸かいがんまで流ながれて、「噴火ふんか警戒けいかい」のレベルがいちばん高たかい５になりました。住すんでいた人ひとたちはみんな島しまを出でて避難ひなんしました。

卒業そつぎょうしたあと日本にっぽんで働はたらく外国人がいこくじん留学生りゅうがくせいが今いままでで最もっとも多おおくなる

发表于 2019-10-25 更新于 2019-11-30 分类于 JapanEasyNews



留学りゅうがくして日本にっぽんの大学だいがくや専門せんもん学校がっこうを卒業そつぎょうしたあと、日本にっぽんで働はたらく外国人がいこくじんが増ふえています。
国くにの出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうによると、日本にっぽんに住すむための資格しかくを「留学りゅうがく」から「働はたらくため」に変かえた外国人がいこくじんは、去年きょねん２万まん５９４２人にんいました。今いままでで最もっとも多おおかった２０１７年ねんより３５００人にん以上いじょう増ふえました。
この中なかの９３％は、通訳つうやくやコンピューターのエンジニアなどの仕事しごとをする資格しかくでした。国くにでは、中国ちゅうごくが最もっとも多おおくて１万まん９００人にんぐらい、次つぎはベトナムで５２００人にんぐらい、その次つぎはネパールで２９００人にんぐらいでした。
出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうは「留学生りゅうがくせいが増ふえているので、卒業そつぎょうしたあと日本にっぽんで働はたらく人ひとも増ふえています。ほかの役所やくしょと一緒いっしょに、留学生りゅうがくせいが相談そうだんできる所ところを作つくったり、学校がっこうで説明会せつめいかいを開ひらいたりしていることも理由りゆうだと思おもいます」と話はなしています。

アフリカでたくさんの象ぞうが死しんでいる

发表于 2019-10-25 更新于 2019-11-30 分类于 JapanEasyNews



アフリカの南みなみの地方ちほうのジンバブエとボツワナでは、先月せんげつから、死しんだ象ぞうが１５０頭とう以上いじょう見みつかっています。
ジンバブエの国立公園こくりつこうえんの人ひとは、雨あめがほとんど降ふっていないため、食たべ物ものがなくなったり水みずが足たりなくなったりしたことが原因げんいんだと考かんがえています。国立公園こくりつこうえんの人ひとは「今いま生いきている象ぞうも食たべ物ものがなくて苦くるしんでいます。雨あめが降ふらない季節きせつになったら、今いまよりもっとひどくなると思おもいます」と話はなしています。
ボツワナの政府せいふは、長ながい間あいだ食たべ物ものがなくて死しんだり、草くさがないため土つちを食たべて、土つちの中なかの細菌さいきんが体からだに入はいって死しんだりした可能性かのうせいがあると考かんがえています。
環境かんきょうを守まもる活動かつどうをしているＷＷＦによると、アフリカの南みなみの地方ちほうには象ぞうが約やく２９万まん頭とういますが、象牙ぞうげを取とるために殺ころされて数かずが減へっています。ＷＷＦは、地球ちきゅうの気温きおんが上あがって雨あめが少すくなくなって環境かんきょうが変かわると、象ぞうがもっと少すくなくなるかもしれないと心配しんぱいしています。

この法律ほうりつを広ひろめるために、４００以上いじょうの市しや町まちなどがつくった会かいが２３日にち、東京とうきょうで式しきを開ひらきました。そして、スーパーやレストランなどの３つの会社かいしゃと一緒いっしょに、食たべ物ものを捨すてないようにするための宣言せんげんを発表はっぴょうしました。宣言せんげんでは、家庭かていやレストランなどで食たべ物ものを残のこさないように、みんなで一緒いっしょに協力きょうりょくしようと言いっています。

この会かいの会長かいちょうは「捨すてる食たべ物ものを減へらすためには、みんなが協力きょうりょくすることが大事だいじです」と話はなしています。

関西電力かんさいでんりょくの社長しゃちょうなど２０人にんが、原子力げんしりょく発電所はつでんしょがある福井県ふくいけん高浜町たかはまちょうの助役じょやくだった人ひとからお金かねや品物しなものをもらって、問題もんだいになっています。
ＮＨＫが調しらべると、助役じょやくだった人ひとは、原発げんぱつで仕事しごとをする４つの会社かいしゃで役員やくいんなどをしていたことがわかりました。助役じょやくだった人ひとは、この中なかの３つの会社かいしゃからたくさんのお金かねをもらっていたこともわかりました。
会社かいしゃの１つは、関西電力かんさいでんりょくから頼たのまれて、５年ねんで１４９億おく円えん以上いじょうの仕事しごとをして、売うり上あげを増ふやしていました。原発げんぱつで使つかうたくさんのお金かねが、助役じょやくだった人ひとと関係かんけいがある会社かいしゃに集あつまっていました。
