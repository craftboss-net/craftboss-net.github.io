スマートフォンと若わかい人ひとの「急性内斜視きゅうせいないしゃし」の関係かんけいを調しらべる

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



日本にほん弱視じゃくし斜視しゃし学会がっかいは、目めが内側うちがわに寄よって戻もどらなくなる「急性内斜視きゅうせいないしゃし」とスマートフォンなどの関係かんけいを調しらべるために、眼科がんかの医者いしゃにアンケートをしました。３６９人にんが答こたえました。４２％の医者いしゃは、去年きょねん、「急性内斜視きゅうせいないしゃし」になった若わかい人ひとが病院びょういんに来きたと答こたえました。
この中なかの７７％は、「スマートフォンなどと関係かんけいがあると思おもった」と答こたえていました。学会がっかいは、スマートフォンなどを使つかいすぎることと関係かんけいがあるかもしれないと考かんがえて、これからも調査ちょうさを続つづけると言いっています。
調査ちょうさでは、スマートフォンなどを使つかう時間じかんや、目めからどのくらい離はなして使つかっているか調しらべます。使つかう時間じかんを短みじかくしたら、よくなったかどうかも調しらべます。学会がっかいは、どうしたら「急性内斜視きゅうせいないしゃし」にならないかや、治なおし方かたを２０２２年ねんぐらいまでに発表はっぴょうしたいと言いっています。

日本にっぽんのタンカーが中東ちゅうとうの海うみで攻撃こうげきされた

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



１３日にち、中東ちゅうとうの海うみで２隻せきのタンカーが攻撃こうげきされました。１隻せきは日本にっぽんの会社かいしゃの船ふねでした。日本にっぽんの時間じかんで１３日にちの昼ひるごろ、ホルムズ海峡かいきょうの近ちかくの海うみで攻撃こうげきされたという連絡れんらくが船ふねから会社かいしゃに入はいりました。
会社かいしゃによると、船ふねは２回かい攻撃こうげきされて、弾たまが２発はつ当あたりました。このあと、船ふねに乗のっていた人ひとはみんなボートで逃にげて、別べつの船ふねに助たすけられました。
この船ふねは、燃もえやすいメタノールを２万まん５０００ｔ運はこんでいましたが、火事かじになったという情報じょうほうはありません。
会社かいしゃの社長しゃちょうは「船ふねに乗のっていた人ひとたちと電話でんわで話はなしをしました。みんな無事ぶじだと聞きいて安心あんしんしています」と話はなしています。

法律ほうりつで犬いぬや猫ねこにチップをつけなければならなくなる

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



法律ほうりつが変かわって、ペットの犬いぬや猫ねこに、誰だれが飼かっているかわかる小ちいさなチップをつけなければならなくなりました。犬いぬや猫ねこが捨すてられたとき、誰だれが捨すてたかすぐにわかるようにするためです。地震じしんなどの災害さいがいで、犬いぬや猫ねこが飼かっている人ひとと別々べつべつになったときも役やくに立たちます。
犬いぬや猫ねこの繁殖はんしょくをするブリーダーやペットを売うる店みせなどは、犬いぬや猫ねこの体からだにチップを入いれなければなりません。チップを入いれたら、飼かっている人ひとの住所じゅうしょなどを国くにに知しらせて登録とうろくしなければなりません。いま飼かっている犬いぬや猫ねこには、できるだけチップをつけるようにします。
法律ほうりつでは、犬いぬや猫ねこは生うまれてから５６日にちまで売うったり展示てんじしたりしてはいけないことにしました。特別とくべつに決きめた犬いぬは４９日にちまでにします。
動物どうぶつを虐待ぎゃくたいしたときの罰ばつも今いままでより厳きびしくしました。

法律ほうりつを守まもらない人ひとは、刑務所けいむしょに入はいったり、罰金ばっきんを払はらったりしなければなりません。

世界中せかいじゅうの新あたらしいおもちゃを紹介しょうかいするイベントが東京とうきょうで始はじまりました。小学校しょうがっこうでは来年らいねんからコンピューターのプログラムを教おしえます。このため、今年ことしは遊あそびながらプログラムを勉強べんきょうするおもちゃが人気にんきになっています。
ソニーのグループの会社かいしゃが作つくったおもちゃは、絵本えほんと２４種類しゅるいのカードとロボットを使つかって遊あそびます。絵本えほんはロボットが冒険ぼうけんをする話はなしです。子こどもは絵本えほんの話はなしに合あうように「右みぎに曲まがる」とか「進すすむ」などと書かいたカードを並ならべます。カードの上うえにロボットを置おくと、ロボットがカードのデータを読よみます。そのあとロボットを絵本えほんの上うえに置おくと、データのとおりに動うごいて、プログラムの考かんがえ方かたがわかります。
おもちゃの会社かいしゃの団体だんたいの人ひとは「子こどもの数かずが少すくなくなっていますが、今いまの子こどもに合あうおもちゃをつくって、たくさん遊あそんでもらいたいです」と話はなしていました。
このイベントは１６日にちまでです。

耳みみに障害しょうがいがある人ひとが体からだにつけて揺ゆれや光ひかりで音おとを感かんじる機械きかい

发表于 2019-06-13 更新于 2019-11-30 分类于 JapanEasyNews



耳みみに障害しょうがいがある人ひとも音おとを感かんじることができる新あたらしい機械きかいができました。
この機械きかいは長ながさが６．５ｃｍ、重おもさが２５ｇで、髪かみや耳みみ、服ふくにつけて使つかいます。中なかには音おとを集あつめるマイクが入はいっています。周まわりで音おとがすると機械きかいが小ちいさく揺ゆれて、髪かみなどに音おとのリズムや大おおきさを伝つたえます。音おとと一緒いっしょに光ひかるＬＥＤも付ついていて、揺ゆれと光ひかりの両方りょうほうで音おとを感かんじることができます。音おとの大おおきさは揺ゆれや光ひかりの強つよさが細こまかく変かわることでわかります。
この機械きかいを使つかうと、音おとが聞きこえなくても、コンサートで音楽おんがくのリズムを感かんじたり、スポーツの会場かいじょうで試合しあいを見みている人ひとの大おおきな声こえを感かんじたりできます。
機械きかいを作つくった富士通ふじつうは、耳みみに障害しょうがいがある子こどもたちが通かよう学校がっこうにこの機械きかいをプレゼントします。映画館えいがかんやスポーツの試合しあいの会場かいじょうなどでも貸かすことができるようにしたいと考かんがえています。

「ギラン?バレー症候群しょうこうぐん」は、免疫めんえきが自分じぶんの神経しんけいを攻撃こうげきするため、手てや足あしに力ちからが入はいらなくなる病気びょうきです。呼吸こきゅうができなくなって亡なくなる人ひともいます。
ペルーの政府せいふによると、首都しゅとのリマや北きたにある観光地かんこうちでは、今年ことしになってからギラン?バレー症候群しょうこうぐんだとわかった人ひとが２０６人にんいました。このうち４人にんが亡なくなりました。
ペルーの政府せいふは今月こんげつ８日ようか、この病気びょうきになる人ひとが急きゅうに増ふえているため、健康けんこうに危険きけんがある状況じょうきょうだと言いいました。そして、体からだに力ちからが入はいらなくなったら、すぐに病院びょういんに行いくように言いっています。
ギラン?バレー症候群しょうこうぐんは、蚊かやウイルスなどでうつる病気びょうきと関係かんけいがあるかもしれないと言いわれています。政府せいふは専門家せんもんかと原因げんいんを調しらべています。
ペルーには世界遺産せかいいさんのマチュピチュがあって、日本人にっぽんじんもたくさん観光かんこうに行いきます。ペルーの日本大使館にっぽんたいしかんは、手てや食たべ物ものをきれいに洗あらって、衛生えいせいに気きをつけるように言いっています。
