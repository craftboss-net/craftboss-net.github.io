チケットを買かった値段ねだんより高たかく売うることが法律ほうりつで禁止きんしになる

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



１４日じゅうよっかから新あたらしい法律ほうりつで、スポーツやコンサートのチケットを買かった値段ねだんより高たかく売うることが禁止きんしになりました。ほかの人ひとに売うるためにチケットをもらうことも禁止きんしになりました。
新あたらしい法律ほうりつでは、コンサートの会場かいじょうの近ちかくなどでチケットを高たかく売うることだけではなくて、インターネットで高たかく売うることも禁止きんしになりました。
最近さいきん、人気にんきがあるチケットの販売はんばいが始はじまるとすぐにたくさん買かって、インターネットでとても高たかく売うる人ひとが増ふえています。買かった値段ねだんの１０倍ばい以上いじょうで売うる人ひともいて、問題もんだいになっています。
法律ほうりつを守まもらない人ひとは、刑務所けいむしょに入はいったり、罰金ばっきんを払はらったりしなければなりません。

耳みみに障害しょうがいがある人ひとが体からだにつけて揺ゆれや光ひかりで音おとを感かんじる機械きかい

发表于 2019-06-13 更新于 2019-11-30 分类于 JapanEasyNews



耳みみに障害しょうがいがある人ひとも音おとを感かんじることができる新あたらしい機械きかいができました。
この機械きかいは長ながさが６．５ｃｍ、重おもさが２５ｇで、髪かみや耳みみ、服ふくにつけて使つかいます。中なかには音おとを集あつめるマイクが入はいっています。周まわりで音おとがすると機械きかいが小ちいさく揺ゆれて、髪かみなどに音おとのリズムや大おおきさを伝つたえます。音おとと一緒いっしょに光ひかるＬＥＤも付ついていて、揺ゆれと光ひかりの両方りょうほうで音おとを感かんじることができます。音おとの大おおきさは揺ゆれや光ひかりの強つよさが細こまかく変かわることでわかります。
この機械きかいを使つかうと、音おとが聞きこえなくても、コンサートで音楽おんがくのリズムを感かんじたり、スポーツの会場かいじょうで試合しあいを見みている人ひとの大おおきな声こえを感かんじたりできます。
機械きかいを作つくった富士通ふじつうは、耳みみに障害しょうがいがある子こどもたちが通かよう学校がっこうにこの機械きかいをプレゼントします。映画館えいがかんやスポーツの試合しあいの会場かいじょうなどでも貸かすことができるようにしたいと考かんがえています。

海うみに出でた日本にっぽんのプラスチックのごみはアメリカまで流ながれる

发表于 2019-06-13 更新于 2019-11-30 分类于 JapanEasyNews



買かい物もののときに使つかう袋ふくろやペットボトルなどプラスチックのごみで海うみが汚よごれることが大おおきな問題もんだいになっています。海うみにいる魚さかなや動物どうぶつなどが間違まちがえてプラスチックを食たべてしまうこともあります。
海うみの水みずや風かぜの流ながれの専門家せんもんかの岩崎いわさき慎介しんすけさんは、日本にっぽんから６３００ｔのプラスチックのごみが海うみに出でた場合ばあい、ごみがどのように海うみを流ながれるかコンピューターを使つかって研究けんきゅうしました。たくさんのごみが太平洋たいへいようを流ながれて、アメリカやカナダ、アラスカの海岸かいがんに着つくことがわかりました。フィリピンなど東南とうなんアジアに着つくごみや、日本にっぽんの周まわりの海うみに残のこるごみもあることがわかりました。
岩崎いわさきさんは「日本にっぽんから出でたプラスチックのごみは、遠とおくの国くににも迷惑めいわくになります。世界せかいの国くにが早はやくプラスチックのごみを少すくなくしなければなりません」と言いっています。

「ギラン?バレー症候群しょうこうぐん」は、免疫めんえきが自分じぶんの神経しんけいを攻撃こうげきするため、手てや足あしに力ちからが入はいらなくなる病気びょうきです。呼吸こきゅうができなくなって亡なくなる人ひともいます。
ペルーの政府せいふによると、首都しゅとのリマや北きたにある観光地かんこうちでは、今年ことしになってからギラン?バレー症候群しょうこうぐんだとわかった人ひとが２０６人にんいました。このうち４人にんが亡なくなりました。
ペルーの政府せいふは今月こんげつ８日ようか、この病気びょうきになる人ひとが急きゅうに増ふえているため、健康けんこうに危険きけんがある状況じょうきょうだと言いいました。そして、体からだに力ちからが入はいらなくなったら、すぐに病院びょういんに行いくように言いっています。
ギラン?バレー症候群しょうこうぐんは、蚊かやウイルスなどでうつる病気びょうきと関係かんけいがあるかもしれないと言いわれています。政府せいふは専門家せんもんかと原因げんいんを調しらべています。
ペルーには世界遺産せかいいさんのマチュピチュがあって、日本人にっぽんじんもたくさん観光かんこうに行いきます。ペルーの日本大使館にっぽんたいしかんは、手てや食たべ物ものをきれいに洗あらって、衛生えいせいに気きをつけるように言いっています。

東京福祉大学とうきょうふくしだいがくでは、ベトナム人じんやネパール人じんなどの留学生りゅうがくせいが大学だいがくに来こなくなってどこにいるかわからなくなりました。
国くにが調しらべると、３年ねんで１６１０人にんが行方不明ゆくえふめいになっていました。この中なかの１１１３人にんは「学部研究生がくぶけんきゅうせい」で、大学だいがくに入はいる前まえの準備じゅんびの勉強べんきょうをしているはずでした。
国くには「しっかり勉強べんきょうできる環境かんきょうではないのに、大勢おおぜいの留学生りゅうがくせいを入いれた大学だいがくに問題もんだいがある」と言いいました。そして、東京福祉大学とうきょうふくしだいがくの学部研究生がくぶけんきゅうせいには留学生りゅうがくせいのビザを出ださないことにしました。
このようなことがないように、国くには日本中にほんじゅうの大学だいがくを調しらべることにしました。大学だいがくをやめた人ひとや、やめたあと国くにに帰かえらないで日本にっぽんに残のこっている人ひとの数かずを調しらべます。そして、大学だいがくに留学生りゅうがくせいの勉強べんきょうや生活せいかつをしっかりチェックするように言いいます。よくならない場合ばあい、その大学だいがくには留学生りゅうがくせいが入学にゅうがくできないようにして、大学だいがくの名前なまえも発表はっぴょうします。
