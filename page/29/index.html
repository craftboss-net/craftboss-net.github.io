九州きゅうしゅうや四国しこく　台風たいふうで雨あめがまだたくさん降ふりそう

发表于 2019-08-06 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふう８号ごうは、６日むいかの朝あさ宮崎県みやざきけんに入はいりました。熊本県くまもとけんや佐賀県さがけんなどを通とおって、九州きゅうしゅうの北きたの海うみを進すすんでいます。
気象庁きしょうちょうによると、九州きゅうしゅうや四国しこくではこれからしばらく雨あめがたくさん降ふる所ところもあります。
気象庁きしょうちょうは、今いままでに雨あめがたくさん降ふった所ところでは、山やまが崩くずれたり川かわの水みずがあふれたりすることがあると言いっています。九州きゅうしゅうの北きたの海うみでは６日むいかの夜よるまで強つよい風かぜや高たかい波なみにも気きをつけるように言いっています。

「ＮＥＷＳ　ＷＥＢ　ＥＡＳＹ」の「台風たいふう」や「大雨おおあめ」のページも読よんでください。

イギリスで、ゴルフのとても有名ゆうめいな大会たいかいの「全英ぜんえい女子じょしオープン」がありました。この大会たいかいで、日本にっぽんの２０歳さいの渋野しぶの日向子ひなこ選手せんしゅが優勝ゆうしょうしました。外国がいこくのとても有名ゆうめいなゴルフの大会たいかいで日本人にっぽんじんが優勝ゆうしょうしたのは、１９７７年ねんの樋口ひぐち久子ひさこさんに続つづいて２人ふたり目めです。

戦たたかいが原因げんいんで亡なくなったりけがをしたりした子こどもが多おおくなる

发表于 2019-08-05 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんは、アフリカなど戦たたかいが続つづいている所ところで生活せいかつしている子こどもたちについて調しらべています。
その結果けっか、去年きょねん戦たたかいが原因げんいんで亡なくなったり、けがをして体からだに障害しょうがいが残のこったりした子こどもは、１万まん２０００人にん以上いじょういました。２００５年ねんに調しらべ始はじめてから最もっとも多おおくなりました。
兵士へいしにされて戦たたかいの場所ばしょに連つれて行いかれた子こどもは７０００人にん以上いじょういました。特とくにアフリカのソマリアやナイジェリア、中東ちゅうとうのシリアに多おおくいました。性せい的てきな暴力ぼうりょくを受うけている子こどももいました。アフリカのマリでは、戦たたかいが原因げんいんで８２７の学校がっこうが使つかうことができなくなって、２４万まん人にん以上いじょうの子こどもが勉強べんきょうできなくなりました。
国連こくれんのグテーレス事務総長じむそうちょうは、戦たたかいをしている人ひとたちに、子こどもや市民しみんなどを攻撃こうげきしないように言いっています。そして、子こどもの安全あんぜんを守まもるためには平和へいわがいちばん大切たいせつなので、戦たたかいを終おわりにするように言いっています。

「いくらお金かねがあるか」と聞きく電話でんわ　３か月げつで３万まん件けん以上いじょう

发表于 2019-08-05 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうと江東区こうとうくのマンションで今年ことし２月がつ、８０歳さいの女性じょせいが部屋へやに入はいってきた男おとこたちに殺ころされる事件じけんがありました。女性じょせいには事件じけんの前まえ、お金かねがいくらあるかと聞きく電話でんわがかかってきていました。
この事件じけんのあと警察けいさつは、家いえにお金かねがいくらあるかなどと聞きく電話でんわがどのくらいあるか調しらべました。その結果けっか、今年ことし４月がつから６月がつまでの３か月げつに、３万まん５２８９件けんありました。東京都とうきょうと、埼玉県さいたまけん、大阪府おおさかふ、千葉県ちばけん、神奈川県かながわけんの順番じゅんばんに多おおくなっていて、この５つで６０％以上いじょうになりました。
電話でんわのあとに男おとこなどが家いえに入はいってきてお金かねを取とられた事件じけんは、２年ねん前まえから７件けんありました。
警察けいさつは、知しらない番号ばんごうからの電話でんわに出でないように留守番るすばん電話でんわを使つかったり、電話でんわの声こえを録音ろくおんしたりしてほしいと言いっています。

手紙てがみの宛先あてさきの書かき方かたがわからない子こどもが増ふえている

发表于 2019-08-02 更新于 2019-11-30 分类于 JapanEasyNews



文部科学省もんぶかがくしょうの今年ことしの「全国学力ぜんこくがくりょくテスト」では、中学校ちゅうがっこう３年生ねんせいの国語こくごで、手紙てがみの宛先あてさきの書かき方かたの問題もんだいが出でました。
正ただしい書かき方かたは、封筒ふうとうの右みぎに住所じゅうしょを書かいて、真まん中なかに「様さま」などをつけた相手あいての名前なまえを書かきます。しかし、正ただしく答こたえた中学生ちゅうがくせいは５７％だけでした。間違まちがった答こたえでは、相手あいての名前なまえを右みぎに書かいたり、住所じゅうしょと一緒いっしょにメールのアドレスを書かいたりしていました。７年ねん前まえのテストで正ただしく答こたえた中学生ちゅうがくせいは７４％で、今年ことしは１７ポイント低ひくくなりました。
インターネットのＳＮＳを使つかうことが増ふえて、子こどもたちが手紙てがみを書かく機会きかいが少すくなくなっているようです。
日本語にほんごを研究けんきゅうしている大学だいがくの先生せんせいは「手紙てがみだから伝つたえることができる気持きもちもあると思おもいます。子こどもたちには手紙てがみのいいところを大切たいせつにしてほしいです」と話はなしています。

防衛省ぼうえいしょう　中止ちゅうししていたＦ３５の訓練くんれんを始はじめる

发表于 2019-08-02 更新于 2019-11-30 分类于 JapanEasyNews



Ｆ３５は、アメリカなどが作つくった攻撃こうげきするための新あたらしい飛行機ひこうきです。今年ことし４月がつ、青森県あおもりけんの三沢基地みさわきちのＦ３５が訓練くんれんをしているときに海うみに落おちる事故じこがありました。このあと防衛省ぼうえいしょうはＦ３５の訓練くんれんを中止ちゅうししました。
防衛省ぼうえいしょうは８月がつ１日ついたち、パイロットの教育きょういくをしっかり行おこなって、飛行機ひこうきに問題もんだいがないかしっかり調しらべて、安全あんぜんに飛とぶことができるようにしたので、また訓練くんれんを始はじめると言いいました。
１日ついたちの午後ごご２時じ４０分ぷんごろ、２機きのＦ３５が三沢基地みさわきちから飛とんで、訓練くんれんを始はじめました。４月がつの事故じこは、夜よるに訓練くんれんをしているときに起おこりました。このため防衛省ぼうえいしょうは、夜よるの間あいだはしばらく飛とばないと言いっています。
防衛省ぼうえいしょうは、Ｆ３５を日本にっぽんの空そらを守まもる飛行機ひこうきの中心ちゅうしんにしようと考かんがえていて、１０年ねんくらいの間あいだに１４７機き買かう計画けいかくです。岩屋いわや防衛ぼうえい大臣だいじんは「計画けいかくが遅おくれないようにしたいです」と話はなしました。

日本にっぽん政府せいふは先月せんげつ、半導体はんどうたいなどを作つくるために必要ひつような３つの材料ざいりょうを韓国かんこくに輸出ゆしゅつするときの決きまりを厳きびしくしました。
