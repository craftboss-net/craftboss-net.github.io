京都きょうとアニメーションの新あたらしい映画えいがが始はじまる

发表于 2019-09-09 更新于 2019-11-30 分类于 JapanEasyNews



京都きょうとアニメーションは、７月がつに建物たてものに火ひをつけられて、仕事しごとをしていた大勢おおぜいの人ひとが亡なくなりました。
６日むいか、この会社かいしゃのアニメで火事かじの前まえの日ひにできていた「ヴァイオレット?エヴァーガーデン外伝がいでん」が映画館えいがかんで始はじまりました。
火事かじで亡なくなったりけがをしたりした人ひとも、この映画えいがを作つくっていました。今いままで京都きょうとアニメーションでは、映画えいがの最後さいごに名前なまえを出だすのは作つくった経験けいけんが１年ねん以上いじょうある人ひとたちだけでした。しかし、この映画えいがの監督かんとくは作つくった人ひとみんなの名前なまえを出だすと決きめて、紹介しょうかいしました。
映画えいがを見みた大学生だいがくせいは「すてきな絵えや細こまかい表現ひょうげんで、作つくった人ひとたちの気持きもちを感かんじました」と話はなしていました。
映画えいがは２６日にちまでです。

横浜よこはまで電車でんしゃとトラックがぶつかる　１人ひとり亡なくなって３４人にんけが

发表于 2019-09-06 更新于 2019-11-30 分类于 JapanEasyNews



５日いつか午前ごぜん１１時じ４０分ぷんごろ、横浜市よこはましにある京急線けいきゅうせんの踏切ふみきりで、電車でんしゃが大おおきなトラックとぶつかって脱線だっせんしました。トラックは大おおきく壊こわれて燃もえました。
警察けいさつによると、トラックを運転うんてんしていた６７歳さいの男性だんせいが亡なくなりました。電車でんしゃに乗のっていた３４人にんがけがをしました。
事故じこがあった踏切ふみきりは神奈川新町駅かながわしんまちえきのすぐ隣となりにあります。ぶつかった電車でんしゃは止とまる駅えきが少すくない電車でんしゃで、いつもはこの踏切ふみきりを時速じそく１２０ｋｍで走はしります。トラックとぶつかったときも速はやいスピードで走はしっていたと考かんがえられています。
警察けいさつなどによると、大おおきなトラックは右みぎに曲まがって踏切ふみきりに入はいりましたが、動うごくことができなくなって、電車でんしゃとぶつかりました。
事故じこのあと、京急線けいきゅうせんは京急川崎駅けいきゅうかわさきえきと上大岡駅かみおおおかえきの間あいだで運転うんてんできなくなりました。

車くるまを運転うんてんしているとき、ほかの車くるまから危険きけんな運転うんてんをされて、トラブルになることが問題もんだいになっています。
このため、保険ほけんの会社かいしゃの東京海上日動とうきょうかいじょうにちどうは、危険きけんな運転うんてんをされたときのための新あたらしい保険ほけんを来月らいげつから始はじめることにしました。この保険ほけんでは、会社かいしゃが有料ゆうりょうで貸かすドライブレコーダーを使つかいます。危険きけんな運転うんてんをされたとき、ドライブレコーダーのボタンを押おすと、警備けいびの会社かいしゃの人ひとと話はなすことができます。ドライブレコーダーで撮とったビデオを警備けいびの会社かいしゃに送おくることもできます。警備けいびの会社かいしゃは、危険きけんだと思おもったら警察けいさつに連絡れんらくします。
別べつの保険ほけんの会社かいしゃも、撮とったビデオを家族かぞくのスマートフォンに送おくるドライブレコーダーを貸かしています。家族かぞくから警察けいさつにすぐ連絡れんらくできるようにするためです。新あたらしい保険ほけんで、危険きけんな運転うんてんでの被害ひがいを少すくなくできるかどうか注目ちゅうもくされています。
