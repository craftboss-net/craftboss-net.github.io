横断歩道おうだんほどうの信号しんごう「目めに障害しょうがいがある人ひとのために音おとを出だして」

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうで去年きょねん１２月がつ、目めに障害しょうがいがある男性だんせいが、朝あさ早はやくに横断歩道おうだんほどうで車くるまにはねられて亡なくなりました。横断歩道おうだんほどうには、信号しんごうが青あおになったことを音おとで知しらせる信号しんごうがありました。しかし、夜よるから朝あさは静しずかにするために音おとを出だしていませんでした。
音おとが出でる信号しんごうは日本にっぽんに２万まん４０００ぐらいあります。しかし、この中なかの多おおくは、朝あさや夜よるは音おとを出ださないようにしています。
障害しょうがいがある人ひとたちの団体だんたいによると、目めに障害しょうがいがある人ひとの中なかには、人ひとや車くるまが少すくない朝あさ早はやくや夜よるに仕事しごとに行いったり帰かえったりする人ひともいます。団体だんたいは「横断歩道おうだんほどうの近ちかくに住すんでいる人ひとと相談そうだんして、朝あさや夜よるでも目めに障害しょうがいがある人ひとが渡わたるときは音おとが出でるようにしてほしいです」と言いっています。
警察けいさつは「近ちかくに住すんでいる人ひとの意見いけんと障害しょうがいがある人ひとの安全あんぜんの両方りょうほうを考かんがえて、どうしたらいいか決きめたいと思おもいます」と話はなしています。

パナソニックがファーウェイに部品ぶひんを売うることをやめる

发表于 2019-05-23 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのパナソニックは、中国ちゅうごくのファーウェイがスマートフォンに使つかう部品ぶひんなどを作つくっています。アメリカの政府せいふは、アメリカの会社かいしゃがファーウェイに部品ぶひんなどを売うることを禁止きんししようとしています。
パナソニックの部品ぶひんなどの中なかには、アメリカの会社かいしゃの部品ぶひんや技術ぎじゅつを使つかって作つくった物ものがあって、ファーウェイに売うることができなくなるかもしれません。このため、パナソニックはファーウェイに部品ぶひんなどを売うることをやめると決きめました。
ファーウェイによると、今いままでにファーウェイに部品ぶひんなどを売うった日本にっぽんの会社かいしゃは１００以上いじょうあります。パナソニックが売うることをやめるため、ほかの会社かいしゃもやめるかもしれません。
日本にっぽんの携帯電話けいたいでんわの会社かいしゃのＫＤＤＩ、ソフトバンク、楽天らくてんモバイルなどは、ファーウェイの新あたらしいスマートフォンを売うり始はじめる日ひを予定よていより遅おそくすることにしました。ＮＴＴドコモは、ファーウェイの新あたらしいスマートフォンの予約よやくの受うけ付つけをやめると言いいました。

发表于 2019-05-22 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんで働はたらく外国人がいこくじんを増ふやすために、４月がつから１４の仕事しごとについて「特定とくてい技能ぎのう」という新あたらしいビザの制度せいどが始はじまりました。このビザは、日本語にほんごの試験しけんと、仕事しごとに必要ひつような技術ぎじゅつをチェックする技能ぎのう試験しけんに合格ごうかくした外国人がいこくじんが取とることができます。
５月がつ２１日にちには新あたらしい制度せいどが始はじまってから日本にっぽんで初はじめて、技能ぎのう試験しけんの結果けっかの発表はっぴょうがありました。レストランなどで働はたらきたい人ひとが４月がつに受うけた試験しけんの結果けっかで、試験しけんを受うけた４６０人にんのうち３４７人にんが合格ごうかくしました。
３月がつに専門せんもん学校がっこうを卒業そつぎょうしたベトナム人じんの男性だんせいは、アルバイトをしていた弁当べんとうの工場こうじょうで仕事しごとを続つづけるために試験しけんを受うけました。男性だんせいは「合格ごうかくしてよかったです。この工場こうじょうで長ながく働はたらきたいと思おもっています」と話はなしていました。
合格ごうかくした人ひとは、日本語にほんごの試験しけんにも合格ごうかくすると、早はやい場合ばあい７月がつごろから新あたらしいビザで働はたらくことになります。

日本にっぽんは、プラスチックのごみを中国ちゅうごくなどに輸出ゆしゅつしていました。しかし、２年ねん前まえに中国ちゅうごくが環境かんきょうに悪わるいという理由りゆうで輸入ゆにゅうをやめました。このため、日本にっぽんは国くにの中なかでごみを片かたづけなければならなくなりました。
環境省かんきょうしょうは、片かたづけることができなくて残のこっているプラスチックのごみについて、調査ちょうさをしました。ごみを片かたづける１８３の会社かいしゃと、１２２の都道府県とどうふけんや都市としが答こたえました。

残のこっているごみが国くにが決きめた量りょうより多おおい会社かいしゃがあるという都道府県とどうふけんなどは１５ありました。去年きょねんの７月がつの３倍ばいです。
