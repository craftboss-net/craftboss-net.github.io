がんが見みつかってから治療ちりょうで５年ねん以上いじょう生いきている人ひとは６６％

发表于 2019-08-09 更新于 2019-11-30 分类于 JapanEasyNews



国立こくりつがん研究けんきゅうセンターは、２０１０年ねんまでの２年ねんの間あいだに２７７の病院びょういんで「がん」の治療ちりょうをした５７万まん人にんについて調しらべました。
治療ちりょうをした人ひとの中なかで、がんが見みつかってから５年ねん以上いじょう生いきている人ひとは６６％いました。この割合わりあいで、どのくらいの人ひとを治療ちりょうで助たすけることができたかがわかります。
割合わりあいがいちばん高たかかったのは、男性だんせいだけにある前立腺ぜんりつせんのがんで９９％でした。次つぎに高たかかった乳にゅうがんは９３％でした。女性じょせいだけのがんの子宮体しきゅうたいがんは８２％、子宮頸しきゅうけいがんは７５％でした。日本にっぽんで多おおい大腸だいちょうがんは７３％、胃いがんは７２％、ぼうこうがんは７０％でした。
５年ねん以上いじょう生いきている人ひとの割合わりあいがいちばん低ひくかったのは、すい臓ぞうがんで１０％でした。次つぎに低ひくかった肝臓かんぞうがんは４０％、肺はいがんは４１％、食道しょくどうがんは４４％でした。

８日ようかの夜よる、東京とうきょうで最後さいごの公演こうえんがありました。吉田よしださんは２０１０年ねんにロイヤルバレエ団だんをやめたときにロンドンで踊おどった「シンデレラ」を踊おどりました。会場かいじょうには、吉田よしださんの最後さいごの踊おどりを見みるためにたくさんの人ひとが集あつまりました。最後さいごの拍手はくしゅは１５分ふん以上いじょう続つづきました。中学生ちゅうがくせいの女おんなの子こは「吉田よしださんのようになりたいと思おもって、バレエの練習れんしゅうを頑張がんばってきました。本当ほんとうに感動かんどうしました」と泣ないて話はなしていました。

羽田空港はねだくうこう　来年らいねん３月がつから新あたらしいルートができて国際線こくさいせんが増ふえる

发表于 2019-08-09 更新于 2019-11-30 分类于 JapanEasyNews



国土交通省こくどこうつうしょうの石井いしい大臣だいじんは８日ようか、飛行機ひこうきが上うえを飛とぶ東京都とうきょうとの区くなどに理解りかいしてもらったので、来年らいねんの３月がつ２９日にちから羽田空港はねだくうこうの新あたらしいルートを使つかうと言いいました。新あたらしいルートでは飛行機ひこうきが新宿しんじゅくや渋谷しぶやの上うえを飛とびます。
新宿しんじゅくで高たかさ９００ｍ以上いじょう、渋谷しぶやで７００ｍ以上いじょう、品川しながわで４５０ｍ以上いじょうのところを飛とびます。風かぜが南みなみから吹ふいているときに羽田空港はねだくうこうに到着とうちゃくする国際線こくさいせんの飛行機ひこうきが使つかいます。このルートを使つかうのは、午後ごご３時じから午後ごご７時じの間あいだになりそうです。
国くには新あたらしいルートを使つかって、国際線こくさいせんの飛行機ひこうきが羽田空港はねだくうこうに到着とうちゃくしたり出発しゅっぱつしたりする回数かいすうを１年ねんに６万まん回かいから９万まん９千せん回かいに増ふやす計画けいかくです。
新あたらしいルートの下したに住すんでいる人ひとたちは、うるさくなったり飛行機ひこうきから物ものが落おちたりすることを心配しんぱいしています。石井いしい大臣だいじんは「音おとや安全あんぜんに気きをつけます。情報じょうほうもしっかり伝つたえて、たくさんの人ひとに理解りかいしてもらうようにします」と話はなしています。

長崎ながさきに原爆げんばくが落おとされてから７４年ねん　平和へいわを祈いのる式しき

发表于 2019-08-09 更新于 2019-11-30 分类于 JapanEasyNews



９日ここのか、長崎ながさきに原爆げんばくが落おとされてから７４年ねんになりました。原爆げんばくで亡なくなった人ひとは１８万まん２６０１人にんです。この中なかで、去年きょねんからの１年ねんに亡なくなった人ひとは３４０２人にんです。
長崎市ながさきしが行おこなった平和へいわを祈いのる式しきには、５９００人にんが出席しゅっせきしました。原爆げんばくが落おとされた午前ごぜん１１時じ２分ふんになると、みんなで静しずかに祈いのりました。
長崎市ながさきしの田上たうえ市長しちょうは、核兵器かくへいきを作つくったり使つかったりすることを禁止きんしすると国連こくれんで決きめた条約じょうやくに入はいるように、日本にっぽん政府せいふに言いいました。
安倍あべ総理大臣そうりだいじんは「核兵器かくへいきをなくすために、核兵器かくへいきを持もっている国くにと持もっていない国くにの両方りょうほうと話はなし合あいをします」と言いいましたが、国連こくれんの条約じょうやくについては何なにも話はなしませんでした。
長崎市ながさきしの７５歳さいの女性じょせいは「私わたしは１歳さい半はんだったので、何なにも覚おぼえていません。母ははは原爆げんばくについて何なにも話はなさないで亡なくなりました。父ちちは原爆げんばくで亡なくなったので顔かおを見みることもできませんでした。戦争せんそうのない平和へいわな世界せかいになってほしいです」と話はなしていました。

浅間山あさまやまで小ちいさな噴火ふんかがあった

发表于 2019-08-08 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうによると、７日なのかの午後ごご１０時じごろ、長野県ながのけんと群馬県ぐんまけんの間あいだにある浅間山あさまやまの頂上ちょうじょうにある火口かこうで、小ちいさな噴火ふんかがありました。噴火ふんかの煙けむりが山やまの上うえから１８００ｍ以上いじょうまで上あがって、大おおきな石いしが２００ｍぐらい飛とびました。噴火ふんかは２０分ぷんぐらい続つづいて、煙けむりは北きたに流ながれました。
噴火ふんかがまたあるかもしれないため、気象庁きしょうちょうは、１から５まである「噴火ふんか警戒けいかい」のレベルを３に上あげて、浅間山あさまやまに登のぼることを禁止きんししました。
気象庁きしょうちょうは、火口かこうから４ｋｍぐらいの所ところまでは、大おおきな石いしが飛とんできたりするかもしれないと言いっています。もっと遠とおくても、風かぜで小ちいさな石いしや灰はいが飛とんでくるかもしれないため、気きをつけるように言いっています。
浅間山あさまやまは２００９年ねんにも「噴火ふんか警戒けいかい」のレベルが３に上あがりました。
