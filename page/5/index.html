学校がっこうでの子こどものいじめ　今いままででいちばん多おおい５４万まん件けん

发表于 2019-10-21 更新于 2019-11-30 分类于 JapanEasyNews



文部科学省もんぶかがくしょうは毎年まいとし、学校がっこうでいじめがどのくらいあったか調しらべています。１７日にち、今年ことしの３月がつまでの１年ねんに５４万まん３９３３件けんあったと発表はっぴょうしました。前まえの年としより１３万まん件けんぐらい増ふえて、今いままででいちばん多おおくなりました。
いじめを知しった学校がっこうがいじめた子こどもに何なにをしたかという質問しつもんには、親おやなどに連絡れんらくしたという答こたえがいちばん多おおくて４５．６％でした。次つぎに多おおかったのは、いじめられた子こどもやその親おやなどに謝あやまるように言いったという答こたえで４３．４％でした。
専門家せんもんかは「小ちいさないじめが見みつかるようになってよかったです。学校がっこうは、いじめが増ふえて忙いそがしくても、急いそいで解決かいけつしないで、よく考かんがえてほしいです」と話はなしています。

ＮＨＫの調査ちょうさ　７０％ぐらいの人ひとが「皇室こうしつに親したしみがある」

发表于 2019-10-21 更新于 2019-11-30 分类于 JapanEasyNews



今年ことし５月がつ１日ついたちに天皇てんのう陛下へいかが即位そくいして、時代じだいが「平成へいせい」から「令和れいわ」に変かわりました。今月こんげつ２２日にちに即位そくいを知しらせる儀式ぎしきがあります。
ＮＨＫは先月せんげつ２８日にちと２９日にち、日本にっぽんの１８歳さい以上いじょうの人ひとに、皇室こうしつについての調査ちょうさをしました。１５３９人にんが質問しつもんに答こたえました。
今いまの皇室こうしつに親したしみがあるかという質問しつもんに、「親したしみがある」と答こたえた人ひとは７０％以上いじょうになりました。年齢ねんれいが高たかくなると、「親したしみがある」と答こたえた人ひとが多おおくなっています。
親したしみがあると答こたえた人ひとが高たかい割合わりあいだったことについて、専門家せんもんかは「平成へいせいの天皇てんのう陛下へいかと皇后こうごうさまが災害さいがいで苦くるしんでいる人ひとのことをいつも考かんがえていたことなどを知しって、親したしみを持もった人ひとが多おおかったのだと思おもいます」と話はなしています。

发表于 2019-10-21 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふう２０号ごうは、四国しこくの南みなみの太平洋たいへいようを北東ほくとうに進すすんでいて、２２日にちまでには低気圧ていきあつになりそうです。この低気圧ていきあつが近ちかくに来くると、強つよい風かぜが吹ふいて、波なみが高たかくなりそうです。雷かみなりが起おこったり、雨あめがたくさん降ふったりする所ところもありそうです。
１０日とおかぐらい前まえに来きた台風たいふう１９号ごうで被害ひがいがあって、堤防ていぼうが壊こわれて川かわの水みずがあふれたりした所ところでは、特とくに気きをつけてください。気象庁きしょうちょうは、いつもなら問題もんだいがない雨あめの量りょうでも、川かわの水みずがあふれたりするかもしれないと言いっています。

台風たいふうの被害ひがいがほとんどない長野県ながのけんの旅館りょかんでキャンセルが続つづく

发表于 2019-10-21 更新于 2019-11-30 分类于 JapanEasyNews



長野県ながのけん上田市うえだしの別所温泉べっしょおんせんには、今いまの季節きせつ、紅葉こうようや秋あきにおいしいマツタケの料理りょうりなどを楽たのしむために、たくさんの人ひとが来きます。
台風たいふう１９号ごうで多おおくの場所ばしょで大おおきな被害ひがいが出でましたが、別所温泉べっしょおんせんの旅館りょかんはほとんど被害ひがいがありませんでした。しかし「上松うえまつや」という旅館りょかんでは、１１日にちから１８日にちまでに５２２人にんの予約よやくのキャンセルがありました。旅館りょかんによると、雨あめが降ふると心配しんぱいなのでキャンセルすると言いう人にんがいました。
旅館りょかんでは、いいにおいを楽たのしみながら食たべるマツタケの料理りょうりを出だしています。キャンセルが続つづいたため、用意よういしていたマツタケを冷凍れいとうしたりしています。
旅館りょかんの人ひとは「近ちかくでも被害ひがいはなくて、旅館りょかんはいつもどおりお客きゃくさんを迎むかえています。心配しんぱいしないで来きて、楽たのしんでほしいです」と話はなしていました。

長野県ながのけんの病院びょういん　専門せんもんの医者いしゃが山やまに登のぼる前まえの健康けんこうをチェック

发表于 2019-10-18 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねん、山やまに登のぼっているときに、けがをしたり、亡なくなったりした人ひとは３１２９人にんいました。山やまの事故じこが増ふえているため、長野県ながのけん松本市まつもとしにある松本まつもと協立きょうりつ病院びょういんでは専門せんもんの医者いしゃが、山やまに登のぼる前まえの健康けんこうのチェックをしています。
自転車じてんしゃの形かたちの機械きかいを使つかって、山やまに登のぼっているときと同おなじような体からだの状態じょうたいにします。そして、心臓しんぞうが動うごく回数かいすうや血圧けつあつを測はかって、無理むりをしないで歩あるくことができる速はやさを調しらべます。心臓しんぞうや肺はいがしっかり働はたらいているかも調しらべて、まだ見みつかっていない病気びょうきがないかチェックします。
山やまの上うえの空気くうきは酸素さんそが少すくなかったり、気温きおんが大おおきく変かわったりします。知しらない間あいだに体からだがとても疲つかれて、心臓しんぞうなどの病気びょうきになる危険きけんが高たかくなります。病院びょういんの医者いしゃは「血圧けつあつが高たかい人ひとや糖尿病とうにょうびょうの人ひとなどは、大丈夫だいじょうぶだと思おもっていても、山やまでは大おおきな病気びょうきになることがあります。山やまに登のぼる前まえにしっかり体からだをチェックして、安全あんぜんに登のぼってほしいです」と話はなしています。

发表于 2019-10-18 更新于 2019-11-30 分类于 JapanEasyNews



イギリスのＢＢＣは毎年まいとし、世界せかいの人ひとの心こころを動うごかした女性じょせいを１００人にん選えらんで発表はっぴょうしています。今年ことしは日本にっぽんから２人ふたりを選えらびました。
１人ひとりは、女優じょゆうの石川いしかわ優実ゆみさんです。石川いしかわさんは、女性じょせいが仕事しごとのときなどに、嫌いやだと思おもってもかかとが高たかい靴くつを履はかなければならないことに反対はんたいしました。「靴くつ」と、痛いたいという意味いみの「苦痛くつう」の発音はつおんが似にているので、「＃ＫｕＴｏｏ」というハッシュタグをつけて、インターネットで多おおくの人ひとに広ひろめました。
ＢＢＣによると、石川いしかわさんは「毎日まいにちの生活せいかつの中なかにある女性じょせいへの差別さべつに気きがついてほしい」と言いっています。
もう１人ひとりは、立命館大学りつめいかんだいがく４年生ねんせいの今こん日和ひよりさんです。今こんさんは、女性じょせいの相撲すもうの大会たいかいにたくさん出でて活躍かつやくしています。相撲すもうの世界せかいには、女性じょせいを入いれないという伝統でんとうがあります。伝統でんとうの壁かべを壊こわそうとする今こんさんの活動かつどうは、映画えいがにもなりました。
ＢＢＣによると、今こんさんは「世界せかいの子こどもたちに相撲すもうを楽たのしんでもらって、オリンピックの競技きょうぎにしたいです」と言いっています。

花粉かふんを出ださない杉すぎを鳥取県とっとりけんの林業試験場りんぎょうしけんじょうが作つくった

发表于 2019-10-17 更新于 2019-11-30 分类于 JapanEasyNews



鳥取県とっとりけんの林業試験場りんぎょうしけんじょうは１５日にち、花粉かふんを出ださない杉すぎの木きができたと発表はっぴょうしました。杉すぎなどの花粉かふんを吸すうとくしゃみが出でたり目めがかゆくなったりする「花粉症かふんしょう」の人ひとを少すくなくしたいと考かんがえました。
この杉すぎは、花粉かふんを出ださない石川県いしかわけんの杉すぎと、虫むしに強つよい鳥取県とっとりけんの杉すぎから林業試験場りんぎょうしけんじょうが作つくりました。
林業試験場りんぎょうしけんじょうでは、花粉かふんを出ださない杉すぎの木きを今いままでに８５本ほん育そだてました。５年ねん後ごには１年ねんに３０００本ぼんを育そだてたいと言いっています。
林業試験場りんぎょうしけんじょうの人ひとは「花粉症かふんしょうの人ひとも少すくなくなるし、いい木材もくざいにもなるので、鳥取県とっとりけんにたくさん植うえていきたいです」と話はなしていました。
