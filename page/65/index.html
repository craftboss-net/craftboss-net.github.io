パリのノートルダム大聖堂だいせいどうで火事かじ　高たかい塔とうが燃もえる

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



フランスのパリで１５日にち午後ごご７時じごろ、ノートルダム大聖堂だいせいどうが火事かじになりました。建物たてものの真まん中なかの部分ぶぶんがひどく燃もえて、高たかさ９０ｍの細ほそい塔とうや、建物たてものの屋根やねの半分はんぶん以上いじょうが落おちました。１０時間じかんぐらいが過すぎたころ、火ひは大体だいたい消きえました。
火事かじになったとき大聖堂だいせいどうは閉しまっていたため、観光かんこうに来きた人ひとは中なかにいなかったようです。
大聖堂だいせいどうでは、去年きょねん４月がつから建物たてものを直なおす工事こうじをしていました。消防しょうぼうは、大聖堂だいせいどうの屋根やねの裏うらから火ひが出でたと考かんがえていますが、原因げんいんはまだわかっていません。
近ちかくの店みせにいた女性じょせいは「歴史れきしがある建物たてものなので、私わたしたちはショックを受うけています」と話はなしていました。
ノートルダム大聖堂だいせいどうには、１年ねんに１２００万まん人にんが世界中せかいじゅうから観光かんこうに来きます。１９９１年ねんにはユネスコの世界遺産せかいいさんになりました。

宮城県みやぎけん柴田町しばたまち　花見はなみに来きた外国人がいこくじんに子こどもたちが英語えいごで案内あんない

发表于 2019-04-16 更新于 2019-11-30 分类于 JapanEasyNews



宮城県みやぎけんの柴田町しばたまちには、たくさんの桜さくらの木きが並ならんでいる「一目ひとめ千本桜せんぼんざくら」という有名ゆうめいな場所ばしょがあります。毎年まいとし２５万まん人にんが花見はなみに来きて、最近さいきんはタイや香港ほんこんなどから来くる人ひとも増ふえています。
１４日じゅうよっか、柴田町しばたまちの小学生しょうがくせいや中学生ちゅうがくせい３８人にんが、駅えきの前まえなど６つの場所ばしょで観光かんこうに来きた外国人がいこくじんを迎むかえました。
この子こどもたちは、学校がっこうで授業じゅぎょうが終おわってから英語えいごの会話かいわを勉強べんきょうしています。子こどもたちは笑顔えがおで、英語えいごや中国語ちゅうごくごで書かいてある地図ちずを渡わたしたり、英語えいごで「写真しゃしんを撮とりましょうか」と話はなしたりして案内あんないしました。
案内あんないした女おんなの子こは「英語えいごを習ならっていますが、外国がいこくの人ひとが何なにを話はなしているかよくわかりませんでした。これからも外国がいこくの人ひとに親切しんせつにして、柴田町しばたまちのきれいな桜さくらを知しってもらいたいです」と話はなしていました。

ＮＡＳＡ「宇宙うちゅうに１年ねんいても健康けんこうに問題もんだいはない」

发表于 2019-04-15 更新于 2019-11-30 分类于 JapanEasyNews



宇宙飛行士うちゅうひこうしのスコット?ケリーさんは２０１５年ねんの３月がつから１年ねんぐらい、国際こくさい宇宙うちゅうステーションにいました。ケリーさんは宇宙うちゅうにいる間あいだに、地球ちきゅうで５０年ねんの間あいだに受うけるより多おおくの放射線ほうしゃせんを受うけて、重力じゅうりょくのない所ところに長ながい間あいだいました。
ＮＡＳＡなどのグループは、ケリーさんの体からだが変かわったかどうか、双子ふたごの兄弟きょうだいと比くらべました。その結果けっか、年としをとると短みじかくなる染色体せんしょくたいの「テロメア」という部分ぶぶんが、宇宙うちゅうでは短みじかくなりにくいことがわかりました。免疫めんえきやＤＮＡを直なおす一部いちぶの遺伝子いでんしも変かわっていました。
地球ちきゅうに帰かえってから６か月げつ過すぎると、テロメアの長ながさは普通ふつうに戻もどりましたが、一部いちぶの遺伝子いでんしは戻もどりませんでした。
グループは「ケリーさんの体からだに大おおきな問題もんだいはありませんでした。宇宙うちゅうに１年ねんぐらいいても健康けんこうに問題もんだいはありません」と言いっています。

发表于 2019-04-15 更新于 2019-11-30 分类于 JapanEasyNews



今年ことしは４月がつ２７日にちから５月がつ６日むいかまで休やすみが１０日とおか間かん続つづきます。ＪＲと高速道路こうそくどうろの会社かいしゃは、この休やすみに鉄道てつどうと高速道路こうそくどうろがどのくらい混こむか調しらべて発表はっぴょうしました。
ＪＲによると、新幹線しんかんせんの下くだりは４月がつ２７日にち、上のぼりは５月がつ５日いつかがいちばん混こみそうです。４月がつ１１日にちに調しらべると、新幹線しんかんせんなどは去年きょねんより６１％多おおい約やく４３６万まん席せきが予約よやくされていました。

高速道路こうそくどうろの会社かいしゃは「出でかけるときはどのくらい混こんでいるか調しらべて、出発しゅっぱつの時間じかんや利用りようする道みちを考かんがえてください」と話はなしています。

バングラデシュの２５歳さいの男性だんせいは「日本にっぽんで仕事しごとをしたいので留学りゅうがくしました。試験しけんに合格ごうかくしたら、ホテルのフロントの仕事しごとなどをやってみたいです」と話はなしていました。
１４日じゅうよっかの試験しけんには７６１人にんが申もうし込こみました。５月がつ２５日にちに結果けっかの発表はっぴょうがあります。

コンビニの会社かいしゃ　「店みせをあまり増ふやさないようにする」

发表于 2019-04-12 更新于 2019-11-30 分类于 JapanEasyNews



コンビニの店みせは日本にっぽんに５万まん５０００あります。競争きょうそうが激はげしくなって、利益りえきが少すくなくなっている店みせが増ふえています。働はたらく人ひとが足たりなくて、１日いちにちに２４時間じかん店みせを開あけていることが難むずかしいと言いう店みせもあります。
このためコンビニの会社かいしゃは今いままでの計画けいかくを変かえて、あまり店みせを増ふやさないことにすると言いっています。
ローソンは１年ねんに７００ぐらいの店みせを新あたらしく作つくって、７００ぐらいの店みせを閉しめる計画けいかくです。このため、店みせの数かずは増ふえなくなりそうです。
セブン－イレブンは、新あたらしく作つくる店みせを前まえの年としより４０％減へらして８５０にします。そして、７５０の店みせを閉しめる計画けいかくです。
ファミリーマートも、店みせを増ふやすのは１００だけにする計画けいかくです。
３つの会社かいしゃは、今いまある店みせの利益りえきを増ふやすことや、新あたらしい機械きかいを店みせに入いれて仕事しごとを少すくなくすることを考かんがえると言いっています。

パレスチナ　１０年ねん前まえに壊こわされたサッカー場じょうを日本にっぽんが直なおす

发表于 2019-04-12 更新于 2019-11-30 分类于 JapanEasyNews



パレスチナのガザ地区ちくにあるサッカー場じょうは１０年ねんぐらい前まえ、飛行機ひこうきで攻撃こうげきされて壊こわされました。ガザ地区ちくを支配しはいしているハマスというイスラム教きょうのグループとイスラエルの軍ぐんが戦たたかっていたためです。
日本にっぽんはパレスチナの人ひとたちに頼たのまれて、去年きょねんからこのサッカー場じょうを直なおす工事こうじをしていました。１０日とおか、工事こうじが終おわったことをお祝いわいする式しきがありました。工事こうじには日本にっぽんのお金かねで１億おく円えんぐらいかかりました。
式しきのあと、ガザ地区ちくとヨルダン川がわ西岸せいがんのチームがサッカーの試合しあいを行おこないました。ガザ地区ちくは２００７年ねんからハマスが支配しはいしていて、ヨルダン川がわ西岸せいがんの人ひとたちと交流こうりゅうすることができなくなっていました。いつもは交流こうりゅうできない２つのチームが試合しあいをしたため、大勢おおぜいの人ひとが見みにきました。
パレスチナにある日本にっぽん政府せいふの事務所じむしょの人ひとは「スポーツでパレスチナを１つにすることができるように、日本にっぽんはこれからも協力きょうりょくします」と話はなしていました。
