天然記念物てんねんきねんぶつの「ニホンヤマネ」　毛布もうふの中なかで眠ねむっていた

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


「ニホンヤマネ」は体からだの大おおきさが８ｃｍぐらいのネズミやリスに似にている動物どうぶつで、日本にっぽんだけにいる種類しゅるいです。１００万まん年ねん以上いじょう前まえからいるので、「生いきた化石かせき」と呼よばれていて、国くにの天然記念物てんねんきねんぶつになっています。
大分県おおいたけん日田市ひたしに住すんでいる永瀬ながせ豊子とよこさんは去年きょねん１２月がつの終おわりに、泊とまりに来くる家族かぞくのために押おし入いれの中なかから布団ふとんなどを出だしていました。そして、毛布もうふの中なかで丸まるくなって眠ねむっている小ちいさい動物どうぶつを見みつけました。
永瀬ながせさんが博物館はくぶつかんの人ひとに相談そうだんすると、この動物どうぶつはニホンヤマネだとわかりました。博物館はくぶつかんの人ひとは「ヤマネは冬ふゆになると、土つちの中なかや木きの穴あななどで眠ねむります。このヤマネは間違まちがって家いえの押おし入いれに入はいって、暖あたたかくて眠ねむってしまったと思おもいます」と話はなしていました。
永瀬ながせさんは、「ヤマネが起おきたら近ちかくの森もりに帰かえすつもりでしたが、先週せんしゅう、いなくなってしまいました。丸まるくてかわいかったので、もう少すこし長ながくいてほしかったです」と話はなしていました。

東京とうきょうの品川区しながわくが「五反田ごたんだバレー」の会社かいしゃを応援おうえんする

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


東京とうきょうの品川区しながわくにある五反田駅ごたんだえきの周まわりには、コンピューターのＡＩを使つかった製品せいひんやサービスを研究けんきゅうする新あたらしい会社かいしゃが集あつまっています。集あつまっている会社かいしゃは「五反田ごたんだバレー」という団体だんたいを作つくって、五反田ごたんだをアメリカのシリコンバレーのような場所ばしょにしたいと考かんがえています。
品川区しながわくは、このような会社かいしゃが仕事しごとをしやすいように応援おうえんすることにしました。品川区しながわくの商店街しょうてんがいをにぎやかにしたり、災害さいがいのとき役やくに立たつ物ものを作つくったりする会社かいしゃには、２年ねんで２０００万まん円えんまで出だします。会社かいしゃが考かんがえたことや作つくった物ものが役やくに立たつかどうかテストする場所ばしょも品川区しながわくが用意よういします。できてからまだ５年ねんになっていない会社かいしゃの場合ばあい、銀行ぎんこうなどからお金かねを借かりやすくします。
品川区しながわくは「たくさんの会社かいしゃが新あたらしいことに挑戦ちょうせんできるようにして、シリコンバレーと同おなじような場所ばしょに育そだてたいです」と話はなしています。

「冷蔵庫れいぞうこに入いれなくても大丈夫だいじょうぶ」新あたらしいかまぼこができた

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


宮城県みやぎけんの仙台せんだいはいろいろなお土産みやげがあって、かまぼこも有名ゆうめいです。しかし、かまぼこは冷蔵庫れいぞうこなどで保存ほぞんしなければならないため、お土産みやげに買かっても、長ながい時間じかん運はこぶことができませんでした。
宮城県みやぎけんの気仙沼市けせんぬましと南三陸町みなみさんりくちょうにあるかまぼこの会社かいしゃが一緒いっしょに、冷蔵庫れいぞうこに入いれなくてもいい新あたらしいかまぼこを作つくりました。会社かいしゃはかまぼこを焼やく時間じかんや材料ざいりょうを研究けんきゅうして、おいしさや香かおりなどが、できたときのまま９０日にち続つづくようにしました。チーズの味あじと、仙台せんだいの有名ゆうめいな食たべ物ものの牛ぎゅうタンの味あじのかまぼこを作つくりました。
このかまぼこはＪＲの仙台駅せんだいえきにあるお土産みやげの店みせなどで売うっています。会社かいしゃは旅行りょこうに来きた外国人がいこくじんにもぜひ買かってほしいと考かんがえています。

タイのバンコク　空気くうきがとても汚きたないため学校がっこうを休やすみにする

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


タイの首都しゅとバンコクとその近ちかくでは、しばらく雨あめが降ふっていません。このため、自動車じどうしゃや工場こうじょうから出でるガスなどで空気くうきがとても汚きたなくなっています。
バンコクでは１月がつ３０日にちの昼ひるごろ、空気くうきを汚きたなくするＰＭ２．５という物質ぶっしつの濃度のうどが、調しらべた場所ばしょ全部ぜんぶでＷＨＯの基準きじゅんよりとても高たかくなりました。このため、タイの政府せいふはバンコクにある学校がっこうや幼稚園ようちえんを２月がつ１日ついたちまで休やすみにするように言いいました。
３０日にちの午後ごごから休やすみになった学校がっこうでは、家族かぞくが子こどもを迎むかえに来きていました。４歳さいと１５歳さいの子こどもを迎むかえに来きた女性じょせいは「子こどもはせきや鼻水はなみずが出でているので心配しんぱいです」と話はなしていました。
日本人にっぽんじんの子こどもが２６００人にんぐらい通かよっている学校がっこうも、３１日にちから２月がつ２日ふつかまで休やすみにすることになりました。

東京とうきょうオリンピックのチケット　申もうし込こみは４月がつごろから

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


２０２０年ねんに東京とうきょうで行おこなうオリンピックのチケットを買かうための申もうし込こみが、今年ことしの４月がつごろからインターネットで始はじまります。
インターネットでチケットを買かうためには、最初さいしょに「ＴＯＫＹＯ ２０２０ ＩＤ」に登録とうろくします。そして４月がつごろから、東京とうきょうオリンピックのウェブサイトで見みたい競技きょうぎや席せきなどを選えらんで申もうし込こみます。チケットの抽ちゅうせんの結果けっかは、６月がつ中旬ちゅうじゅんごろに発表はっぴょうされます。
今年ことしの秋あきからは、抽ちゅうせんがないチケットも売うり始はじめます。来年らいねんの春はるには東京とうきょうにチケットを売うる店みせもできます。
チケットの値段ねだんは、競技きょうぎや席せきで違ちがいます。例たとえば、陸上りくじょうの男子だんしの１００ｍ決勝けっしょうを見みることができるチケットは、５８００円えんから１３万まん円えんです。
パラリンピックのチケットは、今年ことしの夏なつから売うり始はじめる予定よていです。

スマートフォン　去年きょねんはファーウェイがとても多おおくなる

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


アメリカの調査ちょうさ会社がいしゃによると、去年きょねん工場こうじょうで生産せいさんして店みせなどに出だしたスマートフォンは世界せかいで１４億おく台だいでした。おととしに比くらべて４％ぐらい少すくなくなりました。２年ねん続つづけて少すくなくなって、世界せかいでいちばんたくさん売うっている中国ちゅうごくでは１０％ぐらい少すくなくなりました。
店みせなどに出だしたスマートフォンの中なかでいちばん割合わりあいが大おおきかったのは、韓国かんこくのサムスン電子でんしのスマートフォンで２０．８％でした。２番目ばんめはアメリカのアップルで、１４．９％でした。３番目ばんめは中国ちゅうごくのファーウェイで、１４．７％でした。ファーウェイはとても増ふえて、アップルとほとんど同おなじぐらいになりました。
調査ちょうさ会社がいしゃは、今年ことしも店みせなどに出だすスマートフォンが減へると考かんがえています。１つのスマートフォンを長ながく使つかう人ひとが増ふえて、スマートフォンを持もっていない人ひとが減へっているためだと言いっています。

ペットで人気にんきのカワウソ　「輸出ゆしゅつや輸入ゆにゅうを禁止きんししてほしい」

发表于 2019-01-30 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんでは最近さいきん、カワウソの人気にんきが高たかくなっています。カワウソをペットにするために、１匹ぴき１００万まん円えんぐらいで買かう人ひともいます。
インドの政府せいふによると、カワウソは３０年ねん前まえと比くらべて３０％以上いじょう少すくなくなっています。特とくに「コツメカワウソ」と「ビロードカワウソ」という種類しゅるいは将来しょうらいいなくなってしまう可能性かのうせいがあります。
このため、インドとバングラデシュとフィリピンの政府せいふは、２種類しゅるいのカワウソの輸出ゆしゅつや輸入ゆにゅうを「ワシントン条約じょうやく」で禁止きんししてほしいと言いっています。
「ワシントン条約じょうやく」に入はいっている国くには、今年ことし５月がつの会議かいぎでこの問題もんだいについて話はなし合あう予定よていです。インドの政府せいふは「世界せかいの国くにが協力きょうりょくして、カワウソを守まもる必要ひつようがあります」と言いっています。

发表于 2019-01-30 更新于 2019-11-30 分类于 JapanEasyNews


タニタという会社かいしゃは、お年寄としよりがもうすぐ介護かいごが必要ひつようになるくらい体からだが弱よわくなっていないか、調しらべることができる機械きかいを作つくりました。
この機械きかいは体重たいじゅう計けいのような形かたちです。この上うえに乗のって手てでグリップを握にぎると、足あしの裏うらや手てのひらから体からだに弱よわい電気でんきが流ながれます。そして、体からだの筋肉きんにくの量りょうを調しらべることができます。転ころびやすいかどうかなど体からだのバランスもわかります。
今いままでは機械きかいで調しらべたあと医者いしゃが計算けいさんをしていましたが、この機械きかいを使つかうと、調しらべた結果けっかがすぐにわかります。
会社かいしゃは、体からだが弱よわくなってきたことがわかったら、運動うんどうなどをして、たくさんのお年寄としよりにできるだけ長ながい間あいだ、元気げんきに生活せいかつしてほしいと考かんがえています。

「外国人がいこくじんが日本語にほんごを勉強べんきょうできるように国くにもお金かねを出だして」

发表于 2019-01-30 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんでは今年ことし４月がつから法律ほうりつを変かえて、今いまよりたくさんの外国人がいこくじんが働はたらくことができるようにします。
２９日にち、今いまたくさんの外国人がいこくじんが住すんでいる１５の市しや町まちが集あつまる会議かいぎがありました。
群馬県ぐんまけん大泉町おおいずみまちは住すんでいる人ひとの２０％ぐらいが外国人がいこくじんです。大泉町おおいずみまちの村山むらやま俊明としあき町長ちょうちょうは「外国人がいこくじんの子こどもに日本語にほんごを教おしえる人ひとの給料きゅうりょうなどは町まちが全部ぜんぶ払はらっています。国くにもお金かねを出だして助たすけてほしいです」と話はなしました。三重県みえけん四日市市よっかいちしの森もり智広ともひろ市長しちょうは「外国人がいこくじんが働はたらいている会社かいしゃが日本語にほんごを教おしえるようにしてほしいです。国くにはそのためにお金かねを出だしてほしいです」と言いいました。
日本国際交流にほんこくさいこうりゅうセンターの人ひとは「日本にっぽんでは今いままで、働はたらく外国人がいこくじんが日本にっぽんで長ながく生活せいかつするために必要ひつようなことをしっかり考かんがえていませんでした。このため、日本語にほんごがわからない外国人がいこくじんがいます」と話はなしました。
