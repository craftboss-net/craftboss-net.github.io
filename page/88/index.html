日本にっぽんとＥＵ　関税かんぜいをなくす新あたらしい貿易ぼうえきのルールが始はじまる

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんとＥＵの新あたらしい貿易ぼうえきのルールが、２月がつ１日ついたちから始はじまりました。
日本にっぽんはＥＵから輸入ゆにゅうする物もののうち９４％の種類しゅるいの品物しなもので関税かんぜいをなくします。ワインは関税かんぜいを全部ぜんぶなくしました。カマンベールなどのソフトチーズは少すこしずつ関税かんぜいを減へらして、２０３３年ねんに全部ぜんぶなくします。服ふくも関税かんぜいをなくしました。かばんや革かわの靴くつは２０２９年ねんに関税かんぜいをなくします。
日本にっぽんがＥＵに輸出ゆしゅつする物ものは、９９%の種類しゅるいの品物しなもので関税かんぜいがなくなります。しょうゆ、みそ、日本酒にほんしゅ、緑茶りょくちゃ、牛肉ぎゅうにくは関税かんぜいが全部ぜんぶなくなりました。車くるまは２０２６年ねんに関税かんぜいがなくなります。
ＥＵには５億おく人にんが住すんでいます。新あたらしいルールが始はじまって、日本にっぽんからＥＵへの輸出ゆしゅつを増ふやしたいとたくさんの人ひとが考かんがえています。

インフルエンザの流行りゅうこうが続つづいている

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


厚生労働省こうせいろうどうしょうによると、１月がつ２７日にちまでの１週間しゅうかんに、日本にっぽんで約やく２２２万まん６０００人にんがインフルエンザになったと考かんがえられます。前まえの週しゅうより１０万まん人にんぐらい増ふえました。厚生労働省こうせいろうどうしょうによると、インフルエンザだとわかった人ひとは１つの病院びょういんで平均へいきん５７．０９人にんいました。調しらべ始はじめた１９９９年ねんから今いままでで、最もっとも多おおくなりました。埼玉県さいたまけん、新潟県にいがたけん、千葉県ちばけんで特とくに多おおくなっています。
ウイルスのタイプは、ほとんどが「Ａ香港ほんこん型がた」と、１０年ねん前まえに世界せかいで流行りゅうこうした「Ｈ１Ｎ１型」です。
厚生労働省こうせいろうどうしょうは「インフルエンザの流行りゅうこうが続つづいているので、手てをよく洗あらったりマスクをしたりするようにしてください。体からだの具合ぐあいがいつもと違ちがうときは、出でかけないで休やすんだり病院びょういんに行いったりしてください」と言いっています。

この作品さくひんは２月がつ３日みっかまで立川市たちかわしにあるデパートに飾かざってあります。

東京とうきょうの品川区しながわくにある五反田駅ごたんだえきの周まわりには、コンピューターのＡＩを使つかった製品せいひんやサービスを研究けんきゅうする新あたらしい会社かいしゃが集あつまっています。集あつまっている会社かいしゃは「五反田ごたんだバレー」という団体だんたいを作つくって、五反田ごたんだをアメリカのシリコンバレーのような場所ばしょにしたいと考かんがえています。

東京とうきょうオリンピックのチケット　申もうし込こみは４月がつごろから

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


２０２０年ねんに東京とうきょうで行おこなうオリンピックのチケットを買かうための申もうし込こみが、今年ことしの４月がつごろからインターネットで始はじまります。
インターネットでチケットを買かうためには、最初さいしょに「ＴＯＫＹＯ ２０２０ ＩＤ」に登録とうろくします。そして４月がつごろから、東京とうきょうオリンピックのウェブサイトで見みたい競技きょうぎや席せきなどを選えらんで申もうし込こみます。チケットの抽ちゅうせんの結果けっかは、６月がつ中旬ちゅうじゅんごろに発表はっぴょうされます。
今年ことしの秋あきからは、抽ちゅうせんがないチケットも売うり始はじめます。来年らいねんの春はるには東京とうきょうにチケットを売うる店みせもできます。
チケットの値段ねだんは、競技きょうぎや席せきで違ちがいます。例たとえば、陸上りくじょうの男子だんしの１００ｍ決勝けっしょうを見みることができるチケットは、５８００円えんから１３万まん円えんです。
パラリンピックのチケットは、今年ことしの夏なつから売うり始はじめる予定よていです。

スマートフォン　去年きょねんはファーウェイがとても多おおくなる

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


アメリカの調査ちょうさ会社がいしゃによると、去年きょねん工場こうじょうで生産せいさんして店みせなどに出だしたスマートフォンは世界せかいで１４億おく台だいでした。おととしに比くらべて４％ぐらい少すくなくなりました。２年ねん続つづけて少すくなくなって、世界せかいでいちばんたくさん売うっている中国ちゅうごくでは１０％ぐらい少すくなくなりました。

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


タイの首都しゅとバンコクとその近ちかくでは、しばらく雨あめが降ふっていません。このため、自動車じどうしゃや工場こうじょうから出でるガスなどで空気くうきがとても汚きたなくなっています。
バンコクでは１月がつ３０日にちの昼ひるごろ、空気くうきを汚きたなくするＰＭ２．５という物質ぶっしつの濃度のうどが、調しらべた場所ばしょ全部ぜんぶでＷＨＯの基準きじゅんよりとても高たかくなりました。このため、タイの政府せいふはバンコクにある学校がっこうや幼稚園ようちえんを２月がつ１日ついたちまで休やすみにするように言いいました。
３０日にちの午後ごごから休やすみになった学校がっこうでは、家族かぞくが子こどもを迎むかえに来きていました。４歳さいと１５歳さいの子こどもを迎むかえに来きた女性じょせいは「子こどもはせきや鼻水はなみずが出でているので心配しんぱいです」と話はなしていました。
日本人にっぽんじんの子こどもが２６００人にんぐらい通かよっている学校がっこうも、３１日にちから２月がつ２日ふつかまで休やすみにすることになりました。

ペットで人気にんきのカワウソ　「輸出ゆしゅつや輸入ゆにゅうを禁止きんししてほしい」

发表于 2019-01-30 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんでは最近さいきん、カワウソの人気にんきが高たかくなっています。カワウソをペットにするために、１匹ぴき１００万まん円えんぐらいで買かう人ひともいます。
インドの政府せいふによると、カワウソは３０年ねん前まえと比くらべて３０％以上いじょう少すくなくなっています。特とくに「コツメカワウソ」と「ビロードカワウソ」という種類しゅるいは将来しょうらいいなくなってしまう可能性かのうせいがあります。
このため、インドとバングラデシュとフィリピンの政府せいふは、２種類しゅるいのカワウソの輸出ゆしゅつや輸入ゆにゅうを「ワシントン条約じょうやく」で禁止きんししてほしいと言いっています。
「ワシントン条約じょうやく」に入はいっている国くには、今年ことし５月がつの会議かいぎでこの問題もんだいについて話はなし合あう予定よていです。インドの政府せいふは「世界せかいの国くにが協力きょうりょくして、カワウソを守まもる必要ひつようがあります」と言いっています。

髪かみの毛けを調しらべてストレスをチェックする新あたらしい技術ぎじゅつ

发表于 2019-01-30 更新于 2019-11-30 分类于 JapanEasyNews


滋賀大学しがだいがくの研究けんきゅうグループによると、人ひとはストレスがあると体からだの中なかでホルモンが出でます。このホルモンは髪かみの毛けの中なかに残のこるため、髪かみの毛けを調しらべると、ストレスがどのくらいあったかわかります。
グループは、特別とくべつな薬くすりを使つかって、３本ぼんか４本ほんぐらいの髪かみの毛けで簡単かんたんにストレスをチェックできる技術ぎじゅつを作つくりました。
髪かみの毛けは１か月げつに１ｃｍぐらい伸のびるため、髪かみの毛けを１ｃｍずつ切きって調しらべると、いつごろ強つよいストレスがあったかわかります。グループによると、６か月げつ前まえまでのストレスをチェックできます。
グループは「ストレスをなくすために、この技術ぎじゅつが役やくに立たつといいと思おもいます」と言いっています。
