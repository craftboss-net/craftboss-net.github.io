台風たいふう１０号ごう　多おおくの雨あめが降ふる危険きけんがある

发表于 2019-08-15 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうによると、台風たいふう１０号ごうは１５日にちに九州きゅうしゅうと四国しこくの間あいだを通とおって、広島県ひろしまけんに入はいりました。このあと、中国地方ちゅうごくちほうを通とおって日本海にほんかいに出でて、北東ほくとうに進すすみそうです。
四国しこくや紀伊半島きいはんとうなどでは雨あめがたくさん降ふっています。九州きゅうしゅうから北海道ほっかいどうまでの広ひろい場所ばしょでも、雨あめがたくさん降ふるかもしれません。
雨あめがたくさん降ふった場所ばしょでは、山やまが崩くずれたり川かわの水みずがあふれたりするかもしれません。強つよい風かぜや高たかい波なみにも気きをつけてください。
台風たいふうについては、気象庁きしょうちょうや市しなどが出だす新あたらしい情報じょうほうをチェックしてください。台風たいふうが来きたときに気きをつけることは「NEWS　WEB　EASY」の「台風たいふう」のページに書かいてあります。
https://www3.nhk.or.jp/news/easy/article/disaster_typhoon.html

戦争せんそうが終おわってから７４年ねん　亡なくなった人ひとのために祈いのる式しき

发表于 2019-08-15 更新于 2019-11-30 分类于 JapanEasyNews



１５日にち、太平洋戦争たいへいようせんそうが終おわってから７４年ねんになりました。
東京とうきょうの日本武道館にっぽんぶどうかんでは、この戦争せんそうで亡なくなった約やく３１０万まん人にんのために祈いのる式しきがありました。亡なくなった人ひとの家族かぞくなど約やく６０００人にんが出席しゅっせきしました。
式しきの最初さいしょに安倍あべ総理大臣そうりだいじんが「もう戦争せんそうはしないと誓ちかいます。この誓ちかいは昭和しょうわ、平成へいせい、令和れいわの時代じだいになっても決けっして変かわりません」と話はなしました。そして１２時じになると、みんなで目めを閉とじて静しずかに祈いのりました。
今年ことし５月がつに即位そくいした天皇てんのう陛下へいかは初はじめてこの式しきに出席しゅっせきして、「戦争せんそうでたくさんの人ひとが亡なくなったことをとても悲かなしく思おもいます。戦争せんそうがまた起おこらないように強つよく願ねがっています」と話はなしました。
式しきに出席しゅっせきした家族かぞくの８０％は７０歳さい以上いじょうでした。若わかい人ひとたちにも戦争せんそうのことを伝つたえていくため、１８歳さいより若わかい人ひとも９５人にん出席しゅっせきしました。

アメリカ　銃じゅうから子こどもを守まもるリュックサックが売うれている

发表于 2019-08-14 更新于 2019-11-30 分类于 JapanEasyNews



アメリカでは、銃じゅうで撃うたれて人ひとが亡なくなる事件じけんが続つづいています。このため、子こどもが安全あんぜんに学校がっこうへ行いくことができるように、銃じゅうから体からだを守まもる リュックサックがよく売うれています。
アメリカのテキサス州しゅうにある会社かいしゃは、銃じゅうの弾たまが通とおらない特別とくべつな板いたを入いれたリュックサックを作つくっています。警察けいさつの人ひとが体からだを守まもるために着きている服ふくと同おなじぐらいの強つよさで、値段ねだんは１万まん５０００円えんぐらいです。
１３歳さいの子こどもがいる女性じょせいは「安やすくないですが、子こどもを守まもるために買かいたいです」と話はなしています。別べつの男性だんせいは「このようなリュックサックを持もって学校がっこうへ行いく子こどもたちを想像そうぞうできません」と話はなしています。専門家せんもんかは「リュックサックを使つかっても問題もんだいを解決かいけつすることはできません。子こどもの安全あんぜんのためにどうしたらいいか、学校がっこうが考かんがえなければなりません」と言いっています。

发表于 2019-08-14 更新于 2019-11-30 分类于 JapanEasyNews



富士山ふじさんの近ちかくにある山梨県やまなしけん富士河口湖町ふじかわぐちこまちのホテルなどには、去年きょねん６０万まん人にん以上いじょうの外国人がいこくじんが泊とまりました。しかし町まちは、富士山ふじさんが噴火ふんかしたときにどうやって外国人がいこくじんを避難ひなんさせるか決きめていませんでした。
町まちは山梨大学やまなしだいがくと協力きょうりょくして、外国人がいこくじんを安全あんぜんに避難ひなんさせるための実験じっけんを今月こんげつ１９日にちに行おこなうことにしました。
山梨大学やまなしだいがくが作つくった１０の外国がいこくのことばに翻訳ほんやくするシステムを使つかって、文化ぶんかや習慣しゅうかんが違ちがう外国人がいこくじんに情報じょうほうをわかりやすく伝つたえることができるかどうか調しらべます。最初さいしょは、外国人がいこくじんの中なかでいちばん多おおい中国人ちゅうごくじんに伝つたえる実験じっけんをします。
山梨大学やまなしだいがくの教授きょうじゅは「外国人がいこくじんが噴火ふんかのときに困こまらないように、いろいろな外国がいこくのことばで情報じょうほうを伝つたえることができるかどうかを調しらべたいと思おもいます」と話はなしています。

「百日ひゃくにちせき」になる人ひとが増ふえる　赤あかちゃんは特とくに気きをつけて

发表于 2019-08-14 更新于 2019-11-30 分类于 JapanEasyNews



「百日ひゃくにちせき」は激はげしい せきが続つづく病気びょうきで、細菌さいきんが原因げんいんでうつります。生うまれてから６か月げつまでの赤あかちゃんがこの病気びょうきになると、肺炎はいえんになって亡なくなることがあります。
国くにの研究所けんきゅうじょによると、今年ことし１月がつから今月こんげつ４日よっかまでに百日ひゃくにちせきになった人ひとが１万まん人にん以上いじょうになりました。去年きょねん１年ねんでこの病気びょうきになった人ひとと同おなじぐらいです。
百日ひゃくにちせきは子こどもに多おおい病気びょうきですが、去年きょねんこの病気びょうきになった人ひとのうち２０％以上いじょうは２０歳さい以上いじょうの人ひとでした。大人おとなはこの病気びょうきになってもひどくならないため、かぜだと思おもって病院びょういんに行いかないで、周まわりの人ひとにうつしてしまうことがあります。
研究所けんきゅうじょの人ひとは「赤あかちゃんにはできるだけ早はやくワクチンの注射ちゅうしゃをしてください。赤あかちゃんが近ちかくにいる人ひとは、せきが出でたら病院びょういんに行いってください」と話はなしています。

日本航空にほんこうくうの飛行機ひこうきが落おちた事故じこから３４年ねん

发表于 2019-08-13 更新于 2019-11-30 分类于 JapanEasyNews



１９８５年ねん８月がつ１２日にち、東京とうきょうから大阪おおさかに向むかっていた日本航空にほんこうくうの飛行機ひこうきが群馬県ぐんまけん上野村うえのむらの山やまの中なかに落おちました。日本にっぽんで起おこった飛行機ひこうき事故じこの中なかでいちばん多おおい５２０人にんが亡なくなりました。
事故じこから３４年ねんになった１２日にち、亡なくなった人ひとたちの家族かぞくなどが上野村うえのむらの山やまに登のぼりました。家族かぞくたちは飛行機ひこうきが落おちた場所ばしょにある石碑せきひの前まえに着つくと、静しずかに祈いのっていました。
午後ごご６時じからは、山やまの下したのほうにある「慰霊いれいの園その」で家族かぞくや日本航空にほんこうくうの人ひとなどが式しきを行おこなって、悲かなしい事故じこがまた起おこらないように祈いのりました。
事故じこで祖父そふが亡なくなった２７歳さいの男性だんせいは「娘むすめを連つれて、初はじめて山やまを登のぼりました。『元気げんきに育そだっているよ、この子こを守まもってください』という気持きもちで祈いのりました」と話はなしていました。
