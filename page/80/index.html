東北地方とうほくちほうで大おおきな地震じしんの可能性かのうせい「９０％以上いじょうの所ところもある」

发表于 2019-02-27 更新于 2019-11-30 分类于 JapanEasyNews



東北地方とうほくちほうから関東地方かんとうちほうの沖おきの海うみの底そこにある「日本海溝にほんかいこう」の近ちかくでは、何なん度ども地震じしんが起おこっています。８年ねん前まえには東日本大震災ひがしにほんだいしんさいがありました。
地震じしんについて調しらべている政府せいふの委員会いいんかいは、この場所ばしょでこれから３０年ねん以内いないに地震じしんが起おこる 可能性かのうせいを新あたらしく計算けいさんしました。
計算けいさんによると、８年ねん前まえと同おなじマグニチュード９ぐらいの地震じしんが起おこる 可能性かのうせいはほとんど０％でした。しかし、マグニチュード７から７．５ぐらいの地震じしんが起おこる 可能性かのうせいは高たかくなっています。
青森県あおもりけんの東ひがしと岩手県いわてけんの北きたの沖おきでは９０％以上いじょう、宮城県みやぎけんの沖おきでは９０％ぐらいでした。茨城県いばらきけんの沖おきでは８０％ぐらい、福島県ふくしまけんの沖おきでは５０％ぐらい、岩手県いわてけんの南みなみの沖おきでは３０％ぐらいでした。
調しらべた委員会いいんかいの人ひとは「東北地方とうほくちほうではもう大おおきな地震じしんは起おこらないと考かんがえないで、強つよい揺ゆれや高たかい津波つなみが来きたときのことを考かんがえて、大おおきな被害ひがいが出でないように準備じゅんびしてほしいです」と話はなしています。

沖縄県おきなわけん　辺野古へのこの工事こうじに「反対はんたい」と投票とうひょうした人ひとが多おおかった

发表于 2019-02-25 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの政府せいふは、沖縄県おきなわけん宜野湾市ぎのわんしにあるアメリカ軍ぐんの普天間基地ふてんまきちを名護市なごし辺野古へのこに移うつすため、辺野古へのこの海うみを埋うめる 工事こうじをしています。
２４日にじゅうよっか、工事こうじについて沖縄県おきなわけんの人ひとたちの意見いけんを聞きく投票とうひょうがあって、有権者ゆうけんしゃの５２．４８％が投票とうひょうしました。「反対はんたい」は４３万まん４２７３票びょう、「賛成さんせい」は１１万まん４９３３票びょう、「どちらでもない」は５万まん２６８２票ひょうでした。
「反対はんたい」に投票とうひょうした人ひとが有権者ゆうけんしゃの２５％より多おおかったため、沖縄県おきなわけんの玉城たまき知事ちじは日本にっぽんの総理大臣そうりだいじんとアメリカの大統領だいとうりょうに投票とうひょうの結果けっかを伝つたえることになりました。
玉城たまき知事ちじは、沖縄県おきなわけんの人ひとたちが工事こうじに反対はんたいしていることがはっきりわかったため、工事こうじをやめてもらいたいと考かんがえています。
日本にっぽんの政府せいふは、普天間基地ふてんまきちをできるだけ早はやく移うつしたいと考かんがえていて、工事こうじを続つづけるつもりです。
政府せいふと沖縄県おきなわけんの話はなし合あいは難むずかしくなりそうです。

日本にほん文学ぶんがくを５０年ねん以上いじょう研究けんきゅうしていたドナルド?キーンさんが、２４日にじゅうよっか、東京とうきょうの病院びょういんで亡なくなりました。９６歳さいでした。
キーンさんは、１９２２年ねんにアメリカで生うまれて、海軍かいぐんの学校がっこうで日本語にほんごを勉強べんきょうしました。そして京都大学きょうとだいがくに留学りゅうがくしたあと、アメリカのコロンビア大学だいがくの教授きょうじゅになって、日本にほん文学ぶんがくの研究けんきゅうを続つづけました。
キーンさんは、日本にほん文学ぶんがくについてたくさんの本ほんを書かいたり、日本にっぽんで昔むかしから続つづいている能のうや歌舞伎かぶき、日本にっぽんの文化ぶんかについて意見いけんを発表はっぴょうしたりしました。古ふるい時代じだいの本ほんなどたくさんの本ほんを英語えいごに翻訳ほんやくして、日本にほん文学ぶんがくを世界せかいに紹介しょうかいしました。

北海道ほっかいどうで震度しんど６弱じゃくの地震じしん　「去年きょねん９月がつの地震じしんと関係かんけいがある」

发表于 2019-02-22 更新于 2019-11-30 分类于 JapanEasyNews



２１日にち午後ごご９時じ２２分ふんごろ、北海道ほっかいどうの胆振いぶり地方ちほうでマグニチュード５．８の地震じしんがありました。
北海道ほっかいどう厚真町あつまちょうでは震度しんど６弱じゃくの大おおきな揺ゆれがありました。震度しんど５強きょうは安平町あびらちょうとむかわ町ちょう、震度しんど５弱じゃくは札幌市さっぽろしの北きた区くと手稲ていね区く、千歳市ちとせし、長沼町ながぬまちょう、平取町びらとりちょうでした。津波つなみはありませんでした。この地震じしんで大おおきなけがをした人ひとはいませんでした。
胆振いぶり地方ちほうでは去年きょねん９月がつ６日むいかにもマグニチュード６．７の地震じしんがあって、厚真町あつまちょうで震度しんど７の揺ゆれがありました。
気象庁きしょうちょうは「２１日にちの地震じしんは去年きょねん９月がつの地震じしんと関係かんけいがあると考かんがえています。大おおきく揺ゆれた場所ばしょでは家いえが壊こわれたり山やまなどが崩くずれたりする心配しんぱいがあります。これからも同おなじぐらい揺ゆれる地震じしんがあるかもしれないため、気きをつけてください」と言いっています。
