車くるまの新あたらしい技術ぎじゅつを紹介しょうかいする「東京とうきょうモーターショー」が始はじまる

发表于 2019-10-24 更新于 2019-11-30 分类于 JapanEasyNews



２年ねんに１度ど、車くるまの新あたらしい技術ぎじゅつなどを紹介しょうかいする「東京とうきょうモーターショー」が始はじまりました。今年ことしは日本にっぽんや外国がいこくから１８０以上いじょうの会社かいしゃが集あつまりました。
トヨタ自動車じどうしゃやホンダなどは、来年らいねん発売はつばいする電気でんき自動車じどうしゃを紹介しょうかいしています。自動じどう運転うんてんと関係かんけいがある技術ぎじゅつの展示てんじもあります。車くるまの会社かいしゃのほかにも、電気でんき製品せいひんの会社かいしゃなどが未来みらいの生活せいかつをテーマにした展示てんじをしています。
東京とうきょうモーターショーを見みに来くる人ひとは、いちばん多おおいとき２００万まん人にん以上いじょういましたが、２年ねん前まえは７７万まん人にんでした。今年ことしは１００万まん人にんに来きてもらうため、今いままでしたことがない展示てんじやイベントを行おこなっています。
会場かいじょうでは、１人ひとりが乗のる小ちいさい車くるまに乗のったり、インターネットを使つかった車くるまのゲームを楽たのしんだりできます。家族かぞくと一緒いっしょに楽たのしむことができるように、高校生こうこうせい以下いかは無料むりょうです。
東京とうきょうモーターショーは１１月がつ４日よっかまで、東京都とうきょうと江東区こうとうくの東京とうきょうビッグサイトなどで行おこなっています。

がんになった若わかい世代せだいの７８％が女性じょせい

发表于 2019-10-23 更新于 2019-11-30 分类于 JapanEasyNews



１５歳さいから３９歳さいまでの若わかい世代せだいは、ほかの世代せだいと比くらべると、がんになる人ひとが少すくないです。このため、がんの診療しんりょうをする病院びょういんや、がんになったときの支援しえんが少すくないと言いわれています。
国立成育医療研究こくりつせいいくいりょうけんきゅうセンターと国立こくりつがん研究けんきゅうセンターは、２０１６年ねんと２０１７年ねん、約やく８００の病院びょういんなどでがんだとわかった若わかい世代せだいについて調しらべました。
がんだとわかった５万まん７７８８人にんの７８％は女性じょせいでした。もっと年齢ねんれいが上うえの世代せだいは、男性だんせいのほうが多おおくなっています。
２０歳さいから３９歳さいまでのがんだとわかった女性じょせいの３５％が、できてすぐの子宮頸しきゅうけいがんや乳にゅうがんでした。
国立成育医療研究こくりつせいいくいりょうけんきゅうセンターは「若わかい世代せだいの女性じょせいは、仕事しごとや結婚けっこんや出産しゅっさんで忙いそがしい人ひとが多おおいため、がんの診療しんりょうに支援しえんが必要ひつようです」と話はなしています。

台風たいふうのあとのボランティアの仕事しごとをロボットが楽らくにする

发表于 2019-10-23 更新于 2019-11-30 分类于 JapanEasyNews



茨城県いばらきけん大子町だいごまちでは、台風たいふう１９号ごうで川かわの水みずがあふれて家いえの中なかに水みずが入はいったため、ボランティアの人ひとたちが一緒いっしょに片かたづけています。
ボランティアの人ひとたちの仕事しごとが少すこしでも楽らくになるように、「ＨＡＬ」というロボットを使つかうことになりました。茨城県いばらきけんつくば市しにある会社かいしゃがつくったロボットで、腰こしにつけると、モーターが動うごいて４０％少すくない力ちからで物ものを運はこぶことができます。
２０日はつかから、このロボットを６台だい使つかい始はじめました。ボランティアの人ひとたちは、ロボットを体からだにつけて、家いえの中なかに入はいった泥どろを運はこんでいました。東京とうきょうから来きた男性だんせいは「腰こしが痛いたくなりませんでした。少すくない力ちからで仕事しごとができたと思おもいます」と話はなしていました。
ロボットは１か月げつぐらい使つかう予定よていです。

２２日にち、天皇てんのう陛下へいかが即位そくいしたことを知しらせる儀式ぎしきが皇居こうきょでありました。儀式ぎしきには、天皇てんのう陛下へいかと皇后こうごうさまのほか、皇族こうぞくや招待しょうたいされた外国がいこくの人ひとなど、２０００人にんぐらいが出席しゅっせきしました。

避難所ひなんじょでの生活せいかつ　「水みずを十分じゅうぶんに飲のんで、体からだを動うごかして」

发表于 2019-10-23 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふう１９号ごうの被害ひがいで家いえに住すむことができなくなった多おおくの人ひとが、学校がっこうの建物たてものなどの避難所ひなんじょで生活せいかつしています。避難所ひなんじょでの生活せいかつが長ながくなると、狭せまい場所ばしょでずっと座すわったままや寝ねたままでいることが多おおくなって、「エコノミークラス症候群しょうこうぐん」になる危険きけんがあります。
長野市ながのしの避難所ひなんじょで、医者いしゃが２２日にちまでに１０９人にんの体からだの具合ぐあいを調しらべました。この中なかの１人ひとりがエコノミークラス症候群しょうこうぐんだとわかって、病院びょういんに行いきました。エコノミークラス症候群しょうこうぐんかもしれない人ひとは１６人にんいました。ほとんどがお年寄としよりでした。
日本赤十字社にっぽんせきじゅうじしゃによると、避難所ひなんじょでトイレに行いきたくないと思おもって、水みずを飲のまないようにすると、エコノミークラス症候群しょうこうぐんになる危険きけんが高たかくなります。このため、日本赤十字社にっぽんせきじゅうじしゃは「水みずを十分じゅうぶんに飲のんで、ときどき足あしや体からだを動うごかしてください」と言いっています。

使つかったあとの紙かみのおむつをリサイクルする技術ぎじゅつができる

发表于 2019-10-21 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんのユニ?チャームという会社かいしゃは２０１１年ねんから、お年寄としよりが使つかう紙かみのおむつをリサイクルする研究けんきゅうをしています。そして新あたらしい技術ぎじゅつで、使つかったあとの紙かみのおむつの菌きんをなくして、きれいにして、紙かみパルプを取とることができるようになりました。この紙かみパルプは、国くにの健康けんこうのための規則きそくに合あっています。会社かいしゃは、この紙かみパルプを使つかって、紙かみのおむつなどを作つくりました。
お年寄としよりが多おおくなって、紙かみのおむつを使つかう人ひとも多おおくなっています。使つかったあとの紙かみのおむつの捨すて方かたやリサイクルのしかたを考かんがえることが必要ひつようになっています。
会社かいしゃは「使つかったあとの紙かみのおむつは、今いままではごみでした。これからは、紙かみのおむつのリサイクルを進すすめていきたいです」と話はなしています。
