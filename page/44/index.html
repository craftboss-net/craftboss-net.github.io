新潟県にいがたけんと山形県やまがたけんの地震じしん　「知しらない活断層かつだんそうが原因げんいんの可能性かのうせい」

发表于 2019-06-20 更新于 2019-11-30 分类于 JapanEasyNews



１８日にち、マグニチュード６．７（Ｍ６．７）の地震じしんが起おこって、新潟県にいがたけんや山形県やまがたけんで大おおきく揺ゆれました。地震じしんが起おこった場所ばしょは山形県やまがたけんの沖おきの日本海にほんかいで、深ふかさ１４ｋｍの所ところでした。この地震じしんで２８人にんがけがをして、家いえがたくさん壊こわれました。
東京大学とうきょうだいがくの佐藤さとう比呂志ひろし教授きょうじゅは、この地震じしんのデータと、今いままでに調しらべた日本海にほんかいの活断層かつだんそうのデータを比くらべました。活断層かつだんそうは、地層ちそうがずれていて今いまも動うごいていると考かんがえられている所ところです。
その結果けっか、今いままで知しらなかった活断層かつだんそうが原因げんいんで地震じしんが起おこった可能性かのうせいがあることがわかりました。佐藤さとう教授きょうじゅは、これからも同おなじような地震じしんが起おこるかもしれないと言いっています。
日本海にほんかいでは陸りくに近ちかい所ところで津波つなみが起おこることが多おおくて、津波つなみが陸りくに着つくまでの時間じかんが短みじかくなります。佐藤さとう教授きょうじゅは「強つよい揺ゆれを感かんじたら、すぐに逃にげてください」と話はなしました。

高知県こうちけんの人口じんこうが減へって７０万まん人にん以下いかになった

发表于 2019-06-20 更新于 2019-11-30 分类于 JapanEasyNews



高知県こうちけんは毎月まいつき、生うまれた人ひとや亡なくなった人ひと、引ひっ越こして来きた人ひとや出でて行いった人ひとを計算けいさんして、県けんの人口じんこうがどのくらいか発表はっぴょうしています。
高知県こうちけんによると、６月がつ１日ついたちの人口じんこうは６９万まん９５２２人にんになりました。高知県こうちけんの人口じんこうが７０万まん人にん以下いかになったのは、１９４５年ねんに戦争せんそうが終おわってから初はじめてです。
日本にっぽんで人口じんこうが７０万まん人にん以下いかの都道府県とどうふけんは、高知県こうちけんと５５万まん人にんぐらいの鳥取県とっとりけん、６７万まん人にんぐらいの島根県しまねけんの３つになりました。
高知県こうちけんの人口じんこうがいちばん多おおかったのは１９５５年ねんで、８８万まん人にん以上いじょういました。１９９０年ねんからは亡なくなった人ひとが生うまれた人ひとより多おおくなることなどが続つづいて、人口じんこうが減へるのを止とめることができていません。
日本にっぽんでは東京とうきょうなど大おおきなまちに人ひとが集あつまって、人口じんこうが減へる県けんなどが多おおくなっていて、大おおきな問題もんだいになっています。

外国人がいこくじんのための「外国人がいこくじん共生きょうせいセンター」を作つくる計画けいかく

日本にっぽんで働はたらく外国人がいこくじんが増ふえているため、政府せいふは外国人がいこくじんの生活せいかつを手伝てつだうための新あたらしい計画けいかくを決きめました。
政府せいふは２０２１年ねんの３月がつまでに、外国人がいこくじんの手伝てつだいをする役所やくしょなどを１つの場所ばしょに集あつめた「外国人がいこくじん共生きょうせいセンター」をつくります。東京都とうきょうと新宿区しんじゅくくにセンターを作つくって、仕事しごとを紹介しょうかいするハローワークや、ビザの手続てつづきなどをする出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうが仕事しごとをします。
外国人がいこくじんが給料きゅうりょうの高たかい都会とかいだけに集あつまる心配しんぱいがあります。このため、県けんや市しなどとハローワークが協力きょうりょくして、都会とかい以外いがいの仕事しごとも紹介しょうかいします。

发表于 2019-06-19 更新于 2019-11-30 分类于 JapanEasyNews



１８日にち午後ごご１０時じ２２分ふん、マグニチュード６．７（Ｍ６．７）の地震じしんが起おこりました。
新潟県にいがたけん村上市むらかみしは震度しんど６強きょう、山形県やまがたけん鶴岡市つるおかしは震度しんど６弱じゃくで、大おおきく揺ゆれました。新潟県にいがたけんの長岡市ながおかしと柏崎市かしわざきし、阿賀町あがまち、山形県やまがたけんの酒田市さかたしと三川町みかわまち、大蔵おおくら村むら、秋田県あきたけんの由利本荘市ゆりほんじょうしは震度しんど５弱じゃくでした。北海道ほっかいどうから東海地方とうかいちほうまで広ひろい場所ばしょで震度しんど４から震度しんど１の揺ゆれがありました。
１９日にち午前ごぜん１１時じ半はんまでに、新潟県にいがたけんと山形県やまがたけん、宮城県みやぎけん、石川県いしかわけんで、２６人にんがけがをしました。新潟県にいがたけんや山形県やまがたけんでは家いえが壊こわれたり、ブロックの塀へいが倒たおれたりしました。
気象庁きしょうちょうは「これから１週間しゅうかんぐらいの間あいだは、また地震じしんが起おこって、建物たてものが壊こわれるかもしれません。１９日にちは夜よるまで雨あめが強つよく降ふって、山やまなどが崩くずれるかもしれません。十分じゅうぶんに気きをつけてください」と言いっています。

スマートフォンを使つかって給料きゅうりょうを早はやくもらう

发表于 2019-06-18 更新于 2019-11-30 分类于 JapanEasyNews



スマートフォンを使つかって、給料きゅうりょうの一部いちぶを早はやくもらうことができるサービスが始はじまっています。
「ペイミー」という会社かいしゃはスマートフォンのアプリを作つくって、このサービスを始はじめました。アプリで申もうし込こむと、働はたらいた日ひまでの給料きゅうりょうの一部いちぶを、いつも給料きゅうりょうをもらう日ひより早はやくもらうことができます。今いま２５０ぐらいの会社かいしゃがこのアプリを利用りようしています。利用りようしている店みせは「早はやく給料きゅうりょうをもらうこともできると説明せつめいしたら、アルバイトをしたいという人ひとが増ふえました」と言いっています。
同おなじようなサービスを始はじめている会社かいしゃはほかにもあります。
サービスを利用りようするとき、給料きゅうりょうをもらう人ひとが手数料てすうりょうを払はらわなければならないことがあります。国くには「給料きゅうりょうを早はやくもらうことが多おおくなると、生活せいかつが悪わるくなるかもしれないので気きをつけてください」と言いっています。

ブロックの塀へい　安全あんぜんのためにお金かねを出だす市しなどは４０％だけ

发表于 2019-06-18 更新于 2019-11-30 分类于 JapanEasyNews



１年ねん前まえの２０１８年ねん６月がつ１８日にち、大阪府おおさかふで大おおきな地震じしんがありました。この地震じしんで５万まん７０００ぐらいの家いえや建物たてものが壊こわれました。高槻たかつき市しでは、小学校しょうがっこうのブロックの塀へいが倒たおれて、学校がっこうに行いく途中とちゅうだった女おんなの子こが亡なくなりました。
このあと市しや町まちが、ブロックの塀へいが安全あんぜんかどうか調しらべたり、塀へいをなくしたりする人ひとにお金かねを出だすようになりました。国土交通省こくどこうつうしょうによると、塀へいの安全あんぜんのためにお金かねを出だしている市しや町まちなどは、今年ことし４月がつに７７８ありました。これは全部ぜんぶの市しや町まちなどの４０％ぐらいだけです。
ブロックの塀へいの問題もんだいに詳くわしい専門家せんもんかは「安全あんぜんのためには、市しや町まちなどがお金かねを出だす必要ひつようがあると思おもいます」と話はなしています。

１２歳さいの岡本おかもと選手せんしゅがスケートボードの国際こくさい大会たいかいで優勝ゆうしょう

アメリカで１６日にち、スケートボードの国際こくさい大会たいかいの決勝けっしょうがありました。１２歳さいの岡本おかもと碧優みすぐ選手せんしゅが、深ふかい皿さらやおわんをつなげた形かたちのコースを使つかう「パーク」という種目しゅもくの女子じょしで優勝ゆうしょうしました。岡本おかもと選手せんしゅは、コースから飛とび上あがって１回かい半はん回まわる難むずかしい技わざなどに成功せいこうしました。

アメリカの大学だいがくが長崎ながさきの教会きょうかいに手紙てがみを出だす「十字架じゅうじかを返かえしたい」

发表于 2019-06-18 更新于 2019-11-30 分类于 JapanEasyNews



長崎ながさきの教会きょうかいにアメリカのウィルミントン大学だいがくから手紙てがみが来きました。手紙てがみには、第だい２次世界大戦じせかいたいせんが終おわったあと、長崎ながさきにある浦上天主堂うらかみてんしゅどうという教会きょうかいでアメリカの軍人ぐんじんが見みつけた十字架じゅうじかを返かえしたいと書かいてありました。
浦上天主堂うらかみてんしゅどうは、１９４５年ねん８月がつ９日ここのかに長崎ながさきに原爆げんばくが落おとされたとき、建物たてものが壊こわれて燃もえました。
手紙てがみによると、１９４５年ねん１０月がつに軍人ぐんじんのウォルター?フックさんは壊こわれた建物たてものの中なかから１ｍぐらいの木きの十字架じゅうじかを見みつけました。フックさんは十字架じゅうじかをアメリカに送おくって、そのあとウィルミントン大学だいがくにあげました。
手紙てがみには「十字架じゅうじかは希望きぼうなどのシンボルです。浦上天主堂うらかみてんしゅどうに返かえさなければならないと思おもうようになりました」と書かいてあります。
ウィルミントン大学だいがくの人ひとは今年ことしの８月がつに日本にっぽんに来きて十字架じゅうじかを返かえす予定よていです。
