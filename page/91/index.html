インフルエンザの女性じょせいが駅えきのホームから落おちて亡なくなる

发表于 2019-01-24 更新于 2019-11-30 分类于 JapanEasyNews


東京とうきょうメトロの中目黒駅なかめぐろえきで２２日にちの朝あさ、３７歳さいの会社員かいしゃいんの女性じょせいがホームから落おちて、電車でんしゃにはねられて亡なくなりました。
事故じこのあと警察けいさつが調しらべると、女性じょせいの体からだからインフルエンザのウイルスが見みつかりました。警察けいさつによると、女性じょせいは何なん日にちか前まえから具合ぐあいが悪わるいと言いっていましたが、インフルエンザになっていると思おもわないで会社かいしゃに行いこうとしていました。
警察けいさつがホームにいた人ひとに聞きくと、女性じょせいはせきをしながら列れつのいちばん前まえで電車でんしゃを待まっていましたが、突然とつぜん ホームから落おちたようです。
専門家せんもんかは「インフルエンザになると高たかい熱ねつが出でて、体からだの水分すいぶんが足たりなくなって、倒たおれることがあります。インフルエンザかもしれないと思おもったら、出でかけないで家いえで休やすんでほしい」と話はなしています。

２１の火山かざん　注意ちゅういしていた場所ばしょ以外いがいでも噴火ふんかの可能性かのうせいがある

发表于 2019-01-24 更新于 2019-11-30 分类于 JapanEasyNews


群馬県ぐんまけんの草津白根山くさつしらねさんが去年きょねん１月がつに噴火ふんかしたとき、噴火ふんかしたのは、気象庁きしょうちょうが噴火ふんかの可能性かのうせいがあると考かんがえていた場所ばしょとは別べつの場所ばしょでした。このため気象庁きしょうちょうは、カメラを置おいて２４時間じかん見みている日本にっぽんの５０の火山かざんについて、もう一度いちど詳くわしく調しらべました。
気象庁きしょうちょうによると、２１の火山かざんで、今いままで注意ちゅういしていた場所ばしょ以外いがいでも噴火ふんかする可能性かのうせいがあることがわかりました。北海道ほっかいどうの十勝岳とかちだけ、富士山ふじさん、神奈川県かながわけんの箱根山はこねやまなどです。
気象庁きしょうちょうは、箱根山はこねやまなど７つの火山かざんに新あたらしくカメラを置おくことにしました。市しや町まちなどが置おいたカメラも使つかって火山かざんの様子ようすを見みて、注意ちゅういしていくことにしました。

ローマ法王庁ほうおうちょうは、法王ほうおうの話はなしを英語えいごやフランス語ごなどに通訳つうやくするアプリも作つくっています。

羽田空港はねだくうこうで自動じどう運転うんてんのバスが走はしる実験じっけん

发表于 2019-01-23 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんへ旅行りょこうに来くる外国人がいこくじんは５年ねんで３倍ばいぐらいに増ふえました。しかし、空港くうこうで働はたらく人ひとは足たりなくなっています。このため、航空こうくう会社がいしゃの全日空ぜんにっくうなどは、人ひとが運転うんてんしなくても自動じどうで走はしるバスを空港くうこうの中なかで使つかおうと考かんがえて、羽田空港はねだくうこうで実験じっけんを行おこなっています。
実験じっけんは、空港くうこうのビルと飛行機ひこうきがとまっている場所ばしょとの間あいだの６００mで行おこないました。安全あんぜんのため、運転うんてん席せきには人ひとが乗のっています。バスはＧＰＳの情報じょうほうを使つかったり、走はしるコースの下したに入はいっている磁石じしゃくをセンサーでチェックしたりしながら、時速じそく２０ｋｍで走はしりました。
バスは飛行機ひこうきのエンジンから出でるガスが当あたらないように、決きまった場所ばしょで一度いちど止とまることになっています。実験じっけんでも、バスは止とまったあと、周まわりが安全あんぜんかどうか調しらべて、また走はしり始はじめました。
全日空ぜんにっくうは、来年らいねんから一部いちぶの空港くうこうで自動じどう運転うんてんのバスを使つかいたいと考かんがえています。

イギリスは今年ことし３月がつにＥＵから出でることにしていて、貿易ぼうえきなどのいろいろな約束やくそくをＥＵと決きめることにしています。しかし、イギリスの議会ぎかいでは、どんな約束やくそくをするか意見いけんが分わかれているため、約束やくそくを決きめることができないままＥＵから出でるかもしれません。

アメリカでとても大おおきくて赤あかい満月まんげつが見みえる
