ラグビー　日本にっぽんが初はじめて「決勝けっしょうトーナメント」に進すすむ

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



ラグビーのワールドカップで１３日にち、日本にっぽんはスコットランドと試合しあいをして、２８－２１で勝かちました。試合しあいが始はじまってすぐスコットランドがトライをしましたが、そのあと日本にっぽんが４つトライをして勝かちました。
日本にっぽんは、グループで行おこなう４つの試合しあいに全部ぜんぶ勝かって、グループで１番ばんになりました。そして、８つのチームで優勝ゆうしょうを決きめるための「決勝けっしょうトーナメント」に出でることが決きまりました。日本にっぽんがラグビーのワールドカップに出でるのは９回かい目めで、「決勝けっしょうトーナメント」に出でるのは初はじめてです。
試合しあいがあった横浜市よこはましの会場かいじょうには６万まん７０００人にん以上いじょうの人ひとが見みに来きました。試合しあいの前まえ、台風たいふうで亡なくなった人ひとたちのために、みんなで静しずかに祈いのりました。
日本にっぽんは２０日はつかに東京とうきょうで南みなみアフリカと試合しあいをします。日本にっぽんのキャプテンのリーチ　マイケル選手せんしゅは「もっと勝かって上うえに行いって、日本にっぽんの歴史れきしを作つくりたいです」と話はなしています。

サムスン　画面がめんを開あけたり閉しめたりできるスマートフォン

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



韓国かんこくのサムスン電子でんしは２５日にちから、画面がめんを開あけたり閉しめたりすることができるスマートフォンを日本にっぽんで売うります。画面がめんを閉しめたときは４．６インチの大おおきさで、普通ふつうのスマートフォンです。開あけると７．３インチに大おおきくなって、タブレット端末たんまつのように使つかうことができます。
日本にっぽんでは携帯電話けいたいでんわのａｕの会社かいしゃがこのスマートフォンを売うります。値段ねだんは２４万まん円えんぐらいです。
開あけたり閉しめたりできるスマートフォンは、アメリカのマイクロソフトや中国ちゅうごくのファーウェイも売うる予定よていです。
サムスンなどはスマートフォンで動画どうがを見みる人ひとが多おおいので、大おおきい画面がめんは人気にんきが出でると考かんがえています。世界せかいではスマートフォンが前まえより売うれなくなっています。新あたらしい形かたちのスマートフォンが売うれるかどうか注目ちゅうもくが集あつまっています。

水みずが入はいった家いえは病気びょうきに気きをつけながら掃除そうじをしてください

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふう１９号ごうの雨あめで、川かわの水みずがあふれて、たくさんの家いえに水みずが入はいりました。
厚生労働省こうせいろうどうしょうは、家いえの中なかに水みずが入はいると細菌さいきんやかびが増ふえやすくなって、病気びょうきになる危険きけんが高たかくなるので、きれいに掃除そうじをすることが大切たいせつだと言いっています。そして、次つぎのことに気きをつけて掃除そうじをしてほしいと言いっています。
ドアや窓まどを開あけて、きれいな空気くうきが入はいるようにしてください。ほこりを吸すわないように、ゴーグルやマスクをしてください。掃除そうじが終おわったあとは、しっかり手てを洗あらってください。
掃除そうじをしている間あいだにけがをして、傷きずから細菌さいきんなどが入はいらないように、厚あつい手袋てぶくろをして、底そこが厚あつい靴くつをはいて、袖そでが長ながい服ふくを着きてください。泥どろを片かたづけたあとは、家いえの中なかを消毒しょうどくしてください。

テニスの大坂おおさかなおみ選手せんしゅが日本にっぽんの国籍こくせきを選えらぶ

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



テニスの大坂おおさかなおみ選手せんしゅは、今月こんげつ中国ちゅうごくであった大会たいかいで優勝ゆうしょうしました。今年ことし３回かい優勝ゆうしょうして、いま世界せかいランキングは３番ばんです。
大坂おおさか選手せんしゅのお父とうさんはハイチ出身しゅっしんで、お母かあさんは日本人にっぽんじんです。日本にっぽんで生うまれて、３歳さいからアメリカに住すんでいるため、日本にっぽんとアメリカの国籍こくせきを持もっています。法律ほうりつでは２２歳さいになる前まえに１つの国籍こくせきを選えらばなければなりません。大坂おおさか選手せんしゅはＮＨＫの質問しつもんに答こたえて、来年らいねんの東京とうきょうオリンピックに日本にっぽんの代表だいひょうで出でたいので、日本にっぽんの国籍こくせきを選えらんだと話はなしました。
大坂おおさか選手せんしゅは「日本にっぽんのために試合しあいをすることはとても特別とくべつで、すばらしい経験けいけんになります。東京とうきょうオリンピックでいい試合しあいをして、金きんメダルを取とりたいと思おもっています」と話はなしていました。
大坂おおさか選手せんしゅが来年らいねんのオリンピックで日本にっぽんの代表だいひょうになるためには、日本にっぽんのチームに入はいって国際こくさいテニス連盟れんめいが行おこなう国際こくさい大会たいかいにあと１回かい出でることが必要ひつようです。

政府せいふ「台風たいふうや交通こうつうの情報じょうほうを外国人がいこくじんにしっかり伝つたえる」

发表于 2019-10-11 更新于 2019-11-30 分类于 JapanEasyNews



大おおきくてとても強つよい台風たいふう１９号ごうが日本にっぽんの近ちかくに来きています。１２日にちは多おおくの飛行機ひこうきや新幹線しんかんせん、電車でんしゃが止とまります。
今いま日本にっぽんではラグビーのワールドカップを開ひらいているため、多おおくの外国人がいこくじんが来きています。このため政府せいふは、外国人がいこくじんに台風たいふうや交通こうつうの情報じょうほうをしっかり伝つたえるようにすると言いっています。
大おおきな空港くうこうでは、外国語がいこくごができる職員しょくいんが新あたらしい情報じょうほうをすぐに伝つたえます。
日本政府観光局にほんせいふかんこうきょくは２４時間じかん、英語えいごと中国語ちゅうごくご、韓国語かんこくごを使つかって電話でんわで相談そうだんできるようにしています。電話でんわ番号ばんごうは０５０ー３８１６ー２７８７です。
観光庁かんこうちょうが紹介しょうかいしている「Ｓａｆｅｔｙ　ｔｉｐｓ」というアプリやＳＮＳでも情報じょうほうを伝つたえます。観光かんこうの案内あんない所じょなどでは、スマートフォンなどを充電じゅうでんできるようにします。

６０年ねん前まえに１２００人にんが亡なくなった台風たいふうと同おなじぐらいの雨あめ

気象庁きしょうちょうは台風たいふう１９号ごうについて説明せつめいしました。台風たいふう１９号ごうは大おおきくてとても強つよいまま、１２日にちに東海地方とうかいちほうか関東地方かんとうちほうを通とおる可能性かのうせいが高たかくなりました。１２日にちから１３日にちは、西にし日本にほんから東北地方とうほくちほうまでの広ひろい範囲はんいでとても強つよい風かぜが吹ふきそうです。高たかい波なみにも気きをつけてください。
とても強つよい雨あめが広ひろい範囲はんいで降ふりそうです。１９５８年ねんに静岡県しずおかけんや関東地方かんとうちほうで１２００人にん以上いじょうが亡なくなった台風たいふうのように、川かわの水みずがあふれたり山やまなどが崩くずれたりする危険きけんがあります。

ノーベル化学賞かがくしょうはリチウムイオン電池でんちを作つくった吉野よしの彰あきらさん

发表于 2019-10-10 更新于 2019-11-30 分类于 JapanEasyNews



「リチウムイオン電池でんち」を作つくった吉野よしの彰あきらさんとアメリカの２人ふたりの研究けんきゅう者しゃが、今年ことしのノーベル賞しょうの化学かがく賞しょうをもらうことになりました。吉野よしのさんは大阪府おおさかふの出身しゅっしんで、７１歳さいです。ノーベル賞しょうをもらう日本人にっぽんじんは吉野よしのさんが２７人にん目めです。
吉野よしのさんは旭化成あさひかせいという会社かいしゃで、充電じゅうでんすることができる電池でんちの研究けんきゅうをしてきました。そして１９８５年ねん、リチウムイオン電池でんちを作つくることに成功せいこうしました。
リチウムイオン電池でんちは小ちいさくて、電気でんきをたくさんためることができるため、スマートフォンや持もって歩あるくことができるパソコンなどに使つかわれています。電気でんき自動車じどうしゃや、太陽たいようや風かぜで作つくった電気でんきをためるためにも使つかわれていて、地球ちきゅうの環境かんきょうにやさしい技術ぎじゅつだと言いわれています。
吉野よしのさんは「この電池でんちができてからＩＴの技術ぎじゅつが進すすんで、世界せかいが変かわりました。この電池でんちで環境かんきょうの問題もんだいを解決かいけつしてほしいと多おおくの人ひとが考かんがえているため、賞しょうをもらうことができたと思おもっています」と話はなしました。
