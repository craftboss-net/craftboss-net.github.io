若わかい人ひと　残業ざんぎょうの時間じかんが減へっている

发表于 2019-07-30 更新于 2019-11-30 分类于 JapanEasyNews



内閣府ないかくふは、スマートフォンや携帯電話けいたいでんわのデータを使つかって、東京都とうきょうとの２３の区くで何なん時じごろどこに何なん人にんぐらいいるか調しらべました。
前まえの年としに調しらべた人ひとの数かずと比くらべると、昼ひるは２．３％増ふえましたが、夜よる増ふえたのは１．２％だけでした。会社かいしゃを出でる時間じかんが早はやくなっていることがわかります。特とくに２０歳さいから３９歳さいまでの人ひとが前まえの年としより早はやく会社かいしゃを出でているようです。
政府せいふは、会社かいしゃで長ながい時間じかん働はたらいて自分じぶんの生活せいかつの時間じかんがなくなるような働はたらき方かたをやめようと言いっています。内閣府ないかくふは「特とくに若わかい人ひとが残業ざんぎょうの時間じかんが減へっているようです」と話はなしています。

ぬいぐるみの中なかに入はいって練習れんしゅうしていた男性だんせいが熱中症ねっちゅうしょうで亡なくなる

发表于 2019-07-30 更新于 2019-11-30 分类于 JapanEasyNews



２８日にちの夜よる、大阪府おおさかふ枚方市ひらかたしにある遊園地ゆうえんち「ひらかたパーク」で、ぬいぐるみの中なかに入はいってショーの練習れんしゅうをしていた２８歳さいの男性だんせいが倒たおれました。男性だんせいは病院びょういんに運はこばれましたが、亡なくなりました。
警察けいさつによると、この男性だんせいは午後ごご７時じ半はんごろから２５分ふんぐらい、ぬいぐるみの中なかに入はいって練習れんしゅうしていました。警察けいさつは、男性だんせいが熱中症ねっちゅうしょうで亡なくなったと考かんがえて調しらべています。枚方市ひらかたしは２８日にち、昼ひるの気温きおんが３５℃以上いじょうになっていました。
気象庁きしょうちょうは、これから１週間しゅうかんぐらい、いろいろな場所ばしょでとても暑あつい日ひが続つづくと言いっています。急きゅうに暑あつくなって体からだが暑あつさに慣なれていないので、よく水みずを飲のんだりエアコンを使つかったりして、熱中症ねっちゅうしょうにならないように気きをつけてほしいと言いっています。

ＪＡＸＡ　宇宙うちゅうで水みずをきれいにする機械きかいを作つくる

发表于 2019-07-29 更新于 2019-11-30 分类于 JapanEasyNews



宇宙うちゅうでは、水みずがとても大切たいせつです。国際こくさい宇宙うちゅうステーションでは、ＮＡＳＡが作つくった機械きかいで宇宙飛行士うちゅうひこうしの尿にょうなどをきれいにした水みずを使つかっています。
ＪＡＸＡは、汚よごれた水みずを入いれると８５％以上いじょうがきれいになって出でてくる新あたらしい機械きかいを作つくりました。この機械きかいは縦たてが５０ｃｍ、横よこが８０ｃｍ、幅はばが７０ｃｍぐらいで、フィルターや電気でんきを使つかって水みずをきれいにします。飲のむことができる水みずを１日にちに８００ｍＬまでつくることができます。
ＪＡＸＡはこの機械きかいを国際こくさい宇宙うちゅうステーションに打うち上あげて正ただしく動うごくかどうかテストをします。そして、２０２５年ねんごろまでにもっと大おおきい機械きかいを作つくって、新あたらしく作つくることを考かんがえている宇宙うちゅうステーションなどで使つかってほしいと言いっています。
ＪＡＸＡは「人ひとが月つきや火星かせいに行いくためには水みずをリサイクルする技術ぎじゅつが必要ひつようなので、日本にっぽんも役やくに立たちたいです」と話はなしていました。

日本にっぽんと韓国かんこくの関係かんけいが悪わるくなっているため、交流こうりゅうの行事ぎょうじが中止ちゅうしになっています。
岐阜県ぎふけん大垣市おおがきしに来くる予定よていだった韓国かんこくのチャンウォン市しの少年しょうねんサッカーのチームは「国くにと国くにの関係かんけいが難むずかしくなっているので行いきません」と連絡れんらくしてきました。今いままで大垣市おおがきしとチャンウォン市しの少年しょうねんサッカーのチームはそれぞれの市しに行いって交流こうりゅうしていました。
大垣市おおがきしの男おとこの子こは「一緒いっしょに遊あそびたいと思おもっていました。残念ざんねんな気持きもちでいっぱいです」と話はなしていました。韓国かんこくの子こどもたちもとても残念ざんねんがっていると連絡れんらくがありました。
韓国かんこくの市民しみんや子こどもなどと交流こうりゅうする行事ぎょうじは、北海道ほっかいどう旭川市あさひかわしや岡山県おかやまけん備前市びぜんしや長崎県ながさきけん対馬市つしましなどでも中止ちゅうしになっています。
日本にっぽんと韓国かんこくは、戦争せんそう中ちゅうに韓国かんこくの人ひとを日本にっぽんの会社かいしゃで働はたらかせたことについての問題もんだいや、日本にっぽんが韓国かんこくに輸出ゆしゅつするときの決きまりを厳きびしくしたことなどで、関係かんけいが悪わるくなっています。

ＪＴＢなど３つの旅行りょこうの会社かいしゃが、来年らいねんの東京とうきょうオリンピックのチケットとホテルをセットにしたツアーの抽ちゅうせんの受うけ付つけを始はじめました。
ＪＴＢのツアーは全部ぜんぶで３７あります。いちばん高たかいツアーは４５０万まん円えんで、開会かいかい式しきと閉会へいかい式しき、そして柔道じゅうどうや陸上競技りくじょうきょうぎなどの試合しあいをオリンピックの間あいだ、毎日まいにち見みることができます。

发表于 2019-07-26 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねん、東京とうきょうオリンピックを行おこなうときはとても暑あつくなりそうです。
東京とうきょうでは、暑あついとき体からだを冷つめたくするために、体からだにつけて使つかう商品しょうひんの展示てんじ会かいがありました。９０ぐらいの会社かいしゃが商品しょうひんを紹介しょうかいしました。
東京とうきょうの会社かいしゃは、ヘッドホンのような形かたちの物ものを首くびにかけて、体からだを冷つめたくする商品しょうひんを紹介しょうかいしました。中なかに入はいっている半導体はんどうたいの表面ひょうめんが、外そとの気温きおんより１０°Cから１５°Cぐらい冷つめたくなります。この商品しょうひんは６月がつから売うっています。
宮城県みやぎけんの会社かいしゃは、体からだを冷つめたくすることができる服ふくを紹介しょうかいしました。服ふくの中なかにはチューブがあって、リュックサックの中なかの氷こおりが入はいった水みずを送おくるようになっています。会社かいしゃは、東京とうきょうオリンピックの警備けいびの人ひとや外そとで働はたらく人ひとに使つかってほしいと言いっています。
