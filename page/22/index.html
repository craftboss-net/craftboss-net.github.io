发表于 2019-08-28 更新于 2019-11-30 分类于 JapanEasyNews



今月こんげつ２３日にちに大阪市おおさかしと関西空港かんさいくうこうの間あいだを走はしった南海電鉄なんかいでんてつの特急電車とっきゅうでんしゃ「ラピート」の車両しゃりょうの下したの部分ぶぶんに、長ながさ１４ｃｍぐらいのひびが見みつかりました。

アマゾンで続つづく火事かじを消けすために日本にっぽんなどが協力きょうりょくする

发表于 2019-08-27 更新于 2019-11-30 分类于 JapanEasyNews



南みなみアメリカにあるアマゾンの森もりで、火事かじが続つづいています。アマゾンは世界せかいでいちばん大おおきい熱帯ねったいの森もりで、地球ちきゅうの酸素さんその２０％ぐらいがアマゾンから出でていると言いわれています。
アマゾンの６０％以上いじょうがあるブラジルでは、今年ことし１月がつから７万まん９０００件けんの火事かじが起おこっています。今年ことしは雨あめが少すくなくて、去年きょねんより８０％以上いじょう火事かじが多おおくなっています。
ブラジルの政府せいふは、火事かじを消けすために軍ぐんから４万まん４０００人にんを出だしました。飛行機ひこうきを使つかって空そらからも水みずをまいていますが、すぐに消けすことは難むずかしそうです。
フランスで７つの先進国せんしんこくの大統領だいとうりょうや首相しゅしょうが集あつまって開ひらいたＧ７の会議かいぎで、アマゾンの火事かじで困こまっている国くにを助たすけるために２１億おく円えんを出だすことを決きめました。日本にっぽんも２６日にち、アマゾンの火事かじを消けすために協力きょうりょくしたいとブラジルに伝つたえました。

阿寒湖あかんこにマリモがどのくらいあるか調査ちょうさをする

发表于 2019-08-27 更新于 2019-11-30 分类于 JapanEasyNews



マリモはボールのような形かたちをした藻もで、北海道ほっかいどう釧路市くしろしにある阿寒湖あかんこのマリモは国くにの特別とくべつ天然記念物てんねんきねんぶつです。阿寒湖あかんこでは１０年ねんぐらい前まえから湖みずうみの中なかの草くさが急きゅうに増ふえて、マリモが育そだつ場所ばしょが狭せまくなっています。このため、マリモが減へっているかもしれないと心配しんぱいする人ひともいます。
２５日にち、釧路市くしろしは阿寒湖あかんこにマリモがどのくらいあるか、調査ちょうさを始はじめました。前まえの調査ちょうさは２２年ねん前まえでした。調査ちょうさでは、湖みずうみの中なかからマリモを出だして大おおきさや重おもさを調しらべています。２０cm以上いじょうの大おおきさのマリモもありました。
釧路市くしろしは「マリモが減へらないように、湖みずうみの草くさをどうするか考かんがえたいと思おもいます」と話はなしています。

東京とうきょうパラリンピックのメダル　触さわってわかるデザイン

東京とうきょうパラリンピックが始はじまるまであと１年ねんになった２５日にち、メダルの発表はっぴょうがありました。
パラリンピックのメダルは、表おもても裏うらも扇子せんすのデザインになっています。「みんなの心こころを一ひとつにして、世界せかいに新あたらしい風かぜを送おくる」という意味いみです。
表おもてには「ＴＯＫＹＯ２０２０」と書かいてあって、目めの見みえない人ひとがわかるように点字てんじもあります。横よこには丸まるいくぼみがあって、くぼみの数かずでどのメダルかわかります。裏うらには岩いわ、花はな、木き、葉は、水みずの５つの自然しぜんもデザインされていて、触さわってわかるようになっています。
メダルをデザインした人ひとは「日本にっぽんらしいデザインにしたいと思おもって、扇子せんすにしました」と話はなしています。
