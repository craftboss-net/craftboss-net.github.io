羽田空港はねだくうこうで自動じどう運転うんてんのバスが走はしる実験じっけん

发表于 2019-01-23 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんへ旅行りょこうに来くる外国人がいこくじんは５年ねんで３倍ばいぐらいに増ふえました。しかし、空港くうこうで働はたらく人ひとは足たりなくなっています。このため、航空こうくう会社がいしゃの全日空ぜんにっくうなどは、人ひとが運転うんてんしなくても自動じどうで走はしるバスを空港くうこうの中なかで使つかおうと考かんがえて、羽田空港はねだくうこうで実験じっけんを行おこなっています。
実験じっけんは、空港くうこうのビルと飛行機ひこうきがとまっている場所ばしょとの間あいだの６００mで行おこないました。安全あんぜんのため、運転うんてん席せきには人ひとが乗のっています。バスはＧＰＳの情報じょうほうを使つかったり、走はしるコースの下したに入はいっている磁石じしゃくをセンサーでチェックしたりしながら、時速じそく２０ｋｍで走はしりました。
バスは飛行機ひこうきのエンジンから出でるガスが当あたらないように、決きまった場所ばしょで一度いちど止とまることになっています。実験じっけんでも、バスは止とまったあと、周まわりが安全あんぜんかどうか調しらべて、また走はしり始はじめました。
全日空ぜんにっくうは、来年らいねんから一部いちぶの空港くうこうで自動じどう運転うんてんのバスを使つかいたいと考かんがえています。

发表于 2019-01-21 更新于 2019-11-30 分类于 JapanEasyNews


先月せんげつ、世界せかいで有名ゆうめいな芸術家げいじゅつかのバンクシーの絵えが防潮扉ぼうちょうとびらにかいてあると東京都とうきょうとに連絡れんらくが入はいりました。防潮扉ぼうちょうとびらは、台風たいふうなどのときに海うみの水みずが陸りくに入はいらないように閉しめるドアです。
絵えは東京都とうきょうと港区みなとくの防潮扉ぼうちょうとびらにかいてあって、縦たてと横よこが２０ｃｍぐらいの大おおきさです。傘かさを持もっているねずみがかいてあって、バンクシーが前まえにかいた絵えに似にていると言いわれています。

バンクシーは社会しゃかいをからかうような絵えを建物たてものなどにかいていて、誰だれも本当ほんとうの名前なまえなどを知しりません。去年きょねん１０月がつ、オークションでバンクシーの絵えが１億おく５０００万まん円えんに決きまると、額がくに隠かくれていたシュレッダーで絵えの半分はんぶんが細ほそく切きられて、ニュースになりました。
