地震じしんの研究けんきゅう　海うみの底そこを３２６０ｍまで掘ほることができた

发表于 2019-01-16 更新于 2019-11-30 分类于 JapanEasyNews


海洋研究開発機構かいようけんきゅうかいはつきこうなどの研究けんきゅうグループは、１２年ねん前まえから「ちきゅう」という船ふねを使つかって、和歌山県わかやまけんの海うみの底そこを掘ほっています。海うみの底そこの岩いわなどを調しらべて、地震じしんの研究けんきゅうをするためです。
グループは、先月せんげついままででいちばん深ふかい、海うみの底そこから３２６０ｍまで掘ほることができたと発表はっぴょうしました。
グループは海うみの底そこから５２００ｍ下したまで掘ほる予定よていです。５２００ｍ下したには、プレートと呼よばれる大おおきな岩いわがあって、そこで大おおきな地震じしんが起おこる 可能性かのうせいがあると言いわれています。グループは岩いわや石いしを取とって、地震じしんがいつ起おこるかなどの研究けんきゅうに使つかいたいと考かんがえています。

横綱よこづなの稀勢きせの里さとが力士りきしをやめる

发表于 2019-01-16 更新于 2019-11-30 分类于 JapanEasyNews


１６日にち、大相撲おおずもうの横綱よこづなの稀勢きせの里さとが力士りきしをやめました。稀勢きせの里さとは「今いままで本当ほんとうにいろいろな人ひとに助たすけてもらいました。感謝かんしゃの気持きもちでいっぱいです」と話はなしました。
稀勢きせの里さとは、２年ねん前まえに優勝ゆうしょうして横綱よこづなになったあと、次つぎの大会たいかいでも優勝ゆうしょうしました。しかし、けがで長ながく休やすんだり、負まけたりすることが多おおくなりました。今月こんげつ１３日にちに始はじまった大会たいかいでは３日みっか続つづけて負まけていました。
稀勢きせの里さとは、けがをしていても相撲すもうを続つづけたことについて「応援おうえんしてくれる皆みなさんのために頑張がんばってきました。やめることになって申もうし訳わけないと思おもいます」と話はなしました。
