映画えいが「この世界せかいの片隅かたすみに」の上映じょうえいが１０００日にち以上いじょう続つづく

发表于 2019-08-19 更新于 2019-11-30 分类于 JapanEasyNews



アニメーション映画えいがの「この世界せかいの片隅かたすみに」は、太平洋戦争たいへいようせんそうのときの広島ひろしまで、苦くるしいことがあっても明あかるく生いきる女性じょせいが主人公しゅじんこうです。
この映画えいがは３年ねん前まえの１１月がつから毎日まいにち上映じょうえいが続つづいていて、上映じょうえいした日ひは今月こんげつで１０００日にち以上いじょうになりました。映画えいがの上映じょうえいが１０００日にち以上いじょう続つづくのは珍めずらしいことです。
監督かんとくの片渕かたぶち須直すなおさんは、太平洋戦争たいへいようせんそうが終おわった日ひの８月がつ１５日にちに映画館えいがかんであいさつをしました。片渕かたぶちさんは「戦争せんそうが終おわるまでの最後さいごの１年ねんで爆弾ばくだんなどで２５０万まん人にんぐらいが亡なくなりました。この映画えいがで、みなさんに戦争せんそうについていろいろなことを感かんじてほしいです」と話はなしていました。
映画えいがを見みた１８歳さいの女性じょせいは「戦争せんそうのあとに生うまれた私わたしたちも、戦争せんそうのことを将来しょうらいに伝つたえていかなければならないと思おもいました」と話はなしていました。

プラスチックのごみで海うみが汚よごれている問題もんだいについて、世界せかいの国くにが調しらべています。しかし、特とくに海うみの深ふかい所ところがどのくらい汚よごれているかについては、まだよくわかっていません。
日本にっぽんの海洋研究開発機構かいようけんきゅうかいはつきこうは、プラスチックのごみで海うみの深ふかい所ところがどのくらい汚よごれているか、初はじめて詳くわしく調しらべることにしました。２８日にちから３週間しゅうかんぐらい、相模湾さがみわんや房総半島ぼうそうはんとうなどの海うみで深ふかさ１２００ｍから９２００ｍの海うみの底そこにたまっている物ものを集あつめます。
そして、生いき物ものに害がいがあると言いわれている５ｍｍ以下いかの細こまかいプラスチックのごみについて調しらべます。いつからどのくらいたまっているか、海うみの深ふかい所ところにいる生いき物もののおなかの中なかにないかどうかなど調しらべます。

日本にっぽんのウイスキー５４本ほんがオークションで約やく１億おく円えんになる

发表于 2019-08-19 更新于 2019-11-30 分类于 JapanEasyNews



香港ほんこんで１６日にちにオークションがあって、埼玉県さいたまけんでつくられたウイスキー５４本ほんのセットが約やく１億おく円えんになりました。オークションに出でた日本にっぽんのウイスキーの中なかでは、今いままででいちばん高たかい値段ねだんになりました。
「イチローズモルト」というウイスキーで、１９８０年ねん代だいから２０００年ねんにつくられました。ウイスキーが入はいっている木きのたるが違ちがうと味あじや香かおりも違ちがうため、５４種類しゅるいの商品しょうひんにして売うっていました。
日本にっぽんのウイスキーは、おいしいと思おもう人ひとが世界せかいで多おおくなっていて、高たかい値段ねだんで売うれるようになっています。
このウイスキーを売うった会社かいしゃの人ひとは「うれしいですが、飲のんで楽たのしんでほしいので、こんなに高たかい値段ねだんだと本当ほんとうに飲のんでくれるか心配しんぱいです」と話はなしていました。

トゥーンベリさんは、いろいろな国くにの大統領だいとうりょうや首相しゅしょうなどにメッセージを伝つたえるため、来月らいげつアメリカのニューヨークで開ひらく国連こくれんの会議かいぎに出席しゅっせきします。

来年らいねんの東京とうきょうパラリンピックのチケットを買かうための申もうし込こみが８月がつ２２日にちに始はじまります。東京とうきょうパラリンピックのウェブサイトで、１人ひとり６０枚まいまで申もうし込こむことができます。

消費税しょうひぜいが上あがっても客きゃくが減へらないように料理りょうりの値段ねだんを下さげる

发表于 2019-08-15 更新于 2019-11-30 分类于 JapanEasyNews



１０月がつに消費税しょうひぜいが上あがる前まえに、料理りょうりの値段ねだんを下さげるレストランや食堂しょくどうが増ふえています。
「リンガーハット」という店みせは今月こんげつから、ギョーザとごはん、スープなどを３７０円えんで食たべることができるセットを始はじめました。多おおくの人ひとが頼たのむちゃんぽんとギョーザのセットも１０円えん安やすくしました。この店みせは去年きょねん８月がつに料理りょうりの値段ねだんを上あげましたが、それから客きゃくが減へっています。このため、消費税しょうひぜいが上あがっても客きゃくが減へらないように、料理りょうりの値段ねだんを下さげました。
「幸楽苑こうらくえん」というラーメンの店みせは、全部ぜんぶの料理りょうりの値段ねだんを下さげて、消費税しょうひぜいが上あがっても客きゃくが払はらうお金かねが今いまと変かわらないようにします。「ケンタッキーフライドチキン」の店みせも、値段ねだんが変かわらないようにします。
