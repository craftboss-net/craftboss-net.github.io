広島ひろしま　原爆げんばくで亡なくなった６人にんの家族かぞくを写真しゃしんで伝つたえる本ほん

发表于 2019-07-16 更新于 2019-11-30 分类于 JapanEasyNews



今月こんげつ売うり出だした「ヒロシマ消きえたかぞく」は、原爆げんばくで亡なくなった家族かぞくのことを写真しゃしんで伝つたえる本ほんです。
広島市ひろしましの鈴木すずき六郎ろくろうさんは写真しゃしんを撮とることが趣味しゅみで、妻つまと４人にんの子こどもと生活せいかつしていました。しかし、１９４５年ねん８月がつ６日むいかに広島ひろしまに原爆げんばくが落おとされて、家族かぞくがみんな亡なくなりました。この本ほんは、六郎ろくろうさんのおいの鈴木すずき恒明つねあきさんが持もっていた写真しゃしんを使つかって、家族かぞくがどのように生いきていたかを伝つたえています。
本ほんを作つくった作家さっかの指田さしだ和かずさんは１１日にち、広島市ひろしましでこの本ほんを紹介しょうかいする会かいを開ひらいて、恒昭つねあきさんも来きました。恒昭つねあきさんは、小学校しょうがっこうの帰かえりに六郎ろくろうさんの店みせに寄よって、六郎ろくろうさんの子こどもたちと遊あそんだことや、六郎ろくろうさんが子こどもたちの写真しゃしんをよく撮とっていたことなどを話はなしました。
指田さしださんは「子こどもたちの笑顔えがおの写真しゃしんを見みると、温あたたかい家族かぞくだったことがわかります。戦争せんそうや原爆げんばくでたくさんの家族かぞくが亡なくなったことを忘わすれないでほしいです」と言いっていました。

日本にっぽんが輸出ゆしゅつの決きまりを厳きびしくして韓国かんこくとの関係かんけいが悪わるくなる

日本にっぽんの政府せいふは今月こんげつ、半導体はんどうたいなどを作つくるのに必要ひつような３つの材料ざいりょうを韓国かんこくに輸出ゆしゅつするときの決きまりを厳きびしくしました。この材料ざいりょうは兵器へいきを作つくるときにも使つかうことができますが、韓国かんこくの貿易ぼうえきに問題もんだいがあるため厳きびしくしたと日本にっぽんは言いっています。
しかし韓国かんこくは、貿易ぼうえきについてしっかり調しらべているので、厳きびしくするのをやめるように言いっています。半導体はんどうたいは韓国かんこくの大事だいじな産業さんぎょうです。３つのうち２つの材料ざいりょうは９０％以上いじょうを日本にっぽんから輸入ゆにゅうしています。
韓国かんこくのムン大統領だいとうりょうは１５日にち、「日本にっぽんは歴史れきしと経済けいざいの問題もんだいを一緒いっしょにしていてよくない」と言いいました。韓国かんこくでは、第だい２次世界大戦じせかいたいせんのときに日本にっぽんの会社かいしゃに働はたらかされたと言いっている人ひとたちが、賠償ばいしょうのお金かねを払はらうように日本にっぽんの会社かいしゃに言いっています。日本にっぽんの政府せいふは、この問題もんだいを解決かいけつするために話はなし合あいたいと言いっていますが、韓国かんこくは答こたえていません。
日本にっぽんの政府せいふは１６日にち、ムン大統領だいとうりょうが言いっていることは正ただしくないと強つよく言いいました。

雨あめや曇くもりの日ひが多おおい　野菜やさいの値段ねだんが高たかくなる

发表于 2019-07-16 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの太平洋たいへいよう側がわなどでは６月がつの終おわりごろから、曇くもりや雨あめの日ひが続つづいています。太陽たいようが出でている時間じかんが短みじかくて、いつもの年としより気温きおんが低ひくい日ひが多おおくなっています。
東京とうきょうでは気温きおんが２５℃まで上あがらなかった日ひが、今月こんげつ１２日にちまで８日ようか続つづきました。太陽たいようが出でていた時間じかんが３時間じかんより短みじかかった日ひも、１４日じゅうよっかまで１８日にち続つづきました。こんなに長ながく続つづいた年としは、１９８８年ねんからあとはありませんでした。
特とくに関東地方かんとうちほうでは太陽たいようが出でる時間じかんが短みじかくなっていて、野菜やさいが育そだつのが遅おくれています。東京都とうきょうとの市場いちばでは、きゅうりの値段ねだんがいつもの年としより５０％以上いじょう高たかくなっています。なすも２０％、ピーマンも１０％ぐらい高たかくなっています。
気象庁きしょうちょうは、これから１週間しゅうかんぐらいは関東地方かんとうちほうから九州きゅうしゅうまでで、曇くもりや雨あめの日ひが多おおくなりそうだと言いっています。

「ニホンライチョウ」という鳥とりは国くにの特別とくべつ天然記念物てんねんきねんぶつですが、とても少すくなくなっています。長野県ながのけんの中央ちゅうおうアルプスの山やまでは５０年ねん以上いじょう前まえにいなくなったと考かんがえられていました。しかし去年きょねん、雌めすが１羽わいることがわかりました。
環境省かんきょうしょうは、この雌めすに別べつの山やまから持もってきたニホンライチョウの卵たまごを温あたためさせました。そして今月こんげつ１日ついたちに５羽わの赤あかちゃんを見みつけました。しかし１１日にちに調しらべに行いくと、雌めすは見みつかりましたが、赤あかちゃんは１羽わも見みつかりませんでした。

「人工衛星じんこうえいせいが増ふえて星ほしを調しらべることが難むずかしくなりそうだ」

发表于 2019-07-12 更新于 2019-11-30 分类于 JapanEasyNews



通信つうしんなどに使つかう人工衛星じんこうえいせいは、これから急きゅうに増ふえそうです。アメリカの会社かいしゃは新あたらしい通信つうしんの仕事しごとをするために、１万まん２０００の人工衛星じんこうえいせいを打うち上あげようとしています。
国立天文台こくりつてんもんだいは、人工衛星じんこうえいせいが増ふえると、星ほしを調しらべることが難むずかしくなるかもしれないと考かんがえています。そして、人工衛星じんこうえいせいを使つかう会社かいしゃの人ひとたちと協力きょうりょくして、この問題もんだいを解決かいけつしたいと言いいました。
アメリカの天文台てんもんだいが撮とった写真しゃしんには、白しろい光ひかりの線せんがたくさん写うつっていて、星ほしを調しらべることができなくなっているものもあります。白しろい線せんは、アメリカの会社かいしゃが打うち上あげた６０の人工衛星じんこうえいせいに、太陽たいようの光ひかりが反射はんしゃしてできました。
国立天文台こくりつてんもんだいは、人工衛星じんこうえいせいが出だす電波でんぱも、星ほしを調しらべるときに邪魔じゃまになるかもしれないと心配しんぱいしています。そして、「人工衛星じんこうえいせいには光ひかりを反射はんしゃしにくい材料ざいりょうを使つかうことが必要ひつようです。星ほしを調しらべるときに使つかう電波でんぱと違ちがう周波数しゅうはすうの電波でんぱを人工衛星じんこうえいせいに使つかってもらう必要ひつようもあります」と話はなしています。
