暑あつい日ひが続つづいて病院びょういんに運はこばれた人ひとが１週間しゅうかんに２０００人にん以上いじょう

发表于 2019-05-29 更新于 2019-11-30 分类于 JapanEasyNews



総務省消防庁そうむしょうしょうぼうちょうによると、５月がつ２０日はつかから２６日にちまでの１週間しゅうかんに熱中症ねっちゅうしょうのようだと言いって病院びょういんに運はこばれた人ひとは２０００人にん以上いじょういました。その前まえの週しゅうの４倍ばいぐらいです。

６月がつ１日ついたちから１０日とおかもいつもの年としよりずっと暑あつくなりそうです。総務省消防庁そうむしょうしょうぼうちょうは「体からだが暑あつさにまだ慣なれていません。冷房れいぼうを使つかったり、よく水みずを飲のんだりして、熱中症ねっちゅうしょうにならないようにしてください」と言いっています。

道後温泉どうごおんせん「工事こうじ中ちゅうでもアニメと温泉おんせんを楽たのしんで」

发表于 2019-05-28 更新于 2019-11-30 分类于 JapanEasyNews



愛媛県えひめけん松山市まつやましの道後温泉どうごおんせんは、長ながい歴史れきしがある有名ゆうめいな温泉おんせんです。１２０年ねん以上いじょう前まえに建たてられた道後温泉本館どうごおんせんほんかんでは、近ちかくに住すんでいる人ひとも旅行りょこうに来きた人ひとも温泉おんせんを楽たのしむことができます。
道後温泉本館どうごおんせんほんかんは、今いま建物たてものを直なおす工事こうじをしています。工事こうじ中ちゅうでもみんなに楽たのしんでもらうため、アニメを作つくりました。手塚治虫てづかおさむさんがかいた漫画まんがを使つかった「火ひの鳥とり“道後温泉どうごおんせん編へん”」というアニメです。
２４日にじゅうよっか、アニメを発表はっぴょうしました。温泉おんせんに入はいった神かみ様さまたちの病気びょうきが治なおった話はなしや、聖徳太子しょうとくたいしも温泉おんせんで元気げんきになったという話はなしなどで、手塚てづかさんの漫画まんがの「火ひの鳥とり」も出でてきます。建物たてもののそばのテレビで毎日まいにち見みることができて、インターネットでも見みることができます。
温泉おんせんの事務所じむしょの人ひとは「工事こうじ中ちゅうでも温泉おんせんに入はいることができます。アニメで歴史れきしを知しって、たくさんの人ひとに来きてもらいたいです」と話はなしています。
