トヨタ　日本にっぽんの会社かいしゃで初はじめて売うり上あげが３０兆ちょう円えんより多おおくなる

发表于 2019-05-09 更新于 2019-11-30 分类于 JapanEasyNews



トヨタ自動車じどうしゃは、今年ことし３月がつまでの１年ねん間かんの売うり上あげが前まえの年としより２．９％増ふえて、３０兆ちょう２２５６億おく円えんだったと発表はっぴょうしました。利益りえきも前まえの年としより２．８％増ふえて、２兆ちょう４６７５億おく円えんでした。トヨタは２年ねん続つづけて売うり上あげも利益りえきも増ふえました。
トヨタは日本にっぽんの会社かいしゃで初はじめて、１年ねん間かんの売うり上あげが３０兆ちょう円えんより多おおくなりました。
売うった車くるまの数かずは、アメリカや日本にっぽんでは少すくなくなりましたが、アジアでは９％増ふえました。タイではピックアップトラック、中国ちゅうごくでは高級こうきゅうな車くるまがよく売うれました。ヨーロッパでも、ガソリンと電気でんきで走はしるハイブリッド車しゃがよく売うれて、２％増ふえました。
豊田とよだ章男あきお社長しゃちょうは、「お客様きゃくさまやトヨタの車くるまを売うる店みせなど、みんなが８０年ねん頑張がんばってきた結果けっかです」と話はなしました。

「女性じょせいにも迷まよわないでＡＥＤを使つかってほしい」

发表于 2019-05-08 更新于 2019-11-30 分类于 JapanEasyNews



「ＡＥＤ」は、心臓しんぞうが急きゅうに止とまった人ひとの体からだにパッドを貼はって、電気でんきで心臓しんぞうを動うごかす機械きかいです。
京都大学きょうとだいがくなどのグループは、学校がっこうで子こどもの心臓しんぞうが急きゅうに止とまったとき、救急車きゅうきゅうしゃが来くる前まえにＡＥＤを使つかったかどうか調しらべました。２００８年ねんから２０１５年ねんの間あいだに学校がっこうで急きゅうに心臓しんぞうが止とまった２３２人にんの子こどもについて調しらべました。
小学生しょうがくせいと中学生ちゅうがくせいでは、ＡＥＤを使つかった割合わりあいは男子だんしと女子じょしでほとんど同おなじでした。高校生こうこうせいでは、男子だんしは８３％以上いじょうでしたが、女子じょしは５５％だけでした。グループは、近ちかくにいる人ひとが女子じょし高校生こうこうせいの服ふくを脱ぬがせることを迷まようためだと考かんがえています。
グループの人ひとは「服ふくを全部ぜんぶ脱ぬがせなくても、下着したぎをずらしてＡＥＤのパッドを貼はることができます。貼はったあとは、体からだの上うえに服ふくなどをかけても大丈夫だいじょうぶです。命いのちを助たすけるために、女性じょせいにも迷まよわないで使つかってほしいです」と話はなしています。

「動物どうぶつや植物しょくぶつ１００万まん種類しゅるいがいなくなる心配しんぱいがある」

发表于 2019-05-08 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんなど１３０以上いじょうの国くにの科学かがくの専門家せんもんかが集あつまるＩＰＢＥＳによると、今いままでの５００年ねんの間あいだ、人ひとが原因げんいんで６８０種類しゅるいの脊椎動物せきついどうぶつがいなくなりました。ガラパゴス諸島しょとうにいた亀かめ「ピンタゾウガメ」や、インド洋ようのモーリシャスにいた大おおきな鳥とり「ドードー」などです。
ＩＰＢＥＳは、人ひとが農業のうぎょうや漁業ぎょぎょうのために陸りくや海うみの環境かんきょうを変かえたこと、動物どうぶつをたくさんとったこと、地球ちきゅうの気温きおんが上あがったことなどが原因げんいんだと言いっています。
ＩＰＢＥＳによると、今いまも動物どうぶつや植物しょくぶつ１００万まん種類しゅるいがいなくなる心配しんぱいがあります。この動物どうぶつや植物しょくぶつの多おおくが、これから何なん十じゅう年ねんかの間あいだにいなくなるかもしれないため、ＩＰＢＥＳは世界せかいで対策たいさくが必要ひつようだと言いっています。

ＮＰＯの「チャイルドライン支援しえんセンター」は、子こどもからの相談そうだんを電話でんわやインターネットで受うけています。センターによると、休やすみが終おわる少すこし前まえから「学校がっこうへ行いきたくない」とか「学校がっこうの勉強べんきょうがわかるか心配しんぱいだ」という相談そうだんが増ふえています。

「日本語にほんご学校がっこうの基準きじゅんを厳きびしくする」

留学生りゅうがくせいが日本語にほんごを勉強べんきょうするための日本語にほんご学校がっこうは日本にっぽんに７５０ぐらいあります。しかし、国くにによると日本語にほんごの教育きょういくが十分じゅうぶんではない学校がっこうもあります。このため国くには、留学生りゅうがくせいを入いれてもいい学校がっこうかどうかを決きめる基準きじゅんを今いまより厳きびしくしようと考かんがえています。
今いまの基準きじゅんでは、学生がくせいみんなの出席しゅっせき率りつの平均へいきんが１か月げつで５０％より低ひくい学校がっこうは、新あたらしい留学生りゅうがくせいを入いれることができません。新あたらしい基準きじゅんの案あんでは、半年はんとしで７０％より低ひくい学校がっこうにします。
日本語にほんごのレベルも、その学校がっこうの学生がくせいの７０％以上いじょうが、大学だいがくなどへの入学にゅうがくや生活せいかつの会話かいわができるようにします。３年ねん続つづけてこのレベルにならなかった学校がっこうは、新あたらしい留学生りゅうがくせいを入いれることができません。
国くにはこの案あんについてみんなの意見いけんを聞きいたあと、新あたらしい基準きじゅんを決きめる予定よていです。
