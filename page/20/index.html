社員しゃいんになることが難むずかしかった人ひとたちを支援しえんする

发表于 2019-09-03 更新于 2019-11-30 分类于 JapanEasyNews



３５歳さいから４５歳さいぐらいの人ひとたちは学校がっこうを出でたとき景気けいきが悪わるくて、社員しゃいんになることができなかった人ひとがたくさんいます。国くには、アルバイトなどを続つづけているこの年齢ねんれいの人ひとたちについて、３年ねんで３０万まん人にんを社員しゃいんにしたいと考かんがえています。

ハローワークには、この年齢ねんれいの人ひとたちが相談そうだんできる場所ばしょも作つくって、社員しゃいんになることを支援しえんすることにしています。

体からだに傷きずがある４歳さいの子こが亡なくなる　何なん度ども１人ひとりで歩あるいていた

发表于 2019-09-03 更新于 2019-11-30 分类于 JapanEasyNews



鹿児島県かごしまけん出水市いずみしで８月がつ２８日にち、４歳さいの大塚おおつか璃愛來りあらちゃんが風呂ふろで溺おぼれたという連絡れんらくがありました。女おんなの子こは病院びょういんに運はこばれましたが、亡なくなりました。女おんなの子こは体からだのいろいろな場所ばしょに傷きずがありました。警察けいさつは前まえの日ひに女おんなの子こを殴なぐった疑うたがいで、一緒いっしょに住すんでいた２１歳さいの男おとこを逮捕たいほしました。
璃愛來りあらちゃんは、今年ことし７月がつまで薩摩川内市さつませんだいしに住すんでいました。去年きょねん１２月がつから今年ことし２月がつには、この子こが雨あめの日ひに靴くつや下着したぎをはかないで１人ひとりで歩あるいていて、近ちかくの人ひとが短みじかい間あいだ預あずかっていたことが何なん度どもありました。
今年ことし３月がつから４月がつの間あいだには、警察けいさつが夜よるに１人ひとりで歩あるいているこの子こを見みつけたことが４回かいありました。警察けいさつは児童相談所じどうそうだんじょにこの子こを預あずかって世話せわをしたほうがいいと伝つたえました。しかし児童相談所じどうそうだんじょは、この子こが母親ははおやを嫌いやがっていなかったので、問題もんだいないと考かんがえて預あずからなかったと言いっています。

发表于 2019-09-03 更新于 2019-11-30 分类于 JapanEasyNews



モンゴルから来きて大相撲おおずもうの横綱よこづなになった白鵬はくほうが、日本にっぽんの国籍こくせきを取とりました。
白鵬はくほうは３４歳さいです。１５歳さいだった２０００年ねんに日本にっぽんで相撲すもうを始はじめて、２００７年ねんに横綱よこづなになりました。優勝ゆうしょうは４２回かい、勝かった試合しあいの数かずは１１３２回かいで、どちらも今いままでの大相撲おおずもうの中なかでいちばん多おおい数かずです。
白鵬はくほうは、将来しょうらい力士りきしをやめたあと、若わかい力士りきしを育そだてる親方おやかたになりたいと考かんがえています。親方おやかたになるためには日本にっぽんの国籍こくせきが必要ひつようです。
白鵬はくほうは「自分じぶんが生うまれたモンゴルを愛あいしているから、日本にっぽんも愛あいすることができます。将来しょうらい、強つよい力士りきしを育そだてたいです」と話はなしています。

北海道ほっかいどう地震じしん　「助たすかった人ひとのほとんどは２階かいにいた」

发表于 2019-09-03 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねん９月がつ６日むいかに北海道ほっかいどうで大おおきく揺ゆれる地震じしんがありました。震度しんど７だった厚真町あつまちょうでは、山やまから流ながれてきた泥どろが家いえに入はいって、２１の家庭かていで３６人にんが亡なくなりました。
家いえに泥どろが入はいって家族かぞくが亡なくなりましたが、自分じぶんは助たすかったという人ひとが１６人にんいました。ＮＨＫが助たすかった人ひとや親戚しんせきに話はなしを聞きくと、１４人にんは２階かいで寝ねていました。
５７歳さいの男性だんせいは２階かいにいて助たすかりましたが、１階かいにいた７６歳さいの母親ははおやは亡なくなりました。男性だんせいは「家いえに入はいってきた泥どろは、１階かいだけを流ながして、２階かいは残のこりました。母親ははおやも２階かいで寝ねていたら、助たすかったと思おもいます」と話はなしています。
雨あめがたくさん降ふって外そとに逃にげることができなくなったときは、建物たてものの２階かい以上いじょうにいたほうがいいと言いわれています。専門家せんもんかは、地震じしんのときも２階かいにいたほうが安全あんぜんだと言いっています。

スーパーコンピューター「京けい」のスイッチを切きる式しきを行おこなう

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



神戸市こうべしにある理化学りかがく研究所けんきゅうしょでは、新あたらしいスーパーコンピューター「富岳ふがく」を使つかうことになりました。８月がつ３０日にち、今いままで使つかっていた「京けい」のスイッチを切きる式しきがありました。
理化学りかがく研究所けんきゅうしょの人ひとは「少すこし寂さびしいですが、『富岳ふがく』も『京けい』と同おなじように役やくに立たつようにしたいと思おもいます」とあいさつしました。そして理化学りかがく研究所けんきゅうしょの人ひとなどが、「京けい」が置おいてある部屋へやで４つのスイッチを１つずつ切きっていきました。
国くにの計画けいかくでつくった「京けい」は２０１２年ねんにできました。計算けいさんの速はやさで世界せかいで１番ばんになったことがあります。ビッグデータを分析ぶんせきする能力のうりょくでは、何なん回かいも世界せかいで１番ばんになりました。地震じしんの被害ひがいの予測よそくや薬くすりの研究けんきゅうなどにとても役やくに立たちました。
「京けい」のあとに使つかう「富岳ふがく」は、「京けい」の１００倍ばい速はやく計算けいさんすることができます。理化学りかがく研究所けんきゅうしょは、２年ねんあとに「富岳ふがく」を使つかい始はじめたいと言いっています。
