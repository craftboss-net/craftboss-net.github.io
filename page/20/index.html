スーパーコンピューター「京けい」のスイッチを切きる式しきを行おこなう

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



神戸市こうべしにある理化学りかがく研究所けんきゅうしょでは、新あたらしいスーパーコンピューター「富岳ふがく」を使つかうことになりました。８月がつ３０日にち、今いままで使つかっていた「京けい」のスイッチを切きる式しきがありました。
理化学りかがく研究所けんきゅうしょの人ひとは「少すこし寂さびしいですが、『富岳ふがく』も『京けい』と同おなじように役やくに立たつようにしたいと思おもいます」とあいさつしました。そして理化学りかがく研究所けんきゅうしょの人ひとなどが、「京けい」が置おいてある部屋へやで４つのスイッチを１つずつ切きっていきました。
国くにの計画けいかくでつくった「京けい」は２０１２年ねんにできました。計算けいさんの速はやさで世界せかいで１番ばんになったことがあります。ビッグデータを分析ぶんせきする能力のうりょくでは、何なん回かいも世界せかいで１番ばんになりました。地震じしんの被害ひがいの予測よそくや薬くすりの研究けんきゅうなどにとても役やくに立たちました。
「京けい」のあとに使つかう「富岳ふがく」は、「京けい」の１００倍ばい速はやく計算けいさんすることができます。理化学りかがく研究所けんきゅうしょは、２年ねんあとに「富岳ふがく」を使つかい始はじめたいと言いっています。

愛媛大学えひめだいがく　人ひとの体からだにあるたんぱく質しつをほとんど全部ぜんぶ作つくる

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



愛媛大学えひめだいがくは、小麦こむぎから作つくった特別とくべつな液体えきたいを使つかって、人ひとの遺伝子いでんしからたんぱく質しつを作つくる研究けんきゅうをしています。
大学だいがくは、人ひとの体からだにある約やく２万まん４０００種類しゅるい以上いじょうのたんぱく質しつをほとんど全部ぜんぶ、研究室けんきゅうしつで作つくることができたと発表はっぴょうしました。大学だいがくによると、人ひとのたんぱく質しつをほとんど全部ぜんぶ作つくったのは世界せかいで初はじめてです。
たんぱく質しつは異常いじょうがあると病気びょうきの原因げんいんになりますが、新あたらしい薬くすりを作つくるためにも使つかわれています。
たんぱく質しつを使つかってがんの研究けんきゅうをしている東京大学とうきょうだいがくの教授きょうじゅは「病気びょうきにならないようにしたり新あたらしい薬くすりを作つくったりする研究けんきゅうが進すすむと思おもいます」と言いっています。

新幹線しんかんせんに大おおきな荷物にもつを持もって乗のる場合ばあい予約よやくが必要ひつようになる

发表于 2019-09-02 更新于 2019-11-30 分类于 JapanEasyNews



ＪＲによると、東海道とうかいどう?山陽新幹線さんようしんかんせんと九州きゅうしゅう新幹線しんかんせんにスーツケースなど大おおきな荷物にもつを持もって乗のるときは、来年らいねん５月がつから予約よやくが必要ひつようになります。来年らいねん東京とうきょうオリンピックとパラリンピックがあります。安全あんぜんのため、自分じぶんの席せきの近ちかくに大おおきな荷物にもつを置おくことができるようにします。
予約よやくが必要ひつような荷物にもつは、縦たて、横よこ、厚あつさの長ながさを足たすと１６０ｃｍより長ながくて２５０ｃｍ以下いかのスーツケースなどです。駅えきやインターネットで、席せきと一緒いっしょに予約よやくをすることが必要ひつようです。
予約よやくをした場合ばあい、荷物にもつを置おくためのお金かねはかかりませんが、予約よやくをしていない場合ばあいは１０００円えん払はらわなければなりません。
ＪＲは、東北新幹線とうほくしんかんせんと北陸新幹線ほくりくしんかんせんでは、このような予約よやくを行おこなう予定よていはないと言いっています。
