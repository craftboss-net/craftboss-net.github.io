東京とうきょうで韓国かんこくの文学ぶんがくを楽たのしむイベント

发表于 2019-08-08 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうの千代田区ちよだくにある韓国かんこくの本ほんを売うる店みせで７日なのか、韓国かんこくの文学ぶんがくを楽たのしむイベントがありました。約やく２０人にんが参加さんかして、チェ?ウニョンさんが書かいた「ショウコの微笑びしょう」という短みじかい小説しょうせつについて、翻訳ほんやくした人ひとと一緒いっしょに話はなし合あいました。
韓国かんこくではたくさんの人ひとが、村上春樹むらかみはるきさんや東野圭吾ひがしのけいごさんなどが書かいた日本にっぽんの小説しょうせつを読よんでいます。日本にっぽんでも１０年ねんぐらい前まえから韓国かんこくの小説しょうせつの翻訳ほんやくが増ふえて、たくさんの人ひとが読よんでいます。
このイベントに参加さんかした男性だんせいは「韓国かんこくについて知しらないことがたくさんあります。最近さいきんの政治せいじの問題もんだいもあるので、もっと知しりたいと思おもっています」と話はなしていました。
このイベントは、韓国かんこくの文学ぶんがくを日本にっぽんに紹介しょうかいする会社かいしゃのキム?スンボクさんが開ひらきました。キムさんは「文学ぶんがくは人ひとの気持きもちをわかるようにする力ちからがあると思おもいます。韓国かんこくのいい文学ぶんがくをたくさん紹介しょうかいしたいです」と話はなしています。

子こどもへの虐待ぎゃくたい　今いままででいちばん多おおい約やく１６万まん件けん

发表于 2019-08-07 更新于 2019-11-30 分类于 JapanEasyNews



児童相談所じどうそうだんじょは子こどもを守まもる仕事しごとをしています。
厚生労働省こうせいろうどうしょうによると、今年ことし３月がつまでの１年ねんに児童相談所じどうそうだんじょが「親おやなどが子こどもを虐待ぎゃくたいしている」と聞きいて家庭かていを調しらべたりした数かずは、１５万まん９８５０件けんありました。前まえの１年ねんより２万まん６０００件けん以上いじょう増ふえて、今いままででいちばん多おおくなりました。
虐待ぎゃくたいの半分はんぶん以上いじょうは、子こどもにひどいことを言いったり、子こどもの前まえで家族かぞくをたたいたりする心こころの虐待ぎゃくたいで、約やく８万まん８０００件けんありました。次つぎに多おおかったのは子こどもをたたいたり蹴けったりする体からだの虐待ぎゃくたいで、その次つぎは子こどもの世話せわをしない虐待ぎゃくたいでした。
厚生労働省こうせいろうどうしょうは「虐待ぎゃくたいで亡なくなった子こどももいます。子こどもを守まもるため、児童相談所じどうそうだんじょがしっかり仕事しごとができるように手伝てつだいたいと思おもいます」と言いっています。

空そらを飛とぶことができる車くるまのテストをする

发表于 2019-08-07 更新于 2019-11-30 分类于 JapanEasyNews



ＮＥＣは空そらを飛とぶことができる車くるまの研究けんきゅうをしています。５日いつか、まだ人ひとが乗のっていない車くるまを飛とばすテストをしました。
車くるまの長ながさは４ｍぐらいで、重おもさは１５０ｋｇです。電気でんきで動うごくモーターで４つのプロペラを回まわして飛とびます。ＮＥＣの人ひとが無線むせんでモーターを動うごかすと、車くるまがまっすぐ上うえに浮うきました。そして、３ｍぐらいの高たかさまでゆっくり上あがりました。
空そらを飛とんで人ひとや荷物にもつを運はこぶ車くるまは、道みちが混こまないようにするために役やくに立たちます。人口じんこうが少すくない町まちの交通こうつうも便利べんりになると言いわれています。ＮＥＣは、２０２３年ねんに荷物にもつを運はこぶテストをして、そのあと、人ひとが乗のることができるようにしたいと考かんがえています。

７日なのかも日本にっぽんのいろいろな所ところで３５℃以上いじょうになりました。暑あつい日ひが続つづいているので、熱中症ねっちゅうしょうにならないように、よく水みずを飲のんだり、エアコンで部屋へやを涼すずしくしたりしてください。
国くにによると、８月がつ４日よっかまでの１週間しゅうかんに、熱中症ねっちゅうしょうで病院びょういんに運はこばれた人ひとは１万まん８０００人にん以上いじょういました。その前まえの１週間しゅうかんの３倍ばい以上いじょうです。この中なかの５７人にんが亡なくなって、７３０人にんぐらいが長ながい入院にゅういんが必要ひつようになりました。病院びょういんに運はこばれた人ひとは６５歳さい以上いじょうの人ひとが多おおくて、１万まん人にんぐらいいました。熱中症ねっちゅうしょうになった場所ばしょでいちばん多おおかったのは、家いえの中なかでした。
これからも暑あつい日ひが続つづきそうです。熱中症ねっちゅうしょうにならないために気きをつけることは「ＮＥＷＳ　ＷＥＢ　ＥＡＳＹ」の「熱中症ねっちゅうしょう」のページにも書かいてあります。

エジプトの昔むかしの王おう、ツタンカーメンの金きんのひつぎは１９２２年ねんに見みつかりました。そのあとはずっと、ルクソールの「王家おうけの谷たに」にある墓はかにありました。

７４年ねん前まえの１９４５年ねん８月がつ６日むいか、広島ひろしまに原爆げんばくが落おとされました。広島市ひろしましでは毎年まいとし８月がつ６日むいかに平和へいわを祈いのる式しきを行おこなっていて、今年ことしは約やく５万まん人にんが式しきに出席しゅっせきしました。世界せかいから９２の国くにの代表だいひょうも来きました。

戦たたかいが原因げんいんで亡なくなったりけがをしたりした子こどもが多おおくなる

发表于 2019-08-05 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんは、アフリカなど戦たたかいが続つづいている所ところで生活せいかつしている子こどもたちについて調しらべています。
その結果けっか、去年きょねん戦たたかいが原因げんいんで亡なくなったり、けがをして体からだに障害しょうがいが残のこったりした子こどもは、１万まん２０００人にん以上いじょういました。２００５年ねんに調しらべ始はじめてから最もっとも多おおくなりました。
兵士へいしにされて戦たたかいの場所ばしょに連つれて行いかれた子こどもは７０００人にん以上いじょういました。特とくにアフリカのソマリアやナイジェリア、中東ちゅうとうのシリアに多おおくいました。性せい的てきな暴力ぼうりょくを受うけている子こどももいました。アフリカのマリでは、戦たたかいが原因げんいんで８２７の学校がっこうが使つかうことができなくなって、２４万まん人にん以上いじょうの子こどもが勉強べんきょうできなくなりました。
国連こくれんのグテーレス事務総長じむそうちょうは、戦たたかいをしている人ひとたちに、子こどもや市民しみんなどを攻撃こうげきしないように言いっています。そして、子こどもの安全あんぜんを守まもるためには平和へいわがいちばん大切たいせつなので、戦たたかいを終おわりにするように言いっています。
