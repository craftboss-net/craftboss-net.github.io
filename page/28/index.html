国会議員こっかいぎいんの小泉こいずみ進次郎しんじろうさんが滝川たきがわクリステルさんと結婚けっこん

发表于 2019-08-08 更新于 2019-11-30 分类于 JapanEasyNews



自民党じみんとうの国会議員こっかいぎいんの小泉こいずみ進次郎しんじろうさんは８日ようか、アナウンサーの滝川たきがわクリステルさんと結婚けっこんしました。
小泉こいずみ進次郎しんじろうさんは３８歳さいで、１０年ねん前まえに衆議院しゅうぎいんの選挙せんきょに出でて初はじめて議員ぎいんになりました。小泉こいずみさんの父親ちちおやは総理大臣そうりだいじんをしていた小泉こいずみ純一郎じゅんいちろうさんです。自民党じみんとうの中なかでは、小泉こいずみ進次郎しんじろうさんは将来しょうらい総理大臣そうりだいじんになるかもしれない人ひとの中なかの１人ひとりだと言いわれています。
滝川たきがわクリステルさんは４１歳さいで、東京とうきょうでオリンピックとパラリンピックを開ひらくことを決きめた会議かいぎで東京とうきょうを紹介しょうかいしました。そのときに使つかった「お?も?て?な?し」ということばは、この年としに最もっとも流行りゅうこうしたことばの賞しょうをもらいました。
小泉こいずみさんは７日なのか、「政治せいじは戦たたかいの場所ばしょですが、滝川たきがわさんと一緒いっしょにいると戦たたかいから離はなれることができます」と話はなしました。滝川たきがわさんは妊娠にんしんしていて、来年らいねん赤あかちゃんが生うまれると言いっています。滝川たきがわさんは「私わたしたち２人ふたりは、そのままでいることができるいい関係かんけいだと思おもいます」と話はなしました。

浅間山あさまやまで小ちいさな噴火ふんかがあった

发表于 2019-08-08 更新于 2019-11-30 分类于 JapanEasyNews



気象庁きしょうちょうによると、７日なのかの午後ごご１０時じごろ、長野県ながのけんと群馬県ぐんまけんの間あいだにある浅間山あさまやまの頂上ちょうじょうにある火口かこうで、小ちいさな噴火ふんかがありました。噴火ふんかの煙けむりが山やまの上うえから１８００ｍ以上いじょうまで上あがって、大おおきな石いしが２００ｍぐらい飛とびました。噴火ふんかは２０分ぷんぐらい続つづいて、煙けむりは北きたに流ながれました。
噴火ふんかがまたあるかもしれないため、気象庁きしょうちょうは、１から５まである「噴火ふんか警戒けいかい」のレベルを３に上あげて、浅間山あさまやまに登のぼることを禁止きんししました。
気象庁きしょうちょうは、火口かこうから４ｋｍぐらいの所ところまでは、大おおきな石いしが飛とんできたりするかもしれないと言いっています。もっと遠とおくても、風かぜで小ちいさな石いしや灰はいが飛とんでくるかもしれないため、気きをつけるように言いっています。
浅間山あさまやまは２００９年ねんにも「噴火ふんか警戒けいかい」のレベルが３に上あがりました。

发表于 2019-08-08 更新于 2019-11-30 分类于 JapanEasyNews



夏休なつやすみ中ちゅうの子こどもたちが、最高裁判所さいこうさいばんしょで裁判さいばんについて勉強べんきょうするイベントがありました。子こどもたちとお父とうさんやお母かあさんが３５０人にんぐらい来きました。
裁判さいばんをする部屋へやでは、子こどもたちが裁判さいばんのやり方かたを体験たいけんしました。検察官けんさつかんや弁護士べんごしの役やくの子こどもが事件じけんについて質問しつもんしたり、裁判官さいばんかんの役やくの子こどもが裁判さいばんで決きまった罪つみをみんなに伝つたえたりしていました。

小学校しょうがっこう５年生ねんせいの男おとこの子こは「どのように裁判さいばんをしているか知しることができて、よかったです」と話はなしていました。

子こどもへの虐待ぎゃくたい　今いままででいちばん多おおい約やく１６万まん件けん

发表于 2019-08-07 更新于 2019-11-30 分类于 JapanEasyNews



児童相談所じどうそうだんじょは子こどもを守まもる仕事しごとをしています。
厚生労働省こうせいろうどうしょうによると、今年ことし３月がつまでの１年ねんに児童相談所じどうそうだんじょが「親おやなどが子こどもを虐待ぎゃくたいしている」と聞きいて家庭かていを調しらべたりした数かずは、１５万まん９８５０件けんありました。前まえの１年ねんより２万まん６０００件けん以上いじょう増ふえて、今いままででいちばん多おおくなりました。
虐待ぎゃくたいの半分はんぶん以上いじょうは、子こどもにひどいことを言いったり、子こどもの前まえで家族かぞくをたたいたりする心こころの虐待ぎゃくたいで、約やく８万まん８０００件けんありました。次つぎに多おおかったのは子こどもをたたいたり蹴けったりする体からだの虐待ぎゃくたいで、その次つぎは子こどもの世話せわをしない虐待ぎゃくたいでした。
厚生労働省こうせいろうどうしょうは「虐待ぎゃくたいで亡なくなった子こどももいます。子こどもを守まもるため、児童相談所じどうそうだんじょがしっかり仕事しごとができるように手伝てつだいたいと思おもいます」と言いっています。

７４年ねん前まえの１９４５年ねん８月がつ６日むいか、広島ひろしまに原爆げんばくが落おとされました。広島市ひろしましでは毎年まいとし８月がつ６日むいかに平和へいわを祈いのる式しきを行おこなっていて、今年ことしは約やく５万まん人にんが式しきに出席しゅっせきしました。世界せかいから９２の国くにの代表だいひょうも来きました。

エジプト　ツタンカーメンの金きんのひつぎを初はじめて墓はかから出だした

发表于 2019-08-06 更新于 2019-11-30 分类于 JapanEasyNews



エジプトの昔むかしの王おう、ツタンカーメンの金きんのひつぎは１９２２年ねんに見みつかりました。そのあとはずっと、ルクソールの「王家おうけの谷たに」にある墓はかにありました。
エジプトでは、日本にっぽんが協力きょうりょくして「大だいエジプト博物館はくぶつかん」をつくっていて、来年らいねんオープンする予定よていです。この博物館はくぶつかんにひつぎを置おいてみんなに見みせるために、４日よっか、初はじめて墓はかから出だしました。
金きんのひつぎは美うつくしく光ひかっていますが、割われたり壊こわれたりしているため、これから直なおします。エジプトの政府せいふは「直なおすために８か月げつかかります。そのあと、この有名ゆうめいなひつぎをみんなに見みせたいと思おもいます」と話はなしていました。
大だいエジプト博物館はくぶつかんでは、ツタンカーメンの墓はかに入はいっていた５０００個この物ものを見みせることになっています。
