「テロから守まもる施設しせつができない場合ばあいは原発げんぱつを止とめる」

发表于 2019-04-25 更新于 2019-11-30 分类于 JapanEasyNews



福島県ふくしまけんにある原子力げんしりょく発電所はつでんしょの事故じこのあと、国くには全部ぜんぶの原発げんぱつを止とめました。原発げんぱつをもう一度いちど動うごかすためには、安全あんぜんのために必要ひつような工事こうじをしなければならないと決きまりました。
決きまりでは、テロから守まもるために飛行機ひこうきなどがぶつかっても、１００ｍ以上いじょう離はなれた所ところから原発げんぱつを動うごかすことができる施設しせつを作つくらなければなりません。もう一度いちど原発げんぱつを動うごかすための工事こうじの計画けいかくが国くにに認みとめられてから５年ねん以内いないに作つくらなければなりません。
しかし、九州電力きゅうしゅうでんりょく、関西電力かんさいでんりょく、四国電力しこくでんりょくは工事こうじが遅おくれていると言いっています。
国くにの原子力げんしりょく規制きせい委員会いいんかいは２４日にじゅうよっか、予定よてい通どおりに施設しせつができない場合ばあいは、原発げんぱつの運転うんてんを止とめる命令めいれいを出だすと言いいました。
鹿児島県かごしまけんにある九州電力きゅうしゅうでんりょくの川内せんだい原発げんぱつ１号機ごうきは、来年らいねん３月がつまでに施設しせつを作つくらなければなりません。しかし会社かいしゃは、工事こうじが終おわるまで２年ねんぐらいかかりそうだと言いっています。

池袋いけぶくろで母ははと娘むすめが亡なくなった事故じこ　「悔くやしい」と夫おっとが話はなす

发表于 2019-04-25 更新于 2019-11-30 分类于 JapanEasyNews



４月がつ１９日にち、東京とうきょうの池袋いけぶくろで８７歳さいの人ひとが運転うんてんする車くるまが、人ひとや自転車じてんしゃとぶつかる事故じこがありました。この事故じこで、自転車じてんしゃで横断歩道おうだんほどうを渡わたっていた３１歳さいの松永まつなが真菜まなさんと３歳さいの娘むすめの莉子りこちゃんが亡なくなって、１０人にんがけがをしました。
２４日にじゅうよっか、真菜まなさんの３２歳さいの夫おっとが記者きしゃたちの前まえで、「大事だいじな妻つまと娘むすめが突然とつぜんいなくなって、涙なみだが止とまりません。これから娘むすめがどんどん大おおきくなって、大人おとなになって家いえから出でていって、そのあとも私わたしと妻つまはずっと一緒いっしょにいると信しんじていました。しかし、私わたしたちの未来みらいはなくなってしまいました。悔くやしいです」と話はなしました。
去年きょねん、車くるまを運転うんてんしていて人ひとが亡なくなる事故じこを起おこした７５歳さい以上いじょうの人ひとは、４６０人にんいました。おととしより４２人にん増ふえました。
夫おっとは「少すこしでも車くるまの運転うんてんに心配しんぱいがある人ひとは、運転うんてんをやめることを考かんがえてほしいです。家族かぞくの中なかに運転うんてんに心配しんぱいがある人ひとがいたら、家族かぞくでどうしたらいいか話はなし合あってください。みんなで考かんがえたら、交通事故こうつうじこで亡なくなる人ひとを減へらすことができるかもしれません」と話はなしました。

スリランカの警察けいさつによると、９人にんが爆発ばくはつを起おこしました。この中なかには女おんなも１人ひとりいました。警察けいさつは、９人にんはスリランカにいるＮＴＪ＝ナショナル?タウヒード?ジャマートというグループのメンバーだと考かんがえています。スリランカの政府せいふは、ＮＴＪのメンバーなど６０人にんのスリランカ人じんを捕つかまえたと言いいました。

１０日とおか間かんの休やすみ　７５％のホテルや旅館りょかんで予約よやくがいっぱい

发表于 2019-04-23 更新于 2019-11-30 分类于 JapanEasyNews



今年ことしは４月がつ２７日にちから５月がつ６日むいかまで休やすみが１０日とおか間かん続つづきます。
観光庁かんこうちょうは、日本にっぽんのホテルや旅館りょかんの、この１０日とおか間かんの予約よやくについて、調査ちょうさ会社がいしゃが１８日にちに調しらべた結果けっかを発表はっぴょうしました。
１０日とおか間かんの平均へいきんでは、調しらべた約やく２万まん９０００のホテルや旅館りょかんのうち、７５％で予約よやくがいっぱいになっていました。予約よやくがいっぱいのホテルや旅館りょかんの割合わりあいは、京都府きょうとふが９０％でいちばん高たかくて、次つぎが沖縄県おきなわけん、徳島県とくしまけんでした。愛知県あいちけんは６０％、茨城県いばらきけんは６２％で、ほかの所ところより予約よやくしやすそうです。
４月がつ２７日にちから３０日にちまでは、予約よやくがいっぱいのホテルや旅館りょかんは８５％以上いじょうになっています。しかし５月がつ４日よっかから６日むいかまでは５０％ぐらいです。
まだ予約よやくができるホテルや旅館りょかんの料金りょうきんは、４月がつの初はじめより平均へいきんで１．５倍ばいに上あがっています。
