「全国学力ぜんこくがくりょくテスト」　中学校ちゅうがっこうで初はじめて英語えいごのテストを行おこなう

发表于 2019-04-19 更新于 2019-11-30 分类于 JapanEasyNews



「全国学力ぜんこくがくりょくテスト」は、文部科学省もんぶかがくしょうが小学校しょうがっこう６年生ねんせいと中学校ちゅうがっこう３年生ねんせいに行おこなっているテストです。子こどもたちの学力がくりょくを調しらべて授業じゅぎょうをよくするために、２００７年ねんから行おこなっています。
４月がつ１８日にちにテストがあって、約やく２１２万まん人にんが受うけました。今年ことしは国語こくごと算数さんすう?数学すうがくのほかに、中学校ちゅうがっこうでは初はじめて英語えいごのテストがありました。
英語えいごでは「読よむ力ちから」と「聞きく力ちから」のほかに、「話はなす力ちから」と「書かく力ちから」を調しらべるテストもありました。「話はなす力ちから」と「書かく力ちから」のテストは、２０２１年ねんから始はじまる「大学だいがく入学にゅうがく共通きょうつうテスト」にもあります。
「話はなす力ちから」のテストは、パソコンを使つかった５分ふんぐらいのテストです。生徒せいとたちはヘッドホンで問題もんだいを聞きいて、英語えいごで答こたえを言いいます。その答こたえを録音ろくおんして、正ただしいかどうかチェックします。
文部科学省もんぶかがくしょうは、７月がつに全国ぜんこく学力がくりょくテストの結果けっかを発表はっぴょうします。

東京とうきょうオリンピック　チケットの申もうし込こみは５月がつ９日ここのかから

发表于 2019-04-19 更新于 2019-11-30 分类于 JapanEasyNews



来年らいねん東京とうきょうで行おこなうオリンピックのチケットを売うるウェブサイトができました。
チケットの最初さいしょの申もうし込こみは５月がつ９日ここのかから２８日にちまでです。最初さいしょの申もうし込こみは抽ちゅうせんを行おこなって、６月がつ２０日はつかに結果けっかがわかります。
ウェブサイトでは競技きょうぎの会場かいじょうの図ずを見みて、席せきの場所ばしょや値段ねだんがわかるようになっています。チケットは「第だい１希望きぼう」と「第だい２希望きぼう」で３０枚まいずつ申もうし込こむことができますが、抽ちゅうせんに当あたるのは３０枚まいまでです。
東京とうきょうオリンピックのチケットは全部ぜんぶで７８０万まん枚まいぐらいです。この中なかで、５月がつに申もうし込こむ チケットがいちばん多おおくなりそうです。今年ことしの秋あきからは、抽ちゅうせんのないチケットを売うり始はじめます。来年らいねんの春はるには、東京とうきょうにチケットを売うる店みせもできます。
チケットの値段ねだんは、開会かいかい式しきと閉会へいかい式しきでいちばん値段ねだんが高たかい席せきは３０万まん円えんです。陸上りくじょうの男子だんし１００メートルの決勝けっしょうを行おこなう日ひの午後ごごのチケットは、１３万まん円えんから５８００円えんです。

台湾たいわんで大おおきな地震じしん　１７人にんがけが

发表于 2019-04-19 更新于 2019-11-30 分类于 JapanEasyNews



台湾たいわんで１８日にちの午後ごご２時じごろ、マグニチュード６．１の地震じしんがありました。東ひがしのほうにある花蓮県かれんけんでは震度しんど７の大おおきな揺ゆれがありました。台北たいほくでも震度しんど４の揺ゆれがありました。
花蓮県かれんけんにある太魯閣渓谷たろこけいこくという景色けしきのいい場所ばしょでは、石いしが落おちて、旅行りょこうに来きていた男性だんせいと女性じょせいが頭あたまなどにけがをしました。この地震じしんで、１７人にんがけがをしました。花蓮駅かれんえきでは、水道すいどうが壊こわれて駅えきの待合室まちあいしつに水みずが入はいりました。台北たいほくではビルの壁かべのタイルが落おちたり、全部ぜんぶの地下鉄ちかてつが止とまったりしました。
台湾たいわんの東ひがしのほうでは最近さいきん、地震じしんが続つづいています。台湾たいわんの中央気象局ちゅうおうきしょうきょくは、これからも地震じしんに気きをつけてほしいと言いっています。

发表于 2019-04-18 更新于 2019-11-30 分类于 JapanEasyNews



クラゲは、サンゴやイソギンチャクと同おなじグループの生いき物ものです。小ちいさいときはサンゴなどと同おなじように海うみの底そこに付ついていますが、大おおきくなると海うみの中なかを泳およぐようになります。どうしてクラゲだけが泳およぐようになったのか、沖縄おきなわ科学技術かがくぎじゅつ大学院だいがくいん大学だいがくとドイツのチームが研究けんきゅうしました。

发表于 2019-04-17 更新于 2019-11-30 分类于 JapanEasyNews



成田空港なりたくうこうでは、飛行機ひこうきをとめる場所ばしょで働はたらく人ひとが運転うんてんする車くるまの事故じこが増ふえています。成田空港なりたくうこうの会社かいしゃによると、１年ねんに１１９件けんの事故じこがあって、前まえの年としの１．５倍ばいぐらいに増ふえました。
多おおいのは車くるまと車くるまがぶつかる事故じこでした。会社かいしゃが調しらべると、事故じこを起おこした人ひとの半分はんぶん以上いじょうは、空港くうこうで車くるまを運転うんてんする仕事しごとを始はじめてから３年ねんになっていませんでした。

成田空港なりたくうこうの会社かいしゃは「日本にっぽんへ観光かんこうに来くる外国人がいこくじんが増ふえて、空港くうこうを利用りようする飛行機ひこうきはもっと増ふえると思おもいます。事故じこが起おこらないようにしっかり教育きょういくします」と話はなしています。

アメリカのフィッシャープライスという会社かいしゃは、赤あかちゃんのベッドなどを売うっています。この会社かいしゃの「ロックン?プレイ?スリーパー」というベッドで寝ねていた赤あかちゃんが３０人にん以上いじょう亡なくなっていたことがわかりました。
