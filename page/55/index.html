東京とうきょうで雨あめが降ふったときに地面じめんにたまる水みずの深ふかさを予想よそう

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうなどの大おおきなまちで雨あめがたくさん降ふると、地面じめんの上うえに水みずがたまって危険きけんになることがあります。早稲田大学わせだだいがくと東京大学とうきょうだいがくなどのグループは、雨あめが降ふったときに東京とうきょうの２３区くで水みずが地面じめんにどのくらいたまるかすぐに予想よそうするシステムをつくりました。
このシステムでは、雲くもの情報じょうほうを調しらべてどのくらい雨あめが降ふるか予想よそうします。そして、道みちや建物たてもの、川かわ、下水道げすいどうの情報じょうほうも利用りようして、たまる水みずの深ふかさをすぐに計算けいさんします。そして地図ちずに、水みずの深ふかさを色いろで知しらせます。今いまの水みずの深ふかさだけではなくて、３０分ぷんあとにどのくらいの深ふかさになるかもわかります。
グループは、６月がつごろからインターネットでこの予想よそうを見みせる予定よていです。

２０日はつか、プロ野球やきゅうのピッチャーの上原うえはら浩治こうじさんが「選手せんしゅをやめます」と言いいました。上原うえはらさんは４４歳さいで、日本にっぽんのプロ野球やきゅうのピッチャーでは年齢ねんれいがいちばん上うえでした。

北海道ほっかいどう　見みえなくなった島しまを海上保安庁かいじょうほあんちょうが調しらべる

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



３２年ねん前まえ、北海道ほっかいどうの猿払村さるふつむらから５００ｍぐらいの所ところで、海うみの上うえに１．４ｍぐらい出でているとても小ちいさい島しまが見みつかりました。
去年きょねん１０月がつ、猿払村さるふつむらに住すんでいる人ひとから島しまが見みえなくなったという連絡れんらくがありました。今月こんげつ２０日はつか、海上保安庁かいじょうほあんちょうは島しまが見みえていた所ところの周まわりの海うみを調しらべ始はじめました。
世界せかいのルールでは、島しまは海うみの上うえに出でていなければならないと決きまっています。海上保安庁かいじょうほあんちょうは、波なみや氷こおりがぶつかって海うみの上うえに出でていた所ところがなくなったかもしれないと考かんがえています。
海上保安庁かいじょうほあんちょうは「海面かいめんから深ふかさ１ｍぐらいの所ところに岩いわがありました。今日きょうは海面かいめんが高たかくなっていたので、低ひくくなるときに海うみの上うえに出でるかどうか調しらべます」と言いっています。

このゴンドラのビデオはインターネットに出でて、６時間じかんで１００万まん回かいぐらい見みられました。

新あたらしく大学だいがくの入学試験にゅうがくしけんに使つかう英語えいごの試験しけん　公平こうへいかどうか心配しんぱい

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



大学だいがくに入学にゅうがくするために多おおくの人ひとが受うける「センター試験しけん」は、２０２１年ねん１月がつから「大学入学共通だいがくにゅうがくきょうつうテスト」にかわります。「大学入学共通だいがくにゅうがくきょうつうテスト」は国くにが行おこなう試験しけんですが、国くに以外いがいが行おこなう英語えいごの検定けんてい試験しけんも使つかうことができるようになります。使つかうことができるのは「英検えいけん」や「ＴＯＥＩＣ」などです。
高校こうこうで英語えいごを教おしえている先生せんせいは、去年きょねん生徒せいとが受うけた検定けんてい試験しけんの点てんに驚おどろいたと話はなしています。「町まちをきれいにするために何なにができると思おもいますか。理由りゆうも書かいてください」という問題もんだいがありました。生徒せいとが書かいた答こたえは「Ｉ　ｔｈｉｎｋ　ｔｏ　ｉｎｐｏｒｔａｎｔ」だけでした。文法ぶんぽうや単語たんごも間違まちがっていましたが、１６０点てんのうち４１点てんもらっていました。この先生せんせいは、自分じぶんが点てんをつけたら０点てんだと話はなしていました。
国くに以外いがいが行おこなっている試験しけんは、学生がくせいのアルバイトや外国がいこくで点てんをつけていることがあります。このため、専門家せんもんかは公平こうへいかどうか心配しんぱいだと話はなしています。

アメリカ　ファーウェイに部品ぶひんを売うることを禁止きんしする

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



アメリカの政府せいふは１５日にち、中国ちゅうごくの携帯電話けいたいでんわなどの会社かいしゃのファーウェイに、アメリカの会社かいしゃが部品ぶひんなどを売うることを禁止きんししました。ファーウェイが原因げんいんで、アメリカの安全あんぜんや外交がいこうの利益りえきを守まもることができないためだと言いっています。
ロイター通信つうしんは、これからファーウェイは必要ひつような部品ぶひんを集あつめることができなくなって、一部いちぶの商品しょうひんを売うることが難むずかしくなるかもしれないと伝つたえています。
去年きょねん１２月がつ、カナダはアメリカに頼たのまれて、ファーウェイの孟もう晩舟ばんしゅう副ふく会長かいちょうを逮捕たいほしました。トランプ大統領だいとうりょうはそのあと、アメリカの安全あんぜんを守まもるためだと言いって、ファーウェイに厳きびしくしています。
アメリカと中国ちゅうごくは、貿易ぼうえきの問題もんだいについての話はなし合あいがうまくいっていないため、お互たがいに関税かんぜいを上あげています。アメリカが貿易ぼうえきの話はなし合あいをうまく進すすめるために、中国ちゅうごくの会社かいしゃに厳きびしくしていると考かんがえている人ひともいます。

先月せんげつ２４日にじゅうよっか、沖縄県おきなわけん那覇市なはしの高校生こうこうせいの崎元さきもと颯馬そうまさんは、おじさんの葬式そうしきに出でるために空港くうこうに向むかいました。しかし空港くうこうの駅えきに着ついたとき、財布さいふをなくしたことがわかりました。財布さいふには飛行機ひこうきのチケットを買かうための約やく６万まん円えんが入はいっていました。崎元さきもとさんはどうしたらいいかわからなくて、とても困こまっていました。
