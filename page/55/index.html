北海道ほっかいどう　見みえなくなった島しまを海上保安庁かいじょうほあんちょうが調しらべる

发表于 2019-05-21 更新于 2019-11-30 分类于 JapanEasyNews



３２年ねん前まえ、北海道ほっかいどうの猿払村さるふつむらから５００ｍぐらいの所ところで、海うみの上うえに１．４ｍぐらい出でているとても小ちいさい島しまが見みつかりました。
去年きょねん１０月がつ、猿払村さるふつむらに住すんでいる人ひとから島しまが見みえなくなったという連絡れんらくがありました。今月こんげつ２０日はつか、海上保安庁かいじょうほあんちょうは島しまが見みえていた所ところの周まわりの海うみを調しらべ始はじめました。
世界せかいのルールでは、島しまは海うみの上うえに出でていなければならないと決きまっています。海上保安庁かいじょうほあんちょうは、波なみや氷こおりがぶつかって海うみの上うえに出でていた所ところがなくなったかもしれないと考かんがえています。
海上保安庁かいじょうほあんちょうは「海面かいめんから深ふかさ１ｍぐらいの所ところに岩いわがありました。今日きょうは海面かいめんが高たかくなっていたので、低ひくくなるときに海うみの上うえに出でるかどうか調しらべます」と言いっています。

アメリカ　窓まどの掃除そうじのゴンドラが高たかさ２６０ｍでひどく揺ゆれる

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのオクラホマ州しゅうで１５日にち、窓まどの掃除そうじをする２人ふたりが乗のったゴンドラが、ビルの高たかさ２６０ｍの所ところでコントロールできなくなりました。ゴンドラは、前まえや後うしろ、左ひだりや右みぎに揺ゆれて、だんだんひどく揺ゆれるようになって、ビルにぶつかって窓まどが割われました。
この揺ゆれるゴンドラを撮とったビデオには、落おちないようにしっかりつかまっている２人ふたりがうつっています。２人ふたりは、消防しょうぼうの人ひとが用意よういしたロープを使つかって揺ゆれるゴンドラを止とめたあと、無事ぶじに助たすけられました。ビルの周まわりには割われたガラスが落おちましたが、けがをした人ひとはいませんでした。
このゴンドラのビデオはインターネットに出でて、６時間じかんで１００万まん回かいぐらい見みられました。

新あたらしく大学だいがくの入学試験にゅうがくしけんに使つかう英語えいごの試験しけん　公平こうへいかどうか心配しんぱい

发表于 2019-05-20 更新于 2019-11-30 分类于 JapanEasyNews



大学だいがくに入学にゅうがくするために多おおくの人ひとが受うける「センター試験しけん」は、２０２１年ねん１月がつから「大学入学共通だいがくにゅうがくきょうつうテスト」にかわります。「大学入学共通だいがくにゅうがくきょうつうテスト」は国くにが行おこなう試験しけんですが、国くに以外いがいが行おこなう英語えいごの検定けんてい試験しけんも使つかうことができるようになります。使つかうことができるのは「英検えいけん」や「ＴＯＥＩＣ」などです。
高校こうこうで英語えいごを教おしえている先生せんせいは、去年きょねん生徒せいとが受うけた検定けんてい試験しけんの点てんに驚おどろいたと話はなしています。「町まちをきれいにするために何なにができると思おもいますか。理由りゆうも書かいてください」という問題もんだいがありました。生徒せいとが書かいた答こたえは「Ｉ　ｔｈｉｎｋ　ｔｏ　ｉｎｐｏｒｔａｎｔ」だけでした。文法ぶんぽうや単語たんごも間違まちがっていましたが、１６０点てんのうち４１点てんもらっていました。この先生せんせいは、自分じぶんが点てんをつけたら０点てんだと話はなしていました。
国くに以外いがいが行おこなっている試験しけんは、学生がくせいのアルバイトや外国がいこくで点てんをつけていることがあります。このため、専門家せんもんかは公平こうへいかどうか心配しんぱいだと話はなしています。

まだ食たべることができる食たべ物ものを捨すてることが問題もんだいになっています。コンビニやスーパーは、作つくってから長ながい時間じかん売うれないで残のこっている弁当べんとうやおにぎりなどを、３年ねん前まえには１年ねんに６６万まんｔ捨すてていました。このため、コンビニの会社かいしゃは、食たべ物ものを捨すてないようにいろいろなやり方かたを考かんがえています。
ローソンは、来月らいげつ１１日にちから愛媛県えひめけんと沖縄県おきなわけんの店みせで、作つくってから長ながい時間じかんが過すぎた食たべ物ものを買かった客きゃくにポイントを渡わたすことにしました。例たとえば、朝あさや昼ひる店みせに並ならべた弁当べんとうを午後ごご４時じよりあとに買かった客きゃくには、１００円えんの買かい物もので５ポイント渡わたします。ポイントは買かい物もののときお金かねの代かわりに使つかうことができます。
セブン‐イレブンも、秋あきから、同おなじようにポイントを渡わたすことにしています。
ファミリーマートは、クリスマスのケーキなどその季節きせつだけたくさん用意よういする物ものは、予約よやくした人ひとに売うることにしています。

横浜市よこはましの中学校ちゅうがっこうの９０％　昼ひるごはんの時間じかんが「１５分ふん」

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



横浜市よこはましに住すんでいる中学生ちゅうがくせいの親おやなどから、「学校がっこうの昼ひるごはんの時間じかんが短みじかい」という意見いけんが増ふえています。
横浜市よこはましの教育委員会きょういくいいんかいが調しらべると、横浜市よこはましの１４８の中学校ちゅうがっこうのうち１３５の学校がっこうで、昼ひるごはんを食たべる時間じかんは「１５分ふん」でした。そのほかの学校がっこうは「２０分ぷん」でした。
教育委員会きょういくいいんかいによると、ずっと前まえから「１５分ふん」が続つづいているようです。長ながくするのは難むずかしいと考かんがえている中学校ちゅうがっこうもあります。
教育委員会きょういくいいんかいが昼ひるごはんの時間じかんをもっと長ながくするように言いっているため、今年ことしの４月がつから５分ふん長ながくした学校がっこうもあります。しかし、８０％ぐらいの学校がっこうは「１５分ふん」のままです。
教育委員会きょういくいいんかいは「中学生ちゅうがくせいは体からだが大おおきくなる時ときなので、昼ひるごはんの時間じかんを十分じゅうぶんに用意よういするように言いっていきます」と話はなしています。

「暴力ぼうりょく的てきな映像えいぞうをＳＮＳに出ださない」８つの国くになどが国際こくさい会議かいぎ

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



ニュージーランドで今年ことし３月がつ、男おとこが銃じゅうを撃うつ 事件じけんがあって、５１人にんが亡なくなりました。男おとこは事件じけんを起おこしているときに、銃じゅうを撃うっている映像えいぞうをインターネットのＳＮＳに出だしました。
１５日にち、このような暴力ぼうりょく的てきな映像えいぞうの対策たいさくを話はなし合あう国際こくさい会議かいぎがフランスでありました。フランスやニュージーランドなど８つの国くにの大統領だいとうりょうや首相しゅしょうと、フェイスブックやツイッターなどＳＮＳの会社かいしゃの幹部かんぶが出席しゅっせきしました。
会議かいぎでまとめた考かんがえ方かたによると、ＳＮＳの会社かいしゃは、暴力ぼうりょく的てきな映像えいぞうやテロの映像えいぞうをそのままＳＮＳに出だしたり広ひろめたりできないようにします。８つの国くにの政府せいふは、法律ほうりつをしっかり使つかって、暴力ぼうりょく的てきな映像えいぞうを作つくったり広ひろめたりできないようにします。
フランスのマクロン大統領だいとうりょうは、９月がつにある国連こくれんの会議かいぎではもっと多おおくの国くになどに出席しゅっせきしてほしいと言いいました。

アメリカ　ファーウェイに部品ぶひんを売うることを禁止きんしする

发表于 2019-05-17 更新于 2019-11-30 分类于 JapanEasyNews



アメリカの政府せいふは１５日にち、中国ちゅうごくの携帯電話けいたいでんわなどの会社かいしゃのファーウェイに、アメリカの会社かいしゃが部品ぶひんなどを売うることを禁止きんししました。ファーウェイが原因げんいんで、アメリカの安全あんぜんや外交がいこうの利益りえきを守まもることができないためだと言いっています。
ロイター通信つうしんは、これからファーウェイは必要ひつような部品ぶひんを集あつめることができなくなって、一部いちぶの商品しょうひんを売うることが難むずかしくなるかもしれないと伝つたえています。
去年きょねん１２月がつ、カナダはアメリカに頼たのまれて、ファーウェイの孟もう晩舟ばんしゅう副ふく会長かいちょうを逮捕たいほしました。トランプ大統領だいとうりょうはそのあと、アメリカの安全あんぜんを守まもるためだと言いって、ファーウェイに厳きびしくしています。
アメリカと中国ちゅうごくは、貿易ぼうえきの問題もんだいについての話はなし合あいがうまくいっていないため、お互たがいに関税かんぜいを上あげています。アメリカが貿易ぼうえきの話はなし合あいをうまく進すすめるために、中国ちゅうごくの会社かいしゃに厳きびしくしていると考かんがえている人ひともいます。

国際的こくさいてきな労働組合ろうどうくみあいの団体だんたいは、東京とうきょうオリンピックとパラリンピックの会場かいじょうなどの工事こうじをしている人ひとがどのように働はたらいているか調しらべました。
東京とうきょう大会たいかいで選手せんしゅが泊とまる建物たてものや、新あたらしい国立競技場こくりつきょうぎじょうをつくっている人ひとの中なかには、１か月げつに２８日にち続つづけて働はたらいた人ひとがいました。暗くらい所ところや、頭あたまの上うえにコンクリートがつるしてある所ところで働はたらく人ひともいました。団体だんたいは、安全あんぜんではない働はたらき方かたをしている人ひとがいると言いっています。

組織委員会そしきいいんかいなどは、団体だんたいの言いっていることを確たしかめてから、どうするか考かんがえると話はなしています。
