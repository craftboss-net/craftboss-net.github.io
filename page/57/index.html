「うその話はなしでお金かねを取とられないように留守番るすばん電話でんわを使つかって」

发表于 2019-05-15 更新于 2019-11-30 分类于 JapanEasyNews



埼玉県さいたまけんの警察けいさつは今年ことし１月がつから３月がつ、うその話はなしをする詐欺さぎの電話でんわでお金かねを取とられた人ひとに話はなしを聞ききました。
この中なかで、家いえの電話でんわに留守番るすばん電話でんわの機能きのうがあると答こたえた人ひとは２００人にんぐらいいました。留守番るすばん電話でんわは、電話でんわに出でることができないときなどに、電話でんわをかけた人ひとのメッセージを録音ろくおんできる機能きのうです。
１６０人にん以上いじょうの人ひとは留守番るすばん電話でんわを使つかっていませんでした。４０人にんぐらいは使つかっていましたが、詐欺さぎの電話でんわがあったときは自分じぶんで電話でんわに出でていました。
警察けいさつは「家いえにいるときも留守番るすばん電話でんわを使つかって、メッセージや電話でんわ番号ばんごうで、誰だれからの電話でんわかよく確たしかめてください」と言いっています。

レオパレス２１　問題もんだいがある建物たてものは１万まん５０００以上いじょう

发表于 2019-05-15 更新于 2019-11-30 分类于 JapanEasyNews



レオパレス２１という会社かいしゃが建たてたアパートなどで火事かじが広ひろがらないようにする壁かべが足たりないことなどがわかって、問題もんだいになっています。会社かいしゃは、住すんでいる約やく７７００人にんに引ひっ越こすように言いいました。
会社かいしゃは、今いままでに建たてた全部ぜんぶの建物たてものの調査ちょうさをしています。４月がつまでに半分はんぶんぐらいの建物たてものの調査ちょうさが終おわりました。
会社かいしゃによると、１万まん５６２８の建物たてもので問題もんだいが見みつかりました。これは会社かいしゃが建たてた約やく３万まん９０００の建物たてものの３９％になります。７６１３の建物たてものは直なおす工事こうじをしなければなりません。しかし、工事こうじが終おわった建物たてものはまだ８００ぐらいです。
国土交通省こくどこうつうしょうは「１０月がつまでに全部ぜんぶの建物たてものを直なおすように言いいました。急いそいで工事こうじをしてほしいです」と話はなしています。

５５年ねん前まえの東京とうきょうオリンピックの聖火せいか台だいを岩手県いわてけんや福島県ふくしまけんで展示てんじ

发表于 2019-05-15 更新于 2019-11-30 分类于 JapanEasyNews



１９６４年ねんの東京とうきょうオリンピックで使つかった旧国立競技場きゅうこくりつきょうぎじょうの聖火せいか台だいは、４年ねん前まえから宮城県みやぎけん石巻市いしのまきしが借かりていました。そして、イベントで聖火せいか台だいに火ひをつけたりして、８年ねん前まえに大おおきな地震じしんがあった所ところの人ひとたちが元気げんきになるようにしてきました。
聖火せいか台だいを持もっている団体だんたいは、岩手県いわてけんと福島県ふくしまけんを回まわって聖火せいか台だいを展示てんじすることを決きめました。もっとたくさんの人ひとたちを元気げんきにして、来年らいねんの東京とうきょうオリンピックについても知しってもらいたいと考かんがえたためです。
１４日じゅうよっか、聖火せいか台だいを運はこび始はじめました。聖火せいか台だいは高たかさが２．１ｍ、太ふとさが２．１ｍもあるため、クレーンを使つかってトラックにのせました。
石巻市体育協会いしのまきしたいいくきょうかいの人ひとは「うれしいような寂さびしいような気持きもちです。岩手県いわてけんと福島県ふくしまけんでもみんなを元気げんきにしてほしいです」と話はなしていました。

たくさん引用いんようされた科学かがくの論文ろんぶん　１番ばんはアメリカで２番ばんは中国ちゅうごく

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



いい研究けんきゅうの論文ろんぶんは、別べつの論文ろんぶんで一部いちぶが使つかわれることがあります。これを「引用いんよう」と言いいます。
日本にっぽんの科学技術振興機構かがくぎじゅつしんこうきこうは、どの国くにの論文ろんぶんがいちばん引用いんようされたか、１５１の科学かがくの分野ぶんやについて調しらべました。
２０１５年ねんから２０１７年ねんまで平均へいきんすると、１番ばん多おおかったのはアメリカの論文ろんぶんで、８０の分野ぶんやでいちばん引用いんようされていました。２番目ばんめは中国ちゅうごくの論文ろんぶんで、７１の分野ぶんやでいちばん引用いんようされていました。どの分野ぶんやでもいちばん引用いんようされたのはこの２つの国くにの論文ろんぶんだけでした。
次つぎに、引用いんようされた回数かいすうが上うえから５番目ばんめまでだった論文ろんぶんの国くにを調しらべると、日本にっぽんは１８の分野ぶんやで５番ばん以内いないでした。２０年ねん前まえは８３の分野ぶんやで、そのときよりずっと少すくなくなっています。
科学技術振興機構かがくぎじゅつしんこうきこうでは「日本にっぽんはいい研究けんきゅうをする力ちからが低ひくくなっていることがわかりました。大おおきな問題もんだいだと思おもいます」と話はなしています。

日本赤十字社にっぽんせきじゅうじしゃは、小ちいさな子こどもの命いのちを助たすけるために、「ＡＥＤ」をどのように使つかったらいいか教おしえる会かいを開ひらきました。ＡＥＤは、心臓しんぞうが急きゅうに止とまった人ひとの体からだに電気でんきを流ながすパッドを貼はって、心臓しんぞうを動うごかす機械きかいです。
大阪市おおさかしで開ひらいた会かいには、保育園ほいくえんの先生せんせいなど３３人にんが参加さんかしました。大人おとなとは違ちがって子こどもには、パッドは小ちいさなものを使つかって、胸むねと背中せなかに貼はったほうがいいと説明せつめいがありました。そのあと、赤あかちゃんの人形にんぎょうを使つかって、２本ほんの指ゆびで心臓しんぞうのマッサージをするやり方かたやＡＥＤの使つかい方かたを練習れんしゅうしました。
ＡＥＤは事故じこで心臓しんぞうが急きゅうに止とまった場合ばあいに役やくに立たつため、専門家せんもんかは練習れんしゅうして使つかってほしいと言いっています。
滋賀県しがけん大津市おおつしで今月こんげつの初はじめに交通事故こうつうじこがあって、保育園ほいくえんの子こどもが２人ふたり亡なくなりました。事故じこのときなどに、小ちいさな子こどもをどのように助たすけたらいいか考かんがえる人ひとが増ふえています。

アメリカ「中国ちゅうごくから輸入ゆにゅうする品物しなものほとんどの関税かんぜいを上あげる」

发表于 2019-05-14 更新于 2019-11-30 分类于 JapanEasyNews



アメリカと中国ちゅうごくは、貿易ぼうえきの問題もんだいについて話はなし合あいをしていますが、うまく進すすんでいません。アメリカは１０日とおか、中国ちゅうごくから輸入ゆにゅうしている一部いちぶの品物しなものの関税かんぜいを２５％上あげました。関税かんぜいを上あげた品物しなものを全部ぜんぶ足たすと、２０００億おくドルになります。
中国ちゅうごくも１３日にち、アメリカから輸入ゆにゅうしている一部いちぶの品物しなものの関税かんぜいを、来月らいげつ１日ついたちから上あげると言いいました。関税かんぜいが上あがる品物しなものを足たすと６００億おくドルで、この中なかには２５％高たかくなる品物しなものもあります。
中国ちゅうごくが関税かんぜいを上あげると言いったあと、アメリカはまだ関税かんぜいを上あげていなかった品物しなものも、上あげる準備じゅんびを始はじめたと言いいました。この品物しなものを全部ぜんぶ足たすと、３０００億おくドルになります。
アメリカの政府せいふは来月らいげつに会議かいぎを開ひらいて、いろいろな人ひとの意見いけんを聞きいてから決きめると言いっています。残のこっている品物しなものの関税かんぜいが上あがると、アメリカが中国ちゅうごくから輸入ゆにゅうしているほとんど全部ぜんぶの品物しなものの関税かんぜいが上あがることになります。

奈良市ならし　殺ころした犬いぬや猫ねこが１０年ねんで６６３匹びきから１匹ぴきに減へる

发表于 2019-05-13 更新于 2019-11-30 分类于 JapanEasyNews



奈良市ならしでは、誰だれが飼かっているかわからない犬いぬや猫ねこを殺ころすことがあります。２００８年度ねんどは６６３匹びきいました。
奈良市ならしは１匹ぴきも殺ころさないようにしたいと考かんがえて、４年ねん前まえからウェブサイトで犬いぬや猫ねこを紹介しょうかいして、飼かう人ひとを探さがすことを始はじめました。
その結果けっか、殺ころした犬いぬや猫ねこはとても減へって、２０１７年度ねんどは３匹びき、２０１８年度ねんどは１匹ぴきになりました。
２０１８年度ねんどから奈良市ならしは、保健所ほけんじょが助たすけた犬いぬや猫ねこの赤あかちゃんの世話せわを、しばらくボランティアの人ひとにお願ねがいしています。このやり方かたで新あたらしく飼かう人ひとが見みつかった犬いぬや猫ねこは４２匹ひきいました。
奈良市ならしの市長しちょうは「犬いぬや猫ねこと最後さいごまで一緒いっしょに生活せいかつするという考かんがえ方かたをみんなに持もってほしいです。１匹ぴきも殺ころさないようにするために、これからも頑張がんばります」と話はなしています。

「松屋まつや」　店みせで食たべる場合ばあいと持もって帰かえる場合ばあいの値段ねだんは同おなじ

发表于 2019-05-13 更新于 2019-11-30 分类于 JapanEasyNews



１０月がつから消費税しょうひぜいが８%から１０%に上あがります。食たべ物ものや飲のみ物ものを買かう場合ばあいは８％のままですが、店みせの中なかで食たべたり飲のんだりする場合ばあいは１０%になります。
「松屋まつや」は、牛丼ぎゅうどんなどを店みせの中なかで食たべる場合ばあいと持もって帰かえる場合ばあいの値段ねだんを同おなじにしようと考かんがえています。どちらの値段ねだんにするかはまだ決きまっていません。会社かいしゃは、これから客きゃくの意見いけんなどを聞きいて決きめると言いっています。社長しゃちょうは「値段ねだんが違ちがうと、客きゃくだけではなくて、店みせで働はたらく人ひともどちらの値段ねだんか迷まよう心配しんぱいがあります」と話はなしています。
「吉野家よしのや」は、牛丼ぎゅうどんなどを店みせの中なかで食たべる場合ばあいと持もって帰かえる場合ばあいで、消費税しょうひぜいを入いれた値段ねだんを別べつにすると言いっています。
どちらも牛丼ぎゅうどんの店みせですが、違ちがう値段ねだんの決きめ方かたになりそうです。
