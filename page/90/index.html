「嵐あらし」が来年らいねんでグループの仕事しごとを休やすむ「自由じゆうな生活せいかつをしたい」

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


アイドル グループの「嵐あらし」は、２７日にち、グループでする仕事しごとは来年らいねんの１２月がつ３１日にちまでで、そのあとは休やすむと発表はっぴょうしました。
リーダーの大野おおの智さとしさんは「２０１７年ねん、自分じぶんが『嵐あらし』というグループでする仕事しごとを終おわりにしたいとほかの４人にんに話はなしました。そのあと何なん度ども話はなし合あって、２０２０年ねんで休やすむことになりました」と説明せつめいしました。
大野おおのさんは「自由じゆうな生活せいかつをしてみたいです。この仕事しごとを一度いちど離はなれて、今いままで見みたことがない景色けしきを見みてみたいです」と理由りゆうを説明せつめいしました。そして、芸能げいのう人じんの仕事しごとも休やすみたいと話はなしました。
「嵐あらし」の二宮にのみや和也かずなりさん、櫻井さくらい翔しょうさん、相葉あいば雅紀まさきさん、松本まつもと潤じゅんさんは「リーダーの気持きもちを大切たいせつにして決きめたことです」、「２０２０年ねんまで、５人にんでファンのみなさんと楽たのしい時間じかんを過すごすことができるようにしたいです」と話はなしました。

３世代せだいで住すむ家族かぞくを助たすける制度せいどがある県けんと府ふ　５年ねん前まえの７倍ばい

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


２０１５年ねんに国くにが調しらべると、親おやと子こと孫まごの３世代せだいが一緒いっしょに住すむ家族かぞくの割合わりあいは５．７％で、２０年ねんの間あいだに半分はんぶんに減へりました。
ＮＨＫは４７の都道府県とどうふけんに、３世代せだいが一緒いっしょに住すむ家族かぞくを助たすける 制度せいどがあるかどうか聞ききました。その結果けっか、１４の県けんと府ふが、家いえを建たてるときに必要ひつようなお金かねの一部いちぶを出だしたり、税金ぜいきんを安やすくしたりする制度せいどがあると答こたえました。
鳥取県とっとりけんは２００８年ねんから、次つぎの福井県ふくいけんは２０１３年ねんからこのような制度せいどを始はじめています。このような制度せいどがある県けんと府ふは、５年ねん前まえの７倍ばいに増ふえました。
日本にっぽんは子こどもが少すくなくなって、人口じんこうが減へっています。３世代せだいが一緒いっしょに住すむと、祖父そふや祖母そぼが手伝てつだうことができるため、子こどもを育そだてるのが楽らくになると言いわれています。県けんなどは３世代せだいで住すむ家族かぞくを増ふやして、子こどもが減へらないようにしようと考かんがえているようです。

アメリカのボーイングが「空そらを飛とぶタクシー」のテスト

发表于 2019-01-25 更新于 2019-11-30 分类于 JapanEasyNews


飛行機ひこうきをつくっているアメリカの会社かいしゃのボーイングは、パイロットがいなくても自動じどうで空そらを飛とぶことができる乗のり物もののテストを行おこないました。
この乗のり物ものは長ながさと幅はばが９ｍぐらいで、プロペラが付ついています。テストではヘリコプターのように１０ｍ以上いじょうの高たかさまでまっすぐ上あがりました。そして、空そらでしばらく止とまったあと、下したに降おりることができました。

ボーイングは将来しょうらい、客きゃくだけを乗のせて飛とんだり、大おおきな荷物にもつを運はこんだりできるようにしたいと考かんがえています。自動じどうで飛とぶ乗のり物ものは「空そらを飛とぶタクシー」と言いわれていて、世界せかいでは競争きょうそうが激はげしくなっています。

アメリカの科学かがく雑誌ざっしは、地球ちきゅうの最後さいごの日ひまでどのくらい時間じかんが残のこっているか「終末時計しゅうまつどけい」で表あらわしています。この雑誌ざっしは２４日にじゅうよっか、残のこっている時間じかんは去年きょねんと同おなじ「あと２分ふん」だと言いいました。
