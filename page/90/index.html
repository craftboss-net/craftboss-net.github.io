深ふかい海うみにいる「リュウグウノツカイ」が兵庫県ひょうごけんで見みつかる

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


兵庫県ひょうごけん豊岡市とよおかしの海うみで２３日にち、魚さかなをとる網あみの中なかに「リュウグウノツカイ」という魚さかなが入はいっていました。体からだの長ながさは４ｍ４２ｃｍで、重おもさは３２ｋｇありました。
リュウグウノツカイは、世界せかいの海うみの深ふかさが２００ｍ以上いじょうの所ところにいると言いわれていますが、どんな魚さかなかよくわかっていません。
豊岡市とよおかしにある水族館すいぞくかんの「城崎きのさきマリンワールド」では、この魚さかなが何なにを食たべているのかおなかの中なかを調しらべます。そして来月らいげつ、この魚さかなが生いきていたときと同おなじような形かたちにして、みんなに見みせる予定よていです。
水族館すいぞくかんの人ひとは「４ｍ以上いじょうのリュウグウノツカイが見みつかることは珍めずらしいのでびっくりしています。どんな魚さかなか少すこしでもわかるように調しらべたいです」と話はなしていました。

日本にっぽんと中国ちゅうごくの間あいだを飛とぶ飛行機ひこうきが２５％以上いじょう増ふえる

发表于 2019-01-25 更新于 2019-11-30 分类于 JapanEasyNews


中国ちゅうごくの政府せいふは今月こんげつ、新あたらしい国際線こくさいせんを８０認みとめました。このうち４６は、中国ちゅうごくと日本にっぽんの間あいだを飛とぶ飛行機ひこうきです。
日本にっぽんと中国ちゅうごくの間あいだは、今いま１週間しゅうかんに１０００以上いじょうの飛行機ひこうきが飛とんでいます。認みとめられた飛行機ひこうきが全部ぜんぶ飛とび始はじめると１週間しゅうかんに２７７増ふえて、割合わりあいでは２５％以上いじょう多おおくなります。
新あたらしくできる国際線こくさいせんは、札幌さっぽろと山東省さんとうしょう青島ちんたおの間あいだ、福岡ふくおかと天津てんしんの間あいだなど地方ちほうの空港くうこうが多おおくなっています。
去年きょねん、日本にっぽんへ旅行りょこうに行いった中国人ちゅうごくじんは８３８万まん人にんで、２０１３年ねんの６倍ばい以上いじょうに増ふえました。日本にっぽんへ旅行りょこうに行いく人ひとが増ふえているため、飛行機ひこうきを増ふやしたようです。

アメリカのボーイングが「空そらを飛とぶタクシー」のテスト

发表于 2019-01-25 更新于 2019-11-30 分类于 JapanEasyNews


飛行機ひこうきをつくっているアメリカの会社かいしゃのボーイングは、パイロットがいなくても自動じどうで空そらを飛とぶことができる乗のり物もののテストを行おこないました。
この乗のり物ものは長ながさと幅はばが９ｍぐらいで、プロペラが付ついています。テストではヘリコプターのように１０ｍ以上いじょうの高たかさまでまっすぐ上あがりました。そして、空そらでしばらく止とまったあと、下したに降おりることができました。
テストで使つかった乗のり物ものには２人ふたり乗のることができますが、ボーイングは、４人にん乗のることができる乗のり物ものも研究けんきゅうしています。これからは、まっすぐ上あがってから飛行機ひこうきのように飛とぶテストを行おこなう予定よていです。
ボーイングは将来しょうらい、客きゃくだけを乗のせて飛とんだり、大おおきな荷物にもつを運はこんだりできるようにしたいと考かんがえています。自動じどうで飛とぶ乗のり物ものは「空そらを飛とぶタクシー」と言いわれていて、世界せかいでは競争きょうそうが激はげしくなっています。

アメリカの科学かがく雑誌ざっし「地球ちきゅうの最後さいごの日ひまであと２分ふん」

发表于 2019-01-25 更新于 2019-11-30 分类于 JapanEasyNews


アメリカの科学かがく雑誌ざっしは、地球ちきゅうの最後さいごの日ひまでどのくらい時間じかんが残のこっているか「終末時計しゅうまつどけい」で表あらわしています。この雑誌ざっしは２４日にじゅうよっか、残のこっている時間じかんは去年きょねんと同おなじ「あと２分ふん」だと言いいました。
「あと２分ふん」は今いままでで最もっとも短みじかい時間じかんで、アメリカとソビエトが核兵器かくへいきの実験じっけんを行おこなった１９５３年ねんと同おなじです。
アメリカは去年きょねん５月がつ、核兵器かくへいきについてイランと約束やくそくしたグループから出でました。アメリカと北朝鮮きたちょうせんとの間あいだで核兵器かくへいきをなくすための話はなし合あいも進すすんでいません。このため、危険きけんな状態じょうたいになっていると言いっています。
アメリカやロシア、サウジアラビアなどが地球ちきゅうの気温きおんが上あがらないようにするために協力きょうりょくしないことも理由りゆうだと言いっています。

東京都とうきょうとの上野動物園うえのどうぶつえんでは、１９５７年ねんに日本にっぽんで初はじめてモノレールの運転うんてんを始はじめました。動物園どうぶつえんの中なかの３３０ｍぐらいを走はしっていて、乗のると、動物園どうぶつえんを高たかい所ところから見みて楽たのしむことができます。とても人気にんきがあって、１年ねんに１００万まん人にんぐらいが乗のっています。

東京とうきょうメトロの中目黒駅なかめぐろえきで２２日にちの朝あさ、３７歳さいの会社員かいしゃいんの女性じょせいがホームから落おちて、電車でんしゃにはねられて亡なくなりました。
事故じこのあと警察けいさつが調しらべると、女性じょせいの体からだからインフルエンザのウイルスが見みつかりました。警察けいさつによると、女性じょせいは何なん日にちか前まえから具合ぐあいが悪わるいと言いっていましたが、インフルエンザになっていると思おもわないで会社かいしゃに行いこうとしていました。

専門家せんもんかは「インフルエンザになると高たかい熱ねつが出でて、体からだの水分すいぶんが足たりなくなって、倒たおれることがあります。インフルエンザかもしれないと思おもったら、出でかけないで家いえで休やすんでほしい」と話はなしています。
