東京とうきょうオリンピックのとき会社かいしゃに行いかないで仕事しごとをする

发表于 2019-07-08 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックまで１年ねんです。オリンピックの間あいだ、道みちや電車でんしゃが混こむ心配しんぱいがあるため、政府せいふは会社かいしゃに、家いえなど会社かいしゃ以外いがいの場所ばしょで仕事しごとをすることを考かんがえるように言いっています。
ホンダは、東京とうきょうの港区みなとくにある会社かいしゃに行いかないで仕事しごとができるようにします。１２００人にんぐらいの社員しゃいんは、家いえで仕事しごとをしたり、埼玉県さいたまけんにあるホンダの事務所じむしょなどで仕事しごとをしたりします。
トヨタ自動車じどうしゃも、東京とうきょうで働はたらく１２００人にんぐらいは家いえで仕事しごとができるようにします。ホンダとトヨタ自動車じどうしゃは、家いえでうまく仕事しごとができるかどうか、今月こんげつテストする予定よていです。
パソコンをつくっているレノボ?ジャパンは、オリンピックの間あいだ、社員しゃいんが特別とくべつな休やすみを取とるようにします。そして、家いえなどで仕事しごとをして、２０００人にんが会社かいしゃに行いかなくてもいいようにします。
このように、会社かいしゃ以外いがいの場所ばしょで仕事しごとができるようにする会社かいしゃが、どのくらい増ふえるか注目ちゅうもくされています。

「ユニクロ」の会社かいしゃは、プラスチックのごみを減へらすため、今年ことし９月がつから客きゃくが買かった物ものを入いれる袋ふくろを紙かみに変かえます。１２の国くにや地域ちいきにある店みせで紙かみの袋ふくろを使つかいます。
ユニクロでは袋ふくろに入いれて売うっている物ものもあります。この袋ふくろもプラスチック以外いがいに変かえることを考かんがえています。そして、買かった物ものを入いれる袋ふくろを自分じぶんで持もってきてもらうため、日本にっぽんにある店みせでは、来年らいねん１月がつから紙かみの袋ふくろを１枚まい１０円えんで売うる予定よていです。会社かいしゃは来年らいねんまでに、買かい物ものの袋ふくろなどのプラスチックを８５％減へらしたいと考かんがえています。
服ふくを売うる会社かいしゃでは、スウェーデンの「Ｈ＆Ｍ」も、日本にっぽんにある店みせでプラスチックの袋ふくろを紙かみに変かえました。プラスチックのごみを減へらそうと考かんがえる会社かいしゃが増ふえています。

参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまる

发表于 2019-07-04 更新于 2019-11-30 分类于 JapanEasyNews



４日よっか、参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまりました。投票とうひょうは７月がつ２１日にちです。
参議院さんぎいんは３年ねんに１度ど、半分はんぶんの数かずの議員ぎいんを選挙せんきょで選えらびます。去年きょねん、法律ほうりつが変かわって議員ぎいんを増ふやしたため、前まえの選挙せんきょより３人にん多おおい１２４人にんを選えらびます。
参議院さんぎいんの議員ぎいんは２つの方法ほうほうで選えらびます。都道府県とどうふけんの中なかで議員ぎいんを選えらぶ「選挙区せんきょく」の選挙せんきょでは、７４人にんを選えらびます。人口じんこうが少すくない県けんでは、２つの県けんで１人ひとりを選えらぶところもあります。政党せいとうと立候補りっこうほした人ひとが取とった票ひょうの割合わりあいで議員ぎいんの数かずが決きまる「比例代表ひれいだいひょう」の選挙せんきょでは、５０人にんを選えらびます。
３７０人にんが立候補りっこうほしました。立候補りっこうほした人ひとたちは、安倍あべ総理大臣そうりだいじんの政治せいじや、１０月がつに消費税しょうひぜいを１０％に上あげること、年金ねんきんの問題もんだいなどについて意見いけんを言いって、選挙せんきょを戦たたかいます。
