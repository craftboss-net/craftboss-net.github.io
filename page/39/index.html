東京とうきょうオリンピックの組織委員会そしきいいんかい「２回かい目めの抽ちゅうせんをします」

发表于 2019-07-05 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックの組織委員会そしきいいんかいは先月せんげつ、チケットの抽ちゅうせんを行おこないました。大勢おおぜいの人ひとが申もうし込こんだので、当あたらなかった人ひとがたくさんいました。
このため組織委員会そしきいいんかいは、１枚まいも当あたらなかった人ひとのために、２回かい目めの抽ちゅうせんをすると発表はっぴょうしました。
この抽ちゅうせんで売うるのは、最初さいしょの抽ちゅうせんで申もうし込こみが少すくなかったチケットや、当あたった人ひとがお金かねを払はらわなかったチケットです。２回かい目めの抽ちゅうせんは、８月がつに行おこなう予定よていです。申もうし込こむことができるのは、１人ひとり１種類しゅるいだけです。
組織委員会そしきいいんかいは今年ことしの秋あきから、早はやく申もうし込こんだ人ひとから順番じゅんばんにチケットを売うる予定よていでしたが、抽ちゅうせんにすることを考かんがえています。

５７％の家庭かていが「生活せいかつが苦くるしい」

发表于 2019-07-04 更新于 2019-11-30 分类于 JapanEasyNews



厚生労働省こうせいろうどうしょうは去年きょねん７月がつ、家庭かていの収入しゅうにゅうがおととしどのくらいだったか聞ききました。６０００ぐらいの家庭かていが答こたえました。
おととしの家庭かていの収入しゅうにゅうは平均へいきんで５５１万まん６０００円えんで、その前まえの年としより８万まん６０００円えん少すくなくなりました。
収入しゅうにゅうが平均へいきんより少すくない家庭かていは６０％ぐらいで、今いままででいちばん多おおくなりました。
生活せいかつについて聞きくと、５７％の家庭かていが、お金かねが足たりなくて生活せいかつが「苦くるしい」と答こたえました。特とくに、子こどもがいる家庭かていでは６２％、お年寄としよりの家庭かていでは５５％が「苦くるしい」と答こたえました。
お年寄としよりの家庭かていの収入しゅうにゅうは、全体ぜんたいの平均へいきんより２００万まん円えん以上いじょう少すくなくなっています。厚生労働省こうせいろうどうしょうは、お年寄としよりが多おおくなって、収入しゅうにゅうが少すくない家庭かていが増ふえたため、全体ぜんたいの収入しゅうにゅうの平均へいきんが少すくなくなったと言いっています。

雨あめがとてもたくさん降ふった鹿児島県かごしまけんなどでは、地面じめんの中なかに水みずがたくさん入はいっているので、山やまや崖がけなどが崩くずれる心配しんぱいがあります。
鹿児島県かごしまけんと宮崎県みやざきけんでは先月せんげつ２８日にちから降ふった雨あめが、いつもの年としの７月がつの２倍ばい以上いじょうになっている場所ばしょがあります。

気象庁きしょうちょうは「鹿児島県かごしまけんなど九州きゅうしゅうの南みなみ側がわは、少すこしの雨あめや雨あめがやんだあとでも、山やまなどが崩くずれるかもしれません。山やまや崖がけの近ちかくに行いかないでください」と言いっています。

参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまる

发表于 2019-07-04 更新于 2019-11-30 分类于 JapanEasyNews



４日よっか、参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょが始はじまりました。投票とうひょうは７月がつ２１日にちです。
参議院さんぎいんは３年ねんに１度ど、半分はんぶんの数かずの議員ぎいんを選挙せんきょで選えらびます。去年きょねん、法律ほうりつが変かわって議員ぎいんを増ふやしたため、前まえの選挙せんきょより３人にん多おおい１２４人にんを選えらびます。
参議院さんぎいんの議員ぎいんは２つの方法ほうほうで選えらびます。都道府県とどうふけんの中なかで議員ぎいんを選えらぶ「選挙区せんきょく」の選挙せんきょでは、７４人にんを選えらびます。人口じんこうが少すくない県けんでは、２つの県けんで１人ひとりを選えらぶところもあります。政党せいとうと立候補りっこうほした人ひとが取とった票ひょうの割合わりあいで議員ぎいんの数かずが決きまる「比例代表ひれいだいひょう」の選挙せんきょでは、５０人にんを選えらびます。
３７０人にんが立候補りっこうほしました。立候補りっこうほした人ひとたちは、安倍あべ総理大臣そうりだいじんの政治せいじや、１０月がつに消費税しょうひぜいを１０％に上あげること、年金ねんきんの問題もんだいなどについて意見いけんを言いって、選挙せんきょを戦たたかいます。
