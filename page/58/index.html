渋谷駅しぶやえきの近ちかくで外国人がいこくじんがドローンを飛とばす　警察けいさつが注意ちゅうい

发表于 2019-05-13 更新于 2019-11-30 分类于 JapanEasyNews



１２日にちの午後ごご、東京とうきょうの渋谷駅しぶやえきの近ちかくの交差点こうさてんの上うえをドローンが飛とんでいました。大勢おおぜいの人ひとが交差点こうさてんを渡わたっているときに突然とつぜん飛とんできました。
ドローンは１０分ぷんぐらい飛とんでいました。ビルの８階かいぐらいの高たかさまで上あがったり、駅えきの前まえにある有名ゆうめいな「忠犬ちゅうけんハチ公こう」という犬いぬの像ぞうの上うえを飛とんだりしていました。
ドローンを飛とばしていたのは、旅行りょこうに来きた外国人がいこくじんの男性だんせいでした。警察けいさつは男性だんせいに、ここはドローンを飛とばしてはいけない場所ばしょだと注意ちゅういしました。
日本にっぽんの法律ほうりつでは、人ひとが多おおく集あつまる場所ばしょなどでドローンを飛とばしてはいけないと決きまっています。ドローンを飛とばしていた男性だんせいはこのことを知しらなかったと言いっています。
警察けいさつは、今月こんげつの初はじめに皇居こうきょの周まわりをドローンのような物ものが飛とんでいたことと、この男性だんせいは関係かんけいがないと言いっています。

发表于 2019-05-10 更新于 2019-11-30 分类于 JapanEasyNews



鉄道てつどう会社がいしゃの東急電鉄とうきゅうでんてつは８日ようか、駅えきで、切符きっぷを買かう機械きかいから銀行ぎんこうの口座こうざのお金かねを出だすことができるサービスを始はじめました。ゆうちょ銀行ぎんこうと横浜銀行よこはまぎんこうに口座こうざを持もっている人ひとがこのサービスを利用りようできます。
利用りようする人ひとは、まず銀行ぎんこうのアプリを入いれたスマートフォンの画面がめんで、出だしたいお金かねを１万まん円えん、２万まん円えん、３万まん円えんの３つの中なかから選えらびます。そして、画面がめんのＱＲコードを、切符きっぷを買かう機械きかいのセンサーの前まえに出だすと、お金かねが出でます。利用りようするときは１回かい１００円えんから２００円えんぐらいかかります。

東急電鉄とうきゅうでんてつは「利用りようできる銀行ぎんこうを多おおくして、ほかの鉄道てつどう会社がいしゃでもこのサービスを利用りようできるようにしたいです」と話はなしています。
