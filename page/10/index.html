北朝鮮きたちょうせん　ミサイルは潜水艦せんすいかんから発射はっしゃするタイプだと発表はっぴょう

发表于 2019-10-03 更新于 2019-11-30 分类于 JapanEasyNews



２日ふつかの朝あさに発射はっしゃしたミサイルについて、北朝鮮きたちょうせんは３日みっか、朝鮮労働党ちょうせんろうどうとうの新聞しんぶんで、ミサイルは潜水艦せんすいかんから発射はっしゃするタイプだと発表はっぴょうしました。
新聞しんぶんには、海うみから出でたミサイルがオレンジ色いろの火ひを出だしながら空そらへ上あがっていく写真しゃしんが出でています。新聞しんぶんは「軍ぐんを強つよくするための新あたらしい力ちからを持もつことができた」と書かいています。
専門家せんもんかは「潜水艦せんすいかんから発射はっしゃするミサイルは、発射はっしゃした場所ばしょを探さがすのが難むずかしいです。日本にっぽんやほかの国くにの危険きけんが大おおきくなります」と言いっています。
国連こくれんのグテーレス事務総長じむそうちょうは「とても心配しんぱいしています。国連こくれんが決きめたことに違反いはんしています」と言いいました。北朝鮮きたちょうせんとアメリカは５日いつかに話はなし合あう予定よていです。グテーレス事務総長じむそうちょうは「話はなし合あいをして、いい結果けっかを出だしてほしいです」と言いいました。

外国人がいこくじんが日本にっぽんで働はたらくための新あたらしい資格しかく　６か月げつで３００人にんだけ

发表于 2019-10-02 更新于 2019-11-30 分类于 JapanEasyNews



今年ことし４月がつ、日本にっぽんで働はたらく外国人がいこくじんを増ふやすために「特定とくてい技能ぎのう」という新あたらしい資格しかくができました。この中なかの「特定とくてい技能ぎのう１号ごう」という資格しかくを取とった外国人がいこくじんは、ビルなどを建たてる仕事しごとやお年寄としよりの介護かいごの仕事しごとなどをすることができます。
日本にっぽんの政府せいふは、この資格しかくを来年らいねん３月がつまでの１年ねんに４万まん７０００人にんぐらいに出だそうと考かんがえています。しかし、今年ことし９月がつまでに申もうし込こんだ人ひとは２０６２人にんで、資格しかくを取とった人ひとは３００人にんだけでした。
政府せいふは、この資格しかくを取とる人ひとが増ふえるように、申もうし込こんだあと早はやく資格しかくを取とることができるようにしようと考かんがえています。この資格しかくでできる仕事しごとの種類しゅるいを多おおくすることも考かんがえています。
法務省ほうむしょうの河井かわい大臣だいじんは「外国人がいこくじんは日本にっぽんの社会しゃかいに必要ひつような人ひとたちだと考かんがえることが大切たいせつです。外国人がいこくじんが働はたらいたり生活せいかつしたりしやすい国くににしたいと思おもいます」と話はなしています。

困こまっている人ひとを助たすける「赤あかい羽根はね共同募金きょうどうぼきん」が始はじまる

发表于 2019-10-02 更新于 2019-11-30 分类于 JapanEasyNews



１０月がつ１日ついたち、「赤あかい羽根はね共同募金きょうどうぼきん」が始はじまりました。この募金ぼきんは１９４７年ねんに始はじまって、町まちをよくしたり困こまっている人ひとを助たすけたりするために寄付きふを集あつめています。たくさんの人ひとが通とおる場所ばしょや、コンビニなどで寄付きふをすると、赤あかい羽はねをもらうことができます。インターネットで寄付きふをすることもできます。
東京とうきょうでは１日ついたち、この募金ぼきんのポスターに出でている桜井さくらい日奈子ひなこさんや、先月せんげつ大相撲おおずもうで優勝ゆうしょうした御嶽海みたけうみが寄付きふを集あつめました。来年らいねん３月がつまでに１９３億おく円えん集あつめて、お年寄としよりや障害しょうがいがある人ひと、台風たいふうで大おおきな被害ひがいがあった千葉県ちばけんの人ひとなどのために使つかう予定よていです。
桜井さくらいさんは「困こまっている人ひとを助たすけたいと思おもっていても、どうしたらいいかわからない人ひとがたくさんいると思おもうので、頑張がんばって寄付きふを集あつめたいと思おもいます」と話はなしていました。

「ＡＩで新あたらしい星ほしを見みつける」今月こんげつから夜よるの空そらを調しらべ始はじめる

发表于 2019-10-02 更新于 2019-11-30 分类于 JapanEasyNews



東京大学とうきょうだいがくなどの研究けんきゅうチームは、特別とくべつな望遠鏡ぼうえんきょうとＡＩを使つかって、新あたらしい星ほしなどを見みつけるためのシステムを作つくりました。長野県ながのけん木曽町きそまちにある天文台てんもんだいで今月こんげつから夜よるの空そらを調しらべ始はじめます。
望遠鏡ぼうえんきょうにはたくさんのセンサーが付ついていて、夜よるの空そらの全体ぜんたいを２時間じかんで撮とります。そして、撮とった映像えいぞうをＡＩで調しらべて、明あかるさや場所ばしょが前まえと違ちがっている星ほしなどを見みつけます。一晩ひとばんに撮とる映像えいぞうのデータは１万まん本ぼんの映画えいがと同おなじぐらい多おおいため、ＡＩを使つかうことで初はじめて調しらべることができるようになりました。
研究けんきゅうチームは、急きゅうに明あかるく光ひかる「超新星ちょうしんせい」を１年ねんに１０００個こぐらい、ほかの星ほしの周まわりを回まわる「惑星わくせい」を１年ねんに１００個こぐらい見みつけたいと言いっています。

地震じしんで壊こわれた熊本城くまもとじょう　直なおした建物たてものを近ちかくで見みることができる

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



熊本市くまもとしにある熊本城くまもとじょうは、４００年ねん以上いじょうの歴史れきしがある城しろです。３年ねん前まえの大おおきな地震じしんで壊こわれた建物たてものを直なおしているため、今いまも入はいることができない場所ばしょがあります。
城しろでいちばん高たかい「大天守だいてんしゅ」という建物たてものの外側そとがわがほとんど直なおったため、１０月がつ５日いつかから近ちかくで見みることができるようになります。敵てきが登のぼりにくいようにつくった「武者むしゃ返がえし」という石垣いしがきも、すぐ近ちかくで見みることができるようになります。
熊本城くまもとじょうの事務所じむしょは「みなさんのおかげです。少すこしずつですが、前まえのような形かたちに戻もどっているのでぜひ見みに来きてほしいです」と話はなしていました。
熊本城くまもとじょうの大天守だいてんしゅは、１０月がつ５日いつかから１４日じゅうよっかまでは毎日まいにち、そのあとは、来年らいねん３月がつの終おわりまで日曜日にちようびと祝日しゅくじつに見みることができます。

日本にっぽんで働はたらく外国人がいこくじんが増ふえて、外国人がいこくじんの子こどもも増ふえています。国くには、日本にっぽんで生活せいかつしている外国人がいこくじんの子こどもが学校がっこうに通かよっているかどうか初はじめて調査ちょうさしました。
調査ちょうさによると、今年ことし５月がつ、６歳さいから１４歳さいまでの外国人がいこくじんの子こどもは日本にっぽんに１２万まん４０００人にんぐらいいました。この中なかで小学校しょうがっこうや中学校ちゅうがっこうや外国人がいこくじん学校がっこうなどに通かよっていない子こどもが１０００人にんいました。そして、学校がっこうに通かよっているかどうかはっきりわからない子こどもが１万まん９０００人にんぐらいいました。国くには「全部ぜんぶで２万まん人にんぐらいの子こどもが学校がっこうに通かよっていない可能性かのうせいがある」と言いっています。
専門家せんもんかは「驚おどろきました。外国人がいこくじんの子こどもはこれからもっと増ふえます。しっかり調査ちょうさを続つづけてほしいです」と話はなしています。

「核兵器かくへいきを禁止きんしする条約じょうやくに入はいってほしい」署名しょめいを国連こくれんに渡わたす

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんで２年ねん前まえ、核兵器かくへいきを作つくったり持もったりすることを禁止きんしする条約じょうやくができました。今年ことしの９月がつまでに３２の国くにがこの条約じょうやくに入はいることを決きめました。しかし、アメリカなど核兵器かくへいきを持もっている国くにや日本にっぽんは入はいっていません。
広島ひろしまや長崎ながさきで原爆げんばくの被害ひがいを受うけた人ひとたちでつくる団体だんたいの日本被団協にほんひだんきょうなどは、「全部ぜんぶの国くにがこの条約じょうやくに入はいってほしい」と言いって、町まちやインターネットで署名しょめいを集あつめました。世界中せかいじゅうから１０５１万まん人にんの署名しょめいが集あつまりました。
日本被団協にほんひだんきょうは１０月がつにニューヨークの国連こくれんに行いって、兵器へいきを減へらすことなどについて話はなし合あう会議かいぎの議長ぎちょうに署名しょめいを渡わたします。
日本被団協にほんひだんきょうは「たくさん署名しょめいが集あつまって、うれしいです。この条約じょうやくが大おおきな力ちからを持もつことができるように、これからも頑張がんばります」と話はなしています。
