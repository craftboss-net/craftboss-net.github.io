東京とうきょうパラリンピック　聖火せいかリレーに使つかうトーチができる

发表于 2019-03-26 更新于 2019-11-30 分类于 JapanEasyNews



２０２０年ねんに東京とうきょうで行おこなうパラリンピックの聖火せいか リレーに使つかうトーチができました。桜さくらの花はなをデザインしたトーチです。オリンピックのトーチと同おなじ形かたちですが、色いろが少すこし違ちがいます。パラリンピックのトーチは桜さくらの花はなのようなピンク色いろです。
手てで持もつ所ところには、「勇気ゆうき」、「公平こうへい」などパラリンピックで大切たいせつにしていることを表あらわすことばが、目めが見みえない人ひとにわかるように点字てんじでかいてあります。
パラリンピックの聖火せいか リレーは、競技きょうぎを行おこなう静岡県しずおかけん、千葉県ちばけん、埼玉県さいたまけん、東京都とうきょうとで８月がつ１８日にちから２５日にちまで行おこないます。３人にんが１本ぽんずつトーチを持もって一緒いっしょに走はしります。

インドネシア　初はじめての地下鉄ちかてつの運転うんてんが始はじまる

发表于 2019-03-26 更新于 2019-11-30 分类于 JapanEasyNews



２５日にち、インドネシアに初はじめてできた地下鉄ちかてつの運転うんてんが始はじまりました。この地下鉄ちかてつは首都しゅとのジャカルタの中心ちゅうしんを通とおる１６ｋｍぐらいを走はしります。駅えきは１３あります。日本にっぽんの政府せいふがお金かねを貸かしたり、日本にっぽんの会社かいしゃが工事こうじに協力きょうりょくしたりして、地下鉄ちかてつができました。
２５日にちは、地下鉄ちかてつに乗のってみたいという人ひとなどが大勢おおぜい駅えきに来きました。駅員えきいんは、電車でんしゃを待まっている人ひとたちに「ホームにかいてある線せんの所ところで待まってください」とか「人ひとが降おりてから乗のりましょう」などと言いっていました。
仕事しごとに行いくために地下鉄ちかてつに乗のった男性だんせいは「時間じかん通どおりに行いくことができて、とても便利べんりです」と話はなしていました。
地下鉄ちかてつの料金りょうきんはまだ決きまっていないため、しばらくは無料むりょうで乗のることができます。

「幸しあわせな国くに」のランキング　日本にっぽんは去年きょねんより下さがって５８番ばん

发表于 2019-03-25 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんは、７年ねん前まえから「幸しあわせな国くに」のランキングを発表はっぴょうしています。１５６の国くにと地域ちいきの経済けいざいや健康けんこう、自由じゆうなどについて調しらべて点数てんすうを計算けいさんしました。
今年ことしのランキングでは、フィンランドが２年ねん続つづけて１番ばんで、２番ばんがデンマーク、３番ばんがノルウェーでした。十分じゅうぶんな福祉ふくしや教育きょういくが行おこなわれている北きたヨーロッパの国くにが高たかくなりました。
イギリスは１５番ばん、アメリカは１９番ばんで、日本にっぽんは去年きょねんより４つ下さがって５８番ばんでした。Ｇ７の７つの先進国せんしんこくの中なかで、日本にっぽんは最もっとも低ひくいランキングでした。
日本にっぽんは「何なん年ねん健康けんこうに生いきることができるか」は高たかい点数てんすうでしたが、「自由じゆうがある社会しゃかいかどうか」や「ほかの人ひとへの優やさしさ」は低ひくい点数てんすうでした。

日本にっぽんで働はたらく外国人がいこくじんの生活せいかつのためのサービスが増ふえている

发表于 2019-03-25 更新于 2019-11-30 分类于 JapanEasyNews



来月らいげつから法律ほうりつが変かわって、今いまよりたくさんの外国人がいこくじんが日本にっぽんで働はたらくことができるようになります。
このため、いろいろな会社かいしゃが外国人がいこくじんの生活せいかつのためのサービスを始はじめます。外国人がいこくじんがビザを取とるときの手伝てつだいをしている会社かいしゃは、外国人がいこくじんが日本にっぽんに来きてすぐ銀行ぎんこうの口座こうざを作つくることができるようにします。外国人がいこくじんが保険ほけんに入はいったり、住すむ家いえを探さがしたりするためのサービスも始はじめようと考かんがえています。
日本にっぽんに住すむとき、市しや区くなどの役所やくしょに書類しょるいを出だしたり、電気でんきやガスを使つかったりできるように手伝てつだうサービスを始はじめる会社かいしゃもあります。
政府せいふはこれから５年ねんで、日本にっぽんで働はたらく外国人がいこくじんを今いまより３４万まん５０００人にん多おおくしようと考かんがえています。外国人がいこくじんの生活せいかつのためのサービスはもっと増ふえそうです。

乗のる前まえに料金りょうきんを決きめるタクシー　早はやい場合ばあい今年ことし始はじまる予定よてい

发表于 2019-03-25 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの決きまりでは、タクシーの料金りょうきんを乗のる前まえに決きめることができません。しかし、旅行りょこうに来きた外国人がいこくじんの中なかには「タクシーを降おりるまでいくらになるかわからないので心配しんぱいだ」と言いう人ひともいます。
このため国土交通省こくどこうつうしょうは、国くにの許可きょかをもらった会社かいしゃは、乗のる前まえに料金りょうきんを決きめるサービスを始はじめてもいいことにしました。
このサービスでは、スマートフォンなどのアプリを使つかいます。乗のる人ひとが乗のる場所ばしょと行いく場所ばしょを入いれると、料金りょうきんがわかります。この料金りょうきんでいいと思おもったら、申もうし込こんでタクシーに来きてもらいます。道みちが混こんでいて時間じかんがかかっても料金りょうきんは高たかくなりません。いくらになるか心配しんぱいしないで、タクシーに乗のることができます。料金りょうきんは、タクシーの会社かいしゃが今いままでの料金りょうきんを調しらべて決きめます。
このサービスは、早はやい場合ばあい、今年ことし始はじまる予定よていです。

鯨くじらのおなかから見みつかったプラスチックを博物館はくぶつかんで展示てんじ

发表于 2019-03-25 更新于 2019-11-30 分类于 JapanEasyNews



去年きょねん８月がつ、神奈川県かながわけんの海岸かいがんで死しんでいた鯨くじらのおなかの中なかから、プラスチックが見みつかりました。東京とうきょうの上野うえのにある国立科学博物館こくりつかがくはくぶつかんは、この鯨くじらの頭あたまの骨ほねや、おなかの中なかから見みつかったプラスチックなどを見みせる展示てんじを始はじめました。
プラスチックは透明とうめいで、３ｃｍぐらいの大おおきさに折おってありました。広ひろげると７ｃｍぐらいの長ながさになります。この鯨くじらはお母かあさんのミルクを飲のんでいる赤あかちゃんで、間違まちがえて海うみにあったプラスチックを飲のんだと考かんがえられています。博物館はくぶつかんでは、別べつの鯨くじらの体からだの中なかから見みつかったプラスチックもたくさん見みせています。
小学校しょうがっこう４年生ねんせいの女おんなの子こは「人ひとが捨すてたごみを鯨くじらが食たべてしまって、かわいそうだと思おもいました」と話はなしていました。
博物館はくぶつかんの人ひとは「海うみがプラスチックでひどく汚よごれていることをたくさんの人ひとに知しってほしいです」と話はなしていました。
この展示てんじは、６月がつ１６日にちまでです。
