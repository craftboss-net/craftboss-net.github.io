参議院さんぎいんの選挙せんきょ　自民党じみんとうと公明党こうめいとうで７１人にん選えらばれる

发表于 2019-07-22 更新于 2019-11-30 分类于 JapanEasyNews



２１日にち、参議院さんぎいんの議員ぎいんを選えらぶ選挙せんきょの投票とうひょうがあって、１２４人にんの議員ぎいんが決きまりました。
自民党じみんとうが５７人にん、立憲民主党りっけんみんしゅとうが１７人にん、国民民主党こくみんみんしゅとうが６人にん、公明党こうめいとうが１４人にん、共産党きょうさんとうが７人にん、日本にっぽん維新いしんの会かいが１０人にん、社民党しゃみんとうが１人ひとり、れいわ新選組しんせんぐみが２人ふたり、ＮＨＫから国民こくみんを守まもる党とうが１人ひとり、どの政党せいとうでもない人ひとが９人にんでした。
今いまの政府せいふをつくっている自民党じみんとうと公明党こうめいとうは、１２４人にんの半分はんぶんより多おおい７１人にんが選えらばれて、勝かちました。しかし、憲法けんぽうを変かえたいと考かんがえるほかの政党せいとうや、選挙せんきょがなかった議員ぎいんなどを足たしても１６０人にんで、参議院さんぎいんで憲法けんぽうを変かえる話はなし合あいを始はじめるために必要ひつような１６４人にんには足たりませんでした。
この選挙せんきょは、選挙せんきょに出でる男性だんせいと女性じょせいの数かずをできるだけ同おなじにする法律ほうりつができてから初はじめての国くにの選挙せんきょでした。１０４人にんの女性じょせいが選挙せんきょに出でて、いちばん多おおかった３年ねん前まえの選挙せんきょのときと同おなじ２８人にんが選えらばれました。

来年らいねんのオリンピックとパラリンピックで使つかう電気でんき自動車じどうしゃ

发表于 2019-07-22 更新于 2019-11-30 分类于 JapanEasyNews



トヨタ自動車じどうしゃは、来年らいねん夏なつに東京とうきょうで開ひらくオリンピックとパラリンピックで使つかう電気でんき自動車じどうしゃを発表はっぴょうしました。
この自動車じどうしゃは、客きゃくを乗のせて会場かいじょうの中なかを電気でんきで走はしります。選手せんしゅが泊とまる場所ばしょでも走はしります。
運転うんてん席せきはいちばん前まえで、真まん中なかと後うしろの席せきに客きゃくが座すわって、全部ぜんぶで６人にん乗のることができます。真まん中なかの席せきを片かたづけてスロープを出だすと、車くるまいすに座すわったまま乗のったり降おりたりできます。暑あつい季節きせつのため、席せきは温度おんどが上あがりにくい材料ざいりょうを使つかっていて、背中せなかの部分ぶぶんは風かぜが通とおって涼すずしく感かんじるようにしています。
スピードは時速じそく１９ｋｍまで出でて、１回かいの充電じゅうでんで１００ｋｍぐらい走はしることができます。
オリンピックとパラリンピックでは、２００台だいぐらいがいろいろな会場かいじょうで走はしる予定よていです。

ＷＨＯ「エボラ出血熱しゅっけつねつが広ひろがっていて危険きけんだ」

发表于 2019-07-19 更新于 2019-11-30 分类于 JapanEasyNews



「エボラ出血熱しゅっけつねつ」はウイルスでうつって、亡なくなる可能性かのうせいが高たかい病気びょうきです。
アフリカのコンゴでは、この病気びょうきになる人ひとが去年きょねん８月がつから増ふえ続つづけて、今いままでに１６７６人にんが亡なくなりました。今月こんげつには、隣となりの国くにのルワンダに近ちかい町まちの人ひとや、コンゴから隣となりの国くにのウガンダへ行いった人ひとも、この病気びょうきになったことがわかりました。
ＷＨＯは１７日にち、専門家せんもんかを急いそいで集あつめて会議かいぎを開ひらきました。そして、「コンゴの周まわりの国くににもエボラ出血熱しゅっけつねつが広ひろがるかもしれません。とても危険きけんになっています」と発表はっぴょうしました。
ＷＨＯはコンゴとその周まわりの国くにに、国くにに入はいる人ひとや出でる人ひとの中なかにエボラ出血熱しゅっけつねつかもしれない人ひとがいないかどうか、しっかり調しらべるように言いっています。そして、国くにのみんなに正ただしい情報じょうほうを伝つたえて、病気びょうきが広ひろがらないようにしてほしいと言いっています。ＷＨＯは、コンゴやその周まわりの国くにの人ひとを助たすけるために、世界せかいの国くにの協力きょうりょくが必要ひつようだと言いっています。

发表于 2019-07-19 更新于 2019-11-30 分类于 JapanEasyNews



北きた太平洋たいへいようのさんまについて、日本にっぽんや中国ちゅうごく、台湾たいわん、韓国かんこく、ロシアなど８つの国くにと地域ちいきが話はなし合あう会議かいぎがありました。会議かいぎでは、８つの国くにと地域ちいきが来年らいねんとってもいいさんまの量りょうを５５万まん６２５０ｔに決きめました。どのくらいさんまをとってもいいかルールを決きめたのは初はじめてです。
北きた太平洋たいへいようのさんまは、２０年ねんの間あいだに６０％以上いじょう少すくなくなっています。さんまが少すくなくならないように、日本にっぽんはとってもいい量りょうのルールを決きめたいと考かんがえていました。しかし今いままでは、中国ちゅうごくなどの反対はんたいで決きめることができませんでした。
去年きょねん、８つの国くにと地域ちいきがとったさんまは全部ぜんぶで４３万まん７０００ｔでした。来年らいねんとってもいいさんまの量りょうを、去年きょねんとった量りょうより多おおくして、反対はんたいしていた国くにに賛成さんせいしてもらうことができました。
さんまがたくさんとれる宮城県みやぎけん気仙沼市けせんぬましにある会社かいしゃは、さんまが少すくなくなっている問題もんだいについて、大おおきく前まえに進すすむことができたと話はなしていました。

エスカレーターでは歩あるく人ひとのために横よこを空あけて乗のる習慣しゅうかんがあります。この乗のり方かただと、エスカレーターで歩あるく人ひとが転ころんで、事故じこになることがあります。お年寄としよりや体からだに障害しょうがいがある人ひとのためにも、やめてほしいと言いう人ひとが増ふえています。
このため鉄道てつどうの５２の会社かいしゃや空港くうこうなどは、今月こんげつ２２日にちから来月らいげつの終おわりまで、エスカレーターの安全あんぜんな乗のり方かたを伝つたえるキャンペーンを行おこないます。キャンペーンのポスターには、「歩あるかないで立たち止どまる」や「手てすりにつかまる」などと書かいてあります。外国人がいこくじんにもわかりやすいように英語えいごなど３つの国くにの言葉ことばでも書かいてあります。
ＪＲ東日本ひがしにほんは「エスカレーターでの事故じこは減へっていません。ほかの会社かいしゃとも協力きょうりょくして、安全あんぜんな乗のり方かたを伝つたえていきたいです」と話はなしています。

１８日にち午前ごぜん１０時じ半はんごろ、京都市きょうとし伏見区ふしみくにあるアニメを作つくる会社かいしゃの「京都きょうとアニメーション」の建物たてもので、大おおきな火事かじがありました。１階かいの玄関げんかんに入はいってきた男おとこが突然とつぜん、ガソリンと考かんがえられるものをまいて火ひをつけたため、爆発ばくはつのような火事かじになりました。
火ひが全部ぜんぶ消きえるまでに５時間じかんぐらいかかりました。警察けいさつによると、建物たてものの中なかにいた７４人にんのうち３３人にんが亡なくなって、３５人にんがけがをしました。

男おとこは前まえに埼玉県さいたまけんに住すんでいた４１歳さいの男おとこのようで、「京都きょうとアニメーション」で働はたらいたことはありません。男おとこと会社かいしゃの関係かんけいはわかっていません。男おとこが「死しね」などと大おおきな声こえで言いいながら火ひをつけたと話はなす人ひともいます。警察けいさつは、男おとこのけががよくなってから詳くわしく調しらべることにしています。

原発げんぱつの事故じこから８年ねん　福島県ふくしまけん飯舘村いいたてむらで乳牛にゅうぎゅうをまた育そだて始はじめる

发表于 2019-07-18 更新于 2019-11-30 分类于 JapanEasyNews



福島県ふくしまけん飯舘村いいたてむらは牛うしを育そだてる仕事しごとが盛さかんでした。しかし、８年ねん前まえの原発げんぱつ（原子力げんしりょく発電所はつでんしょ）の事故じこで、村むらに住すむことができなくなって、牛うしを育そだてる仕事しごともできなくなりました。
２年ねん前まえに村むらに戻もどってもいいことになってから、２つの農家のうかの人ひとたちはまた村むらで牛うしを育そだてるための準備じゅんびをしてきました。そして１６日にち、２２頭とうの子こどもの牛うしを牧場ぼくじょうに運はこんで、乳牛にゅうぎゅうを育そだてる仕事しごとを始はじめました。乳牛にゅうぎゅうをまた育そだてるまでに８年ねん４か月げつかかりました。
この牧場ぼくじょうでは、生うまれてから８か月げつの牛うしを２歳さいぐらいまで育そだてます。乳牛にゅうぎゅうは２００頭とうぐらいまで増ふやす予定よていです。村むらは牛うしを育そだてる仕事しごとをもう一度いちど盛さかんにしたいと考かんがえています。
農家のうかの人ひとは「また牛うしを育そだてることができてうれしいです。将来しょうらいは、この仕事しごとを若わかい人ひとや女性じょせいでも簡単かんたんにできるようにして、飯舘村いいたてむらを元気げんきにしたいです」と話はなしていました。

ジャニーズ事務所じむしょの「ＳＭＡＰ」にいた稲垣いながき吾郎ごろうさんと草くさなぎ剛つよしさんと香取かとり慎吾しんごさんの３人にんは事務所じむしょをやめました。３人にんは今いま、ＮＨＫ以外いがいのテレビに出演しゅつえんしていません。
