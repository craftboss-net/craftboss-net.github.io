ブロックの塀へい　安全あんぜんのためにお金かねを出だす市しなどは４０％だけ

发表于 2019-06-18 更新于 2019-11-30 分类于 JapanEasyNews



１年ねん前まえの２０１８年ねん６月がつ１８日にち、大阪府おおさかふで大おおきな地震じしんがありました。この地震じしんで５万まん７０００ぐらいの家いえや建物たてものが壊こわれました。高槻たかつき市しでは、小学校しょうがっこうのブロックの塀へいが倒たおれて、学校がっこうに行いく途中とちゅうだった女おんなの子こが亡なくなりました。
このあと市しや町まちが、ブロックの塀へいが安全あんぜんかどうか調しらべたり、塀へいをなくしたりする人ひとにお金かねを出だすようになりました。国土交通省こくどこうつうしょうによると、塀へいの安全あんぜんのためにお金かねを出だしている市しや町まちなどは、今年ことし４月がつに７７８ありました。これは全部ぜんぶの市しや町まちなどの４０％ぐらいだけです。
ブロックの塀へいの問題もんだいに詳くわしい専門家せんもんかは「安全あんぜんのためには、市しや町まちなどがお金かねを出だす必要ひつようがあると思おもいます」と話はなしています。

「自分じぶんで運転うんてんする車くるま」と答こたえた人ひとは５６％ぐらいいました。「歩あるいて行いく」と答こたえた人ひとも５６％ぐらいでした。「自転車じてんしゃ」と答こたえた人ひとは２２％ぐらいでした。８０歳さい以上いじょうで自分じぶんで車くるまを運転うんてんしている人ひとは２６％ぐらいいました。

发表于 2019-06-17 更新于 2019-11-30 分类于 JapanEasyNews



中国ちゅうごくで、宮崎駿みやざきはやお監督かんとくのアニメ映画えいが「千せんと千尋ちひろの神かみ隠かくし」の初はじめての上映じょうえいが１５日にちから始はじまりました。
映画館えいがかんでは、大勢おおぜいの子こどもや若わかい人ひとたちが映画えいがを楽たのしんでいました。これから中国ちゅうごくの９０００ぐらいの映画館えいがかんで上映じょうえいする予定よていです。
映画えいがを見みた人ひとは「アニメの絵えが本当ほんとうにきれいでした。人ひとがどのように生いきていったらいいかを教おしえてくれて、とても感動かんどうしました」と話はなしていました。

中国ちゅうごくでは最近さいきん、たくさんの日本にっぽんの映画えいがを上映じょうえいするようになりました。日本にっぽんと中国ちゅうごくの関係かんけいがよくなっていて、日本にっぽんの映画えいがを上映じょうえいする許可きょかを中国ちゅうごくの政府せいふからもらいやすくなっているからだと考かんがえられています。

子こどもを育そだてるために仕事しごとを休やすむ日本にっぽんの男性だんせいは５％だけ

发表于 2019-06-17 更新于 2019-11-30 分类于 JapanEasyNews



ユニセフは、４１の国くにの「育児休業いくじきゅうぎょう」について調しらべました。育児休業いくじきゅうぎょうは、子こどもを育そだてるために仕事しごとを休やすむことです。
育児休業いくじきゅうぎょうの間あいだも給料きゅうりょうをもらうことができる制度せいどがあっても、子こどものお父とうさんは休やすまない人ひとが多おおいとユニセフは言いっています。
特とくに日本にっぽんはほかの国くにより休やすみやすい制度せいどがあるのに、２０１７年ねんに制度せいどを使つかって休やすんだ人ひとは５％だけでした。同おなじような制度せいどがあって、１７％が休やすんだ韓国かんこくに比くらべるととても少すくないと言いっています。
ユニセフは、日本にっぽんでは働はたらく人ひとが足たりないことや、会社かいしゃなどで男性だんせいが育児休業いくじきゅうぎょうをすると言いいにくいことが原因げんいんだと考かんがえています。
ユニセフは、お父とうさんが育児休業いくじきゅうぎょうをすると、家族かぞくの関係かんけいがよくなるだけではなくて、お母かあさんを手伝てつだうことができると言いっています。男性だんせいと女性じょせいが平等びょうどうな社会しゃかいをつくることにも役やくに立たつので、お父とうさんがもっと休やすみやすい制度せいどにしてほしいと世界せかいの国くにに言いっています。

法律ほうりつで犬いぬや猫ねこにチップをつけなければならなくなる

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



法律ほうりつが変かわって、ペットの犬いぬや猫ねこに、誰だれが飼かっているかわかる小ちいさなチップをつけなければならなくなりました。犬いぬや猫ねこが捨すてられたとき、誰だれが捨すてたかすぐにわかるようにするためです。地震じしんなどの災害さいがいで、犬いぬや猫ねこが飼かっている人ひとと別々べつべつになったときも役やくに立たちます。
犬いぬや猫ねこの繁殖はんしょくをするブリーダーやペットを売うる店みせなどは、犬いぬや猫ねこの体からだにチップを入いれなければなりません。チップを入いれたら、飼かっている人ひとの住所じゅうしょなどを国くにに知しらせて登録とうろくしなければなりません。いま飼かっている犬いぬや猫ねこには、できるだけチップをつけるようにします。
法律ほうりつでは、犬いぬや猫ねこは生うまれてから５６日にちまで売うったり展示てんじしたりしてはいけないことにしました。特別とくべつに決きめた犬いぬは４９日にちまでにします。
動物どうぶつを虐待ぎゃくたいしたときの罰ばつも今いままでより厳きびしくしました。

日本にほん弱視じゃくし斜視しゃし学会がっかいは、目めが内側うちがわに寄よって戻もどらなくなる「急性内斜視きゅうせいないしゃし」とスマートフォンなどの関係かんけいを調しらべるために、眼科がんかの医者いしゃにアンケートをしました。３６９人にんが答こたえました。４２％の医者いしゃは、去年きょねん、「急性内斜視きゅうせいないしゃし」になった若わかい人ひとが病院びょういんに来きたと答こたえました。
この中なかの７７％は、「スマートフォンなどと関係かんけいがあると思おもった」と答こたえていました。学会がっかいは、スマートフォンなどを使つかいすぎることと関係かんけいがあるかもしれないと考かんがえて、これからも調査ちょうさを続つづけると言いっています。
調査ちょうさでは、スマートフォンなどを使つかう時間じかんや、目めからどのくらい離はなして使つかっているか調しらべます。使つかう時間じかんを短みじかくしたら、よくなったかどうかも調しらべます。学会がっかいは、どうしたら「急性内斜視きゅうせいないしゃし」にならないかや、治なおし方かたを２０２２年ねんぐらいまでに発表はっぴょうしたいと言いっています。

日本にっぽんのタンカーが中東ちゅうとうの海うみで攻撃こうげきされた

发表于 2019-06-14 更新于 2019-11-30 分类于 JapanEasyNews



１３日にち、中東ちゅうとうの海うみで２隻せきのタンカーが攻撃こうげきされました。１隻せきは日本にっぽんの会社かいしゃの船ふねでした。日本にっぽんの時間じかんで１３日にちの昼ひるごろ、ホルムズ海峡かいきょうの近ちかくの海うみで攻撃こうげきされたという連絡れんらくが船ふねから会社かいしゃに入はいりました。
会社かいしゃによると、船ふねは２回かい攻撃こうげきされて、弾たまが２発はつ当あたりました。このあと、船ふねに乗のっていた人ひとはみんなボートで逃にげて、別べつの船ふねに助たすけられました。
この船ふねは、燃もえやすいメタノールを２万まん５０００ｔ運はこんでいましたが、火事かじになったという情報じょうほうはありません。
会社かいしゃの社長しゃちょうは「船ふねに乗のっていた人ひとたちと電話でんわで話はなしをしました。みんな無事ぶじだと聞きいて安心あんしんしています」と話はなしています。
