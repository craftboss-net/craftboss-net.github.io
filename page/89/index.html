ヤギの子こどもがジャンプしてシマウマの背中せなかに乗のる

发表于 2019-01-29 更新于 2019-11-30 分类于 JapanEasyNews


静岡県しずおかけん東伊豆町ひがしいずちょうにある動物園どうぶつえんで、去年きょねん５月がつにヤギの子こどもが生うまれました。マーコールという種類しゅるいのヤギで、名前なまえは「きなこ」です。
きなこは去年きょねんの終おわりごろから、同おなじ場所ばしょで育そだてられているシマウマの背中せなかにジャンプして乗のるようになりました。２週間しゅうかんぐらい前まえからは毎日まいにち乗のっています。
マーコールは中東ちゅうとうの山やまに住すむヤギで、高たかくジャンプします。きなこは今いま、体からだの重おもさが１７ｋｇぐらいですが、高たかさ１ｍぐらいのシマウマの狭せまい背中せなかに簡単かんたんにジャンプして、上手じょうずに立たっています。
動物園どうぶつえんに来きた人ひとは「かわいいですね」とか「珍めずらしいですね」などと話はなしながら、写真しゃしんを撮とっていました。

エルサレムで２７日にち、第だい２次世界大戦じせかいたいせんのときにたくさんのユダヤ人じんを助たすけた杉原すぎはら千畝ちうねさんを記念きねんする式しきがありました。
日本にっぽんの外交官がいこうかんだった杉原すぎはらさんは、リトアニアで仕事しごとをしていました。このとき、ナチスからリトアニアに逃にげてきたユダヤ人じんが、日本にっぽんを通とおるビザを出だしてほしいと杉原すぎはらさんに頼たのみました。杉原すぎはらさんは、日本にっぽんの政府せいふの命令めいれいに反対はんたいして多おおくのユダヤ人じんにビザを出だしました。杉原すぎはらさんは、約やく６０００人にんの命いのちを助たすけたと言いわれています。
式しきではユダヤ人じんの団体だんたいの人ひとが「杉原すぎはらさんが勇気ゆうきを出だして、私わたしたちを助たすけたことを忘わすれません」と言いいました。
式しきに出席しゅっせきした男性だんせいは「私わたしの父ちちは２０歳さいのときに杉原すぎはらさんに助たすけてもらいました。父ちちは亡なくなりましたが、私わたしの家族かぞくは６０人にんになりました。とても感謝かんしゃしています」と話はなしていました。

普通ふつうの望遠鏡ぼうえんきょうで太陽系たいようけいの外側そとがわにある小ちいさい星ほしを見みつける

发表于 2019-01-29 更新于 2019-11-30 分类于 JapanEasyNews


太陽系たいようけいでいちばん遠とおくを回まわっている海王星かいおうせいの外側そとがわには、直径ちょっけいが２０ｋｍ以下いかの小ちいさい星ほしがたくさんあると考かんがえられています。このような小ちいさい星ほしがぶつかったり１つになったりして、地球ちきゅうなどの惑星わくせいができたと考かんがえられています。しかし海王星かいおうせいは地球ちきゅうから４３億おくｋｍも離はなれているため、その外側そとがわにある小ちいさい星ほしは暗くらくて見みつけることができませんでした。
日本にっぽんの研究けんきゅうグループが、海王星かいおうせいの外側そとがわにある直径ちょっけい２．６ｋｍの小ちいさい星ほしを世界せかいで初はじめて見みつけました。グループは、光ひかりを出だす大おおきい星ほしの前まえを小ちいさい星ほしが通とおるときの様子ようすを調しらべて、小ちいさい星ほしを見みつけました。
日本にっぽんのグループは普通ふつうの望遠鏡ぼうえんきょう２つとカメラを使つかっているため、研究けんきゅうにかかったお金かねは３５０万まん円えんぐらいでした。ほかの国くにのグループは３００倍ばいぐらいのお金かねをかけて、同おなじような星ほしを探さがす研究けんきゅうをしています。
日本にっぽんのグループは、海王星かいおうせいの外側そとがわには小ちいさい星ほしが２０億おく以上いじょうあると言いっています。

３世代せだいで住すむ家族かぞくを助たすける制度せいどがある県けんと府ふ　５年ねん前まえの７倍ばい

发表于 2019-01-28 更新于 2019-11-30 分类于 JapanEasyNews


２０１５年ねんに国くにが調しらべると、親おやと子こと孫まごの３世代せだいが一緒いっしょに住すむ家族かぞくの割合わりあいは５．７％で、２０年ねんの間あいだに半分はんぶんに減へりました。
ＮＨＫは４７の都道府県とどうふけんに、３世代せだいが一緒いっしょに住すむ家族かぞくを助たすける 制度せいどがあるかどうか聞ききました。その結果けっか、１４の県けんと府ふが、家いえを建たてるときに必要ひつようなお金かねの一部いちぶを出だしたり、税金ぜいきんを安やすくしたりする制度せいどがあると答こたえました。
鳥取県とっとりけんは２００８年ねんから、次つぎの福井県ふくいけんは２０１３年ねんからこのような制度せいどを始はじめています。このような制度せいどがある県けんと府ふは、５年ねん前まえの７倍ばいに増ふえました。
日本にっぽんは子こどもが少すくなくなって、人口じんこうが減へっています。３世代せだいが一緒いっしょに住すむと、祖父そふや祖母そぼが手伝てつだうことができるため、子こどもを育そだてるのが楽らくになると言いわれています。県けんなどは３世代せだいで住すむ家族かぞくを増ふやして、子こどもが減へらないようにしようと考かんがえているようです。

アイドル グループの「嵐あらし」は、２７日にち、グループでする仕事しごとは来年らいねんの１２月がつ３１日にちまでで、そのあとは休やすむと発表はっぴょうしました。
リーダーの大野おおの智さとしさんは「２０１７年ねん、自分じぶんが『嵐あらし』というグループでする仕事しごとを終おわりにしたいとほかの４人にんに話はなしました。そのあと何なん度ども話はなし合あって、２０２０年ねんで休やすむことになりました」と説明せつめいしました。

中国ちゅうごくの政府せいふは今月こんげつ、新あたらしい国際線こくさいせんを８０認みとめました。このうち４６は、中国ちゅうごくと日本にっぽんの間あいだを飛とぶ飛行機ひこうきです。
日本にっぽんと中国ちゅうごくの間あいだは、今いま１週間しゅうかんに１０００以上いじょうの飛行機ひこうきが飛とんでいます。認みとめられた飛行機ひこうきが全部ぜんぶ飛とび始はじめると１週間しゅうかんに２７７増ふえて、割合わりあいでは２５％以上いじょう多おおくなります。

上野動物園うえのどうぶつえん　モノレールが古ふるくなって１１月がつから運転うんてんを休やすむ

发表于 2019-01-25 更新于 2019-11-30 分类于 JapanEasyNews


東京都とうきょうとの上野動物園うえのどうぶつえんでは、１９５７年ねんに日本にっぽんで初はじめてモノレールの運転うんてんを始はじめました。動物園どうぶつえんの中なかの３３０ｍぐらいを走はしっていて、乗のると、動物園どうぶつえんを高たかい所ところから見みて楽たのしむことができます。とても人気にんきがあって、１年ねんに１００万まん人にんぐらいが乗のっています。
しかし、モノレールの車両しゃりょうが古ふるくなったため、東京都とうきょうとは今年ことし１１月がつから運転うんてんを休やすむことにしました。電気でんき自動車じどうしゃなどを代かわりに使つかう予定よていです。
東京都とうきょうとによると、新あたらしい車両しゃりょうを作つくるのに３年ねんぐらいかかります。お金かねも、１７年ねん使つかっている今いまの車両しゃりょうの４倍ばい以上いじょうの１８億おく円えんかかります。このため東京都とうきょうとは、動物園どうぶつえんに来きた人ひとや東京都とうきょうとに住すんでいる人ひとの意見いけんも聞きいて、モノレールの車両しゃりょうを新あたらしくするかどうか考かんがえたいと言いっています。
