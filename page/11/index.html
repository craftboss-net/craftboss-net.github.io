地震じしんで壊こわれた熊本城くまもとじょう　直なおした建物たてものを近ちかくで見みることができる

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



熊本市くまもとしにある熊本城くまもとじょうは、４００年ねん以上いじょうの歴史れきしがある城しろです。３年ねん前まえの大おおきな地震じしんで壊こわれた建物たてものを直なおしているため、今いまも入はいることができない場所ばしょがあります。
城しろでいちばん高たかい「大天守だいてんしゅ」という建物たてものの外側そとがわがほとんど直なおったため、１０月がつ５日いつかから近ちかくで見みることができるようになります。敵てきが登のぼりにくいようにつくった「武者むしゃ返がえし」という石垣いしがきも、すぐ近ちかくで見みることができるようになります。
熊本城くまもとじょうの事務所じむしょは「みなさんのおかげです。少すこしずつですが、前まえのような形かたちに戻もどっているのでぜひ見みに来きてほしいです」と話はなしていました。
熊本城くまもとじょうの大天守だいてんしゅは、１０月がつ５日いつかから１４日じゅうよっかまでは毎日まいにち、そのあとは、来年らいねん３月がつの終おわりまで日曜日にちようびと祝日しゅくじつに見みることができます。

蔵王山ざおうざんで紅葉こうよう　木きの葉はが黄色きいろや赤あかに変かわり始はじめる

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんではこれからいろいろな所ところで紅葉こうようが始はじまって、きれいな秋あきの景色けしきを楽たのしむことができます。
山形県やまがたけんと宮城県みやぎけんの間あいだにある蔵王山ざおうざんでも紅葉こうようが始はじまりました。山やまの頂上ちょうじょうの近ちかくにあるロープウエーの駅えきの周まわりで、楓かえでなどの木きの葉はが黄色きいろや赤あかに変かわり始はじめています。蔵王山ざおうざんに来きた人ひとは駅えきの近ちかくを歩あるきながら、紅葉こうようが始はじまった木きを見みたり写真しゃしんを撮とったりして、秋あきの山やまを楽たのしんでいました。
山形市やまがたしから来きた女性じょせいは「紅葉こうようを見みながら山やまに登のぼりたいと思おもって来きました。初はじめて来きましたが、緑みどりと赤あかと黄色きいろの３つの色いろがきれいです」と話はなしていました。

消費税しょうひぜいが１０％に上あがる

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



１０月がつ１日ついたち、消費税しょうひぜいが８％から１０％に上あがりました。食たべ物ものと酒さけ以外いがいの飲のみ物ものを買かって帰かえる場合ばあいは８％のままですが、店みせで食たべたり飲のんだりするときは１０％になります。
２４時間じかん開あいているコンビニなどでは午前ごぜん０時じから、売うり場ばの値段ねだんを変かえていました。消費税しょうひぜいが８％の品物しなものには「軽けい」と書かいて、わかるようにする店みせもありました。店みせの人ひとや買かい物ものをする人ひとは、どんなときに安やすい消費税しょうひぜいになるのかなどがわかりにくいと言いっています。
前まえに消費税しょうひぜいが上あがったのは２０１４年ねん４月がつでした。政府せいふは、消費税しょうひぜいを上あげて増ふえた収入しゅうにゅうを、小学校しょうがっこうに入はいる前まえの小ちいさい子こどものためなど、福祉ふくしに使つかうと言いっています。

「核兵器かくへいきを禁止きんしする条約じょうやくに入はいってほしい」署名しょめいを国連こくれんに渡わたす

发表于 2019-10-01 更新于 2019-11-30 分类于 JapanEasyNews



国連こくれんで２年ねん前まえ、核兵器かくへいきを作つくったり持もったりすることを禁止きんしする条約じょうやくができました。今年ことしの９月がつまでに３２の国くにがこの条約じょうやくに入はいることを決きめました。しかし、アメリカなど核兵器かくへいきを持もっている国くにや日本にっぽんは入はいっていません。
広島ひろしまや長崎ながさきで原爆げんばくの被害ひがいを受うけた人ひとたちでつくる団体だんたいの日本被団協にほんひだんきょうなどは、「全部ぜんぶの国くにがこの条約じょうやくに入はいってほしい」と言いって、町まちやインターネットで署名しょめいを集あつめました。世界中せかいじゅうから１０５１万まん人にんの署名しょめいが集あつまりました。
日本被団協にほんひだんきょうは１０月がつにニューヨークの国連こくれんに行いって、兵器へいきを減へらすことなどについて話はなし合あう会議かいぎの議長ぎちょうに署名しょめいを渡わたします。
日本被団協にほんひだんきょうは「たくさん署名しょめいが集あつまって、うれしいです。この条約じょうやくが大おおきな力ちからを持もつことができるように、これからも頑張がんばります」と話はなしています。

发表于 2019-09-30 更新于 2019-11-30 分类于 JapanEasyNews



高知県こうちけん四万十市しまんとしにある「学遊館がくゆうかん」という博物館はくぶつかんで、体からだの半分はんぶんが雄おすで半分はんぶんが雌めすの珍めずらしいクワガタをみんなに見みせています。このクワガタは、マレーシアなどにいる体からだの長ながさが４ｃｍぐらいの「モセリオウゴンオニクワガタ」です。大阪おおさかの男性だんせいが卵たまごから育そだてました。
クワガタは普通ふつう、雄おすはあごが長ながくて少すこし光ひかっていて、雌めすはあごが短みじかくて少すこし黒くろく見みえます。博物館はくぶつかんで見みせているクワガタは、左側ひだりがわが雄おすで右側みぎがわが雌めすです。博物館はくぶつかんによると、雄おすと雌めすの体からだを持もつ「モセリオウゴンオニクワガタ」が生うまれる割合わりあいは４万まん匹びきのうち１匹ぴきです。

博物館はくぶつかんを管理かんりしている人ひとは「この珍めずらしいクワガタを見みて、自然しぜんの世界せかいの不思議ふしぎを感かんじてほしいです」と話はなしていました。

发表于 2019-09-30 更新于 2019-11-30 分类于 JapanEasyNews



山口県やまぐちけん下関市しものせきしの南風泊はえどまり市場しじょうには、とらふぐというふぐが日本にっぽんでいちばんたくさん集あつまります。ふぐがおいしい季節きせつが来きて、最初さいしょの競せりが３０日にちにありました。
午前ごぜん３時じ２０分ぷんにベルが鳴なると、「袋ふくろ競ぜり」という特別とくべつなやり方かたでふぐの値段ねだんをどんどん決きめていました。袋ふくろ競ぜりでは、ふぐを売うる人ひとと買かう人ひとがほかの人ひとに見みえないように袋ふくろの中なかで指ゆびを使つかって値段ねだんを決きめます。
海うみでとったふぐと養殖ようしょくで人ひとが育そだてたふぐが、全部ぜんぶで３ｔ以上いじょう競せりに出でて、１０分ぷんぐらいで全部ぜんぶの値段ねだんが決きまりました。
いちばん大おおきなふぐは重おもさが４ｋｇぐらいあって、１ｋｇの値段ねだんが２万まん５０００円えんになりました。去年きょねんより６０００円えん高たかくなりました。

地球ちきゅうの気温きおんが上あがって海うみでとれる魚さかなが減へるかもしれない

发表于 2019-09-27 更新于 2019-11-30 分类于 JapanEasyNews



専門家せんもんかなどが集あつまる国連こくれんのＩＰＣＣは、地球ちきゅうの気温きおんが上あがると海うみが将来しょうらいどうなるかという予測よそくを発表はっぴょうしました。
地球ちきゅうの気温きおんが上あがると、海面かいめんの温度おんども上あがります。ＩＰＣＣは「これが原因げんいんで、世界せかいの海うみにいる生物せいぶつが２１００年ねんまでに２０％減へるかもしれません。海うみでとれる魚さかなも２４％減へるかもしれません」と言いっています。
ほかにも、「地球ちきゅうの気温きおんが上あがってグリーンランドや南極なんきょくの氷こおりがとけ続つづけると、海面かいめんの高たかさが２１００年ねんまでに１ｍ以上いじょう上あがるかもしれません。海面かいめんの高たかさが上あがると、台風たいふうの高潮たかしおなどで１００年ねんに一度いちどぐらいの大おおきな災害さいがいが毎年まいとし起おこるようになる危険きけんがあります」と言いっています。
このため、ＩＰＣＣは「二酸化炭素にさんかたんそなどのガスを少すくなくしたり、世界せかいの海うみの研究けんきゅうデータを集あつめたりするために、世界せかいの国くにが協力きょうりょくしなければなりません」と言いっています。
