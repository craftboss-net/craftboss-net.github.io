東京都とうきょうと中野区なかのくの新あたらしい小学校しょうがっこう　名前なまえは「令和れいわ小学校しょうがっこう」

发表于 2019-05-07 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうと中野区なかのくの上高田かみたかだ小学校しょうがっこうと新井あらい小学校しょうがっこうでは、子こどもが少すくなくなっています。このため来年らいねん４月がつに、２つの小学校しょうがっこうを１つにすることになっています。子こどもの親おややまちの人ひとたちは、会議かいぎをして新あたらしい小学校しょうがっこうの名前なまえを考かんがえていました。
会議かいぎでは、新あたらしい元号げんごうが「令和れいわ」なので、名前なまえを「令和れいわ小学校しょうがっこう」にすることを決きめました。子こどもたちが、まちの人ひとたちに大切たいせつにしてもらって、みんなで一緒いっしょに元気げんきに大おおきくなってほしいという気持きもちでこの名前なまえにしました。
４月がつ２５日にちまでに「令和れいわ」を名前なまえに使つかっている学校がっこうはないようだと中野区なかのくは話はなしています。中野区なかのくの議会ぎかいなどが賛成さんせいすると、夏なつごろに新あたらしい小学校しょうがっこうの名前なまえが正式せいしきに「令和れいわ小学校しょうがっこう」に決きまる予定よていです。

奈良県ならけん大和郡山市やまとこおりやまし　金魚きんぎょすくいの大会たいかいを「世界せかい大会たいかい」にする

发表于 2019-04-26 更新于 2019-11-30 分类于 JapanEasyNews



奈良県ならけんの大和郡山市やまとこおりやましでは、金魚きんぎょをたくさん育そだてています。市しは毎年まいとし夏なつに「金魚きんぎょすくい」の大会たいかいを開ひらいています。金魚きんぎょすくいは、紙かみを貼はったスプーンのような道具どうぐで金魚きんぎょをすくいます。
市しは、金魚きんぎょすくいを世界せかいの人ひとに知しってもらうために、今年ことしから外国人がいこくじんにも参加さんかしてもらって、「世界せかい大会たいかい」を開ひらくことにしました。市しはウェブサイトに、英語えいごで申もうし込こみをするページや、ルールを紹介しょうかいするビデオを用意よういしました。
大和郡山市やまとこおりやましの上田うえだ清きよし市長しちょうは「金魚きんぎょすくいは日本にっぽんの文化ぶんかで、スポーツです。外国人がいこくじんと交流こうりゅうするためにも、たくさんの人ひとに金魚きんぎょすくいを楽たのしんでもらいたいです」と話はなしています。
初はじめての世界せかい大会たいかいは８月がつ１８日にちです。

发表于 2019-04-26 更新于 2019-11-30 分类于 JapanEasyNews



広島市ひろしましの原爆資料館げんばくしりょうかんは１９５５年ねんにできました。建物たてものが古ふるくなって工事こうじをしていた本館ほんかんが新あたらしくなって、２５日にちからまた展示てんじを見みることができるようになりました。
１９９１年ねんから変かえていなかった展示てんじのしかたも新あたらしくなりました。本館ほんかんは４つのコーナーに分わかれています。原爆げんばくが落おちた１９４５年ねん８月がつ６日むいかのまちの様子ようすを紹介しょうかいするコーナーでは、原爆げんばくの強つよい風かぜで曲まがった建物たてものの鉄材てつざいや、亡なくなった子こどもたちの服ふくを展示てんじしています。
別べつのコーナーでは、原爆げんばくで亡なくなった人ひとたちのことを知しってもらうため、使つかっていたベルトなどをその人ひとの写真しゃしんと一緒いっしょに展示てんじしています。
原爆資料館げんばくしりょうかんは、亡なくなった人ひとたちが使つかっていた物ものをたくさん展示てんじして、原爆げんばくでどんなことが起おこったか、今いままで以上いじょうにわかりやすく伝つたえたいと考かんがえています。

日本にっぽんのＪＡＸＡは、地球ちきゅうから３億おくｋｍの所ところにあるリュウグウという小ちいさい星ほしを調しらべるため、４年ねんぐらい前まえに「はやぶさ２」を打うち上あげました。
「はやぶさ２」は４月がつ５日いつか、リュウグウに金属きんぞくの塊かたまりを強つよくぶつけました。そのあと２５日にち、「はやぶさ２」は、強つよくぶつけた所ところの画像がぞうを高たかさ１７００ｍから撮とりました。

ＪＡＸＡは「世界せかいで初はじめて人工じんこうのクレーターができて、とてもうれしいです。これからクレーターを詳くわしく調しらべたいと思おもいます」と話はなしています。

４月がつ２７日にちから５月がつ６日むいかまで休やすみが１０日とおか間かん続つづきます。この間あいだに病気びょうきやけがをした人ひとが困こまらないように、特別とくべつに病院びょういんを開あける日ひを決きめました。
病院びょういんが開あいている日ひは、都道府県とどうふけんが調しらべてウェブサイトに出だしています。厚生労働省こうせいろうどうしょうのウェブサイトからもさがすことができます。

旅行りょこうの計画けいかくをしやすくするインターネットの新あたらしいサービス

发表于 2019-04-25 更新于 2019-11-30 分类于 JapanEasyNews



最近さいきん、日本にっぽんの国内こくないで１泊ぱく以上いじょうの旅行りょこうに行いく人ひとがあまり増ふえていません。もっとたくさんの人ひとが旅行りょこうをするように、いろいろな会社かいしゃがインターネットで新あたらしいサービスを始はじめています。
東京都とうきょうとにある会社かいしゃが始はじめたサービスは、まずウェブサイトのページに出発しゅっぱつの場所ばしょ、旅行りょこうに行いきたい季節きせつ、一緒いっしょに行いく人ひとを入いれます。そして、ページの中なかから好すきな写真しゃしんを３枚まい選えらぶと、その人ひとの気分きぶんに合あった国内こくない旅行りょこうの計画けいかくを自動じどうで考かんがえてくれます。ホテルや飛行機ひこうきなどの予約よやくもできます。
別べつの会社かいしゃのサービスは、スマートフォンなどに入いれたアプリで旅行りょこうの予約よやくをします。１０万まん円えんまでの旅行りょこうのお金かねは２か月げつあとに払はらうこともできます。
