台風たいふうや雨あめで水みずが入はいった家いえを掃除そうじするボランティアが足たりない

发表于 2019-10-30 更新于 2019-11-30 分类于 JapanEasyNews



先月せんげつの台風たいふう１５号ごうと今月こんげつの台風たいふう１９号ごう、そして２５日にちにたくさん降ふった雨あめで、たくさんの場所ばしょで大おおきな被害ひがいがありました。ボランティアの団体だんたいによると、掃除そうじなどを手伝てつだうボランティアの人ひとが足たりません。
特とくに宮城県みやぎけん、福島県ふくしまけん、栃木県とちぎけんで足たりません。福島県ふくしまけんいわき市しでは、家いえの掃除そうじなど６００以上いじょうの仕事しごとを頼たのまれていますが、ボランティアの人ひとが手伝てつだうことができたのは２００ぐらいです。宮城県みやぎけん丸森町まるもりまちでは家いえに入はいった泥どろを掃除そうじすることができないため、たくさんの人ひとが学校がっこうの建物たてものなどで生活せいかつしています。だんだん寒さむくなっていますが、服ふくや毛布もうふが足たりません。ボランティアの団体だんたいは、もっとたくさんの人ひとにボランティアに来きてほしいと言いっています。

「エスカレーターは歩あるかないで止とまって乗のってください」

发表于 2019-10-29 更新于 2019-11-30 分类于 JapanEasyNews



エスカレーターは、安全あんぜんのために手てすりをつかんで、歩あるかないで止とまって乗のるのが正ただしい乗のり方かたです。しかし、歩あるいて上のぼったり下おりたりする人ひとのために、自分じぶんの隣となりをあける習慣しゅうかんがあります。この乗のり方かたでは、お年寄としよりや体からだに障害しょうがいがある人ひとは、利用りようしにくいことがあります。
東京とうきょうの練馬駅ねりまえきの近ちかくで２７日にち、子こどもと親おやにエスカレーターの正ただしい乗のり方かたを教おしえる会かいがありました。子こどもたちは、止とまって乗のったほうが安全あんぜんで、２人ふたりずつ並ならんで乗のったほうが大勢おおぜいの人ひとが早はやく利用りようできることを習ならいました。
東京とうきょうでは、エスカレーターの左側ひだりがわに止とまって乗のる人ひとがいて、右側みぎがわを歩あるく人ひとが多おおいです。しかし体からだの障害しょうがいなどで左ひだりの手てにつえを持もっている人ひとは、右みぎの手てで手てすりをつかむため、右側みぎがわに立たつ必要ひつようがあります。子こどもたちは、つえを持もって乗のってみてわかりました。
この会かいを開ひらいた人ひとは「正ただしい乗のり方かたでみんなが安全あんぜんに利用りようできるようにしたいです」と話はなしていました。

发表于 2019-10-29 更新于 2019-11-30 分类于 JapanEasyNews



今年ことし７月がつ、京都きょうとアニメーションの建物たてもので、男おとこがガソリンに火ひをつけて、大勢おおぜいの人ひとが亡なくなりました。男おとこはガソリンスタンドでガソリンを買かって、缶かんに入いれて運はこびました。
このため、総務省消防庁そうむしょうしょうぼうちょうは、ガソリンスタンドがガソリンを缶かんに入いれて売うるときの規則きそくを新あたらしく作つくることにしました。
新あたらしい規則きそくでは、ガソリンスタンドは、ガソリンを缶かんで買かう人ひとに運転免許証うんてんめんきょしょうなどを見みせてもらって、名前なまえや住所じゅうしょを確認かくにんしなければなりません。何なんのためにガソリンを買かうかも聞きいて、いつどのくらいガソリンを売うったか書かいて残のこしておかなければなりません。よくガソリンを買かう人ひとの場合ばあいは、確認かくにんはいつもではなくてもいいようにすることも考かんがえています。
総務省消防庁そうむしょうしょうぼうちょうは、１１月がつ２７日にちまでこの規則きそくについての意見いけんをメールなどで集あつめます。そして、来年らいねん２月がつ１日ついたちから新あたらしい規則きそくを始はじめる予定よていです。

ＪＲ東日本ひがしにほんは、上越新幹線じょうえつしんかんせんで新あたらしく使つかう予定よていだった６０の車両しゃりょうを北陸新幹線ほくりくしんかんせんで使つかうことを考かんがえています。しかし、十分じゅうぶんではないので、故障こしょうしたり検査けんさしたりする車両しゃりょうが出でたら、運転うんてんする数かずが少すくなくなるかもしれないと言いっています。

東京とうきょうの渋谷駅しぶやえきのそばに「渋谷しぶやスクランブルスクエア」という新あたらしいビルができました。４７階かい建だてで、高たかさは２３０ｍ、地下ちかは７階かいまであります。渋谷駅しぶやえきの周まわりでいちばん高たかいビルです。

このビルは、１１月がつ１日ついたちにオープンします。
