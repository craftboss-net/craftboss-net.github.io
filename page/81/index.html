「学校がっこうに通かようことができない子こどもは２億おく６０００万まん人にん」

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



イギリスの首相しゅしょうだったブラウンさんは、今いまは国連こくれんの教育きょういく特使とくしで、世界せかいの教育きょういくをよくするための仕事しごとをしています。
ブラウンさんは１９日にち、「難民なんみんになった子こどもの中なかで、小学校しょうがっこうの教育きょういくを受うけることができる子こどもは２０％だけです。ギリシャでシリアからの難民なんみんなどがたくさん住すんでいる所ところでは、１０歳さいぐらいの２人ふたりの男おとこの子こが自殺じさつしようとしました。難民なんみんたちが住すんでいる所ところでも教育きょういくを行おこなうことが必要ひつようです」と言いいました。
そして「学校がっこうに通かようことができない子こどもは今いま、世界せかいに２億おく６０００万まん人にんもいます。世界せかいの国くには、２０３０年ねんまでにみんなが十分じゅうぶんな教育きょういくを受うけることができるようにすると決きめました。しかし、このままでは２０３０年ねんには２億おく人にんの子こどもが学校がっこうに通かようことができません」と言いいました。
ブラウンさんは、子こどもの教育きょういくのために、世界せかいの国くにはもっと協力きょうりょくしてほしいと言いっています。

发表于 2019-02-21 更新于 2019-11-30 分类于 JapanEasyNews



アメリカのトランプ大統領だいとうりょうは来週らいしゅう、ベトナムで北朝鮮きたちょうせんのキム?ジョンウン朝鮮ちょうせん労働党ろうどうとう委員長いいんちょうと会あう予定よていです。安倍あべ総理大臣そうりだいじんは２０日はつかにトランプ大統領だいとうりょうと電話でんわで話はなして、日本にっぽんとアメリカは北朝鮮きたちょうせんから核兵器かくへいきをなくすために協力きょうりょくすると約束やくそくしました。

「学校がっこうに携帯電話けいたいでんわを持もって来くることができるように考かんがえる」

发表于 2019-02-20 更新于 2019-11-30 分类于 JapanEasyNews



文部科学省もんぶかがくしょうは２００９年ねん、小学校しょうがっこうや中学校ちゅうがっこうに通かよう子こどもたちは学校がっこうに携帯電話けいたいでんわを持もって来きてはいけないと決きめました。
去年きょねん６月がつ、子こどもたちが学校がっこうに行いく時間じかんに大阪おおさかで大おおきな地震じしんがありました。大阪府おおさかふは、災害さいがいのときに子こどもたちに連絡れんらくすることができるように、携帯電話けいたいでんわを持もって学校がっこうに来きてもいいことにしようと考かんがえています。そして１８日にち、携帯電話けいたいでんわの使つかい方かたの案あんを発表はっぴょうしました。
柴山しばやま文部科学大臣もんぶかがくだいじんは「大阪府おおさかふの案あんは携帯電話けいたいでんわの問題もんだいや心配しんぱいなことについて考かんがえて、携帯電話けいたいでんわを学校がっこうに持もって来くる場合ばあいのルールを決きめてあります」と言いいました。そして、国くにも学校がっこうに携帯電話けいたいでんわを持もって来くることができるように考かんがえたいと言いいました。

ホンダ「イギリスで車くるまを作つくることを２０２１年ねんにやめる」

发表于 2019-02-20 更新于 2019-11-30 分类于 JapanEasyNews



ホンダは、イギリスにある工場こうじょうで車くるまを作つくることを２０２１年ねんにやめると発表はっぴょうしました。
ヨーロッパにあるホンダの工場こうじょうはこの１つだけです。工場こうじょうでは３５００人にんが働はたらいて、１年ねんに１６万まん台だいの車くるまを作つくっています。ここで作つくった車くるまの３５％ぐらいはヨーロッパで売うっています。
しかし、ホンダは最近さいきんヨーロッパでは車くるまがあまり売うれなくなったと言いっています。これから電気でんき自動車じどうしゃを作つくるためには、新あたらしい機械きかいの準備じゅんびなどにたくさんのお金かねがかかります。ホンダは、このような理由りゆうで工場こうじょうをやめると言いいました。
イギリスは３月がつ２９日にちにＥＵから出でることになっています。そのあと、イギリスで作つくった車くるまを売うることは今いまより難むずかしくなるかもしれません。これもホンダがイギリスで車くるまを作つくることをやめる理由りゆうだと言いわれています。
