英語えいごなど１０の言葉ことばで同時どうじに話はなしてもわかるＡＩの技術ぎじゅつ

发表于 2019-02-15 更新于 2019-11-30 分类于 JapanEasyNews



三菱電機みつびしでんきがコンピューターのＡＩの新あたらしい技術ぎじゅつを発表はっぴょうしました。この技術ぎじゅつでは、いろいろな国くにの言葉ことばで２人ふたり以上いじょうが同時どうじにＡＩに話はなしても、ＡＩは何語なにごで何なんと言いわれたかわかります。
この技術ぎじゅつの発表はっぴょう会かいでは、１人ひとりが日本語にほんごで「中国料理ちゅうごくりょうりを食たべたい」、もう１人ひとりが英語えいごで「お土産みやげを買かいたい」と同時どうじに言いいました。そのあと、画面がめんの中なかの地図ちずに中国料理ちゅうごくりょうりとお土産みやげの２つの店みせの場所ばしょが出でました。中国料理ちゅうごくりょうりの店みせの説明せつめいは日本語にほんご、お土産みやげの店みせの説明せつめいは英語えいごで書かいてありました。
三菱電機みつびしでんきによると、２つ以上いじょうの言葉ことばが同時どうじにわかるＡＩの技術ぎじゅつは世界せかいで初はじめてです。１０の言葉ことばを同時どうじに理解りかいすることもできます。
この技術ぎじゅつをつくった人ひとは「観光かんこうの案内あんない所じょで使つかうことができたら、質問しつもんしに来きた人ひとは待まつ時間じかんが少すくなくなります。ほかにも、便利べんりな使つかい方かたをいろいろ考かんがえたいです」と話はなしています。

池江いけえ選手せんしゅの発表はっぴょうで骨髄こつずいバンクの登録とうろくの方法ほうほうを聞きく人ひとが増ふえる

发表于 2019-02-15 更新于 2019-11-30 分类于 JapanEasyNews



水泳すいえいの池江いけえ璃花子りかこ選手せんしゅが１２日にち、白血病はっけつびょうという病気びょうきになったと発表はっぴょうしました。白血病はっけつびょうの治療ちりょうの１つに、白血球はっけっきゅうの型かたが合あう人ひとから骨髄こつずいをもらうという方法ほうほうがあります。
「日本骨髄にほんこつずいバンク」は、１８歳さいから５４歳さいまでの骨髄こつずいをあげてもいいという人ひとを登録とうろくして、白血病はっけつびょうの人ひとに紹介しょうかいしています。池江いけえ選手せんしゅのニュースのあと、事務所じむしょには登録とうろくの方法ほうほうなどを聞きく電話でんわがたくさん来きました。資料しりょうを送おくってほしいというメールも、１日にちに２７０件けんぐらいありました。
白血球はっけっきゅうの型かたが合あう割合わりあいは兄弟きょうだいでも２５％で、家族かぞく以外いがいだともっと少すくなくなります。日本骨髄にほんこつずいバンクは「白血病はっけつびょうはいつ誰だれがなるかわかりません。骨髄こつずいをもらうのを待まっている人ひとは日本にっぽんに１５００人にんぐらいいます。ぜひ大勢おおぜいの人ひとに登録とうろくしてほしいです」と話はなしています。

愛媛県えひめけんの県庁けんちょうに１億おく円えんぐらいのお金かねが入はいった箱はこが届とどく

发表于 2019-02-15 更新于 2019-11-30 分类于 JapanEasyNews



愛媛県えひめけんの県庁けんちょうに１月がつ２９日にち、箱はこが１つ届とどきました。箱はこの中なかには、１万まん円えん札さつがいっぱい入はいっていました。
一緒いっしょに入はいっていた手紙てがみには「何なにかの役やくに立たつように使つかってください。送おくった人ひとの名前なまえや住所じゅうしょは本当ほんとうではありません」と書かいてありました。
愛媛県えひめけんによると、箱はこに入はいっていたお金かねは１億おく円えんぐらいあるようですが、とても古ふるくなっていて使つかうことはできません。このため、愛媛県えひめけんは日本銀行にっぽんぎんこうに調しらべてもらって、新あたらしいお金かねに換かえてもらう予定よていです。
愛媛県えひめけんの中村なかむら知事ちじは「長ながい間あいだためていたお金かねだと思おもいます。心こころからお礼れいを言いいたいです。去年きょねんの大雨おおあめで被害ひがいがあった人ひとやまちが元気げんきになるようにお金かねを使つかいたいと考かんがえています」と話はなしています。

「同性どうせいの結婚けっこんを認みとめてほしい」国くにを訴うったえる

发表于 2019-02-15 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんでは、男性だんせいと男性だんせいや女性じょせいと女性じょせいの同性どうせいが結婚けっこんすることができません。法律ほうりつに書かいてある「夫婦ふうふ」について、国くには「夫おっとは男性だんせいで、妻つまは女性じょせい」だと考かんがえて、同性どうせいが結婚けっこんすることを認みとめていません。
１４日じゅうよっか、東京とうきょうなどに住すんでいる同性どうせいのパートナー１３組くみが、同性どうせいが結婚けっこんできないのは、人ひとはみんな平等びょうどうだと決きめている憲法けんぽうに違反いはんしていると言いって、国くにを訴うったえました。
最近さいきんは、東京とうきょうの渋谷区しぶやくや大阪市おおさかしなど同性どうせいのパートナーを結婚けっこんと同おなじような関係かんけいだと認みとめる役所やくしょや会社かいしゃが増ふえています。外国がいこくでは、ヨーロッパなどで同性どうせいの結婚けっこんを認みとめる国くにが増ふえています。
国くにを訴うったえた人ひとは「私わたしたちは平等びょうどうにしてほしいだけです」と言いっています。

「はしか」は高たかい熱ねつやせきが出でたり、体からだに赤あかい発疹はっしんが出でたりする病気びょうきです。はしかのウイルスはとてもうつりやすくて、はしかがひどくなって亡なくなる人ひともいます。
大阪府おおさかふは１４日じゅうよっか、新大阪駅しんおおさかえきと東京駅とうきょうえきの間あいだで新幹線しんかんせんに乗のった女性じょせいがはしかになっていたことがわかったと言いいました。この女性じょせいは６日むいかに熱ねつが出でたあと、８日ようか午前ごぜん１１時じ５６分ぷんに新大阪駅しんおおさかえきを出発しゅっぱつする新幹線しんかんせんに乗のって、午後ごご２時じ半はんに東京駅とうきょうえきに着つきました。１０日とおかにも午後ごご６時じに東京駅とうきょうえきを出発しゅっぱつする新幹線しんかんせんに乗のって、午後ごご８時じ３３分ぷんに新大阪駅しんおおさかえきに着つきました。
大阪府おおさかふは、この女性じょせいと同おなじ新幹線しんかんせんに乗のっていた人ひとなどに、熱ねつが出でたら、電話でんわで連絡れんらくをしてから病院びょういんへ行いってほしいと言いっています。
国くにの研究所けんきゅうじょによると、今年ことし２月がつ３日みっかまでにはしかになった人ひとは１４８人にんです。いちばん多おおいのは三重みえ県けんで４９人にん、次つぎは大阪府おおさかふで４３人にん、愛知県あいちけんで１４人にん、東京都とうきょうとで１０人にん、和歌山県わかやまけんと岐阜県ぎふけんが５人にんです。

今年ことしも風疹ふうしんになる人ひとが増ふえている「ワクチンの注射ちゅうしゃをして」

发表于 2019-02-14 更新于 2019-11-30 分类于 JapanEasyNews



風疹ふうしんは熱ねつが出でたり体からだに赤あかい発疹はっしんが出でたりする病気びょうきで、ウイルスでうつります。
国くにの研究所けんきゅうじょによると、去年きょねん、風疹ふうしんになった人ひとは２９１７人にんで、今いままでの１０年ねんで２番目ばんめに多おおくなりました。今年ことしも２月がつ３日みっかまでに３６７人にんが風疹ふうしんになっています。この中なかの６０％以上いじょうは東京都とうきょうととその周まわりの県けんの人ひとで、大阪府おおさかふや福岡県ふくおかけんなどでも増ふえています。
研究所けんきゅうじょによると、風疹ふうしんになる人ひとは毎年まいとし春はるから多おおくなるため、これからもっと増ふえる心配しんぱいがあります。
風疹ふうしんになる人ひとが特とくに多おおい３９歳さいから５６歳さいの男性だんせいは、ワクチンの注射ちゅうしゃをしていない人ひとがたくさんいます。このため、国くにはこれから、この年齢ねんれいの男性だんせいに無料むりょうでワクチンの注射ちゅうしゃをすることができるようにします。
妊娠にんしんしている女性じょせいが風疹ふうしんになると、赤あかちゃんの目めや耳みみなどに障害しょうがいが出でることがあります。研究所けんきゅうじょは「女性じょせいは妊娠にんしんする前まえにワクチンの注射ちゅうしゃを２回かいしてください」と言いっています。
