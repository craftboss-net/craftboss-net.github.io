茨城県いばらきけん水戸市みとし　たくさんの人ひとが「偕楽園かいらくえん」の梅うめを楽たのしむ

发表于 2019-02-19 更新于 2019-11-30 分类于 JapanEasyNews



茨城県いばらきけん水戸市みとしで１６日にちから、梅うめまつりが始はじまりました。梅うめの花はなが有名ゆうめいな「偕楽園かいらくえん」には、たくさんの人ひとが来きていました。
偕楽園かいらくえんには、１００種類しゅるい、３０００本ぼんぐらいの梅うめの木きがあります。梅うめまつりが始はじまる前まえに調しらべたときは、１５％以上いじょうの木きで花はなが咲さいていました。
見みに来きた人ひとは、白しろやピンク色いろのかわいい花はなを見みながら、散歩さんぽしたり写真しゃしんを撮とったりしていました。そして「いいにおいで、きれいです」とか「春はるが来きたようでうれしくなります」と話はなしていました。
梅うめまつりは３月がつの終おわりまでです。梅うめの実みでつくった梅酒うめしゅを１４０種類しゅるいぐらい飲のむことができるイベントもあって、５０万まん人にんぐらいの人ひとが来くると考かんがえられています。

ｉＰＳ細胞さいぼうを使つかった「脊髄損傷せきずいそんしょう」の治療ちりょうが始はじまる

发表于 2019-02-19 更新于 2019-11-30 分类于 JapanEasyNews



ｉＰＳ細胞さいぼうは、神経しんけいなど体からだのいろいろな部分ぶぶんに変かわることができる細胞さいぼうです。慶応大学けいおうだいがくのグループは、「脊髄損傷せきずいそんしょう」の治療ちりょうにｉＰＳ細胞さいぼうを使つかう研究けんきゅうをしています。国くには１８日にち、このグループが作つくった計画けいかくを行おこなってもいいと言いいました。
事故じこなどで背中せなかの骨ほねの中なかにある神経しんけいが傷きずついて「脊髄損傷せきずいそんしょう」になると、手てや足あしなどを動うごかすことができなくなります。日本にっぽんでは毎年まいとし５０００人にんぐらいの人ひとがなっていて、治なおすことは難むずかしいと言いわれています。
グループの計画けいかくによると、まず人ひとのｉＰＳ細胞さいぼうから神経しんけいに変かわることができる細胞さいぼうを２００万まん個こぐらい作つくります。そして、神経しんけいが傷きずついた所ところにこの細胞さいぼうを入いれます。入いれた細胞さいぼうが神経しんけいに変かわって、体からだの動うごかなかった部分ぶぶんを動うごかすことができるようにしたいと考かんがえています。

群馬県ぐんまけんで生活せいかつする外国人がいこくじんのための「健康けんこう手帳てちょう」ができる

发表于 2019-02-18 更新于 2019-11-30 分类于 JapanEasyNews



今年ことし４月がつから法律ほうりつが変かわって、外国人がいこくじんが日本にっぽんで働はたらきやすくなります。このため、病気びょうきやけがをして病院びょういんに行いく外国人がいこくじんが増ふえると考かんがえられています。
群馬県ぐんまけんにある通訳つうやくの団体だんたいなどが、外国人がいこくじんのための「健康けんこう手帳てちょう」を作つくりました。英語えいごやベトナム語ご、ポルトガル語ごなど９つの言葉ことばで書かいた手帳てちょうがあります。
手帳てちょうには、病院びょういんを紹介しょうかいする場所ばしょや消防署しょうぼうしょの電話でんわ番号ばんごうなどが書かいてあります。そのほか、自分じぶんの病気びょうきの名前なまえや飲のんでいる薬くすりなど、医者いしゃに伝つたえたほうがいいことを書かく所ところもあります。インフルエンザのワクチンを受うけることなど、健康けんこうのために気きをつけることも書かいてあります。
団体だんたいなどはこの手帳てちょうを１万まん１０００冊さつ作つくって、群馬県ぐんまけんにある日本語にほんご学校がっこうや外国人がいこくじんが働はたらいている会社かいしゃなどで配くばる予定よていです。団体だんたいの人ひとは「言葉ことばがよくわからなくて病院びょういんに行いくのが遅おくれる人ひとがいます。みんなにこの手帳てちょうを使つかってほしいです」と話はなしていました。

携帯電話けいたいでんわを使つかいながら運転うんてん　亡なくなる事故じこの割合わりあいが２倍ばい高たかい

发表于 2019-02-18 更新于 2019-11-30 分类于 JapanEasyNews



警察庁けいさつちょうによると、携帯電話けいたいでんわなどを使つかいながら車くるまを運転うんてんする「ながら運転うんてん」が原因げんいんの交通事故こうつうじこが、去年きょねん２７９０件けんありました。事故じこは、携帯電話けいたいでんわで話はなしたり、メールやカーナビの画面がめんを見みたりしながら運転うんてんしているときに起おこっています。スマートフォンを使つかう人ひとが増ふえたため、１０年ねんの間あいだで２倍ばい以上いじょうに増ふえました。
「ながら運転うんてん」が原因げんいんで人ひとが亡なくなった事故じこは４２件けんありました。人ひとが亡なくなった事故じこの割合わりあいは、「ながら運転うんてん」以外いがいが原因げんいんのときより２倍ばいも高たかくなっています。
「ながら運転うんてん」をすると、危険きけんに気きがついて、スピードを下さげたり車くるまを止とめたりすることが遅おそくなります。そして車くるまのスピードが十分じゅうぶんに下さがらないまま事故じこになってしまうことが多おおいと警察庁けいさつちょうは考かんがえています。
警察庁けいさつちょうは、今いま開ひらいている国会こっかいに「ながら運転うんてん」の罰ばつを厳きびしくした法律ほうりつの案あんを出だす予定よていです。

若わかい人ひとの耳みみを守まもる　ＷＨＯがイヤホンで聞きく音おとの大おおきさを発表はっぴょう

发表于 2019-02-18 更新于 2019-11-30 分类于 JapanEasyNews



ＷＨＯは、世界せかいの若わかい人ひとの半分はんぶんが長ながい時間じかん大おおきな音おとを聞きいているため、耳みみが悪わるくなって音おとが聞きこえにくくなるかもしれないと心配しんぱいしています。音楽おんがくを楽たのしむスマートフォンなどの小ちいさなプレーヤーで、イヤホンを使つかって大おおきな音おとで音楽おんがくを聞きく人ひとが増ふえたことなどが原因げんいんだと考かんがえています。

「同性どうせいの結婚けっこんを認みとめてほしい」国くにを訴うったえる

发表于 2019-02-15 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんでは、男性だんせいと男性だんせいや女性じょせいと女性じょせいの同性どうせいが結婚けっこんすることができません。法律ほうりつに書かいてある「夫婦ふうふ」について、国くには「夫おっとは男性だんせいで、妻つまは女性じょせい」だと考かんがえて、同性どうせいが結婚けっこんすることを認みとめていません。
１４日じゅうよっか、東京とうきょうなどに住すんでいる同性どうせいのパートナー１３組くみが、同性どうせいが結婚けっこんできないのは、人ひとはみんな平等びょうどうだと決きめている憲法けんぽうに違反いはんしていると言いって、国くにを訴うったえました。
最近さいきんは、東京とうきょうの渋谷区しぶやくや大阪市おおさかしなど同性どうせいのパートナーを結婚けっこんと同おなじような関係かんけいだと認みとめる役所やくしょや会社かいしゃが増ふえています。外国がいこくでは、ヨーロッパなどで同性どうせいの結婚けっこんを認みとめる国くにが増ふえています。
国くにを訴うったえた人ひとは「私わたしたちは平等びょうどうにしてほしいだけです」と言いっています。
