アスベストのような物質ぶっしつが見みつかる　軍艦島ぐんかんじまに入はいることを禁止きんし

发表于 2019-08-01 更新于 2019-11-30 分类于 JapanEasyNews



長崎市ながさきしの軍艦島ぐんかんじまは昔むかし、石炭せきたんを取とっていた島しまで、軍艦ぐんかんのように見みえます。今いまは誰だれも住すんでいませんが、世界遺産せかいいさんになっていて、多おおくの人ひとが観光かんこうに来きます。
長崎市ながさきしが７月がつに調しらべると、軍艦島ぐんかんじまにある３つの広場ひろばからアスベストのような物質ぶっしつが見みつかりました。アスベストは建物たてものなどを作つくるときに使つかっていましたが、体からだに悪わるいことがわかって、今いまは使つかうことができません。長崎市ながさきしは安全あんぜんのため、観光かんこうで軍艦島ぐんかんじまに入はいることを７月がつ３１日にちから禁止きんししました。今いままでの調査ちょうさではアスベストが見みつかったことがありませんでした。
長崎市ながさきしは「見みつかった物質ぶっしつを詳くわしく調しらべて、また安全あんぜんに島しまに入はいることができるようにしたいです」と話はなしています。

ボールを投なげすぎてひじの手術しゅじゅつをする子こどもが多おおい

发表于 2019-08-01 更新于 2019-11-30 分类于 JapanEasyNews



「トミー?ジョン手術しゅじゅつ」は、ひじの具合ぐあいが悪わるくなった野球やきゅうの選手せんしゅが受うける手術しゅじゅつです。群馬県ぐんまけんの病院びょういんで働はたらく医者いしゃの古島ふるしま弘三こうぞうさんは今いままでに６００回かい以上いじょう、この手術しゅじゅつをしました。
古島ふるしまさんによると、手術しゅじゅつをした人ひとの４０％ぐらいが高校生こうこうせい以下いかの子こどもでした。この中なかには小学生しょうがくせいもいました。手術しゅじゅつを受うけた多おおくの子こどもは、ボールを投なげることができなくなっていて、手術しゅじゅつ以外いがいに治なおす方法ほうほうがありませんでした。
古島ふるしまさんは「骨ほねがまだ大おおきくなっていない子こどもが休やすまないでボールをたくさん投なげると、ひじが悪わるくなる危険きけんが高たかくなります。野球やきゅうを教おしえる人ひとは、ひじの具合ぐあいが悪わるくならないように気きをつけなければなりません」と話はなしています。

雑誌ざっしの図書館としょかん「大宅壮一文庫おおやそういちぶんこ」を続つづけるために寄付きふを集あつめる

发表于 2019-08-01 更新于 2019-11-30 分类于 JapanEasyNews



東京都とうきょうと世田谷区せたがやくにある「大宅壮一文庫おおやそういちぶんこ」は、１９７１年ねんにできた日本にっぽんで最初さいしょの雑誌ざっしの図書館としょかんです。有名ゆうめいなジャーナリストの大宅おおや壮一そういちさんが集あつめたりした１万まん２６００種類しゅるいの雑誌ざっしが８０万まん冊さつあります。しかし、みんながインターネットを使つかうようになってから、大宅壮一文庫おおやそういちぶんこを利用りようする人ひとが減へって、経営けいえいが難むずかしくなりました。
このため、大宅壮一文庫おおやそういちぶんこは８月がつから寄付きふを集あつめることにしました。寄付きふは１人ひとり１万まん円えんから、会社かいしゃや団体だんたいは１０万まん円えんからです。
大宅壮一文庫おおやそういちぶんこは「ここにある雑誌ざっしには、インターネットにはない情報じょうほうがたくさん書かいてあります。この場所ばしょをなくしてはいけないと思おもいます」と話はなしています。

发表于 2019-07-31 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんでは都道府県とどうふけんで「最低さいてい賃金ちんぎん」が決きまっていて、会社かいしゃなどは働はたらいた人ひとに最低さいてい賃金ちんぎん以上いじょうのお金かねを払はらわなければなりません。
最低さいてい賃金ちんぎんをいくらぐらいにしたらいいか話はなし合あう国くにの会議かいぎがありました。そして、１時間じかんの最低さいてい賃金ちんぎんは、都道府県とどうふけんの平均へいきんで、今いまより２７円えん上あがって９０１円えんになりました。
物価ぶっかなどが高たかい東京都とうきょうとや大阪府おおさかふなどは今いまより２８円えん、京都府きょうとふや兵庫県ひょうごけんなどは２７円えん、北海道ほっかいどうや宮城県みやぎけん、青森県あおもりけん、愛媛県えひめけんなどは２６円えん上あげます。
都道府県とどうふけんがこのとおりに最低さいてい賃金ちんぎんを上あげると、東京都とうきょうとは１０１３円えん、神奈川県かながわけんは１０１１円えんになって、初はじめて１０００円えんより高たかくなります。
最低さいてい賃金ちんぎんは去年きょねんまで３年ねん続つづけて大おおきく上あがっていて、今年ことしも大おおきく上あげることに決きまりました。
