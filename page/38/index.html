囲碁いごの仲邑なかむら菫すみれさん　１０歳さいでプロの試合しあいで初はじめて勝かつ

发表于 2019-07-09 更新于 2019-11-30 分类于 JapanEasyNews



１０歳さいの仲邑なかむら菫すみれさんはいちばん若わかい囲碁いごのプロです。８日ようか、仲邑なかむらさんは６７歳さいの田中たなか智恵子ちえこさんと試合しあいをしました。４月がつにプロになってから２回かい目めの試合しあいです。
仲邑なかむらさんは試合しあいのとき、いすが高たかくて、足あしをのせる 台だいを使つかいました。試合しあいは仲邑なかむらさんが勝かちました。
今いままで、プロの試合しあいで初はじめて勝かったときにいちばん若わかかった人ひとの年齢ねんれいは１１歳さい８か月げつでした。仲邑なかむらさんはそれより若わかい１０歳さい４か月げつで初はじめて勝かちました。試合しあいのあと仲邑なかむらさんは「勝かつことができてうれしかったです」と話はなしていました。
試合しあいをした田中たなかさんは「仲邑なかむらさんは強つよかったです。これからも頑張がんばってほしいです」と話はなしていました。

東京とうきょうオリンピックのとき会社かいしゃに行いかないで仕事しごとをする

发表于 2019-07-08 更新于 2019-11-30 分类于 JapanEasyNews



東京とうきょうオリンピックまで１年ねんです。オリンピックの間あいだ、道みちや電車でんしゃが混こむ心配しんぱいがあるため、政府せいふは会社かいしゃに、家いえなど会社かいしゃ以外いがいの場所ばしょで仕事しごとをすることを考かんがえるように言いっています。
ホンダは、東京とうきょうの港区みなとくにある会社かいしゃに行いかないで仕事しごとができるようにします。１２００人にんぐらいの社員しゃいんは、家いえで仕事しごとをしたり、埼玉県さいたまけんにあるホンダの事務所じむしょなどで仕事しごとをしたりします。
トヨタ自動車じどうしゃも、東京とうきょうで働はたらく１２００人にんぐらいは家いえで仕事しごとができるようにします。ホンダとトヨタ自動車じどうしゃは、家いえでうまく仕事しごとができるかどうか、今月こんげつテストする予定よていです。
パソコンをつくっているレノボ?ジャパンは、オリンピックの間あいだ、社員しゃいんが特別とくべつな休やすみを取とるようにします。そして、家いえなどで仕事しごとをして、２０００人にんが会社かいしゃに行いかなくてもいいようにします。
このように、会社かいしゃ以外いがいの場所ばしょで仕事しごとができるようにする会社かいしゃが、どのくらい増ふえるか注目ちゅうもくされています。

旅行会社りょこうがいしゃのＪＴＢは、ツアーに予約よやくしている人ひとの数かずなどから今年ことしの夏なつに外国がいこくへ旅行りょこうに行いく人ひとの数かずを計算けいさんしました。その結果けっか、今月こんげつ１５日にちから来月らいげつ３１日にちまでに外国がいこくへ旅行りょこうに行いく人ひとは２９９万まん人にんぐらいになりました。去年きょねんより３．５％多おおくなって、今いままででいちばん多おおくなりそうです。

旅行りょこうに使つかうお金かねは、平均へいきんで１人ひとり２２万まん７７００円えんで、去年きょねんより６．２％増ふえそうです。

发表于 2019-07-05 更新于 2019-11-30 分类于 JapanEasyNews



手足口病てあしくちびょうは子こどもに多おおい病気びょうきで、手てや足あし、口くちの中なかなどに小ちいさな発疹はっしんができます。脳のうに炎症えんしょうが出でることもあります。ウイルスでうつる病気びょうきで、毎年まいとし、夏なつに病気びょうきになる人ひとが多おおくなります。
東京都とうきょうとによると、６月がつ３０日にちまでの１週間しゅうかんに東京都とうきょうとでは、手足口病てあしくちびょうの子こどもは１３２５人にんになりました。１つの病院びょういんの平均へいきんで５．１人にんでした。東京都とうきょうとは４日よっか、平均へいきんが５人にんより多おおくなったため、手足口病てあしくちびょうが大おおきな流行りゅうこうになっているので気きをつけるように発表はっぴょうしました。前まえにこのような発表はっぴょうをしたのは、２年ねん前まえの７月がつでした。
手足口病てあしくちびょうは、せきやくしゃみでうつります。ほかにも、手てについたウイルスが口くちの中なかに入はいってうつるため、東京都とうきょうとは小ちいさな子こどもがいる家庭かていでは、よく手てを洗あらうように言いっています。
