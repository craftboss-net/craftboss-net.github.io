災害さいがいのとき少すくない水みずで流ながすことができるトイレ

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



ＬＩＸＩＬという会社かいしゃは、地震じしんなどの災害さいがいのとき、普通ふつうより少すくない水みずで流ながすことができるトイレを作つくりました。９日ここのか、この新あたらしいトイレの説明会せつめいかいが熊本市くまもとしでありました。
普通ふつうのトイレは流ながすとき１回かい５Ｌの水みずを使つかいます。新あたらしいトイレは、いつもは５Ｌの水みずで流ながしますが、１Ｌの水みずで流ながすように変かえることができます。水みずが流ながれていく所ところに開あいたり閉しまったりするふたがあって、においも出でにくくなっています。掃除そうじもしやすくて、きれいに使つかうことができるようにしました。

ノーベル賞しょうの本庶ほんじょさん「薬くすりの特許とっきょの契約けいやくを変かえてほしい」

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



京都大学きょうとだいがくの特別とくべつ教授きょうじゅの本庶ほんじょ佑たすくさんは、薬くすりの会社かいしゃの小野薬品工業おのやくひんこうぎょうと一緒いっしょに、がんを治療ちりょうする新あたらしい薬くすりをつくりました。この薬くすりの研究けんきゅうなどで、本庶ほんじょさんは去年きょねんノーベル賞しょうをもらいました。
本庶ほんじょさんと小野薬品工業おのやくひんこうぎょうは、この薬くすりをつくる技術ぎじゅつの特許とっきょを持もっています。本庶ほんじょさんは、ほかの会社かいしゃがこの技術ぎじゅつを使つかうときに払はらったお金かねを、小野薬品工業おのやくひんこうぎょうと分わける 契約けいやくをしています。本庶ほんじょさんは１０日とおか、「今いまの契約けいやくでは、私わたしがもらうお金かねがとても少すくないと思おもいます」と言いいました。
京都大学きょうとだいがくの弁護士べんごしは「今いまの契約けいやくで本庶ほんじょさんがもらうお金かねは２６億おく円えんぐらいですが、正ただしく契約けいやくをしたら１０００億おく円えんぐらいになります」と話はなしています。
本庶ほんじょさんはもらったお金かねを若わかい人ひとたちの研究けんきゅうのために使つかいたいと考かんがえていて、「会社かいしゃと大学だいがくが正ただしく協力きょうりょくして、若わかい研究けんきゅう者しゃを育そだてなければなりません」と話はなしています。
小野薬品工業おのやくひんこうぎょうは「今いまの契約けいやくとずいぶん違ちがうので、相談そうだんを続つづけています」と話はなしています。

世界せかいで初はじめてブラックホールの画像がぞうを撮とる

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



ブラックホールはとても強つよい力ちからで光ひかりや電波でんぱを吸すい込こむため、見みることができません。
日本にっぽんやアメリカ、ヨーロッパなどのグループは、光ひかりの速はやさで地球ちきゅうから５５００万まん年ねん離はなれた所ところにあるブラックホールを調しらべました。南みなみアメリカのチリなど世界中せかいじゅうにある６つの電波でんぱ望遠鏡ぼうえんきょうを使つかって、同おなじ時間じかんに、ブラックホールの周まわりにあるガスやごみなどが出だす電波でんぱのデータを集あつめました。そしてデータから画像がぞうをつくりました。
画像がぞうでは、ブラックホールの周まわりにあるガスなどが出だす電波でんぱが赤あかや白しろ、その中なかが黒くろい穴あなのように見みえます。
グループは「世界せかいで初はじめて、ブラックホールの画像がぞうを撮とることができました。これからブラックホールのことや宇宙うちゅうがどうやってできたかなどがもっとわかるようになると思おもいます」と話はなしています。

发表于 2019-04-10 更新于 2019-11-30 分类于 JapanEasyNews



８年ねん前まえの原子力げんしりょく発電所はつでんしょ（原発げんぱつ）の事故じこのあと、国くには放射線ほうしゃせんの量りょうが多おおい所ところには住すまないように言いいました。福島県ふくしまけん大熊町おおくままちも住すむことができなくなりました。
町まちでは放射線ほうしゃせんの量りょうを少すくなくする作業さぎょうを続つづけていました。国くには、大熊町おおくままちの一部いちぶは住すむことができるようになったと考かんがえて、１０日とおかから戻もどってもいいと言いいました。事故じこがあった原子力げんしりょく発電所はつでんしょがある町まちで初はじめてです。
町まちの役場やくばは５月がつから仕事しごとを始はじめます。６月がつには新あたらしい建物たてもので５０の家族かぞくが生活せいかつを始はじめます。コンビニもできます。
しかし、去年きょねん１月がつに町まちの人ひとに質問しつもんすると「町まちに戻もどりたい」と答こたえた人ひとは１２．５％だけでした。事故じこの前まえと同おなじような生活せいかつにするためには、産業さんぎょうや教育きょういくについて考かんがえていかなければなりません。
