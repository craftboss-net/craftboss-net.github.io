ノーベル賞しょうの本庶ほんじょさん「薬くすりの特許とっきょの契約けいやくを変かえてほしい」

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



京都大学きょうとだいがくの特別とくべつ教授きょうじゅの本庶ほんじょ佑たすくさんは、薬くすりの会社かいしゃの小野薬品工業おのやくひんこうぎょうと一緒いっしょに、がんを治療ちりょうする新あたらしい薬くすりをつくりました。この薬くすりの研究けんきゅうなどで、本庶ほんじょさんは去年きょねんノーベル賞しょうをもらいました。
本庶ほんじょさんと小野薬品工業おのやくひんこうぎょうは、この薬くすりをつくる技術ぎじゅつの特許とっきょを持もっています。本庶ほんじょさんは、ほかの会社かいしゃがこの技術ぎじゅつを使つかうときに払はらったお金かねを、小野薬品工業おのやくひんこうぎょうと分わける 契約けいやくをしています。本庶ほんじょさんは１０日とおか、「今いまの契約けいやくでは、私わたしがもらうお金かねがとても少すくないと思おもいます」と言いいました。
京都大学きょうとだいがくの弁護士べんごしは「今いまの契約けいやくで本庶ほんじょさんがもらうお金かねは２６億おく円えんぐらいですが、正ただしく契約けいやくをしたら１０００億おく円えんぐらいになります」と話はなしています。
本庶ほんじょさんはもらったお金かねを若わかい人ひとたちの研究けんきゅうのために使つかいたいと考かんがえていて、「会社かいしゃと大学だいがくが正ただしく協力きょうりょくして、若わかい研究けんきゅう者しゃを育そだてなければなりません」と話はなしています。
小野薬品工業おのやくひんこうぎょうは「今いまの契約けいやくとずいぶん違ちがうので、相談そうだんを続つづけています」と話はなしています。

オリンピックの仕事しごとをしている桜田さくらだ大臣だいじんがやめる

发表于 2019-04-11 更新于 2019-11-30 分类于 JapanEasyNews



オリンピックとパラリンピックの仕事しごとをしている桜田さくらだ大臣だいじんは１０日とおか、岩手県いわてけんの高橋たかはし議員ぎいんのパーティーに出席しゅっせきしました。岩手県いわてけんは２０１１年ねんに地震じしんと津波つなみで大おおきな被害ひがいを受うけました。このパーティーで桜田さくらだ大臣だいじんは、被害ひがいを受うけた町まちを元気げんきにすることより大事だいじなのが高橋たかはし議員ぎいんだと言いいました。
パーティーのあと桜田さくらだ大臣だいじんは「被害ひがいを受うけた人ひとたちの気持きもちを傷きずつけるようなことを言いって、とても申もうし訳わけないと思おもっています」と言いって、大臣だいじんをやめました。
安倍あべ総理大臣そうりだいじんは「これからも私わたしたちは被害ひがいを受うけた人ひとたちの気持きもちを大切たいせつにして、町まちを元気げんきにするために頑張がんばります」と言いって、謝あやまりました。
野党やとうは、大臣だいじんを選えらんだ安倍あべ総理大臣そうりだいじんにも責任せきにんがあると言いっています。今月こんげつ５日いつかにも、自分じぶんが話はなしたことが原因げんいんで副ふく大臣だいじんがやめたばかりでした。

警察庁けいさつちょう　ドローンを使つかったテロがないように新あたらしい機械きかいを使つかう

发表于 2019-04-10 更新于 2019-11-30 分类于 JapanEasyNews



警察庁けいさつちょうはドローンを使つかった事件じけんなどがないように、新あたらしい機械きかいを使つかうことにしました。
法律ほうりつに違反いはんして飛とんでいるドローンがいた場合ばあい、この機械きかいから出でる電波でんぱを当あてると、ドローンは飛とぶことができなくなります。アメリカではもう使つかっています。
外国がいこくではドローンを使つかった事件じけんがあって、ベネズエラでは大統領だいとうりょうが攻撃こうげきされました。
日本にっぽんでは今年ことし、２０の国くにの首相しゅしょうや大統領だいとうりょうが集あつまる「Ｇ２０」という会議かいぎがあります。来年らいねんはオリンピックがあります。このため、警察庁けいさつちょうは新あたらしい機械きかいを使つかって、ドローンを使つかったテロなどがないようにしたいと考かんがえています。

福島県ふくしまけん三春町みはるまちに１０００年ねん前まえからある桜さくらは高たかさが１３ｍで、国くにの天然記念物てんねんきねんぶつになっています。この桜さくらはベニシダレザクラという種類しゅるいで、花はなが咲さくと滝たきのように見みえるため、「滝桜たきざくら」と呼よばれています。

原発げんぱつの事故じこがあった福島県ふくしまけん大熊町おおくままちの一部いちぶに戻もどれるようになる

发表于 2019-04-10 更新于 2019-11-30 分类于 JapanEasyNews



８年ねん前まえの原子力げんしりょく発電所はつでんしょ（原発げんぱつ）の事故じこのあと、国くには放射線ほうしゃせんの量りょうが多おおい所ところには住すまないように言いいました。福島県ふくしまけん大熊町おおくままちも住すむことができなくなりました。
町まちでは放射線ほうしゃせんの量りょうを少すくなくする作業さぎょうを続つづけていました。国くには、大熊町おおくままちの一部いちぶは住すむことができるようになったと考かんがえて、１０日とおかから戻もどってもいいと言いいました。事故じこがあった原子力げんしりょく発電所はつでんしょがある町まちで初はじめてです。
町まちの役場やくばは５月がつから仕事しごとを始はじめます。６月がつには新あたらしい建物たてもので５０の家族かぞくが生活せいかつを始はじめます。コンビニもできます。
しかし、去年きょねん１月がつに町まちの人ひとに質問しつもんすると「町まちに戻もどりたい」と答こたえた人ひとは１２．５％だけでした。事故じこの前まえと同おなじような生活せいかつにするためには、産業さんぎょうや教育きょういくについて考かんがえていかなければなりません。

トルコの新あたらしい空港くうこう　１年ねんに２億おく人にんが利用りようできるようにする

发表于 2019-04-09 更新于 2019-11-30 分类于 JapanEasyNews



トルコでは去年きょねん、「イスタンブール空港くうこう」という新あたらしい空港くうこうがオープンしました。今いままで使つかっていた「アタチュルク国際こくさい空港くうこう」より大おおきい空港くうこうが必要ひつようになったためです。アタチュルク国際こくさい空港くうこうを使つかっていた飛行機ひこうきは、７日なのかから全部ぜんぶ新あたらしい空港くうこうを使つかうことになりました。
新あたらしい空港くうこうを利用りようした人ひとは「本当ほんとうに大おおきくてすごいです」と言いって喜よろこんでいました。空港くうこうは、広ひろさが野球やきゅう場じょうの東京とうきょうドームの３０倍ばいぐらいで、滑走路かっそうろが２本ほんあります。１年ねんに９０００万まん人にんが利用りようできます。
トルコでは、イスタンブール空港くうこうをだんだん広ひろくする計画けいかくです。計画けいかくでは、２０２８年ねんまでに滑走路かっそうろを６本ぽんにして、１年ねんに２億おく人にんが利用りようできるようにします。とても大おおきな空港くうこうになるため、トルコの人ひとは経済けいざいがよくなってほしいと考かんがえています。
