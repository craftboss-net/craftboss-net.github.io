山口県やまぐちけんと韓国かんこくの大学生だいがくせいが今年ことしも交流こうりゅうをする

发表于 2019-08-22 更新于 2019-11-30 分类于 JapanEasyNews



山口県やまぐちけんの大学生だいがくせいは毎年まいとし、韓国かんこくの大学生だいがくせいと交流こうりゅうをしています。今年ことしも韓国かんこくのキョンサン南道なんどうやプサン市しなどの大学生だいがくせい１７人にんが山口県やまぐちけんに来きました。
２１日にち、山口県やまぐちけんの大学生だいがくせい２０人にんと韓国かんこくの大学生だいがくせいたちは、一緒いっしょに長門市ながとしの海岸かいがんを掃除そうじしました。ペットボトルや缶かんを拾ひろったり、大おおきな網あみをみんなで引ひいて海うみの中なかのごみを取とったりしました。
最近さいきん、日本にっぽんと韓国かんこくの関係かんけいが悪わるくなっていて、いろいろな交流こうりゅうのイベントが中止ちゅうしになったりしています。韓国かんこくの大学生だいがくせいは「政治せいじの関係かんけいはよくないですが、みんなが交流こうりゅうをしたら関係かんけいもよくなると思おもいます」と話はなしていました。山口県やまぐちけんの大学生だいがくせいは「冬ふゆには私わたしたちが韓国かんこくへ行いきたいです」と話はなしていました。

宇宙うちゅうの研究けんきゅうをする「ハイパーカミオカンデ」をつくる計画けいかく

发表于 2019-08-22 更新于 2019-11-30 分类于 JapanEasyNews



岐阜県ぎふけん飛騨市ひだしに「カミオカンデ」と「スーパーカミオカンデ」という建物たてものがあります。宇宙うちゅうの研究けんきゅうのために、物質ぶっしつを作つくっている最もっとも小ちいさい粒子りゅうしの１つ「ニュートリノ」を調しらべています。この地下ちかにある建物たてもので研究けんきゅうを行おこなった２人ふたりが、ノーベル賞しょうをもらっています。
文部科学省もんぶかがくしょうは、ニュートリノなどをもっとよく調しらべるため、飛騨市ひだしに新あたらしく「ハイパーカミオカンデ」をつくる予定よていです。地下ちか６５０ｍの所ところに、直径ちょっけいが７０ｍぐらい、深ふかさが６０ｍの水槽すいそうをつくって、宇宙うちゅうから来くるニュートリノを調しらべます。ハイパーカミオカンデをつくるためには６７０億おく円えん以上いじょう必要ひつようです。
文部科学省もんぶかがくしょうは、２０２５年ねんから２０２９年ねんの間あいだにハイパーカミオカンデを使つかい始はじめる計画けいかくです。ニュートリノと反対はんたいの性質せいしつがある「反はんニュートリノ」などを調しらべることにしています。

「在留資格ざいりゅうしかくの取とり消けし」になった外国人がいこくじんが去年きょねんは８３２人にんいた

发表于 2019-08-22 更新于 2019-11-30 分类于 JapanEasyNews



外国人がいこくじんがビザに書かいてある目的もくてきなどと違ちがう生活せいかつをしていると「在留資格ざいりゅうしかくの取とり消けし」になって、日本にっぽんにいることができなくなります。
出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうによると、去年きょねん「在留資格ざいりゅうしかくの取とり消けし」になった人ひとは８３２人にんでした。前まえの年としの２倍ばい以上いじょうで、調しらべ始はじめてから今いままででいちばん多おおくなりました。ベトナム人じんがいちばん多おおくて４１６人にん、次つぎは中国人ちゅうごくじんで１５２人にんでした。
「在留資格ざいりゅうしかくの取とり消けし」になった人ひとの半分はんぶんぐらいが「留学りゅうがく」のビザで来きた人ひとでした。出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうは「勉強べんきょうするために日本にっぽんに来きたのに、日本語学校にほんごがっこうや大学だいがくをやめたあとも、働はたらきながら日本にっぽんに残のこっている外国人がいこくじんがいます。大学だいがくから留学生りゅうがくせいが行方不明ゆくえふめいになる問題もんだいもあったので、これからも厳きびしく調しらべます」と言いっています。

昭和天皇しょうわてんのうとの会話かいわを書かいた大事だいじな資料しりょうが見みつかった

发表于 2019-08-21 更新于 2019-11-30 分类于 JapanEasyNews



昭和天皇しょうわてんのうは、今いまの天皇てんのう陛下へいかの祖父そふで、上皇じょうこうさまの父ちちです。
戦争せんそうが終おわったあと最初さいしょの宮内庁くないちょうの長官ちょうかんだった田島たじま道治みちじさんが、昭和天皇しょうわてんのうとの会話かいわを書かいたノートなどの大事だいじな資料しりょうが見みつかりました。ノートには１９４９年ねんから１９５３年ねんまでの会話かいわが書かいてあります。
この中なかで、昭和天皇しょうわてんのうは戦争せんそうへの後悔こうかいについて何なん度ども話はなしていました。１９５２年ねん、日本にっぽんの独立どくりつが戻もどったことを祝いわう式しきがありました。この式しきで昭和天皇しょうわてんのうは、戦争せんそうへの深ふかい後悔こうかいと、二に度どと戦争せんそうをしないための反省はんせいの気持きもちを伝つたえたいと強つよく思おもっていました。しかし、吉田よしだ茂しげる総理大臣そうりだいじんが「戦争せんそうを始はじめた責任せきにんがあると言いわれるかもしれない」と言いって反対はんたいしました。このため、昭和天皇しょうわてんのうは自分じぶんが言いいたかった後悔こうかいの気持きもちを話はなしませんでした。
専門家せんもんかは「昭和天皇しょうわてんのうは、みんなの前まえで戦争せんそうへの後悔こうかいや反省はんせいについてはっきり話はなしたことがありません。とても深ふかい後悔こうかいの気持きもちを伝つたえたいと思おもっていたことに驚おどろきました」と話はなしています。

北海道ほっかいどうには韓国かんこくからたくさんの人ひとが観光かんこうに来きます。しかし、日本にっぽんと韓国かんこくの関係かんけいが悪わるくなったあと、北海道ほっかいどうと韓国かんこくの間あいだでは一部いちぶの飛行機ひこうきが飛とばないことになりました。北海道ほっかいどうは、韓国かんこくから来くる人ひとが少すくなくなることを心配しんぱいしています。
このため、北海道ほっかいどうなどの職員しょくいんたちは韓国かんこくの人ひとたちを空港くうこうで歓迎かんげいすることにしました。新千歳空港しんちとせくうこうでは１９日にち、職員しょくいんたちが「北海道ほっかいどうへようこそ」と韓国語かんこくごで書かいた幕まくを持もって、韓国かんこくからの飛行機ひこうきで着ついた人ひとを迎むかえました。そして、韓国語かんこくごで「こんにちは」と言いいながら、うちわや北海道ほっかいどうのメロンのゼリーなどを渡わたしました。
韓国人かんこくじんの若わかい男性だんせいは「日本にっぽんに来くるのをやめようとは思おもいませんでした。歓迎かんげいしてくれてうれしいです」と話はなしていました。北海道ほっかいどうの職員しょくいんは「韓国かんこくのみなさんに、北海道ほっかいどうに安心あんしんして行いくことができると思おもってほしいです」と話はなしていました。
