タピオカの輸入ゆにゅうが関西かんさいでとても増ふえている

发表于 2019-08-21 更新于 2019-11-30 分类于 JapanEasyNews



大阪おおさかの税関ぜいかんによると、今年ことしの１月がつから６月がつまでに関西かんさいで輸入ゆにゅうしたタピオカは６３３ｔで、去年きょねんの同おなじときの２０倍ばい以上いじょうになりました。全部ぜんぶ台湾たいわんから輸入ゆにゅうしています。
タピオカは暑あつい所ところでできる芋いもから作つくった食たべ物もので、最近さいきん若わかい人ひとの間あいだで紅茶こうちゃなどに入いれて飲のむことが流行りゅうこうしています。大阪おおさかではタピオカを入いれた飲のみ物ものを売うる店みせがたくさんできていて、今月こんげつの売うり上あげが去年きょねんの４月がつの１０倍ばいになったという店みせもあります。この店みせの店長てんちょうは「台湾たいわんの文化ぶんかのタピオカを日本にっぽんの人ひとが楽たのしんでくれてうれしいです」と言いっていました。

９月がつ２０日はつかに始はじまるラグビーのワールドカップの日本にほん大会たいかいまで１か月げつになりました。試合しあいをする会場かいじょうの近ちかくでは、大会たいかいをたくさんの人ひとに知しってもらうため、幕まくなどを飾かざってＰＲしています。
東京都とうきょうと調布市ちょうふしにある東京とうきょうスタジアムでは、大会たいかいの最初さいしょの試合しあいの日本にっぽんとロシアの試合しあいを行おこないます。この会場かいじょうでは、大会たいかいのマークやキャラクターの絵えがかいてある長ながさ２．７ｍの幕まくを飾かざりました。
近ちかくにある商店街しょうてんがいには、大会たいかいのマークがかいてある旗はたをたくさん飾かざりました。まちの人ひとたちは、外国がいこくからもたくさんの人ひとが試合しあいを見みに来きてくれると楽たのしみにしています。小学生しょうがくせいの男おとこの子こは「日本にっぽんに決勝けっしょうまで進すすんでほしいです」と話はなしていました。

「核兵器かくへいきをなくしてほしい」高校生こうこうせいが国連こくれんに署名しょめいを届とどける

发表于 2019-08-21 更新于 2019-11-30 分类于 JapanEasyNews



毎年まいとし８月がつ、日本にっぽんの高校生こうこうせいが国連こくれんへ行いって「核兵器かくへいきをなくしてほしい」と伝つたえる活動かつどうをしています。この活動かつどうは、インドとパキスタンが核兵器かくへいきの実験じっけんをした１９９８年ねんに長崎ながさきの人ひとたちが始はじめました。
今年ことしは１６の都道府県とどうふけんから集あつまった２３人にんの高校生こうこうせいがスイスにある国連こくれんのヨーロッパ本部ほんぶへ行いきました。原爆げんばくが落おとされた広島ひろしまと長崎ながさきの高校生こうこうせいも行いきました。高校生こうこうせいたちは、核兵器かくへいきをなくしてほしいと願ねがっている世界せかいの人ひとたち２１万まん５５４７人にんの署名しょめいを届とどけました。
広島ひろしまの高校こうこう２年生ねんせいの牟田むた悠一郎ゆういちろうさんは、祖父そふが被爆ひばくしました。牟田むたさんは「祖父そふたちから聞きいた経験けいけんをこれからも伝つたえたいと強つよく思おもいました」と話はなしていました。
高校生こうこうせいたちは１９９８年ねんからの活動かつどうで２００万まん人にん以上いじょうの署名しょめいを国連こくれんに届とどけました。

昭和天皇しょうわてんのうとの会話かいわを書かいた大事だいじな資料しりょうが見みつかった

发表于 2019-08-21 更新于 2019-11-30 分类于 JapanEasyNews



昭和天皇しょうわてんのうは、今いまの天皇てんのう陛下へいかの祖父そふで、上皇じょうこうさまの父ちちです。
戦争せんそうが終おわったあと最初さいしょの宮内庁くないちょうの長官ちょうかんだった田島たじま道治みちじさんが、昭和天皇しょうわてんのうとの会話かいわを書かいたノートなどの大事だいじな資料しりょうが見みつかりました。ノートには１９４９年ねんから１９５３年ねんまでの会話かいわが書かいてあります。
この中なかで、昭和天皇しょうわてんのうは戦争せんそうへの後悔こうかいについて何なん度ども話はなしていました。１９５２年ねん、日本にっぽんの独立どくりつが戻もどったことを祝いわう式しきがありました。この式しきで昭和天皇しょうわてんのうは、戦争せんそうへの深ふかい後悔こうかいと、二に度どと戦争せんそうをしないための反省はんせいの気持きもちを伝つたえたいと強つよく思おもっていました。しかし、吉田よしだ茂しげる総理大臣そうりだいじんが「戦争せんそうを始はじめた責任せきにんがあると言いわれるかもしれない」と言いって反対はんたいしました。このため、昭和天皇しょうわてんのうは自分じぶんが言いいたかった後悔こうかいの気持きもちを話はなしませんでした。
専門家せんもんかは「昭和天皇しょうわてんのうは、みんなの前まえで戦争せんそうへの後悔こうかいや反省はんせいについてはっきり話はなしたことがありません。とても深ふかい後悔こうかいの気持きもちを伝つたえたいと思おもっていたことに驚おどろきました」と話はなしています。

和歌山県わかやまけん太地町たいじちょうは、昔むかしから鯨くじらをとっているため「鯨くじらの町まち」で有名ゆうめいです。
太地町たいじちょうの「くじらの博物館はくぶつかん」は毎年まいとし夏なつ、近ちかくの海うみに鯨くじらと泳およぐことができる場所ばしょを作つくっています。今年ことしは１９日にちまで、１日にちに２回かい、博物館はくぶつかんで育そだてている鯨くじらとたくさんの人ひとが一緒いっしょに泳およいで楽たのしみました。
体からだの長ながさが３ｍぐらいの鯨くじらは、海うみにいる人ひとたちの中なかをゆっくり泳およぎながら、ときどき手てを振ふるような動うごきをしていました。海うみに来きた人ひとたちは、鯨くじらが近ちかくに来くると喜よろこんで写真しゃしんを撮とっていました。三重県みえけんから来きた７歳さいの男おとこの子こは「ひれを速はやく動うごかしたり、声こえを出だしたりしていてかわいかったです」と話はなしていました。
博物館はくぶつかんの人ひとは「鯨くじらと一緒いっしょに遊あそんで、太地町たいじちょうにも興味きょうみを持もってほしいと思おもいます」と話はなしていました。

北海道ほっかいどうには韓国かんこくからたくさんの人ひとが観光かんこうに来きます。しかし、日本にっぽんと韓国かんこくの関係かんけいが悪わるくなったあと、北海道ほっかいどうと韓国かんこくの間あいだでは一部いちぶの飛行機ひこうきが飛とばないことになりました。北海道ほっかいどうは、韓国かんこくから来くる人ひとが少すくなくなることを心配しんぱいしています。
このため、北海道ほっかいどうなどの職員しょくいんたちは韓国かんこくの人ひとたちを空港くうこうで歓迎かんげいすることにしました。新千歳空港しんちとせくうこうでは１９日にち、職員しょくいんたちが「北海道ほっかいどうへようこそ」と韓国語かんこくごで書かいた幕まくを持もって、韓国かんこくからの飛行機ひこうきで着ついた人ひとを迎むかえました。そして、韓国語かんこくごで「こんにちは」と言いいながら、うちわや北海道ほっかいどうのメロンのゼリーなどを渡わたしました。
韓国人かんこくじんの若わかい男性だんせいは「日本にっぽんに来くるのをやめようとは思おもいませんでした。歓迎かんげいしてくれてうれしいです」と話はなしていました。北海道ほっかいどうの職員しょくいんは「韓国かんこくのみなさんに、北海道ほっかいどうに安心あんしんして行いくことができると思おもってほしいです」と話はなしていました。

８月がつ１０日とおか、茨城県いばらきけんの高速道路こうそくどうろで白しろい車くるまが危険きけんな運転うんてんをして、会社員かいしゃいんの車くるまを止とめました。白しろい車くるまを運転うんてんしていた男おとこは、会社員かいしゃいんを殴なぐってけがをさせました。
会社員かいしゃいんの車くるまには、車くるまの周まわりのビデオを撮とるドライブレコーダーがついていました。警察けいさつはビデオを調しらべて、白しろい車くるまを運転うんてんしていた大阪市おおさかしの男おとこを捕つかまえました。白しろい車くるまと同おなじナンバープレートをつけた車くるまは、愛知県あいちけんと静岡県しずおかけんでも、同おなじように危険きけんな運転うんてんをしていました。

お盆ぼんが終おわる１６日にちの夜よる、３００年ねん以上いじょう前まえから続つづいている「京都五山送きょうとござんおくり火び」がありました。京都きょうとの町まちの周まわりにある５つの山やまに火ひで字じや形かたちなどを作つくって、お盆ぼんで戻もどってきた先祖せんぞの霊れいを送おくる行事ぎょうじです。
今年ことしは台風たいふうが近ちかくに来きたため心配しんぱいしていましたが、午後ごご８時じに予定よていどおり始はじまりました。まず、大文字山だいもんじやまで「大だい」の字じの形かたちに並ならべた木きに火ひをつけると、暗くらい山やまに大おおきな「大だい」という字じが明あかるく光ひかりました。そのあと、ほかの山やまでも火ひをつけて、船ふねや鳥居とりいの形かたちなどができました。

日本にっぽんのウイスキー５４本ほんがオークションで約やく１億おく円えんになる

发表于 2019-08-19 更新于 2019-11-30 分类于 JapanEasyNews



香港ほんこんで１６日にちにオークションがあって、埼玉県さいたまけんでつくられたウイスキー５４本ほんのセットが約やく１億おく円えんになりました。オークションに出でた日本にっぽんのウイスキーの中なかでは、今いままででいちばん高たかい値段ねだんになりました。
「イチローズモルト」というウイスキーで、１９８０年ねん代だいから２０００年ねんにつくられました。ウイスキーが入はいっている木きのたるが違ちがうと味あじや香かおりも違ちがうため、５４種類しゅるいの商品しょうひんにして売うっていました。
日本にっぽんのウイスキーは、おいしいと思おもう人ひとが世界せかいで多おおくなっていて、高たかい値段ねだんで売うれるようになっています。
このウイスキーを売うった会社かいしゃの人ひとは「うれしいですが、飲のんで楽たのしんでほしいので、こんなに高たかい値段ねだんだと本当ほんとうに飲のんでくれるか心配しんぱいです」と話はなしていました。
