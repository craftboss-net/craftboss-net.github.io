東京都とうきょうと「電車でんしゃが混こまないように仕事しごとに行いく時間じかんを変かえよう」

发表于 2019-01-22 更新于 2019-11-30 分类于 JapanEasyNews


東京都とうきょうとは２１日にち、「時差じさＢｉｚ」のキャンペーンを始はじめました。「時差じさＢｉｚ」は、朝あさの電車でんしゃが混こまないように仕事しごとに行いく時間じかんを早はやくしたり遅おそくしたりすることで、２年ねん前まえから夏なつに行おこなっています。今年ことしは冬ふゆも行おこなってみんなに知しってもらって、来年らいねんの東京とうきょうオリンピックとパラリンピックで電車でんしゃが混こまないようにしたいと考かんがえました。
電車でんしゃの会社かいしゃは何なん時じごろが混こんでいるかみんなに知しらせます。社員しゃいんにできるだけ混こんでいない時間じかんに電車でんしゃを利用りようするように言いう会社かいしゃもあります。
駅えきにいた会社員かいしゃいんは「６時じ半はんごろ電車でんしゃに乗のるようにしています。早はやく起おきるのは大変たいへんでしたが、慣なれました」と話はなしていました。

店みせの人ひとが気きをつけて見みていると、２１日にちの朝あさ、コーヒーのカップにカフェラテを入いれている男おとこを見みつけました。店みせの人ひとが質問しつもんすると、男おとこはコーヒーを入いれる機械きかいのボタンを間違まちがえて押おしたと言いいました。

新宿御苑しんじゅくぎょえん　３月がつからは夜よるも楽たのしむことができる

发表于 2019-01-22 更新于 2019-11-30 分类于 JapanEasyNews


東京とうきょうにある新宿御苑しんじゅくぎょえんは木きが１万まん本ぼんぐらいある広ひろい公園こうえんで、外国人がいこくじんにも人気にんきがあります。２０１７年度ねんどにこの公園こうえんに来きた２５０万まん人にんのうち、半分はんぶんが外国人がいこくじんだと言いわれています。
新宿御苑しんじゅくぎょえんが開あいている時間じかんは午前ごぜん９時じから午後ごご４時じ半はんまでですが、多おおくの人ひとが「夜よるも公園こうえんを楽たのしみたい」と言いっていました。
このため３月がつからは公園こうえんが閉しまる時間じかんが遅おそくなります。３月がつ１９日にちから６月がつ３０日にちまでと８月がつ２１日にちから９月がつ３０日にちまでは午後ごご６時じまで、７月がつ１日ついたちから８月がつ２０日はつかまでは午後ごご７時じまで楽たのしむことができます。
公園こうえんは桜さくらや紅葉こうようの季節きせつに木きがきれいに見みえるようにライトをつけたり、夜よるにイベントを行おこなったりすることも考かんがえています。

北海道ほっかいどう足寄町あしょろちょうに住すんでいる野中のなか正造まさぞうさんは、１９０５年ねん７月がつ２５日にちに生うまれました。去年きょねん４月がつには、「世界せかいでいちばん長生ながいきの男性だんせい」というギネスの世界記録せかいきろくになりました。
２０日はつか、野中のなかさんが１１３歳さいで亡なくなりました。
野中のなかさんは長ながい間あいだ、旅館りょかんを経営けいえいしていました。去年きょねんの誕生日たんじょうびには、家族かぞくと一緒いっしょに好すきなケーキを食たべました。いつもはテレビで相撲すもうを楽たのしんでいました。１９日にちまではいつもと同おなじように生活せいかつしていましたが、２０日はつか午前ごぜん１時じ半はんごろ、寝ねている野中のなかさんが呼吸こきゅうをしていないことがわかりました。
野中のなかさんと一緒いっしょに住すんでいた孫まごの女性じょせいは「おじいちゃんのおかげで、私わたしたち家族かぞくも幸しあわせな楽たのしい毎日まいにちを過すごすことができました。寂さびしいですが、おじいちゃんが亡なくなるまで家いえにいることができてよかったと思おもいます」と話はなしていました。

千葉県ちばけん　介護かいごの仕事しごとをする外国人がいこくじんを応援おうえんするための計画けいかく

发表于 2019-01-21 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんでは１９４７年ねんから１９４９年ねんに生うまれた人ひとたちがとても多おおくて、２０２５年ねんにはこの人ひとたちがみんな７５歳さい以上いじょうになります。２０２５年ねんに千葉県ちばけんは日本にっぽんで最もっとも、このようなお年寄としよりの介護かいごの仕事しごとをする人ひとが足たりなくなると言いわれています。このため千葉県ちばけんは、介護かいごの仕事しごとをする外国人がいこくじんを応援おうえんする計画けいかくを作つくりました。
計画けいかくでは、介護かいごの技術ぎじゅつを習ならっている技能実習生ぎのうじっしゅうせいが、日本語にほんごを勉強べんきょうするためのお金かねの一部いちぶを千葉県ちばけんが出だします。外国人がいこくじんが生活せいかつや仕事しごとで困こまったことを相談そうだんできる場所ばしょを新あたらしく作つくります。
千葉県ちばけんで介護かいごの仕事しごとをする予定よていの留学生りゅうがくせいには、学校がっこうで介護かいごの勉強べんきょうをするためのお金かねや、日本にっぽんで住すむ家いえや部屋へやの家賃やちんの一部いちぶを出だします。
千葉県ちばけんは、この計画けいかくのために、２０１９年度ねんどに約やく１億おく８００万まん円えん使つかいたいと考かんがえています。
