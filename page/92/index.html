「世界せかいでいちばん長生ながいきの男性だんせい」の野中のなかさんが亡なくなる

发表于 2019-01-21 更新于 2019-11-30 分类于 JapanEasyNews


北海道ほっかいどう足寄町あしょろちょうに住すんでいる野中のなか正造まさぞうさんは、１９０５年ねん７月がつ２５日にちに生うまれました。去年きょねん４月がつには、「世界せかいでいちばん長生ながいきの男性だんせい」というギネスの世界記録せかいきろくになりました。
２０日はつか、野中のなかさんが１１３歳さいで亡なくなりました。
野中のなかさんは長ながい間あいだ、旅館りょかんを経営けいえいしていました。去年きょねんの誕生日たんじょうびには、家族かぞくと一緒いっしょに好すきなケーキを食たべました。いつもはテレビで相撲すもうを楽たのしんでいました。１９日にちまではいつもと同おなじように生活せいかつしていましたが、２０日はつか午前ごぜん１時じ半はんごろ、寝ねている野中のなかさんが呼吸こきゅうをしていないことがわかりました。
野中のなかさんと一緒いっしょに住すんでいた孫まごの女性じょせいは「おじいちゃんのおかげで、私わたしたち家族かぞくも幸しあわせな楽たのしい毎日まいにちを過すごすことができました。寂さびしいですが、おじいちゃんが亡なくなるまで家いえにいることができてよかったと思おもいます」と話はなしていました。

去年きょねん１１月がつ２０日はつかに出発しゅっぱつして、今いままで日本人にっぽんじんが全部ぜんぶ歩あるくことができなかった難むずかしいルートを、荷物にもつを乗のせたそりを引ひきながら歩あるきました。そして約やく２か月げつが過すぎた今月こんげつ１７日にち、９２０ｋｍを歩あるいて南極点なんきょくてんに着ついたと阿部あべさんから日本にっぽんに電話でんわがありました。

御嶽山おんたけさんの噴火ふんかを３Ｄで見みせる「危あぶない場所ばしょを知しっておいて」

发表于 2019-01-17 更新于 2019-11-30 分类于 JapanEasyNews


長野県ながのけんと岐阜県ぎふけんの間あいだにある御嶽山おんたけさんは５年ねん前まえに噴火ふんかがあって、飛とんできた石いしなどで大勢おおぜいの人ひとが亡なくなりました。
防衛大学校ぼうえいだいがっこうの先生せんせいたちは、コンピューターの３Ｄの技術ぎじゅつを使つかってこの噴火ふんかの映像えいぞうを作つくりました。山やまの様子ようすがはっきり見みえる映像えいぞうで、噴火ふんかのときに石いしがどう飛とぶかがわかります。
高校生こうこうせいが協力きょうりょくして、この映像えいぞうにＶＲ＝バーチャルリアリティの技術ぎじゅつを使つかいました。専用せんようの眼鏡めがねをかけて映像えいぞうを見みると、自分じぶんがその場所ばしょに立たっているようにいろいろな方向ほうこうを見みることができます。そして、石いしがどこに飛とんでくるか、どこに逃にげたらいいかがよくわかります。
長野県ながのけん木曽町きそまちは、この映像えいぞうを御嶽山おんたけさんに登のぼる人ひとたちに見みせて、噴火ふんかのとき危あぶない場所ばしょがわかるようにしたいと考かんがえています。

２０１６年ねんに「がん」が見みつかった人ひとは１００万まん人にん

发表于 2019-01-17 更新于 2019-11-30 分类于 JapanEasyNews


日本人にっぽんじんが亡なくなる理由りゆうで最もっとも多おおいのは「がん」という病気びょうきです。厚生労働省こうせいろうどうしょうは２０１６年ねんから、がんになった人ひとの情報じょうほうを日本中にほんじゅうの病院びょういんから集あつめる新あたらしい制度せいどを始はじめました。がんの研究けんきゅうなどに役やくに立たてるためです。
厚生労働省こうせいろうどうしょうによると、２０１６年ねんにがんが見みつかった人ひとは９９万まん５１３２人にんでした。
男性だんせいは胃いのがんが最もっとも多おおくて９万まん２６００人にんでした。膀胱ぼうこうの近ちかくにある前立腺ぜんりつせんのがんは８万まん９７００人にん、大腸だいちょうのがんは８万まん９６００人にんでした。
女性じょせいは乳房ちぶさのがんが最もっとも多おおくて９万まん４８００人にんでした。大腸だいちょうのがんは６万まん８４００人にん、胃いのがんは４万まん１９００人にんでした。
厚生労働省こうせいろうどうしょうは「集あつまった情報じょうほうをよく調しらべて、がんの治療ちりょうに役やくに立たつようにしたいです。がんになる人ひとも少すくなくしたいです」と話はなしています。
