日本にっぽんのスーパーコンピューター　２つのランキングで１番ばん

发表于 2019-06-26 更新于 2019-11-30 分类于 JapanEasyNews



スーパーコンピューターの世界せかいランキングは１年ねんに２回かい発表はっぴょうされています。新あたらしいランキングが６月がつに発表はっぴょうされました。
日本にっぽんの理化学研究所りかがくけんきゅうしょの「菖蒲しょうぶシステムＢ」が、使つかうエネルギーが少すくないコンピューターのランキングで、４回かい続つづけて１番ばんになりました。
ビッグデータを分析ぶんせきする能力のうりょくのランキングでも、理化学研究所りかがくけんきゅうしょの「京けい」が９回かい続つづけて１番ばんになりました。
日本にっぽんのスーパーコンピューターは、実際じっさいに会社かいしゃなどが使つかうときに役やくに立たつ２つのランキングで１番ばんになりました。
そのほか、計算けいさんの速はやさのランキングでは、アメリカの国立研究所こくりつけんきゅうしょの「Ｓｕｍｍｉｔ」が１秒びょうに１４京けい８６００兆ちょう回かい計算けいさんして、１番ばんになりました。１京けいは１兆ちょうの１万まん倍ばいです。日本にっぽんの産業技術総合研究所さんぎょうぎじゅつそうごうけんきゅうしょの「ＡＢＣＩ」は８番ばんでした。

手足口病てあしくちびょうになる子こどもが増ふえている「よく手てを洗あらって」

发表于 2019-06-26 更新于 2019-11-30 分类于 JapanEasyNews



手足口病てあしくちびょうは５歳さい以下いかの子こどもに多おおい病気びょうきで、手てや足あし、口くちの中なかなどに小ちいさい発疹はっしんができます。毎年まいとし、夏なつに多おおくなっていて、せきやくしゃみ、便べんなどに入はいっているウイルスでうつります。
国立感染症研究所こくりつかんせんしょうけんきゅうしょによると、６月がつ１６日にちまでの１週間しゅうかんに手足口病てあしくちびょうの子こどもは１万まん２７００人にん以上いじょういました。その前まえの１週間しゅうかんより３８００人にんぐらい増ふえて、６月がつごろでは今いままでの１０年ねんでいちばん多おおくなりました。特とくに佐賀県さがけん、福岡県ふくおかけん、鹿児島県かごしまけん、そして大阪府おおさかふで多おおくなっています。
国立感染症研究所こくりつかんせんしょうけんきゅうしょは「今年ことしの手足口病てあしくちびょうのウイルスはうつりやすいです。よく手てを洗あらいましょう。赤あかちゃんのおむつを取とり替かえたあとはきれいに片付かたづけてください」と言いっています。

荷物にもつを届とどける会社かいしゃでは働はたらく人ひとが足たりなくなっています。荷物にもつが増ふえたため、特とくに、家いえや会社かいしゃなどに１軒けんずつ荷物にもつを届とどける人ひとが足たりません。このため、荷物にもつを届とどける会社かいしゃや国くには、人ひとの代かわりに自動じどうで走はしるロボットを使つかおうと考かんがえています。
２４日にじゅうよっか、経済産業省けいざいさんぎょうしょうは日本にっぽんやアメリカ、中国ちゅうごくなどの会社かいしゃが作つくったロボットを集あつめて、どのように走はしるかなどをチェックしました。
アメリカのロボットは、狭せまい道みちでも人ひとなどにぶつからないように走はしることができます。日本にっぽんのロボットはカメラやセンサーで周まわりを見みて、人ひとの近ちかくを通とおるとき「こんにちは」と言いいます。
アメリカやヨーロッパでは、このようなロボットが道みちを走はしることができる新あたらしい法律ほうりつを作つくり始はじめています。しかし、日本にっぽんの今いまの法律ほうりつではみんなが利用りようする道みちを走はしることはできません。
国くには来年らいねん３月がつまでに、ロボットが安全あんぜんに走はしるためのルールの案あんをいろいろな会社かいしゃと一緒いっしょに考かんがえます。
