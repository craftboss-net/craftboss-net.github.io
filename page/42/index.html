障害しょうがいがある人ひと　去年きょねんは８２万まん人にん以上いじょうが会社かいしゃで働はたらいた

发表于 2019-06-27 更新于 2019-11-30 分类于 JapanEasyNews



厚生労働省こうせいろうどうしょうは５年ねんに１回かい、日本にっぽんの会社かいしゃで体からだや心こころに障害しょうがいがある人ひとが何なん人にん働はたらいているか計算けいさんしています。去年きょねん働はたらいた人ひとは８２万まん１０００人にんになりました。５年ねん前まえより１９万まん人にん増ふえて、今いままででいちばん多おおくなりました。
働はたらいている人ひとのうち、体からだに障害しょうがいがある人ひとの５０％以上いじょうが会社かいしゃの社員しゃいんになっています。しかし、心こころや知能ちのうに障害しょうがいがある人ひとで社員しゃいんになっている人ひとは２０％ぐらいです。
厚生労働省こうせいろうどうしょうは「会社かいしゃの中なかの障害しょうがいがある人ひとの割合わりあいは法律ほうりつで決きまっていて、去年きょねん上あがりました。障害しょうがいがある人ひとに働はたらいてほしいと考かんがえる会社かいしゃも増ふえています」と言いっています。

「子こども食堂しょくどう」が増ふえている

发表于 2019-06-27 更新于 2019-11-30 分类于 JapanEasyNews



「子こども食堂しょくどう」は、親おやが働はたらいていて１人ひとりで食事しょくじをしなければならない子こどもなどが、無料むりょうか少すくないお金かねで食事しょくじができる所ところです。
子こども食堂しょくどうを応援おうえんしているＮＰＯによると、子こども食堂しょくどうは今年ことし５月がつに３７１８ありました。去年きょねん４月がつより１４００以上いじょう増ふえて、１．６倍ばいになりました。子こども食堂しょくどうがいちばん多おおい所ところは東京都とうきょうとで、４８８ありました。人口じんこうが多おおい所ところで多おおくなっています。少すくない所ところは１１ある秋田県あきたけんなどでした。
このＮＰＯは、小学校しょうがっこうがある所ところに子こども食堂しょくどうも１つあるのがいちばんいいと考かんがえています。沖縄県おきなわけんにある子こども食堂しょくどうの数かずは小学校しょうがっこうの数かずの６０．５％で、日本にっぽんでいちばん高たかい割合わりあいになっています。いちばん低ひくい所ところは秋田県あきたけんで５．５％でした。
ＮＰＯは「子こども食堂しょくどうは、お金かねに困こまっている家庭かていの子こどものためだけではなくて、子こどもからお年寄としよりまでいろいろな人ひとが交流こうりゅうできる所ところです。このことをみなさんに知しってほしいです」と話はなしています。

发表于 2019-06-27 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどうの清里町きよさとちょうにある斜里川しゃりがわでは今いまの季節きせつ、卵たまごを産うむために海うみから戻もどったサクラマスという魚さかなが川かわを上のぼります。
斜里川しゃりがわには高たかさ３．７ｍの滝たきがあります。サクラマスが川かわを上のぼっていくためには、この滝たきを越こえなければなりません。
大おおきさ４０ｃｍぐらいのサクラマスは滝たきに向むかって高たかくジャンプをします。しかし、水みずの流ながれが速はやくて、なかなか滝たきを越こえることができません。
何なん度どもジャンプするサクラマスを見みた人ひとは「心こころの中なかで『頑張がんばって』と思おもいました」などと話はなしていました。ビデオを撮とっている人ひともいました。
斜里川しゃりがわでは、８月がつの初はじめごろまでサクラマスのジャンプを見みることができます。

发表于 2019-06-26 更新于 2019-11-30 分类于 JapanEasyNews



山形県やまがたけん鶴岡市つるおかしでは、今月こんげつ１８日にちの大おおきな地震じしんで屋根やねの瓦かわらが落おちた家いえがたくさんありました。
地震じしんのあと、市しに住すんでいる人ひとから警察けいさつに「あやしい人ひとが屋根やねを直なおしに来きた」という相談そうだんが１２件けんありました。
鶴岡市つるおかしの人ひとによると、あやしい人ひとが急きゅうに来きて「雨あめが入はいらないように、無料むりょうで屋根やねにシートをかける」と言いって、屋根やねに上あがりました。そのあと、家いえの人ひとに屋根やねの写真しゃしんを見みせながら「直なおす必要ひつようがある」と言いって、また屋根やねに上あがりました。そして、とても高たかいお金かねを払はらうように言いいました。
鶴岡市つるおかしでは、市しが頼たのんだ会社かいしゃが壊こわれた屋根やねに無料むりょうでシートをかけています。しかし、そのときに屋根やねを直なおしたり、お金かねを払はらうように言いったりすることはありません。市しは「おかしいと思おもったら、消費生活しょうひせいかつセンターなどに相談そうだんしてください」と言いっています。

日本にっぽんの理化学研究所りかがくけんきゅうしょの「菖蒲しょうぶシステムＢ」が、使つかうエネルギーが少すくないコンピューターのランキングで、４回かい続つづけて１番ばんになりました。

大阪おおさかでＧ２０　交通こうつうの情報じょうほうがわかるカードを外国人がいこくじんに配くばる

发表于 2019-06-25 更新于 2019-11-30 分类于 JapanEasyNews



今月こんげつ２８日にちと２９日にち、大阪おおさかでＧ２０サミットがあります。この会議かいぎには国くにや国連こくれんなど３７の代表だいひょうが出席しゅっせきします。会場かいじょうの近ちかくの道みちなどでは車くるまが通とおることができなくなります。
関西空港かんさいくうこうでは２４日にじゅうよっか、旅行りょこうに来きた外国人がいこくじんに交通こうつうの情報じょうほうなどを知しらせるための小ちいさいカードを配くばりました。
カードの表おもてには空港くうこう、鉄道てつどう、バスなどを意味いみする絵えが、英語えいご、中国語ちゅうごくご、韓国語かんこくごと一緒いっしょに書かいてあります。裏うらには、空港くうこうや鉄道てつどう会社がいしゃなどのウェブサイトのＱＲコードがあって、必要ひつような情報じょうほうを簡単かんたんに調しらべることができます。
アメリカから観光かんこうに来きた女性じょせいは「大阪おおさかでＧ２０があるのは知しっていましたが、車くるまが通とおることができなくなる所ところがあることは知しりませんでした。このカードは便利べんりだと思おもいます」と話はなしていました。
