東京とうきょうオリンピック　マラソンと競歩きょうほは札幌さっぽろで行おこなうことを考かんがえる

发表于 2019-10-17 更新于 2019-11-30 分类于 JapanEasyNews



ＩＯＣは１６日にち、来年らいねんの東京とうきょうオリンピックのマラソンと競歩きょうほを北海道ほっかいどうの札幌さっぽろで行おこなうことを考かんがえていると言いいました。東京とうきょうの暑あつさが心配しんぱいなためだと言いっています。
９月がつからカタールのドーハで陸上競技りくじょうきょうぎの世界せかい選手権せんしゅけんがありました。とても暑あつかったため、女子じょしのマラソンや５０ｋｍの競歩きょうほで、４０％の選手せんしゅが途中とちゅうでやめました。
ＩＯＣが会場かいじょうを変かえる案あんを出だすことは、今いままでほとんどありませんでした。３０日にちからのＩＯＣの会議かいぎでどうするか話はなし合あいます。ＩＯＣのバッハ会長かいちょうは「選手せんしゅがいちばんいい成績せいせきを出だすことができるようにしたいと思おもいます」と話はなしています。
日本にっぽんでは９月がつ、東京とうきょうのマラソンのコースで、オリンピックに出でる選手せんしゅを選えらぶ大会たいかいがありました。選手せんしゅの監督かんとくは「選手せんしゅたちは東京とうきょうのコースを考かんがえて練習れんしゅうしてきたので、どうしようという気持きもちだと思おもいます。札幌さっぽろになったら気持きもちを変かえて頑張がんばります」と話はなしています。

台風たいふう　避難所ひなんじょで生活せいかつするときに気きをつけること

发表于 2019-10-16 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふうで家いえが壊こわれたりして避難所ひなんじょで生活せいかつするときは、次つぎのことに気きをつけてください。
赤あかちゃんのミルクが足たりないときは、コップ１杯ぱいのお湯ゆに砂糖さとうを大おおさじ１杯ぱい入いれて、熱あつくなくなったら、飲のませてください。水みずを多おおく入いれてやわらかく煮にたおかゆのスープでもいいです。
避難所ひなんじょではかぜなどの病気びょうきが広ひろがりやすいため、特とくにお年寄としよりはしっかり手てを洗あらって、うがいをしてください。いつもマスクをしていてください。朝あさと夜よるに歯はや入いれ歯ばを洗あらうことも必要ひつようです。
長ながい時間じかん体からだを動うごかさないと「エコノミークラス症候群しょうこうぐん」になって、亡なくなる人ひともいます。足首あしくびなど体からだをときどき動うごかしてください。
トイレに行いくのが嫌いやで、あまり水みずを飲のまないようにすると、体からだの水分すいぶんが足たりなくなる心配しんぱいがあります。しっかり水みずを飲のんでください。

観光かんこうで有名ゆうめいな箱根町はこねまち　台風たいふうで道みちが崩くずれて線路せんろが流ながされる

发表于 2019-10-16 更新于 2019-11-30 分类于 JapanEasyNews



神奈川県かながわけん箱根町はこねまちはこれから紅葉こうようの季節きせつで、毎年まいとしたくさんの人ひとが観光かんこうに来きます。
１２日にちから１３日にちに来きた台風たいふう１９号ごうで、箱根町はこねまちには雨あめがとてもたくさん降ふりました。山やまや道みちが崩くずれたり、道みちに水みずがあふれたりしました。落おちてきた大おおきな岩いわや倒たおれた木きがあって、通とおることができない道みちもあります。多おおくのバスが運転うんてんを休やすんでいます。

菓子かしを売うる店みせの人ひとは「交通こうつうが止とまっていると、みんなが観光かんこうに来くることができないので、困こまります」と話はなしていました。

信号機しんごうきの情報じょうほうを利用りようして自動じどう運転うんてんをもっと安全あんぜんにする実験じっけん

发表于 2019-10-16 更新于 2019-11-30 分类于 JapanEasyNews



１５日にち、人ひとが運転うんてんしない「自動じどう運転うんてん」の車くるまが、信号機しんごうきが送おくる情報じょうほうを利用りようして、安全あんぜんに走はしることができるかどうか調しらべる実験じっけんが始はじまりました。実験じっけんは、東京とうきょうのお台場だいばや羽田空港はねだくうこうの道みちで行おこないます。
実験じっけんでは、カメラや機械きかいを付つけた１００台だいぐらいの車くるまやバスが、信号機しんごうきから情報じょうほうを受うけながら、自動じどう運転うんてんで走はしります。お台場だいばでは３０か所しょぐらいの信号機しんごうきが、今いまどんな色いろか、あと何なん秒びょうで色いろが変かわるかなどの情報じょうほうを車くるまに送おくります。
今いままでは自動じどう運転うんてんの車くるまがカメラを使つかって信号機しんごうきの色いろを調しらべていましたが、暗くらい場所ばしょなどでは色いろがよくわかりませんでした。国くにや車くるまの会社かいしゃなどは、信号機しんごうきが出だす情報じょうほうを利用りようして、もっと自動じどう運転うんてんを安全あんぜんにしたいと考かんがえています。
実験じっけんは２０２１年ねん３月がつまで行おこないます。

水みずが入はいった家いえは病気びょうきに気きをつけながら掃除そうじをしてください

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



台風たいふう１９号ごうの雨あめで、川かわの水みずがあふれて、たくさんの家いえに水みずが入はいりました。
厚生労働省こうせいろうどうしょうは、家いえの中なかに水みずが入はいると細菌さいきんやかびが増ふえやすくなって、病気びょうきになる危険きけんが高たかくなるので、きれいに掃除そうじをすることが大切たいせつだと言いっています。そして、次つぎのことに気きをつけて掃除そうじをしてほしいと言いっています。
ドアや窓まどを開あけて、きれいな空気くうきが入はいるようにしてください。ほこりを吸すわないように、ゴーグルやマスクをしてください。掃除そうじが終おわったあとは、しっかり手てを洗あらってください。
掃除そうじをしている間あいだにけがをして、傷きずから細菌さいきんなどが入はいらないように、厚あつい手袋てぶくろをして、底そこが厚あつい靴くつをはいて、袖そでが長ながい服ふくを着きてください。泥どろを片かたづけたあとは、家いえの中なかを消毒しょうどくしてください。

ラグビー　日本にっぽんが初はじめて「決勝けっしょうトーナメント」に進すすむ

发表于 2019-10-15 更新于 2019-11-30 分类于 JapanEasyNews



ラグビーのワールドカップで１３日にち、日本にっぽんはスコットランドと試合しあいをして、２８－２１で勝かちました。試合しあいが始はじまってすぐスコットランドがトライをしましたが、そのあと日本にっぽんが４つトライをして勝かちました。
日本にっぽんは、グループで行おこなう４つの試合しあいに全部ぜんぶ勝かって、グループで１番ばんになりました。そして、８つのチームで優勝ゆうしょうを決きめるための「決勝けっしょうトーナメント」に出でることが決きまりました。日本にっぽんがラグビーのワールドカップに出でるのは９回かい目めで、「決勝けっしょうトーナメント」に出でるのは初はじめてです。
試合しあいがあった横浜市よこはましの会場かいじょうには６万まん７０００人にん以上いじょうの人ひとが見みに来きました。試合しあいの前まえ、台風たいふうで亡なくなった人ひとたちのために、みんなで静しずかに祈いのりました。
日本にっぽんは２０日はつかに東京とうきょうで南みなみアフリカと試合しあいをします。日本にっぽんのキャプテンのリーチ　マイケル選手せんしゅは「もっと勝かって上うえに行いって、日本にっぽんの歴史れきしを作つくりたいです」と話はなしています。
