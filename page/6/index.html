鹿児島県かごしまけんいちき串木野くしきの市し「サメをおいしく食たべよう」

发表于 2019-03-14 更新于 2019-11-30 分类于 JapanEasyNews



鹿児島県かごしまけんの薩摩半島さつまはんとうの沖おきでは、小ちいさいサメが魚さかなをとる道具どうぐを壊こわしたり、サメが網あみの中なかに入はいってほかの魚さかながとれなくなったりしています。いちき串木野くしきの市しは、このような困こまった被害ひがいを出だすサメを捕つかまえて、おいしく食たべようと考かんがえています。
１２日にち、サメの料理りょうりを紹介しょうかいする会かいがありました。集あつまった人ひとたちは、高校生こうこうせいなどが作つくった１０種類しゅるいの料理りょうりを食たべました。サメを切きって油あぶらで揚あげた料理りょうりや、サメを使つかったカレーパンがありました。食たべたとき鶏肉とりにくに似にていると感かんじる料理りょうりもありました。
料理りょうりを作つくった高校生こうこうせいは「サメのにおいを消けすのが大変たいへんでしたが、いろいろやり方かたを考かんがえたらおいしくなることがわかりました」と話はなしていました。
いちき串木野くしきの市しは、サメの栄養えいようやサメを保存ほぞんする方法ほうほうを調しらべて、みんなにもっと食たべてほしいと考かんがえています。

アフリカで仕事しごとをしたい会社かいしゃに留学生りゅうがくせいを紹介しょうかいするイベント

发表于 2019-03-14 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんの会社かいしゃにアフリカの留学生りゅうがくせいを紹介しょうかいするイベントが東京とうきょうでありました。これからアフリカで仕事しごとをしようと考かんがえている１１０の会社かいしゃが参加さんかしました。会場かいじょうでは、日本にっぽんの大学だいがくで勉強べんきょうしている約やく１００人にんの留学生りゅうがくせいが、会社かいしゃの人ひとの所ところに行いって熱心ねっしんに説明せつめいを聞ききました。
バイクを作つくる会社かいしゃには大勢おおぜいの留学生りゅうがくせいが集あつまりました。この会社かいしゃは、川かわの水みずをきれいにして飲のむことができるようにする機械きかいも売うりたいと考かんがえています。機械きかいは太陽たいようの光ひかりで電気でんきを作つくって動うごきます。
会社かいしゃの人ひとは「アフリカで生活せいかつの役やくに立たつ仕事しごとをしたいので、アフリカをよく知しっている人ひとと一緒いっしょに仕事しごとをしたいです」と話はなしていました。
参加さんかした留学生りゅうがくせいは「日本にっぽんは自然しぜんを壊こわさない技術ぎじゅつが多おおいです。ビジネスのやり方かたを勉強べんきょうして、自分じぶんの国くにに伝つたえたいと思おもいます」と話はなしていました。

先生せんせいの仕事しごとを減へらすため家庭かていとの連絡れんらくにＩＴの技術ぎじゅつを使つかう

发表于 2019-03-14 更新于 2019-11-30 分类于 JapanEasyNews



学校がっこうの先生せんせいは仕事しごとが多おおくて、中学校ちゅうがっこうの先生せんせいの６０％は１か月げつの残業ざんぎょう時間じかんが８０時間じかん以上いじょうになりそうだと言いわれています。
神奈川県かながわけん横浜市よこはましが中学校ちゅうがっこうの先生せんせいの仕事しごとを調しらべると、家庭かていとの連絡れんらくに１日にち平均へいきん１３分ぷん、１年ねんで４４時間じかん以上いじょうかかっていました。先生せんせいはいつも、学校がっこうを休やすむという電話でんわを受うけたり、家庭かていに渡わたすプリントを作つくったりしています。
横浜市よこはましは先生せんせいの仕事しごとを減へらすため、来月らいげつから６つの学校がっこうで家庭かていとの連絡れんらくにＩＴの技術ぎじゅつを使つかってみることにしました。
学校がっこうには２４時間じかんいつでも電話でんわできて、機械きかいがいろいろな国くにの言葉ことばで案内あんないします。休やすむ理由りゆうは番号ばんごうを押おして伝つたえます。学校がっこうはプリントではなくてメールで家庭かていに連絡れんらくします。学校がっこうに来きた連絡れんらくはコンピューターがまとめます。
横浜市よこはましは６つの学校がっこうでうまくいったら、全部ぜんぶの学校がっこうもこのようにしたいと考かんがえています。

１０日とおか、エチオピアから隣となりのケニアに行いくエチオピア航空こうくうの飛行機ひこうきが、飛とび始はじめてすぐに落おちました。乗のっていた１５７人にんが亡なくなりました。
飛行機ひこうきが落おちる前まえ、パイロットは、問題もんだいが起おこったため空港くうこうに戻もどると連絡れんらくしましたが、落おちた原因げんいんはまだわかっていません。

札幌市さっぽろしの円山まるやま動物園どうぶつえん　象ぞうを見みせるための場所ばしょと建物たてものがオープン

发表于 2019-03-13 更新于 2019-11-30 分类于 JapanEasyNews



北海道ほっかいどう札幌市さっぽろしの円山まるやま動物園どうぶつえんに、ミャンマーから来きた３頭とうの象ぞうをみんなに見みせるための広ひろい場所ばしょと建物たてものができました。３頭とうはアジアゾウという象ぞうで、５歳さいから２７歳さいまでの雌めすです。
１２日にち、この場所ばしょと建物たてものがオープンしました。広ひろさは５０００m²以上いじょうあります。建物たてものの中なかには象ぞうのための深ふかさ３ｍのプールがあります。象ぞうの足あしが痛いたくならないように、歩あるく所ところにはたくさんの砂すなが敷しいてあって、象ぞうが生活せいかつしやすくなっています。

９日ここのか、川口市かわぐちしは入学にゅうがくする生徒せいとを集あつめて、学校がっこうの生活せいかつやルールなどを説明せつめいしました。入学にゅうがくする８０人にんのうち４４人にんが外国人がいこくじんのため、説明せつめいがわかるように資料しりょうには漢字かんじの読よみ方かたや英語えいごなども書かきました。授業じゅぎょうでは、外国人がいこくじんのために日本語にほんごや日本にっぽんの習慣しゅうかんや文化ぶんかも教おしえる予定よていです。

ＮＴＴ東日本ひがしにほんは、町まちの中なかや駅えきにある「公衆電話こうしゅうでんわ」の使つかい方かたを子こどもに教おしえるウェブサイトを作つくりました。ウェブサイトでは、ビデオやクイズで公衆電話こうしゅうでんわの使つかい方かたを紹介しょうかいしています。公衆電話こうしゅうでんわがどこにあるか調しらべることができます。
公衆電話こうしゅうでんわは、大おおきな地震じしんがあったときや電気でんきが止とまったときでも、ほかの電話でんわよりつながりやすくなっています。８年ねん前まえの東日本大震災ひがしにほんだいしんさいのとき、携帯電話けいたいでんわはつながりにくくなりましたが、公衆電話こうしゅうでんわで多おおくの人ひとが連絡れんらくすることができました。
しかしＮＴＴ東日本ひがしにほんが調しらべると、「公衆電話こうしゅうでんわを知しらない」とか「使つかったことがない」という子こどもがたくさんいることがわかりました。
ＮＴＴ東日本ひがしにほんの人ひとは「子こどもたちに公衆電話こうしゅうでんわを使つかってみてほしいです。そして、地震じしんなどのとき使つかうことができるようにしてほしいです」と話はなしていました。

发表于 2019-03-12 更新于 2019-11-30 分类于 JapanEasyNews



スキーのジャンプの小林こばやし陵侑りょうゆう選手せんしゅは、去年きょねんから今年ことしのワールドカップの試合しあいで１１回かい優勝ゆうしょうしています。全部ぜんぶの成績せいせきを足たした「個人こじん総合そうごう」では１番ばんになっています。
１０日とおかにノルウェーで試合しあいがあって、小林こばやし選手せんしゅは５番ばんでした。まだ試合しあいは５つありますが、個人こじん総合そうごうの成績せいせきでほかの選手せんしゅが小林こばやし選手せんしゅより上うえになる可能性かのうせいはなくなりました。このため、小林こばやし選手せんしゅが総合そうごう優勝ゆうしょうすることが決きまりました。

小林こばやし選手せんしゅは岩手県いわてけんの出身しゅっしんで２２歳さいです。「すごくうれしいです。こんなに早はやく決きまるとは思おもっていませんでした。周まわりの人ひとにすごく感謝かんしゃしています」と話はなしていました。
