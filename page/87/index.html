中国ちゅうごく　春節しゅんせつの長ながい休やすみに大勢おおぜいの人ひとが外国がいこくを旅行りょこうする

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


中国ちゅうごくでは、昔むかしのカレンダーで正月しょうがつの「春節しゅんせつ」をお祝いわいします。今年ことしは４日よっかから１０日とおかまでが春節しゅんせつの休やすみです。
中国ちゅうごくの空港くうこうは、春節しゅんせつの休やすみに外国がいこくを旅行りょこうする人ひとたちで混こんでいます。旅行会社りょこうがいしゃによると、中国ちゅうごくの人ひとに人気にんきがある国くには１番ばんはタイで、２番ばんは日本にっぽんです。
去年きょねん日本にっぽんを旅行りょこうした中国ちゅうごくの人ひとは約やく８３８万まん人にんで、今いままででいちばん多おおくなりました。先月せんげつから日本にっぽんのビザが取とりやすくなって、今年ことしの春節しゅんせつの休やすみも旅行りょこうしたり買かい物ものしたりする人ひとが多おおくなりそうです。
家族かぞく３人にんで初はじめて日本にっぽんを旅行りょこうするという男性だんせいは「鉄瓶てつびんやウイスキーのグラスを買かうつもりです。景色けしきを見みたり、いろいろな人ひとと話はなしたりして楽たのしみたいです」と話はなしていました。

「未来みらいのミライ」がアメリカでアニメ映画えいがの賞しょうをもらう

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


アニメの「アカデミー賞しょう」と言いわれている「アニー賞しょう」は、アニメのすばらしい映画えいがや監督かんとくなどに贈おくる賞しょうで、１９７２年ねんに始はじまりました。
アメリカで２日ふつか、日本にっぽんの細田守ほそだまもる監督かんとくの「未来みらいのミライ」が、アニー賞しょうの中なかの「インディペンデント作品さくひん賞しょう」をもらいました。この賞しょうは、アメリカで上映じょうえいした映画館えいがかんが少すくない映画えいがの中なかから選えらばれます。細田ほそだ監督かんとくは５１歳さいで、今いままでも外国がいこくの映画えいがのイベントなどで賞しょうをもらっています。
「未来みらいのミライ」は４歳さいの男おとこの子こが、未来みらいから来きた妹いもうとに会あっていろいろな経験けいけんをしながら、家族かぞくの愛あいを知しる話はなしです。
細田ほそだ監督かんとくは「僕ぼくの２人ふたりの子こどもと一緒いっしょにいて、小ちいさな子こどもと過すごすことは本当ほんとうにすばらしいと思おもって、この映画えいがを作つくりました」と話はなしました。

岡山県おかやまけん総社市そうじゃし　久ひさしぶりに集あつまってみんなで「恵方巻えほうまき」を食たべる

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


岡山県おかやまけん総社市そうじゃしの下原地区しもばらちくは、去年きょねん７月がつに雨あめがたくさん降ふって、川かわの水みずがあふれたり、家いえに水みずが入はいってきたりしました。工場こうじょうで爆発ばくはつもありました。今いまも８０の家族かぞくが家いえに住すむことができなくて、別べつの場所ばしょで生活せいかつしています。
３日みっかは節分せつぶんで、別べつの場所ばしょで生活せいかつしている人ひとたち１００人にんぐらいが下原地区しもばらちくに集あつまりました。そして、節分せつぶんに食たべるといいことがあると言いわれている「恵方巻えほうまき」というすしを作つくりました。
下原地区しもばらちくの米こめを使つかって、６種類しゅるいの材料ざいりょうを一緒いっしょにのりで巻まいて恵方巻えほうまきを作つくりました。みんなは恵方巻えほうまきを食たべて、久ひさしぶりに会あった人ひとたちと楽たのしそうに話はなしていました。
恵方巻えほうまきを食たべた女性じょせいは「早はやく家いえに帰かえることができるように願ねがって食たべました。近所きんじょの人ひとたちに会あうことができてよかったです」と話はなしていました。

作つくった学生がくせいは「色いろをつけるのが難むずかしかったですが、うまくできました。たくさんの人ひとに見みてほしいです」と話はなしていました。
この作品さくひんは２月がつ３日みっかまで立川市たちかわしにあるデパートに飾かざってあります。

国連こくれん　北朝鮮きたちょうせんは石油せきゆ製品せいひんの輸入ゆにゅうや石炭せきたんの輸出ゆしゅつを続つづけている

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


国連こくれんは、北朝鮮きたちょうせんが核兵器かくへいきやミサイルの実験じっけんをしたあと、北朝鮮きたちょうせんの輸入ゆにゅうや輸出ゆしゅつについての制裁せいさいを決きめました。そして、北朝鮮きたちょうせんについての制裁せいさいが守まもられているかどうか調しらべています。
国連こくれんによると、北朝鮮きたちょうせんの船ふねは去年きょねん１月がつから８月がつまでに１５０回かいぐらい、外国がいこくの船ふねから海うみの上うえでガソリンなど石油せきゆの製品せいひんをもらっていました。そして、国連こくれんが決きめたよりたくさんの石油せきゆの製品せいひんを輸入ゆにゅうしていました。
国連こくれんは、北朝鮮きたちょうせんが石炭せきたんや鉄てつを輸出ゆしゅつすることを禁止きんししています。しかし北朝鮮きたちょうせんは、ロシアの石炭せきたんだとうそを言いって、ロシアの港みなとを通とおって石炭せきたんを輸出ゆしゅつしていました。うそを言いって輸出ゆしゅつした石炭せきたんを売うると、５５８万まんドルぐらいになります。
国連こくれんによると、北朝鮮きたちょうせんはほかにも、近ちかくの海うみで魚さかなをとる権利けんりを中国ちゅうごくの船ふねに売うったりして、制裁せいさいに違反いはんすることをいろいろ行おこなっていました。

天然記念物てんねんきねんぶつの「ニホンヤマネ」　毛布もうふの中なかで眠ねむっていた

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


「ニホンヤマネ」は体からだの大おおきさが８ｃｍぐらいのネズミやリスに似にている動物どうぶつで、日本にっぽんだけにいる種類しゅるいです。１００万まん年ねん以上いじょう前まえからいるので、「生いきた化石かせき」と呼よばれていて、国くにの天然記念物てんねんきねんぶつになっています。
大分県おおいたけん日田市ひたしに住すんでいる永瀬ながせ豊子とよこさんは去年きょねん１２月がつの終おわりに、泊とまりに来くる家族かぞくのために押おし入いれの中なかから布団ふとんなどを出だしていました。そして、毛布もうふの中なかで丸まるくなって眠ねむっている小ちいさい動物どうぶつを見みつけました。
永瀬ながせさんが博物館はくぶつかんの人ひとに相談そうだんすると、この動物どうぶつはニホンヤマネだとわかりました。博物館はくぶつかんの人ひとは「ヤマネは冬ふゆになると、土つちの中なかや木きの穴あななどで眠ねむります。このヤマネは間違まちがって家いえの押おし入いれに入はいって、暖あたたかくて眠ねむってしまったと思おもいます」と話はなしていました。
永瀬ながせさんは、「ヤマネが起おきたら近ちかくの森もりに帰かえすつもりでしたが、先週せんしゅう、いなくなってしまいました。丸まるくてかわいかったので、もう少すこし長ながくいてほしかったです」と話はなしていました。

日本にっぽんとＥＵ　関税かんぜいをなくす新あたらしい貿易ぼうえきのルールが始はじまる

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


日本にっぽんとＥＵの新あたらしい貿易ぼうえきのルールが、２月がつ１日ついたちから始はじまりました。
日本にっぽんはＥＵから輸入ゆにゅうする物もののうち９４％の種類しゅるいの品物しなもので関税かんぜいをなくします。ワインは関税かんぜいを全部ぜんぶなくしました。カマンベールなどのソフトチーズは少すこしずつ関税かんぜいを減へらして、２０３３年ねんに全部ぜんぶなくします。服ふくも関税かんぜいをなくしました。かばんや革かわの靴くつは２０２９年ねんに関税かんぜいをなくします。
日本にっぽんがＥＵに輸出ゆしゅつする物ものは、９９%の種類しゅるいの品物しなもので関税かんぜいがなくなります。しょうゆ、みそ、日本酒にほんしゅ、緑茶りょくちゃ、牛肉ぎゅうにくは関税かんぜいが全部ぜんぶなくなりました。車くるまは２０２６年ねんに関税かんぜいがなくなります。
ＥＵには５億おく人にんが住すんでいます。新あたらしいルールが始はじまって、日本にっぽんからＥＵへの輸出ゆしゅつを増ふやしたいとたくさんの人ひとが考かんがえています。

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


厚生労働省こうせいろうどうしょうによると、１月がつ２７日にちまでの１週間しゅうかんに、日本にっぽんで約やく２２２万まん６０００人にんがインフルエンザになったと考かんがえられます。前まえの週しゅうより１０万まん人にんぐらい増ふえました。厚生労働省こうせいろうどうしょうによると、インフルエンザだとわかった人ひとは１つの病院びょういんで平均へいきん５７．０９人にんいました。調しらべ始はじめた１９９９年ねんから今いままでで、最もっとも多おおくなりました。埼玉県さいたまけん、新潟県にいがたけん、千葉県ちばけんで特とくに多おおくなっています。

東京とうきょうの品川区しながわくが「五反田ごたんだバレー」の会社かいしゃを応援おうえんする

发表于 2019-01-31 更新于 2019-11-30 分类于 JapanEasyNews


東京とうきょうの品川区しながわくにある五反田駅ごたんだえきの周まわりには、コンピューターのＡＩを使つかった製品せいひんやサービスを研究けんきゅうする新あたらしい会社かいしゃが集あつまっています。集あつまっている会社かいしゃは「五反田ごたんだバレー」という団体だんたいを作つくって、五反田ごたんだをアメリカのシリコンバレーのような場所ばしょにしたいと考かんがえています。
品川区しながわくは、このような会社かいしゃが仕事しごとをしやすいように応援おうえんすることにしました。品川区しながわくの商店街しょうてんがいをにぎやかにしたり、災害さいがいのとき役やくに立たつ物ものを作つくったりする会社かいしゃには、２年ねんで２０００万まん円えんまで出だします。会社かいしゃが考かんがえたことや作つくった物ものが役やくに立たつかどうかテストする場所ばしょも品川区しながわくが用意よういします。できてからまだ５年ねんになっていない会社かいしゃの場合ばあい、銀行ぎんこうなどからお金かねを借かりやすくします。
品川区しながわくは「たくさんの会社かいしゃが新あたらしいことに挑戦ちょうせんできるようにして、シリコンバレーと同おなじような場所ばしょに育そだてたいです」と話はなしています。
