「さっぽろ雪ゆきまつり」が始はじまる　今年ことしで７０回かい目め

发表于 2019-02-05 更新于 2019-11-30 分类于 JapanEasyNews


北海道ほっかいどう札幌市さっぽろしで行おこなう「さっぽろ雪ゆきまつり」は今年ことしで７０回かい目めです。大通おおどおり公園こうえんなどでは４日よっかから始はじまりました。会場かいじょうには、市民しみんなどが雪ゆきや氷こおりで作つくった像ぞうが１９０ぐらいあります。高たかさ１０ｍ以上いじょうの大おおきな像ぞうは５つあります。
そのうちの１つは、札幌さっぽろの有名ゆうめいな観光地かんこうちの前まえを走はしる馬うまの像ぞうです。去年きょねん９月がつの地震じしんで大おおきな被害ひがいがあった町まちでは、馬うまをたくさん育そだてています。この像ぞうは、町まちが元気げんきになるように願ねがって作つくられました。
３０歳さいの男性だんせいは「すばらしい像ぞうです。馬うまのように元気げんきに走はしって前まえのような町まちに戻もどってほしいです」と話はなしていました。
「さっぽろ雪ゆきまつり」は今月こんげつ１１日にちまでで、２５０万まん人にんぐらいが見みに来くると言いわれています。

日本にっぽんとドイツの首相しゅしょう「経済けいざいの関係かんけいを強つよくする」

发表于 2019-02-05 更新于 2019-11-30 分类于 JapanEasyNews


ドイツのメルケル首相しゅしょうが日本にっぽんに来きて、安倍あべ総理大臣そうりだいじんと会あって話はなしをしました。
安倍あべ総理大臣そうりだいじんは最初さいしょに、「日本にっぽんとＥＵの間あいだで関税かんぜいをなくす新あたらしい貿易ぼうえきのルールが始はじまりました。世界せかいの中なかで、日本にっぽんとドイツの役割やくわりが大おおきくなっています」と話はなしました。
メルケル首相しゅしょうは「日本にっぽんは、ドイツと同おなじようにルールを大切たいせつにする国くにです。ドイツと日本にっぽんが協力きょうりょくすることが大切たいせつです」と言いいました。
この話はなし合あいで、日本にっぽんとドイツが経済けいざいの関係かんけいを強つよくして、コンピューターのＡＩなど新あたらしい技術ぎじゅつで協力きょうりょくすることが決きまりました。
イギリスは今年ことし３月がつにＥＵから出でることになっています。安倍あべ総理大臣そうりだいじんは、イギリスとＥＵが貿易ぼうえきなどの約束やくそくを決きめることができるように協力きょうりょくしてほしいと言いいました。

日本にっぽんに住すんでいる中国人ちゅうごくじんの団体だんたいなどは春節しゅんせつのお祝いわいをするため、４日よっかの夜よる、東京とうきょうタワーを赤あかい光ひかりでライトアップしました。中国ちゅうごくでは赤あかは幸しあわせの色いろだと言いわれています。東京とうきょうタワーの下したにある広場ひろばでは、中国ちゅうごくの子こどもたちが歌うたを歌うたいました。

モネの「睡蓮すいれん」　破やぶれたり汚よごれたりした絵えを修理しゅうりする

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


東京とうきょうの国立西洋美術館こくりつせいようびじゅつかんは去年きょねん４月がつから、フランスの画家がかのクロード?モネがかいた「睡蓮すいれん、柳やなぎの反映はんえい」を修理しゅうりしています。
この絵えは１００年ねんぐらい前まえにロンドンやパリで絵えをたくさん集あつめた松方まつかた幸次郎こうじろうがモネから買かったと言いわれています。第だい２次世界大戦じせかいたいせんのあと、この絵えはどこにあるのかわからなくなりましたが、２０１６年ねんにパリのルーブル美術館びじゅつかんで見みつかりました。
絵えの大おおきさは縦たてが２ｍ、横よこが４mぐらいあります。しかし、上うえの半分はんぶんは破やぶれてなくなっていて、残のこっている部分ぶぶんもひどく汚よごれています。美術館びじゅつかんの専門家せんもんかたちは絵えについているかびなどの汚よごれを取とったり、絵えの裏うらに別べつの紙かみを貼はって絵えを強つよくしたりしています。
専門家せんもんかは「これほど悪わるくなった大おおきな絵えを修理しゅうりした話はなしは世界せかいでも聞きいたことがないので、とても大事だいじな仕事しごとになると思おもいます」と話はなしていました。
国立西洋美術館こくりつせいようびじゅつかんは今年ことし６月がつに「松方まつかたコレクション展てん」を開ひらいて、この絵えを初はじめて見みせる予定よていです。

眠ねむくなる働はたらきをする新あたらしい遺伝子いでんし　名前なまえは「ｎｅｍｕｒｉ」

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


アメリカのペンシルベニア大学だいがくで研究けんきゅうしている戸田とだ浩史ひろふみさんたちのグループが、眠ねむくなる働はたらきをする遺伝子いでんしを見みつけました。
ショウジョウバエというハエで、この遺伝子いでんしの働はたらきを強つよくすると、ハエが長ながい時間じかん眠ねむりました。遺伝子いでんしの働はたらきを止とめると、ハエが起おきている時間じかんが長ながくなりました。この遺伝子いでんしが働はたらくと、ハエは細菌さいきんに感染かんせんしても長ながく生いきたため、免疫めんえきにも関係かんけいがあるとグループは言いっています。
グループは、この遺伝子いでんしの名前なまえを日本語にほんごで「眠ねむること」という意味いみの「ｎｅｍｕｒｉ」にしました。
この遺伝子いでんしは、眠ねむる時間じかんが足たりなかったり細菌さいきんに感染かんせんしたりして、体からだにストレスがあるときに眠ねむくなるように働はたらくとグループは考かんがえています。
人ひとの体からだでこのハエと同おなじ遺伝子いでんしは見みつかっていませんが、戸田とださんは「風邪かぜのときに眠ねむくなって、眠ねむると治なおることも多おおいです。人ひとでも同おなじような働はたらきが起おこっていると思おもいます」と話はなしています。

アメリカで２日ふつか、日本にっぽんの細田守ほそだまもる監督かんとくの「未来みらいのミライ」が、アニー賞しょうの中なかの「インディペンデント作品さくひん賞しょう」をもらいました。この賞しょうは、アメリカで上映じょうえいした映画館えいがかんが少すくない映画えいがの中なかから選えらばれます。細田ほそだ監督かんとくは５１歳さいで、今いままでも外国がいこくの映画えいがのイベントなどで賞しょうをもらっています。

岡山県おかやまけん総社市そうじゃし　久ひさしぶりに集あつまってみんなで「恵方巻えほうまき」を食たべる

发表于 2019-02-04 更新于 2019-11-30 分类于 JapanEasyNews


岡山県おかやまけん総社市そうじゃしの下原地区しもばらちくは、去年きょねん７月がつに雨あめがたくさん降ふって、川かわの水みずがあふれたり、家いえに水みずが入はいってきたりしました。工場こうじょうで爆発ばくはつもありました。今いまも８０の家族かぞくが家いえに住すむことができなくて、別べつの場所ばしょで生活せいかつしています。
３日みっかは節分せつぶんで、別べつの場所ばしょで生活せいかつしている人ひとたち１００人にんぐらいが下原地区しもばらちくに集あつまりました。そして、節分せつぶんに食たべるといいことがあると言いわれている「恵方巻えほうまき」というすしを作つくりました。
下原地区しもばらちくの米こめを使つかって、６種類しゅるいの材料ざいりょうを一緒いっしょにのりで巻まいて恵方巻えほうまきを作つくりました。みんなは恵方巻えほうまきを食たべて、久ひさしぶりに会あった人ひとたちと楽たのしそうに話はなしていました。
恵方巻えほうまきを食たべた女性じょせいは「早はやく家いえに帰かえることができるように願ねがって食たべました。近所きんじょの人ひとたちに会あうことができてよかったです」と話はなしていました。

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


国連こくれんは、北朝鮮きたちょうせんが核兵器かくへいきやミサイルの実験じっけんをしたあと、北朝鮮きたちょうせんの輸入ゆにゅうや輸出ゆしゅつについての制裁せいさいを決きめました。そして、北朝鮮きたちょうせんについての制裁せいさいが守まもられているかどうか調しらべています。

天然記念物てんねんきねんぶつの「ニホンヤマネ」　毛布もうふの中なかで眠ねむっていた

发表于 2019-02-01 更新于 2019-11-30 分类于 JapanEasyNews


「ニホンヤマネ」は体からだの大おおきさが８ｃｍぐらいのネズミやリスに似にている動物どうぶつで、日本にっぽんだけにいる種類しゅるいです。１００万まん年ねん以上いじょう前まえからいるので、「生いきた化石かせき」と呼よばれていて、国くにの天然記念物てんねんきねんぶつになっています。
大分県おおいたけん日田市ひたしに住すんでいる永瀬ながせ豊子とよこさんは去年きょねん１２月がつの終おわりに、泊とまりに来くる家族かぞくのために押おし入いれの中なかから布団ふとんなどを出だしていました。そして、毛布もうふの中なかで丸まるくなって眠ねむっている小ちいさい動物どうぶつを見みつけました。
永瀬ながせさんが博物館はくぶつかんの人ひとに相談そうだんすると、この動物どうぶつはニホンヤマネだとわかりました。博物館はくぶつかんの人ひとは「ヤマネは冬ふゆになると、土つちの中なかや木きの穴あななどで眠ねむります。このヤマネは間違まちがって家いえの押おし入いれに入はいって、暖あたたかくて眠ねむってしまったと思おもいます」と話はなしていました。
永瀬ながせさんは、「ヤマネが起おきたら近ちかくの森もりに帰かえすつもりでしたが、先週せんしゅう、いなくなってしまいました。丸まるくてかわいかったので、もう少すこし長ながくいてほしかったです」と話はなしていました。
