日本にっぽんと韓国かんこく　市民しみんや子こどもが交流こうりゅうする行事ぎょうじが中止ちゅうしになる

发表于 2019-07-29 更新于 2019-11-30 分类于 JapanEasyNews



日本にっぽんと韓国かんこくの関係かんけいが悪わるくなっているため、交流こうりゅうの行事ぎょうじが中止ちゅうしになっています。
岐阜県ぎふけん大垣市おおがきしに来くる予定よていだった韓国かんこくのチャンウォン市しの少年しょうねんサッカーのチームは「国くにと国くにの関係かんけいが難むずかしくなっているので行いきません」と連絡れんらくしてきました。今いままで大垣市おおがきしとチャンウォン市しの少年しょうねんサッカーのチームはそれぞれの市しに行いって交流こうりゅうしていました。
大垣市おおがきしの男おとこの子こは「一緒いっしょに遊あそびたいと思おもっていました。残念ざんねんな気持きもちでいっぱいです」と話はなしていました。韓国かんこくの子こどもたちもとても残念ざんねんがっていると連絡れんらくがありました。
韓国かんこくの市民しみんや子こどもなどと交流こうりゅうする行事ぎょうじは、北海道ほっかいどう旭川市あさひかわしや岡山県おかやまけん備前市びぜんしや長崎県ながさきけん対馬市つしましなどでも中止ちゅうしになっています。
日本にっぽんと韓国かんこくは、戦争せんそう中ちゅうに韓国かんこくの人ひとを日本にっぽんの会社かいしゃで働はたらかせたことについての問題もんだいや、日本にっぽんが韓国かんこくに輸出ゆしゅつするときの決きまりを厳きびしくしたことなどで、関係かんけいが悪わるくなっています。

電気でんきで飛とぶ飛行機ひこうき　アメリカと日本にっぽんの会社かいしゃが一緒いっしょに研究けんきゅうする

发表于 2019-07-26 更新于 2019-11-30 分类于 JapanEasyNews



飛行機ひこうきを作つくっている会社かいしゃは、電気でんきで飛とぶ飛行機ひこうきを作つくる研究けんきゅうをしています。地球ちきゅうの気温きおんが上あがる原因げんいんになっている二酸化炭素にさんかたんそを少すくなくするためです。
飛行機ひこうきを作つくっているアメリカのボーイングは、日本にっぽんの会社かいしゃと一緒いっしょに電気でんきで飛とぶ飛行機ひこうきを作つくる研究けんきゅうをすることになりました。日本にっぽんでは、電気でんきで走はしる車くるまをたくさん作つくっています。このため、日本にっぽんの電池でんちやモーターなどを作つくる会社かいしゃには、安全あんぜんでいい品物しなものを作つくるための高たかい技術ぎじゅつがあります。
２５日にち、ボーイングの人ひとが経済産業省けいざいさんぎょうしょうに来きて、日本にっぽんの２０の会社かいしゃと話はなし合あいました。ボーイングの人ひとは「未来みらいの技術ぎじゅつを一緒いっしょに作つくることができる国くには日本にっぽんだけです」と話はなしました。
電池でんちを作つくる会社かいしゃの「ＧＳユアサ」は、飛行機ひこうきに使つかうための新あたらしい電池でんちを研究けんきゅうする計画けいかくがあると言いいました。
経済産業省けいざいさんぎょうしょうは、研究けんきゅうのためのお金かねを日本にっぽんの会社かいしゃに出だして協力きょうりょくします。

来年らいねん、東京とうきょうオリンピックを行おこなうときはとても暑あつくなりそうです。
東京とうきょうでは、暑あついとき体からだを冷つめたくするために、体からだにつけて使つかう商品しょうひんの展示てんじ会かいがありました。９０ぐらいの会社かいしゃが商品しょうひんを紹介しょうかいしました。

宮城県みやぎけんの会社かいしゃは、体からだを冷つめたくすることができる服ふくを紹介しょうかいしました。服ふくの中なかにはチューブがあって、リュックサックの中なかの氷こおりが入はいった水みずを送おくるようになっています。会社かいしゃは、東京とうきょうオリンピックの警備けいびの人ひとや外そとで働はたらく人ひとに使つかってほしいと言いっています。

東京とうきょうオリンピックの試合しあいを見みるツアーの受うけ付つけが始はじまる

发表于 2019-07-26 更新于 2019-11-30 分类于 JapanEasyNews



ＪＴＢなど３つの旅行りょこうの会社かいしゃが、来年らいねんの東京とうきょうオリンピックのチケットとホテルをセットにしたツアーの抽ちゅうせんの受うけ付つけを始はじめました。
ＪＴＢのツアーは全部ぜんぶで３７あります。いちばん高たかいツアーは４５０万まん円えんで、開会かいかい式しきと閉会へいかい式しき、そして柔道じゅうどうや陸上競技りくじょうきょうぎなどの試合しあいをオリンピックの間あいだ、毎日まいにち見みることができます。
いちばん安やすいツアーは２８万まん８０００円えんで、バレーボールなど１つのスポーツの試合しあいを５日いつか間かん見みることができます。このほかに、メダルが決きまる試合しあいだけを見みることができるツアーもあります。
８月がつ１５日にちまでにウェブサイトで申もうし込こんだ人ひとの中なかから、抽ちゅうせんで選えらばれた７８０人にんぐらいがツアーを買かうことができます。
近畿日本きんきにっぽんツーリストと東武とうぶトップツアーズも、ウェブサイトでツアーの受うけ付つけを始はじめました。３つの会社かいしゃは、これからツアーをもっと多おおくする予定よていです。

日本にっぽんには、１つの会社かいしゃに長ながく勤つとめて、何なん年ねん勤つとめたかで給料きゅうりょうや昇進しょうしんが決きまる習慣しゅうかんがあります。レポートでは、この習慣しゅうかんが日本にっぽんの発展はってんを邪魔じゃましていると言いっています。

東京とうきょうオリンピックが始はじまるまであと１年ねんになった２４日にじゅうよっか、メダルのデザインの発表はっぴょうがありました。
表おもては、ギリシャの女神めがみなどをデザインすることに決きまっています。裏うらには東京とうきょうオリンピックのマークが入はいっています。このメダルは、選手せんしゅが首くびにかけたときに、どこから見みても美うつくしく光ひかるデザインになっています。

ハンセン病びょうになった人ひとの家族かぞくは、国くにが間違まちがった考かんがえ方かたでハンセン病びょうの人ひとを療養りょうよう所じょに入いれ続つづけたため家族かぞくも差別さべつされたと言いって、裁判さいばんに訴うったえました。裁判所さいばんしょは先月せんげつ２８日にち、家族かぞくが差別さべつされたのは国くにに責任せきにんがあると言いいました。
