羽田空港はねだくうこう　日本にっぽんを出でる外国人がいこくじんの顔かおを機械きかいでチェックする

发表于 2019-07-25 更新于 2019-11-30 分类于 JapanEasyNews



羽田空港はねだくうこうなど５つの空港くうこうでは、日本人にっぽんじんが外国がいこくへ行いくときや帰かえってきたとき、機械きかいで顔かおのチェックをしています。
来年らいねん、東京とうきょうオリンピックなどで大勢おおぜいの外国人がいこくじんが日本にっぽんに来くると考かんがえられています。このため、羽田空港はねだくうこうでは２４日にじゅうよっかから、日本にっぽんを出でる外国人がいこくじんの顔かおのチェックもこの機械きかいでできるようにしました。旅行りょこうや仕事しごとなどで日本にっぽんにいた期間きかんが９０日にち以内いないだった外国人がいこくじんのチェックに使つかいます。
羽田空港はねだくうこうでは、外国人がいこくじんがパスポートを開ひらいて台だいに置おくと、カメラが顔かおの写真しゃしんを撮とって、パスポートの写真しゃしんと同おなじかどうかチェックしました。５秒びょうぐらいでゲートを通とおることができました。
出入国在留管理庁しゅつにゅうこくざいりゅうかんりちょうによると、機械きかいを使つかうと、日本にっぽんを出でる外国人がいこくじんより、日本にっぽんに入はいる外国人がいこくじんをチェックする職員しょくいんを多おおくできます。このため、テロなどがないように厳きびしくチェックできると言いっています。

うなぎ　今年ことしも高たかい値段ねだんが続つづきそう

发表于 2019-07-24 更新于 2019-11-30 分类于 JapanEasyNews



２７日にちは「土用どようのうしの日ひ」です。日本にっぽんではこの日ひにうなぎを食たべる習慣しゅうかんがあります。日本にっぽんで食たべる多おおくのうなぎは、海うみから子こどもを取とって育そだてて養殖ようしょくしています。
うなぎの子こどもは、取とれる量りょうが少すくなくなっています。水産庁すいさんちょうによると、今年ことし４月がつまでの６か月げつに日本にっぽんで取とれたうなぎの子こどもは３．７ｔで、今いままででいちばん少すくなくなりました。輸入ゆにゅうしたものを足たしても、養殖ようしょくするためのうなぎの子こどもは少すくなくなっています。
このため今年ことしも、うなぎは高たかい値段ねだんが続つづきそうです。養殖ようしょくする会社かいしゃが売うるときの値段ねだんは、２２日にちに１ｋｇ５２００円えんでした。これは今いままでで２番目ばんめに高たかい値段ねだんです。
うなぎの子こどもをたくさん取とっている中国ちゅうごくや韓国かんこく、台湾たいわんと一緒いっしょに増ふやす方法ほうほうを考かんがえることができるかどうか問題もんだいになっています。

国際天文学連合こくさいてんもんがくれんごう「星ほしの名前なまえをみんなで考かんがえよう」

发表于 2019-07-24 更新于 2019-11-30 分类于 JapanEasyNews



星ほしを研究けんきゅうしている専門家せんもんかなどが集あつまる国際天文学連合こくさいてんもんがくれんごうができてから、今年ことしで１００年ねんです。国際天文学連合こくさいてんもんがくれんごうは１００年ねんの記念きねんに、まだ名前なまえがない星ほしの名前なまえを、みんなに考かんがえてもらうことにしました。
国際天文学連合こくさいてんもんがくれんごうは、７０以上いじょうの国くにと地域ちいきがどの星ほしの名前なまえを考かんがえるか決きめています。日本にっぽんでは、１９年ねん前まえに日本人にっぽんじんが見みつけた恒星こうせいと、その周まわりを回まわっている星ほしの名前なまえを考かんがえることになりました。この恒星こうせいは地球ちきゅうから４１０光年こうねんの所ところにあります。
日本にっぽんの国立天文台こくりつてんもんだいのウェブサイトに、星ほしの名前なまえとその名前なまえにした理由りゆうを書かいて応募おうぼします。名前なまえは４字じから１６字じまでのローマ字じです。９月がつ４日よっかまで応募おうぼできます。
ウェブサイトに集あつまった中なかから日本にっぽんで名前なまえを選えらんだあと、国際天文学連合こくさいてんもんがくれんごうがチェックをして、１２月がつに名前なまえを発表はっぴょうします。
