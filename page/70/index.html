新あたらしい元号げんごうは「令和れいわ」に決きまる

发表于 2019-04-01 更新于 2019-11-30 分类于 JapanEasyNews



元号げんごうは日本にっぽんにある年としの数かぞえ方かたで、天皇てんのうがかわるときに新あたらしくなります。５月がつ１日ついたちに皇太子こうたいしさまが新あたらしい天皇てんのうになるときに、「平成へいせい」が終おわって新あたらしい元号げんごうにかわります。
政府せいふは４月がつ１日ついたち、いろいろな専門家せんもんかから意見いけんを聞きいたあと、新あたらしい元号げんごうを「令和れいわ」にすると決きめて、発表はっぴょうしました。「令和れいわ」は、１２００年ねん以上いじょう前まえの「万葉集まんようしゅう」にある言葉ことばから選えらばれました。
安倍あべ総理大臣そうりだいじんは「令和れいわ」について「みんなの心こころが集あつまって文化ぶんかが生うまれて育そだつという意味いみです。寒さむさのあと春はるが来くるときれいに咲さく梅うめの花はなのように、１人ひとり１人ひとりの日本人にっぽんじんがそれぞれの花はなを大おおきく咲さかせる日本にっぽんになってほしいと思おもいました」と話はなしました。

世界せかいで明あかりを消けすイベント「地球ちきゅうを大切たいせつにしよう」

发表于 2019-04-01 更新于 2019-11-30 分类于 JapanEasyNews



世界せかいのいろいろなまちで３月がつ３０日にち、有名ゆうめいな建物たてものの電気でんきなどの明あかりを消けす「アースアワー」というイベントがありました。
地球ちきゅうの気温きおんが上あがらないようにすることや、環境かんきょうを大切たいせつにすることを考かんがえてもらうために、毎年まいとし行おこなわれています。今年ことしは１８０以上いじょうの国くにや地域ちいきで、午後ごご８時じ半はんから１時間じかん、明あかりを消けしました。
日本にっぽんでは、東京とうきょうタワーや東京とうきょうスカイツリーなど全部ぜんぶで１４００以上いじょうの場所ばしょで明あかりを消けしました。
インドのニューデリーでは、国くにの旗はたの色いろでライトアップしたインド門もんの明あかりを消けしました。ロシアのサンクトペテルブルクでは、エルミタージュ美術館びじゅつかんなど、まちの中なかの建物たてものの明あかりを少すこしずつ消けしました。
ドイツのベルリンでは、集あつまった大勢おおぜいの人ひとが、明あかりが消きえるブランデンブルク門もんをスマートフォンで撮とったり、拍手はくしゅをしたりしていました。

４月がつ１日ついたちからいろいろな会社かいしゃが、食たべ物ものや飲のみ物ものの値段ねだんを上あげました。働はたらく人ひとに払はらうお金かねや、物ものを運はこぶときにかかるお金かねが上あがっているためです。

スーパーなどが売うる値段ねだんも上あがりそうです。

日本にっぽんの二酸化炭素にさんかたんその濃度のうどが今いままででいちばん高たかくなる

发表于 2019-03-29 更新于 2019-11-30 分类于 JapanEasyNews



二酸化炭素にさんかたんそは地球ちきゅうの気温きおんが上あがる原因げんいんになります。気象庁きしょうちょうは、二酸化炭素にさんかたんその濃度のうどを日本にっぽんの３つの場所ばしょで調しらべています。去年きょねんの平均へいきんの濃度のうどは、岩手県いわてけんの大船渡市おおふなとしで４１２ｐｐｍ、沖縄県おきなわけんの与那国島よなぐにじまで４１１．７ｐｐｍ、小笠原諸島おがさわらしょとうの南鳥島みなみとりしまで４０９．４ｐｐｍでした。全部ぜんぶの場所ばしょで今いままででいちばん高たかくなりました。
国連こくれんは、地球ちきゅうの気温きおんが２℃以上いじょう上あがらないように、世界せかいの二酸化炭素にさんかたんその平均へいきんの濃度のうどを４２０ｐｐｍぐらいまでにしようと言いっています。しかし、日本にっぽんの二酸化炭素にさんかたんその濃度のうどは高たかくなり続つづけています。あと５年ねんぐらいで４２０ｐｐｍ以上いじょうになるかもしれません。
気象庁きしょうちょうは「二酸化炭素にさんかたんそを出ださないように、もっと努力どりょくしなければなりません」と話はなしています。
