hexo博客验证百度站长

发表于 2019-10-23 分类于 问题集

问题

验证百度站长有3个方法(1.文件验证，2.HTML标签验证，3.CNAME验证)。这里要解决的是hexo博客文件验证方法。
下载文件放在source的根目录下。

回到验证页面，点击完成验证按钮。
提示验证失败。

原因

hexo发布网站的时候渲染了百度验证文件，导致百度无法辨认自己的验证文件。

解决方法

打开验证文件，添加一行代码。
layout: false
hexo发布网站的时候就不会渲染此百度验证文件。


重新验证，恭喜，验证成功。

hexo发布之后GitHub Pages自定义域名失效

发表于 2019-10-22 更新于 2019-10-23 分类于 问题集

问题

hexo deploy之后，重新访问自己的域名会404，重新查看仓库的设置发现Custom domain项变成空白。

原因

deploy内容中没有关于域名的文件

解决方法

source 目录下新建一个CNAME文件(没有后缀)，然后在这个文件中填入自己的域名。
重新deploy，访问自己的域名没有404。
