飼かっている犬いぬには必かならず狂犬病きょうけんびょうのワクチンの注射ちゅうしゃをして

发表于 2019-11-01 更新于 2019-12-03 分类于 JapanEasyNews



東京都とうきょうと瑞穂町みずほまちで今年ことし８月がつ、高校生こうこうせいが犬いぬに足あしをかまれました。警察けいさつがこの犬いぬと一緒いっしょに散歩さんぽしていた人ひとを調しらべて、犬いぬが狂犬病きょうけんびょうのワクチンを受うけていないことがわかりました。検査けんさした結果けっか、この犬いぬは狂犬病きょうけんびょうになっていませんでした。
犬いぬを飼かっている人ひとは１年ねんに１回かい、犬いぬに狂犬病きょうけんびょうのワクチンを注射ちゅうしゃしなければならないと法律ほうりつで決きまっています。この犬いぬを飼かっていた人ひとは「必かならず注射ちゅうしゃをしなければならないとは思おもっていませんでした」と話はなしていました。警察けいさつは調しらべたことを検察庁けんさつちょうに伝つたえました。
狂犬病きょうけんびょうは人ひとにもうつる病気びょうきです。狂犬病きょうけんびょうの犬いぬにかまれて、ウイルスが傷きずから体からだの中なかに入はいって病気びょうきになると、ほとんどの人ひとが亡なくなります。
厚生労働省こうせいろうどうしょうは、犬いぬを飼かっている人ひとは住すんでいる所ところの役所やくしょに犬いぬの登録とうろくをして、犬いぬには必かならず狂犬病きょうけんびょうのワクチンを注射ちゅうしゃするように言いっています。

「渋谷駅しぶやえきが変かわる」地下ちかに新あたらしい広場ひろばができる

发表于 2019-11-01 更新于 2019-12-03 分类于 JapanEasyNews



東京とうきょうの渋谷駅しぶやえきでは、駅えきを変かえる大おおきな計画けいかくが進すすんでいます。
１１月がつ１日ついたち、渋谷駅しぶやえきの地下ちかに新あたらしい広場ひろばができました。この広場ひろばはＪＲの山手線やまのてせんや東急電鉄とうきゅうでんてつの東横線とうよこせん、地下鉄ちかてつの銀座線ぎんざせんなど、たくさんの電車でんしゃの改札かいさつをつなぐ場所ばしょになっています。広ひろさは１６００m²です。
駅えきのそばには、４７階かい建だての新あたらしいビルもオープンしました。
渋谷駅しぶやえきは、１００年ねんぐらい前まえに駅えきの建物たてものが今いまの場所ばしょに移うつったあと、いちばん大おおきく変かわります。
広場ひろばができたことをお祝いわいする式しきで、渋谷区しぶやくの区長くちょうは「渋谷しぶやは地下ちかも大おおきく発展はってんしています。たくさんの人ひとが広場ひろばを利用りようして、渋谷しぶやがもっとにぎやかになったらうれしいです」と言いいました。

猿さるが果物くだものを食たべに来こないように働はたらく犬いぬ

发表于 2019-10-31 更新于 2019-11-30 分类于 JapanEasyNews



愛媛県えひめけん松山市まつやましでは、山やまから来きた猿さるが、農家のうかが育そだてている果物くだものや野菜やさいを食たべてしまって、その被害ひがいは１年ねんに２００万まん円えん以上いじょうになっています。
このため松山市まつやましでは、特別とくべつな訓練くんれんをした５匹ひきの犬いぬが、猿さるが来こないようにする仕事しごとをしています。果物くだものがある所ところの近ちかくを歩あるいて猿さるをさがしたり、見みつけた猿さるを脅おどしたりします。この犬いぬは「モンキードッグ」と呼よばれています。
２８日にち、３か月げつの訓練くんれんが終おわった犬いぬが、６匹ぴき目めのモンキードッグになりました。名前なまえは「キヨ」で、３歳さいの雌めすです。キヨはこれから、みかんや栗くりなどの被害ひがいが特とくに多おおい所ところで仕事しごとを始はじめます。キヨを飼かっている男性だんせいは「早はやくしっかり仕事しごとができる犬いぬになってほしいです」と話はなしていました。
